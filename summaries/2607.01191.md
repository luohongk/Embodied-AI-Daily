# Perceive-to-Reason: Decoupling Perception and Reasoning for Fine-Grained Visual Reasoning

- arXiv: https://arxiv.org/abs/2607.01191v1
- arXiv ID: 2607.01191

---

🔍 **核心问题**：解决视觉语言模型在高分辨率图像中难以捕捉微小关键视觉线索，导致细粒度视觉推理能力不足的问题。

🛠 **方法**：提出Perceive-to-Reason（P2R）框架，将细粒度视觉推理解耦为感知（定位问题相关证据）和推理（基于标注图像和裁剪区域回答问题）两个阶段，并引入角色感知的强化学习策略PRA-GRPO，交替优化感知与推理能力。

✨ **主要贡献**：在Qwen3-VL-Instruct系列模型上，P2R显著提升细粒度视觉推理性能，例如P2R-4B在V-Star、HR-Bench-4K和HR-Bench-8K上分别达到93.2%、81.9%和80.5%，超越原骨干模型，且该方法可泛化至更广泛的多模态推理任务。
