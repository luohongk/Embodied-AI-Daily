# Embodied Human-Robot Interaction via Acoustics: A MARL Approach with AcoustoBots for Spatial Data Physicalization

- arXiv: https://arxiv.org/abs/2607.06563v1
- arXiv ID: 2607.06563

---

🔍 **核心问题**：传统数据物理化方式静态且脱离真实环境，难以传达空间动态并吸引用户交互，尤其在体现空间数据与机器人协作方面存在局限。

🛠 **方法**：提出AcoustoBots平台，利用搭载8x8超声相控阵的TurtleBot3机器人，通过声悬浮粒子高度（1-10厘米）编码城市标量数据；采用基于MADDPG的多智能体强化学习策略实现碰撞感知导航，结合GS-PAT声学控制器维持悬浮稳定并动态调整相位。

✨ **主要贡献**：首次将声悬浮技术用于移动机器人数据物理化，实现运动中的稳定悬浮与位置相关高度渲染；在4m×3m英国地图上验证单/双机器人任务，成功率分别达90%和80%，碰撞次数低，为具身人机交互提供简洁直观的通信线索。
