# Locality-Aware Density Control for Efficient Gaussian-based Image Representation

- arXiv: https://arxiv.org/abs/2607.17896v1
- arXiv ID: 2607.17896

---

🔍 **核心问题**：现有基于2D高斯泼溅的图像表示方法在优化过程中无法高效分配高斯容量，导致欠重建区域碎片化细化、过重建区域冗余保留，降低了表示质量。

🛠 **方法**：提出Locality-Aware Density Control（LocoADC）框架，包含区域级高斯致密化（RGD）策略和相似性驱动高斯合并（SDGM）策略，并引入局部颜色一致性约束以提升合并可靠性。

✨ **主要贡献**：通过利用图像空间和参数空间的双重局部性，LocoADC显著提升了高斯容量利用率，在CLIC数据集上以相同3万高斯预算相比GI方法获得2.93 dB的PSNR增益，且作为即插即用模块可稳定改进多种基线模型。
