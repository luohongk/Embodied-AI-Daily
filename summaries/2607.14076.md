# From Pixels to States: Rethinking Interactive World Models as Game Engines

- arXiv: https://arxiv.org/abs/2607.14076v1
- arXiv ID: 2607.14076

---

🔍 **核心问题**：如何利用视频生成模型构建能实时响应玩家动作、遵循游戏规则并保持长期一致性的交互式游戏世界。

🛠 **方法**：以传统游戏引擎的“动作-状态-观测”循环为框架，从玩家动作控制、游戏状态动态、状态-观测持久性和实时交互生成四个维度分析现有方法，并构建了一个包含90小时《黑神话：悟空》游戏数据（含对齐的玩家动作、真实游戏状态和视觉观测）的可扩展数据引擎。

✨ **主要贡献**：系统梳理了交互式游戏世界建模的四个关键维度及各类方法的优劣，同时提供了首个大规模、带结构化语义标注的状态感知游戏数据集，为推进状态感知的交互式游戏世界建模奠定了基础。
