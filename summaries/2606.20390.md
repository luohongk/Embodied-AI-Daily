# Geometry-Aware Superpixel Graph Transformer with Metadata for Skin Lesion Classification

- arXiv: https://arxiv.org/abs/2606.20390v1
- arXiv ID: 2606.20390

---

🔍 **核心问题**：皮肤镜图像中良恶性病变分类因病灶结构异质、类内差异大及视觉差异细微而面临挑战，现有CNN/ViT方法依赖全局或块级特征且多模态融合受限。

🛠 **方法**：提出基于超像素区域的图学习框架，将病灶建模为空间连贯超像素区域构成的图，编码区域间几何关系作为边属性，并引入元数据上下文节点实现人口学/临床变量的结构化融合，通过边缘感知图变换器更新节点表示。

✨ **主要贡献**：在四个公开数据集上验证了显式区域级关系建模和图原生多模态融合优于现有方法，建立了以CNN特征为关系节点并通过上下文集成增强表达的新图视角。
