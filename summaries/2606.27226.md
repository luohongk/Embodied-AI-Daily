# Ask, Don't Judge: Binary Questions for Interpretable LLM Evaluation and Self-Improvement

- arXiv: https://arxiv.org/abs/2606.27226v1
- arXiv ID: 2606.27226

---

🔍 **核心问题**：当前LLM评估存在人工评估昂贵缓慢、词汇指标与人类判断相关性差、整体性LLM评分器输出不透明且难以调试的问题。

🛠 **方法**：提出BINEVAL框架，将评估标准分解为原子化的二元问题，由LLM独立回答每个问题，汇总后生成可解释的多维评分和细粒度反馈。

✨ **主要贡献**：在SummEval、Topical-Chat和QAGS等基准上匹配或超越UniEval、G-Eval等强基线，尤其在事实一致性任务上表现突出；能更好匹配人类评分分布、避免天花板效应；支持基于问题级反馈的迭代提示优化，兼具强实证性能与实用诊断价值。
