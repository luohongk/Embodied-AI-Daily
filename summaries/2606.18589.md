# DREAM-Chunk: Reactive Action Chunking with Latent World Model

- arXiv: https://arxiv.org/abs/2606.18589v1
- arXiv ID: 2606.18589

---

🔍 **核心问题**：动作分块策略在开环执行时，面对随机动力学、硬件误差和部分可观测性时鲁棒性不足。

🛠 **方法**：提出DREAM-Chunk，在测试时使用轻量级潜在世界模型对多个候选动作分块进行未来状态预测，并选择与观测轨迹最匹配的分块执行。

✨ **主要贡献**：无需微调策略模型，通过增加测试时计算量提升动作分块策略在随机环境中的鲁棒性，在Kinetix基准和多种机器人任务上验证了有效性。
