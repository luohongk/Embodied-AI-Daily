# HeteroMosaic: Exposing and Exploiting Heterogeneous Execution Opportunities for Energy-Efficient Edge LLM Inference

- arXiv: https://arxiv.org/abs/2607.12839v1
- arXiv ID: 2607.12839

---

🔍 **核心问题**：现有边缘LLM推理运行时通常仅进行粗略的设备选择或孤立优化算子，未能充分利用CPU、iGPU和NPU等异构资源，尤其在统一内存平台上性能受限于设备放置与任务图协调。

🛠 **方法**：提出HeteroMosaic框架，通过异构屋顶模型识别iGPU与NPU协同执行的收益，将推理分解为保留依赖的微批次以暴露跨加速器重叠，并采用迹引导的调度与设备分配联合优化，考虑内存竞争、DVFS、设备差异及NPU运行时开销等实际因素。

✨ **主要贡献**：在AMD Ryzen AI平台上，HeteroMosaic相比iGPU基线实现最高1.73倍加速，比NPU基线快1.78倍，比llama.cpp等框架快2.05倍，同时能耗降低45.3%；相比现有异构边缘AI方案性能提升最高2.35倍。
