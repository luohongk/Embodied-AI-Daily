# HTNav: A Hybrid Navigation Framework with Tiered Structure for Urban Aerial Vision-and-Language Navigation

- arXiv: https://arxiv.org/abs/2604.08883v1
- arXiv ID: 2604.08883

---

🔍 **核心问题**：解决城市空中视觉语言导航（VLN）在复杂环境中泛化能力差、长距离路径规划效果不佳及空间连续性理解不足的问题。

🛠 **方法**：提出HTNav混合导航框架，融合模仿学习（IL）与强化学习（RL），采用分阶段训练机制，并集成层级决策机制实现宏观路径规划与细粒度动作控制的协同，同时引入地图表示学习模块增强空间连续性理解。

✨ **主要贡献**：在CityNav基准上达到所有场景难度下的最优性能，显著提升了复杂城市环境中的导航精度与鲁棒性。
