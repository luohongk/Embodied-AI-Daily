# Semantic Occupancy Prediction with Dual Range-Voxel Representation

- arXiv: https://arxiv.org/abs/2606.31688v1
- arXiv ID: 2606.31688

---

🔍 **核心问题**：解决基于LiDAR点云的3D语义占用预测中，多帧点云堆叠带来的计算负担和位姿噪声问题，实现单帧点云的高效准确预测。

🛠 **方法**：提出双范围-体素表示（DRVR），利用单帧点云的范围视图上下文和体素视图几何信息，通过范围视图编码器提取紧凑场景特征，设计几何感知体素视图编码器提取多尺度特征，并采用范围-体素融合模块实现双向特征交互。

✨ **主要贡献**：在nuScenes-Occupancy数据集上，单帧DRVR相比多帧方法mIoU提升5.4%，推理速度加速2.1倍，并在SemanticKITTI和SemanticPOSS上验证了优越性。
