# CaP-X: A Framework for Benchmarking and Improving Coding Agents for Robot Manipulation

- arXiv: https://arxiv.org/abs/2603.22435v2
- arXiv ID: 2603.22435

---

🔍 **核心问题**：当前“代码即策略”（Code-as-Policy）方法在机器人操作任务中作为自主控制器的有效性尚未被系统评估，且现有模型过度依赖人工设计的抽象层，缺乏鲁棒性。

🛠 **方法**：提出CaP-X框架，包含交互环境CaP-Gym和评估基准CaP-Bench，通过多轮交互、结构化执行反馈、视觉差分、自动技能合成和集成推理等测试时计算扩展策略，提升代码代理在低层级原语下的表现；并引入CaP-RL，利用可验证奖励的强化学习实现仿真到真实的无缝迁移。

✨ **主要贡献**：系统揭示了代码代理对人工抽象层的依赖，并证明通过扩展测试时计算可显著弥补这一差距；提出无需训练的CaP-Agent0框架，在仿真和真实机器人上恢复人类级可靠性；CaP-RL方法在sim2real迁移中实现极小性能差距，为具身代码代理研究提供了开源、可复现的平台。
