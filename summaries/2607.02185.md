# RadiomicNet: A Hybrid Radiomics-Guided Lightweight Architecture for Interpretable Medical Image Segmentation

- arXiv: https://arxiv.org/abs/2607.02185v1
- arXiv ID: 2607.02185

---

🔍 **核心问题**：解决深度学习在医学图像分割中参数庞大、数学不可解释且缺乏临床可解释性的问题。

🛠 **方法**：提出RadiomicNet双流混合架构，通过Radiomics Attention Gate（RAG）将GLCM和LBP手工特征融入MobileNetV2编码器-解码器的跳跃连接注意力机制，并引入Radiomics Consistency Loss约束纹理复杂度与预测不确定性。

✨ **主要贡献**：在BUSI和Kvasir-SEG数据集上分别达到0.763和0.854的Dice系数，优于U-KAN（p<0.05），参数仅3.27M（比U-Net少9.5倍）；通过梯度分析揭示GLCM差异度（15.24%）、能量（14.56%）和LBP熵（11.49%）为关键可解释特征，实现轻量级、可解释的临床级分割性能。
