# AdaJEPA: An Adaptive Latent World Model

- arXiv: https://arxiv.org/abs/2606.32026v1
- arXiv ID: 2606.32026

---

🔍 **核心问题**：解决潜在世界模型在测试时因分布偏移导致预测不准确，进而使规划失败的问题。

🛠 **方法**：提出AdaJEPA，一种自适应潜在世界模型，在模型预测控制（MPC）闭环中执行测试时自适应，利用观测到的下一状态转换作为自监督信号，在每次重规划时更新模型。

✨ **主要贡献**：仅需每个MPC重规划步骤进行一次梯度更新，即可显著提升多种目标到达任务中的规划成功率，无需额外专家演示。
