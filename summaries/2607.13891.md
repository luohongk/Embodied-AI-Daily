# PiVoT: A Variational Solution for Real-time Large-scale Multi-object Detection and Tracking under Heavy Clutter

- arXiv: https://arxiv.org/abs/2607.13891v1
- arXiv ID: 2607.13891

---

🔍 **核心问题**：在严重杂波、大量目标和全分辨率多普勒点云场景下，现有贝叶斯跟踪器难以兼顾精度与效率，且依赖外部聚类或检测器。

🛠 **方法**：提出PiVoT，一种基于变分推理的实时多目标检测与跟踪方法，联合推断目标状态、形状、存在概率、数据关联和测量率，无需训练或外部模块。

✨ **主要贡献**：通过理论化的出生剪枝、二次到线性复杂度优化及高效多普勒泊松模型，实现千级目标的可扩展性、对视觉不可分杂波的鲁棒性，并在全尺寸汽车雷达数据集上达到与深度学习基准相当的实时性能。
