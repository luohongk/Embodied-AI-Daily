# PLATE: Plasticity-Tunable Efficient Adapters for Geometry-Aware Continual Learning

- arXiv: https://arxiv.org/abs/2602.03846v2
- arXiv ID: 2602.03846

---

🔍 **核心问题**：解决预训练模型在持续学习中无法访问旧任务数据时的灾难性遗忘问题，尤其适用于预训练分布不可用的实际场景。

🛠 **方法**：提出PLATE方法，利用预训练网络的几何冗余性，通过结构化低秩更新（ΔW = B A Q^⊤）限制参数更新空间，其中B和Q从预训练权重中一次性计算并冻结，仅训练矩阵A。

✨ **主要贡献**：无需旧任务数据即可显式控制可塑性-保留权衡，通过冗余神经元构建受保护更新子空间，降低功能漂移并提升最坏情况下的保留保证。
