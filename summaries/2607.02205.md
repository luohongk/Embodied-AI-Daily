# Actuator Reality Shaping for Zero-Shot Sim-to-Real Robot Learning

- arXiv: https://arxiv.org/abs/2607.02205v1
- arXiv ID: 2607.02205

---

🔍 **核心问题**：解决机器人学习中仿真到现实迁移时，因物理电机非线性、硬件依赖行为与仿真中理想执行器动力学不匹配导致的性能下降问题。

🛠 **方法**：提出“执行器现实塑造”范式，通过为每个关节配备二自由度前馈-反馈控制器，使物理执行器的闭环行为匹配仿真中使用的理想二阶参考动力学，而非修改仿真器。

✨ **主要贡献**：实现了零样本的仿真到现实迁移，无需任务级微调或学习执行器模型；在单关节伺服、7自由度机械臂、轮腿机器人和人形机器人上验证，显著降低跟踪误差并提升任务性能。
