# Automating Potential-based Reward Shaping with Vision Language Model Guidance

- arXiv: https://arxiv.org/abs/2606.27180v1
- arXiv ID: 2606.27180

---

🔍 **核心问题**：强化学习在稀疏奖励环境中缺乏中间反馈，导致探索困难且易出现奖励破解问题，传统奖励塑形方法需专家设计势函数。

🛠 **方法**：提出VLM-PBRS框架，利用轻量级视觉语言模型（VLM）对图像对进行偏好比较，基于这些偏好训练势函数模型，并将其用于基于势能的奖励塑形（PBRS），从而保留最优策略集。

✨ **主要贡献**：首次将VLM偏好学习用于合成PBRS势函数；提出低成本的轻量VLM解决方案；在Meta-World和Franka Kitchen环境中验证了该方法能提升样本效率并增强对奖励破解的鲁棒性。
