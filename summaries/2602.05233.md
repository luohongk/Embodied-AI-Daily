# MobileManiBench: Simplifying Model Verification for Mobile Manipulation

- arXiv: https://arxiv.org/abs/2602.05233v2
- arXiv ID: 2602.05233

---

🔍 **核心问题**：现有视觉-语言-动作模型依赖大规模遥操作采集的静态桌面场景数据集，缺乏对移动机器人操作任务的验证框架和基准。

🛠 **方法**：提出基于NVIDIA Isaac Sim的仿真优先框架，利用强化学习自动生成多样化操作轨迹，并构建包含2种移动平台、630个物体、5类技能和30万条轨迹的MobileManiBench基准。

✨ **主要贡献**：首次提供大规模移动操作仿真基准，支持可控的机器人形态、感知模态和策略架构研究，通过代表性VLA模型评估揭示了复杂仿真环境中的感知、推理与控制挑战，并开源全部代码、数据集和模型。
