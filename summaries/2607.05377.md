# Cortex: A Bidirectionally Aligned Embodied Agent Framework for Long-horizon Manipulation

- arXiv: https://arxiv.org/abs/2607.05377v1
- arXiv ID: 2607.05377

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在处理长周期操作任务时，因马尔可夫性质（仅依赖当前观测）而表现不佳，且分层双系统方法存在高层规划语义与低层执行运动学之间的鸿沟。

🛠 **方法**：提出Cortex框架，通过双向对齐的规划接口将高层VLM的可行子任务计划传递给低层VLA。具体包括：将操作子任务标准化为32种规范技能原语，注入可执行性原理（如对象属性与轨迹可达性）到数据生成流程，自动标注超4000小时开源视频并生成30小时仿真数据，并设计事件平衡采样策略与推理时的任务上下文约束。

✨ **主要贡献**：在Libero-long和RoboTwin基准上分别比单模型基线提升3.1%和4.1%；其通用VLM结合微调VLA可零样本完成未见过的真实世界长周期任务（如多阶段化学实验），而单独微调VLA无法实现。
