# Latency Prediction for LLM Inference on NPU Systems

- arXiv: https://arxiv.org/abs/2606.18042v1
- arXiv ID: 2606.18042

---

🔍 **核心问题**：在NPU系统上部署大语言模型（LLM）时，由于配置空间庞大（包括并行策略、批处理技术和调度策略），难以通过穷举测量来预测推理延迟，而现有方法无法直接适用于NPU，因其面临微架构不公开、编译器优化不可预测以及分桶导致的延迟非线性等挑战。

🛠 **方法**：提出LENS延迟预测器，无需微架构或编译器信息，通过每个桶的两次端到端测量来建模分桶导致的非线性延迟，并组合结果预测任意输入-输出长度组合的延迟。

✨ **主要贡献**：在多个厂商的NPU、多种LLM及多样化工作负载上验证，平均预测误差仅为2.15%，并通过与两个方法相关的基线对比，确认了其有效性。
