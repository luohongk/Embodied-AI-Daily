# Uncertainty Quantification for EO Regression Tasks: Building Height, Tree Canopy Height and Above-ground Biomass Estimation

- arXiv: https://arxiv.org/abs/2607.11412v1
- arXiv ID: 2607.11412

---

🔍 **核心问题**：针对地球观测回归任务（如建筑高度、树冠高度和地上生物量估算）中深度学习模型仅输出确定性预测、缺乏逐像素可靠性评估的问题，提出不确定性量化方法以提升推断可靠性。

🛠 **方法**：基于全年Sentinel-1 SAR和Sentinel-2 MSI时间序列数据，提出两种互补的不确定性建模方法：高斯不确定性（假设高斯分布联合预测均值与标准差）和分位数不确定性（估计第10、50、90百分位数以捕捉非对称异方差误差分布）。

✨ **主要贡献**：两种方法在10米空间分辨率的三项代表性回归任务中均达到或超越确定性基准与现有全球产品，提供校准良好、可解释且实用的置信度估计；尤其在树冠高度估算中，性能优于当前最先进的10米分辨率不确定性感知模型。
