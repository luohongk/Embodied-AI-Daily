# KYON: Semi-Modular Wheel-Legged Quadruped With Agile Bimanual Capability

- arXiv: https://arxiv.org/abs/2606.30243v1
- arXiv ID: 2606.30243

---

🔍 **核心问题**：现有轮腿混合机器人通常难以兼顾敏捷的轮式/腿式运动与双臂操作能力，尤其在非结构化环境中缺乏鲁棒性。

🛠 **方法**：提出半模块化设计，将驱动器置于机身并通过传动机构降低远端惯性；结合全身控制框架与强化学习策略，分别处理运动与操作的动态非线性及扰动鲁棒性。

✨ **主要贡献**：实现了动态轮腿运动与双臂灵巧操作的独立执行，实验验证了机器人在复杂非结构化场景中的有效性与鲁棒性。
