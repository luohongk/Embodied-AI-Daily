# Semantic Anchoring for Robotic Action Representations

- arXiv: https://arxiv.org/abs/2607.13597v1
- arXiv ID: 2607.13597

---

🔍 **核心问题**：视觉-语言-动作（VLA）模型在微调时丢失了预训练视觉-语言模型中的语义结构，导致泛化能力下降，因此需要明确什么是良好的动作表征。

🛠 **方法**：提出一种即插即用的方法，将动作表征锚定到语义流形上，并将其分解为共享语义通道和私有通道，推理时丢弃私有通道，保持部署模型不变。

✨ **主要贡献**：验证了动作表征的语义结构与任务成功率和分布外泛化能力同步；在仿真和真实世界基准上，该方法在分布内任务中提升最高达+18.7%，在分布外泛化中提升最高达+21.5%。
