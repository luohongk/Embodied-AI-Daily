# Cognition to Control - Multi-Agent Learning for Human-Humanoid Collaborative Transport

- arXiv: https://arxiv.org/abs/2603.03768v1
- arXiv ID: 2603.03768

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）系统多强调反应式行为（System 1），难以将长期推理（System 2）与低延迟连续控制有效整合，尤其在多人机协作中，协调决策与物理执行需在接触、可行性和安全约束下共同演化。

🛠 **方法**：提出三层认知-控制（C2C）层次架构：①基于VLM的接地层，维护场景参照并推断具身感知的可行性与约束；②基于去中心化多智能体强化学习（MARL）的推理层，将人机耦合建模为马尔可夫势博弈，优化长期技能选择与序列；③全身控制层，高频执行选定技能并保证运动学/动力学可行性与接触稳定性。

✨ **主要贡献**：通过残差策略实现推理层与标称控制器的结合，无需显式角色分配即可内化伙伴动态；在协作操作任务中，相比单智能体和端到端基线，显著提升成功率和鲁棒性，并涌现稳定的领导者-跟随者行为。
