# Hierarchical Fine-Grained Aerial Object Detection

- arXiv: https://arxiv.org/abs/2606.16448v1
- arXiv ID: 2606.16448

---

🔍 **核心问题**：现有细粒度航空目标检测方法沿用粗粒度检测范式，仅依赖单标签监督，难以区分结构差异细微的型号级目标（如波音787与737）。

🛠 **方法**：提出ExpertDet方案，包含两个关键技术：视觉感知掩码属性建模（VMAM）通过重建随机掩码属性对齐视觉结构与属性语义；层次视觉实例提升（HierVIP）基于层次关系构建视觉原型树，施加分类感知约束以保持跨层级语义连续性。

✨ **主要贡献**：构建了包含106种舰船和30种飞机型号的PSP细粒度检测基准数据集，是目前型号类别最丰富的航空目标检测数据集；实验表明ExpertDet在各层级上均优于现有细粒度检测方法。
