# BLaDA: Bridging Language to Functional Dexterous Actions within 3DGS Fields

- arXiv: https://arxiv.org/abs/2604.08410v2
- arXiv ID: 2604.08410

---

🔍 **核心问题**：在非结构化环境中，功能性灵巧抓取需要语义理解、3D功能定位和物理可解释执行的紧密集成，但现有模块化方法依赖预定义功能标签，缺乏语义与姿态的紧密耦合。

🛠 **方法**：提出BLaDA框架，通过知识引导的语言解析模块将自然语言解析为结构化六元组约束，利用3D高斯泼溅作为连续场景表示，结合三角功能点定位模块在几何约束下识别功能区域，最后通过3D关键点抓取矩阵变换执行模块解码为物理可行的腕部姿态和手指级命令。

✨ **主要贡献**：实现零样本、可解释的功能性灵巧操作，在复杂基准测试中显著优于现有方法，在功能定位精度和操作成功率上均表现突出。
