# MIL-LC: A Robust Magnetometer-Inertial-LiDAR Fusion Multimodal Localization Framework

- arXiv: https://arxiv.org/abs/2606.25796v1
- arXiv ID: 2606.25796

---

🔍 **核心问题**：在GNSS拒止、几何重复或纹理缺失的复杂环境中（如办公室、酒店、地下停车场），现有单模态定位方法受限于传感器自身缺陷，而多模态融合方法仍过度依赖几何/纹理特征或基础设施信标，导致部署成本高、灵活性差。

🛠 **方法**：提出一种磁力计-惯性-激光雷达融合的多模态定位框架MIL-LC，通过定制传感器套件，在激光雷达几何退化或长期部署中磁场地图变化时，仍能利用环境磁场作为互补模态实现鲁棒定位。

✨ **主要贡献**：首次将环境磁场融合应用于自主移动机器人系统，设计并验证了MIL-LC框架在仿真和真实环境中的鲁棒性与准确性，解决了现有方法在退化场景下的定位失效问题。
