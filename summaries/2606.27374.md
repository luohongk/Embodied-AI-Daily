# World Action Models Enable Continual Imitation Learning with Recurrent Generative Replays

- arXiv: https://arxiv.org/abs/2606.27374v1
- arXiv ID: 2606.27374

---

🔍 **核心问题**：解决机器人在持续模仿学习中因存储原始演示数据受限而导致的灾难性遗忘问题。  
🛠 **方法**：提出基于世界动作模型（WAM）的循环生成回放（REGEN）框架，通过递归生成伪回放轨迹，使机器人策略在不存储原始数据的情况下复习先前学过的任务。  
✨ **主要贡献**：在仿真和真实场景中，REGEN将灾难性遗忘降低高达50%，性能接近需要真实回放数据的特权经验回放方法，并揭示了长时视觉退化与动作-观测不一致性作为生成回放的主要瓶颈。
