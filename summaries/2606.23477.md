# Sublinearly Structured Deep Neural Networks Achieve Feature Learning Consistency for Compositional Functions

- arXiv: https://arxiv.org/abs/2606.23477v1
- arXiv ID: 2606.23477

---

🔍 **核心问题**：本文旨在从统计学习角度回答深度神经网络能否像经典模型一样实现特征学习与预测的一致性，特别是针对层次组合函数的学习任务。

🛠 **方法**：提出并分析了一类“次线性结构”深度神经网络，其输入/输出维度及隐藏神经元数量随样本量次线性增长，并在过参数化条件下验证其一致性。

✨ **主要贡献**：证明了次线性结构DNN在层次组合函数上具有特征学习一致性，且在大样本下具备通用逼近能力；实验表明其性能不逊于宽网络，并发现AlexNet、ResNet等主流CNN实际符合次线性结构，从而从统计角度解释了大规模深度学习在图像任务中的成功。
