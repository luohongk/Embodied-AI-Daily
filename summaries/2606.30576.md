# Beyond 2D Matching: A Unified Single-Stage Framework for Geometry-Aware Cross-View Object Geo-Localization

- arXiv: https://arxiv.org/abs/2606.30576v1
- arXiv ID: 2606.30576

---

🔍 **核心问题**：现有跨视角目标地理定位方法过度依赖2D外观匹配，且受限于缺乏几何元数据、多样化提示和标准视场图像的数据集，导致在未见场景和新型跨视角设置中泛化能力不足。

🛠 **方法**：提出单阶段几何感知定位框架GAGeo，基于置换等变3D基础模型π³，通过融合视觉特征、提示信息和可学习任务令牌，在单次前向传播中联合预测边界框、分割掩码和相机位姿；并引入以卫星视图为通用锚点的对比损失，隐式对齐地面与无人机表示。

✨ **主要贡献**：构建了包含22万对地面-卫星和无人机-卫星图像的大规模高保真建筑数据集，提供多模态提示和相机位姿；提出的GAGeo框架在多个基准上显著超越现有方法，并首次实现无需三元组训练的零样本地面-无人机定位。
