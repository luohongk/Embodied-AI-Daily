# On-Policy Self-Distillation with Sampled Demonstrations Reduces Output Diversity

- arXiv: https://arxiv.org/abs/2606.26091v1
- arXiv ID: 2606.26091

---

🔍 **核心问题**：论文揭示了在线自蒸馏方法在提升单次生成准确率（pass@1）的同时，会降低输出多样性，导致多次采样（pass@k）的收益递减，尤其在需要多样化策略的分布外场景中表现不佳。

🛠 **方法**：采用在线自蒸馏技术，使用同一模型同时作为教师和学生，教师基于一个正确的采样演示提供密集的token级反馈，并通过理论分析最优策略，发现其会放大已有概率差距，集中质量于主导模式。

✨ **主要贡献**：首次指出在线自蒸馏会因教师反馈中的复合偏差而减少输出多样性，理论上证明其与最优在线强化学习的差异，并通过图路径寻找和科学问答实验验证了该方法在平均性能上虽可媲美强化学习，但功能与语义多样性显著降低。
