# DIM-WAM: World-Action Modeling with Diverse Historical Event Memory

- arXiv: https://arxiv.org/abs/2606.27677v1
- arXiv ID: 2606.27677

---

🔍 **核心问题**：现有世界-动作模型依赖短期历史与短时未来预测，难以应对需要长期依赖早期观察和任务进度的复杂任务，存在长期遗忘与全局状态感知不足的问题。  
🛠 **方法**：提出DiM-WAM，一种记忆增强的世界-动作模型，通过多尺度历史事件记忆库（基于相似性合并与时间嵌入）提取紧凑视觉信息，结合局部未来动态与全局任务进度监督，联合去噪视频与动作预测。  
✨ **主要贡献**：在RMBench上将平均成功率从28.4%提升至69.8%（超越显式记忆基线42.0%），在真实Franka机器人任务中将阶段成功率从70.7%提升至91.5%，完整任务成功率从52.5%提升至80.0%。
