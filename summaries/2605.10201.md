# HeteroGenManip: Generalizable Manipulation For Heterogeneous Object Interactions

- arXiv: https://arxiv.org/abs/2605.10201v2
- arXiv ID: 2605.10201

---

🔍 **核心问题**：机器人需要处理跨类型物体交互任务，但现有方法将“接触点定位”与“交互轨迹规划”混为一谈，导致长程任务中误差累积，且单一模型难以适应异构物体的多样化特征。

🛠 **方法**：提出HeteroGenManip两阶段框架，先通过基础模型引导的抓取模块（利用结构先验对齐初始接触状态），再采用多基础模型扩散策略（MFMDP），将物体路由至类别专用模型，并通过双流交叉注意力融合几何与部件特征。

✨ **主要贡献**：在仿真任务中平均性能提升31%，在四种真实世界交互任务中提升36.7%，实现了跨类别形状与姿态的鲁棒泛化。
