## 0. 概览（Concise Summary）

- **主题（Topic）**：合规性监控（Compliance Monitoring），特别是在规则主导的领域（如税务、临床协议、环境法规）。
- **问题（Problem）**：现有机器学习方法在合规监控中面临三大结构性障碍：1）部署时缺乏标注数据；2）不合规实体选择性隐瞒证据（非随机缺失，MNAR）；3）法规环境变化快，监督模型无法及时重训练。
- **方法（Method）**：提出规则状态推断（Rule-State Inference, RSI）框架。这是一个贝叶斯框架，它将权威的、形式化的规则集作为结构化的先验知识，并通过平均场变分推断（Mean-Field Variational Inference）来推断总体的潜在合规状态。
- **创新（Innovation）**：1）逆转了传统范式，从“从数据中学习规则”变为“将规则作为先验来推断合规状态”；2）提出一个包含全局合规文化因子和每个规则特定状态（激活、合规水平、参数漂移）的联合潜在状态模型；3）通过链接函数（Link Function）将异构信号统一到高斯共轭框架下，实现精确的坐标上升更新；4）提供了三项形式化保证（定理1-3）。
- **意义（Significance）**：为在缺乏标签、数据缺失严重且法规频繁变动的现实世界中，进行高效、可解释且理论上可靠的合规监控提供了全新的解决方案。特别适用于资源有限、法规变化快的发展中国家。
- **一句话总结**：本文提出RSI框架，通过将形式化法规作为贝叶斯先验，利用变分推断从噪声和不完整信号中推断总体合规状态，有效解决了合规监控中的零标签、数据非随机缺失和法规动态变化三大难题。

## 1. 研究动机（Motivation）

论文的研究动机源于现实世界中合规监控（如税务审计、临床协议遵守、环境法规执行）所面临的独特且严峻的挑战，这些挑战是标准机器学习方法无法同时解决的。

1.  **核心问题**：在规则主导的领域，规则本身是已知且明确的（如税法条款），但实体（如企业）是否真正合规是未知的。审计员需要从**不完整、可能被操纵的信号**（如申报数据）中推断出真实的合规状态。这个“知道规则”与“从噪声证据推断合规”之间的鸿沟是论文的核心研究对象。

2.  **三大结构性障碍**：
    - **缺乏标签（Absence of Labels）**：监督学习需要大量标注好的合规/不合规样本。但在合规监控中，标签（审计结果）的获取成本高昂、受法律约束，且覆盖率极低。任何依赖标签的方法在初始部署或法规变更后都不可用。
    - **战略性缺失（Strategic Missingness）**：不合规实体有动机隐瞒能暴露其问题的证据。例如，逃税公司可能不申报，超标工厂可能关闭传感器。这种缺失与待推断的潜在状态（合规性）相关，属于**非随机缺失（MNAR）**。基于观测数据训练的模型会系统性低估不合规率。
    - **法规易变性（Regulatory Volatility）**：在快速发展的经济体中，税法等法规一年可能修改多次。需要完全重训练的模型无法满足在几小时内更新合规评分的要求。

3.  **现有方法的不足**：监督分类器、马尔可夫逻辑网络、神经符号方法等主流范式都无法同时满足上述三个约束。RSI通过逆转推理方向（将规则作为先验而非学习目标）来应对这些挑战。

## 2. 创新点（Innovation）

论文的核心创新在于提出并形式化了**规则状态推断（RSI）框架**，其关键创新点如下：

1.  **范式逆转（Paradigm Reversal）**：与从数据中学习规则的常见方法（如马尔可夫逻辑网络）不同，RSI将权威的、形式化的规则集视为**结构化的贝叶斯先验**。这从根本上解决了零标签问题，因为推理不再依赖历史审计数据，而是基于规则本身。

2.  **层次化潜在状态模型（Hierarchical Latent State Model）**：提出了一个新颖的联合潜在状态模型，包含两个层次：
    - **全局因子（Global Factor, $\eta$）**：一个标量潜在变量，代表整个总体的“合规文化”倾向。负值表示系统性不合规。
    - **每个规则的特定状态（Per-Rule State, $s_i = (a_i, \phi_i, \delta_i)$）**：包括规则是否激活（$a_i$）、总体合规水平（$\phi_i$）和参数漂移（$\delta_i$）。其中，$\delta_i$ 是一个关键创新，它量化了法律规定的阈值与实际执行之间的差距，这是现有方法无法提供的。

3.  **链接函数与高斯共轭（Link Function & Gaussian Conjugacy）**：通过一个单调的链接函数 $g_i$，将各种类型的合规信号（如比率、正数、实数）映射到实数域，从而在**高斯-高斯共轭**的框架下进行推理。这使得变分推断的坐标上升更新具有**精确的闭式解**，保证了算法的效率和收敛性。

4.  **依赖参数 $\rho$ 的变分估计（Variational Estimation of Dependence Parameter $\rho$）**：引入一个标量 $\rho \in [0, 1)$ 来控制不同规则之间的依赖程度。$\rho=0$ 恢复独立模型。该参数通过变分推断进行估计，使模型能够灵活地捕捉系统性合规行为。

5.  **三项形式化保证（Three Formal Guarantees）**：
    - **定理1（O(nk+K) 法规适应性）**：当单个规则更新时，RSI的更新成本与规则影响的实体数（$n_k$）和规则总数（$K$）成线性关系，远优于监督学习的全量重训练。
    - **定理2（Bernstein–von Mises 一致性）**：证明了对于可识别的连续分量（$\psi_i = \phi_i + \delta_i$ 和 $\eta$），后验分布会随着数据量增加而收敛到真实值，并具有渐近正态性。
    - **定理3（单调ELBO收敛）**：证明了算法每一步迭代都能保证证据下界（ELBO）单调不减，并收敛到一个驻点。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

RSI框架的核心架构是一个生成式概率模型，其推理过程是贝叶斯的。

1.  **问题设定**：给定一个权威规则集 $R = \{R_1, ..., R_K\}$ 和一个实体群 $X = \{x_1, ..., x_J\}$。目标是基于不完整、有噪声的信号 $D$，推断每个实体对每条规则的合规状态。

2.  **核心假设**：规则必须以机器可读的形式化表示存在（如通过“规则即代码”系统，如Catala）。RSI是此编码层的下游推理引擎。

3.  **模型架构**：模型定义了联合潜在状态 $S = (\eta, s_1, ..., s_K)$ 的生成过程：
    - 全局合规文化因子 $\eta$ 从标准正态分布中采样。
    - 对于每个规则 $R_i$，其状态 $s_i = (a_i, \phi_i, \delta_i)$ 的生成依赖于 $\eta$。
    - 观测信号 $t_{ji}$ 的生成依赖于 $a_i, \phi_i, \delta_i$。

4.  **推理方法**：由于精确后验推断不可行，采用**平均场变分推断（Mean-Field Variational Inference, MFVI）**。它将后验分布近似为可分解的形式 $q(S) = q(\eta) \prod_i q(a_i) q(\phi_i) q(\delta_i)$，并通过坐标上升法（Coordinate Ascent Variational Inference, CAVI）最大化证据下界（ELBO）。

5.  **实体评分**：实体级别的审计评分是**确定性的**，直接与法律标准比较。对于有观测的实体，评分公式为 $NC_{ji} = g_i^{-1}(\mu_i - t_{ji})$。对于缺失数据的实体，则使用后验预测概率 $NC_{miss}^{ji}$，这提供了对MNAR的部分抵抗能力。

### 3.2 关键算法与数学推导

**1. 先验设定（Prior Specification）**：
$$P(S) = P(\eta) \prod_{i=1}^K P(a_i) P(\phi_i | \eta, a_i) P(\delta_i | a_i)$$
其中：
$$\eta \sim \mathcal{N}(0, 1)$$
$$a_i \sim \text{Bern}(\pi_i)$$
$$\phi_i | \eta, a_i=1 \sim \mathcal{N}(\mu_i + \rho \tau_i \eta, \tau_i^2 (1-\rho^2))$$
$$\delta_i | a_i=1 \sim \mathcal{N}(0, \omega_i^2)$$
当 $a_i=0$ 时，$\phi_i = \delta_i = 0$（退化分布）。

**2. 似然函数（Likelihood）**：
对于规则 $R_i$ 下的观测信号 $t_{ji}$：
$$t_{ji} | a_i=1, \phi_i, \delta_i \sim \mathcal{N}(\phi_i + \delta_i, \sigma_i^2)$$
$$t_{ji} | a_i=0 \sim \mathcal{N}(0, \sigma_{bg,i}^2)$$

**3. 变分更新公式（Variational Update Equations）**：
这是算法的核心，所有更新都是闭式解。

- **更新 $q(\eta) = \mathcal{N}(m_\eta, v_\eta)$**:
  $$v_\eta = \left(1 + \frac{\rho^2}{1-\rho^2} \sum_{i=1}^K \varrho_i \right)^{-1}$$
  $$m_\eta = v_\eta \sum_{i=1}^K \varrho_i \frac{\rho (m_{\phi_i} - \mu_i)}{\tau_i (1-\rho^2)}$$

- **更新 $q(\phi_i) = \mathcal{N}(m_{\phi_i}, v_{\phi_i})$**:
  首先计算有效先验：$\tilde{\mu}_i = \mu_i + \rho \tau_i m_\eta$, $\tilde{\psi}_i^2 = \tau_i^2 (1-\rho^2) + (\rho \tau_i)^2 v_\eta$。
  $$v_{\phi_i} = \left( \frac{1}{\tilde{\psi}_i^2} + \frac{\varrho_i n_i}{\sigma_i^2} \right)^{-1}$$
  $$m_{\phi_i} = v_{\phi_i} \left( \frac{\tilde{\mu}_i}{\tilde{\psi}_i^2} + \frac{\varrho_i n_i (\bar{t}_i - m_{\delta_i})}{\sigma_i^2} \right)$$

- **更新 $q(\delta_i) = \mathcal{N}(m_{\delta_i}, v_{\delta_i})$**:
  $$v_{\delta_i} = \left( \frac{1}{\omega_i^2} + \frac{\varrho_i n_i}{\sigma_i^2} \right)^{-1}$$
  $$m_{\delta_i} = v_{\delta_i} \cdot \frac{\varrho_i n_i (\bar{t}_i - m_{\phi_i})}{\sigma_i^2}$$

- **更新 $q(a_i) = \text{Bern}(\varrho_i)$**:
  $$\varrho_i = \sigma \left( \log \frac{\pi_i}{1-\pi_i} + \log \tilde{L}_{1,i} - \log \tilde{L}_{0,i} \right)$$
  其中 $\sigma(\cdot)$ 是逻辑函数，$\tilde{L}_{1,i}$ 和 $\tilde{L}_{0,i}$ 是边际似然。

- **更新 $\rho$**:
  使用投影梯度上升法：$\rho \leftarrow \Pi_{[0,1-\epsilon]} \left( \rho + \alpha \frac{\partial \text{ELBO}}{\partial \rho} \right)$。

### 3.3 模型、训练与数据集

- **模型**：RSI是一个概率生成模型，其参数是变分分布 $q$ 的参数（$m_\eta, v_\eta, m_{\phi_i}, v_{\phi_i}, m_{\delta_i}, v_{\delta_i}, \varrho_i, \rho$）。
- **训练**：训练过程是**无监督的**，不需要任何合规标签。它通过坐标上升法（Algorithm 1）迭代优化ELBO来学习变分参数。算法在每一步都保证ELBO单调不减（定理3）。
- **数据集**：论文创建了一个名为 **RSI-Togo-Fiscal-Synthetic v2.0** 的合成基准数据集，基于多哥官方税法。它包含：
    - **实体**：2000家来自四个经济部门的企业。
    - **规则**：5条税务规则（增值税、公司税、最低包干税、非正规部门税、员工所得税）。
    - **信号**：主要信号是“申报/理论税额比”（经logit变换），辅以支付延迟和银行账户注册等信号。
    - **数据缺失**：每条规则有18-20%的信号是缺失的。
    - **法规变更事件**：编码了增值税起征点从6000万FCFA提高到1亿FCFA的法规变更，为验证定理1提供了自然实验。

### 3.4 实验设置与结果

论文设计了7个实验来验证RSI的性能和理论性质。由于是预印本，部分数值结果标注为“forthcoming”（即将公布），但实验设计和预期结果已明确。

- **实验1：整体性能**：在F1、AUC和召回率上，将RSI（零样本）与基于规则的系统（RBS）、XGBoost和MLP（全监督）进行比较。预期RSI在零样本下能达到有竞争力的性能。
- **实验2：法规适应性（定理1）**：测量RSI吸收增值税起征点变更所需的挂钟时间，并与监督模型的全量重训练时间对比。预期RSI的更新成本远低于监督模型。
- **实验3：BvM一致性（定理2）**：追踪后验标准差 $SD(\psi_i|D)$ 随样本量 $n$ 增加的变化，验证其以 $1/\sqrt{n}$ 的速率衰减。
- **实验4：缺失数据鲁棒性**：在MCAR缺失率从0%增加到50%的情况下，评估各规则的F1值。预期RSI的贝叶斯回退机制（公式24）比RBS的固定赋值退化得更慢。
- **实验5：规则间依赖（$\rho$）**：报告收敛后的 $\rho$ 值，并比较 $\rho=0$（独立模型）与变分 $\rho$ 的实体级F1。预期 $\rho > 0$ 证实了系统性合规文化的存在。
- **实验6：ELBO收敛（定理3）**：绘制ELBO随迭代次数的变化曲线，验证其单调不减的特性。
- **实验7：MNAR鲁棒性（计划中）**：通过引入非对称缺失率来模拟MNAR场景，评估后验预测评分（公式24）对偏差的缓解能力。

### 3.5 技术细节

- **链接函数**：对于比率信号（0,1），使用logit链接：$g(s) = \log(s/(1-s))$。对于正数信号，使用log链接。对于实数信号，使用恒等链接。对于二元信号，直接使用Beta-Bernoulli共轭。
- **超参数设定**：表2给出了多哥税务实例的超参数，如先验激活概率 $\pi_i$、先验均值 $\mu_i$、方差 $\tau_i^2$ 等。这些参数基于领域知识设定。
- **$\phi_i$ 和 $\delta_i$ 的可识别性**：在静态模型中，只有它们的和 $\psi_i = \phi_i + \delta_i$ 是可识别的。分解依赖于先验。在顺序RSI扩展中，通过纵向观测可以自然解决这个问题。
- **实体评分的确定性**：论文刻意将实体评分设计为确定性的，以避免在单次横截面数据中引入不可识别的实体级随机效应。实体级的不确定性被推迟到顺序RSI中处理。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    - **解决了合规监控的“不可能三角”**：同时应对了零标签、数据非随机缺失和法规动态变化三大挑战。
    - **提供了可解释的、理论上有保证的框架**：RSI的后验分布（如 $E[\phi_i|D]$, $E[\delta_i|D]$, $E[\eta|D]$）本身就是可解释的审计情报，无需额外的解释工具。定理1-3提供了坚实的理论基础。
    - **开创了“规则即先验”的新范式**：为将形式化知识融入机器学习提供了一种优雅且高效的贝叶斯方法，其思想可推广到其他领域。

- **对未来研究/应用的影响**：
    - **税务与公共管理**：为发展中国家的税务部门提供了一个无需大量历史审计数据即可部署的智能审计系统，能快速适应法规变化，并发现系统性逃税趋势和执法偏差。
    - **临床协议与医疗质量**：可用于监控医院对临床指南的遵守情况，从电子健康记录中推断合规状态，并识别系统性偏差。
    - **环境法规监控**：可用于监控工厂的排放数据，即使传感器被故意关闭，也能通过后验预测概率提供合理的风险评估。
    - **顺序RSI扩展**：将RSI扩展到时间序列，形成一个精确的卡尔曼滤波器，可以追踪合规轨迹，并为每个实体提供带有不确定性区间的贝叶斯评分，这具有巨大的应用潜力。

    - **对“规则即代码
