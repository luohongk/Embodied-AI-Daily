# Diffusion for Long-Horizon Multi-Robot Path Planning in Human-Shared Environments

- arXiv: https://arxiv.org/abs/2607.09911v1
- arXiv ID: 2607.09911

---

🔍 **核心问题**：在人类共享环境中，多机器人长期路径规划需要兼顾机器人间的协调与社交行为，但现有扩散模型生成路径时存在固定时长和高延迟问题，难以适应不同目标距离和实时部署需求。

🛠 **方法**：提出多机器人滚动扩散（MRRD）框架，结合滚动时域方案适应人类运动预测限制、并行化扩散推理生成类人路径、基于冲突搜索机制解决机器人间碰撞，并引入基于紧迫性的时间条件生成变速路径，以及差异化引导项最大化社交意识与协调效率。

✨ **主要贡献**：MRRD在密集人群中成功实现15个机器人的实时导航，显著提升了安全性和任务成功率，优于现有基线方法。
