# LiveFigure: Generating Editable Scientific Illustration with VLM Agents

- arXiv: https://arxiv.org/abs/2605.23527v1
- arXiv ID: 2605.23527

---

🔍 **核心问题**：现有生成模型只能输出不可编辑的栅格图像，无法生成支持结构化修改（如元素、比例、文本调整）的矢量科学插图，难以满足学术出版对可编辑图形的需求。

🛠 **方法**：提出LiveFigure框架，利用VLM智能体模拟人类研究者的多步绘图流程，包括从高质量参考中规划蓝图、通过PowerPoint接口生成可执行脚本，以及基于视觉诊断优化输出，最终产生完全矢量化的可编辑插图。

✨ **主要贡献**：生成的插图天然具备可编辑性，仅需17次手动修改即可达到80%的出版就绪率，远超最强基线NanoBanana的24%；人类偏好实验中以60%胜率显著领先。
