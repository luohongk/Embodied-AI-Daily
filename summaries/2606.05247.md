## 0. 概览（Concise Summary）

- **主题（Topic）**：约束神经网络预测。具体来说，是解决在神经网络输出中强制执行非线性不等式约束的问题。
- **问题（Problem）**：现有的硬约束方法（如投影层）在处理大规模、耦合的非线性不等式约束时，要么对约束集的结构有严格限制，要么计算开销巨大。软约束方法则无法保证推理时的可行性。
- **方法（Method）**：提出了一种名为 **DiffSlack** 的可微投影层。其核心思想是：1）通过引入**可学习的松弛变量**，将不等式约束重新表述为等式约束，从而构建一个增广的可行流形；2）使用**阻尼高斯-牛顿法**将网络原始输出投影到这个流形上；3）采用**两阶段课程学习**策略来稳定训练过程。
- **创新（Innovation）**：
    1.  将松弛变量从传统的求解器内部变量，提升为**网络输出的一部分**，使其能够进行数据驱动的预测，为投影过程提供“热启动”。
    2.  提出了一种**可微的、基于阻尼高斯-牛顿法的投影层**，适用于大规模非线性不等式约束，并保持了端到端的可微性。
    3.  设计了**两阶段课程学习**，先通过软约束和松弛校准进行预训练，再激活硬投影层，从而稳定训练并提高约束满足度。
- **意义（Significance）**：DiffSlack 提供了一种**可扩展、实用且高效**的方法，将硬不等式约束嵌入到神经网络中。它在车辆路径规划等安全关键工程应用中表现出色，能以较低的计算成本获得更高的成功率，并显著降低对监督信号质量的敏感性。
- **一句话总结**：DiffSlack 通过将不等式约束转化为带可学习松弛变量的等式流形，并利用可微的高斯-牛顿投影层，实现了高效、可扩展且鲁棒的非线性约束神经网络预测。

## 1. 研究动机（Motivation）

- **问题与空白**：在许多安全关键的应用（如自动驾驶、机器人控制）中，神经网络的输出必须满足一系列物理、安全或操作约束（通常是非线性不等式）。然而，标准神经网络只优化预测损失，无法保证输出在推理时满足这些约束。现有的解决方案存在明显不足：
    - **软约束方法**：将约束作为惩罚项加入损失函数。这种方法简单，但无法保证推理时的可行性，且对惩罚权重和训练数据分布敏感。
    - **事后滤波器**：在模型预测后使用外部安全滤波器（如控制障碍函数）进行修正。这会阻断端到端的梯度流，导致网络输出与修正动作之间出现不匹配。
    - **现有硬约束方法**：虽然能提供更强的可行性保证，但处理大规模、耦合的非线性不等式约束时面临挑战。例如，DC3 的梯度修正收敛慢；ENFORCE 的 Fischer-Burmeister 公式化会引入大量额外变量；KKT-HardNet 构建的原始-对偶系统规模巨大，计算开销高昂。
- **重要性**：解决这个问题对于提升学习模型在真实世界部署中的**可靠性、安全性和物理一致性**至关重要。一个能同时保证预测精度和约束满足的模型，是实现安全自主系统的关键。
- **挑战**：核心挑战在于如何设计一个**可微的、计算高效的投影层**，能够处理**大规模（如200个）、耦合的、一般形式的非线性不等式约束**，同时保持端到端训练的能力。

## 2. 创新点（Innovation）

- **关键创新**：DiffSlack 的核心创新在于将**可学习的松弛变量**与**可微的高斯-牛顿投影层**相结合。
- **与现有方案的不同**：
    1.  **可学习的松弛变量**：与 DC3、ENFORCE 等将松弛变量作为求解器内部变量不同，DiffSlack 的松弛变量是**网络输出的一部分**。网络不仅预测任务输出（如路径点），还预测松弛变量。这使得松弛变量能够从数据中学习，为每个约束的“非活跃侧”边界（即约束已经满足时的裕量）提供一个**数据驱动的初始估计**。这相当于为后续的投影过程提供了一个“热启动”，减少了投影所需的迭代次数和修正幅度。
    2.  **阻尼高斯-牛顿投影层**：该层将原始网络输出（包含任务输出和松弛变量）投影到由松弛变量重新表述的等式流形上。该层是**可微的**，可以通过隐式微分进行端到端训练。与 DC3 的梯度下降法相比，高斯-牛顿法具有二次收敛速度，效率更高。与 ENFORCE 的 Fischer-Burmeister 方法相比，DiffSlack 的松弛变量公式化更直接，避免了引入额外的乘子空间，从而降低了问题规模。
    3.  **两阶段课程学习**：直接训练带有硬投影层的网络非常不稳定。DiffSlack 提出的两阶段策略（先软约束训练，再硬投影训练）有效地解决了这个问题，使网络能够逐步学习如何接近可行流形，从而稳定了端到端训练。
- **为什么能奏效**：
    - 可学习的松弛变量通过估计非活跃约束的裕量，**减小了原始输出到可行流形的距离**，使投影更容易、更稳定。
    - 高斯-牛顿法在局部具有良好的收敛性，而松弛变量提供的热启动确保了原始输出位于其收敛域内。
    - 两阶段训练策略避免了从随机初始化直接进行硬投影的失败风险，为网络提供了一个“软着陆”过程。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

DiffSlack 的整体框架如图 2 所示，包含以下核心组件：

1.  **网络与增广输出**：一个神经网络 $f_\theta$ 接收输入 $x$，输出一个增广的原始预测 $\hat{y} = [\hat{p}^\top, \hat{s}^\top]^\top$。其中 $\hat{p}$ 是任务输出（如路径点），$\hat{s}$ 是可学习的松弛变量。
2.  **松弛变量重新表述**：将原始的不等式约束 $g(p) \le 0$ 转化为等式约束 $h(y) = g(p) + s \odot s = 0$。这里 $s \odot s$ 确保了松弛项非负，因此满足 $h(y)=0$ 自动意味着 $g(p) \le 0$。这定义了一个增广的可行流形 $\mathcal{M}$。
3.  **可微投影层**：一个基于阻尼高斯-牛顿法的迭代投影层，将原始增广输出 $\hat{y}$ 投影到流形 $\mathcal{M}$ 上，得到可行输出 $y^*$。该层是可微的，允许梯度反向传播。
4.  **两阶段课程训练**：
    - **阶段一（软训练）**：禁用投影层。网络通过软约束损失和松弛校准损失进行训练，目标是让原始输出 $\hat{y}$ 接近流形 $\mathcal{M}$ 的“可达管”（reach tube）。
    - **阶段二（硬投影训练）**：激活投影层。网络输出经过投影层得到 $y^*$，任务损失在 $y^*$ 上计算。同时保留软约束损失作为辅助正则化项。

### 3.2 关键算法与数学推导

**1. 松弛变量重新表述与定理1**

不等式约束 $g(p) \le 0$ 通过引入松弛变量 $s \in \mathbb{R}^{N_C}$ 被重新表述为等式：
$$h(y) = g(p) + s \odot s = 0$$
其中 $y = [p^\top, s^\top]^\top$。

**定理1（不等式满足）**：对于任何 $y = [p^\top, s^\top]^\top$，如果 $h(y) = 0$，则 $g(p) \le 0$。
*证明*：由 $h(y)=0$ 得 $g(p) = -s \odot s$。由于 $s \odot s \ge 0$ 逐元素成立，因此 $g(p) \le 0$。

**2. 可学习的松弛变量作为边界预测**

定义 $q(\hat{p}) := [-g(\hat{p})]_+$ 为原始预测 $\hat{p}$ 的非活跃约束裕量。可学习的松弛变量 $\hat{s}$ 的目标是估计这个裕量，即 $\hat{q} = \hat{s} \odot \hat{s} \approx q(\hat{p})$。

等式残差可以分解为：
$$h(\hat{y}) = g(\hat{p}) + \hat{q} = [g(\hat{p})]_+ + (\hat{q} - q(\hat{p}))$$
其中 $[g(\hat{p})]_+$ 是活跃约束的违反量，$(\hat{q} - q(\hat{p}))$ 是松弛变量的估计误差。

当使用零初始化松弛变量时，残差为 $h(\hat{y}_0) = g(\hat{p})$，其期望的 $L_2$ 范数平方为：
$$\mathbb{E}[\|h(\hat{y}_0)\|_2^2] = \mathbb{E}[\|[g(\hat{p})]_+\|_2^2] + \mathbb{E}[\|[-g(\hat{p})]_+\|_2^2]$$
当使用学习到的松弛变量 $\hat{y}_L$ 时，假设其估计误差为 $\varepsilon_L^2$，则：
$$\mathbb{E}[\|h(\hat{y}_L)\|_2^2] = \mathbb{E}[\|[g(\hat{p})]_+\|_2^2] + \varepsilon_L^2$$
由于 $\varepsilon_L^2$ 通常远小于 $\mathbb{E}[\|[-g(\hat{p})]_+\|_2^2]$，因此学习到的松弛变量显著减小了等式残差，从而减小了原始输出到流形 $\mathcal{M}$ 的距离（公式12, 13）。

**3. 可微的高斯-牛顿投影层**

给定原始增广输出 $\hat{y}$，投影层求解以下加权最小范数问题：
$$y^* = \arg\min_y \frac{1}{2} \|y - \hat{y}\|_W^2 \quad \text{s.t.} \quad h(y) = 0$$
其中 $W = \text{diag}(w_o \mathbf{1}_{N_O}, w_s \mathbf{1}_{N_C})$，$w_o > w_s$，优先保持任务输出不变。

该问题通过迭代的阻尼高斯-牛顿法求解。在第 $k$ 步，线性化约束 $h(y_k) + J_h(y_k)(y - y_k) = 0$，得到闭式更新：
$$y_{k+1} = y_k - W^{-1} J_h(y_k)^\top \left( J_h(y_k) W^{-1} J_h(y_k)^\top \right)^{-1} h(y_k)$$
其中 $J_h(y) = [J_g(p) \quad \text{diag}(2s)]$ 是增广雅可比矩阵。迭代直到 $\|h(y_k)\|_\infty < \epsilon$ 或达到最大迭代次数 $I_{\max}$。

**局部收敛性**：在满足正则性条件的可行点 $y^*$ 附近，更新满足：
$$\|e^{(k+1)}\|_W \le \|M_W\|_W \|e^{(k)}\|_W + O(\|e^{(k)}\|_W^2)$$
其中 $e^{(k)} = y_k - y^*$，$M_W$ 是 $W$-加权切空间投影算子。这表明误差的法向分量被一阶抑制。

**数值条件改善**：Gram 矩阵为：
$$J_h W^{-1} J_h^\top = w_o^{-1} J_g J_g^\top + 4 w_s^{-1} \text{diag}(s^2)$$
非零的松弛变量 $s$ 提升了 Gram 矩阵的谱下界，改善了病态条件，尤其是在 $J_g$ 接近秩亏时。

**4. 梯度流与隐式约束监督**

通过隐式微分，投影层输出的梯度为：
$$\frac{\partial y^*}{\partial \hat{y}} = M_W$$
这意味着反向传播的梯度被投影到了流形 $\mathcal{M}$ 的切空间上。梯度中与约束流形法向的分量被抑制，而切向分量得以保留。这被称为**隐式约束监督**：即使外部任务监督信号粗糙，投影层也会将梯度过滤到与约束几何兼容的方向上。

### 3.3 模型、训练与数据集

- **任务**：车辆路径规划，生成一个由 $T=40$ 个离散路径点 $P \in \mathbb{R}^{T \times 2}$ 表示的路径。
- **约束**：共 200 个非线性不等式约束，包括：
    - 碰撞避免（$3T=120$ 个）：通过 Log-Sum-Exp 近似将车辆和障碍物表示为可微的符号距离函数。
    - 曲率约束（$T=40$ 个）：基于自行车模型，限制最大曲率。
    - 路径点间距约束（$T=40$ 个）：限制相邻路径点最大距离。
- **网络**：一个全连接神经网络，输入为 66 维向量（目标坐标和障碍物顶点），隐藏层维度为 [128, 256, 512, 512]，激活函数为 ReLU。
- **数据集**：生成 200,000 个随机场景，每个场景包含 8 个随机四边形障碍物。按 6:3:1 划分为训练、验证和测试集。
- **监督信号**：使用**全局引导人工势场（G-APF）** 作为廉价监督信号，而非昂贵的 NMPC 专家演示。G-APF 通过 Dijkstra 算法和波前扩展构建，提供全局拓扑引导。

### 3.4 实验设置与结果

- **基线方法**：Hybrid A*， Informed RRT*， NMPC， IL (Pure)， IL+Soft， DC3， ENFORCE。
- **主要结果（表2）**：
    - **成功率（SR）**：DiffSlack 以 **93.80%** 的成功率显著优于所有学习基线（DC3: 86.44%， ENFORCE: 84.05%， IL+Soft: 72.66%），接近非学习方法（NMPC: 82.35% 但计算时间极长）。
    - **计算时间（CT）**：DiffSlack (0.0983s) 与 DC3 (0.2025s) 和 ENFORCE (0.1101s) 相当，远快于非学习方法。
    - **约束满足**：DiffSlack 在运动学合规性（$S_{kin}$: 99.48%）和间距合规性（$S_{spc}$: 99.89%）上也取得了最佳或接近最佳的结果。
- **动态验证（表3）**：在 CARLA 模拟器中，DiffSlack 规划的路径在跟踪误差（RMSE CTE: 0.0802m）和控制平滑度（$J_{smooth}$: 0.0519）上均优于其他学习基线。真实车辆实验也验证了其可执行性。
- **消融研究**：
    - **监督信号质量（图7）**：在 $2\times2$ 实验（硬/软约束 × 专家/粗糙信号）中，硬约束（DiffSlack）将因监督信号质量差异导致的成功率差距缩小了约 **83%**，证明了其强大的隐式约束监督能力。
    - **两阶段训练（表4，图8）**：移除阶段一预训练会导致 DiffSlack 的成功率下降（从 93.80% 降至 89.73%），且路径长度和终点距离显著恶化，证实了阶段一的重要性。
    - **最大迭代次数 $I_{\max}$（图9）**：$I_{\max}=50$ 在成功率和计算时间之间取得了最佳平衡。

### 3.5 技术细节

- **阻尼因子**：在求解 Gram 矩阵时，使用 Cholesky 分解并添加对角阻尼 $\delta = 10^{-4}$，以增强数值稳定性。
- **权重设置**：任务输出权重 $w_o = 5.0$，松弛变量权重 $w_s = 1.0$，优先保持任务输出不变。
- **训练超参数**：阶段一训练 300 个 epoch，学习率 $10^{-4}$，批次大小 512；阶段二训练 100 个 epoch，学习率 $3 \times 10^{-5}$，批次大小 256。
- **停止条件**：投影迭代在 $\|h(y_k)\|_\infty < 10^{-3}$ 或达到 $I_{\max}=50$ 时停止。
