# Hybrid Topological Data Analysis and LSTM Networks for Enhanced Network Intrusion Detection Using CIC-IDS2017 Dataset

- arXiv: https://arxiv.org/abs/2606.31619v1
- arXiv ID: 2606.31619

---

🔍 **核心问题**：提升网络入侵检测系统对复杂攻击（如DDoS、暴力破解等）的检测能力，解决传统方法在捕捉网络流量拓扑结构和时序特征方面的不足。

🛠 **方法**：提出混合模型，将拓扑数据分析（TDA）的持续同调（Betti曲线、持久性图）与长短期记忆（LSTM）网络结合，先提取流量的拓扑特征，再模拟其时序依赖关系。

✨ **主要贡献**：在CIC-IDS2017数据集上，混合模型AUC和F1分数均达1.000，5折交叉验证平均F1为0.999±0.001；消融实验证实拓扑与时序特征互补（F1分别为0.990和1.000），且性能优于TDA+随机森林（F1=0.994）和孤立森林（F1=0.835）基线。
