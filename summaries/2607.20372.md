# Notes to Self: Can LLMs Benefit from Experiential Abstractions?

- arXiv: https://arxiv.org/abs/2607.20372v1
- arXiv ID: 2607.20372

---

🔍 **核心问题**：研究大型语言模型（LLMs）是否能够像人类一样，从自身经验中提取可复用的抽象知识（如策略或提醒），并利用这些知识逐步提升问题解决能力。

🛠 **方法**：从LLMs在MATH训练集上的解题轨迹中，由更强的教师模型或LLMs自身提取自然语言抽象知识，构建可检索的库；并探索两种使用模式：推理时检索，以及将抽象知识增强到训练提示中进行强化学习。

✨ **主要贡献**：实验表明，经验性抽象能显著提升LLMs在数学和逻辑推理基准上的表现；自提取的抽象知识与教师提取的效果相当；该抽象使用框架可迁移至其他数据集和模型，验证了LLMs能像人类一样从经验中提取和应用抽象知识。
