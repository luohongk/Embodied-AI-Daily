# GeneralVLA-2: Geometry-Aware Reconstruction and Governed Memory for Robot Planning

- arXiv: https://arxiv.org/abs/2606.17480v1
- arXiv ID: 2606.17480

---

🔍 **核心问题**：通用视觉-语言-动作系统在机器人规划中面临两个瓶颈：单目3D物体重建易产生姿态和几何幻觉，以及知识库缺乏对记忆质量、冲突和几何相关性的有效控制。

🛠 **方法**：提出GeoFuse-MV3D重建分支，利用几何先验引导多视图SAM3D重建，通过输入视图掩码验证外部几何线索、软视觉外壳支持及轴对齐优化，仅融合几何信息并保留外观；同时将知识库升级为受控长期记忆系统，引入质量、置信度、生命周期、验证器和冲突元数据，结合精度导向检索。

✨ **主要贡献**：GeoFuse-MV3D在GSO-30上相比MV-SAM3D基线降低CD和LPIPS 2.20%和2.02%，提升PSNR和SSIM 2.36%和1.03%；知识库在Terminal-Bench 2.0和SWE-Bench Verified上分别提升成功率4.53%和3.73%，同时降低动作序列长度4.95%和5.65%。
