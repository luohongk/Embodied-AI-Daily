# Feel Robot Feels: Tactile Feedback Array Glove for Dexterous Manipulation

- arXiv: https://arxiv.org/abs/2603.28542v1
- arXiv ID: 2603.28542

---

🔍 **核心问题**：解决遥操作灵巧操作中手-机器人运动映射不准确和缺乏触觉反馈的问题，导致操作灵巧性受限且依赖视觉感知。

🛠 **方法**：提出TAG低成本手套系统，采用非接触磁感应实现21自由度关节跟踪（误差<1°），并在每个手指集成32个触觉执行器阵列（2 cm²模块），通过空间激活模式传递触觉反馈。

✨ **主要贡献**：实现实时感知接触几何与动态力，显著提升接触密集型遥操作任务的成功率，并增强基于学习的操作演示数据收集的可靠性。
