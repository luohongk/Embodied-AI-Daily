# ObjRetarget: An Object-Aware Motion Retargeting Framework with Anthropomorphic Arm Constraints and Polyhedral Hand Modeling

- arXiv: https://arxiv.org/abs/2607.03828v1
- arXiv ID: 2607.03828

---

🔍 **核心问题**：从人类操作视频中可靠地将动作意图迁移到机器人，同时保持手-物接触稳定，是具身智能中的关键挑战。现有方法缺乏显式接触建模或依赖强化学习，导致精度和泛化性不足。

🛠 **方法**：提出ObjRetarget框架，结合拟人化手臂轨迹约束与结构化手-物几何建模。手臂运动通过人类视频提取参考轨迹，经拟人约束和冗余感知优化生成自然动作；手部操作利用多面体簇表示多指接触，并通过几何不变量保持接触结构。

✨ **主要贡献**：在真实机器人实验中，ObjRetarget提升了多种灵巧操作任务的成功率和接触稳定性，并能良好泛化至不同演示、物体姿态和任务设置。
