# A Spatial-Spectral-Frequency Interactive Network for Multimodal Remote Sensing Classification

- arXiv: https://arxiv.org/abs/2510.04628v3
- arXiv ID: 2510.04628

---

🔍 **核心问题**：解决多模态遥感图像分类中，现有方法难以从异质冗余数据中有效提取结构性和细节特征的问题。

🛠 **方法**：提出空间-光谱-频率交互网络（S²Fin），通过高频稀疏增强Transformer、自适应频率通道模块、高频共振掩膜及空间-光谱注意力融合模块，实现跨域特征交互与增强。

✨ **主要贡献**：首次将频域学习引入多模态遥感分类，通过稀疏注意力优化高频滤波器参数，并设计两级空间-频率融合策略，在四个基准数据集上以有限标注数据取得优于现有方法的分类性能。
