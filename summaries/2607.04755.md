# Continual Model Merging with Test-Time Adaptation for Whole-Slide Image Analysis

- arXiv: https://arxiv.org/abs/2607.04755v1
- arXiv ID: 2607.04755

---

🔍 **核心问题**：在无需保留历史训练数据的情况下，如何通过模型合并实现全切片图像分析中的持续学习，并应对分布偏移问题。

🛠 **方法**：采用测试时自适应引导的模型合并方法，在推理阶段利用无标签目标数据调整合并相关变量，以适配当前数据分布并保持历史知识。

✨ **主要贡献**：首次将测试时自适应模型合并方法系统应用于无重放的持续全切片图像分类，在六个TCGA癌症亚型数据集上验证其能有效提升任务特异性性能并缓解遗忘，同时揭示了任务顺序与分布适应对性能的影响。
