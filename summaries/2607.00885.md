# Improving Sparse-View 3DGS Generalization via Flat Minima Optimization

- arXiv: https://arxiv.org/abs/2607.00885v1
- arXiv ID: 2607.00885

---

🔍 **核心问题**：在稀疏视角监督下，3D高斯泼溅（3DGS）容易过拟合到观测图像，导致对新视角的泛化能力差。

🛠 **方法**：提出基于平坦最小值优化的轻量训练框架，通过引入各向异性感知的高斯扰动和周期性重初始化，在不改变网络结构的情况下正则化优化过程。

✨ **主要贡献**：在LLFF和Mip-NeRF360数据集上，该方法在稀疏视角下显著提升了定量指标和感知质量，生成更清晰、稳定且泛化性更好的新视角重建。
