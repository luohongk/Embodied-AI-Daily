# RoboNaldo: Accurate, Stable and Powerful Humanoid Soccer Shooting via Motion-Guided Curriculum Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.11092v1
- arXiv ID: 2606.11092

---

🔍 **核心问题**：解决仿人机器人足球射门中全身稳定性、高冲击力交互与目标精度难以兼顾的问题，现有方法在适应不同球位和击球时机上存在局限。

🛠 **方法**：提出三阶段运动引导课程强化学习框架RoboNaldo，先以单个人类踢球参考学习稳定踢球先验，再适应静止球自由踢场景，最后通过运动指令与踢球触发接口扩展至移动球射门，训练中由高层启发式规划器控制接口，推理时可由其他高层控制器驱动。

✨ **主要贡献**：在仿真中自由踢射门误差降低48.6%、射门速度提升2.96倍；在真实Unitree G1机器人上，3米距离自由踢和移动球平均射门误差分别为0.73米和0.86米，触球后速度达13.10米/秒，达到专业球员射门速度的59-71%。
