# A Classifier-Agnostic Zero-Shot Adversarial Attack Detection via CLIP

- arXiv: https://arxiv.org/abs/2606.30342v1
- arXiv ID: 2606.30342

---

🔍 **核心问题**：解决深度学习模型在面对对抗性攻击时，缺乏通用且无需攻击先验知识的检测方法问题。  
🛠 **方法**：提出一种完全黑盒、零样本的对抗攻击检测框架 \(A^4D\)，利用CLIP模型基于提示的相似性分数，通过观察CLIP对微小扰动的敏感性和嵌入空间偏移的规律性来检测攻击。  
✨ **主要贡献**：首次将CLIP用于对抗攻击检测，在攻击无关和分类器无关的设置下，跨多种攻击、数据集和分类器实现了最先进的检测性能。
