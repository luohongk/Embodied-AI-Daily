# Guided Action Flow: Q-Guided Inference for Flow-Matching Vision-Language-Action Policies

- arXiv: https://arxiv.org/abs/2607.02092v1
- arXiv ID: 2607.02092

---

🔍 **核心问题**：如何在无需重新训练基础策略的前提下，通过测试时引导提升预训练视觉-语言-动作（VLA）流匹配策略的机器人操作成功率。

🛠 **方法**：提出Guided Action Flow框架，冻结预训练的SmolVLA策略，额外学习一个动作块评论器（critic），在反向流采样过程中通过动作梯度引导生成更优的动作序列。

✨ **主要贡献**：在LIBERO操作任务上，单任务评论器将成功率从68.0%提升至82.0%（一个种子窗口），多任务评论器将验证成功率从46.0%提升至56.0%，验证了Q引导推理对冻结流匹配VLA策略的可行性，同时指出评论器泛化与不确定性感知引导仍是主要瓶颈。
