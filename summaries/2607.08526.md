# A Quantized Native Runtime for On-Device Semantic Audio Generation

- arXiv: https://arxiv.org/abs/2607.08526v1
- arXiv ID: 2607.08526

---

🔍 **核心问题**：在资源受限的消费级和嵌入式设备（如树莓派）上，实现无需Python或深度学习框架的语义音频生成（如文本到音乐），同时解决内存和计算效率问题。

🛠 **方法**：提出一个无依赖的原生运行时 *aria*，通过量化技术（8位和4位精度）降低模型数值精度以节省内存，并利用运行时对内部张量的完全控制实现激活引导（activation steering），以低成本调控生成内容。

✨ **主要贡献**：8位量化在无质量损失下显著降低内存并实现GPU最快速度；4位量化使12亿参数模型能在8GB树莓派5上运行，且启动速度比官方实现快约7倍；通过激活引导实现可控的“声音调味”生成，为物联网场景下的设备端语义音频提供了实用基础。
