# Ill-Posed by Design: Probing Evidence Use in VLMs

- arXiv: https://arxiv.org/abs/2606.24335v1
- arXiv ID: 2606.24335

---

🔍 **核心问题**：现有视觉语言模型（VLM）在“良态”任务中难以通过反事实分析有效评估其证据使用能力，因为多个线索可能同时支持同一答案，移除单一线索不改变预测结果。

🛠 **方法**：提出单目度量物体尺寸估计作为“病态”诊断任务，构建Metric VQA数据集（含10,813个维度查询和331个真实场景），对12个开源VLM（3B-397B参数）进行反事实分析，分解六种视觉与语言证据通道。

✨ **主要贡献**：发现最大VLM（如Qwen3-VL-235B）在真实场景中仍落后于纯文本大模型；目标身份是最关键线索，全局场景几何信息几乎未被利用；LoRA微调虽可学习任务，但模型未能学会利用场景几何。
