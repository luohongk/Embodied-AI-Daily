# A Bilateral Teleoperation Framework for Dexterous Manipulation

- arXiv: https://arxiv.org/abs/2606.15434v1
- arXiv ID: 2606.15434

---

🔍 **核心问题**：解决灵巧操作中双臂-手协调、低延迟反馈及复杂接触环境下的鲁棒交互问题。

🛠 **方法**：提出模块化双边遥操作框架，集成操作端输入接口与机器人端灵巧手、柔性臂，采用基于位置的手部重定向、差分臂控制、多尺度触觉反馈及共享控制策略。

✨ **主要贡献**：通过真实灵巧操作任务验证框架可行性，揭示跨具身不匹配、触觉反馈粒度及共享控制的设计洞见，为未来基于示教学习的高质量数据采集提供基础平台。
