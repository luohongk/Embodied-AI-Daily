# MuseVLA: An Adaptive Multimodal Sensing Vision-Language-Action Model for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.17598v1
- arXiv ID: 2606.17598

---

🔍 **核心问题**：现有机器人视觉-语言-动作（VLA）模型仅依赖RGB摄像头，无法感知温度、声音、雷达等物理属性，限制了复杂操作任务的执行能力。

🛠 **方法**：提出MuseVLA模型，通过生成传感器令牌和目标描述动态调用多种传感器，并将异构传感数据统一转换为“接地传感器图像”进行多模态融合与动作生成；同时引入数据合成流水线，利用现有RGB视频数据生成模拟传感器图像以降低训练成本。

✨ **主要贡献**：在真实机器人上实现温度引导抓取、音频驱动搜索、雷达辅助隐藏物体检索等任务，平均成功率80.6%，显著优于仅RGB和多传感器基线，并展现出强大的零样本泛化能力。
