# Antaeus: Hunting Repository-Level Logic Vulnerabilities via Context-Grounded LLM Reasoning

- arXiv: https://arxiv.org/abs/2607.01138v1
- arXiv ID: 2607.01138

---

🔍 **核心问题**：现有基于LLM的漏洞检测工具难以识别逻辑漏洞，因为这类漏洞需要推断应用特定的安全不变量和隐含假设，而这些信息常被埋没在无关代码中。

🛠 **方法**：提出Antaeus框架，通过仓库级代码上下文引导LLM推理，采用函数优先级排序、上下文推理、对比验证和结构化报告四步流程，结合本地代码与全局功能、安全资源和信任边界进行分析。

✨ **主要贡献**：在28个含确认逻辑漏洞的仓库上检测并解释了15个漏洞，性能优于函数级和智能体模型，且token使用量和成本相当。
