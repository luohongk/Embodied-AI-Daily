# You've Got a Golden Ticket: Improving Generative Robot Policies With A Single Noise Vector

- arXiv: https://arxiv.org/abs/2603.15757v3
- arXiv ID: 2603.15757

---

🔍 **核心问题**：如何在不重新训练或修改预训练生成式机器人策略（如扩散模型或流匹配策略）的前提下，仅通过优化输入噪声来提升其在任务中的表现。

🛠 **方法**：提出“黄金票”方法，即用精心选择的固定初始噪声向量替代从高斯分布中随机采样的噪声，并通过蒙特卡洛策略评估进行简单搜索来找到最优噪声，无需额外训练或新增网络。

✨ **主要贡献**：在51个模拟和真实机器人操作任务中，46个任务性能得到提升，成功率最高提升55%（模拟）和28%（真实）；该方法可扩展至多任务场景，单一噪声向量使VLA策略在7个任务上平均提升14%，且一个任务的“黄金票”还能提升相关任务表现。
