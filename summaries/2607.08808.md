# StereoSplat+: Feed-Forward Stereo Gaussian Splatting with Diffusion-Assisted Progressive Inference

- arXiv: https://arxiv.org/abs/2607.08808v1
- arXiv ID: 2607.08808

---

🔍 **核心问题**：从单一立体相机观测中重建高质量3D高斯场景（3DGS）面临遮挡、视野有限和几何缺失的挑战，而现有方法通常依赖多视角观测或未来帧的非因果访问。

🛠 **方法**：提出StereoSplat+框架，包含两个关键部分：一是StereoSplat，一种输入不变的馈送式3D高斯估计器，通过代价体积分支和三平面3D体积分支融合几何线索，并利用连续姿态编码适应不同视角和相机配置；二是扩散增强的渐进推理方案，从单个立体对开始，渲染新视角并用一步扩散增强器优化，再反馈更新3DGS。

✨ **主要贡献**：在KITTI-360数据集上，StereoSplat+在遮挡区域和强视角外推下显著提升了新视角渲染质量和几何精度，超越了近期馈送式3DGS基线方法。
