# Wh0: Generative World Models as Scalable Sources of Egocentric Human Hand Manipulation Data

- arXiv: https://arxiv.org/abs/2606.22136v1
- arXiv ID: 2606.22136

---

🔍 **核心问题**：现有灵巧操作数据来源在规模与场景/本体对齐之间存在矛盾——遥操作数据对齐好但成本高，仿真可扩展但存在仿真到现实差距，真实第一人称视频规模大但与机器人部署不对齐。

🛠 **方法**：提出Wh0框架，利用生成式视频世界模型作为可控的第一人称人手操作数据源，通过语言、物体和场景条件生成50k集WM-H数据集，再经手部运动重建和视觉编辑转化为机器人可训练监督信号。

✨ **主要贡献**：在18个真实灵巧操作任务中，Wh0将未见任务的零样本成功率从8.3%提升至38.9%，并证明可扩展生成与场景/本体对齐是性能提升的关键。
