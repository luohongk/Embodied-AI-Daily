# Parameter Efficient Machine Unlearning on Hybrid Resistive Memory based Compute-in-Memory Accelerators

- arXiv: https://arxiv.org/abs/2601.10037v3
- arXiv ID: 2601.10037

---

🔍 **核心问题**：电阻式存储器存算一体加速器在边缘模型适配（如机器遗忘和持续学习）中，因模拟权重频繁重编程的高成本（设备变异性和迭代写入验证操作）而受限。

🛠 **方法**：提出硬件-软件协同设计，将冻结的预训练权重映射到模拟电阻式存储阵列，而可训练的低秩适应分支（LoRA风格）置于SRAM连接的数字计算单元中，仅更新少量数字参数，避免重复编程模拟骨干网络。

✨ **主要贡献**：首次在制造出的电阻式存储器CIM加速器上实验验证了近似机器遗忘；在180nm 128x128 1T1R宏上实现人脸识别，并通过电路精确仿真验证说话人认证和风格化图像生成任务；相比直接更新模拟权重，混合映射将模拟训练/更新成本降低148倍，片上部署开销降低388倍，推理能耗降低59倍，同时保持竞争性任务性能。
