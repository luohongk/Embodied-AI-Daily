# Interpretable Sperm Morphology Classification via Attention-Guided Deep Learning

- arXiv: https://arxiv.org/abs/2606.20438v1
- arXiv ID: 2606.20438

---

🔍 **核心问题**：现有精子形态分类的深度学习模型缺乏可解释性，限制了其在临床诊断中的应用。

🛠 **方法**：提出基于注意力引导的深度学习框架，结合预训练的EfficientNet-B0与卷积块注意力模块（CBAM），并使用Grad-CAM++进行可视化解释。

✨ **主要贡献**：在SMIDS和HuSHem数据集上分别达到90.2%和93.9%的准确率（宏F1分数0.913和0.948），优于SimpleCNN和标准EfficientNet-B0，同时提供了可解释的决策依据。
