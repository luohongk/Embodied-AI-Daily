# Uncertainty-Aware Longitudinal Forecasting of Alzheimer's Disease Progression Using Deep Learning

- arXiv: https://arxiv.org/abs/2606.24604v1
- arXiv ID: 2606.24604

---

🔍 **核心问题**：现有深度学习模型通常将阿尔茨海默病进展简化为单步分类，无法预测患者随时间变化的轨迹，也无法量化预测的不确定性。

🛠 **方法**：提出一个概率框架，结合时序融合Transformer编码器、CORAL有序输出层、非对称损失加权和转换器过采样，并利用自回归混合密度网络生成五年期概率轨迹，同时通过解析混合方差和五成员bootstrap集成分离偶然不确定性与认知不确定性。

✨ **主要贡献**：在ADNI数据集上，模型在下一诊断预测中优于线性、循环和Transformer基线，尤其在MCI与痴呆区分上表现突出；生成的轨迹达到近90%的置信区间覆盖率，不确定性随预测时间增加而扩大，且认知不确定性在罕见进展模式、MCI和痴呆患者中更高，并在外部OASIS-3数据集上随预测误差增加而上升。
