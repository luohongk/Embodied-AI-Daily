## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人运动规划，具体为**任务空间（笛卡尔空间）下的反应式运动规划**，并考虑**关节空间约束**。
- **问题（Problem）**：传统的任务空间规划器（如Bug2算法）使用固定的笛卡尔步长，忽略了机器人的关节角度限制。当雅可比矩阵病态时，即使很小的笛卡尔步长也可能导致关节变化超出允许范围。对关节进行“裁剪”会导致跟踪漂移，甚至无法到达目标。
- **方法（Method）**：提出一种基于**平方和（SOS）规划**的方法，在每个规划步骤中，计算一个**可被认证的最大笛卡尔超矩形**，该矩形内的所有点都能在当前关节位移约束下被可靠地到达。将此认证集与Bug2规划器集成，实现自适应步长。
- **创新（Innovation）**：
    1.  首次提出使用SOS规划来**认证任务空间中的可达区域**，而非传统的状态/关节空间中的安全漏斗。
    2.  利用**二阶多项式近似逆运动学**和**S-过程**，将认证问题转化为一个可高效求解的小型半定规划（SDP）。
    3.  针对二次型结构，提出了一种**等效的二分法**，实现亚毫秒级的求解速度，适用于实时应用。
- **意义（Significance）**：解决了层级式任务空间规划器中一个根本性的安全漏洞，即固定步长在病态区域会违反关节约束。该方法为任何任务空间规划器提供了一个**通用的、每步可行的认证过滤器**，在不牺牲性能的前提下，从根本上消除了关节限制违规和由此产生的跟踪漂移，提升了机器人运动的安全性和可靠性。
- **一句话总结**：本文通过SOS规划实时计算并利用一个“认证可达盒”作为自适应步长，使得任务空间规划器在关节约束下既能保证安全（零违规），又能提升性能（100%成功率）。

## 1. 研究动机（Motivation）

- **核心问题**：在机器人运动规划中，任务空间（目标、障碍物）和关节空间（物理约束）之间存在**不匹配**。高层规划器在任务空间生成路径时，隐含地假设每个路径点都存在逆运动学（IK）解。这个假设在关节角度限制、速度限制或运动学奇异性存在时**经常失效**。
- **为什么重要**：在工业操作和人机协作等安全关键应用中，违反关节约束不仅是次优的，而且是**危险的**。例如，关节撞到机械限位可能导致损坏或伤害。
- **挑战在哪里**：层级式规划器（如Bug算法）的脆弱性在于，一个在某个构型下安全的固定笛卡尔步长，在另一个构型下可能因**雅可比矩阵条件数**的变化而违反关节限制。关节位移的放大倍数与雅可比矩阵的最小奇异值成反比（$\|\Delta\theta\| \leq \|\Delta z\| / \sigma_{\min}(J(\theta))$）。直接“裁剪”关节位移会导致末端执行器落在非预期位置，产生**累积的跟踪漂移**，最终导致任务失败。

## 2. 创新点（Innovation）

- **关键创新**：提出并求解了一个**认证可达集**问题：给定当前关节构型和每步关节位移界限，找到**最大的、可被保证到达的笛卡尔区域**。
- **与现有方案的不同**：
    - **现有方法**：使用固定步长或基于条件数的启发式变步长（如$s^{(t)} = \delta / \kappa(\theta^{(t)})$）。这些方法要么不安全，要么过于保守（仅考虑最坏情况），且依赖一阶线性模型。
    - **本文方法**：
        1.  **更精确的模型**：使用**二阶多项式**近似逆运动学，捕捉了线性模型忽略的非线性耦合。
        2.  **形式化保证**：利用**SOS规划**和**S-过程**，为**整个笛卡尔盒子**（而非单个方向）提供严格的数学保证，确保盒子内所有点都满足关节约束。
        3.  **自适应步长**：认证集的半宽度$\lambda^*$自动根据局部运动学条件（雅可比条件数）调整，在病态区域缩小，在良好区域扩大。
- **为什么能奏效**：
    - 二阶IK近似比一阶更准确，能更好地预测关节位移。
    - SOS规划提供了一个**充分且必要**（对于二次型问题）的条件来验证约束，保证了认证的**无损性（exactness）**。
    - 将认证集作为“可行性过滤器”集成到规划器中，确保了**每一步**的关节位移都在界限内，从而从根本上消除了裁剪和漂移。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

整体架构分为三个部分：
1.  **多项式逆运动学（Polynomial IK）**：在当前构型$\theta_0$处，构建一个从笛卡尔位移$\Delta z$到关节位移$\Delta \hat{\theta}$的二阶多项式映射，作为真实IK的局部近似。
2.  **SOS认证可达集（SOS-Based Reachable Set Certification）**：利用SOS规划，求解一个优化问题，找到最大的$\lambda^*$，使得对于所有$\|\Delta z\|_\infty \le \lambda^*$，多项式IK预测的关节位移都满足给定的界限$\delta$。这个$\lambda^*$定义了认证的笛卡尔盒子。
3.  **与Bug2规划器集成（Integration with Bug2）**：将标准的Bug2规划器的固定步长替换为自适应步长$s^{(t)} = \alpha \lambda^*$。每一步的笛卡尔位移$\Delta z^{(t)}$被限制在这个认证盒子内，然后通过多项式IK更新关节构型。

### 3.2 关键算法与数学推导

**1. 多项式逆运动学（二阶近似）**

对于第$i$个关节，其位移的二阶近似为：
$$\Delta \hat{\theta}_i(\Delta z) = \hat{\theta}_i(\Delta z) - \theta_{0,i} = A_{i1}\Delta z_1 + A_{i2}\Delta z_2 + b_{i,11}\Delta z_1^2 + b_{i,12}\Delta z_1\Delta z_2 + b_{i,22}\Delta z_2^2$$
其中$A = J(\theta_0)^\dagger$是雅可比伪逆（一阶项），$B_i$是二阶修正矩阵，通过有限差分计算。

**2. SOS认证（S-过程）**

将上述多项式写成二次型形式：
$$\Delta \hat{\theta}_i(\Delta z) = y^\top Q_i y$$
其中$y = (1, \Delta z_1, \Delta z_2)^\top$，$Q_i \in \mathbb{S}^3$。

认证问题转化为：找到最大$\lambda$，使得对所有$i$和符号$\sigma \in \{+1, -1\}$，满足：
$$p_{i,\sigma}(\Delta z) := \sigma \cdot y^\top Q_i y - \delta_i^{\text{eff}} \le 0, \quad \forall \Delta z \in [-\lambda, \lambda]^2$$

使用**S-过程**，将域约束$g_1(\Delta z) = \lambda^2 - \Delta z_1^2 \ge 0$和$g_2(\Delta z) = \lambda^2 - \Delta z_2^2 \ge 0$引入，得到一个充分条件：存在乘子$c_1, c_2 \ge 0$，使得以下矩阵半正定：
$$S_{i,\sigma} := -\sigma Q_i + \delta_i^{\text{eff}} E_{11} + c_1 G_1(\lambda) + c_2 G_2(\lambda) \succeq 0$$
其中$E_{11} = \text{diag}(1,0,0)$，$G_1, G_2$是编码域约束的矩阵。

**3. 优化问题**

最终求解的优化问题是：
$$\lambda^* = \max_{\lambda, c_1^{(i,\sigma)}, c_2^{(i,\sigma)}} \lambda$$
$$\text{s.t.} \quad S_{i,\sigma}(\lambda, c_1, c_2) \succeq 0, \quad \forall i, \sigma$$
$$c_1^{(i,\sigma)}, c_2^{(i,\sigma)} \ge 0, \quad 0 \le \lambda \le \lambda_{\max}$$

这是一个双线性SDP。由于$S_{i,\sigma}$是$3\times3$矩阵，可以通过**西尔维斯特准则**（所有前主子式非负）将其转化为一组多项式不等式，从而用非线性规划求解。

**4. 高效的二分法实现**

论文证明，对于二次多项式在盒子域上的最大值问题，SOS松弛是**无损的**。因此，可以通过**二分法**搜索$\lambda$，并在每次迭代中，在一个密集网格上评估$|\Delta \hat{\theta}_i(\Delta z)|$的最大值，以此作为可行性判断。这种方法避免了求解SDP，速度极快（亚毫秒级）。

### 3.3 模型、训练与数据集

- **模型**：一个**平面3连杆旋转机械臂**（$n=3$），连杆长度分别为1.0m, 0.8m, 0.6m。
- **训练**：本文方法**无需训练**。它是一个基于模型和优化的规划方法。
- **数据集**：通过一个**对抗性场景生成**流程，自动生成94个具有挑战性的规划场景。每个场景都满足：起始条件数适中、沿直线路径条件数显著增加（确保标准Bug2会失败）、且SOS规划器理论上可行。测试了6种不同的关节位移界限$\delta \in \{0.020, 0.025, 0.030, 0.035, 0.040, 0.050\}$ rad。

### 3.4 实验设置与结果

- **对比方法**：
    - **Vanilla Bug2（基线）**：固定步长$s = \delta / \kappa_0$，使用一阶伪逆IK，关节超限时进行裁剪。
    - **SOS-Verified Bug2（本文）**：自适应步长$s^{(t)} = 0.75 \cdot \lambda^*$，使用二阶多项式IK。
- **主要结果**（见论文表II, III, IV）：
    - **关节限制违规（RQ1）**：SOS规划器在所有94个场景和所有6种$\delta$设置下，实现了**零违规**。而Vanilla Bug2在6-11%的步骤中发生违规。
    - **任务成功率（RQ2）**：SOS规划器在所有设置下实现了**100%的成功率**。当$\delta \ge 0.035$时，Vanilla Bug2的成功率下降到82-93%。
    - **路径质量（RQ2）**：SOS规划器路径更短（路径长度比约为1.2-1.5），而Vanilla Bug2因漂移导致路径长度比高达8.8-10.3。
    - **计算成本**：SOS规划器每步约**0.3毫秒**（通过二分法实现），远低于典型规划周期，满足实时性要求。

### 3.5 技术细节

- **近似误差界**：使用一个$7\times7$的网格评估多项式IK的近似误差$\varepsilon(\rho)$，并从关节界限中减去它（$\delta_i^{\text{eff}} = \delta_i - \varepsilon(\rho)$），以提供保守但安全的保证。
- **自适应步长**：步长因子$\alpha=0.75$，用于提供额外的安全裕度。
- **雅可比非奇异假设**：方法假设雅可比矩阵非奇异，以确保多项式近似的有效性。如果$\lambda^*$太小，会尝试减小近似半径$\rho$。

- **贡献是什么**：

## 5. 澄清与简化（Clarifications and Simplifications）

- **核心思想类比**：想象一个在黑暗中行走的人（机器人），他想去一个地方（目标），但每走一步的步长（关节位移）是有限制的。他手里有一个手电筒（认证集），但手电筒的光束（步长）是固定的。在平坦开阔的地方（良好条件数），固定光束没问题。但当他走到一个布满坑洼和障碍物的地方（病态条件数），固定光束可能会让他一脚踩空（违反关节限制）。本文的方法就是给他一个**智能手电筒**，它能根据地形自动调节光束范围：在平坦处照得远（大步长），在崎岖处照得近（小步长），确保他**每一步都能安全地踩在光束照亮的地方**（认证可达集内）。
- **SOS规划的作用**：SOS规划就像一个**数学放大镜**，它用来检查一个“盒子”（笛卡尔超矩形）里的所有点。它要回答的问题是：“这个盒子里的每一个点，机器人是不是都能在不超出关节限制的情况下到达？” 如果答案是“是”，这个盒子就是“认证”过的。我们不断放大这个盒子，直到找到最大的那个。
- **为什么是“无损”的**：因为这个问题（二次多项式在二次域上的最大值）恰好是SOS规划能完美解决的。就像用一把精确的尺子去量一个正方形，结果就是精确的，没有近似误差。这保证了我们找到的$\lambda^*$是理论上最紧的界。
- **为什么SOS规划器路径更短**：Vanilla Bug2的路径变长，不是因为算法随机，而是因为**裁剪**。当它想走一步，但关节被“剪掉”一截后，它实际到达的位置和计划的位置不同。这个误差会累积，导致它“跑偏”，需要走更多弯路来纠正。SOS规划器因为每一步都“说到做到”（关节位移完全符合预期），所以它走的路径就是Bug2算法在理想无约束情况下应该走的直线或最优路径。
