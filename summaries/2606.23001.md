# EnerInfer: Energy-Aware On-Device LLM Inference

- arXiv: https://arxiv.org/abs/2606.23001v2
- arXiv ID: 2606.23001

---

🔍 **核心问题**：设备端大语言模型推理面临能耗与散热瓶颈，现有系统仅优化解码速度，忽略了可通过调整NPU/内存频率来兼顾能效与用户体验的潜力。

🛠 **方法**：提出EnerInfer框架，采用解耦的模型结构感知预测与排名驱动的在线反馈机制，无需逐模型配置或传感器，通过预测不同频率下的吞吐与功耗，动态选择满足服务质量的高能效配置，并结合轻量级热预测切换节能与散热模式。

✨ **主要贡献**：在手机、笔记本和开发板上分别实现高达65%、12%和24%的能效提升，且不降低用户体验质量，首次实现设备端LLM推理的能效、吞吐与热舒适联合管理。
