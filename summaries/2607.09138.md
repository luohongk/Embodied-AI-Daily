# BeyondSight: Object Permanence for End-to-End Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.09138v1
- arXiv ID: 2607.09138

---

🔍 **核心问题**：自动驾驶在部分可观测环境中，被遮挡的物体（如车辆或行人）会因瞬时观测缺失而消失，导致下游预测和规划忽略潜在关键目标。

🛠 **方法**：提出BeyondSight框架，通过时间维度传播并更新持久化的物体查询（actor queries），结合观测证据维持被遮挡物体的假设，实现感知、预测和规划的联合推理。

✨ **主要贡献**：引入nuScenes-Permanence数据集，提供被遮挡物体的监督与评估；实验表明，BeyondSight将被遮挡物体的检测mAP从0提升至0.249，并将规划误差L2avg从0.61降至0.54，验证了物体恒存性对鲁棒端到端驾驶的重要性。
