# Pause and Think: A Dataset and Benchmark for Video-Grounded Assistive Action Suggestion

- arXiv: https://arxiv.org/abs/2606.00616v2
- arXiv ID: 2606.00616

---

🔍 **核心问题**：现有视觉-语言模型在视频理解中缺乏基于场景的推理、时间一致性和上下文感知规划能力，难以生成可操作的辅助建议。

🛠 **方法**：构建了pause-and-think-T推理训练数据集，引导模型在生成答案前先暂停并基于视觉证据进行结构化推理；微调了一个4B参数的紧凑模型，并在pause-and-think-B基准上评估。

✨ **主要贡献**：该紧凑模型在场景理解和目标规划任务中达到58.0%准确率，参数仅为Qwen3-VL-235B的1/59，性能接近GPT-5.2并超越GPT-4o；在EgoThink和TempCompass等分布外任务上表现出色，尤其在功能识别、辅助推理和时间顺序方面显著提升，无需大规模模型扩展。
