# Ensuring Deterministic Timing in a Federated GNSS Correction Pipeline with Lingua Franca

- arXiv: https://arxiv.org/abs/2606.31415v1
- arXiv ID: 2606.31415

---

🔍 **核心问题**：解决联邦式GNSS校正数据流水线中，因硬件中断、缓冲和分布式通信导致的时序不确定性与分析困难问题。

🛠 **方法**：采用Lingua Franca框架，将端到端流水线分解为具有显式时间语义的反应器（reactors），包括时间触发的GNSS接收器、基于波特率和FIFO缓冲的UART中断流、周期性转发任务及下游抖动监控。

✨ **主要贡献**：通过联邦式执行与运行时日志验证了分析推导的确定性时序结构（如中断节奏、环形缓冲演化、分包行为及物理-逻辑抖动），实现了可复现、可预测的时序轮廓。
