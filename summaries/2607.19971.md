# Unified Prediction and Planning via Conflict-Aware Disjoint Parameter Training

- arXiv: https://arxiv.org/abs/2607.19971v1
- arXiv ID: 2607.19971

---

🔍 **核心问题**：在资源受限的边缘设备上，统一预测与规划模型因预测邻居行为与自我安全规划的目标不同，导致参数重叠引发“技能冲突”，损害模型性能。

🛠 **方法**：提出基于模型合并的“分离参数训练”（DPT）框架，通过分布式参数学习分离各任务的关键参数区域，并在合并时采用稀疏合并策略，仅整合对每个任务最关键的参数。

✨ **主要贡献**：首次识别并解决统一模型中的技能冲突问题；DPT框架可并行应用于多种合并方法；在JRDB和JTA基准上验证了其优越性能，适用于安全且资源高效的机器人导航。
