# Vulnerability of Natural Language Classifiers to Evolutionary Generated Adversarial Text

- arXiv: https://arxiv.org/abs/2606.27215v1
- arXiv ID: 2606.27215

---

🔍 **核心问题**：自然语言分类模型易受对抗性攻击，现有方法常需模型内部结构信息，且攻击的语义相似性不足。

🛠 **方法**：提出GAversary，一种基于遗传算法的黑盒攻击方法，利用GloVe词嵌入指导词替换（变异操作），仅需模型输出的logit值即可生成对抗样本。

✨ **主要贡献**：在多个基准数据集上，GAversary显著降低目标模型准确率（最佳从76.8%降至5.8%），优于BAE和A2T攻击；代价是扰动词数略多（约两倍）、语义相似性稍低，且运行时间增加约5%。
