# Co-policy: Responsive Human-Robot Co-Creation for Musical Performances

- arXiv: https://arxiv.org/abs/2606.19914v1
- arXiv ID: 2606.19914

---

🔍 **核心问题**：在机器人音乐共创中，如何将语义音乐理解与实时、可物理执行的演奏动作有效连接，避免仅作为用户指令的机械回放。

🛠 **方法**：提出Co-policy框架，包含三个模块：使用预推理语义锚点和微调Qwen-vl规划器（F-Qwen）将语音、音乐种子和视觉观测转化为结构化共创计划；引入高斯混合视觉运动策略（GMP），通过条件混合密度网络将目标音符和视觉上下文映射为多模态机器人动作，实现单次前向推理的低延迟执行。

✨ **主要贡献**：在真实机器人钟琴实验中，Co-policy相比扩散策略和消融基线，在意图对齐、执行准确性和响应频率上均有提升，验证了物理约束下的动作生成是实现具身人机共创的关键。
