# TypiCore: A Hybrid Active Query Strategy for Class-Incremental Learning on Time Series

- arXiv: https://arxiv.org/abs/2607.17632v1
- arXiv ID: 2607.17632

---

🔍 **核心问题**：针对多变量时间序列数据，在标注预算有限且需增量学习新类别的场景下，现有主动学习方法难以平衡模型的可塑性、稳定性与标签效率。

🛠 **方法**：提出TypiCore混合查询策略，在主动学习周期中交替使用基于典型性和多样性的样本选择，构建兼具代表性和多样性的记忆缓冲区。

✨ **主要贡献**：在TSCIL基准测试中，TypiCore显著优于所有基线方法，并在多个数据集上达到或超越全监督持续学习性能，同时仅需使用部分标签。
