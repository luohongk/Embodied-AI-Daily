# Selective Time Series Forecasting via Metalearning

- arXiv: https://arxiv.org/abs/2606.23448v1
- arXiv ID: 2606.23448

---

🔍 **核心问题**：时间序列预测中，不同样本的预测难度差异大，现有拒绝机制（允许模型放弃高风险预测）多依赖与训练领域强相关的代理指标，导致跨领域泛化能力不足。

🛠 **方法**：提出基于元学习的选择性预测框架，通过元学习从近期滞后数据中提取结构特征，建模预测误差的经验百分位数（一种尺度不变的统计量），从而将拒绝决策与预测本身解耦，并基于领域无关特征实现跨异构时间序列的拒绝迁移。

✨ **主要贡献**：首次将拒绝机制与元学习结合用于时间序列预测，在域内和迁移学习实验中证明，拒绝被判定为困难的样本能持续提升各覆盖水平下的预测精度。
