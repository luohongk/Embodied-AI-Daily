# EA-Nav: Learning Safe Visual Navigation Policies with Embodiment Awareness

- arXiv: https://arxiv.org/abs/2607.19880v1
- arXiv ID: 2607.19880

---

🔍 **核心问题**：跨形态（cross-embodiment）视觉导航中，不同智能体面对相同视觉观察时动作预测存在歧义，现有方法依赖强化学习难以支持可扩展预训练和真实场景适应。

🛠 **方法**：提出基于模仿学习的形态感知导航框架（EA-Nav），采用模块化多阶段设计。预训练阶段从互联网视频构建跨形态数据集，引入形态几何作为条件令牌减少动作歧义；微调阶段通过解耦架构注入多模态信息，并设计轨迹增强策略生成高风险样本，分别训练空间感知与风险感知校正。

✨ **主要贡献**：首次将形态几何显式融入模仿学习导航框架，有效提升不同形态设置下的导航性能，验证了形态几何在具身导航中的有效性。
