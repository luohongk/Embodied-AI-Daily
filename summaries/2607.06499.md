# Clustering-Embedded Model Predictive Path Integral Control: Avoiding Averaging-Induced Failure and Enabling Efficient Cluster Selection for Dynamic Obstacles

- arXiv: https://arxiv.org/abs/2607.06499v1
- arXiv ID: 2607.06499

---

🔍 **核心问题**：标准MPPI控制方法在复杂环境中因重要性加权更新平均了不兼容的轨迹，导致在障碍物前方出现犹豫甚至碰撞的“平均诱导失败”问题。

🛠 **方法**：提出聚类嵌入MPPI（CE-MPPI）框架，通过集成DBSCAN聚类和基于碰撞参考点的几何方向特征，从不可行轨迹噪声中分离可行模式，并结合智能选择逻辑（静态场景最小成本、动态场景主动避开障碍物通量）重新定义控制律。

✨ **主要贡献**：在2D仿真中缓解了障碍物前的犹豫和动态场景中的持续耦合；在6自由度UR5e机械臂真实实验中，实现目标时间减少48%，末端执行器路径缩短12%。
