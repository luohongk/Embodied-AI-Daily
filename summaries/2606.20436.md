# Multi-View Decompilation for LLM-Based Malware Classification

- arXiv: https://arxiv.org/abs/2606.20436v1
- arXiv ID: 2606.20436

---

🔍 **核心问题**：现有基于大语言模型的恶意软件分类方法通常只依赖单一反编译器（如Ghidra或RetDec）的输出，但反编译器是启发式工具，可能丢失或扭曲二进制文件中的关键信息，导致分类不准确。

🛠 **方法**：构建包含良性工具和恶意程序的基准数据集，将每个样本分别用Ghidra和RetDec反编译，生成匹配的伪C代码视图，然后通过多视图提示（同时输入两个反编译器的输出）让大语言模型进行分类。

✨ **主要贡献**：实验表明，提供多个反编译器视图可显著提升恶意样本分类的F1分数（主要通过提高召回率），且不同反编译器的错误具有互补性，证明多反编译器提示是一种简单、无需额外训练即可提升LLM恶意软件分类性能的有效方法。
