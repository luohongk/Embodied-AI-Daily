# ASALT: Adaptive State Alignment for Lateral Transfer in Multi-agent Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.24601v1
- arXiv ID: 2606.24601

---

🔍 **核心问题**：现有多智能体强化学习中的迁移学习方法通常要求源域与目标域的观测空间和全局状态空间维度一致，限制了跨异构域的策略迁移能力。

🛠 **方法**：提出ASALT方法，通过引入观测级和状态级适配器，将目标域的观测和全局状态映射到共享嵌入空间，从而在演员和评论家网络间实现知识迁移。

✨ **主要贡献**：在标准基准环境的多种配置下，ASALT在样本效率和合作任务的总回报上优于现有基线，并能有效缓解因域间观测与动作空间不匹配导致的负迁移问题。
