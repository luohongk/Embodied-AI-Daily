# From Weights to Features: SAE-Guided Activation Regularization for LLM Continual Learning

- arXiv: https://arxiv.org/abs/2606.26629v1
- arXiv ID: 2606.26629

---

🔍 **核心问题**：解决大语言模型在持续学习中因权重空间正则化方法（如EWC）过于粗糙而导致的灾难性遗忘问题，特别是权重多义性导致知识保护不精准。

🛠 **方法**：提出基于稀疏自编码器（SAE）的激活空间正则化方法，利用SAE作为单义特征字典，通过约束优化推导新损失函数，在激活空间中显式平衡稳定性与可塑性，并仅保留紧凑的SAE特征掩码，无需存储先前任务数据。

✨ **主要贡献**：在TRACE和MedCL基准上超越EWC等传统方法，且无需任务特定架构；实验证明任务相关表示在SAE特征空间线性可分，而权重空间保护几乎无概念选择性，验证了多义性假说。
