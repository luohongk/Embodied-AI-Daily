# CoSAG: Compact Semantic Anchor Gaussians via Training-Free Rate-Distortion Coding

- arXiv: https://arxiv.org/abs/2607.10237v1
- arXiv ID: 2607.10237

---

🔍 **核心问题**：现有开放词汇3D场景理解方法将高维视觉-语言特征附加到每个高斯体上，导致单场景存储达数GB，成为部署瓶颈。

🛠 **方法**：提出CoSAG，通过闭式透射率加权提升、空间语义锚点、多视图去噪构建语义场，并采用空间预测熵编码压缩每个高斯体到锚点表的绑定关系，无需逐场景训练。

✨ **主要贡献**：实现亚兆字节级存储，在2D渲染、3D选择、密集LSeg协议上匹配或超越现有最优方法，相比LangSplatV2将场景大小压缩37-76倍且精度更高。
