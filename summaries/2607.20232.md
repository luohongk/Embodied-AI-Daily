# DINS-IO: Learned Inertial Odometry via Differentiable INS Consistency

- arXiv: https://arxiv.org/abs/2607.20232v1
- arXiv ID: 2607.20232

---

🔍 **核心问题**：现有学习型惯性里程计训练依赖昂贵且难以大规模获取的高精度位置真值（如动作捕捉或SLAM），本文旨在无需位置标签，仅从原始IMU数据学习惯性里程计。

🛠 **方法**：提出DINS-IO，利用捷联惯导速度递归作为可微分的自监督先验，构建滑动窗口最小二乘问题并解析求解，将残差作为自监督损失；同时设计高频网络输出体坐标系速度，并通过少量带标签轨迹进行低秩适配（LoRA）校准。

✨ **主要贡献**：首次实现无需位置标签的自监督惯性里程计学习，在标准基准上，自监督预训练后仅用少量标签微调即可达到或超越全监督基线，显著降低了对昂贵标注数据的依赖。
