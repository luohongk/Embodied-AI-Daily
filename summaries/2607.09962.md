# Task Planning for Mobile Manipulation in Retail Stores using Foundation Models with Iterative Re-planning

- arXiv: https://arxiv.org/abs/2607.09962v1
- arXiv ID: 2607.09962

---

🔍 **核心问题**：解决零售场景（如超市）中机器人自动补货的任务规划问题，应对货架环境多变且以人为中心的特点。  
🛠 **方法**：基于大型语言模型（LLMs）和视觉-语言模型（VLMs）进行任务规划，结合用户驱动提示和基于反馈的迭代重规划机制实现错误修正，并在全向移动操作平台上验证。  
✨ **主要贡献**：提出了一种结合基础模型与迭代重规划的端到端任务规划方法，在PyBullet仿真环境中成功验证了拾取与放置任务，为零售业自动化提供了可扩展的解决方案。
