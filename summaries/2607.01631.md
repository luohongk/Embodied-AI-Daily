# Path planning for unmanned naval surface vehicles

- arXiv: https://arxiv.org/abs/2607.01631v1
- arXiv ID: 2607.01631

---

🔍 **核心问题**：解决无人水面艇（USV）在已知固定障碍物和未知移动障碍物（如船只、游泳者等）环境中的实时路径规划问题。  
🛠 **方法**：结合全局路径规划器（融合Grassfire、改进Grassfire和新版概率路线图三种算法）与局部路径规划器（基于障碍物相对全局路径的运动方向，采用高逻辑决策引导USV从障碍物后方绕行）。  
✨ **主要贡献**：提出新型全局与局部路径规划组合方案，局部规划器通过系统性绕行策略（而非平行移动）提升避障效率，仿真验证其优于D*等传统算法。
