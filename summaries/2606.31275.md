# CLIMB: Centroid-Based Hierarchical Memory for Online Continual Self-Supervised Learning

- arXiv: https://arxiv.org/abs/2606.31275v1
- arXiv ID: 2606.31275

---

🔍 **核心问题**：在线持续自监督学习（OCSSL）面临无标签数据流、无任务边界和内存限制的挑战，现有方法要么依赖重放缓冲区，要么仅使用正则化，难以兼顾表示质量与内存效率。

🛠 **方法**：提出CLIMB方法，结合基于质心的层次化记忆库（总存储图像数有限）与知识蒸馏技术，通过将相似图像分组为质心，提供难区分的对比学习样本，同时覆盖数据分布的多样性。

✨ **主要贡献**：在Split CIFAR-100和Split ImageNet-100等标准基准及不规则任务分布新协议上，CLIMB均超越现有OCSSL方法，首次同时实现重放与正则化的有效协同。
