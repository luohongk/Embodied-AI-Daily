# When Does Continual Learning Require Learning

- arXiv: https://arxiv.org/abs/2607.07847v1
- arXiv ID: 2607.07847

---

🔍 **核心问题**：论文探讨在大型语言模型（LLM）中，持续学习是否真正需要模型内部权重更新，还是可以通过外部框架（如上下文管理）实现，并分析不同环境变化模式下的学习需求。

🛠 **方法**：将环境变化分解为“空间”（新领域）和“时间”（数据漂移）两个维度，设计统一评估协议，比较提示方法（GEPA、ACE）、监督学习（SFT、SDFT）、强化学习（GRPO、SDPO）和上下文压缩（Cartridges、In-place TTT）在连续任务中的表现。

✨ **主要贡献**：揭示持续学习并非单一能力，不同变化模式需要不同更新策略；提示方法快速适应但遗忘快，蒸馏方法稳定但难更新旧知识，强化学习适应性强但对噪声敏感；为设计更强持续学习系统提供指导。
