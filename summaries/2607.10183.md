# Automated Tensor Scheduling for Hybrid CPU-GPU LLM Inference on Consumer Devices

- arXiv: https://arxiv.org/abs/2607.10183v2
- arXiv ID: 2607.10183

---

🔍 **核心问题**：在消费级设备（如笔记本）上运行大语言模型时，模型权重常超出GPU显存，需借助CPU内存进行卸载推理，但现有系统仅按层或专家粒度调度，忽略了张量间的异构性，且难以适应动态硬件负载。

🛠 **方法**：提出ATSInfer系统，采用张量粒度的卸载策略，结合静态张量放置与负载感知的动态传输，并通过异步CPU-GPU协调来高效调度存储、数据移动与计算。

✨ **主要贡献**：在消费级平台上，ATSInfer相比现有系统将预填充吞吐量提升最高1.94倍，解码吞吐量提升最高3.29倍，同时提高了GPU利用率和PCIe带宽利用率，显著改善了本地LLM部署的用户体验。
