# Skill-Constrained Model Predictive Control for Resilient Manufacturing Supply Chains

- arXiv: https://arxiv.org/abs/2606.17269v1
- arXiv ID: 2606.17269

---

🔍 **核心问题**：在技能约束的生产-库存系统中，如何通过动态决策（生产、库存、积压和培训）应对技能瓶颈、需求冲击、人员缺勤等扰动，提升供应链韧性。

🛠 **方法**：提出一种闭环技能约束模型预测控制器，每班次求解有限时域混合整数规划，包含二进制认证状态、硬性生产资格和可解释的终端价值项，仅执行第一期动作后重新规划。

✨ **主要贡献**：通过合成场景实验发现，预测控制仅在技能或劳动力瓶颈可提前预测时有效；在突发冲击、需求-产能边界附近等情况下，静态交叉培训保险策略更难被超越，揭示了预测控制的有效性取决于可预测性而非适应性本身。
