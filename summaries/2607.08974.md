# CLAP: Direct VLM-to-VLA Adaptation via Language-Action Grounding

- arXiv: https://arxiv.org/abs/2607.08974v1
- arXiv ID: 2607.08974

---

🔍 **核心问题**：将预训练的视觉语言模型（VLM）直接转化为视觉语言动作模型（VLA）时，预测动作作为纯数字序列会导致输出分布与VLM预训练的语言分布不匹配，从而削弱其原有能力。

🛠 **方法**：提出CLAP（因果语言-动作预测），在每个数字动作序列前添加自然语言动作描述，通过语言动作计划因果地约束精确动作标记预测，且不修改骨干架构。

✨ **主要贡献**：仅通过单轮微调，2B参数CLAP在LIBERO上达到90.8%准确率（比VLA-0高14.9个百分点），并在LIBERO-PRO上提升了对语言、物体和空间扰动的鲁棒性；同时发布0.8B、2B和4B多尺度紧凑VLA模型族，支持可控的VLM到VLA能力迁移分析。
