# Nonparametric Control Koopman Operators

- arXiv: https://arxiv.org/abs/2405.07312v5
- arXiv ID: 2405.07312

---

🔍 **核心问题**：现有控制系统的Koopman算子方法依赖显式的字典或输入参数化，限制了模型在无限维空间中的灵活性和高维系统的适用性。

🛠 **方法**：在再生核希尔伯特空间中提出无参数化的Koopman复合算子表示框架，利用不同模型表示间的等价性统一无限维回归与算子学习，并通过草图技术提升高维系统的可扩展性。

✨ **主要贡献**：实现了无限维空间中的任意精度有限秩近似，无需预先限制函数或输入跨度；数值实验表明在高维场景下预测精度显著优于双线性EDMD，且学习模型可直接与线性参数变化技术结合用于模型预测控制。
