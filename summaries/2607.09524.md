# Balancing Usefulness and Naturalness: An LLM-based Curation Pipeline for Code Review Comments

- arXiv: https://arxiv.org/abs/2607.09524v1
- arXiv ID: 2607.09524

---

🔍 **核心问题**：现有代码评审数据集存在噪声大、不一致或结构差的问题，限制了大型语言模型（LLM）生成准确、有用且类人评审注释的能力。

🛠 **方法**：提出两种基于LLM的策展流水线：第一种系统性地重写所有评审注释以提升清晰度、简洁性和礼貌性；第二种通过高质量范例分离数据集，保留优质注释并作为上下文示例引导低质量注释的重写，增强真实性和多样性。

✨ **主要贡献**：构建了更清洁、高质量的CuREV数据集，显著提升下游任务（如注释生成和代码优化）的性能；通过范例引导的流水线使重写后的注释更清晰、可操作，并展现更丰富的写作风格，更接近人类评审。
