# Climate-Driven Mortality Forecasting Using Deep Learning

- arXiv: https://arxiv.org/abs/2606.26980v1
- arXiv ID: 2606.26980

---

🔍 **核心问题**：传统死亡率模型无法预测由极端气候事件引发的死亡率突然飙升，需要建立能捕捉气候驱动超额死亡率的预测框架。

🛠 **方法**：提出两阶段建模框架，先构建区域周度Lee-Carter基线模型捕捉长期趋势与季节模式，再结合两种深度学习架构——CNN-LSTM（捕捉区域时序响应）和GNN-LSTM（建模空间死亡率依赖与气候影响传播），并扩展为分位数LSTM框架生成时变预测区间。

✨ **主要贡献**：两种架构均优于Lee-Carter基线和MortFCNet模型，测试MSE降低约24%，尤其在高龄群体中表现显著；时变预测区间为保险公司和养老基金提供了更准确的气候相关长寿风险评估工具。
