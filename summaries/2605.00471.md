# Stereo Multistage Spatial Attention for Real-Time Mobile Manipulation Under Visual Scale Variation and Disturbances

- arXiv: https://arxiv.org/abs/2605.00471v1
- arXiv ID: 2605.00471

---

🔍 **核心问题**：机器人在开放非结构化环境中移动时，车载摄像头视角的持续变化会导致目标物体视觉尺度显著变化，影响基于视觉的运动生成，需要解决视觉尺度变化和干扰下的实时移动操作问题。

🛠 **方法**：提出基于立体多阶段空间注意力的深度预测学习方法，从立体图像中提取任务相关的空间注意力点，并通过层次化循环架构与机器人状态融合，实现闭环动作预测。

✨ **主要贡献**：在四种真实移动操作任务（刚性放置、铰接物体操作、可变形物体交互）中，相比代表性模仿学习和视觉-语言-动作基线方法，在随机初始位置和视觉干扰条件下显著提升了鲁棒性和任务成功率，验证了结构化立体空间注意力结合预测性时间建模的有效性。
