# Switch-JustDance: Benchmarking Whole Body Motion Tracking Controllers Using a Commercial Console Game

- arXiv: https://arxiv.org/abs/2511.17925v3
- arXiv ID: 2511.17925

---

🔍 **核心问题**：当前缺乏低成本、可复现且能直接对比人类与机器人全身运动控制性能的标准化基准测试方法。

🛠 **方法**：利用Nintendo Switch上的《Just Dance》游戏，通过动作流传输、重建和重定向模块，将游戏编舞转化为机器人可执行动作，并借助游戏内置评分系统评估控制器性能。

✨ **主要贡献**：验证了《Just Dance》作为基准测试平台的可靠性、有效性和敏感性，并在真实硬件上对三种先进的人形机器人全身控制器进行了基准测试，揭示了其相对优势与局限。
