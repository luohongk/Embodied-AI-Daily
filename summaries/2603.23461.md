# End-to-End Efficient RL for Linear Bellman Complete MDPs with Deterministic Transitions

- arXiv: https://arxiv.org/abs/2603.23461v2
- arXiv ID: 2603.23461

---

🔍 **核心问题**：在满足线性贝尔曼完备性的马尔可夫决策过程中，如何设计一个计算高效的强化学习算法，以处理确定性转移、随机初始状态和随机奖励，并避免对动作空间或特征空间的强假设。

🛠 **方法**：提出了一种端到端高效的算法，对于有限动作空间直接实现多项式复杂度；对于大或无限动作空间，仅需标准的argmax oracle（即对动作进行最大值查询），结合线性函数近似和贝尔曼备份的线性性质进行学习。

✨ **主要贡献**：首次在确定性转移的线性贝尔曼完备MDP中实现了计算与样本复杂度均为多项式（关于时间步长、特征维度和1/ε）的算法，且无需强oracle假设，显著提升了实际可行性。
