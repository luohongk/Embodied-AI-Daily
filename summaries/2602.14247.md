# Path Planning Optimisation for SParse, AwaRe and Cooperative Networked Aerial Robot Teams (SpArC-NARTs): Optimisation Tool and Ground Sensing Coverage Use Cases

- arXiv: https://arxiv.org/abs/2602.14247v2
- arXiv ID: 2602.14247

---

🔍 **核心问题**：针对稀疏感知与协作的无人机网络团队（SpArC-NART）在探索任务中，如何优化路径规划以平衡有限能量、通信约束和任务目标（如区域覆盖或目标搜索）。

🛠 **方法**：提出一种路径规划优化工具，同时考虑环境先验信息、用户自定义通信模型和软运动约束，通过动态奖励机制（基于运动价值和预期通信可用性）促进代理间协作。

✨ **主要贡献**：首次在离线规划阶段整合稀疏通信、能量限制和协作行为，提升团队全局态势感知与任务重规划效率，并通过地面传感覆盖用例验证工具的有效性。
