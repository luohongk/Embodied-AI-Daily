# Reinforcement Learning without Ground-Truth Solutions can Improve LLMs

- arXiv: https://arxiv.org/abs/2606.27369v1
- arXiv ID: 2606.27369

---

🔍 **核心问题**：现有强化学习方法依赖真实答案（ground-truth）来分配奖励，限制了其在无法获得真实解的任务（如启发式优化）中的应用。

🛠 **方法**：提出RiVER框架，利用确定性执行反馈作为连续值监督信号，通过实例级校准奖励（解决规模主导和频率主导问题），并强调排名靠前的解，在无真实解的情况下训练LLM。

✨ **主要贡献**：在AtCoder启发式竞赛任务上训练后，RiVER在ALE-Bench上提升Qwen3-8B和GLM-Z1-9B-0414的评分排名8.9%和9.4%，并意外地在精确解基准（LiveCodeBench、USACO）上平均提升2.4%和3.5%，而基线方法无法迁移。这表明无真实解的评分优化任务结合适当奖励校准可有效提升通用编码能力。
