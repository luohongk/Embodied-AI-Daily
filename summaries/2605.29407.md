# Phase-Conditioned Imitation Learning with Autonomous Failure Recovery for Robust Deformable Object Manipulation

- arXiv: https://arxiv.org/abs/2605.29407v1
- arXiv ID: 2605.29407

---

🔍 **核心问题**：标准模仿学习策略（如ACT）在推理时依赖马尔可夫假设，导致视觉相似但需不同动作时出现状态混淆，且无法自主从执行失败中恢复。

🛠 **方法**：提出一种闭环分层架构，包括FiLM条件编码器（根据任务阶段调制特征提取）、多模态阶段预测器（融合视觉、力与位姿反馈实时估计阶段并检测接触失败），以及混合阻抗控制器与力觉遥操作接口。

✨ **主要贡献**：FiLM调制显著优于无条件与token级条件基线，t-SNE分析显示其生成阶段特异性特征；在双臂挂取T恤任务中，闭环系统通过自主错误恢复将成功率从56%提升至87%。
