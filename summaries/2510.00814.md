# RTFF: Random-to-Target Fabric Flattening Policy using Dual-Arm Manipulator

- arXiv: https://arxiv.org/abs/2510.00814v2
- arXiv ID: 2510.00814

---

🔍 **核心问题**：解决随机褶皱织物在双机械臂操作中同时实现展平与位姿对齐的难题，两者相互耦合（展平会改变位姿，对齐会引入褶皱）。

🛠 **方法**：提出混合模仿学习-视觉伺服（IL-VS）策略，通过将当前与目标织物状态锚定到同一模板网格实现顶点级评估，并设计网格动作分块变换器（MACT）进行粗对齐，再通过视觉伺服实现精确收敛。

✨ **主要贡献**：首次定义随机到目标织物展平任务，提出无需配准的网格表示方法，并在真实双机械臂遥操作系统中验证了对未见目标位姿、织物类型和尺度的精准对齐能力。
