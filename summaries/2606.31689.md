# ScratchWorld: Evaluating If World Models Compute Executable Consequences

- arXiv: https://arxiv.org/abs/2606.31689v1
- arXiv ID: 2606.31689

---

🔍 **核心问题**：现有世界模型评估常因复制静态状态而虚高准确率，无法真正检验模型是否理解可执行规则与因果后果。

🛠 **方法**：提出ScratchWorld离线诊断基准，基于Scratch项目与可执行虚拟机，提供回放验证的转移、隐藏变量、因果轨迹与反事实结果，并采用价值感知的变化字段F1指标。

✨ **主要贡献**：在659个样例的测试中，七个语言/推理模型在状态部分观测压力测试下最高仅达13.8%变化字段F1；复制输入状态可达到98.0%全状态字段准确率但变化字段F1为0%，揭示重叠混淆问题；辅助诊断显示模型常对动作或干预做出反应，却未遵循决定变化值的可执行规则。
