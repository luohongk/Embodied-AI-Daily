# KeepLoRA++: Continual Learning with Layer-Scaled Residual Gradient Adaptation

- arXiv: https://arxiv.org/abs/2606.16256v1
- arXiv ID: 2606.16256

---

🔍 **核心问题**：如何平衡持续学习中的三个竞争目标——保留预训练知识、保持已学任务知识、维持获取新知识的可塑性。

🛠 **方法**：提出KeepLoRA++，通过层缩放残差梯度适配方法，将新任务的LoRA参数更新限制在残差子空间，并结合浅层到深层的缩放策略，同时将梯度投影到与预训练主子空间和旧任务主导方向正交的子空间。

✨ **主要贡献**：揭示了知识在Transformer层间和层内的分布规律（浅层和主子空间编码通用知识，深层和残差子空间编码任务特定知识），并在图像分类、视觉问答和视频理解任务上持续优于现有基线方法。
