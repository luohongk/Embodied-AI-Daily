# Efficient Multi-Robot Motion Planning with Precomputed Translation-Invariant Edge Bundles

- arXiv: https://arxiv.org/abs/2605.09801v1
- arXiv ID: 2605.09801

---

🔍 **核心问题**：解决多机器人运动规划中，因机器人间密集时空约束导致的高计算复杂度和低规划效率问题。

🛠 **方法**：提出KiTE-Extend机制，通过离线预计算轨迹段库，在线规划时引导动作选择，不改变状态传播、碰撞检测或代价评估，且保持原有理论保证。

✨ **主要贡献**：显著减少规划时间并提升可扩展性，在集中式、优先级和冲突基三种主流多机器人规划范式上均验证有效，尤其适用于多机器人密集交互场景。
