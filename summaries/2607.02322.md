# The Moving Eye: Enhancing VLA Spatial Generalization via Hybrid Dynamic Data Collection

- arXiv: https://arxiv.org/abs/2607.02322v1
- arXiv ID: 2607.02322

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在机器人操作中空间泛化能力不足，容易陷入“捷径学习”，即依赖物体间或相机与机器人基座间的固定相对位姿等虚假相关性，而非真正理解空间关系。

🛠 **方法**：提出一种数据驱动的混合动态数据采集策略，利用双臂机器人系统，其中一臂执行操作任务，另一臂作为移动环境相机。系统对比了固定视角、多固定视角和连续移动视角三种数据分布模式，最终采用连续相机运动与多样化静态视角相结合的混合方案。

✨ **主要贡献**：该混合策略通过显著减少虚假相关性并保持训练稳定性，使VLA模型能泛化到未见过的相机姿态和物体配置，且优于单纯增加静态视角的方法。实验表明，捷径学习与空间泛化困难是不同架构（如ACT、Diffusion、Pi0和Gr00t）的共性问题，而该策略对所有模型均有显著提升。
