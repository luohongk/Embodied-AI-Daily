# AllDayNav: Lifelong Navigation via Real-World Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.10927v1
- arXiv ID: 2606.10927

---

🔍 **核心问题**：动态环境中机器人需要从零散观测中形成持久场景理解，现有依赖显式地图或场景图的方法难以在非结构化场景中泛化。

🛠 **方法**：提出AllDayNav框架，通过强化学习将场景动态隐式编码到大模型参数中，并利用自进化多模态记忆（维护视觉关键帧、语义描述和时间上下文）自主生成开放词汇指令、图像目标和结构化奖励。

✨ **主要贡献**：在跨房间、跨回合和跨任务场景中，成功率接近100%，路径效率和鲁棒性显著优于基于地图、VLM和强化学习的基线方法，证明隐式记忆驱动的强化学习可作为显式地图的可靠替代方案。
