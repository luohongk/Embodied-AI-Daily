# Maximizing Parallel Execution of Series-Parallel Task Graphs for Safety-Critical Embedded Control

- arXiv: https://arxiv.org/abs/2606.31481v1
- arXiv ID: 2606.31481

---

🔍 **核心问题**：针对安全关键嵌入式控制程序中，因任务依赖结构存在可并行子任务而导致的串行执行瓶颈，研究如何最小化系列-并行任务图的批次执行总时间。

🛠 **方法**：将最大并行执行问题建模为加权团划分问题，提出基于拉格朗日迭代启发式算法（LIH），通过定价过滤构建可行候选批次池，结合拉格朗日定价引导列选择与修复过程，恢复合法团划分。

✨ **主要贡献**：LIH在91.25%的实例中匹配精确最优解，平均差距仅0.073%，运行时间平均18.19毫秒；在最大精确参考节点设置中，精确基线需数百秒而LIH低于50毫秒；并通过PLC梯形逻辑案例验证了端到端优化效果。
