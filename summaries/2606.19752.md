# Temporal Self-Imitation Learning

- arXiv: https://arxiv.org/abs/2606.19752v1
- arXiv ID: 2606.19752

---

🔍 **核心问题**：在长周期机器人操作任务中，基于奖励塑形的强化学习策略容易因低效交互而利用密集奖励，同时罕见的高效行为可能在训练中被遗忘。

🛠 **方法**：提出时间自模仿学习（TSIL）框架，从学习过程中挖掘时间上高效的成功轨迹，将其转化为可复用的监督信号，通过效率加权自模仿学习和配置条件自适应时间目标来逐步优化策略。

✨ **主要贡献**：在15个长周期操作任务中，TSIL一致提升了学习效率、任务完成效率、高效行为的重访率以及对不稳定训练的鲁棒性，表明成功行为的时间结构本身可作为超越人工奖励塑形的可扩展自监督信号。
