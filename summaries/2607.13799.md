# Vision-Based Obstacle Separation for Strawberry Harvesting in Clusters Using Hierarchical Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.13799v1
- arXiv ID: 2607.13799

---

🔍 **核心问题**：在簇状草莓环境中，成熟果实常被未成熟果实遮挡，导致直接抓取不可靠，需要解决障碍分离与选择性采摘的难题。

🛠 **方法**：提出分层强化学习框架VGPA，包含视觉引导决策机制和渐进自适应探索策略（PAES），将任务分解为障碍分离与目标抓取两个阶段，高层优化选项选择，低层提升连续控制中的探索效率。

✨ **主要贡献**：仿真实验成功率达96.7%，在自研并联机器人上的虚实迁移实验中成功率为71.7%-88.3%，优于直接采摘，平均采摘时间仅增加1.22秒，验证了该方法在复杂簇状环境中的有效性、泛化能力和实用潜力。
