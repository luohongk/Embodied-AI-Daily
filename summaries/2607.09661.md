# PanoWorld: Real-World Panoramic Generation

- arXiv: https://arxiv.org/abs/2607.09661v1
- arXiv ID: 2607.09661

---

🔍 **核心问题**：解决全景世界模型中长程记忆的挑战，尤其是在大规模空间变化和多样光照条件下保持物理一致性。

🛠 **方法**：提出PanoWorld，利用全景表示的旋转等变性，通过密集全景射线条件（DPRC）和几何感知记忆增强（GMA）简化相机轨迹，并采用三阶段训练流程逐步优化各组件。

✨ **主要贡献**：构建了包含真实无人机采集和高质量模拟数据的World360数据集，实验表明PanoWorld在物理一致性上大幅优于现有方法，相关模型、代码和数据集将开源。
