# Parallel OctoMapping: A Scalable Framework for Enhanced Path Planning in Autonomous Navigation

- arXiv: https://arxiv.org/abs/2603.22508v2
- arXiv ID: 2603.22508

---

🔍 **核心问题**：固定分辨率地图在复杂环境中会过度保守地表示障碍物，导致路径规划失败或生成次优路径。

🛠 **方法**：提出Parallel OctoMapping（POMP），一种基于OctoMap的高效建图技术，在保持固定栅格分辨率的同时优化自由空间表示，并支持多线程并行计算。

✨ **主要贡献**：首次在固定分辨率下精细化自由空间表示，兼容现有搜索规划器，显著提升杂乱环境中的路径规划成功率、缩短路径长度，并大幅提高计算效率。
