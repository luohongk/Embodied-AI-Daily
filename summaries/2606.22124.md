# Surgical Anatomy Recognition with Context Learning using Foundation Representations

- arXiv: https://arxiv.org/abs/2606.22124v1
- arXiv ID: 2606.22124

---

🔍 **核心问题**：解决微创手术中因标注数据有限和现有方法主要针对自然场景而导致的解剖结构识别不准确问题。

🛠 **方法**：构建了包含超过12万帧标注的ATLAS-120k数据集，并提出ATLAS模型，该模型利用基础模型嵌入和轻量级时间推理，结合手术类型、阶段等上下文信息实现实时视频语义分割。

✨ **主要贡献**：提供了大规模、多模态、高质量的手术视频语义分割数据集，并设计了兼顾时间一致性与实时性的解剖识别模型，为临床手术引导系统奠定基础。
