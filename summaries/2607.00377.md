# SAOT: Self-Supervised Continual Graph Learning with Structure-Aware Optimal Transport

- arXiv: https://arxiv.org/abs/2607.00377v1
- arXiv ID: 2607.00377

---

🔍 **核心问题**：现有自监督持续图学习方法仅关注单个节点或节点对的稳定性，忽略了全局节点间关系结构，导致跨任务时关系逐渐扭曲。

🛠 **方法**：提出结构感知最优传输（SAOT）框架，利用最优传输理论捕获全局节点对应关系，并结合跨任务知识蒸馏机制保留先前任务的结构知识。

✨ **主要贡献**：在四个基准数据集上超越现有方法，在Class-IL设置下，CoraFull-CL和Products-CL的平均准确率分别提升高达5%和15%。
