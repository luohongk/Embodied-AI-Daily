# LLM vs. SAST: A Technical Analysis on Detecting Coding Bugs of GPT4-Advanced Data Analysis

- arXiv: https://arxiv.org/abs/2506.15212v2
- arXiv ID: 2506.15212

---

🔍 **核心问题**：量化评估大型语言模型（LLM）与静态应用安全测试（SAST）工具在代码漏洞检测中的实际效果差异。

🛠 **方法**：在32个精心设计的常见编码缺陷场景上，对比GPT-4（高级数据分析模式）与两个SAST工具（SonarQube和Cloud Defence）的检测能力，使用逻辑或聚合SAST结果，并通过McNemar检验进行统计显著性分析。

✨ **主要贡献**：实验表明GPT-4正确检测了30/32个场景（93.75%），显著优于聚合SAST基线（11/32），并讨论了将LLM集成到安全开发流程中的操作约束与安全考量。
