# Simple Domain Generalization for Strong Pixel-Level Image Tampering Detection in Modern VLMs

- arXiv: https://arxiv.org/abs/2607.18230v1
- arXiv ID: 2607.18230

---

🔍 **核心问题**：解决现代视觉语言模型（如ChatGPT、Gemini等）生成的图像在像素级篡改检测中，跨模型和分布外场景下的鲁棒性不足问题。

🛠 **方法**：提出一个简单的域泛化训练框架，包含两个策略：一是平衡小批量采样方案，确保每个批次中篡改与真实图像比例均衡，防止优化偏向；二是晚期注入策略，先在大规模基础数据上训练检测器至稳定收敛，再引入少量新兴VLM分布的支持数据，提升适应性而不过拟合。

✨ **主要贡献**：在GPT-Images-2.0、Gemini-3.1等分布外VLM上，平均gIoU和cIoU分别相对提升26.1%和26.8%，大幅超越此前最先进的PIXAR方法，且代码已开源。
