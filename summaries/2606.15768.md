# LaWAM: Latent World Action Models for Efficient Dynamics-Aware Robot Policies

- arXiv: https://arxiv.org/abs/2606.15768v1
- arXiv ID: 2606.15768

---

🔍 **核心问题**：现有视觉-语言-动作模型（VLA）缺乏对动作如何改变场景的显式预测能力，而基于像素的世界动作模型（WAM）计算成本高、存在像素级冗余。

🛠 **方法**：提出LaWAM，通过潜在世界模型（LaWM）在预训练视觉基础模型的潜在空间中预测未来观测特征，并以此作为潜在视觉子目标来条件化动作生成，实现动力学感知的机器人控制。

✨ **主要贡献**：在LIBERO（98.6%成功率）、RoboTwin（91.22%成功率）及真实世界操作任务上达到最优或竞争性能，同时推理延迟仅187毫秒/动作块，比像素级WAM降低最多24倍延迟。
