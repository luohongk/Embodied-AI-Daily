# Isaac Sim-to-Real: Reinforcement Learning based Locomotion for Quadrupeds

- arXiv: https://arxiv.org/abs/2607.18135v1
- arXiv ID: 2607.18135

---

🔍 **核心问题**：解决四足机器人强化学习策略从仿真环境迁移到真实机器人时存在的“仿真到现实”差距问题，实现零样本迁移。

🛠 **方法**：基于NVIDIA Isaac Sim仿真平台和Isaac Lab强化学习框架，训练具备全身控制能力的四足机器人运动策略，并在Unitree Go1机器人上直接部署验证。

✨ **主要贡献**：实现了零样本的仿真到现实迁移，在真实机器人上达到与内置控制器相当的线速度（2.0 m/s）和角速度（1.8 rad/s）跟踪性能，且具备更强的抗扰动恢复能力。
