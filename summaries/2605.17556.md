# Visual Sculpting: Visually-Aligned Planning Representations for Long-Horizon Robot Clay Sculpting

- arXiv: https://arxiv.org/abs/2605.17556v1
- arXiv ID: 2605.17556

---

🔍 **核心问题**：解决机器人进行长时程黏土雕塑任务中，如何实现视觉对齐的变形材料动力学建模与规划，以匹配目标形状并保留纹理等视觉特征。  
🛠 **方法**：提出一种视觉对齐的表示方法，通过参数化单端执行器推压动作，结合动力学模型预测变形，并利用视觉规划实现形状到形状的匹配。  
✨ **主要贡献**：首次将视觉对齐表示用于机器人黏土雕塑，在多种材料与工具下达到与现有技术相当的动力学建模性能，并支持超过100步的长时程规划，同时分析了视觉表示相比3D表示在规划中的挑战。
