# Accelerating Signal-Temporal-Logic-Based Task and Motion Planning of Bipedal Navigation using Benders Decomposition

- arXiv: https://arxiv.org/abs/2508.13407v2
- arXiv ID: 2508.13407

---

🔍 **核心问题**：在双足机器人导航中，基于信号时序逻辑（STL）的任务与运动规划问题因包含非凸约束（如运动学可达性和脚步旋转）而计算复杂度极高，传统混合整数规划方法难以高效求解。

🛠 **方法**：采用Benders分解技术，将原问题分解为一个主问题（用于生成满足任务规范的规划方案）和一系列子问题（用于验证运动学与动力学可行性），通过迭代切割平面方法逐步优化。

✨ **主要贡献**：实验表明，该方法在处理非线性约束的优化问题时，相比传统算法能显著加快规划速度，有效解决了双足机器人导航中混合整数规划的计算瓶颈。
