# E3DGS: Unified Geometric-Photometric Equivariance for 3D Gaussian Splatting via Color-as-Geometry Embedding

- arXiv: https://arxiv.org/abs/2607.15536v1
- arXiv ID: 2607.15536

---

🔍 **核心问题**：现有3D高斯泼溅方法中，颜色被视为信号而非几何实体，导致在相机坐标系变化时，几何与外观的对称性难以统一，且球谐系数在旋转下处理不一致，破坏了等变性。

🛠 **方法**：提出基于表示论的统一矩阵嵌入，将球谐系数（ℓ≤2）映射为3×3几何张量，使所有高斯属性在统一的载体空间𝔤𝔩(3)中处理，并构建了无需Clebsch-Gordan张量积的SE(3)等变架构E3DGS。

✨ **主要贡献**：首次实现颜色作为几何实体的统一等变建模，在物体视觉和动作条件高斯世界建模任务中，显著提升了相机坐标系变化下的鲁棒性和数据效率。
