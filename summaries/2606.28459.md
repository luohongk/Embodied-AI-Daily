# scKDGM: KAN-guided Dynamic Graph Masked Learning for Single-Cell RNA-seq Clustering

- arXiv: https://arxiv.org/abs/2606.28459v1
- arXiv ID: 2606.28459

---

🔍 **核心问题**：单细胞RNA测序数据存在高维度、稀疏性、缺失值和技术噪声，导致细胞表达表示和图构建困难，现有方法未能将表达恢复与图优化有效结合。

🛠 **方法**：提出scKDGM框架，包含图感知分布保持基因掩码（GDP-Mask）扰动细胞身份、基于KAN的TAKGCN编码器学习掩码视图、掩码引导的表达恢复构建动态图，以及跨视图对比学习将恢复信号传递到拓扑更新，并用ZINB损失处理过离散和零膨胀。

✨ **主要贡献**：在12个真实scRNA-seq数据集上，scKDGM在平均NMI和ARI指标上优于10个基线方法，首次将KAN网络引入单细胞聚类，并通过动态图机制实现表达恢复与图优化的协同学习。
