# Online Segment 3D Gaussians via Launching Virtual Drones

- arXiv: https://arxiv.org/abs/2607.01628v1
- arXiv ID: 2607.01628

---

🔍 **核心问题**：现有3D高斯分割方法在交互前需要耗时数十秒甚至数分钟的场景预配置（如多视角掩膜准备、掩膜提升和特征蒸馏），严重阻碍了在线实时应用。

🛠 **方法**：提出SAGO框架，通过引入虚拟无人机，将3D分割问题转化为马尔可夫过程中的在线“下一最佳视角”规划任务，无需任何预配置即可直接交互。

✨ **主要贡献**：首次实现无预配置的亚秒级3D高斯交互分割（<1秒），相比现有无预配置方法提速超50倍，并支持物体操控和场景编辑等下游应用。
