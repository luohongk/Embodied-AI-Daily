# GARAGE: Characterizing the Automation Boundary in LLM-based Attack Graph Generation

- arXiv: https://arxiv.org/abs/2607.18108v1
- arXiv ID: 2607.18108

---

🔍 **核心问题**：当前自动化工具难以处理非结构化数据及汽车领域特有的架构细节，导致车辆安全威胁情报（CTI）合成效率低下。

🛠 **方法**：提出GARAGE框架，基于检索增强生成（RAG）技术，将碎片化CTI转化为符合STIX 2.1和Auto-ISAC ATM标准的知识库，并通过细粒度杀伤链分析实现战术模式级威胁生成。

✨ **主要贡献**：构建了包含12,786个CVE和140份事故报告的知识库；通过320次留一实验验证框架能准确将安全知识迁移至全新车辆架构；作为可扩展的TARA支持工具，提供成本-性能分析以指导不同LLM层级的部署。
