# Probing LLMs for Syntactic Structure Beyond Universal Dependencies: A Minimalist Phase Account in English

- arXiv: https://arxiv.org/abs/2605.26431v3
- arXiv ID: 2605.26431

---

🔍 **核心问题**：探究大型语言模型（LLMs）是否编码了超越通用依存关系（UD）树距离的句法结构，特别是英语wh-疑问句中有限与不定从句的句法差异。

🛠 **方法**：使用结构探针测量嵌入主语与动词之间的探针距离，结合因果干预（如替换矩阵动词）和跨模型（13个模型）分层分析，检验最小主义阶段理论（Minimalist phase theory）的预测。

✨ **主要贡献**：发现所有模型在多数层中均表现出有限从句内探针距离更短、不定从句更长的符号反转现象，该现象无法用UD距离、线性顺序或单调结构复杂性解释，而最小主义阶段理论能一致解释；因果干预排除了词汇语义、从句长度和表面线索的替代解释，揭示了LLMs中超越UD目标的深层句法结构。
