## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人学中的接触式操作（Contact-Rich Manipulation），特别是高精度装配任务。结合了强化学习（RL）、人类参与循环（Human-in-the-Loop）和知识蒸馏（Knowledge Distillation）。
- **问题（Problem）**：基于视觉的策略在接触式操作中容易过拟合于训练时的视觉环境（如光照、背景、物体纹理），导致在视觉条件变化或面对未见过的任务时鲁棒性和泛化能力差。而仅依赖本体感觉（如位置、力/力矩）的策略虽然鲁棒，但难以从零开始有效学习。
- **方法（Method）**：提出一个名为 **VE2VF** 的两阶段框架。首先，利用人类参与循环的强化学习（HIL-RL）训练一个**视觉-本体感觉融合的教师策略**，利用视觉信息加速学习。然后，通过知识蒸馏，将教师策略的知识转移到一个**仅依赖本体感觉（位置、速度、力/力矩）的学生策略**中，该策略在部署时对视觉变化具有不变性。
- **创新（Innovation）**：核心创新在于**跨模态蒸馏**。与传统的“特权教师”蒸馏不同（通常会导致性能下降），本文的动机是视觉在训练时是有效的“加速器”，但在部署时却可能成为“干扰项”。蒸馏过程旨在**抛弃不可靠的视觉模态，同时保留其带来的探索优势**，从而在保持样本效率的同时，获得一个鲁棒且泛化的纯本体感觉策略。
- **意义（Significance）**：该方法在真实世界的 NIST 装配基准板上，仅需约 50 分钟的训练时间，即在 3 个代表性任务上达到 95% 的成功率，并能零样本泛化到 8 个未见过的任务变体。这证明了在真实世界中高效学习鲁棒、可泛化的接触式操作策略的可行性，对工业自动化等领域具有重要价值。
- **一句话总结**：本文提出了一种通过真实世界人类参与循环强化学习进行跨模态蒸馏的方法，将视觉辅助的教师策略转化为鲁棒的、无视觉的学生策略，实现了对接触式操作任务的高效学习和强泛化能力。

## 1. 研究动机（Motivation）

- **核心问题**：在接触式机器人操作（如装配）中，基于视觉的强化学习策略虽然能利用丰富的感知信息加速学习，但**严重依赖于训练时的视觉环境**。一旦环境发生变化（如光照、背景、物体纹理改变），策略性能会急剧下降，即存在**视觉过拟合**问题。
- **为什么重要**：在真实世界的工业场景中，环境变化是常态。一个无法适应新环境的策略是不实用的。同时，高精度接触式操作任务（如装配）对鲁棒性和适应性要求极高，微小的偏差就可能导致失败。
- **挑战**：
    1.  **视觉过拟合**：视觉策略将任务相关的几何关系与不相关的视觉外观特征（如颜色、纹理）混淆。
    2.  **纯本体感觉策略学习困难**：仅依赖位置、力/力矩等低维信号进行学习，由于部分可观测性和缺乏探索引导，从零开始训练非常困难且耗时（已有工作报道需要3小时）。
    3.  **模拟到现实的鸿沟**：在仿真中训练的策略难以直接迁移到真实世界，尤其是接触动力学建模极其困难。
    4.  **样本效率**：在真实机器人上直接进行无模型强化学习需要大量的交互时间，且存在安全风险。

## 2. 创新点（Innovation）

- **跨模态蒸馏框架（Cross-modal Distillation Framework）**：这是本文最核心的创新。与传统的“特权信息蒸馏”（如从拥有完美状态信息的教师蒸馏到仅有噪声观测的学生）不同，本文的蒸馏是从**视觉-本体感觉模态**到**纯本体感觉模态**。其动机并非为了压缩模型或弥补信息缺失，而是为了**在训练阶段利用视觉的探索优势，在部署阶段抛弃视觉的干扰**。这是一种“取其精华，去其糟粕”的策略。
- **高效的技能泛化（Efficient Skill Generalization）**：通过在少量（3个）具有代表性的任务上联合训练，蒸馏出的学生策略能够零样本泛化到8个未见过的任务变体（包括不同几何形状的插头/插座）。对于更具挑战性的任务，通过**带蒸馏的微调**（fine-tuning with distillation）可以在几分钟内达到100%的成功率。这展示了强大的任务泛化能力。
- **真实世界验证（Real-world Validation）**：整个训练和评估流程完全在真实机器人上完成，无需仿真或域随机化。在约50分钟的机器人交互时间内，完成了从教师训练到学生蒸馏的全过程，并在标准化的NIST装配基准板上进行了全面的实验验证，结果优于多种基线方法。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

VE2VF 框架包含三个主要阶段，如图 1 和图 2 所示：

1.  **阶段 I：训练视觉-本体感觉教师策略**
    - 使用人类参与循环的强化学习（HIL-RL）算法。
    - 输入：多模态观测 $o_t = \{I_t, x_t, z\}$，包括来自两个摄像头的图像 $I_t$，以及经过变换的本体感觉状态 $x_t$（包含相对于初始位姿的末端执行器位置、速度、力、力矩）。
    - 输出：末端执行器的位姿增量 $\Delta p_t$。
    - 目标：在选定的3个代表性训练任务上达到100%的成功率。这个教师策略具备视觉伺服能力，对控制误差和物理扰动具有鲁棒性，但对视觉环境敏感。

2.  **阶段 II：蒸馏视觉无关的学生策略**
    - 将阶段 I 训练好的教师策略 $\pi_t$ 的知识蒸馏到一个学生策略 $\pi_s$ 中。
    - 学生策略的输入：仅包含本体感觉状态 $o_s = x_s$，其中 $x_s$ 是相对于任务坐标系的位姿、速度、力、力矩。这确保了学生策略对视觉外观完全不变。
    - 蒸馏方法：在 HIL-RL 的训练损失函数中加入额外的蒸馏项。
        - **评论家（Critic）蒸馏**：最小化学生评论家 $Q_s$ 与教师评论家 $Q_t$ 之间的均方误差（MSE），让学生学会教师的“价值判断”。
        - **演员（Actor）蒸馏**：在学生演员的损失函数中加入 KL 散度项 $D_{KL}(\pi_t(a|o) \parallel \pi_s(a|o_s))$，鼓励学生策略的动作分布与教师策略的动作分布尽可能接近。

3.  **阶段 III（可选）：针对挑战性任务的微调**
    - 对于零样本泛化效果不佳的挑战性任务（如 DSUB 连接器），进行微调。
    - 方法：针对该新任务，重新训练一个专门的视觉教师策略 $\pi_{t'}$，然后使用与阶段 II 相同的蒸馏损失，对之前训练好的学生策略进行微调。这个过程非常高效，仅需几分钟。

### 3.2 关键算法与数学推导

论文使用 Soft Actor-Critic (SAC) 算法作为基础 RL 框架。

- **SAC 基础**：
    - **评论家更新**：最小化时序差分误差。
      $$L_Q(\phi) = \mathbb{E}_{(s,a,r,s')\sim D} \left[ (Q_\phi(s, a) - y)^2 \right]$$
      其中目标值 $y = r + \gamma \left( \min_{i=1,2} Q_{\bar{\phi}_i}(s', a') - \alpha \log \pi_\theta(a'|s') \right)$。
    - **演员更新**：最大化期望 Q 值并保持熵。
      $$L_\pi(\theta) = \mathbb{E}_{s \sim D} \left[ \mathbb{E}_{a \sim \pi_\theta(\cdot|s)} \left[ \alpha \log \pi_\theta(a|s) - Q_\phi(s, a) \right] \right]$$

- **教师策略训练（阶段 I）**：
    - 使用 HIL-RL 变体，从混合数据缓冲区 $D_{HIL}$（包含人类演示、干预和自主探索数据）中采样。
    - 评论家损失：$L^t_Q(\phi) = \mathbb{E}_{(o_t, a, r, o_t')\sim D^t_{HIL}} \left[ (Q^t_\phi(o_t, a) - y)^2 \right]$
    - 演员损失：$L^t_\pi(\theta) = \mathbb{E}_{o_t \sim D^t_{HIL}} \left[ \mathbb{E}_{a \sim \pi^t_\theta(\cdot|o_t)} \left[ \alpha \log \pi^t_\theta(a|o_t) - Q^t_\phi(o_t, a) \right] \right]$

- **学生策略训练（阶段 II，带蒸馏）**：
    - **评论家损失**：在标准 SAC 损失基础上，增加了与教师评论家的 MSE 项。
      $$L^s_Q(\phi) = \mathbb{E}_{(o, a, r, o')\sim D^s_{HIL}} \left[ (Q^s_\phi(o_s, a) - y)^2 + \left( Q^t(o_t, a) - Q^s(o_s, a) \right)^2 \right]$$
      其中 $y = r + \gamma \left( \min_{i=1,2} Q^s_{\bar{\phi}_i}(o_s', a') - \alpha \log \pi^s_\theta(a'|o_s') \right)$。
    - **演员损失**：在标准 SAC 损失基础上，增加了与教师策略动作分布的 KL 散度项。
      $$L^s_\pi(\theta) = \mathbb{E}_{o_s \sim D^s_{HIL}} \left[ \mathbb{E}_{a \sim \pi^t_\theta(\cdot|o_s)} \left[ \alpha \log \pi^s_\theta(a|o_s) - Q^s_\phi(o_s, a) + D_{KL}(\pi^t(a|o) \parallel \pi^s(a|o_s)) \right] \right]$$

### 3.3 模型、训练与数据集

- **模型架构**：
    - **视觉编码器**：预训练的 ResNet-10（冻结权重），处理两个 128x128 的 RGB 图像。
    - **本体感觉编码器**：一个全连接层（FC64）。
    - **演员/评论家网络**：均为 256:256 的全连接 MLP。
- **训练**：
    - **算法**：SAC 的 HIL-RL 变体（RLPD）。
    - **硬件**：Franka FR3 机器人，两个腕部 Intel RealSense D435i 摄像头，3D Space mouse 用于人类遥操作。
    - **时间**：总共约 50 分钟。其中教师训练 40 分钟，学生蒸馏 10 分钟。
    - **控制频率**：10Hz。
- **数据集**：
    - **训练数据**：在训练开始前，为每个任务提供 5 次人类演示。在训练过程中，人类操作员可以随时通过遥操作干预机器人，这些干预数据也被存入缓冲区。数据来自真实世界的机器人交互。
    - **任务**：基于 NIST Assembly Board I 基准。训练任务为 3 个（图 3b）：中号 peg、以太网接口、中号齿轮。测试任务包括 8 个（图 3c），涵盖训练任务变体和全新的几何形状。

### 3.4 实验设置与结果

- **实验设置**：在真实机器人上评估，分为三类任务：训练任务、受干扰任务（添加视觉干扰物和位姿噪声）、分布外任务（全新几何形状的连接器）。
- **基线方法**：
    1.  **HIL-SERL (VPTW)**：与教师策略相同，使用视觉+本体感觉。
    2.  **HIL-SERL (PTW)**：仅使用本体感觉，从零开始训练。
    3.  **DMP**：基于单一示教的动态运动基元，开环控制。
    4.  **DMP + Residual RL**：在 DMP 基础上，使用 RL 学习残差修正。
- **主要结果（表 II）**：
    - **VE2VF (PTW)** 以 **95.0%** 的总体成功率显著优于所有基线。
    - 视觉策略（VPTW）在训练任务上表现完美，但在受干扰和分布外任务上崩溃（总体 34.3%），证实了视觉过拟合。
    - 纯本体感觉策略（PTW）从零开始学习效果不佳（总体 47.1%）。
    - DMP + Residual RL 表现较好（总体 85.7%），但 VE2VF 在鲁棒性和泛化性上更胜一筹，尤其是在 USB 和 DSUB 等困难任务上。
- **消融实验（表 III）**：
    - 验证了**蒸馏的必要性**：纯本体感觉策略（PTW）即使训练更长时间（75分钟），性能也无法提升，反而会过拟合（图 6）。
    - 验证了**多模态本体感觉的重要性**：学生策略 `sPTW`（位姿+速度+力/力矩）优于仅使用位姿的 `sP` 或仅使用力/力矩的 `sTW`。
- **微调实验（表 IV）**：
    - 对于挑战性任务 DSUB，零样本泛化成功率仅 50%。
    - 使用**带蒸馏的微调**（10分钟）可将成功率提升至 **100%**。
    - 而不带蒸馏的微调（仅使用 RL）则完全失败（0%），再次证明蒸馏的关键作用。

### 3.5 技术细节

- **状态表示**：
    - 教师策略使用**相对于初始位姿**的变换 $T_{p_0}(p)$，这赋予了其视觉伺服行为。
    - 学生策略使用**相对于任务位姿**的变换 $T_z(p)$，这需要为每个任务提供一个估计的目标位姿。这种表示方式使得学生策略能够泛化到不同位置的任务。
- **奖励函数**：使用**稀疏奖励**（成功=1，失败=0），避免了复杂的奖励塑形，使其易于迁移到新任务。稀疏奖励与人类演示和干预相结合，提供了足够的学习信号。
- **人类参与**：人类操作员通过 3D Space mouse 提供初始演示和在线干预。干预发生时，机器人动作被覆盖，但干预数据被记录并用于训练，这极大地提高了样本效率并引导了探索。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了视觉策略鲁棒性差的问题**：通过蒸馏，成功地将视觉策略的“探索能力”和“任务知识”转移到了对视觉变化鲁棒的纯本体感觉策略上。
    2.  **提出了一种高效的现实世界学习方法**：整个流程在真实机器人上仅需约50分钟，远快于以往的工作（如3小时），证明了 HIL-RL 与蒸馏结合的巨大潜力。
    3.  **展示了强大的泛化能力**：在少量任务上训练后，能零样本泛化到多种未见过的任务，并通过快速微调解决最困难的任务，这对于高混合、低产量的工业场景至关重要。
- **对未来研究/应用的影响**：
    - **对机器人学习领域**：提供了一种新的范式，即“训练时用视觉，部署时不用视觉”，挑战了“传感器越多越好”的传统观念。为设计鲁棒的、可部署的机器人策略提供了新思路。
    - **对工业应用**：该方法有望应用于自动化装配、精密操作等场景，减少对昂贵视觉系统和复杂环境控制的依赖，提高生产线的灵活性和鲁棒性。
- **局限与开放问题**：
    - **任务位姿估计**：学生策略需要预先知道每个任务的目标位姿 $T_z(p)$，这在实际应用中可能需要额外的感知或标定步骤。
    - **任务范围**：目前仅验证了插入类任务。作者提到未来工作将扩展到拧螺丝、非抓取操作等更广泛的接触式技能。
    - **安全性**：作者提到未来可以加入无源性约束来保证交互过程中的能量安全交换。

## 5. 澄清与简化（Clarifications and Simplifications）

- **类比理解“跨模态蒸馏”**：可以把教师策略想象成一个经验丰富的**导游**，他既会看地图（视觉），也会感受脚下的路（力觉）。在训练新导游（学生策略）时，导游会带着他走一遍路线，并告诉他“这里要左转”、“那里要小心台阶”。训练结束后，新导游被派去带团，但他被要求**蒙上眼睛**，只靠脚下的感觉和记忆来完成任务。VE2VF 的蒸馏过程，就是让蒙上眼睛的新导游，学会像睁着眼睛的老导游一样走路的技巧。视觉在训练时提供了丰富的“教学信号”，但在实际工作中，依赖视觉反而会让他分心（比如被路边不同的商店招牌干扰），而专注于脚下的感觉（力/力矩）才是完成“走路”这个任务最可靠的方式。
- **为什么蒸馏比直接训练纯本体感觉策略更有效？**
    - **直接训练**：相当于让一个盲人从零开始在迷宫里摸索，他需要无数次撞墙才能找到出口，非常低效。
    - **蒸馏**：相当于先让一个视力正常的人（教师）在迷宫里走几遍，记住路线。然后，他蒙上眼睛（学生），由视力正常的人（教师）在旁边“口述”指导（通过蒸馏损失函数），告诉他“向左”、“向前”、“停”。这样，学生就能在极短的时间内学会在迷宫中行走，并且由于他最终不依赖视觉，所以即使迷宫里的灯光变化，他也能走对。蒸馏提供了强大的**学习先验**和**探索引导**。
