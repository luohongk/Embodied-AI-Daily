# GeoHAT: Geometry-Adaptive Hybrid Action Transformer for Mobile Manipulation

- arXiv: https://arxiv.org/abs/2606.13394v1
- arXiv ID: 2606.13394

---

🔍 **核心问题**：移动操作机器人需要协调移动底盘与机械臂，但现有方法在几何感知上缺乏密集空间结构，且将臂与底盘编码在同一动作向量中，忽略了它们不同的控制需求。

🛠 **方法**：提出GeoHAT，一种端到端的扩散框架，使用轻量傅里叶空间编码器将密集3D坐标转为几何令牌，并通过深度有效性调制的门控融合选择性注入视觉基础模型特征；同时采用混合全身动作解码器，将臂与底盘分解为独立子空间，通过稀疏交叉注意力与因果时序建模生成协调动作。

✨ **主要贡献**：在ManiSkill-HAB仿真基准上平均成功率达79.3%，超越最强基线23.7%；真实世界实验也一致优于所有基线。
