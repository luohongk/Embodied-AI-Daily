# Privacy-Preserving Depth-Only Open-Vocabulary 3D Semantic Segmentation Via Uncertainty-Guided Test-Time Optimization

- arXiv: https://arxiv.org/abs/2607.00978v1
- arXiv ID: 2607.00978

---

🔍 **核心问题**：在保护隐私的前提下，如何仅利用深度信息实现开放词汇的3D语义分割，避免RGB图像泄露敏感视觉信息。

🛠 **方法**：提出UTTO框架，将深度预测中的不确定性转化为引导信号，识别不可靠的语义响应，并利用基础模型的语义先验进行测试时优化，无需额外训练。

✨ **主要贡献**：在ScanNet20、ScanNet40和ScanNet200数据集上，UTTO一致提升了仅深度输入的开放词汇3D分割性能，并在隐私保护条件下优于代表性基线方法。
