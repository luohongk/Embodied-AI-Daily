# KEPLA: A Knowledge-Enhanced Deep Learning Framework for Accurate Protein-Ligand Binding Affinity Prediction

- arXiv: https://arxiv.org/abs/2506.13196v5
- arXiv ID: 2506.13196

---

🔍 **核心问题**：现有深度学习模型预测蛋白质-配体结合亲和力时，仅依赖结构特征，忽略了基因本体和配体属性等生化知识，导致预测精度受限。

🛠 **方法**：提出KEPLA框架，以蛋白质序列和配体分子图为输入，通过知识图谱关系对齐全局表示以整合领域知识，并利用局部表示间的交叉注意力构建细粒度联合嵌入，同时优化两个互补目标。

✨ **主要贡献**：在域内和跨域基准数据集上均优于现有方法，且通过知识图谱关系和交叉注意力图的可解释性分析，揭示了预测机制的内在逻辑。
