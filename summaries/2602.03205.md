# HUSKY: Humanoid Skateboarding System via Physics-Aware Whole-Body Control

- arXiv: https://arxiv.org/abs/2602.03205v2
- arXiv ID: 2602.03205

---

🔍 **核心问题**：解决人形机器人在高动态、复杂交互任务（如滑板）中的稳定控制难题，尤其是非完整约束下的人-物耦合动态平衡问题。  
🛠 **方法**：提出HUSKY框架，通过物理感知全身控制建模滑板倾斜与转向的耦合关系，结合对抗运动先验（AMP）学习类人蹬地动作，并采用物理引导的航向策略实现倾斜转向，辅以轨迹引导机制平滑切换蹬地与转向。  
✨ **主要贡献**：在Unitree G1人形机器人上实现了真实场景中的稳定滑板操控，验证了框架对高动态、欠驱动平台控制的可行性，并开源项目页面。
