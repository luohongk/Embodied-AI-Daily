# Slot-RAE: Streamlining Object-Centric Learning via Direct Representation Auto-Encoders

- arXiv: https://arxiv.org/abs/2607.11196v1
- arXiv ID: 2607.11196

---

🔍 **核心问题**：现有基于扩散的对象中心模型依赖复杂的预训练生成先验（如Stable Diffusion）和外部VAE潜空间，导致计算开销大且流程繁琐，难以实现高效场景分解与高质量生成。

🛠 **方法**：提出Slot-RAE框架，直接在冻结的视觉基础模型（如DINOv3）的连续语义特征空间中运行，使用Slot注意力机制和从头训练的扩散Transformer（DiT）解码器，并引入表示对齐（REPA）头进行特征空间扩散。

✨ **主要贡献**：无需VAE瓶颈和任务无关的生成预训练，在COCO数据集上实现与现有方法相当的无监督对象发现、更高保真度的图像重建和鲁棒的零样本组合能力，同时显著提升计算效率和速度。
