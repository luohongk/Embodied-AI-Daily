# Disentangling Pictorial Cue Understanding from Language Bias in VLMs via Depth Ordering Task

- arXiv: https://arxiv.org/abs/2607.01503v1
- arXiv ID: 2607.01503

---

🔍 **核心问题**：研究视觉语言模型（VLM）在深度感知任务中如何分离图像线索与语言偏差的影响，并评估其对深度顺序的理解能力。

🛠 **方法**：结合深度排序与“找出不同”心理物理学任务，通过控制图像中单个深度线索（如遮挡、阴影）的呈现，并变化语言描述的清晰度，构建了包含3.7万张图像和14.7万个问答对的O3-D数据集。

✨ **主要贡献**：发现所有12个测试模型的深度排序准确率仅47%-56%（低于随机水平），且存在显著语言偏差；链式思维和上下文学习无法提升性能，表明静态图像数据可能不足以让模型理解深度。
