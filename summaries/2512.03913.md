# Hierarchical Vision Language Action Model Using Success and Failure Demonstrations

- arXiv: https://arxiv.org/abs/2512.03913v1
- arXiv ID: 2512.03913

---

🔍 **核心问题**：现有视觉-语言-动作模型仅使用成功演示数据训练，忽略了数据收集过程中大量自然产生的失败样本，导致策略鲁棒性不足。

🛠 **方法**：提出VINE分层模型，将高层推理（System 2）与低层控制（System 1）分离。System 2基于场景图抽象进行可行性引导的树搜索，利用成功和失败数据预测子目标成功率并剪枝脆弱分支；System 1执行选定的子目标序列。

✨ **主要贡献**：首次将失败数据作为结构化学习信号融入VLA决策循环，在复杂操作任务中显著提升成功率和鲁棒性，证明失败数据是增强模型执行能力的关键资源。
