# Cutting the Cord: System Architecture for Low-Cost, GPU-Accelerated Bimanual Mobile Manipulation

- arXiv: https://arxiv.org/abs/2603.09051v2
- arXiv ID: 2603.09051

---

🔍 **核心问题**：现有双臂移动机器人平台成本高昂且依赖外部计算设备，限制了其在科研与教育中的普及。

🛠 **方法**：基于开源XLeRobot平台，设计低成本双臂移动机器人，采用优化机械结构、Tri-Bus电源拓扑隔离电机干扰，并集成NVIDIA Jetson Orin Nano实现机载自主计算。

✨ **主要贡献**：构建了总成本低于1300美元的无缆化双臂移动机器人，支持遥操作、自主SLAM导航与视觉操控，为机器人学习研究提供低成本替代方案。
