# LLMCFG-TGen: Using LLM-Generated Control Flow Graphs to Automatically Create Test Cases from Use Cases

- arXiv: https://arxiv.org/abs/2512.06401v2
- arXiv ID: 2512.06401

---

🔍 **核心问题**：现有基于大语言模型的测试用例生成方法难以保证全面且无冗余的覆盖，尤其无法充分捕捉复杂条件逻辑，导致测试用例不完整。

🛠 **方法**：提出LLMCFG-TGen方法，分三步：先用LLM将自然语言用例转换为JSON格式的控制流图（CFG），再遍历CFG提取执行路径，最后从路径生成测试用例。

✨ **主要贡献**：在六个跨领域数据集上验证，相比基线方法，生成的测试用例更完整、结构一致，能更好捕捉行为逻辑与执行流，经LLM和从业者评估，提升了全面性与逻辑连贯性，同时减少人工工作量。
