# Posterior Sampling Reinforcement Learning with Gaussian Processes for Continuous Control: Sublinear Regret Bounds for Unbounded State Spaces

- arXiv: https://arxiv.org/abs/2603.08287v2
- arXiv ID: 2603.08287

---

🔍 **核心问题**：针对连续控制问题中状态空间无界的情况，现有高斯过程后验采样强化学习（GP-PSRL）算法的理论遗憾界存在增长率次优、平滑性假设过强或未正确处理无界状态空间的问题。  
🛠 **方法**：通过递归应用Borell-Tsirelson-Ibragimov-Sudakov不等式，证明算法访问的状态高概率位于近恒定半径的球内，并利用链式方法在弱平滑条件下控制遗憾。  
✨ **主要贡献**：首次为GP-PSRL在无界状态空间下建立了$\widetilde{\mathcal{O}}(H\sqrt{γ_TT})$量级的贝叶斯遗憾界，解决了先前理论工作的局限性，为复杂场景下的PSRL分析提供了理论基础和工具。
