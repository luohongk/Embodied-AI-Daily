# Open-Weather Robust 3D Detection via Dual-Critic Diffusion Alignment

- arXiv: https://arxiv.org/abs/2607.01983v1
- arXiv ID: 2607.01983

---

🔍 **核心问题**：解决自动驾驶中恶劣天气下3D目标检测的鲁棒性问题，现有方法受限于封闭世界假设，无法泛化到未见过的天气类型和严重程度。

🛠 **方法**：提出双判别器引导扩散对齐框架（DCDA），利用4D雷达条件扩散过程逐步恢复退化LiDAR特征，并通过检测引导判别器和天气对抗判别器分别确保目标级可判别性和分布一致性。

✨ **主要贡献**：无需配对数据或天气标签即可泛化到未知天气类型和严重程度，并构建了结构化开放天气基准，实验验证了DCDA的显著优势。
