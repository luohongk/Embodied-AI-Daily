# AEGIR: Modeling Area Emitters for Indoor Inverse Rendering using Gaussian Splatting

- arXiv: https://arxiv.org/abs/2606.28635v2
- arXiv ID: 2606.28635

---

🔍 **核心问题**：现有可重光照的高斯泼溅方法将光源简化为点光源或环境贴图，忽略了真实光源的物理空间范围，导致光照衰减错误和阴影不真实。

🛠 **方法**：提出AEGIR框架，在可重光照的高斯泼溅表示中显式建模局部面光源，并通过可微延迟渲染管线结合多重重要性采样与针对性正则化，联合优化光源、材质和几何。

✨ **主要贡献**：首次在可重光照高斯泼溅中引入显式面光源，准确模拟局部光传输，实现更一致的光照与材质分解，在复杂局部光照场景中显著提升新视角合成、可控重光照和虚拟物体插入等任务效果。
