# VisionAId: An Offline-First Multimodal Android Assistant for People with Visual Impairment, Featuring Personalized Object Retrieval

- arXiv: https://arxiv.org/abs/2607.02371v1
- arXiv ID: 2607.02371

---

🔍 **核心问题**：全球超2.85亿视障人士在日常生活中面临障碍物避让、个人物品寻找、人脸识别和现金处理等自主性难题，现有辅助应用多依赖云端、识别类别固定或需专用硬件。

🛠 **方法**：开发了一款名为VisionAId的Android应用，集成六种设备端深度学习模型（单目深度估计、实例分割、视觉/面部嵌入、人脸检测及自定义纸币检测器），通过ONNX Runtime本地运行；仅场景描述和自动标签标注可选云端大语言模型（Google Gemini Flash）。创新性地提出个人物品少样本流水线：用户多角度拍照后，系统通过增强现实标记、空间音频和距离比例触觉反馈引导用户定位物品。

✨ **主要贡献**：实现全离线多模态实时视觉辅助，INT8量化将深度估计延迟从~1200ms降至~491ms；自定义纸币检测器mAP@50达0.986；3米内度量深度误差低于1厘米。
