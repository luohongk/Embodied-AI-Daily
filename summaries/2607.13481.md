# GPOcc++: Unified Sparse Gaussian Occupancy Prediction with Visual Geometry Priors

- arXiv: https://arxiv.org/abs/2607.13481v1
- arXiv ID: 2607.13481

---

🔍 **核心问题**：从视觉观测中恢复完整的3D场景体积表示（包括物体、结构和自由空间）仍具挑战，尤其在遮挡和未观测区域，现有视觉几何先验仅提供表面信息，无法直接用于体积占用预测。

🛠 **方法**：提出GPOcc++，将视觉几何先验转化为占用感知的稀疏高斯表示，并在统一框架中建模多视角观测和时间序列，支持室内外场景的占用预测。

✨ **主要贡献**：通过稀疏高斯表示桥接表面几何与体积占用，实现高效且表达力强的场景建模；在室内外基准上均取得优异性能，并展现出良好的效率与泛化能力。
