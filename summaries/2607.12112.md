# Continual Learning with Elastic Regularization and Synthetic Replay for Federated MLLM Fine-Tuning

- arXiv: https://arxiv.org/abs/2607.12112v1
- arXiv ID: 2607.12112

---

🔍 **核心问题**：联邦微调多模态大模型时，顺序任务更新会导致灾难性遗忘，即模型在视觉、语言和跨模态表示上丢失先前学到的知识。

🛠 **方法**：提出FedCMM框架，在参数层使用模态感知弹性权重合并（为不同模态计算独立Fisher矩阵），在数据层训练轻量生成模块合成无原始数据的多模态回放元组，在聚合层通过梯度余弦相似度自主过滤和重加权客户端更新。

✨ **主要贡献**：在多个基准上持续优于现有方法，验证了模态感知的全局优化能实现异构网络AI的鲁棒演化适应，有效提升准确率和反向迁移性能。
