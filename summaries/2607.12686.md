# Segregate, Refine, Integrate: Decomposing Multimodal Fusion for Sentiment Analysis

- arXiv: https://arxiv.org/abs/2607.12686v1
- arXiv ID: 2607.12686

---

🔍 **核心问题**：多模态情感分析中，模态特定信号的精炼与跨模态交互建模这两个目标通常在同一操作中纠缠，导致性能受限。

🛠 **方法**：提出SeRIn框架，通过架构设计将模态特定表示沿独立路径演化并分别精炼，同时用专用跨模态路径累积联合演化，最终在预测步骤才进行完整跨模态交互。

✨ **主要贡献**：在CH-SIMS和CMU-MOSEI基准上实现所有指标的最优结果；消融实验证明结构化交互（而非参数增加）是性能提升关键；门控分析显示模型能在无显式监督下自动重加权受损模态。
