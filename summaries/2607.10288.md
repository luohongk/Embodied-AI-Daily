# PIER-Flow: Physics-Informed Efficient Rectified Flow for Real-Time Mobile Robot Navigation

- arXiv: https://arxiv.org/abs/2607.10288v1
- arXiv ID: 2607.10288

---

🔍 **核心问题**：在密集动态环境中，移动机器人导航需要兼顾物理可行控制与低延迟重规划，但现有方法（如MPC、行为克隆或扩散策略）在实时性与多模态避障能力之间存在权衡。

🛠 **方法**：提出PIER-Flow，通过将MPC专家策略蒸馏为连续时间常微分方程（ODE），结合并行潜在采样与轻量可行性选择实现单步动作生成；并引入物理信息训练目标保证运动学一致性，以及异步动作分块架构支持仿真到真实部署。

✨ **主要贡献**：在仿真中实现98.85%成功率与零碰撞，平均推理仅1.29毫秒，比MPC加速37.2倍，比标准扩散模型加速超800倍；在资源受限的边缘计算机上实际部署仍保持约5.3毫秒稳定推理延迟，避免基线方法的延迟尖峰与冻结问题。
