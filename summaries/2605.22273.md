# Exposing Vulnerabilities in Visible-Infrared VLMs: A Unified Geometric Adversarial Framework with Cross-Task Transferability

- arXiv: https://arxiv.org/abs/2605.22273v1
- arXiv ID: 2605.22273

---

🔍 **核心问题**：现有视觉语言模型（VLM）在可见光-红外（VIS-IR）跨模态场景下的对抗鲁棒性尚未被充分研究，而这类感知系统在真实世界中至关重要。

🛠 **方法**：提出CFGPatch框架，基于贝塞尔曲线改进三角分形几何生成对抗补丁，并设计模态特定的Fraser螺旋渲染机制注入纹理干扰，同时采用期望变换（EOT）增强鲁棒性。

✨ **主要贡献**：CFGPatch能有效欺骗VIS-IR VLM，在攻击效果和鲁棒性上优于标准补丁方法，且对抗样本在零样本分类、图像描述和视觉问答等下游任务中展现出强跨任务迁移性。
