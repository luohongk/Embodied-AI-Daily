# Hallucination Detection and Correction in Medical VLMs via Counter-Evidence Verification

- arXiv: https://arxiv.org/abs/2606.18609v1
- arXiv ID: 2606.18609

---

🔍 **核心问题**：医学视觉语言模型（VLMs）在诊断中常产生幻觉，现有检测方法未能验证模型关注的视觉区域是否真正支持生成的文本内容。  
🛠 **方法**：提出CoEV框架，通过文本断言与视觉证据的双向验证，将每个陈述映射到四象限诊断图（结合文本事实性与视觉基础），无需训练即可检测并修正幻觉。  
✨ **主要贡献**：在四个医学数据集上，CoEV在幻觉检测中PR-AUC和ROC-AUC分别提升3.0%和3.9%（特定场景最高达18.5%），在报告生成中降低幻觉率超11.9%，并提升VQA准确率，提供可靠的临床诊断依据。
