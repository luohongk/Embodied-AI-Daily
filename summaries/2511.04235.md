# Shared Spatial Memory Through Predictive Coding

- arXiv: https://arxiv.org/abs/2511.04235v4
- arXiv ID: 2511.04235

---

🔍 **核心问题**：多智能体系统中，部分可观测性和有限带宽导致协调失败，需要构建一致的共享空间记忆。

🛠 **方法**：提出多智能体预测编码框架，通过信息瓶颈目标最小化智能体间的互不确定性，并利用自监督运动预测自发产生类网格细胞的空间编码，结合分层强化学习策略主动探索以减少联合不确定性。

✨ **主要贡献**：在Memory-Maze基准上，该方法在带宽从128降至4 bits/步时，成功率仅从73.5%降至64.4%，而全广播基线从67.6%骤降至28.6%；首次从统一预测驱动中自发涌现出类似海马社会位置细胞的社会表征，为集体智能提供了理论原则和生物合理性基础。
