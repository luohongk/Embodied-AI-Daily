# ViTexQA: A Multi-Frame Temporal Perception Dataset for Video Text Question Answering

- arXiv: https://arxiv.org/abs/2606.24602v1
- arXiv ID: 2606.24602

---

🔍 **核心问题**：当前多模态大模型在视频文本理解中，难以处理需跨多帧融合时序文本线索才能回答的问题，而现有数据集大多仅依赖单帧即可回答，无法反映真实需求。

🛠 **方法**：提出ViTexQA数据集，通过质量受控的思维链标注流程并加入时序约束，确保所有问答对必须跨帧融合文本；同时设计FrameThinker模型，采用两阶段训练：先进行思维链引导的监督微调生成帧感知推理链，再使用时序强化学习优化多帧一致性。

✨ **主要贡献**：构建了首个强制依赖多帧时序推理的视频文本问答数据集ViTexQA，并提出了FrameThinker方法，在ROUGE-L指标上比当前最优模型提升6.3%。
