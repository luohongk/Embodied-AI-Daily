# WARP-RM: A Warp-Augmented Relative Progress Reward Model for Data Curation

- arXiv: https://arxiv.org/abs/2606.28320v1
- arXiv ID: 2606.28320

---

🔍 **核心问题**：如何从人类遥操作产生的混合质量演示数据中自动筛选高质量样本，以提升模仿学习在大规模数据集上的鲁棒性。

🛠 **方法**：提出WARP-RM，一种全自监督的密集相对进度奖励模型，通过时间扭曲增强（变速播放和倒放）生成逐帧进度目标，并训练模型预测帧间归一化时间差；进一步利用WARP-BC在行为克隆中根据帧级奖励聚合的块级优势加权高价值动作块。

✨ **主要贡献**：在双臂机器人折叠T恤任务中，当训练数据包含大量低效演示时，WARP-BC保持19/20成功率，而标准行为克隆降至2/20，吞吐量提升高达18倍。
