# Learn to Memorize: Scalable Continual Learning in Semiparametric Models with Mixture-of-Neighbors Induction Memory

- arXiv: https://arxiv.org/abs/2303.01421v2
- arXiv ID: 2303.01421

---

🔍 **核心问题**：现有半参数语言模型中的非参数记忆（如kNN-LM）缺乏学习能力，与参数模型信息流脱节，限制了可扩展性和持续学习性能。  
🛠 **方法**：提出可学习的混合邻居归纳记忆（MoNIM），将注意力头的归纳能力与前馈网络的记忆能力结合，作为类似FFN的旁路层集成到Transformer架构中，使非参数记忆能够动态学习新知识。  
✨ **主要贡献**：MoNIM实现了数据维度和模型维度的可扩展持续学习，显著提升了半参数语言模型的记忆保持能力和持续学习性能。
