# Robustness-Based Synthesis for Time Window Temporal Logic Specifications via Mixed-Integer Linear Programming

- arXiv: https://arxiv.org/abs/2606.30820v1
- arXiv ID: 2606.30820

---

🔍 **核心问题**：针对离散时间线性系统，如何自动生成控制输入以严格满足带有显式时间约束的时序任务规范（TWTL），并确保鲁棒性。

🛠 **方法**：将TWTL公式的鲁棒满足性编码为混合整数线性约束，构建混合整数线性规划（MILP）最大化鲁棒度；提出开环优化与闭环模型预测控制（MPC）两种框架，其中MPC利用TWTL的确定性有限自动机（DFA）动态调整预测时域。

✨ **主要贡献**：首次将TWTL鲁棒性度量的优化问题转化为MILP求解，证明正鲁棒度可保证布尔满足性；提出任务自适应时域MPC方法，通过仅预测当前子任务剩余窗口显著降低计算成本。
