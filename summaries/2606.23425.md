# Interpretable Kolmogorov-Arnold Network with Feature-Isolated Temporal Attention Mechanism for Electricity Load Forecasting

- arXiv: https://arxiv.org/abs/2606.23425v1
- arXiv ID: 2606.23425

---

🔍 **核心问题**：现有深度学习电力负荷预测模型虽性能优异，但作为黑箱缺乏可解释性，而直接应用KAN处理时间序列数据时难以建模复杂时序模式。

🛠 **方法**：提出LoadKAN框架，将特征隔离时序注意力机制与KAN模块协同结合，先独立提取各输入特征（如历史负荷、人类流动性）的时序动态，再通过KAN进行可解释预测。

✨ **主要贡献**：在三个美国电力市场数据集上，LoadKAN性能与顶尖黑箱模型相当，且通过KAN可学习激活函数，首次量化揭示了六种流动性模式与电力负荷之间复杂的非线性关系，为电网运营提供透明洞察。
