# Duet: Dual-Robot Understanding via Efficient Teaching

- arXiv: https://arxiv.org/abs/2606.20990v1
- arXiv ID: 2606.20990

---

🔍 **核心问题**：双机器人协作系统面临数据采集效率低下的主要瓶颈，难以高效训练移动操作任务。  
🛠 **方法**：提出DUET框架，包含统一的双机器人VR遥操作系统用于高效数据采集，以及基于Action Chunking Transformer的架构，先利用人类协作演示预训练策略，再通过少量真实机器人轨迹微调。  
✨ **主要贡献**：通过引入人类协作先验，在四个协作任务中实现比纯机器人数据训练策略更优的性能，同时将数据采集人力成本降低5.4倍，且任务表现不逊于甚至优于基线方法。
