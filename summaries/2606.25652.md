# Auto-Labelling-Based Domain Transfer for 3D Object Detection on a Bicycle-Mounted LiDAR Platform

- arXiv: https://arxiv.org/abs/2606.25652v1
- arXiv ID: 2606.25652

---

🔍 **核心问题**：针对自行车搭载的LiDAR平台，缺乏从骑行者视角标注的3D数据，导致车辆训练的检测器难以直接应用于弱势道路使用者（如骑行者、行人）的感知场景。

🛠 **方法**：提出一种基于自动标注的域迁移方法，利用专为VRU设计的自动标注流水线生成训练标签，无需人工标注，并对预训练的nuScenes检测器进行微调。

✨ **主要贡献**：构建了包含1,027个LiDAR关键帧（超过18,000个3D边界框）的基准数据集，验证了自动标注可替代人工标注，微调后检测器在VRU类别上的平均精度提升高达23.4点，甚至超越自动标签本身的质量。
