# Gated Multimodal Learning for Interpretable Property Energy Performance Prediction and Retrofit Scenario Analysis

- arXiv: https://arxiv.org/abs/2605.05088v1
- arXiv ID: 2605.05088

---

🔍 **核心问题**：针对住宅建筑能耗评估依赖现场检查、难以大规模应用的问题，提出一种可解释的多模态方法，用于预测能效等级并支持改造方案分析。

🛠 **方法**：构建门控多模态模型，融合EPC表格变量、评估师自由文本和GIS空间特征，通过样本级门控学习各模态权重，并引入辅助分类头稳定训练。

✨ **主要贡献**：在伦敦威斯敏斯特案例中，模型预测SAP和EI分数的MAE为4.03和4.76，R²达0.757和0.748；消融实验证明多模态融合优于单/双模态基线；可解释性分析揭示文本、燃料类型、建筑形态等关键因素，并成功应用于墙体、屋顶和窗户改造场景的能耗与碳排放预测。
