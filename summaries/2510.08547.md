# R2RGEN: Real-to-Real 3D Data Generation for Spatially Generalized Manipulation

- arXiv: https://arxiv.org/abs/2510.08547v2
- arXiv ID: 2510.08547

---

🔍 **核心问题**：机器人操作的空间泛化能力不足，现有方法依赖大量人工演示覆盖不同空间配置，且存在仿真到现实的差距，难以在移动操作等复杂场景中应用。

🛠 **方法**：提出R2RGen框架，通过无仿真器、无渲染的“真实到真实”3D数据生成，直接增强点云观测-动作对；包含场景/轨迹解析、分组回溯增强和相机感知后处理三阶段，对齐生成数据与真实传感器分布。

✨ **主要贡献**：显著提升数据效率，在移动操作等任务中实现强泛化能力，且框架即插即用，无需仿真环境，具备规模化应用潜力。
