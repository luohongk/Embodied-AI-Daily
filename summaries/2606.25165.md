# The Gentle Collapse: Distributional Metrics for Continual Learning

- arXiv: https://arxiv.org/abs/2606.25165v1
- arXiv ID: 2606.25165

---

🔍 **核心问题**：传统灾难性遗忘评估指标（如准确率）在任务边界处离散下降，无法揭示遗忘的内部结构，且在高/低准确率区间趋于饱和。

🛠 **方法**：提出六种基于softmax的连续指标（包括真实标签排名、预测置信度、分布散度），每个指标归一化至[0,1]且不修改训练过程，并利用指标斜率进行小窗口采样。

✨ **主要贡献**：在CIFAR-100上，这些指标在准确率为0%时仍能区分不同类别的遗忘程度（混淆区间四分位距[0.32,0.50]）；作为损失权重时，遗忘率比均匀经验回放降低1.3个百分点；基于对数TLR斜率的小窗口采样在TinyImageNet上比基线降低7.7个百分点的遗忘率。
