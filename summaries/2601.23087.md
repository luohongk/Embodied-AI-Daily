# CoLA-Flow Policy: Temporally Coherent Imitation Learning via Continuous Latent Action Flow Matching for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2601.23087v5
- arXiv ID: 2601.23087

---

🔍 **核心问题**：现有生成式策略在长时程机器人操作任务中难以同时实现高表达能力、实时推理和稳定执行，扩散模型推理慢，而直接对原始动作空间进行流匹配则执行不稳定。

🛠 **方法**：提出CoLA-Flow Policy，在连续潜在动作空间中进行流匹配，将动作序列编码为时间连贯的潜在轨迹，并学习显式的潜在空间流，从而分离全局运动结构与低级控制噪声。

✨ **主要贡献**：实现近单步推理，轨迹平滑度提升最高93.7%，任务成功率比原始动作空间流基线提升最多25个百分点，且推理速度显著快于基于扩散的策略。
