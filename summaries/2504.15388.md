# Deep learning with missing data

- arXiv: https://arxiv.org/abs/2504.15388v3
- arXiv ID: 2504.15388

---

🔍 **核心问题**：解决多元非参数回归中协变量缺失时的预测问题，提升缺失数据场景下的模型性能。

🛠 **方法**：提出模式嵌入神经网络（PENNs），将观测指示向量通过第二个神经网络生成紧凑表示，与基于插补数据训练的主神经网络输出结合，再通过第三个网络进行最终预测。

✨ **主要贡献**：理论上证明PENN估计量在典型情况下可达到极小极大最优收敛速率（仅差多对数因子），实验表明该方法在模拟、半合成和真实数据上显著优于标准神经网络。
