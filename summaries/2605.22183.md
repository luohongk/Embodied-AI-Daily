# Action with Visual Primitives

- arXiv: https://arxiv.org/abs/2605.22183v2
- arXiv ID: 2605.22183

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型将指令理解、场景感知和运动控制耦合在单一目标中，导致动作专家需隐式学习预训练VLM已具备的认知能力，限制了学习效率和泛化性。

🛠 **方法**：提出AVP（Action with Visual Primitives）架构，让VLM推断下一阶段目标并生成视觉基元令牌，这些令牌作为条件输入流匹配动作专家，通过末端执行器运动学进行监督。

✨ **主要贡献**：在通用抓取放置任务中，AVP相比pi_0.5成功率提升27.61%，并优于其他最新方法，同时在数据效率、空间组合泛化和物体级迁移上表现一致提升。
