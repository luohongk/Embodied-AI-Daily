# Agent Journey Beyond RGB: Hierarchical Semantic-Spatial Representation Enrichment for Vision-and-Language Navigation

- arXiv: https://arxiv.org/abs/2412.06465v5
- arXiv ID: 2412.06465

---

🔍 **核心问题**：解决视觉语言导航中智能体因简单拼接RGB与辅助模态特征，未能充分利用各模态独特贡献，导致在未知环境中导航性能受限的问题。

🛠 **方法**：提出层次化语义理解与空间感知（SUSA）架构，包含文本语义理解（TSU）模块生成视图级描述以捕捉细粒度语义，以及深度增强空间感知（DSP）模块构建轨迹级深度探索图以表征全局空间布局。

✨ **主要贡献**：通过层次化表示丰富策略显著提升离散VLN基准（REVERIE、R2R、SOON）的导航性能，并有效泛化至连续R2R-CE基准。
