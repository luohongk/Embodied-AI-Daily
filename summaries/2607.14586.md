# SoftNav: Injecting 3D Scene Tokens into VLMs for Embodied Navigation

- arXiv: https://arxiv.org/abs/2607.14586v1
- arXiv ID: 2607.14586

---

🔍 **核心问题**：在目标导向的具身导航任务中，现有方法通过文本将3D场景信息传递给视觉语言模型（VLM），导致表示鸿沟，限制了导航性能。

🛠 **方法**：提出SoftNav，通过轻量投影器将每个检测到的物体或前沿的实体级3D连续表示作为软令牌注入VLM的隐藏空间，冻结3D编码器和VLM，仅需约1200个样本和1700万可训练参数。

✨ **主要贡献**：在HM3D-OVON上达到74.2%/68.3%/66.7%的成功率（SR），超越所有先前方法；零样本迁移至GOAT-Bench（67.2% SR）、SG3D（47.2% s-SR）及真实机器人部署，无需重新训练或架构修改，弥合了表示鸿沟。
