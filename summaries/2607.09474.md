# ProofCouncil: An LLM Agent for Solving Open Mathematical Problems

- arXiv: https://arxiv.org/abs/2607.09474v1
- arXiv ID: 2607.09474

---

🔍 **核心问题**：解决大型语言模型在开放式数学问题求解中性能不足的问题，提升其自主解决真实数学难题的能力。  
🛠 **方法**：提出基于“作者-评论者”架构的数学智能体ProofCouncil，通过智能体工作流模拟数学实践中的协作与验证过程。  
✨ **主要贡献**：在FirstProof挑战赛的10个问题中正确解决6个（仅需少量修正），并在30个开放数学问题中5个完全正确、2个接近完成、8个提供有效进展；同时开源了构建该智能体的工具库。
