# PNE-SGAN: Probabilistic NDT-Enhanced Semantic Graph Attention Network for LiDAR Loop Closure Detection

- arXiv: https://arxiv.org/abs/2504.08280v2
- arXiv ID: 2504.08280

---

🔍 **核心问题**：解决LiDAR回环检测在复杂环境中因噪声、动态物体和视角变化导致的鲁棒性与精度不足问题，尤其是现有语义图方法几何表示粗糙且缺乏时间域鲁棒性。

🛠 **方法**：提出PNE-SGAN模型，通过正态分布变换（NDT）协方差矩阵增强语义图的几何节点特征，并利用图注意力网络（GAT）处理；同时将图相似度分数集成到概率时间滤波框架（HMM/贝叶斯滤波器）中，结合不确定里程计运动模型和前向后向平滑处理歧义。

✨ **主要贡献**：在KITTI数据集上实现平均精度96.2%（序列00）和95.1%（序列08），显著优于现有方法，尤其在困难的双向回环场景中表现突出，通过NDT几何细节与概率时间推理的协同提升SLAM可靠性。
