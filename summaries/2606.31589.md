# From Failure to Alignment: A Requirements Engineering Framework for Machine Learning Systems

- arXiv: https://arxiv.org/abs/2606.31589v1
- arXiv ID: 2606.31589

---

🔍 **核心问题**：如何系统化地建模和推理机器学习系统的需求，使其与利益相关者的需求对齐，并确保系统可信赖。  
🛠 **方法**：提出REAL框架，基于三个原则——整合数据、模型和系统需求，利用失败驱动替代需求探索，以及迭代可追溯的需求细化。  
✨ **主要贡献**：通过自动驾驶案例验证，REAL框架能有效支持开发更符合利益相关者需求的机器学习系统，并提供可复现的在线资源。
