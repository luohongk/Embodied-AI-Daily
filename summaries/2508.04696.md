# Achieving Precise and Reliable Locomotion with Differentiable Simulation-Based System Identification

- arXiv: https://arxiv.org/abs/2508.04696v1
- arXiv ID: 2508.04696

---

🔍 **核心问题**：解决双足机器人运动控制中因系统参数不准确导致的轨迹漂移问题，提升轨迹跟踪的精确性和可靠性。  
🛠 **方法**：提出一种将系统辨识集成到强化学习训练循环中的控制框架，利用可微分仿真器MuJoCo-XLA，仅通过轨迹数据（位置、速度）和控制输入优化系统参数，并支持通过神经网络近似处理复杂非线性行为（如高级摩擦模型）。  
✨ **主要贡献**：无需直接力矩测量即可实现参数优化，显著提升轨迹跟踪性能；框架可扩展且灵活，能同时优化质量、惯性等物理参数及复杂非线性模型。
