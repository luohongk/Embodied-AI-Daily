# Lagrange: An Open-Vocabulary, Energy-Based Sparse Framework for Generalized End-to-End Driving

- arXiv: https://arxiv.org/abs/2606.20274v1
- arXiv ID: 2606.20274

---

🔍 **核心问题**：现有端到端自动驾驶模型在开放世界中面临效率与泛化能力的矛盾——密集模型计算开销大且语义推理弱，稀疏模型依赖封闭集定义，难以应对分布外事件，而视觉-语言-动作模型因离散生成方式不适用于连续控制需求。

🛠 **方法**：提出Lagrange框架，利用视觉-语言模型编码类别无关的目标提议为连续语义视觉令牌，通过意图驱动的掩码交叉注意力模块过滤无关实体，并将注意力解码为空间坐标上的隐式连续能量场，将决策建模为能量场上的拉格朗日动作最小化问题。

✨ **主要贡献**：在nuScenes和CODA基准上验证了Lagrange能实现鲁棒、可解释且符合运动学的开放世界自动驾驶，为稀疏、开放词汇的端到端驾驶提供了新范式。
