# PagedWeight: Efficient MoE LLM Serving with Dynamic Quality-Aware Weight Quantization

- arXiv: https://arxiv.org/abs/2607.16184v1
- arXiv ID: 2607.16184

---

🔍 **核心问题**：在KV缓存密集型的MoE大模型服务场景中，模型权重与不断增长的KV缓存之间存在GPU内存竞争，导致内存瓶颈和性能下降。

🛠 **方法**：提出PagedWeight，一种运行时动态量化MoE模型权重的方法，根据任务需求灵活调整专家权重的精度，以平衡模型精度与KV缓存的内存占用。

✨ **主要贡献**：首次在MoE服务中显式建模并优化精度-内存-吞吐量的复杂权衡；在多个内存敏感场景下，相比现有量化方法，PagedWeight在同等内存预算下最高提升39.3%的模型质量，同时实现高达72.0%的GPU内存节省和1.94倍的吞吐量提升，且精度损失不超过4.1%。
