# Hy-Embodied-VLM-1.0: Efficient Physical-World Agents

- arXiv: https://arxiv.org/abs/2607.12894v1
- arXiv ID: 2607.12894

---

🔍 **核心问题**：如何构建一个高效且强大的具身基础模型，使其能够在物理世界中具备多模态感知、动作推理和自适应交互能力。

🛠 **方法**：提出Hy-Embodied-VLM-1.0模型，基于Hy3-A3B语言骨干和Hy-ViT2视觉编码器，采用高效的混合专家架构，并设计以动作为核心的能力分类体系，从预训练到后训练系统化构建数据流程。

✨ **主要贡献**：在38个具身基准测试中，模型在19个上达到同类最佳，平均性能比上一代提升8.4%，仅激活3B参数即可接近32B参数模型的性能，并在多轮交互和长程推理任务中表现优异。
