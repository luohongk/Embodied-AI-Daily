# Fail-RAG : A Retrieval Augmented Generation Informed Framework for Robot Failure Identification

- arXiv: https://arxiv.org/abs/2606.19598v1
- arXiv ID: 2606.19598

---

🔍 **核心问题**：针对仓储自动化中机器人因环境和任务动态变化导致的意外故障，传统基于规则的检测方法难以适应，需要一种能灵活识别多种操作故障的智能检测方案。

🛠 **方法**：提出Fail-RAG框架，通过将故障图像与上下文信息嵌入向量并计算相似度，从故障数据库中检索匹配案例，再结合视觉语言模型（VLMs）按指令模板分析故障细节。

✨ **主要贡献**：在五种常见仓储机器人操作任务中，Fail-RAG的故障检测准确率平均比直接使用现成VLMs高出25个百分点，验证了其在真实场景中的有效性。
