# Optimization and Deep Learning based Resource Allocation for UAV-Aided Wireless Communication with Rotatable Antenna Array

- arXiv: https://arxiv.org/abs/2607.08420v1
- arXiv ID: 2607.08420

---

🔍 **核心问题**：针对无人机辅助无线通信中，传统固定天线阵列（FAA）无法灵活调整波束方向以优化多用户信号传输和干扰管理的问题，提出利用可旋转天线阵列（RAA）提升系统容量和服务质量。

🛠 **方法**：联合优化RAA的朝向和波束赋形，先提出基于惩罚对偶分解（PDD）的迭代优化框架，再设计基于图神经网络（GNN）的深度学习框架以加速求解。

✨ **主要贡献**：验证了RAA相比FAA在无人机通信中的显著优势；优化框架能高可靠地满足严格服务质量约束，而深度学习框架在保持相近和速率性能的同时大幅降低计算时间，并对用户位置误差具有鲁棒性。
