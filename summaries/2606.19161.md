# HT-Bench: Benchmarking and Learning Dexterous Full-Hand Tactile Representations with Egocentric Vision

- arXiv: https://arxiv.org/abs/2606.19161v1
- arXiv ID: 2606.19161

---

🔍 **核心问题**：现有触觉表示学习缺乏统一基准，难以评估不同传感器和机器人形态下的触觉表征质量。  
🛠 **方法**：构建大规模多任务基准HT-Bench（含10M RGB帧和7.8M触觉帧），并提出HandTouch向量量化视觉-触觉编码器，通过空间、跨模态和时间渐进训练学习触觉表示。  
✨ **主要贡献**：HT-Bench从接触几何编码、视觉-触觉对齐和任务泛化三个维度评估触觉表征；HandTouch在细粒度触觉检索（Recall@5提升至85.23%）、掩膜触觉修复（RMSE降至0.010）和视觉到触觉合成（OOD cIoU达0.705）等任务上显著优于基线方法。
