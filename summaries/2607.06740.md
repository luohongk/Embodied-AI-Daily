# A Continual Learning Framework for Adaptive Control of Modular Soft Robots

- arXiv: https://arxiv.org/abs/2607.06740v1
- arXiv ID: 2607.06740

---

🔍 **核心问题**：模块化软机器人因高度非线性动力学和形态变化，导致控制器需从零重新训练，缺乏适应性。  
🛠 **方法**：提出基于持续学习的控制框架，通过增量学习适应机器人形态变化，同时保留先前知识；对固定形态的MSR，采用分布式方式学习模块特定动力学。  
✨ **主要贡献**：在仿真和真实三模块气动软机械臂上验证了闭环轨迹跟踪性能，并通过选择性激活模块的到达实验展示了自适应能力，降低了计算开销。
