# Holo-World: Unified Camera, Object and Weather Control for Video World Model

- arXiv: https://arxiv.org/abs/2606.20083v1
- arXiv ID: 2606.20083

---

🔍 **核心问题**：现有视频世界模型无法同时控制相机运动、物体运动和环境天气，且天气生成通常依赖已有视频或重建场景，缺乏从单张图像出发的联合控制能力。

🛠 **方法**：提出Holo-World模型，包含HoloStateData数据集（将多样视频转化为统一控制样本）和Unified Scene Adapter（将世界保持与天气转移分解为独立参数子空间），并引入Scene-Weather Decomposed CFG（分别引导场景与天气残差，增强目标天气效果而不过度放大整体条件）。

✨ **主要贡献**：首次实现从单张图像对相机、物体和天气的联合控制，在保持精确场景结构的同时生成多样目标天气状态，性能优于视频到视频的天气编辑基线。
