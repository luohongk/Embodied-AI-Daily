# D-SafeMPC: Diffusion-Driven Safe Model Predictive Control with Discrete-Time Control Barrier Functions

- arXiv: https://arxiv.org/abs/2607.10842v1
- arXiv ID: 2607.10842

---

🔍 **核心问题**：扩散模型在机器人规划中无法天然保证安全或动力学约束，导致输出可能物理不可行或不安全，而结合模型预测控制（MPC）的混合方法因扩散模型初始轨迹质量差而难以收敛到安全可行解。

🛠 **方法**：提出D-SafeMPC，利用控制障碍函数（CBF）和控制李雅普诺夫函数（CLF）引导反向扩散过程，并在每个去噪步骤中采用迭代投影方案，由MPC优化轨迹，从而将采样导向安全、目标导向的区域，并为MPC提供可靠的初始解。

✨ **主要贡献**：在Franka机械臂的四个仿真场景（一个静态障碍物、三个动态障碍物）及真实机器人实验中，D-SafeMPC相比现有方法显著提升了安全性、任务成功率和规划效率，并开源了代码与配置。
