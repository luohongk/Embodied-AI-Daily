# Mosaic: An Extensible Framework for Composing Rule-Based and Learned Motion Planners

- arXiv: https://arxiv.org/abs/2604.13853v2
- arXiv ID: 2604.13853

---

🔍 **核心问题**：自动驾驶中规则型规划器可解释但适应性差，学习型规划器适应性强但透明度和安全性不足，如何融合两者优势实现安全、可解释的决策。

🛠 **方法**：提出Mosaic框架，通过仲裁图（arbitration graphs）将轨迹生成与验证、选择解耦，使集中式验证作为安全底线、逐轨迹选择作为性能上限，无需重新训练或额外数据。

✨ **主要贡献**：在nuPlan基准上达到95.56 CLS-NR和94.18 CLS-R，创下新纪录；在interPlan高交互场景中得分54.10 CLS-R，比最佳单一规划器提升22.8%，且将各规划器的责任碰撞从25次降至16次。
