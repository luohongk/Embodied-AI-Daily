# Learning New Tasks via Reusable Skills: Skill-Compositional Experts for Embodied Continual Learning

- arXiv: https://arxiv.org/abs/2606.15685v1
- arXiv ID: 2606.15685

---

🔍 **核心问题**：解决具身持续学习（ECL）中机器人在闭环控制下执行新操作任务时，因特征漂移累积导致的严重灾难性遗忘问题，以及缺乏结构化技能复用机制以支持连贯任务执行。  
🛠 **方法**：提出技能组合专家框架（SCE），通过组合技能基础（CSG）将任务演示分解为可复用技能，并设计双执行-转换专家（DETE），一个分支确保技能执行，另一个分支支持技能间平滑转换，实现新任务的技能组合学习。  
✨ **主要贡献**：在LIBERO基准和真实操作任务中，SCE显著提升了旧任务保留率与整体任务性能，并通过特征漂移分析和消融实验验证了其有效性，为机器人持续学习提供了结构化技能复用的新范式。
