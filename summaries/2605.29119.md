# PRO-CUA: Process-Reward Optimization for Computer Use Agents

- arXiv: https://arxiv.org/abs/2605.29119v1
- arXiv ID: 2605.29119

---

🔍 **核心问题**：计算机使用代理（CUA）在训练中面临模仿瓶颈（如专家演示分布偏移、缺乏负反馈信号）和轨迹级强化学习的稀疏奖励、信用分配模糊及高基础设施成本问题。

🛠 **方法**：提出PRO-CUA框架，通过解耦在线环境交互与策略优化，利用过程奖励模型（PRM）对每个步骤提供密集反馈，并结合组相对优势进行迭代步骤级强化学习。

✨ **主要贡献**：实现了无需黄金答案或离线专家轨迹的密集、灵活信用分配，减少了分布偏移，并在真实网页基准测试中验证了PRM引导的步骤级训练的有效性和可靠性。
