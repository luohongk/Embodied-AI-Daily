# LinkAnchor: An Autonomous LLM-Based Agent for Issue-to-Commit Link Recovery

- arXiv: https://arxiv.org/abs/2508.12232v5
- arXiv ID: 2508.12232

---

🔍 **核心问题**：软件仓库中问题（issue）与提交（commit）之间的链接恢复准确率低（仅约42.2%），现有方法受限于LLM的上下文窗口大小，且仅处理单对问题-提交关系，忽略了提交链的时序依赖和整体修复逻辑。

🛠 **方法**：提出首个基于LLM的自主智能体LinkAnchor，采用“懒加载”架构，动态检索最相关的上下文数据（如提交、问题评论、代码文件），避免超出token限制。

✨ **主要贡献**：首次将自主LLM智能体应用于问题-提交链接恢复，通过动态上下文管理克服了传统方法的上下文窗口限制和计算效率问题。
