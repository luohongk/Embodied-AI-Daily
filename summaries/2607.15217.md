# NeuronSoup: Evolving Asynchronous, Shared-Neuron Temporal Graphs without Backpropagation

- arXiv: https://arxiv.org/abs/2607.15217v1
- arXiv ID: 2607.15217

---

🔍 **核心问题**：当前深度学习依赖可微计算图和同步逐层处理，难以适应异步信号传播和跨路径的神经元共享机制，且计算深度固定、缺乏灵活性。

🛠 **方法**：提出NeuronSoup架构，通过共享神经元池实现异步、延迟介导的信号传播，路径间共享隐藏神经元产生信号干涉；采用遗传算法在扁平实数基因组上协同进化拓扑、权重、延迟和连接性。

✨ **主要贡献**：无需反向传播即可进化出异步、共享神经元的时序图，在MNIST分类任务上达到85.9%测试准确率，模型仅115KB；计算深度可逐样本自适应，自动发现路径间的横向交互，并通过替换编码器与输出结构泛化到任意领域。
