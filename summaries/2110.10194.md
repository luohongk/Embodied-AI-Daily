# CoFi: Coarse-to-Fine ICP for LiDAR Localization in an Efficient Long-lasting Point Cloud Map

- arXiv: https://arxiv.org/abs/2110.10194v2
- arXiv ID: 2110.10194

---

🔍 **核心问题**：解决基于ICP的激光雷达定位方法因非凸性和局部迭代策略易陷入局部最优、需要精确初始化的难题。

🛠 **方法**：提出CoFi算法，通过多体素分辨率下采样输入点云，从粗粒度到细粒度逐步优化变换矩阵；并结合Cylinder3D语义分割提取特征点，在高效点云地图上实现定位。

✨ **主要贡献**：在KITTI里程计基准上达到最先进性能，显著优于现有方法，实现了鲁棒且高效的激光雷达定位。
