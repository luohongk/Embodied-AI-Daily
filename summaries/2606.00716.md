# Graph Transfer Learning via Shared Latent Geometry: Theory and Applications

- arXiv: https://arxiv.org/abs/2606.00716v1
- arXiv ID: 2606.00716

---

🔍 **核心问题**：解决物理系统部署时因需反复求解数值优化而导致的“物理成本”高昂问题，以及现有物理信息学习方法在微调时丢失潜在几何结构、缺乏可量化迁移保证的缺陷。

🛠 **方法**：提出一种非对称双路径架构，教师编码器从高保真模拟器中学习基于算子多项式的稳定潜在几何，学生编码器从稀疏现场数据中学习相同几何，部署时仅使用冻结的学生编码器进行单次前向传播，并基于Wasserstein距离建立迁移条件与零样本误差界。

✨ **主要贡献**：首次实现带证书的零样本迁移，在电力系统估计任务中成功迁移至100种未见拓扑，证书通过率达95%，精度媲美拓扑感知的牛顿-拉夫逊方法，推理时间低于毫秒级，为物理系统的零样本推理与控制提供了理论基础。
