# Learning to Move Before Learning to Do: Task-Agnostic pretraining for VLAs

- arXiv: https://arxiv.org/abs/2607.02466v1
- arXiv ID: 2607.02466

---

🔍 **核心问题**：视觉-语言-动作（VLA）模型因专家演示数据稀缺而受限，传统方法将“如何移动”（物理能力）与“做什么”（语义对齐）两个目标混淆，导致数据效率低下。

🛠 **方法**：提出任务无关预训练（TAP）两阶段框架：第一阶段利用廉价无标签交互数据（如离轨轨迹、自主机器人探索），通过自监督逆动力学目标学习可迁移的运动先验；第二阶段用少量专家数据将运动先验与语言指令对齐。

✨ **主要贡献**：在SIMPLER基准上，TAP仅用极少量标注数据即可匹配百万级专家轨迹训练的性能，绝对成功率提升10%；在真实WidowX平台上，TAP在相机扰动下保持25%成功率，而大规模基线模型完全失效，证明任务无关预训练能生成鲁棒、可迁移的物理表征。
