# EchoSonar-R: A Multi-View Reasoning-Enabled Model for Disease Classification and Report Generation in Echocardiography

- arXiv: https://arxiv.org/abs/2606.28164v1
- arXiv ID: 2606.28164

---

🔍 **核心问题**：现有超声心动图疾病分类模型缺乏明确的诊断推理能力和空间定位的解剖学证据，导致临床医生信任度不足。

🛠 **方法**：提出EchoSonar-R模型，结合时空视频编码器与结构感知心脏检测器，通过两阶段训练（监督微调+基于GRPO的强化学习）联合完成多标签疾病分类和报告生成。

✨ **主要贡献**：在私有数据集上宏平衡准确率提升17.1%，在MIMICEchoQA上提升6.1%，临床忠实度GREEN得分达0.800，并能生成基于多视图视觉证据的可解释推理轨迹。
