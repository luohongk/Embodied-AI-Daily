# TabPack: Efficient Hyperparameter Ensembles for Tabular Deep Learning

- arXiv: https://arxiv.org/abs/2607.05380v1
- arXiv ID: 2607.05380

---

🔍 **核心问题**：表格数据深度学习中的多层感知机（MLP）集成方法依赖超参数调优才能达到最佳性能，导致计算资源浪费和调优成本高。

🛠 **方法**：提出TabPack框架，在单次运行中并行采样并训练多个不同超参数的MLP，训练过程中动态选择集成成员，仅需指定超参数采样范围而非精确值。

✨ **主要贡献**：默认配置下性能与经过大量调优的现有方法相当，显著降低调优所需计算资源（如在现代MacBook上运行时间少于部分基线在工业级GPU上的调优时间）。
