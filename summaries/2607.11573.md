# Knowledge-Guided Synthetic Bug Feedback for LLM-Based Unit Test Generation

- arXiv: https://arxiv.org/abs/2607.11573v1
- arXiv ID: 2607.11573

---

🔍 **核心问题**：基于大语言模型（LLM）的单元测试生成虽然能产生可执行测试，但难以有效检测真实缺陷。  
🛠 **方法**：提出一个框架，将历史真实缺陷机制转化为结构化和语义化表示，检索适用于目标方法的机制，并实例化为合成缺陷，引导LLM迭代增强测试生成。  
✨ **主要贡献**：在Defects4J上的实验表明，该方法在真实缺陷检测上优于基于执行、覆盖、变异、知识和搜索的基线方法，证明将真实缺陷机制组织为可检索、可执行的反馈目标是提升测试缺陷触发能力的有效途径。
