# Learning Residual Kinematic Corrections for Continuous Neural Decoding via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.11530v1
- arXiv ID: 2607.11530

---

🔍 **核心问题**：针对非侵入式脑机接口在连续三维运动想象解码中存在的信号变异性与系统性残差误差问题，提升解码精度。

🛠 **方法**：提出两阶段解码框架，先使用CNN-LSTM模型进行初始运动学解码，再通过强化学习离线对预测轨迹进行残差校正，无需额外脑电数据输入。

✨ **主要贡献**：在2D和VR场景下，该方法将平均相关系数分别提升41.5%和21.2%，均方根误差降低40.2%和38.2%，显著增强三维运动想象解码性能，可推广至神经康复、假肢控制及虚拟交互领域。
