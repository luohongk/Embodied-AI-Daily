# Can LLM Agents Infer World Models? Evidence from Agentic Automata Learning

- arXiv: https://arxiv.org/abs/2606.16576v1
- arXiv ID: 2606.16576

---

🔍 **核心问题**：评估当前具备工具调用能力的大语言模型（LLM）代理能否通过交互式查询，自主推断隐藏的确定性有限自动机（DFA）所代表的“世界模型”。

🛠 **方法**：提出“代理自动机学习”框架，让LLM代理通过与预言机进行成员查询（判断字符串是否属于目标语言）和等价查询（判断DFA是否正确）来逐步发现目标DFA，并以经典自动机学习算法作为性能基线。

✨ **主要贡献**：实验表明，随着DFA规模增大，LLM代理性能急剧下降；推理模型显著优于非推理模型，但轨迹分析揭示其在查询规划、证据整合和假设构建方面存在系统性失败。当前LLM代理虽能完成部分非平凡交互发现，但鲁棒性和效率远不及经典算法。
