# SA-LIVO: Efficient LiDAR-Inertial-Visual Odometry with Subspace-Aware Degeneracy Handling

- arXiv: https://arxiv.org/abs/2606.25699v1
- arXiv ID: 2606.25699

---

🔍 **核心问题**：现有紧耦合LiDAR-视觉-惯性里程计（LIVO）中，LiDAR和视觉传感器各自存在独立失效模式（如LiDAR在几何约束不足时退化，视觉在光照差或无纹理时退化），但现有方法仅在模态层面处理退化，未考虑联合信息矩阵的方向依赖性，导致视觉残差在LiDAR已良好约束的方向上仍被引入，而在退化方向上视觉补偿分散于全状态空间而非集中所需方向。

🛠 **方法**：提出SA-LIVO系统，通过子空间感知信息融合（SAIF）框架对联合LiDAR-视觉信息矩阵进行特征分解，并在每个特征方向上应用线性-钳位软门控，衰减退化方向同时保持可观测方向的全强度；随后在一个共享线性化点的InEKF循环中联合优化LiDAR和视觉残差，且视觉雅可比矩阵仅在循环前组装一次并在迭代中复用，避免传统迭代滤波器的逐次计算开销。

✨ **主要贡献**：在三个基准（HILTI'22、New College、Oxford Spires）的29个序列及并发退化场景中，SA-LIVO达到与最强基线相当的精度，并在竞争系统发散时保持有界漂移；在笔记本电脑CPU上平均每帧12.3毫秒，嵌入式ARM板上26.8毫秒，峰值内存降低3.6-6.3倍，代码将开源。
