# RADAR: Closed-Loop Robotic Data Generation via Semantic Planning and Autonomous Causal Environment Reset

- arXiv: https://arxiv.org/abs/2603.11811v2
- arXiv ID: 2603.11811

---

🔍 **核心问题**：解决机器人数据采集依赖人工干预、成本高昂且难以规模化扩展的瓶颈问题。

🛠 **方法**：提出RADAR全自动闭环数据生成引擎，包含四个模块：视觉语言模型（VLM）基于3D演示生成语义任务、图神经网络（GNN）策略执行模仿学习、VLM通过视觉问答自动评估任务成功、有限状态机（FSM）结合因果序列实现自主环境重置与失败恢复。

✨ **主要贡献**：实现了无需人工干预的持续数据采集，在复杂长时任务中达到90%成功率，并在真实场景中通过少样本适应成功执行可变形物体操作等接触密集型技能，无需领域微调。
