# Topological Shape Representation for Aneurysm -- Bifurcation Detection

- arXiv: https://arxiv.org/abs/2607.05317v1
- arXiv ID: 2607.05317

---

🔍 **核心问题**：CT血管造影中颅内动脉瘤自动检测受高假阳性率严重制约，尤其是小动脉瘤（<3 mm）检测灵敏度低于60%，主要原因是卷积神经网络难以区分囊状动脉瘤与血管分叉结构。

🛠 **方法**：提出一种即插即用的拓扑感知假阳性降低框架，利用平滑欧拉特征变换（SECT）编码全局3D血管几何形状（不依赖像素强度），并与持久性图像和持久性景观等摘要方法对比，在RSNA 2025数据集分层子集上测试。

✨ **主要贡献**：SECT在整体检测中达到0.943的AUC，远超方向无关方法（AUC约0.68）；在亚3毫米队列中保持0.943 AUC和78.5%灵敏度（95%特异度），且跨四家制造商的留一扫描仪验证中平均AUC达0.927，可作为混合深度学习诊断管线的鲁棒下游滤波器。
