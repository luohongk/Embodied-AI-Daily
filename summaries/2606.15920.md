# OmniOPSD: Rationale-Privileged On-Policy Self-Distillation for Affective Computing

- arXiv: https://arxiv.org/abs/2606.15920v2
- arXiv ID: 2606.15920

---

🔍 **核心问题**：多模态大语言模型在复杂推理任务中面临奖励稀疏问题，尤其在涉及人类情感、意图等主观因素时，高质量思维链标注昂贵且难以获取，直接使用标签进行监督微调易导致捷径学习并缺乏透明度。

🛠 **方法**：提出OmniOPSD框架，采用“理由特权”在线策略自蒸馏方法，将前沿模型生成的推理理由作为教师端的特权证据（而非学生模仿目标），教师基于此提供密集的令牌级监督，学生则从原始多模态输入中自行采样轨迹进行学习。

✨ **主要贡献**：在MER-UniBench上以平均84.19分达到最先进性能，消融实验验证了理由特权教师指导的有效性；推理时无需标签、理由或闭源模型访问，提升了安全关键人机交互的透明性。
