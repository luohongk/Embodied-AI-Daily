# Dynamic Object Detection and Tracking in Construction: A Fisheye Camera and LiDAR Sensor Fusion Model

- arXiv: https://arxiv.org/abs/2607.06896v1
- arXiv ID: 2607.06896

---

🔍 **核心问题**：在建筑工地等复杂环境中，如何实现机器人对动态物体的高精度实时检测与跟踪，尤其是处理物体在动态与静态状态间切换的难题。

🛠 **方法**：提出一种融合鱼眼相机与LiDAR传感器的框架，利用LiDAR点云检测运动物体，再通过将3D坐标投影到2D圆柱全景图上，与实时图像检测结果对齐，为卡尔曼滤波器提供观测更新，实现语义标签分配与跟踪。

✨ **主要贡献**：该方法无需预训练神经网络或额外后处理，具有高精度、简单性和鲁棒性，能有效处理物体状态切换，适用于真实建筑工地的机器人部署。
