# MoCo-AIS: A Contrastive Learning Framework for Similarity Computation of Vessel Trajectories

- arXiv: https://arxiv.org/abs/2606.17978v1
- arXiv ID: 2606.17978

---

🔍 **核心问题**：传统轨迹相似度计算方法计算成本高，监督学习方法依赖大量标签且泛化能力有限，自监督学习缺乏统一框架，难以有效比较不同深度学习模型在轨迹表示上的一致性。

🛠 **方法**：提出MoCo-AIS框架，基于动量对比学习（MoCo）范式，通过构建正负轨迹对进行对比学习，并在大规模真实船舶跟踪AIS数据集上评估多种主流深度学习模型。

✨ **主要贡献**：显著提升了轨迹相似度学习性能，优于现有基线方法，同时提供了一个统一的基准平台，用于评估和比较不同轨迹表示模型。
