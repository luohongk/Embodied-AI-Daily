# Reasoning Beyond Prediction: From Data-Driven to Causal Software Engineering

- arXiv: https://arxiv.org/abs/2606.27960v1
- arXiv ID: 2606.27960

---

🔍 **核心问题**：当前基于深度学习的软件工程方法仅能进行模式预测，无法满足现代软件开发对智能支持的更高要求，缺乏因果推理能力。

🛠 **方法**：提出人机协作的新范式，从数据驱动转向因果驱动，让机器通过因果推理增强工程师的决策与推理能力。

✨ **主要贡献**：呼吁并定义了因果软件工程的新方向，强调机器应超越自动化与预测，主动提升工程师的因果推理能力，以应对日益复杂的软件系统。
