# Learning Physics-Guided Residual Dynamics for Deformable Object Simulation

- arXiv: https://arxiv.org/abs/2607.13451v1
- arXiv ID: 2607.13451

---

🔍 **核心问题**：解决可变形物体在机器人操作中动力学模拟不准确的问题，现有纯物理或纯学习方法均存在局限。

🛠 **方法**：提出物理引导残差动力学（PGRD）混合框架，以可优化的弹簧-质点模型为骨干，结合滑动窗口Transformer网络预测物理模拟的残差修正，并采用基于速度的公式确保稳定性。

✨ **主要贡献**：在多种真实可变形物体上，PGRD的模拟精度优于纯物理和纯学习方法；成功应用于模型预测控制的操控规划（含语言条件目标图像生成）及基于3D高斯泼溅的动作条件视频预测交互模拟。
