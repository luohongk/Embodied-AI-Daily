# MANGO: Automated Multi-Agent Test Oracle Generation for Vision-Language-Action Models

- arXiv: https://arxiv.org/abs/2606.24815v1
- arXiv ID: 2606.24815

---

🔍 **核心问题**：现有VLA机器人测试依赖人工构建的符号化测试预言，成本高、领域依赖性强，且仅能评估最终状态，无法定位中间行为故障。

🛠 **方法**：提出MANGO多智能体框架，通过自然语言任务描述自动生成细粒度测试预言，包含原子任务库构建、模拟器验证的预言定义，以及复杂指令分解为有序原子动作与对应预言，并利用生成器、评估器和裁判智能体迭代优化。

✨ **主要贡献**：在LIBERO_10和RoboCasa基准上验证，MANGO生成的细粒度预言能检测与符号预言相当的故障数量，同时准确定位故障并提供更丰富的诊断信息，消融实验进一步证实了各组件的有效性。
