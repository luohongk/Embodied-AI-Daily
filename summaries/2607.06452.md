# From Voting to Agent Collaboration: Answer-Type-Aware LLM Pipelines for BioASQ 14b

- arXiv: https://arxiv.org/abs/2607.06452v1
- arXiv ID: 2607.06452

---

🔍 **核心问题**：解决生物医学问答中不同问题类型（是非题、事实题、列表题）需要不同推理策略，而单一提示方法难以兼顾准确性与证据可靠性的问题。

🛠 **方法**：提出问题类型感知的LLM框架，对是非题采用片段打乱与自我反思，对事实题使用全片段输入结合思维链上下文学习，对列表题构建多智能体协作架构（证据提取、候选生成、答案验证、最终聚合）。

✨ **主要贡献**：在BioASQ 14b官方评测中取得竞争性表现，并在第4批次事实题子任务中获第一名，验证了问题类型特异性推理、集成预测与基于智能体的验证相结合的有效性。
