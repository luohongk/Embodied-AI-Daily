# XR-1: Towards Versatile Vision-Language-Action Models via Learning Unified Vision-Motion Representations

- arXiv: https://arxiv.org/abs/2511.02776v3
- arXiv ID: 2511.02776

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型难以从高维观测生成精确的低层动作，且无法有效融合异构数据（如不同机器人形态和人类演示）中的多模态知识。

🛠 **方法**：提出XR-1框架，通过双分支VQ-VAE学习离散潜在表示“统一视觉-运动编码（UVMC）”，联合编码视觉动态与机器人运动，并采用三阶段训练范式（自监督学习、跨形态预训练、任务微调）。

✨ **主要贡献**：在6种机器人形态、120余项任务、超14,000次实际部署中，XR-1显著优于π₀.₅、π₀等基线，并展现出对新颖物体、背景变化等干扰的强泛化能力。
