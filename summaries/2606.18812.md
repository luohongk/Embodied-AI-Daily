# Reinforcement Learning Foundation Models Should Already Be A Thing

- arXiv: https://arxiv.org/abs/2606.18812v2
- arXiv ID: 2606.18812

---

🔍 **核心问题**：当前基础模型在语言和视觉领域依赖互联网规模数据，但在结构化任务（如表格预测）中需用合成数据替代，而强化学习领域尚未系统性地利用合成马尔可夫决策过程（MDP）来构建基础模型。

🛠 **方法**：提出通过采样合成MDP作为先验，并利用注意力架构（如图注意力网络）处理MDP的固定大小充分统计量，以策略头替代监督目标，从而在上下文中解决强化学习任务。

✨ **主要贡献**：首次论证了强化学习基础模型的可行性，并在合成MDP上训练的模型无需任务特定调优，即可在在线和离线表格基准测试中显著优于UCB-VI和表格Q-learning等传统方法。
