# Varying Bundle Size Reactive Multi-Task Assignment using Selective Cost Estimation for Multi-Agent Systems

- arXiv: https://arxiv.org/abs/2606.24462v1
- arXiv ID: 2606.24462

---

🔍 **核心问题**：在多机器人系统中，任务执行成本估算（如路径规划）计算开销大，导致传统组合拍卖方法难以实时处理动态任务分配问题。

🛠 **方法**：提出一种分布式两阶段多保真度束生成框架，先用低保真启发式（如欧氏距离）快速搜索束空间，再对最有潜力的候选方案进行高保真路径规划，最后由中央协调器求解集合包装问题以保障全局可行性。

✨ **主要贡献**：实现了动态束大小的实时反应式任务分配，在不暴露智能体状态和内部成本模型的前提下，显著提升了拍卖式任务分配的性能。
