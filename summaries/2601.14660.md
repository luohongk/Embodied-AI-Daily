# NeuroFilter: Activation-Based Guardrails for Privacy-Conscious LLM Agents

- arXiv: https://arxiv.org/abs/2601.14660v2
- arXiv ID: 2601.14660

---

🔍 **核心问题**：解决智能体大语言模型在推理时因访问敏感信息而导致的隐私泄露风险，尤其是无法持续遵守隐私规范的问题。  
🛠 **方法**：提出基于激活探测（activation probing）的隐私过滤器，通过分析模型内部状态来检测和限制不当信息泄露，适用于单轮和多轮对话场景。  
✨ **主要贡献**：首次系统性地研究对话轨迹中的模型内部状态探测，超越了静态单提示分析，实现了高效且计算成本低的隐私保护，并验证了其对语义审查攻击的鲁棒性。
