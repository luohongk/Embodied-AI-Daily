# A Single Diffusion-Policy Controller for Multi-Task Block Pushing with Zero-Shot Sim-to-Real Transfer

- arXiv: https://arxiv.org/abs/2607.10892v1
- arXiv ID: 2607.10892

---

🔍 **核心问题**：如何在没有人类演示的情况下，利用强化学习训练一个单一的扩散策略模型，使其能够完成多形状积木推动任务，并实现零样本从仿真到真实环境的迁移。

🛠 **方法**：提出一个基于强化学习的框架，使用重加权证据下界作为策略损失函数，结合逆向课程生成和目标中心化表示，以解决稀疏奖励下的探索难题，并利用扩散策略的表达能力学习多任务策略。

✨ **主要贡献**：首次在无演示条件下，通过强化学习成功训练出单一扩散策略控制器，在仿真中完成多形状积木推动任务，并验证了该策略在真实环境中对目标位置、积木形状、重量和表面摩擦等变化具有零样本迁移能力。
