# AnoleVLA: Lightweight Vision-Language-Action Model with Deep State Space Models for Mobile Manipulation

- arXiv: https://arxiv.org/abs/2603.15046v1
- arXiv ID: 2603.15046

---

🔍 **核心问题**：解决语言引导的机器人操作任务中，视觉-语言-动作模型（VLA）因标准Transformer骨干网络计算成本高，难以在资源受限环境中部署的问题。

🛠 **方法**：提出AnoleVLA轻量级模型，采用深度状态空间模型高效处理多模态序列，利用其快速顺序状态建模能力处理视觉和文本输入，从而高效生成机器人轨迹。

✨ **主要贡献**：在真实世界评估中，AnoleVLA的任务成功率比代表性大规模VLA高出21个百分点，同时推理速度提升约3倍，实现了轻量化与高性能的平衡。
