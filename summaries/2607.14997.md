# AeroAct: Action-Centered World-Action Models for Language-Conditioned Quadrotor Flight

- arXiv: https://arxiv.org/abs/2607.14997v1
- arXiv ID: 2607.14997

---

🔍 **核心问题**：语言条件四旋翼飞行需要将语义目标与视觉预测结合，并输出平滑可执行的动态控制指令，现有方法缺乏对飞行动作如何改变未来观测的密集监督。

🛠 **方法**：提出AeroAct，首个用于真实空中飞行的世界-动作模型（WAM），基于预训练视频扩散Transformer，从第一人称视觉历史、本体感知和语言中预测局部轨迹-动作块，训练时利用未来帧作为密集后果监督，部署时直接解码动作。

✨ **主要贡献**：构建了DiffAero数据生成管道及低成本手持采集设备，引入自引导过程提升轨迹块的时间一致性；在闭环仿真和真实实验中验证了时间视觉上下文对目标跟踪和搜索性能的提升，并成功在物理四旋翼上执行WAM策略。
