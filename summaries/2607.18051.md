# SAR Vessel Detection and Gross Tonnage Estimation from Heterogeneous Datasets for Dark Vessel Identification

- arXiv: https://arxiv.org/abs/2607.18051v1
- arXiv ID: 2607.18051

---

🔍 **核心问题**：针对非法船只关闭应答器以逃避监控的“暗船”问题，现有深度学习模型虽能检测SAR图像中的船只，但缺乏公开数据集来同时实现船只检测与总吨位（GT）回归，从而无法区分必须携带应答器的船只。

🛠 **方法**：提出一个结合异构图像与表格数据的框架，采用多任务深度学习模型预测船只位置、类型和物理尺寸，再通过基于混合欧氏与分类距离的KNN非参数模型，从尺寸和类别回归总吨位。

✨ **主要贡献**：首次实现端到端的船只检测与总吨位回归，在各项子任务上达到与现有最优模型相当的性能，并开源代码，为暗船识别提供可行方案。
