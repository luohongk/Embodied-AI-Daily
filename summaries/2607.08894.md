# GATS: Graph-Augmented Tree Search with Layered World Models for Efficient Agent Planning

- arXiv: https://arxiv.org/abs/2607.08894v1
- arXiv ID: 2607.08894

---

🔍 **核心问题**：现有LLM智能体在规划任务中依赖大量模型推理，导致计算成本高、结果随机且成功率不稳定。

🛠 **方法**：提出GATS框架，结合基于UCB1的树搜索与三层世界模型（L1符号匹配、L2执行日志统计、L3未知动作的LLM预测），在规划阶段完全消除LLM调用。

✨ **主要贡献**：在合成任务中实现100%成功率（LATS为92%，ReAct为64%），在12个复杂场景中保持100%成功率（LATS降至88.9%），且规划阶段零LLM调用、结果零方差。
