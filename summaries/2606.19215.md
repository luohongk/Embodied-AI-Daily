# GUMP-Net: An interpretable model-data-driven intelligent algorithm for multi-class pelvic segmentation

- arXiv: https://arxiv.org/abs/2606.19215v1
- arXiv ID: 2606.19215

---

🔍 **核心问题**：解决骨盆多类别分割中精度不足、小样本训练效果差以及深度学习模型可解释性弱的问题。  
🛠 **方法**：将改进的测地线活动轮廓模型与深度神经网络结合，设计GUMP-Net框架，包含目标检测模块（自动初始化水平集）、边缘检测模块（学习解剖感知边缘函数）和迭代模块（深度水平集演化）。  
✨ **主要贡献**：在骨盆数据集上相比现有方法分割更准确、鲁棒且一致，尤其在小样本场景下优势显著；在踝关节数据集上验证了泛化性，并为深度学习分割提供了可解释的几何视角。
