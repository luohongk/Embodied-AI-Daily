# Attitude Estimation Using Inertial and Barometric Measurements

- arXiv: https://arxiv.org/abs/2607.13254v1
- arXiv ID: 2607.13254

---

🔍 **核心问题**：在GNSS信号缺失或高加速度飞行条件下，仅依靠惯性测量单元（IMU）难以准确估计飞行器姿态，因为重力加速度与惯性加速度难以区分。

🛠 **方法**：提出基于气压计辅助的姿态估计架构，利用气压高度测量提供垂直运动信息，并设计两种非线性观测器：一种是与互补滤波器级联的确定性Riccati观测器，另一种是在SO(3)×R²流形上演化的统一观测器，融合IMU、气压计和磁力计数据。

✨ **主要贡献**：首次将气压计作为轻量级辅助传感模态用于姿态估计，在均匀可观测条件下实现几乎全局渐近稳定（AGAS）或局部指数稳定（LES），并通过仿真和实际飞行数据验证了其在最小传感器配置下的可靠性与有效性。
