# Reinforcement Learning for the Full Strawberry Harvesting Process: Obstacle Separation, Detachment, and Placement

- arXiv: https://arxiv.org/abs/2607.14708v1
- arXiv ID: 2607.14708

---

🔍 **核心问题**：解决草莓采摘过程中因严重遮挡和植物结构变形导致的复杂接触动力学问题，实现障碍分离、果实摘取和放置的完整流程自动化。

🛠 **方法**：提出基于策略驱动的强化学习框架，结合启发式阶段协调，将采摘任务建模为序列决策问题；采用共享交互感知策略生成笛卡尔运动，并通过分层架构融合高层策略与低层阻抗控制，同时利用零样本仿真到现实迁移技术。

✨ **主要贡献**：在仿真和真实实验中分别达到89.7%和82.0%的成功率，验证了交互感知采摘行为可有效迁移至不同结构的机器人平台，且随遮挡程度增加，执行时间从12.99秒升至21.73秒，反映了对复杂交互的适应能力。
