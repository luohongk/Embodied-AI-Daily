# HumanHalo -- Safe and Efficient 3D Navigation Among Humans via Minimally Conservative MPC

- arXiv: https://arxiv.org/abs/2510.17525v3
- arXiv ID: 2510.17525

---

🔍 **核心问题**：现有机器人导航方法多局限于简化2D人群环境，未能充分处理人体动态复杂性，导致在3D空间中与人类共存的导航安全性与效率不足。

🛠 **方法**：提出HumanHalo框架，基于模型预测控制（MPC）结合可达性安全理论，仅约束初始控制输入以保证安全，同时利用数据驱动模型预测人体运动，实现3D微型飞行器（MAV）在人群中的高效导航。

✨ **主要贡献**：创新性地将安全约束仅作用于初始控制输入，避免过度保守，并通过仿真与真实实验验证，该方法在目标导航和人形跟踪任务中比基线方法更高效、可靠，且可推广至其他机器人平台。
