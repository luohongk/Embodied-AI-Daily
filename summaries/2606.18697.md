# Stealthy World Model Manipulation via Data Poisoning

- arXiv: https://arxiv.org/abs/2606.18697v1
- arXiv ID: 2606.18697

---

🔍 **核心问题**：基于模型的学习代理在微调世界模型时，可能受到数据投毒攻击，导致学习到的动态模型被操纵，进而破坏下游规划性能。  
🛠 **方法**：提出SWAAP框架，分两阶段实施攻击：首先通过一阶双层优化（基于转移梯度定理）识别一个有害但接近干净动态的目标世界模型；然后通过隐身约束梯度匹配，仅修改少量微调转移目标，使训练梯度引导受害模型朝向该目标，同时用预测误差正则化保持投毒数据接近自然误差。  
✨ **主要贡献**：首次提出针对世界模型的两阶段数据投毒攻击，在多种连续控制任务中显著降低性能，且能规避非自适应残差/CUSUM/TRIM等防御，揭示了世界模型适应管道的实际脆弱性。
