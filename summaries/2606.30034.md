# Scalable Intention Sharing for ETSI VAMs

- arXiv: https://arxiv.org/abs/2606.30034v1
- arXiv ID: 2606.30034

---

🔍 **核心问题**：在密集的V2X（车联网）环境中，现有符合ETSI标准的机动协调方法（如轨迹向量传输）在邻居数量增多和预测时间延长时，通信与计算开销过大，扩展性差。

🛠 **方法**：从意图共享角度重新审视机动协调，分析三种ETSI兼容编码（轨迹向量、N边形、不确定性椭圆）的复杂度与CPU开销，并采用扩展卡尔曼滤波器（EKF）生成短时预测，将预测结果编码为不确定性椭圆以表示预期机动。

✨ **主要贡献**：证明不确定性椭圆编码相比轨迹向量可将计算复杂度降低一个数量级，且消息大小恒定；基于真实自行车GNSS轨迹的评估表明，该方法在密集V2X环境中能实现可靠的多秒预测，同时保持良好扩展性。
