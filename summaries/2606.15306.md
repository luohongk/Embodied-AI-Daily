# LatentGym: A Testbed For Cross-Task Experiential Learning With Controllable Latent Structure

- arXiv: https://arxiv.org/abs/2606.15306v1
- arXiv ID: 2606.15306

---

🔍 **核心问题**：现有框架无法提供可控的共享潜在结构，难以衡量智能体在跨任务序列中是否真正从经验中学习并提升表现。

🛠 **方法**：提出LatentGym测试平台，每个环境围绕一个控制任务间结构的真实潜在变量构建，并设计指标分离探索（收集潜在信息）与利用（使用信息）行为。

✨ **主要贡献**：揭示了前沿模型在跨任务适应中的失败原因，验证了后训练能提升泛化能力并分析其来源，同时展示了任务间反馈等设计选择对训练动态的影响，为研究LLM智能体的跨任务经验学习提供了可控基础。
