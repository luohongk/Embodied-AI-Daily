# Unified Gradient Projection: Language-Balanced Continual Learning for Multilingual Low-Resource ASR

- arXiv: https://arxiv.org/abs/2607.11163v1
- arXiv ID: 2607.11163

---

🔍 **核心问题**：在多语言低资源ASR场景中，微调Whisper等预训练模型时，主流语言会主导优化过程，导致跨任务干扰和灾难性遗忘。

🛠 **方法**：提出统一梯度投影（UGP），通过语言平衡回放生成参考梯度，在统一投影空间中约束参数更新，均衡各语言贡献，并融合梯度级投影与数据级回放以增强稳定性与可塑性。

✨ **主要贡献**：UGP有效减少主流语言偏差，提升跨语言稳定性，在Whisper-large-v3上实现接近零的平均遗忘，显著缓解灾难性遗忘。
