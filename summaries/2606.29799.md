# The CRISTAL Method: Neurosymbolic analysis from AI-synthesized world models

- arXiv: https://arxiv.org/abs/2606.29799v1
- arXiv ID: 2606.29799

---

🔍 **核心问题**：解决投资分析等复杂分析任务中，传统LLM代理在数值推理、不确定性感知和可重复性方面的不足，以及人类分析师受认知偏差和注意力限制的困境。

🛠 **方法**：提出CRISTAL神经符号框架，通过自然语言先验知识构建动态可解释的概率程序，结合统计模型合成、连续学习和主动学习，实现全贝叶斯推理（含不确定性量化与预算感知数据采集），并利用LLM进行代码合成与模型持续优化。

✨ **主要贡献**：在合成股票分类任务中，CRISTAL仅用5个样本和5秒预算即达到贝叶斯最优精度，而最先进LLM在输入数据和计算量高一个数量级时仍仅约40%准确率，显著提升了自动化分析的效率、可靠性和可解释性。
