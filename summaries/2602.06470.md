# Improve Large Language Model Systems with User Logs

- arXiv: https://arxiv.org/abs/2602.06470v3
- arXiv ID: 2602.06470

---

🔍 **核心问题**：如何利用真实用户交互日志中嘈杂、非结构化的反馈信号，持续改进大语言模型系统，克服数据稀缺和计算成本效益递减的挑战。

🛠 **方法**：提出UNO框架，首先将用户日志蒸馏为半结构化规则和偏好对，然后通过查询与反馈驱动的聚类处理数据异质性，最后量化模型先验知识与日志数据之间的认知差距，自适应过滤噪声并构建主经验与反思经验模块。

✨ **主要贡献**：在真实部署场景下，UNO显著优于检索增强生成（RAG）和基于记忆的基线方法，实现了最先进的效果与效率，并开源了代码。
