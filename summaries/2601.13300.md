# OI-Bench: An Option Injection Benchmark for Evaluating LLM Susceptibility to Directive Interference

- arXiv: https://arxiv.org/abs/2601.13300v2
- arXiv ID: 2601.13300

---

🔍 **核心问题**：评估大语言模型（LLM）在选择题界面中受误导性指令干扰（如社交暗示、威胁框架等）的脆弱性。

🛠 **方法**：提出“选项注入”基准测试方法，在标准多选题中嵌入一个包含误导性指令的额外选项，构建包含3000道题（覆盖知识、推理、常识）和16种指令类型的OI-Bench基准，并测试12个LLM的鲁棒性。

✨ **主要贡献**：系统揭示了LLM在选择题界面中普遍存在对指令干扰的脆弱性，并分析了不同模型间的鲁棒性差异，同时探索了推理时提示与后训练对齐等缓解策略。
