# PWM-ArtGen: Part World Model for Articulated Object Generation

- arXiv: https://arxiv.org/abs/2607.02045v1
- arXiv ID: 2607.02045

---

🔍 **核心问题**：从单张图像生成可动3D物体时，现有方法难以准确预测其内部运动结构，要么依赖静态图像缺失动态关系，要么通过两步估计导致误差累积，且标注数据规模有限影响泛化能力。

🛠 **方法**：提出统一的部分世界模型PWM-ArtGen，通过联合学习视觉动态与运动参数的分布，并利用动作扩散与图像扩散的独立时间步耦合，实现无标注数据的视觉分支协同训练；同时构建了1.97万对无运动标注的逼真部分级图像数据集。

✨ **主要贡献**：在静止状态下显著超越现有基线方法，并展现出对分布外物体的强零样本泛化能力，解决了数据稀缺和两步误差累积问题。
