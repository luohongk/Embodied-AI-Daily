# StereoGS: Sparse-View 3D Gaussian Splatting via Stereo Priors

- arXiv: https://arxiv.org/abs/2606.30545v1
- arXiv ID: 2606.30545

---

🔍 **核心问题**：3D高斯泼溅（3DGS）在稀疏视角设置下因几何约束不足而严重过拟合，现有单目深度先验方法存在尺度模糊和跨视角不一致问题。

🛠 **方法**：提出StereoGS框架，通过优化过程中构建虚拟立体对并利用基础立体模型强制绝对尺度和双目一致性，引入立体深度正则化；同时设计梯度感知透明度衰减策略动态抑制冗余高斯体，并结合一致性感知密集初始化。

✨ **主要贡献**：在LLFF、DTU、Mip-NeRF360和Blender数据集上，稀疏视角设置下达到最先进性能，且不增加额外推理开销。
