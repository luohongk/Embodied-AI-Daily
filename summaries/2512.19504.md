# FusionNet: Physics-Aware Representation Learning for Multi-Spectral and Thermal Data via Trainable Signal-Processing Priors

- arXiv: https://arxiv.org/abs/2512.19504v2
- arXiv ID: 2512.19504

---

🔍 **核心问题**：现有遥感监测水泥生产设施的方法依赖热红外信号，易受背景热源干扰，且无法捕捉持续的环境变化，导致检测精度不足。

🛠 **方法**：提出FusionNet框架，通过中间层多光谱数据融合，将可训练的差分信号处理先验嵌入卷积层，结合混合池化和更大感受野，集成热红外与短波红外数据，并利用地质SWIR比值特征检测土壤性质变化。

✨ **主要贡献**：在SWIR比值数据集上达到90.6%的最高准确率，比最强单模态模型提升1.1%；DGCNN相比传统CNN精度提高4.1-6.8%；发现ImageNet预训练会降低TIR和SWIR性能，强调跨光谱任务需模态感知训练。
