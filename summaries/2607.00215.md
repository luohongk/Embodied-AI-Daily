# ELMP: Efficient Learning for Motion Planning via Analytical Policy Gradients

- arXiv: https://arxiv.org/abs/2607.00215v1
- arXiv ID: 2607.00215

---

🔍 **核心问题**：神经运动规划器（NMPs）在新环境中适应时，通常需要重新收集大量专家数据集，计算成本过高。

🛠 **方法**：提出ELMP框架，通过自监督微调直接优化策略，利用可微运动学层结合密集碰撞、目标到达和平滑性目标，替代专家数据生成，并引入点云显式编码工具几何结构。

✨ **主要贡献**：将单样本适应成本降低约两个数量级，在未见环境中将成功率从57.3%（零样本）提升至89.8%，平均成功率84.8%，冷启动延迟比经典方法低数个数量级，并在真实机器人上验证。
