## 0. 概览（Concise Summary）

- **主题（Topic）**：无人机路径规划、计算几何、组合优化。具体问题为：使用有限续航能力的多架无人机，对位于一条直线上的线段集合进行全覆盖巡检，并最小化最大完工时间（makespan）。
- **问题（Problem）**：在无人机电池续航有限（即单次飞行距离有上限L）的约束下，如何为多架无人机规划一组从基站出发并返回的回路（tours），使得所有线段被完全覆盖，并且所有无人机中飞行总距离最长的那个（即makespan）最小化。该问题被证明是强NP难的。
- **方法（Method）**：
    1.  **理论证明**：通过将问题归约到经典的“有限划分问题”（Finite Partition Problem），严格证明了2架无人机情况下的该问题是强NP难的。
    2.  **近似算法**：提出一个基于贪心策略的算法（G2D-algorithm）。该算法首先利用单无人机问题的多项式时间精确解（来自[Bereg et al., 2024]）生成一组最优的回路集合，然后将这些回路贪心地分配给多架无人机，以平衡各无人机的总飞行距离。
    3.  **算法改进**：在贪心算法基础上，提出了“切割策略”（Cutting Strategy）和“延长策略”（Enlarging Strategy）来进一步优化解的质量。
- **创新（Innovation）**：
    1.  **复杂性理论贡献**：首次证明了在连续空间（非离散化）中，多无人机（k≥2）沿直线巡检线段的最小化最大完工时间问题是强NP难的。这填补了该领域从单无人机多项式可解到多无人机NP难的理论空白。
    2.  **实用近似算法**：提出了一个简单、高效且具有理论保证的常数因子近似算法，并通过实验证明其在实际场景中能达到接近最优的性能。
- **意义（Significance）**：该研究为太阳能电厂、管道、电力线等线性基础设施的无人机巡检提供了理论基础和高效算法。它证明了在更复杂的场景下（如多无人机、连续路径规划），问题本质上是困难的，但通过巧妙的算法设计，仍然可以获得高质量的实用解。其“自底向上”（从单线到多线）的建模思路也为解决更复杂的优化问题提供了借鉴。
- **一句话总结**：本文证明了使用有限续航无人机巡检直线线段以最小化最大完工时间的问题是强NP难的，并提出了一个高效的贪心近似算法及其改进版本，在理论和实验上均表现出色。

## 1. 研究动机（Motivation）

- **解决什么问题**：本文旨在解决一个实际的工业巡检优化问题：如何高效地调度多架电池续航有限的无人机，对一组位于同一直线上的线段（如太阳能电厂中的集热管）进行全覆盖检查，并使得整个任务完成时间（即所有无人机中飞行时间最长的那个）最短。
- **为什么重要**：
    - **实际应用价值**：在太阳能电厂、石油/天然气管道、高压输电线等场景中，人工巡检成本高、效率低、风险大。使用无人机进行自动巡检是迫切需求。最小化任务完成时间（makespan）意味着能更快地发现故障（如破损的玻璃管），从而减少停机损失，提高电厂运营效率。
    - **理论意义**：该问题属于“弧路径问题”（Arc Routing Problem, ARP）的一个变种。与经典ARP不同，无人机可以直线飞行，并且可以任意点进入和退出线段，这使得问题成为一个连续的优化问题，比离散图上的问题更复杂。研究其计算复杂性，有助于理解这类问题的本质。
- **挑战在哪里**：
    - **连续性与离散性的矛盾**：无人机可以在线段上任意点进入和退出，这导致解空间是无限的，无法直接使用经典的图论算法。
    - **多机协同与负载均衡**：需要将巡检任务（即一系列回路）分配给多架无人机，并确保每架无人机的总飞行距离不超过其电池续航，同时最小化最繁忙无人机的飞行时间（makespan）。这是一个负载均衡问题。
    - **计算复杂性**：作者证明了即使是2架无人机、所有线段都在一条直线上的简化版本，该问题也是强NP难的，意味着不存在高效的精确算法。

## 2. 创新点（Innovation）

- **关键创新1：理论上的复杂性证明**。
    - **新贡献**：严格证明了“2-Min-Makespan”问题（即2架无人机，最小化最大完工时间）是**强NP难**的。
    - **与现有方案的不同**：已有工作[Bereg et al., 2024]证明了单无人机版本可以在多项式时间内精确求解。本文的工作首次揭示了从单机到多机（k≥2）时，问题性质发生了根本性变化，从“易解”变为“难解”。这为后续研究划定了理论边界。
    - **为什么能奏效**：作者巧妙地构造了一个从“Minmax Partition Problem”（一个已知的强NP难问题）到“2-Min-Makespan”问题的多项式时间归约。通过精心设计线段的位置和长度，使得任何对2-Min-Makespan问题的解都直接对应一个对Minmax Partition问题的解，从而证明了后者的难度不低于前者。

- **关键创新2：实用且高效的近似算法**。
    - **新贡献**：提出了一个名为G2D（Greedy tour distribution for two drones）的贪心算法及其改进版本。
    - **与现有方案的不同**：现有方法（如[Campbell et al., 2018, 2021]）通常将连续问题离散化为图上的问题，需要大量采样点，计算量大且精度受限。本文的算法直接处理连续问题，利用单机精确解作为“积木”，然后通过贪心分配和局部优化策略来求解多机问题，更加简洁高效。
    - **为什么能奏效**：
        1.  **模块化设计**：将问题分解为“生成最优回路”和“分配回路”两个子问题。前者有精确解，后者用贪心算法，保证了整体效率。
        2.  **理论保证**：算法具有常数因子近似比，并且该近似比在特定条件下（如需要较多回路时）非常接近1（即接近最优）。
        3.  **局部优化**：提出的“切割”和“延长”策略能够有效调整回路分配，进一步平衡各无人机的负载，提升解的质量。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文的整体架构是“理论证明 + 算法设计 + 实验验证”。

1.  **问题形式化**：首先严格定义了`k-Min-Makespan Problem`，即给定一组位于直线上的线段、k架相同无人机、一个基站B和最大航程L，目标是找到k组回路，使得所有线段被覆盖，并且所有无人机中总飞行距离的最大值（makespan）最小。
2.  **复杂性分析**：通过归约证明2架无人机的情况是强NP难的。
3.  **算法设计**：
    - **基础算法（G2D）**：
        1.  使用[Bereg et al., 2024]的算法，为单架无人机计算出覆盖所有线段的最优回路集合 `T = {t1, t2, ..., tm}`。这个集合的总长度最小。
        2.  将回路集合 `T` 中的回路，按照长度，贪心地分配给两架无人机（T1和T2）。每次分配时，将当前回路分配给当前总长度较小的那架无人机。
    - **改进算法**：在G2D的基础上，迭代地应用两种局部搜索策略：
        - **切割策略（CS）**：如果一架无人机的某个回路过长，可以将其“切断”成两个更短的回路，并将其中一个分配给另一架无人机，以平衡负载。
        - **延长策略（ES）**：如果一架无人机的某个回路较短，而另一架无人机的相邻回路较长，可以将短回路“延长”以覆盖更多区域，同时相应缩短长回路，从而平衡负载。

### 3.2 关键算法与数学推导（公式用 LaTeX，行内用 $...$，独立公式用 $$...$$）

**1. 问题归约与NP-hard证明核心**

论文的核心归约是将一个已知的强NP难问题——**Minmax Partition Problem**（问题4.2）——归约到**2-Min-Makespan Problem**。

- **Minmax Partition Problem**：给定一个包含 $2n$ 个正整数的集合 $S = \{s_1, s_2, ..., s_{2n}\}$，找到一个大小为 $n$ 的子集 $A \subset S$，使得 $A$ 中元素之和 $M_2$ 与 $S \setminus A$ 中元素之和 $M_1$ 的差最小，且 $M_2 \ge M_1$。

- **构造过程**：给定一个 $S$ 的实例，作者构造了一个线段集合 $\alpha_S$，使得求解 $\alpha_S$ 上的 2-Min-Makespan 问题等价于求解 $S$ 上的 Minmax Partition 问题。
    - **关键构造**：通过一系列步骤（Step 1-4），为 $S$ 中的每个元素 $s_i$ 构造一个唯一的线段 $a_i$。这些线段的位置和长度被精心设计，使得：
        1.  任何一条长度不超过 $L$ 的回路**最多只能覆盖一个完整的线段**（引理4.10）。
        2.  覆盖第 $i$ 个线段的最优回路的长度 $s'_i$ 与 $s_i$ 成线性关系：$s'_i = K s_i + C$，其中 $K$ 和 $C$ 是构造中确定的常数（命题4.4）。
        3.  因此，2-Min-Makespan 问题的最优解，本质上就是将 $2n$ 个长度为 $s'_i$ 的回路分配给两架无人机，使得两架无人机的总长度尽可能相等。这正好等价于 Minmax Partition 问题。

**2. 近似算法（G2D）及其理论分析**

- **G2D算法**：将单机最优回路集 $T = \{t_1, ..., t_m\}$ 贪心地分配给两架无人机。
- **定理5.1**：设 $\{T_1^*, T_2^*\}$ 为最优解，$\{T_1, T_2\}$ 为G2D算法得到的解，且 $\ell(T_1) \le \ell(T_2)$，$\ell(T_1^*) \le \ell(T_2^*)$。则存在 $a \in [0, 1]$ 使得 $\ell(T_2) = \ell(T_1) + aL$，并且：
    - (a) $\ell(T_1) + \frac{aL}{2} \le \ell(T_2^*) \le \ell(T_1) + aL$
    - (c) 近似比 $\Delta = \frac{\ell(T_2)}{\ell(T_2^*)} \le \frac{\Gamma + 2}{\Gamma + 1}$，其中 $\Gamma = \frac{2\ell(T_1)}{aL}$。

    **推导**：
    由 (a) 可知，$\ell(T_2^*) \ge \ell(T_1) + \frac{aL}{2}$。
    因此，近似比 $\Delta = \frac{\ell(T_2)}{\ell(T_2^*)} \le \frac{\ell(T_1) + aL}{\ell(T_1) + \frac{aL}{2}}$。
    令 $\Gamma = \frac{2\ell(T_1)}{aL}$，则 $\ell(T_1) = \frac{\Gamma aL}{2}$。代入上式得：
    $$\Delta \le \frac{\frac{\Gamma aL}{2} + aL}{\frac{\Gamma aL}{2} + \frac{aL}{2}} = \frac{\Gamma + 2}{\Gamma + 1}$$
    这个公式表明，当 $\ell(T_1)$ 很大（即需要很多回路）时，$\Gamma$ 很大，$\Delta$ 趋近于1，算法性能很好。

### 3.3 模型、训练与数据集

- **模型**：本文没有训练任何机器学习模型。其核心是一个**组合优化模型**，即`k-Min-Makespan Problem`。
- **数据集**：实验数据是**随机生成的**。生成方式如下：
    - 在区间 $[m, M] = [-1000, 1000]$ 内随机生成 $N$ 个线段。
    - 线段长度分布具有不同的**同质性**（Coefficient of Variation, CV），分为高（0.2）、中（0.6）、低（0.8）三档，但均值固定为10。
    - 基站位置 $(X_b, Y_b)$ 随机生成，$Y_b$ 在 $[1, 10000]$ 之间。
    - 无人机最大航程 $L$ 在 $[L_{min}, L_{max}]$ 之间随机选取，其中 $L_{min}$ 是到达最远线段并返回的最小距离，$L_{max}$ 是覆盖整个线段区间所需的最小距离。
    - 总共生成了987个不同的场景。

### 3.4 实验设置与结果

- **实验设置**：
    - **对比基准**：将问题离散化（将线段细分为小段），然后使用**混合整数线性规划（MILP）** 求解器（Gurobi）求解离散化后的问题，将其解作为“近似最优解”。
    - **评估指标**：**近似因子** $\Delta = \ell(T_2) / \ell(T_2^*)$，其中 $\ell(T_2)$ 是算法得到的最大完工时间，$\ell(T_2^*)$ 是MILP求解器得到的最优解。
    - **算法对比**：对比了基础G2D算法和改进算法（G2D + 切割/延长策略）。

- **实验结果**：
    1.  **整体性能**：
        - G2D算法：平均近似因子 $\bar{\Delta} = 1.094$，标准差0.11。10.84%的案例达到最优（$\Delta=1$）。
        - 改进算法：平均近似因子 $\bar{\Delta}_I = 1.04$，标准差0.045。16.31%的案例达到最优。改进算法的性能显著优于基础算法（t检验，p<0.001）。
    2.  **影响因素分析**：
        - **线段同质性**：对算法性能无显著影响。
        - **无人机航程L**：对算法性能有显著影响。当L较小时（需要更多回路），G2D算法的近似因子更接近1，性能更好。改进算法在所有L下都表现稳定，且远优于基础G2D。
        - **交互效应**：L和同质性对改进算法有交互作用，但对基础算法没有。

    - **图8**：展示了近似因子随L变化的关系。G2D算法的近似因子随L增大而恶化，而改进算法则没有这种趋势，表现更稳健。

### 3.5 技术细节

- **单机精确解**：论文引用了[Bereg et al., 2024]的工作，该工作使用动态规划算法在 $O(n^2 + nm)$ 时间内解决了单机问题，其中 $n$ 是线段数，$m$ 是回路数。
- **MILP公式**：为了获得对比基准，论文将连续问题离散化，并建立了一个MILP模型（公式7-11）。该模型通过二进制变量 $z^d_{ij}$ 表示无人机 $d$ 是否使用了一个从点 $a_i$ 到 $a_j$ 的回路，并通过约束确保所有子区间都被覆盖且回路长度不超过 $L$。目标是最小化所有无人机总长度的最大值 $T$。
- **改进算法的实现**：切割和延长策略的实现需要在线性时间内找到所有可能的候选操作，并选择能最大程度降低makespan的那个。算法迭代进行，直到无法进一步改进。
