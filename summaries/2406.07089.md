# RS-Agent: Automating Remote Sensing Tasks through Intelligent Agent

- arXiv: https://arxiv.org/abs/2406.07089v4
- arXiv ID: 2406.07089

---

🔍 **核心问题**：现有多模态大语言模型在遥感任务中仅能执行基础指令，无法处理需要多源数据融合、精细空间推理和领域专业知识的真实场景。

🛠 **方法**：提出RS-Agent智能体，包含中央控制器、动态工具包、解决方案空间和知识空间四个模块，并引入任务感知检索和双路径检索增强生成技术，支持光学与SAR影像等多模态数据。

✨ **主要贡献**：在9个数据集和18项遥感任务上，任务规划准确率超95%，在场景分类、目标计数和遥感视觉问答等任务中显著优于现有最先进模型，验证了将大语言模型推理与遥感专业知识结合的可行性。
