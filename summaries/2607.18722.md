# Stale but Stable: Staleness-Adaptive Trust Regions for Stabilizing Asynchronous Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.18722v2
- arXiv ID: 2607.18722

---

🔍 **核心问题**：异步强化学习中，由于策略滞后、引擎延迟和专家混合路由等因素导致的“陈旧性”问题，使得训练与推理之间的策略不匹配加剧，而PPO裁剪仅作为采样代理而非全局约束，无法有效控制高陈旧性更新。

🛠 **方法**：提出**陈旧性自适应信任域（SAT）**，利用分离的采样对数比作为陈旧性代理，通过基于陈旧性的核缩放识别批次内高失配尾部，并仅收缩PPO名义区间中符号选择的端点，从而在保留普通令牌基线行为的同时，对新拦截的外向带施加更保守的更新。

✨ **主要贡献**：证明了SAT相对于PPO的局部区间包含性和逐点悲观性，展示了自适应规则如何在异质陈旧性下重塑更新几何；在基于Qwen3-30B-A3B-Base的异步RL设置中，SAT-GSPO w/ R3在AIME24 avg@8上达到最优（滞后1时35.83，滞后8时34.79），表明对齐裁剪区间与陈旧性异质性可有效稳定异步RL。
