# CREG: Compass Relational Evidence Graph for Characterizing Directional Structure in VLM Spatial-Reasoning Attribution

- arXiv: https://arxiv.org/abs/2603.20475v4
- arXiv ID: 2603.20475

---

🔍 **核心问题**：现有视觉-语言模型（VLM）的归因热图仅显示模型关注区域，但无法区分这些证据是否真正基于查询的空间关系，还是仅仅反映了图像的整体布局。

🛠 **方法**：提出CREG（罗盘关系证据图），一种无需训练的诊断框架，将词元级归因转换为以参考点为中心的罗盘分布，并测量其方向对齐程度，同时与几何控制进行显式比较。

✨ **主要贡献**：实验表明，纯框几何方法的方向对齐误差比最佳模型归因方法低28.4至34.4度，揭示当前归因方法的方向结构有限且常与图像布局混合；此外，任务准确率的提升并不保证方向归因的改善，为评估空间推理改进提供了可控协议。
