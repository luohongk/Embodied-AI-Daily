# Physics-enhanced reinforcement learning for real-time optimal control of dynamical systems

- arXiv: https://arxiv.org/abs/2607.16177v1
- arXiv ID: 2607.16177

---

🔍 **核心问题**：解决强化学习在控制高维、参数化动态系统时样本效率低、需要大量环境交互的问题。

🛠 **方法**：提出物理增强强化学习（PEARL）范式，采用演员-伴随算法，利用自动微分计算短时策略梯度，并通过神经网络近似未来回报的伴随敏感性，减少环境交互次数并缓解长期梯度不稳定。

✨ **主要贡献**：在非定常流中的参数化导航问题中，PEARL显著优于现有强化学习算法，样本效率高，能泛化至多场景，并支持高维状态与动作空间，无需低维表示或多智能体策略。
