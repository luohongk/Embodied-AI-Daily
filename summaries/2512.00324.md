# MILE: A Mechanically Isomorphic Exoskeleton Data Collection System with Fingertip Visuotactile Sensing for Dexterous Manipulation

- arXiv: https://arxiv.org/abs/2512.00324v3
- arXiv ID: 2512.00324

---

🔍 **核心问题**：现有灵巧手操作数据采集系统存在运动重定向不准确、采集效率低、缺乏高分辨率指尖触觉感知等问题，限制了模仿学习的有效性。

🛠 **方法**：提出MILE系统，通过从人手到外骨骼再到机械手的机械同构设计，实现关节位置的一对一映射，避免非线性重定向；外骨骼基于人体测量学设计，机械手集成紧凑指尖视觉触觉模块，采集多模态数据（触觉、RGB-D图像、关节位置）。

✨ **主要贡献**：外骨骼多关节平均绝对角度误差低于1度，遥操作成功率平均提升64%；加入指尖触觉观测后，相比纯视觉基线成功率再提升25%，验证了数据集的高保真度和实用性。
