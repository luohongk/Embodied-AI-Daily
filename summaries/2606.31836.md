# RoboTacDex: A Dexterous Visual-Tactile-Action Dataset for Humanoid Manipulation

- arXiv: https://arxiv.org/abs/2606.31836v1
- arXiv ID: 2606.31836

---

🔍 **核心问题**：解决机器人学习领域缺乏大规模、多模态、多样化灵巧操作数据集的问题，尤其是针对类人机器人双臂与灵巧手协同完成复杂任务的数据需求。

🛠 **方法**：基于公开的人形机器人Unitree G1，构建包含6k条轨迹、覆盖19个任务、23种技能和22个物体的数据集，同步采集多视角RGB-D图像、触觉反馈及语义标注，并开发改进的多相机同步系统实现毫秒级数据记录。

✨ **主要贡献**：提供了首个面向人形机器人灵巧操作的视觉-触觉-动作多模态数据集，包含需双臂与灵巧手完成的挑战性任务，并通过三种模仿学习模型验证了数据集的有效性与泛化能力。
