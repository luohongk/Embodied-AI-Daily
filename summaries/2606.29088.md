# Diff-Based Code Corruption using LLMs for Large-Scale Bugfix Benchmarking

- arXiv: https://arxiv.org/abs/2606.29088v2
- arXiv ID: 2606.29088

---

🔍 **核心问题**：现有的大语言模型（LLM）代码修复基准测试规模小、bug类型单一且与现实修复实践脱节，导致评估结果不可靠且缺乏代表性。

🛠 **方法**：提出MegaBugFix基准，利用LLM通过生成代码变更（diff）的方式，从正确的Python程序中合成12,629个有bug的程序，避免注入过于简单或未修改程序的常见问题。

✨ **主要贡献**：构建了大规模、更贴近真实场景的bug修复基准，评估13个开源模型发现其在MegaBugFix上性能显著低于现有基准，揭示了现有评估中隐藏的模型缺陷。
