# Learning Bipedal Walking for Humanoids with Current Feedback

- arXiv: https://arxiv.org/abs/2303.03724v2
- arXiv ID: 2303.03724

---

🔍 **核心问题**：解决人形机器人在从仿真环境迁移到真实硬件时，因执行器扭矩跟踪不准确导致的“仿真-现实差距”（sim2real gap）问题。  
🛠 **方法**：在仿真中人为降低扭矩跟踪精度，并利用真实机器人执行器的电流反馈训练策略，结合前馈策略架构与针对性动力学随机化，实现零样本迁移。  
✨ **主要贡献**：首次在真实HRP-5P人形机器人上成功部署端到端强化学习策略实现双足行走，无需复杂记忆网络，且在崎岖地形上鲁棒性优于传统模型控制器。
