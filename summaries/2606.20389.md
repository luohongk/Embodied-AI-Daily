# CoLI: A Reproducible Platform for Continuum Robot Learning via Monolithic 3D Printing and Isomorphic Teleoperation

- arXiv: https://arxiv.org/abs/2606.20389v1
- arXiv ID: 2606.20389

---

🔍 **核心问题**：连续体机器人因制造复杂、运动学建模困难及缺乏直观控制接口，导致可重复性差，限制了其研究与应用。  
🛠 **方法**：采用多材料3D打印制造整体柔性臂（减少组装），并通过同构遥操作界面实现直接执行器级映射（无需运动学建模），同时支持模仿学习自主控制。  
✨ **主要贡献**：提出开源、可重复的连续体机器人平台，简化制造流程并消除运动学奇异性，实验验证其在操作任务中的有效性，为算法开发与基准测试提供基础。
