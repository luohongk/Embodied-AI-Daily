# Dual-Mind World Models: A General Framework for Learning in Dynamic Wireless Networks

- arXiv: https://arxiv.org/abs/2510.24546v1
- arXiv ID: 2510.24546

---

🔍 **核心问题**：现有基于强化学习的无线网络方法数据效率低、缺乏泛化能力，难以应对高动态和长期规划需求，尤其在毫米波V2X场景中优化信息年龄（CAoI）时表现不足。

🛠 **方法**：提出一种受认知心理学启发的双思维世界模型框架，包含模式驱动的系统1和逻辑驱动的系统2，通过端到端可微分的想象轨迹学习链路调度，并在无观测间隔内仍能高效决策。

✨ **主要贡献**：在基于Sionna的真实物理信道仿真中，该方法相比现有强化学习基线及仅含系统1的世界模型，显著提升了数据效率，并展现出对未知环境的强泛化与适应能力。
