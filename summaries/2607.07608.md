# Dual Latent Memory in Vision-Language-Action Models for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2607.07608v1
- arXiv ID: 2607.07608

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型基于马尔可夫假设，仅依赖当前观测预测动作，难以处理需要长期依赖的机器人操作任务。

🛠 **方法**：提出LaMem-VLA框架，通过四个组件（策展人、搜寻者、压缩器、编织者）将历史经验重构为短期和长期潜在记忆令牌，并与当前观测和指令在连续潜在空间中融合，使记忆直接参与VLA推理。

✨ **主要贡献**：首次将历史经验以潜在记忆形式无缝嵌入VLA推理过程，在SimplerEnv和LIBERO基准上验证了该方法在长时域任务中的优越性。
