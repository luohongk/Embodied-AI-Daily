# MRS: Multi-Resolution Skills for HRL Agents

- arXiv: https://arxiv.org/abs/2505.21410v2
- arXiv ID: 2505.21410

---

🔍 **核心问题**：分层强化学习中，管理者的子目标表示缺乏可达性和时间距离约束，导致在需要敏捷性的任务上性能下降，且最优子目标距离因任务和状态而异。

🛠 **方法**：提出多分辨率技能（MRS），学习多个固定时间跨度的目标预测模块，并联合训练一个元控制器，根据当前状态动态选择最合适的模块。

✨ **主要贡献**：MRS在DeepMind控制套件、Gym-Robotics和长时域AntMaze任务上持续优于固定分辨率基线，显著缩小了HRL与非HRL最先进方法之间的性能差距。
