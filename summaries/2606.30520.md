# Staged Hybridisation for Visual Quantum Reinforcement Learning via Knowledge Distillation

- arXiv: https://arxiv.org/abs/2606.30520v1
- arXiv ID: 2606.30520

---

🔍 **核心问题**：视觉量子强化学习面临高维观测、不稳定优化与受限变分量子电路难以联合训练的挑战，直接端到端训练效果不佳。

🛠 **方法**：提出分阶段混合策略，先训练经典视觉教师模型并冻结其编码器作为特征接口，再通过知识蒸馏将教师策略行为压缩到下游小型量子兼容头（经典或VQC基）中。

✨ **主要贡献**：在CartPole Pixels和Acrobot Pixels任务上，浅层VQC头通过分阶段蒸馏能习得非平凡视觉控制行为，角度编码VQC头保持接近教师性能，振幅编码头实现极致紧凑性，将视觉QRL转化为紧凑头学习问题，绕开标准端到端RL循环。
