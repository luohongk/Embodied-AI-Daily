# Improvement of Robot's Simultaneous Localization and Mapping Using an Effective Transformation to Achieve Linear Model

- arXiv: https://arxiv.org/abs/2606.28475v1
- arXiv ID: 2606.28475

---

🔍 **核心问题**：解决基于扩展卡尔曼滤波（EKF）的同步定位与地图构建（SLAM）中因模型非线性和线性化误差导致的算法发散问题。  
🛠 **方法**：通过引入简单指南针并应用有效变换，将非线性状态空间模型转化为线性模型，再使用原始卡尔曼滤波（KF）处理，提出LMKF SLAM方法。  
✨ **主要贡献**：LMKF SLAM在精度、收敛性和计算复杂度上显著优于现有方法（尤其是EKF-SLAM），且对传感器噪声和参数变化具有更强稳定性，实验验证了其优越性。
