# TRACE: A Two-Channel Robust Attribution Watermark via Complementary Embeddings for LLM-Agent Trajectories

- arXiv: https://arxiv.org/abs/2607.08400v1
- arXiv ID: 2607.08400

---

🔍 **核心问题**：LLM智能体通过经销商触达用户时，经销商可能篡改或替换模型，导致归属争议。现有水印方法无法在经销商完全读写轨迹日志（工具调用、观察和动作记录）的对抗环境下可靠地验证归属。

🛠 **方法**：提出TRACE，一种双通道鲁棒归属水印。它叠加两个互补通道：选择通道（基于局部内容密钥，无失真采样器选择动作，抗删除）和计数通道（基于日志骨架密钥，控制每组决策记录数，抗重写）。

✨ **主要贡献**：首个在动作选择上无失真、在删除下自同步、在重写下无条件不变的水印。在ToolBench和ALFWorld上，TRACE保持与无水印智能体相同的成功率，选择通道在长程轨迹上检测z值接近100，70%步骤删除下仍可检测，计数通道在任意强度LLM重写下完全不变。
