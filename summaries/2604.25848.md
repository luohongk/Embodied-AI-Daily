# A Distributionally Robust Reinforcement Learning Framework for Constrained Urban EV Dispatch

- arXiv: https://arxiv.org/abs/2604.25848v2
- arXiv ID: 2604.25848

---

🔍 **核心问题**：解决城市级电动网约车车队在充电桩容量和电网馈线限制下，面对空间相关的不确定需求和行驶时间时，如何联合优化派单、重定位和充电决策的问题。

🛠 **方法**：将问题建模为六边形网格半马尔可夫决策过程，结合离散动作（服务、重定位、充电）与连续充电功率；通过带掩码和温度退火的演员网络输出高层意图，再经时间受限的滚动混合整数线性规划严格满足物理约束；采用基于Wasserstein-1模糊集和图形对齐马氏距离的分布鲁棒软演员-评论家算法，利用Kantorovich-Rubinstein对偶与原始-对偶风险预算更新进行鲁棒优化。

✨ **主要贡献**：提出分布鲁棒强化学习框架PD-RSAC，在纽约出租车数据的大规模仿真中实现最高净利润122万美元（对比基线58-70万美元），且保持零馈线违规，显著优于贪婪、单智能体及多智能体基线方法。
