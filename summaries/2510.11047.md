# Benchmarking Deep Learning Models for Laryngeal Cancer Staging Using the LaryngealCT Dataset

- arXiv: https://arxiv.org/abs/2510.11047v2
- arXiv ID: 2510.11047

---

🔍 **核心问题**：喉癌影像研究缺乏标准化公开数据集，导致深度学习模型开发难以复现和比较。  
🛠 **方法**：构建LaryngealCT数据集（1029例CT扫描），采用弱监督参数搜索提取喉部感兴趣区域，并测试6种3D深度学习架构（如3D CNN、ResNet系列等）进行早期/晚期及T4分类。  
✨ **主要贡献**：提供首个标准化喉癌CT基准数据集，3D CNN在早期/晚期分类中达最优性能（准确率0.854，F1-macro 0.841），并集成可解释性工具（GradCAMpp）验证模型解剖合理性，为临床决策支持奠定可复现基础。
