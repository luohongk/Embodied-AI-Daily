# Language Models as Interfaces, Not Oracles: A Hybrid LLM-ML System for Pediatric Appendicitis

- arXiv: https://arxiv.org/abs/2606.19183v1
- arXiv ID: 2606.19183

---

🔍 **核心问题**：解决大型语言模型直接作为诊断引擎时对提示词敏感、输出不稳定且可能产生错误结果的问题，同时克服结构化机器学习模型难以处理临床自由文本的局限。

🛠 **方法**：提出ClaMPAPP混合系统，利用LLM从临床叙事文本中提取受模式约束的特征，经确定性合理性检查后，将验证后的特征输入XGBoost分类器进行诊断预测。

✨ **主要贡献**：在两个独立儿科阑尾炎队列中，ClaMPAPP取得最优诊断性能且漏诊率最低；相比端到端LLM，其敏感性与特异性更稳定，对文本顺序变化更鲁棒，验证了“LLM作为接口、ML作为预测器”的可审计临床决策支持设计。
