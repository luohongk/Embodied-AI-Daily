# Improving Robotic Imitation Learning via Trajectory Standardization

- arXiv: https://arxiv.org/abs/2606.22907v1
- arXiv ID: 2606.22907

---

🔍 **核心问题**：机器人模仿学习中，人类演示轨迹因操作速度不均、间歇停顿和动作密度不一致而存在噪声和时间不规则性，传统时间均匀降采样无法有效消除这些非均匀性，导致数据质量下降并阻碍策略学习。

🛠 **方法**：提出信息标准化轨迹重采样（ISR），通过将轨迹映射到信息调制的黎曼流形上，并执行测地线等距参数化，利用速度和加速度范数构建信息强度场，从而在相邻点之间强制实现近似相等的信息距离。

✨ **主要贡献**：在三个真实世界操作任务上，ISR相比基线时间均匀3倍降采样将任务成功率提升约25%，对不同操作员采集的数据集保持鲁棒性，并同时减少数据集大小和训练成本。
