# parallelcbf: A composable safety-filter and auditability framework for tensor-parallel reinforcement learning

- arXiv: https://arxiv.org/abs/2605.15509v1
- arXiv ID: 2605.15509

---

🔍 **核心问题**：现有框架无法统一大规模并行无人机仿真、约束强化学习基准测试和安全屏障函数合成工具，缺乏端到端的安全约束训练与可审计性支持。

🛠 **方法**：提出ParallelCBF框架，集成张量并行无人机环境、硬门控CBF安全过滤器、分片行为克隆到强化学习流水线，并通过可组合API实现操作审计（预注册、监控注册表、故障取证和数据集审计）。

✨ **主要贡献**：首个统一上述能力的框架，提供四层可组合API、双屏障CBF参考实现、39项属性测试在1.67秒内完成，并通过审计层自动阻止未达预注册收敛标准的训练阶段，防止退化模型传播，确保可复现性。
