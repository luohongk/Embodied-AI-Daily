# Deconfounded Lifelong Learning for Autonomous Driving via Dynamic Knowledge Spaces

- arXiv: https://arxiv.org/abs/2603.14354v3
- arXiv ID: 2603.14354

---

🔍 **核心问题**：端到端自动驾驶系统在持续学习中面临灾难性遗忘、跨场景知识迁移困难，以及传感器噪声等混杂因素导致驾驶意图与观测数据间产生虚假关联的问题。

🛠 **方法**：提出DeLL框架，利用狄利克雷过程混合模型构建动态轨迹知识空间和隐式特征知识空间，并通过因果推断中的前门调整机制，以知识空间为中介变量消除混杂偏差，同时采用非自回归进化轨迹解码器实现规划。

✨ **主要贡献**：首次将非参数贝叶斯与因果推断结合用于自动驾驶持续学习，实现知识空间自适应扩展与增量更新；提出基于Bench2Drive的评估协议与指标；在CARLA闭环仿真中验证了该方法在抗遗忘、新场景适应性和整体驾驶性能上的显著提升。
