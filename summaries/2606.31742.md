# FedXDS: Leveraging Model Attribution Methods to counteract Data Heterogeneity in Federated Learning

- arXiv: https://arxiv.org/abs/2606.31742v1
- arXiv ID: 2606.31742

---

🔍 **核心问题**：联邦学习中客户端数据分布存在统计异质性，导致模型性能下降，而现有方法难以精准识别并共享关键数据以缓解这一问题。

🛠 **方法**：提出FedXDS框架，利用可解释AI（XAI）中的特征归因技术（如传播式归因），通过单次反向传播识别任务相关特征，指导客户端间选择性数据共享，并引入度量隐私技术保护敏感信息。

✨ **主要贡献**：首次将XAI应用于联邦学习以对抗数据异质性，实验表明该方法在不同客户端数量和异质性设置下均能实现更高精度和更快收敛，同时提供理论隐私保证并抵御成员推断和特征反转攻击。
