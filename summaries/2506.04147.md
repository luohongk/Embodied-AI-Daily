# SLAC: Safe and Efficient Real-Robot Reinforcement Learning via Unsupervised Simulation Pre-Training

- arXiv: https://arxiv.org/abs/2506.04147v5
- arXiv ID: 2506.04147

---

🔍 **核心问题**：解决高自由度机器人（如双臂移动操作平台）在真实世界中难以安全、高效地通过强化学习自主获取控制策略的问题，同时避免传统仿真到现实迁移方法因“现实差距”导致的脆弱性。

🛠 **方法**：提出SLAC框架，利用低精度仿真器通过无监督技能发现方法预训练一个与任务无关的潜在动作空间，该空间设计注重时间抽象性、解耦性和安全性；随后将该空间作为动作接口，结合新型离策略强化学习算法，让机器人在真实环境中自主学习下游任务。

✨ **主要贡献**：在双臂移动操作任务上达到最先进性能，无需任何演示或手工行为先验，仅需不到一小时的现实交互即可学会包含接触的全身控制任务，显著提升了真实世界机器人强化学习的可行性与效率。
