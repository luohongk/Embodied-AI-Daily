# RigPI: Dynamic Parameter Identification of Rigid Body via VLM-Seeded Differentiable Simulation

- arXiv: https://arxiv.org/abs/2606.25212v1
- arXiv ID: 2606.25212

---

🔍 **核心问题**：解决机器人操作中物体（刚体及多连杆刚体）的物理参数（惯性、摩擦等）难以从真实交互中准确识别的问题，克服传感噪声、建模误差和先验知识不足的挑战。

🛠 **方法**：提出RigPI框架，集成视觉语言模型（VLM）提供语义先验和约束搜索空间，结合可微物理模拟器的梯度信息，采用两阶段优化策略，从力-力矩测量和运动观测中高效稳定地识别动态参数。

✨ **主要贡献**：首次将VLM与可微模拟结合用于刚体动态参数识别，通过两阶段优化提升噪声鲁棒性和物理合理性；在真实机器人实验中，对旋转和滑动关节物体实现精确参数估计，并成功复现操作轨迹，验证了系统的有效性和鲁棒性。
