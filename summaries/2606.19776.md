# Occ-VLM: Occupancy Grounded Vision Language Model for Indoor Scene Understanding

- arXiv: https://arxiv.org/abs/2606.19776v1
- arXiv ID: 2606.19776

---

🔍 **核心问题**：现有3D场景理解的视觉语言模型（VLM）通常依赖显式3D输入（如点云）或额外3D编码器，导致3D几何感知与2D语义在结构上分离，难以形成统一的3D视觉语言表征。

🛠 **方法**：提出Occ-VLM框架，仅使用带位姿的RGB图像和单一2D视觉编码器，通过重建3D场景占用作为几何先验，将前景2D标记与3D空间关联，并由大语言模型（LLM）解码实现统一场景理解。

✨ **主要贡献**：在多视角占用预测任务上达到最先进性能，同时在3D视觉问答和密集描述基准上取得与使用3D输入的VLM相当的结果，实现了精确几何感知与鲁棒视觉语言推理的统一。
