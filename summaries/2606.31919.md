# MVP-Nav: Multi-layer Value Map Planner Navigator

- arXiv: https://arxiv.org/abs/2606.31919v1
- arXiv ID: 2606.31919

---

🔍 **核心问题**：解决仅依赖RGB视觉的零样本物体目标导航中，因缺乏深度信息导致的物理不确定性及语义与物理空间错位问题。

🛠 **方法**：提出MVP-Nav框架，利用3D基础模型将2D语义实例投影为3D有向边界框以重建物理占用，并设计多层价值图（MVM）将语义优先级与几何信息统一到共享代价空间，实现物理约束下的几何规划。

✨ **主要贡献**：在零样本物体导航基准上显著超越现有无深度方法，达到最优性能，验证了结构化物理先验可有效补偿主动深度传感器的缺失。
