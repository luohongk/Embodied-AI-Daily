# Information-Theoretic Adaptive Cooling for Deterministic MPPI via Entropy Feedback

- arXiv: https://arxiv.org/abs/2607.14245v1
- arXiv ID: 2607.14245

---

🔍 **核心问题**：在确定性MPPI控制中，如何设计有效的温度冷却策略以恢复最优解，避免现有开环调度方法效率低、鲁棒性差的问题。

🛠 **方法**：提出信息论自适应冷却（ITAC）框架，利用重要性权重的香农熵作为在线反馈信号动态调节温度，实现基于采样状态的冷却速率自适应调整。

✨ **主要贡献**：证明了ITAC方案渐近收敛到确定性最优解，推导出防止权重过早坍缩的临界熵阈值；在非光滑信号时序逻辑运动规划任务中，ITAC显著提升采样效率并比现有基线方法更快收敛，且保持MPPI的无导数特性。
