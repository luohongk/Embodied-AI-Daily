# ECHO: Prune To Act, Trace To Learn With Selective Turn Memory In Agentic RL

- arXiv: https://arxiv.org/abs/2606.31650v2
- arXiv ID: 2606.31650

---

🔍 **核心问题**：在长程语言智能体强化学习中，有限上下文窗口导致历史信息被压缩后，无法追溯哪些早期证据支持最终答案，造成策略行动与结果导向学习的信用分配不匹配。

🛠 **方法**：提出ECHO框架，通过选择性回合记忆实现可追溯的上下文重建。它将每个环境回合压缩为带源索引的紧凑记录，在重建策略上下文时选择有用记录，并利用所选源索引将正向结果信用路由到最终轨迹段、重用证据、记忆发现及记忆选择动作。

✨ **主要贡献**：在BrowseComp-Plus上达到43.4%的留出准确率，优于GRPO（28.9%）和滚动摘要基线SUPO（36.1%），且使用更少的回合和轨迹量。训练后的策略在密集和MoE骨干网络上，对多目标问答、代码生成和深度信息检索基准的零样本泛化能力均有提升。
