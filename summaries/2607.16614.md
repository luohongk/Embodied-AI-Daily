# An Indoor Navigation System for the Visually Impaired based on UWB Positioning and D* Lite Path Planning Algorithm

- arXiv: https://arxiv.org/abs/2607.16614v1
- arXiv ID: 2607.16614

---

🔍 **核心问题**：解决视障人士在GPS信号缺失的室内环境中难以精准导航和规避动态障碍的问题。  
🛠 **方法**：采用超宽带（UWB）定位技术实现高精度室内定位，并集成D* Lite路径规划算法，支持动态障碍下的快速路径重规划。  
✨ **主要贡献**：系统在复杂室内空间中实现了低延迟、高可靠性的导航，显著提升了视障用户的安全性与灵活性。
