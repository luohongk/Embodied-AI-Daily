# KineBench: Benchmarking Embodied World Models via IDM-Free Kinematic Grounding

- arXiv: https://arxiv.org/abs/2607.19876v1
- arXiv ID: 2607.19876

---

🔍 **核心问题**：现有具身世界模型评估依赖逆动力学模型（IDM）从生成视频中提取动作，但IDM对分布外数据脆弱，导致模型误差与提取误差难以区分。

🛠 **方法**：提出KineBench基准，采用无IDM的显式运动学管线，通过级联视觉基础模型直接从视频帧提取6D末端执行器位姿，并在物理模拟器中执行闭环验证，同时引入SPARC和可操作性指数等运动学指标。

✨ **主要贡献**：首次实现无IDM的闭环评估，消除归因歧义；在20个操作任务上构建四类渐进测试套件，揭示具身视频生成中任务复杂度相关的非线性缩放规律，为数据扩展策略提供实证指导。
