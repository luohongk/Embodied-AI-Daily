# E-CoDrive: A Co-Simulation Framework for Testing Energy-Critical Driving Scenarios

- arXiv: https://arxiv.org/abs/2607.04803v1
- arXiv ID: 2607.04803

---

🔍 **核心问题**：现有自动驾驶研究主要关注安全性，忽视了能耗与可持续性等非功能属性，缺乏评估复杂交通场景下电动汽车能否在电池耗尽前完成行程的测试方法。

🛠 **方法**：提出E-CoDrive框架，集成能耗模型、微观交通模拟器与高保真驾驶模拟器，实现可复现的闭环驾驶协同仿真，并基于Autoware自动驾驶软件栈进行测试验证。

✨ **主要贡献**：首次将能耗关键场景纳入自动驾驶系统测试，实验表明不同交通条件会导致车辆能耗显著差异，为能量感知的场景化测试提供了开源工具与可复现方案。
