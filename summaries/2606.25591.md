# WOLF-VLA: Whole-Body Humanoid Optimal Locomotion Framework for Vision-Language-Action Learning

- arXiv: https://arxiv.org/abs/2606.25591v1
- arXiv ID: 2606.25591

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在机器人操作任务中表现良好，但缺乏适用于全身、高接触性人形机器人运动的数据集和动态一致性演示，难以直接通过自然语言指令生成运动策略。

🛠 **方法**：提出WOLF-VLA框架，结合全身最优控制运动合成与大规模多模态数据集，训练VLA模型，使其能够从自然语言指令、第一人称视觉和关节轨迹中学习人形机器人运动策略。

✨ **主要贡献**：构建了包含六类运动任务、参数化环境变化的动态可行数据集，训练出的VLA策略展现出强推理能力和对初始条件变化的鲁棒性，并公开数据集、模型和仿真环境，为指令驱动的人形运动控制提供可复现基准。
