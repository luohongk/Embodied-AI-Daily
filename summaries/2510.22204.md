# Human-Inspired Neuro-Symbolic World Modeling and Logic Reasoning for Interpretable Safe UAV Landing Site Assessment

- arXiv: https://arxiv.org/abs/2510.22204v3
- arXiv ID: 2510.22204

---

🔍 **核心问题**：解决无人机在非结构化环境中安全着陆点评估的可靠性问题，现有方法在数据分布变化时性能下降且缺乏可解释性。

🛠 **方法**：提出NeuroSymLand框架，将轻量级分割模型构建的概率语义场景图与离线合成、人工优化的符号安全规则相结合，实现感知驱动的世界建模与逻辑推理的分离。

✨ **主要贡献**：在72个仿真和硬件在环场景中完成61次成功评估，优于基线方法（37-57次），同时提供可解释的推理过程和极低的边缘计算开销。
