# Diagnosing Semantic Handoff Failures in Agent-Orchestrated Vision-Language-Action Skill Composition

- arXiv: https://arxiv.org/abs/2607.06256v1
- arXiv ID: 2607.06256

---

🔍 **核心问题**：解决机器人长时域任务中，由视觉-语言-动作（VLA）技能组合执行时，因技能间语义交接不明确导致的失败问题。

🛠 **方法**：在BEHAVIOR-1K环境中构建基于智能体编排的执行框架，使用π₀.₅技能检查点、多视角视觉-语言模型验证，以及技能边界状态与链式状态两种初始分布对比评估。

✨ **主要贡献**：发现单个技能在干净状态下成功率高达77-100%，但组合执行时因链式状态导致频繁失败，揭示了技能间交接、目标定位和底层控制的关键瓶颈，为诊断长时域任务失败提供了可操作的分析方法。
