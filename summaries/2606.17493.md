# When Robots Sleep: Offline Skill Consolidation for Shared-Policy Robot Learning

- arXiv: https://arxiv.org/abs/2606.17493v1
- arXiv ID: 2606.17493

---

🔍 **核心问题**：机器人在长期部署中学习新技能时，容易破坏已有技能的共享策略结构，导致技能间可靠性下降（即“技能耦合崩溃”）。

🛠 **方法**：提出“睡眠机器人”框架，采用“醒-睡”机制——清醒时学习新技能，睡眠时离线巩固共享策略，利用冻结的技能记忆（如冻结评论家与状态缓冲区）通过纳什议价合并梯度，并结合自适应锚定与局部兴奋性实现稳定巩固。

✨ **主要贡献**：在Meta-World MT5上，平均成功率提升64%，成对可靠性提升2倍；在SurgicAI上，平均成功率和反向迁移优于持续模仿基线，成对可靠性保持竞争力。
