# Optimal any-angle path planning in static and dynamic environments

- arXiv: https://arxiv.org/abs/2607.00065v1
- arXiv ID: 2607.00065

---

🔍 **核心问题**：解决在静态和动态障碍物环境下，实现任意角度路径规划的最优解，并提升计算效率。  
🛠 **方法**：提出两种加速技术——椭圆前向扩展（限制搜索空间）和视野方法（替代传统视线检测），并引入反向扫描与前向扫描整合技术，基于此开发Zeta*和Zeta*-SIPP算法。  
✨ **主要贡献**：Zeta*在静态环境中性能与最优算法Anya相当，且更易扩展；Zeta*-SIPP在动态环境中比现有最优规划器TO-AA-SIPP快20倍以上，实现了统一的高效最优任意角度路径规划框架。
