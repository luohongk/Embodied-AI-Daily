# T2T-VICL: Cross-Task Visual In-Context Learning via Implicit Text-Driven VLMs

- arXiv: https://arxiv.org/abs/2511.16107v3
- arXiv ID: 2511.16107

---

🔍 **核心问题**：现有视觉上下文学习（VICL）方法要求演示样本与查询属于同一视觉任务，但实际应用中常出现任务不匹配的情况，因此需要研究跨任务VICL，即演示与查询任务不同时模型能否正确推理。

🛠 **方法**：提出T2T-VICL框架，利用大型教师VLM生成跨任务对的视觉变化与任务差异的结构化描述，构建隐式跨任务关系数据集，再蒸馏到轻量学生VLM中，使其能从任务A的演示和任务B的查询生成内容相关的隐式文本提示，引导冻结的图像编辑VLM，并引入基于分数的推理策略排序候选结果。

✨ **主要贡献**：在12个低级视觉任务和20余个跨任务对上验证，T2T-VICL相比固定提示显著提升任务对齐性，并常改善图像保真度，揭示了跨任务VICL的潜力与局限。
