# SymbOmni: Evolving Agentic Omni Models via Symbolic Concept Learning

- arXiv: https://arxiv.org/abs/2607.12042v1
- arXiv ID: 2607.12042

---

🔍 **核心问题**：现有视觉生成模型存在“永久新手”问题，无法累积学习经验、自主进化，导致组合泛化能力差且知识保留效率低。

🛠 **方法**：提出SymbOmni智能全模态模型，通过符号概念学习实现累积进化。核心是“符号概念盒”这一可优化记忆模块，将底层操作抽象为可复用的符号工作流指令，并采用归纳-转导循环（经验抽象为符号概念，再组合解决新任务），训练通过基于语言反馈的言语反向传播实现持续自我改进。

✨ **主要贡献**：在图像质量和任务成功率上显著超越现有智能体系统及闭源模型（如Nano Banana、GPT-Image-1）；将令牌消耗降低40%以上且保持生成质量；在多个在线学习基准上实现累积增益，创下新最优结果。
