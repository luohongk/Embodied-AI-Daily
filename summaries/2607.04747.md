# MergeSurv: Merging-Based Continual Learning for Survival Analysis on Whole-Slide Images

- arXiv: https://arxiv.org/abs/2607.04747v1
- arXiv ID: 2607.04747

---

🔍 **核心问题**：现有生存分析模型通常针对每种癌症队列独立训练，导致在千兆像素级全切片图像（WSI）上进行持续适应时计算成本高昂，且难以避免灾难性遗忘。

🛠 **方法**：提出MergeSurv框架，基于模型合并的持续学习方法。对每个任务独立微调病理视觉-语言基础模型，随后将学习到的参数顺序合并为统一模型，无需存储历史训练数据；并探索了两种推理策略：统一模型（OFA）和投票专家聚合（VEA）。

✨ **主要贡献**：在四个TCGA队列上，MergeSurv优于朴素微调及代表性的正则化与重放式持续学习方法，有效缓解灾难性遗忘，为计算病理学中可扩展且保护隐私的持续学习提供了新方向。
