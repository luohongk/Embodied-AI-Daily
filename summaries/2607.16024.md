# DiffTestGen: Change-Directed LLM-Based Testing for Exposing Behavioral Differences

- arXiv: https://arxiv.org/abs/2607.16024v1
- arXiv ID: 2607.16024

---

🔍 **核心问题**：软件版本迭代中，如何自动生成测试用例以暴露新旧版本间的行为差异，确保代码变更符合开发者预期。

🛠 **方法**：提出DiffTestGen方法，结合静态调用图分析与项目文档识别有效测试入口，引导大语言模型（LLM）生成测试；通过迭代优化新提出的“联合覆盖率”指标（覆盖新旧版本修改代码），并向LLM反馈覆盖率信息以提升测试效果。

✨ **主要贡献**：在463个PR数据集上，DiffTestGen在78.2%的PR中暴露行为差异，平均联合覆盖率达90.7%；相比基线方法，额外暴露99个PR的行为差异，代码覆盖率分别提升12.5%和15.6个百分点；与Testora回归检测器集成后，能发现现有方法遗漏的回归缺陷。
