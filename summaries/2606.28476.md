# FADA: Few-Shot Domain Adaptation via Dynamics Alignment for Humanoid Control

- arXiv: https://arxiv.org/abs/2606.28476v1
- arXiv ID: 2606.28476

---

🔍 **核心问题**：人形机器人控制中，目标域（如地形、负载变化）与源域动力学不匹配导致控制精度下降，现有方法难以在少量目标域数据下高效适应。

🛠 **方法**：提出FADA三阶段框架，先训练一个使用特权信息的“先知”策略，再通过DAgger蒸馏为可部署的规划器-逆动力学模型（Planner-IDM），部署时仅用约2分钟目标域数据微调IDM，无需最优演示或奖励。

✨ **主要贡献**：仅需少量目标域交互数据即可实现高效动力学对齐，在仿真和真实人形机器人上超越现有上下文适应和端到端方法，成功执行高精度全身任务。
