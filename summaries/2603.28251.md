# DiffAttn: Diffusion-Based Drivers' Visual Attention Prediction with LLM-Enhanced Semantic Reasoning

- arXiv: https://arxiv.org/abs/2603.28251v3
- arXiv ID: 2603.28251

---

🔍 **核心问题**：解决智能车辆中驾驶员视觉注意力预测不准确的问题，以提升对潜在危险的感知和驾驶安全性。  
🛠 **方法**：提出DiffAttn框架，将视觉注意力预测建模为条件扩散-去噪过程，采用Swin Transformer编码器、特征融合金字塔解码器，并集成大语言模型（LLM）增强语义推理。  
✨ **主要贡献**：在四个公开数据集上达到最先进性能，超越多种基线方法，并支持可解释的驾驶员场景理解，有助于改善人机交互和风险感知。
