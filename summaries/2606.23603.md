# MORL-A2C: Multi-Objective Reinforcement Learning Reranker for Optimizing Healthiness in MOPI-HFRS

- arXiv: https://arxiv.org/abs/2606.23603v1
- arXiv ID: 2606.23603

---

🔍 **核心问题**：现有食品推荐系统（如MOPI-HFRS）虽能通过帕累托优化平衡偏好、健康与多样性，但依赖静态单步权衡，无法捕捉饮食决策的序列性，导致健康优化不足。

🛠 **方法**：提出MORL-A2C，将推荐建模为K步重排序问题，利用冻结的图神经网络嵌入，结合优势演员-评论家算法和标量化奖励（相关性+健康），并通过行为克隆初始化策略。

✨ **主要贡献**：在修正评估基线后，MORL-A2C以轻微牺牲排序质量（Recall@20下降2%，NDCG@20下降3%）为代价，显著提升健康对齐度（H-Score@20从46%升至70%），验证了策略驱动的序列优化能有效导航多目标食品推荐中的健康-偏好权衡。
