# MCNav: Memory-Aware Dynamic Cognitive Map for Zero-shot Goal-oriented Navigation

- arXiv: https://arxiv.org/abs/2605.19594v1
- arXiv ID: 2605.19594

---

🔍 **核心问题**：在复杂环境中进行零样本实例级目标导航时，现有方法过度关注探索新区域，忽视对已探索区域信息的深度利用，导致目标被遗漏或误识别时导航失败。

🛠 **方法**：提出MCNav框架，构建动态认知地图存储已探索区域中相关对象的可查询信息，并引入两种记忆感知策略：目标重新验证（纠正匹配错误）和遗漏目标重新探索（根据上下文线索估计目标存在概率），同时结合黑名单机制和双重检查机制提升稳定性。

✨ **主要贡献**：在HM3Dv1和HM3Dv2数据集上的三个任务中取得最先进性能，尤其在实例级目标导航任务上表现突出，有效解决了零样本导航中已探索区域信息利用不足的问题。
