## 0. 概览（Concise Summary）

- **主题（Topic）**：多无人水面艇（USV）协同目标环绕跟踪。这是一个结合了多智能体系统、状态估计与协同控制的机器人学问题。
- **问题（Problem）**：在缺乏外部定位（如GPS）的拒止环境下，如何仅依靠USV自身搭载的有限传感器（如测距、测向、惯性测量单元），实现多艘USV对非合作目标的协同环绕跟踪。核心挑战在于：1) 感知的异构性（对合作队友可测距，对非合作目标只能测向）；2) 传感器噪声的非高斯性（如UWB测距的拖尾噪声）；3) 状态估计的可观性与控制目标（形成均匀圆形编队）之间的耦合与冲突。
- **方法（Method）**：提出一个异构感知与协同控制框架。1) **异构感知**：对队友使用主动测距（UWB）+ 惯性位移，对目标使用被动测向（视觉）。2) **状态估计**：对队友间的相对定位，采用最大相关熵卡尔曼滤波器（MCKF）以抑制非高斯噪声；对USV与目标间的相对状态估计，采用伪线性卡尔曼滤波器（PLKF）。3) **协同控制**：设计一个基于耦合振荡器的编队控制器，该控制器不仅能引导USVs形成均匀圆形编队，还能主动产生满足“持续激励（PE）”条件的相对运动，从而保证滤波器的可观性。
- **创新（Innovation）**：1) 首次在无外部定位的多USV协同环绕问题中，明确并利用了感知的异构性（合作vs非合作），提出了更贴合实际的传感器配置方案。2) 将MCKF引入基于测距和位移的USV间相对定位，有效解决了UWB传感器非高斯噪声导致的估计精度下降问题。3) 设计了一个“一石二鸟”的控制器，同时实现了编队控制和可观性保证，并给出了严格的理论证明。
- **意义（Significance）**：该工作为在GPS拒止等复杂环境下，多USV系统执行自主、鲁棒的协同目标跟踪任务（如海上搜救、环境监测）提供了可行的理论框架和工程解决方案。它解决了传统方法中假设理想定位或同质感知的局限性，提升了系统的实用性和鲁棒性。
- **一句话总结**：本文提出了一种基于异构感知（测距+测向）和双目标控制（编队+可观性）的框架，使多艘无人艇在无外部定位条件下，仅靠自身传感器即可实现对非合作目标的协同环绕跟踪。

## 1. 研究动机（Motivation）

- **核心问题**：多USV协同目标环绕跟踪任务中，一个关键前提是每个USV需要知道自身、队友以及目标的精确位置。然而，在GPS拒止等缺乏外部定位基础设施的环境中，这个前提无法满足。
- **现有方案的不足**：
    1.  **理想化假设**：许多现有研究假设所有USV和目标的位置都是完美已知的（例如通过GPS），这在拒止环境中不成立。
    2.  **同质感知假设**：一些基于有限传感器（仅测距或仅测向）的研究，忽略了系统中“合作”与“非合作”的内在不对称性。它们假设对所有对象（队友和目标）使用同一种感知方式，这在实际应用中存在问题：
        - **仅用测向（如视觉）**：对非合作目标很合适（被动、隐蔽），但用于队友间定位时，受限于视场角（FOV）和非视距（NLOS）问题，不可靠。
        - **仅用测距（如UWB）**：对队友间定位很合适（全向、穿透性好），但无法单独用于估计目标状态（需要结合其他信息）。
    3.  **噪声模型简化**：基于测距的定位研究中，很多工作假设UWB测量噪声是高斯分布的。然而，实际UWB噪声具有非高斯、重尾（heavy-tailed）的特性，这会导致传统卡尔曼滤波器（KF）性能严重下降。
    4.  **可观性与控制的冲突**：在部分观测（如仅有测距或测向）下，状态估计必须满足可观性条件（即相对运动要足够“丰富”）。然而，编队控制的目标是让USVs稳定在一个相对静止的圆形编队上，这恰恰会削弱相对运动，导致可观性丧失。这是一个复杂的双重控制问题。

- **为什么重要**：解决这些问题对于将多USV协同技术从理论推向实际应用至关重要。海上搜救、环境监测、海域执法等任务都要求在无外部基础设施支持的环境下，系统能够自主、可靠地运行。

## 2. 创新点（Innovation）

本文的主要创新点可以概括为以下三点：

1.  **提出并实现了异构感知策略**：
    - **创新点**：首次在无外部定位的多USV协同环绕问题中，明确区分了与合作队友（USVs）和非合作目标（Target）的感知关系。针对前者，采用主动测距（UWB）+ 惯性位移；针对后者，采用被动测向（视觉）。
    - **与现有方案的不同**：不同于以往假设对所有对象使用同一种传感器（如全测距或全测向）的工作，该策略更贴合实际，充分利用了不同传感器的优势，规避了其劣势（如视觉的FOV限制，UWB对非合作目标的无效性）。
    - **为何奏效**：这种“扬长避短”的配置，使得系统在复杂环境下依然能获得可靠的多源信息，为后续的状态估计和控制奠定了基础。

2.  **引入最大相关熵卡尔曼滤波器（MCKF）用于USV间相对定位**：
    - **创新点**：将MCKF应用于基于测距和位移的USV间相对定位问题，以应对UWB传感器的非高斯、重尾噪声。
    - **与现有方案的不同**：传统KF或EKF基于最小均方误差（MMSE）准则，在高斯噪声下最优，但对重尾噪声中的异常值非常敏感。MCKF采用最大相关熵准则（MCC），通过高斯核函数自动降低异常值的权重，从而提供更鲁棒的估计。
    - **为何奏效**：MCC本质上是一种局部相似性度量，它对大误差（异常值）不敏感，因此能有效抑制非高斯噪声的影响，提高估计的准确性和稳定性。

3.  **设计了一个“双目标”协同控制器**：
    - **创新点**：设计了一个基于耦合振荡器的编队控制器，该控制器不仅能引导USVs形成并维持一个均匀的圆形编队（控制目标），还能主动产生满足“持续激励（PE）”条件的相对运动，从而保证MCKF和PLKF的可观性（估计目标）。
    - **与现有方案的不同**：以往工作要么假设PE条件自动满足而不加保证，要么将控制与估计分开设计。本文通过精心设计期望轨迹（由耦合振荡器生成），将两个目标统一在一个框架下，并给出了严格的理论证明。
    - **为何奏效**：控制器中的前馈项（$v^*_{io,k}$）是产生PE的关键。通过合理选择参数（如期望半径$r^*_{io}$和期望偏航率$\omega^*_i$），可以使前馈项产生的相对运动足够“丰富”，从而“压倒”编队控制项和未知目标运动带来的干扰，保证可观性。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文提出的整体框架如图3所示，是一个“感知-估计-控制”的闭环系统。

1.  **感知层**：每个USV $V_i$ 通过异构传感器获取数据：
    - **主动感知**：通过IMU获取自身位移 $\tilde{\delta}_{i,k}$；通过UWB传感器获取与所有其他USV $V_j$ 的相对距离 $\tilde{r}_{ij,k}$。
    - **被动感知**：通过视觉传感器（如单目相机）获取指向目标 $o$ 的方位向量 $\tilde{\mathbf{b}}_{io,k}$。
    - **通信**：USVs之间通过通信网络交换各自的位移 $\tilde{\delta}_{j,k}$ 和期望相位角 $\phi_{j,k}$。

2.  **估计层**：
    - **USV间相对定位**：利用MCKF，融合自身位移、接收到的队友位移以及相对距离测量，估计出与队友的相对位置 $\hat{\mathbf{p}}^+_{ij,k}$。
    - **USV-目标相对状态估计**：利用PLKF，融合自身位移和目标的方位测量，估计出与目标的相对位置 $\hat{\mathbf{p}}^+_{io,k}$ 和相对速度 $\hat{\mathbf{v}}^+_{io,k}$。

3.  **控制层**：
    - **期望轨迹生成**：通过耦合振荡器模型（公式28），USVs相互协商，生成各自在圆形编队上的期望相位角 $\phi_{i,k}$，进而计算出期望的相对位置 $\mathbf{p}^*_{io,k}$ 和 $\mathbf{p}^*_{ij,k}$。
    - **编队控制器**：基于估计的相对状态和期望的相对位置，设计一个位移式编队控制器（公式31-33），输出USV的期望速度 $v_{i,k}$ 和偏航率 $\omega_{i,k}$。
    - **避障模块**：集成一个基于向量场的避障控制器（公式34），在检测到障碍物时生成避障速度项 $v^s_{i,k}$，叠加到编队控制器上。

### 3.2 关键算法与数学推导

**1. USV间相对定位：MCKF**

- **核心思想**：利用连续时刻的测距和位移信息，通过余弦定理构建一个关于相对位置 $\mathbf{p}_{ij,k}$ 的伪线性测量模型。
- **伪线性测量模型**（公式11-12）：
  $$z_{ij,k} = \tilde{\boldsymbol{\delta}}^T_{ij,k} \mathbf{p}_{ij,k} + \mu_{z_{ij},k}$$
  其中 $z_{ij,k} = \frac{1}{2}(\tilde{r}^2_{ij,k} - \tilde{r}^2_{ij,k-1} + \|\tilde{\boldsymbol{\delta}}_{ij,k}\|^2)$，$\tilde{\boldsymbol{\delta}}_{ij,k} = \tilde{\boldsymbol{\delta}}_{i,k} - \tilde{\boldsymbol{\delta}}_{j,k}$。等效噪声 $\mu_{z_{ij},k}$ 具有对称、尖峰、重尾的特性。
- **MCKF迭代更新**（公式15-21）：
  - **预测**：$\hat{\mathbf{p}}^-_{ij,k} = \hat{\mathbf{p}}^+_{ij,k-1} + \tilde{\boldsymbol{\delta}}_{ij,k}$
  - **迭代更新**：通过固定点迭代求解后验估计 $\hat{\mathbf{p}}^+_{ij,k}$。核心在于计算一个修正的卡尔曼增益 $\bar{\mathbf{K}}_{ij,k}$，该增益通过高斯核函数 $G_\sigma(\cdot)$ 对异常值进行加权，从而降低其对估计结果的影响。
    $$\hat{\mathbf{p}}^{(s)}_{ij,k} = \hat{\mathbf{p}}^-_{ij,k} + \bar{\mathbf{K}}_{ij,k}(z_{ij,k} - \tilde{\boldsymbol{\delta}}^T_{ij,k}\hat{\mathbf{p}}^-_{ij,k})$$
    其中 $\bar{\mathbf{K}}_{ij,k}$ 的计算依赖于由核函数构建的权重矩阵 $\bar{\mathbf{C}}_{p_{ij},k}$ 和 $\bar{\mathbf{C}}_{z_{ij},k}$。

**2. USV-目标相对状态估计：PLKF**

- **核心思想**：利用方位向量 $\tilde{\mathbf{b}}_{io,k}$ 构建一个正交投影矩阵 $\mathbf{G}_{\tilde{\mathbf{b}}_{io,k}} = \mathbf{I} - \tilde{\mathbf{b}}_{io,k}\tilde{\mathbf{b}}^T_{io,k}$。由于真实方位 $\mathbf{b}_{io,k}$ 与相对位置 $\mathbf{p}_{io,k}$ 平行，因此 $\mathbf{G}_{\mathbf{b}_{io,k}}\mathbf{p}_{io,k} = \mathbf{0}$。
- **伪线性测量模型**（公式22-23）：
  $$\mathbf{z}_{io,k} \triangleq \mathbf{0}_{2\times1} = \mathbf{G}_{\tilde{\mathbf{b}}_{io,k}} \mathbf{p}_{io,k} + \boldsymbol{\mu}_{z_{io},k}$$
  这个模型将非线性的方位测量转化为一个关于状态的线性方程，使得标准卡尔曼滤波框架得以应用。
- **PLKF更新**（公式24-25）：
  - **预测**：$\hat{\mathbf{x}}^-_{io,k} = \mathbf{A} \hat{\mathbf{x}}^+_{io,k-1}$
  - **更新**：$\hat{\mathbf{x}}^+_{io,k} = \hat{\mathbf{x}}^-_{io,k} - \mathbf{K}_{io,k}\mathbf{H}_{io,k}\hat{\mathbf{x}}^-_{io,k}$，其中 $\mathbf{H}_{io,k} = [\mathbf{G}_{\tilde{\mathbf{b}}_{io,k}} \quad \mathbf{0}_{2\times2}]$。

**3. 可观性分析**

- **核心思想**：证明在提出的控制器下，USV间的相对速度 $\mathbf{v}_{ij,k}$ 和USV-目标的相对速度 $\mathbf{v}_{io,k}$ 满足持续激励（PE）条件，从而保证MCKF和PLKF的可观性。
- **PE条件**：一个向量序列 $\{\mathbf{v}_k\}$ 满足PE条件，当且仅当存在正整数 $N$ 和正定矩阵 $\lambda_1 \mathbf{I}, \lambda_2 \mathbf{I}$，使得对于所有 $l$，有：
  $$\lambda_1 \mathbf{I} \le \sum_{k=l}^{l+N-1} \mathbf{v}_k \mathbf{v}_k^T \le \lambda_2 \mathbf{I}$$
  这意味着该向量序列在 $N$ 步内能“张成”整个空间，即运动足够丰富。
- **引理1 & 2**：证明了如果期望相对速度 $\mathbf{v}^*_{ij,k}$ 和 $\mathbf{v}^*_{io,k}$ 满足PE条件，并且控制参数选择得当（使得 $\lambda^*_{ij,1} > \sqrt{N}(U_{vf_i} + U_{vf_j} + 2U_{v_o})$ 等），那么实际相对速度 $\mathbf{v}_{ij,k}$ 和 $\mathbf{v}_{io,k}$ 也满足PE条件。
- **定理1 & 2**：基于PE条件，证明了MCKF和PLKF系统的Gramian矩阵是正定的，从而证明了系统的可观性。例如，对于PLKF，其可观性Gramian矩阵为：
  $$\boldsymbol{\Phi}_{io,l} = \sum_{k=l}^{l+N-1} \mathbf{G}^T_{\mathbf{b}_{io,k}} \mathbf{G}_{\mathbf{b}_{io,k}} > 0$$
  这个条件要求方位向量 $\mathbf{b}_{io,k}$ 在 $N$ 步内不能是常数，即USV必须相对目标做非直线运动。

### 3.3 模型、训练与数据集

- **模型**：
    - **USV运动模型**：简化的离散时间“surge-yaw”运动学模型（公式1），忽略了水动力学等复杂因素。
    - **目标运动模型**：噪声驱动的双积分器模型（公式3），用于描述非合作目标的未知机动。
- **训练**：本文的方法不涉及离线训练。MCKF和PLKF都是在线递推滤波器，其参数（如核带宽 $\sigma$、噪声协方差矩阵）需要根据经验或传感器特性进行手动调整。
- **数据集**：
    - **仿真数据**：主要基于仿真生成的数据进行验证。仿真参数见表I。

- **实验设置**：通过100次蒙特卡洛仿真，评估不同算法在不同噪声模型下的性能。主要对比了MCKF与RLS、KF、AKF、Huber-RLS等算法。
- **主要结果**：

        - 在**真实UWB数据集**（图6）上，MCKF同样表现出最快的收敛速度和最小的估计误差，验证了其在实际场景中的有效性。

        - 论文声称，在提出的整体框架下，多USV系统能够成功形成并维持一个均匀的圆形编队，并实现对目标的跟踪。具体结果（如编队误差、定位误差等）在论文的图7-12等图表中展示（由于文本截断，未提供具体数值，但根据描述，性能良好）。

### 3.5 技术细节

- **MCKF核带宽选择**：$\sigma_1, \sigma_2$ 用于处理过程噪声（假设为高斯），设为较大值（9999），使MCKF在过程更新上接近标准KF。$\sigma_3$ 用于处理等效测量噪声（重尾），设为较小值（0.8），以增强对异常值的鲁棒性。
- **控制器参数设计**：为了满足可观性条件（引理1, 2），需要选择足够大的期望半径 $r^*_{io}$ 和期望偏航率 $\omega^*_i$，使得期望相对速度 $v^*_{io,k}$ 足够大，从而“压倒”编队控制项 $v^f_{i,k}$ 和目标速度 $v_{o,k}$ 的影响。这为
