# Model Predictive Path Integral PID Control for Learning-Based Path Following

- arXiv: https://arxiv.org/abs/2603.29499v2
- arXiv ID: 2603.29499

---

🔍 **核心问题**：传统MPPI控制直接优化高维控制输入序列，导致采样效率低、输入变化剧烈，且优化维度随预测时域增长。  
🛠 **方法**：提出MPPI-PID控制，利用MPPI在线优化PID增益（低维参数空间），而非直接优化控制输入序列，同时保留PID结构。  
✨ **主要贡献**：通过将高维输入优化转为低维增益优化，提升采样效率并生成更平滑的控制输入；在基于残差学习动力学模型的微型叉车路径跟踪任务中，相比固定增益PID和传统MPPI，跟踪性能更优且输入增量更小。
