# SparseGS: Sparse View Synthesis using 3D Gaussian Splatting

- arXiv: https://arxiv.org/abs/2312.00206v4
- arXiv ID: 2312.00206

---

🔍 **核心问题**：3D Gaussian Splatting在稀疏视角输入时，会出现重建质量下降，产生“漂浮物”和“背景塌陷”等伪影。

🛠 **方法**：提出SparseGS训练流程，引入深度先验、新型深度渲染技术和剪枝启发式方法减少漂浮物，并设计“未见视角正则化模块”缓解背景塌陷。

✨ **主要贡献**：在Mip-NeRF360、LLFF和DTU数据集上，仅用12张（无界场景）或3张（前向场景）输入图像即可实现高质量重建，同时保持快速训练和实时渲染能力。
