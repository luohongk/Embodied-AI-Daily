# ATHENA: Accelerated Multi-Task Heterogeneous Influence Functions for Robot Data Curation

- arXiv: https://arxiv.org/abs/2606.16208v1
- arXiv ID: 2606.16208

---

🔍 **核心问题**：在机器人模仿学习中，将影响函数扩展到十亿参数级别的视觉-语言-动作（VLA）模型时，面临计算开销大和多任务瓶颈的挑战。

🛠 **方法**：提出ATHENA框架，利用线性层梯度的Kronecker结构降低投影成本，并采用秩r随机截断近似来逼近稠密Hessian逆矩阵，实现约313.4倍的影响计算加速；同时设计全局与局部交互影响机制，平衡50个联合训练任务的数据筛选。

✨ **主要贡献**：在模拟和真实机器人实验中，仅使用50%的模拟演示数据和66.7%的真实任务数据，即可达到或超越全数据联合微调的性能，验证了其在十亿参数多任务VLA微调中数据筛选的有效性。
