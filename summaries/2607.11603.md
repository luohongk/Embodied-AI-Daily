# WarpMPC: Large-Batch MPC on GPU via ADMM with Unrolled $LDL^\top$ Factorization

- arXiv: https://arxiv.org/abs/2607.11603v1
- arXiv ID: 2607.11603

---

🔍 **核心问题**：如何在GPU上高效求解大批量（1万至10万以上）具有相同结构的序贯二次规划（SQP）迭代，以加速模型预测控制（MPC）的计算。

🛠 **方法**：提出WarpMPC工具箱，基于JAX和Warp实现。通过展开稀疏线性因子分解（如$LDL^\top$）和求解过程，优化交替方向乘子法（ADMM）求解器；采用优化内存布局、减少填充的分段展开因子分解以及依赖级别调度的回代，避免内存访问瓶颈并加速灵敏度计算。

✨ **主要贡献**：在非线性倒立摆、四旋翼和人形机器人基准测试中，实现每秒8000至25万次SQP迭代，吞吐量比基线方法提升3至25倍；在硬件实验中，仅用不到4分钟即可合成数据集并训练神经网络近似MPC，成功稳定纳米四旋翼飞行器。
