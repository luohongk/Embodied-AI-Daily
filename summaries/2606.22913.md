# Intend, Reflect, Refine: An Adaptive Multimodal Reflection Framework for Autonomous Driving

- arXiv: https://arxiv.org/abs/2606.22913v1
- arXiv ID: 2606.22913

---

🔍 **核心问题**：现有VLA模型直接生成最终轨迹，缺乏对未来后果的显式检查，导致在复杂动态环境中可靠性不足。

🛠 **方法**：提出IRR-Drive框架，先生成初步文本意图并预测未来语义BEV表示，构建“文本+BEV”双模态反思空间，使模型能自我修正初始意图，再生成最终轨迹；同时设计自适应反思奖励，根据场景复杂度动态调整推理模式。

✨ **主要贡献**：在NAVSIM基准上达到PDMS和EPDMS双指标最优；首次将自适应反思机制直接集成到规划框架中，实现基于场景复杂度的决策感知轨迹修正；验证了多模态反思与自适应策略的有效性。
