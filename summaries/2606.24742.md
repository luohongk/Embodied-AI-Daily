# World Value Models for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.24742v1
- arXiv ID: 2606.24742

---

🔍 **核心问题**：现有机器人价值模型基于视觉-语言模型（VLM）构建，缺乏对时间动态的深度理解，难以准确评估混合质量数据中的任务进展。

🛠 **方法**：提出世界价值模型（WVM），将世界模型的时间建模与未来规划能力与价值估计相结合，用于评估数据质量并指导策略学习。

✨ **主要贡献**：在标准基准和新增的Suboptimal-Value-Bench（含800条次优轨迹）上达到最先进的价值排序相关性（VOC）性能，并在仿真和真实场景中通过混合质量数据提升机器人操作策略学习效果。
