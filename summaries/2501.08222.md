# Data-driven Spatial Classification using Multi-Arm Bandits for Monitoring with Energy-Constrained Mobile Robots

- arXiv: https://arxiv.org/abs/2501.08222v2
- arXiv ID: 2501.08222

---

🔍 **核心问题**：如何利用能量受限的移动机器人团队，在数据噪声和有限能源条件下，快速将监测环境中的区域分类为“感兴趣”和“不感兴趣”。  
🛠 **方法**：提出双层规划策略，高层使用多臂赌博机框架基于在线数据动态识别潜在兴趣区域，低层基于整数规划生成无碰撞路径，并协调机器人访问这些区域。  
✨ **主要贡献**：理论证明了方法的任意时间保证和任务完成时间特性，并通过仿真和物理实验验证了其在噪声环境下的分类效率与能量约束下的可行性。
