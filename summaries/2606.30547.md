# Teaching Prompt-Based Programming with LLMs: A 45-Minute Lesson with Guided Practice for End-User Programmers

- arXiv: https://arxiv.org/abs/2606.30547v1
- arXiv ID: 2606.30547

---

🔍 **核心问题**：如何通过简短的教学干预（45分钟），帮助非计算机专业学生提升向大语言模型（LLM）清晰表达计算目标的能力，以解决提示词模糊或过于笼统的问题。

🛠 **方法**：设计了一节45分钟的提示词编程课程，包含引导式实践，并与同等时长的传统计算机科学实验活动（代码追踪）进行随机对照实验，评估55名工程专业学生的前后测表现。

✨ **主要贡献**：实验组在提示词编程能力上平均提升10.8个百分点（对照组仅提升1.1个百分点），且在提示词自我效能感上显著提升35.4个百分点（对照组提升21.9个百分点），表明短时干预能有效改善学习者的提示词技能，但效果有限，提示该技能需更长时间练习。
