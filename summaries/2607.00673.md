# Path Planning in Physically Viable World Models

- arXiv: https://arxiv.org/abs/2607.00673v1
- arXiv ID: 2607.00673

---

🔍 **核心问题**：机器人在非结构化户外环境中使用部署前采集的静态地图进行长时程路径规划，但环境物理变化（如洪水）可能导致原可行路径在执行时变得不安全或不可达。

🛠 **方法**：提出一种“物理可行世界模型”，将重建的3D高斯泼溅场景与物理仿真结合，在不重新采集数据或重建地图的情况下生成环境物理变化后的版本，并集成地形感知规划器评估路径可行性。

✨ **主要贡献**：首次将物理仿真与3D场景重建结合，使机器人能在部署前评估未来地形变化对路径可行性的影响，实验证明该方法能暴露传统静态规划无法发现的路径失效和重规划行为。
