# Adapting Generalist Robot Policies with Semantic Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.31958v1
- arXiv ID: 2606.31958

---

🔍 **核心问题**：通用机器人策略在预训练后，难以通过标准强化学习适应复杂或长周期任务，因为其动作分布与高性能策略差距较大。

🛠 **方法**：提出语义动作强化学习（SARL），通过在线交互优化语言提示空间，将通用策略作为可控技能先验，利用其已有技能组合解决新任务。

✨ **主要贡献**：SARL实现了结构化、语义有意义的探索和高效在线改进，在真实场景和模拟基准中显著超越现有方法，使视觉-语言-动作模型能解决复杂长周期任务。
