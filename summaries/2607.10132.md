# TAC-LOCO: Unified Whole-Body Control for Quadrupedal TACtile-Informed LOCO-Manipulation

- arXiv: https://arxiv.org/abs/2607.10132v1
- arXiv ID: 2607.10132

---

🔍 **核心问题**：四足机器人在动态全身操作中，如何在不确定外力下协调肢体运动并稳定抓取物体，同时缺乏触觉反馈在全身控制中的应用研究。

🛠 **方法**：提出TAC-LOCO框架，将柔性夹爪的触觉阵列观测编码为紧凑潜在表示，与本体感知结合，通过强化学习统一控制腿部、手臂和夹爪，并设计抓取稳定性奖励函数。

✨ **主要贡献**：在Unitree Go2机器人上零样本部署，实现动态触觉感知操作，抓取力降低47%，物体掉落率低于1%，首次将触觉反馈有效融入四足机器人全身控制。
