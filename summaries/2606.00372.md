# LFA: Layer Feature Attention for Run-Time Introspection of 2D Object Detectors in Automated Driving

- arXiv: https://arxiv.org/abs/2606.00372v1
- arXiv ID: 2606.00372

---

🔍 **核心问题**：自动驾驶中2D目标检测器即使性能先进仍会出错，现有故障预测方法仅利用最后一层特征或手工统计，忽略了早期层中不同抽象级别的视觉信息。

🛠 **方法**：提出轻量级Layer Feature Attention（LFA）方法，通过注意力机制学习聚合多个骨干网络层的特征，并端到端地学习各层重要性权重。

✨ **主要贡献**：在KITTI和BDD100K数据集上，LFA实现了最先进的故障预测性能，优于基于单层特征的基线方法，同时提供了可解释的层重要性分析。
