# CRC-HGD: A Histopathological Image Dataset for Grading Colorectal Cancer

- arXiv: https://arxiv.org/abs/2607.12750v1
- arXiv ID: 2607.12750

---

🔍 **核心问题**：结直肠癌组织学分级对预后和治疗规划至关重要，但缺乏公开、多放大倍率标注的病理图像数据集用于自动化分级研究。

🛠 **方法**：构建了包含1914张H&E染色病理图像的CRC-HGD数据集，图像来自214名患者，按WHO标准分为三级（高、中、低分化），并提供4x、10x、20x、40x四种放大倍率。

✨ **主要贡献**：首次提供涵盖所有分化等级且多放大倍率标注的结直肠癌病理图像数据集，为基于深度学习的自动分级研究提供了标准化资源。
