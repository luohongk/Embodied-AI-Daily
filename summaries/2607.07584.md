# PHaul: A PPO-based forwarding agent for Sub6 enhanced Integrated Access and Backhaul networks

- arXiv: https://arxiv.org/abs/2607.07584v1
- arXiv ID: 2607.07584

---

🔍 **核心问题**：解决Sub6增强型IAB网络中毫米波回传链路依赖视距条件导致的性能瓶颈问题，以及如何高效利用混合频谱资源进行流量转发。

🛠 **方法**：提出PHaul转发引擎，结合离线路径选择启发式算法与基于近端策略优化（PPO）的在线深度强化学习（DRL）智能体，并通过网络数字孪生定期采样输入流量并更新流到路径的映射。

✨ **主要贡献**：在多种网络配置下，PHaul相比两种对比启发式算法，吞吐效率提升高达36%，公平性提升20%，且对训练与推理阶段的拓扑差异（如链路故障）具有鲁棒性，执行时间在真实拓扑中低于10秒。
