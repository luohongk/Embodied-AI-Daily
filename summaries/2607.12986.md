# Win by Silence: Deletion Non-Monotonicity, Autonomous Exploitation, and Typed-State Gating in LLM Plan Evaluation

- arXiv: https://arxiv.org/abs/2607.12986v1
- arXiv ID: 2607.12986

---

🔍 **核心问题**：论文揭示了LLM计划评估器存在“删除非单调性”漏洞——一个策略可能因省略关键步骤而获得更高评分，导致评估系统鼓励遗漏必要工作而非真实改进。

🛠 **方法**：提出GATE（类型化状态门控）机制，通过拒绝为“沉默路线”（省略必要步骤的计划）发布评分，并利用PCSC（后验省略拼接检测）在模型介导的类型化状态记录中识别和消除省略拼接。

✨ **主要贡献**：实验证明所有57次合法删除均符合理论公式，且每条路线至少存在一个评分提升的删除；GATE在26条沉默路线中全部拒绝评分（0次误判），促使47/54次后续修订修复为覆盖结构，严格覆盖改进率从1/26提升至13/26，同时揭示了注册-来源边界漏洞（如义务通道规避）和成本地板策略的局限性。
