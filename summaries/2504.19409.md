# GSFF-SLAM: 3D Semantic Gaussian Splatting SLAM via Feature Field

- arXiv: https://arxiv.org/abs/2504.19409v2
- arXiv ID: 2504.19409

---

🔍 **核心问题**：现有语义SLAM系统依赖2D真值先验进行监督，但在真实环境中这些信号稀疏且含噪声，导致语义重建性能受限。

🛠 **方法**：提出GSFF-SLAM，基于3D高斯泼溅（3D Gaussian Splatting）构建特征场，联合渲染外观、几何与N维语义特征，并通过独立优化特征梯度，支持稀疏噪声2D先验的语义重建。

✨ **主要贡献**：在跟踪精度和逼真渲染质量上超越先前方法；利用2D真值先验时，语义分割mIoU达95.03%（SOTA），且实现最高2.9倍加速，性能仅轻微下降。
