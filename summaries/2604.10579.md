# AffordGen: Generating Diverse Demonstrations for Generalizable Object Manipulation with Afford Correspondence

- arXiv: https://arxiv.org/abs/2604.10579v2
- arXiv ID: 2604.10579

---

🔍 **核心问题**：机器人模仿学习因数据多样性不足，难以泛化到几何形状变化大的新物体。

🛠 **方法**：提出AffordGen框架，利用3D生成模型和视觉基础模型在大规模3D网格上建立语义关键点对应，生成多样化的机器人操作轨迹，并训练闭环视觉运动策略。

✨ **主要贡献**：通过语义可泛化的功能对应与端到端学习的结合，显著提升数据效率，在仿真和真实实验中实现对新物体的零样本泛化，并保持高成功率。
