# ChatImage: Navigating Long-Form LLM Answers through Interactive Images

- arXiv: https://arxiv.org/abs/2607.05290v1
- arXiv ID: 2607.05290

---

🔍 **核心问题**：大语言模型生成的详细回答通常以密集线性文本呈现，导致用户难以进行细粒度查看、导航和回访。

🛠 **方法**：提出ChatImage系统，将长文本回答转化为交互式图像，先结构化内容并规划视觉布局，再通过视觉定位模型（如LocateAnything）识别可交互区域，叠加透明热点，支持点击查看详情和局部追问。

✨ **主要贡献**：创新性地在渲染后基于视觉内容定位交互目标，提升视觉与点击区域一致性；发布参考实现和30题基准测试，评估交互循环完成度、视觉对齐门控和掩码完整性。
