# GigaWorld-Policy: An Efficient Action-Centered World--Action Model

- arXiv: https://arxiv.org/abs/2603.17240v2
- arXiv ID: 2603.17240

---

🔍 **核心问题**：现有世界-动作模型（WAM）在机器人策略学习中存在推理开销大、动作预测依赖未来视频质量的问题。

🛠 **方法**：提出GigaWorld-Policy，一种以动作为中心的WAM，通过耦合动作预测与视频生成两个任务进行训练，并采用因果设计使未来视频生成在推理时可选，从而加速动作解码。

✨ **主要贡献**：相比基线Motus，推理速度提升9倍，任务成功率提高7%；相比pi-0.5，在RoboTwin 2.0上性能提升95%。
