# Fast 3D Foundation Model Initialized Gaussian Splatting

- arXiv: https://arxiv.org/abs/2607.03209v1
- arXiv ID: 2607.03209

---

🔍 **核心问题**：传统3D高斯泼溅（3DGS）重建依赖耗时的运动恢复结构（SfM）进行初始化，且训练时间长，难以满足近实时应用需求。

🛠 **方法**：利用3D基础模型（3DFM）直接初始化相机位姿和点云，通过深度引导损失函数联合优化位姿与高斯图元，并引入MLP位姿精化模块提升稀疏视图下的重建质量。

✨ **主要贡献**：在Mip-NeRF 360等数据集上达到23.61 dB PSNR和0.19 LPIPS的竞争性重建质量，同时将每场景训练时间缩短至约3分钟，为机器人、VR和自主导航等近实时应用提供了高效方案。
