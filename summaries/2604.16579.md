# EviDep: Trustworthy Multimodal Depression Estimation via Disentangled Evidential Learning

- arXiv: https://arxiv.org/abs/2604.16579v2
- arXiv ID: 2604.16579

---

🔍 **核心问题**：在非受控环境下，现有抑郁症自动评估方法仅给出点估计，无法量化预测不确定性，易导致过度自信的不可靠评估。

🛠 **方法**：提出EviDep框架，通过正态-逆伽马分布联合估计抑郁严重度与偶然/认知不确定性；设计基于小波的混合专家模块解耦行为线索的时频异质性，并采用解缠证据学习策略在贝叶斯融合前分离跨模态共享与模态特有特征。

✨ **主要贡献**：在AVEC 2013/2014、DAIC-WOZ和E-DAIC四个数据集上，EviDep实现了最优预测精度与不确定性校准，为抑郁症评估提供了可信赖的风险感知决策工具。
