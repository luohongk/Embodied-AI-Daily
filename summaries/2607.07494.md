# GIFT: Geometry-Informed Low-precision Gradient Communication for LLM Pretraining

- arXiv: https://arxiv.org/abs/2607.07494v1
- arXiv ID: 2607.07494

---

🔍 **核心问题**：在大语言模型预训练中，梯度通信是主要扩展瓶颈，而现有低精度量化方法（如FP8）因梯度高度各向异性导致方向依赖的失真，降低模型性能。

🛠 **方法**：提出GIFT方法，通过几何感知坐标变换将梯度映射到近各向同性空间后再进行低精度量化通信，并采用低秩近似与选择性应用简化变换，以平衡计算开销与通信缩减。

✨ **主要贡献**：在Llama-600M模型上，GIFT在64个NVIDIA GH200超级芯片上实现端到端预训练时间减少7.6%，同时相比直接欧几里得FP8通信，下游任务保留性能更优，且不改变优化器、训练方案或通信格式。
