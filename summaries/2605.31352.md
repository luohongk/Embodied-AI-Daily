# Haptic Sorter: A Unified Planning Framework for Online Shape Estimation and Real-Time Pose Inference

- arXiv: https://arxiv.org/abs/2605.31352v1
- arXiv ID: 2605.31352

---

🔍 **核心问题**：机器人操作中，物体的形状和位姿通常假设已知，但实际中几何信息不精确，且位姿推断受传感器不确定性和遮挡影响。

🛠 **方法**：提出统一模型框架，结合贝叶斯优化引导触觉探索以推断物体形状（用超椭圆近似），并设计在线常微分方程基于模型预测和触觉反馈实时推断位姿。

✨ **主要贡献**：首次将贝叶斯优化用于触觉形状推断，提出自适应操作势能编码物体几何，并在2D分拣任务中验证了框架在仿真和真实多臂系统中的鲁棒性与泛化能力。
