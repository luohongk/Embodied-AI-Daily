# DSP-SLAM++: A Unified Framework for Multi-Class, High-Fidelity Object SLAM in the Wild

- arXiv: https://arxiv.org/abs/2606.25953v1
- arXiv ID: 2606.25953

---

🔍 **核心问题**：现有面向物体的SLAM系统难以同时兼顾实时性、多类别物体支持以及高保真、语义一致的物体模型生成。

🛠 **方法**：提出DSP-SLAM++框架，通过引入异步建图流水线提升实时性能，并针对单目鱼眼-激光雷达传感器套件进行专用融合适配。

✨ **主要贡献**：在支持多类别物体精细几何形状重建的同时，将最大物体处理延迟相比现有最优基线降低高达70%，实现了在25Hz多类别数据集上的鲁棒实时性能，并开源代码。
