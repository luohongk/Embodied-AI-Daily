# AGI Maze as a Benchmark Framework for World-Modeling Agents

- arXiv: https://arxiv.org/abs/2607.00627v1
- arXiv ID: 2607.00627

---

🔍 **核心问题**：当前大语言模型（LLM）在部分可观测、有状态的环境中，无法可靠地构建和利用外部世界的内部表征，而仅依赖局部模式匹配。

🛠 **方法**：提出AGI Maze框架，通过一系列基于网格的迷宫任务，构建需要记忆和隐藏状态推理的环境，并评估了多种标准LLM及一个利用消息历史作为工作记忆的基线智能体。

✨ **主要贡献**：证明了标准LLM在推理时无法内部表征迷宫结构，即使允许使用工作记忆，LLM智能体在人类可轻松完成的步数预算内仍无法可靠解决小型迷宫，揭示了当前模型在持久世界建模上的根本局限。
