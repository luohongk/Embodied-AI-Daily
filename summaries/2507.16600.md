## 0. 概览（Concise Summary）

- **主题（Topic）**：基于5G网络的地面定位系统，旨在替代或补充全球导航卫星系统（GNSS），特别是在GNSS信号不可用（GNSS-denied）的场景下。
- **问题（Problem）**：在GNSS信号被干扰或遮挡的城市、室内等环境中，如何利用现有5G基础设施实现高精度、鲁棒的定位，并解决载波相位定位中的整数模糊度、非视距（NLOS）识别以及信号中断时的连续跟踪问题。
- **方法（Method）**：提出一个融合多传感器信息的混合定位系统。核心方法包括：
    1.  **多载波载波相位（CP）测距**：利用虚拟波长技术消除整数模糊度，实现高精度测距。
    2.  **深度学习LOS/NLOS分类**：使用时间卷积神经网络（T-CNN）分析信道冲激响应（CIR），实时识别并排除NLOS链路。
    3.  **传感器融合**：当LOS信号不足时，通过误差状态扩展卡尔曼滤波（ES-EKF）融合视觉里程计（VO）和惯性测量单元（IMU）数据，保持定位连续性。
- **创新（Innovation）**：
    1.  提出一个完整的、端到端的5G定位流水线，将CP定位、ML分类和传感器融合有机整合。
    2.  在CP测距中，提出一种基于线性平均和相位连续性约束的多载波相位差处理方法，比传统的矢量求和法更鲁棒。
    3.  在LOS/NLOS分类中，引入数据驱动的下采样（Stride操作）和置信度评分（κ score），实现软决策。
- **意义（Significance）**：为GNSS拒止环境提供了一种可行的、高精度的替代定位方案。实验表明，在KITTI城市数据集上，该系统能达到与商用GNSS服务相当的定位精度（<5米），展示了其在自动驾驶、工业自动化等领域的应用潜力。
- **一句话总结**：本文提出了一种融合5G载波相位测距、深度学习NLOS识别和视觉/惯性传感器融合的鲁棒地面定位系统，在GNSS失效场景下实现了米级定位精度。

## 1. 研究动机（Motivation）

- **关键挑战/空白**：全球导航卫星系统（GNSS）是当前主流的定位技术，但其信号易受有意或无意的干扰（如射频噪声干扰），且在室内、城市峡谷等复杂环境中信号会严重衰减或完全不可用。这暴露了依赖单一信息源的定位系统的脆弱性。因此，亟需一种更鲁棒、更具弹性的替代或补充定位方案。
- **为什么重要**：自动驾驶、工业物联网（IIoT）、增强现实等新兴应用对定位的精度、可靠性和连续性提出了极高要求。在GNSS失效的场景下，这些应用将无法正常运行，甚至可能引发安全事故。因此，开发不依赖GNSS的鲁棒定位系统具有重大的战略意义和商业价值。
- **挑战在哪里**：
    1.  **高精度测距**：传统基于时间（TOA/TDOA）或角度（AOA）的定位方法在NLOS和多径环境下精度会急剧下降。载波相位（CP）测距精度高，但面临**整数模糊度（IA）** 这一核心难题。
    2.  **环境感知**：CP测距依赖LOS传播，NLOS信号会引入巨大误差。因此，需要实时、准确地识别并排除NLOS链路。
    3.  **定位连续性**：在城市环境中，LOS链路可能频繁中断。如何在LOS中断期间保持连续、可靠的定位，是系统鲁棒性的关键。

## 2. 创新点（Innovation）

论文的主要创新点可以归纳为以下三点：

1.  **一种无需显式求解整数模糊度的多载波CP测距方法**：
    - **不同之处**：传统方法需要复杂的整数模糊度求解过程。本文利用5G系统中多子载波的特性，通过计算两个子载波的相位差，构造一个**虚拟波长**。通过选择合适的子载波间隔，使虚拟波长大于基站与用户间的最大可能距离，从而使得相位差测量本身不包含整数模糊度。
    - **为何奏效**：该方法将整数模糊度问题转化为一个设计问题（选择合适的子载波间隔），而非一个计算问题。它利用了5G系统固有的多载波结构，避免了复杂的搜索和估计过程，降低了计算复杂度。
    - **进一步创新**：在计算平均相位差时，论文提出了一种基于**线性平均**和**相位连续性约束**的方法（公式8），而非传统的矢量求和法（公式7）。这种方法对稀疏多径环境下的强反射分量更具鲁棒性，能更好地隔离出真实的LOS相位。

2.  **一个基于T-CNN的、置信度感知的LOS/NLOS链路分类模块**：
    - **不同之处**：不同于使用手工特征的传统机器学习方法，本文直接使用从SRS信号中提取的CIR作为输入，利用T-CNN自动学习区分LOS和NLOS的时空特征。
    - **为何奏效**：CNN的局部感受野和权重共享特性非常适合处理CIR这类具有局部时域结构的信号。T-CNN能有效捕捉多径传播的延迟域特征。
    - **进一步创新**：论文引入了**数据驱动的下采样（Stride操作）** 来减少模型参数，并提出了一个**置信度评分（κ score）** 用于模型选择和软决策。这意味着模型不仅输出分类结果，还输出一个置信度，系统可以根据置信度选择性地排除不可靠的链路，而不是简单地硬判决。

3.  **一个融合CP、VO和IMU的ES-EKF传感器融合框架**：
    - **不同之处**：将高精度的CP定位（作为绝对参考）与视觉里程计（VO）和IMU（提供短时相对运动估计）进行松耦合融合。当CP可用时，它用于修正VO/IMU的累积漂移；当CP不可用时，系统平滑地切换到VO/IMU模式，保持定位连续性。
    - **为何奏效**：这种混合架构充分利用了不同传感器的互补优势。CP提供长期稳定的绝对位置，VO/IMU提供短期高频的相对运动。ES-EKF提供了一个优雅的框架来融合这些异构信息，并处理不同传感器的不确定性。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文提出的系统架构如图1所示，主要包含三个核心模块：

1.  **CP定位模块**：该模块利用5G NR的PRS（下行）或SRS（上行）信号进行载波相位测距。它接收来自多个gNB的信号，通过多载波CP方法计算到每个gNB的距离，然后使用三边测量法（Trilateration）计算用户位置。该模块仅在至少三个gNB提供LOS条件时激活。
2.  **LOS/NLOS分类模块**：这是一个基于T-CNN的深度学习模型，它分析从SRS信号中提取的CIR，实时判断每个gNB链路是LOS还是NLOS。只有被判定为LOS的链路才会被送入CP定位模块。该模块实现了对测量数据的筛选。
3.  **传感器融合模块**：当LOS链路数量不足时，CP定位模块失效。此时，系统切换到传感器融合模式。该模块使用ES-EKF融合视觉里程计（VO）和IMU数据。VO提供基于图像特征的相对运动估计，IMU提供加速度和角速度。ES-EKF将CP模块最后一次有效的位置作为初始值，然后利用VO/IMU进行状态预测和更新，直到CP信号恢复。

### 3.2 关键算法与数学推导

**1. 多载波CP测距与虚拟波长**

- **基本原理**：发射两个频率为 $f_1$ 和 $f_2$ 的导频信号，接收端测量其相位 $\phi_1$ 和 $\phi_2$。由于相位测量存在 $2\pi$ 模糊度，实际相位为 $\phi_i + 2\pi N_i$，其中 $N_i$ 是整数模糊度。
- **相位差与虚拟波长**：通过计算相位差 $\Delta\phi = \phi_2 - \phi_1$，可以消除收发机之间的未知相位偏移 $\theta$：
  $$\Delta\phi + 2\pi\Delta N = 2\pi d \cdot \frac{\Delta f}{c} = \frac{2\pi d}{\lambda_v}$$
  其中 $\Delta f = f_2 - f_1$，$\Delta N = N_2 - N_1$，**虚拟波长** $\lambda_v = c / \Delta f$。
- **距离估计**：距离 $d$ 可以表示为：
  $$d = \left( \frac{\Delta\phi}{2\pi} + \Delta N \right) \lambda_v$$
  如果 $\lambda_v$ 大于可能的距离 $d$，则 $\Delta N = 0$，从而无需显式求解整数模糊度。

**2. 改进的相位差平均方法**

- **问题**：传统方法（公式7）使用矢量求和计算平均相位差，在稀疏多径环境中易受强反射分量的影响。
- **本文方法**：提出基于线性平均和相位连续性约束的方法（公式8）：
  $$\Delta\phi_{\text{avg}} = \frac{1}{K-k} \left( \sum_{i=1}^{K-k} \arg\left(A_{i+k}e^{j\phi_{i+k}}\right) - \arg\left(A_i e^{j\phi_i}\right) \right)$$
  其中 $K$ 是子载波总数，$k$ 是固定间隔。关键在于，在计算前需要对相位进行**解缠绕（unwrapping）**，确保相位随频率单调递增，以符合物理传播规律。这能有效识别并剔除由强反射引起的相位异常值。

**3. 误差状态扩展卡尔曼滤波（ES-EKF）**

- **状态向量**：误差状态 $\delta\mathbf{x}$ 通常包含位置误差、速度误差、姿态误差以及IMU的偏置误差。
- **预测步骤**：利用IMU的测量值（加速度和角速度）通过运动学模型预测名义状态 $\mathbf{x}$，同时利用IMU噪声协方差预测误差状态的协方差 $\mathbf{P}$。
- **更新步骤**：
    - **CP更新**：当CP定位有效时，将CP输出的位置作为观测值 $\mathbf{z}$，计算观测残差 $\mathbf{y} = \mathbf{z} - \mathbf{H}\delta\mathbf{x}$，然后更新误差状态和协方差。
    - **VO更新**：VO提供相邻帧之间的相对位姿变化，也可以作为观测值来更新误差状态。
- **误差注入**：在更新后，将估计出的误差状态 $\delta\mathbf{x}$ 注入到名义状态 $\mathbf{x}$ 中进行修正，然后将误差状态重置为零。

### 3.3 模型、训练与数据集

- **LOS/NLOS分类模型**：一个轻量级的T-CNN。输入是经过FFT和批归一化处理的SRS波形（3276个实数值功率）。网络包含卷积层和全连接层，输出层是两个神经元的Softmax，分别对应LOS和NLOS的概率。
- **训练**：模型在仿真数据上训练。仿真场景基于3GPP TR 38.901的UMi（城市微蜂窝）信道模型，使用QUADRIGA框架生成。标签（LOS/NLOS）通过已知的发射机和接收机位置计算得出。
- **数据集**：主要使用**KITTI数据集**进行系统级评估。KITTI数据集包含一辆在城市环境中行驶的车辆所采集的相机、IMU和GPS数据。论文使用Sionna射线追踪工具，基于KITTI场景的3D建筑模型，模拟了5G基站的部署和LOS/NLOS可见性。然后，基于这些可见性状态，使用3GPP UMi信道模型生成用于CP测距和ML分类的信道实现。

### 3.4 实验设置与结果

- **实验设置**：
    - **仿真场景1（CP性能评估）**：在3GPP UMi场景下，部署3个TRP，评估CP测距和定位精度。
    - **仿真场景2（系统级评估）**：在KITTI数据集对应的城市环境中，部署多个5G基站，模拟车辆行驶过程中的信道变化，评估整个混合定位系统的性能。
- **主要结果**：
    1.  **CP定位精度**：在纯LOS条件下，CP定位可以达到厘米级精度（90%分位误差23.4 cm）。但在混合LOS/NLOS条件下，如果不进行NLOS排除，误差会急剧增大（90%分位误差68.9 m），这凸显了NLOS识别的重要性（见表I）。
    2.  **系统级定位精度**：在KITTI城市场景下，提出的混合定位系统可以实现**小于5米**的定位误差，这与公共商用GNSS服务的精度相当。这表明该系统在GNSS拒止环境下具有作为鲁棒替代方案的潜力。

### 3.5 技术细节

- **网络规划**：论文强调了为定位服务优化网络规划的重要性。传统的通信网络规划追求覆盖和容量，最小化基站间重叠。而为定位服务规划时，需要最大化LOS可见性，即确保用户在任何位置都能同时看到至少3个基站。仿真表明，将基站从4个增加到12个，可以将满足此条件的区域从16%提升到64%（图3）。
- **5G信号选择**：论文选择使用5G NR中的**定位参考信号（PRS）** 和**探测参考信号（SRS）** 进行CP测距。这些信号是3GPP专门为定位设计的，具有确定性的序列和可配置的参数（如带宽、周期、梳状结构），非常适合高精度相位测量。
- **SRS配置示例**：论文给出了一个SRS配置示例，使用comb-6结构（每6个子载波一个导频），子载波间隔120 kHz，则频率间隔为720 kHz，对应的虚拟波长约为416米，远大于典型的UMi小区半径（200米），从而实现了无模糊测距。

## 4. 意义与影响（Significance and Impact）

- **解决了什么问题**：
    1.  为GNSS拒止环境提供了一个可行的、基于现有5G基础设施的高精度定位解决方案。
    2.  系统性地解决了CP定位在5G网络中应用面临的三大挑战：整数模糊度、NLOS识别和LOS中断。
    3.  提出了一个完整的、端到端的定位流水线，将理论方法、机器学习与传感器融合技术有机结合。

- **对未来研究/应用的影响**：
    1.  **推动5G-Advanced定位标准化**：论文的工作与3GPP Rel.18及后续版本中关于CP定位的标准化方向高度一致，为实际部署提供了有价值的参考。
    2.  **赋能新兴应用**：为自动驾驶、工业自动化、无人机导航等对定位鲁棒性要求极高的应用提供了关键技术支撑。
    3.  **促进多传感器融合研究**：论文提出的CP+VO+IMU融合框架为其他异构传感器融合定位系统提供了设计范式。

- **局限与开放问题**：
    1.  **依赖仿真数据**：LOS/NLOS分类模型是在仿真数据上训练的，其在实际复杂环境中的泛化能力有待验证。
    2.  **计算复杂度**：虽然T-CNN是轻量级的，但实时运行整个流水线（包括CP处理、ML推理和EKF）对UE的计算能力仍有一定要求。
    3.  **网络部署成本**：为实现高精度定位，可能需要更密集的基站部署，这会增加运营商的成本。
    4.  **同步要求**：CP测距对收发机之间的同步精度要求很高，虽然差分方法可以消除部分误差，但残余的同步误差仍会影响最终精度。

## 5. 澄清与简化（Clarifications and Simplifications）

- **载波相位（CP）测距 vs. 传统时间测距**：想象一下用尺子量距离。传统时间测距（如TOA）就像用一把厘米刻度的尺子，精度有限。CP测距则像用一把毫米甚至更精细的尺子，因为它测量的是无线电波的“波峰”和“波谷”的位置。但问题在于，这把“精细尺子”是周期性的，你不知道你量的是第几个周期，这就是**整数模糊度**问题。
- **虚拟波长**：为了解决整数模糊度，论文巧妙地用了“差频”的方法。就像用两个不同频率的音叉产生一个“拍音”，这个“拍音”的频率很低，波长很长。这个长波长的“虚拟波”就像一把“粗尺子”，它的刻度足够大，保证你第一次测量时就知道是第几个周期，从而解决了整数模糊度问题。
- **LOS/NLOS分类**：LOS就像你和朋友在空旷的广场上面对面说话，声音清晰直接。NLOS就像你们中间隔了一堵墙，你听到的是回声和混响，声音失真。T-CNN模型就像一个训练有素的声呐兵，它能通过分析接收到的“回声”（CIR）的特征，判断出你和朋友之间是否有墙（NLOS），从而决定是否使用这个“失真”的声音来测距。
- **传感器融合（ES-EKF）**：想象你在一个没有GPS的陌生城市里导航。CP定位就像你偶尔看到的、非常精确的路牌，告诉你绝对位置。VO就像你用自己的脚步和方向感（IMU）来估计自己走了多远、拐了几个弯。ES-EKF就像一个聪明的导航员，它会在看到路牌时（CP更新）修正你的脚步估计（VO/IMU）的累积误差（漂移）。当你看不到路牌时（LOS中断），它就完全依靠你的脚步和方向感继续导航，直到下一个路牌出现。
