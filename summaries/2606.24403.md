# RE4: Transformation-aware Imitation of Object Interactions Using Manipulation Modes

- arXiv: https://arxiv.org/abs/2606.24403v1
- arXiv ID: 2606.24403

---

🔍 **核心问题**：现有模仿学习方法在物体交互任务中难以兼顾高性能与可解释性，端到端方法虽表现优异但缺乏透明度。

🛠 **方法**：提出RE4框架，包含四个步骤：基于自监督的轻量级无模型物体姿态估计、操作模式感知的演示检索、模式感知变换、以及保持模式约束的重规划与演示执行。

✨ **主要贡献**：在Push-T和Robomimic基准上验证了RE4框架的有效性，尤其在图像输入的稀疏数据场景下展现出鲁棒性，表明简单可解释的模块也能有效学习操作技能。
