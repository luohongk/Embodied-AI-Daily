# Learning to control switching nonlinear systems with Koopman operator regression

- arXiv: https://arxiv.org/abs/2607.11344v1
- arXiv ID: 2607.11344

---

🔍 **核心问题**：针对具有有限动作空间的非线性系统，如何从有限样本中识别其动态特性并实现有效控制。

🛠 **方法**：利用再生核希尔伯特空间中的Koopman算子回归估计未知动态，得到线性切换预测模型（切换由控制变量值决定）；随后通过模型预测控制求解无限时域最优控制问题，并考虑了时变阶段成本。

✨ **主要贡献**：理论推导了Koopman动态近似的学习率，并在精确动态与学习动态两种情形下量化了模型预测控制的次优性；通过Duffing振荡器数值仿真验证了理论结果。
