# MemBoost: A Memory-Boosted Framework for Cost-Aware LLM Inference

- arXiv: https://arxiv.org/abs/2603.26557v2
- arXiv ID: 2603.26557

---

🔍 **核心问题**：大语言模型在服务中处理重复或近似查询时推理成本高昂，缺乏高效复用历史答案的机制。  
🛠 **方法**：提出MemBoost框架，通过轻量模型复用历史答案、检索相关支持信息，并将困难查询路由至强模型，同时支持持续记忆增长与成本感知调度。  
✨ **主要贡献**：显著减少强模型调用次数与总体推理成本，在模拟负载下保持与强模型基线相当的回答质量。
