# Toward Semantic Communication for Real-time Mobile 3D Reconstruction

- arXiv: https://arxiv.org/abs/2607.16128v1
- arXiv ID: 2607.16128

---

🔍 **核心问题**：实时移动3D重建中，通信导致的图像失真会严重影响相机位姿和场景几何的在线估计精度，现有语义通信方法缺乏针对几何估计的可靠性信息。

🛠 **方法**：提出一种语义通信框架，包含输出重建图像及逐像素置信度图的语义收发器，并引入置信度引导的几何估计方法，将置信度融入RANSAC位姿初始化和光束法平差中。

✨ **主要贡献**：在保持高图像质量的同时，显著提升了位姿估计精度和3D结构一致性，优于现有语义通信及传统分离式信源信道编码方法。
