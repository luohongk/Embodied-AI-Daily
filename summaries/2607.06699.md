# RoboSnap: One-Shot Real-to-Sim Scene Generation for Generalizable Robot Learning and Evaluation

- arXiv: https://arxiv.org/abs/2607.06699v1
- arXiv ID: 2607.06699

---

🔍 **核心问题**：如何从单张真实世界RGB图像快速生成既物理稳定又视觉逼真的可交互仿真场景，以支持通用机器人学习和可重复策略评估。

🛠 **方法**：提出RoboSnap框架，采用分层设计——碰撞感知的前景资产用于稳定机器人交互，3D高斯泼溅视觉层在新视角下保留逼真背景外观，将单张图像转化为仿真就绪场景。

✨ **主要贡献**：在DROID场景和真实机器人任务中实现可靠轨迹回放，支持任务特定合成数据生成，并建立有意义的仿真-真实相关性；同时构建包含564个真实场景的DROID-Sim数据集，证明真实到仿真方法的价值在于将真实环境转化为可复用的机器人学习基础设施。
