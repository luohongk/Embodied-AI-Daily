# Quantifying Frontier LLM Capabilities for Container Sandbox Escape

- arXiv: https://arxiv.org/abs/2603.02277v2
- arXiv ID: 2603.02277

---

🔍 **核心问题**：大型语言模型（LLM）作为自主代理时，可能利用工具执行代码、访问网络等，带来安全风险，而现有沙箱环境（如Docker容器）能否有效隔离这些模型尚需量化评估。

🛠 **方法**：提出SANDBOXESCAPEBENCH基准测试，采用嵌套沙箱架构（外层含无漏洞的flag），通过Inspect AI的CTF评估框架，模拟恶意代理在容器内利用配置错误、内核漏洞等机制逃逸的场景。

✨ **主要贡献**：首次系统量化LLM突破容器沙箱的能力，发现当存在漏洞时，LLM能主动识别并利用漏洞逃逸，证明需持续使用此类基准测试以确保沙箱对高能力模型的有效封装。
