# SeerGuard: A Safety Framework for Mobile GUI Agents via World Model Prediction

- arXiv: https://arxiv.org/abs/2607.15550v1
- arXiv ID: 2607.15550

---

🔍 **核心问题**：移动GUI代理在执行复杂任务时存在安全风险，现有安全机制多为事后反应，缺乏执行前的风险预判能力。  
🛠 **方法**：提出SeerGuard框架，通过指令级筛选和动作级风险评估实现执行前安全控制，并构建统一的安全增强世界模型（SAWM），结合语义下一状态预测与安全风险评估。  
✨ **主要贡献**：在Qwen3-VL-8B-Instruct上，安全-效用评分从0.191提升至0.596（ω=0.8），风险成本评分从0.347降至0.130（α=0.8），验证了指令筛选、动作风险评估及状态预测的有效性。
