# Exploring Extrinsic and Intrinsic Properties for Effective Reasoning with Code Interpreter

- arXiv: https://arxiv.org/abs/2606.16934v1
- arXiv ID: 2606.16934

---

🔍 **核心问题**：探究代码解释器（CI）增强大语言模型推理能力时，有效推理行为的外在属性（关键token）和内在属性（代码特定认知行为）及其作用机制。

🛠 **方法**：通过分析多个LLM在CI推理中的关键token分布和认知行为（验证、回溯、反向链），并在推理阶段附加关键token、训练阶段增强认知行为来验证效果。

✨ **主要贡献**：首次系统刻画CI有效推理的行为特征，发现关键token和认知行为能提升数学、排序等推理性能，减少错误回答中的过度思考并提高token效率，同时揭示了部分模型增益受限的因素。
