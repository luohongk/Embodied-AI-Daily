# Visual graphs for image classification: does the structure affect performance?

- arXiv: https://arxiv.org/abs/2607.06295v1
- arXiv ID: 2607.06295

---

🔍 **核心问题**：现有深度学习模型在图像分类中难以有效编码图像的视觉结构，忽略了空间、拓扑和语义信息，而图神经网络在此方面的应用尚未被充分探索。

🛠 **方法**：在固定三层图卷积网络（GCN）架构下，系统比较了多种当前主流的图构建技术，并通过实证研究分析网络结构对分类性能的影响。

✨ **主要贡献**：揭示了图结构对分类性能的关键影响，并为图使用前的计算阶段提供了重要方法论指导，强调结构本身会显著影响后续处理流程。
