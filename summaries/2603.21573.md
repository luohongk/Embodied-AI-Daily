# Rethinking Visual Privacy: A Compositional Privacy Risk Framework for Severity Assessment with VLMs

- arXiv: https://arxiv.org/abs/2603.21573v2
- arXiv ID: 2603.21573

---

🔍 **核心问题**：现有视觉隐私基准将隐私视为二元属性（私密/非私密），忽略了多个无害属性组合可能引发严重隐私泄露的风险。

🛠 **方法**：提出组合隐私风险分类法（CPRT），基于独立可识别性和组合危害潜力对视觉属性分级，并设计可解释的评分函数；构建6.7K图像数据集，通过视觉语言模型（VLM）评估组合隐私风险。

✨ **主要贡献**：揭示了前沿VLM在结构化引导下能较好评估组合隐私风险，但系统低估组合驱动风险；小模型难以内化分级隐私推理；提出8B SFT模型，在组合隐私评估上接近前沿性能。
