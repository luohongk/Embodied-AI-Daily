# TraceLab: Characterizing Coding Agent Workloads for LLM Serving

- arXiv: https://arxiv.org/abs/2606.30560v1
- arXiv ID: 2606.30560

---

🔍 **核心问题**：当前缺乏真实、多代理、多模型家族的编码智能体工作负载数据，导致LLM服务系统优化缺乏实际依据。

🛠 **方法**：收集并公开了约4,300个编码智能体会话（含35万LLM步骤和43万工具调用）的真实追踪数据，并进行了系统性负载特征分析。

✨ **主要贡献**：揭示了编码智能体工作负载的四大特征（长自主循环、长上下文短输出、多样化工具调用、高但不完美的前缀缓存命中率），并据此提出四项具体优化方向（低开销工具调用、追加长度感知预填充、语义感知工具延迟预测、人机间隙KV缓存管理）。
