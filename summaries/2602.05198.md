# Informative Path Planning with Guaranteed Estimation Uncertainty

- arXiv: https://arxiv.org/abs/2602.05198v3
- arXiv ID: 2602.05198

---

🔍 **核心问题**：在资源受限的环境监测中，如何规划机器人路径，既能高效采集数据，又能保证对未知区域的估计不确定性（如高斯过程后验方差）低于用户指定的阈值。

🛠 **方法**：提出三阶段方法：①从先验数据学习非平稳高斯过程模型；②将核函数转化为二值覆盖图，标识能降低不确定性的位置；③规划近似最短路径，满足全局不确定性约束，并支持非凸障碍环境。

✨ **主要贡献**：首次在复杂环境中实现带估计不确定性保证的信息路径规划，提供传感点选择和联合路径规划的近似最优性保证；在真实地形数据和自主水面/水下机器人实验中，相比基线方法，用更少的传感点和更短的路径达到不确定性目标。
