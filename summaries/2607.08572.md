# Switch-Reasoner: Learn When to Think in Multitask Mixtures via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.08572v1
- arXiv ID: 2607.08572

---

🔍 **核心问题**：多模态大语言模型在异构多任务场景中采用固定的“先思考后回答”范式，导致简单输入浪费计算资源，而复杂任务又需要推理，且训练中容易陷入“总是思考”或“总是直接回答”的不稳定状态。

🛠 **方法**：提出Switch-Reasoner框架，基于GRPO（分组相对策略优化），将思考视为虚拟工具调用，让模型自适应选择直接回答或先推理再回答，并引入双层调节机制平衡两种模式的使用，同时根据样本级相对收益提供监督。

✨ **主要贡献**：在11个多模态任务上，Switch-Reasoner在保持强性能的同时减少了不必要的推理，实现了更好的准确率-效率权衡。
