# DL-VINS-Factory: A Modular Framework for Learned Visual Front-Ends in Visual-Inertial SLAM

- arXiv: https://arxiv.org/abs/2607.01757v1
- arXiv ID: 2607.01757

---

🔍 **核心问题**：深度学习特征在紧耦合视觉惯性SLAM（VI-SLAM）中的实际价值尚未被充分验证，缺乏统一框架来系统评估其性能。

🛠 **方法**：提出DL-VINS-Factory框架，集成多种学习型特征提取器（如ALIKED、SuperPoint）与光流跟踪或描述子匹配，后端采用滑动窗口Ceres优化，并可选AnyLoc闭环与4自由度位姿图优化。

✨ **主要贡献**：在四个数据集上全面基准测试，证明学习型前端在实时嵌入式VI-SLAM中可行但非普遍优于经典方法；ALIKED+LG在EuRoC和NTU-VIRAL上分别降低ATE 5%-12%，SuperPoint+LK在Botanic Garden上降低ATE 29%；所有配置在Jetson AGX Orin上实时运行（单目29-47 FPS），AnyLoc闭环检测比BRIEF+DBoW2多2-7倍有效回环。
