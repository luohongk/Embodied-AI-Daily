# Automated Summarization of Software Documents: An LLM-based Multi-Agent Approach

- arXiv: https://arxiv.org/abs/2606.24689v1
- arXiv ID: 2606.24689

---

🔍 **核心问题**：如何利用大语言模型（LLM）自动生成软件文档的简洁准确摘要，以提升软件工程中的文档处理效率。

🛠 **方法**：提出基于LLM的多智能体系统Metagente，采用教师-学生架构，让多个LLM智能体协作优化摘要的相关性和精确性。

✨ **主要贡献**：在真实数据集上验证了Metagente的有效性，其摘要质量优于现有基线方法，显著改进了需求分析和技术文档的自动摘要能力，并揭示了LLM在软件工程中的潜力与未来研究方向。
