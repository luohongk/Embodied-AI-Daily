# Belt-Finger: An Affordable Soft Belt-Driven Gripper for Dexterous In-Hand Manipulation

- arXiv: https://arxiv.org/abs/2606.20193v1
- arXiv ID: 2606.20193

---

🔍 **核心问题**：传统平行夹爪缺乏手指内操作能力，导致机械臂在狭小空间内灵活性不足，需依赖大幅臂部运动。

🛠 **方法**：设计一种双软带驱动的指模块，在保留平行夹爪开合功能基础上，增加平移、俯仰和滚动三个手指内自由度，并集成模型预测控制与轻量遥操作接口。

✨ **主要贡献**：提出低成本、易集成的软带夹爪，显著提升手指内操作灵巧性，在遥操作、MPC及训练策略的多项复杂任务中，均优于传统平行夹爪。
