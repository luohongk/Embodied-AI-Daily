# NavVerse: Benchmarking Indoor-to-Outdoor Embodied Navigation in Continuous Robot Simulation

- arXiv: https://arxiv.org/abs/2607.19695v1
- arXiv ID: 2607.19695

---

🔍 **核心问题**：现有导航基准通常将室内与室外场景分开评估，忽略了机器人从室内到室外连续导航中面临的出口寻找、边界穿越、环境适应及运动动力学失败等挑战。

🛠 **方法**：提出NavVerse，一个基于物理引擎的连续仿真基准，包含100个室内场景、50个城市室外场景及50个室内-室外混合场景，并设计了10,000个任务实例，涵盖目标导航、视觉-语言导航和地点导航三类任务，通过可执行机器人接口评估成功率、路径效率与安全性。

✨ **主要贡献**：构建了首个支持室内到室外连续导航的物理仿真基准，并通过零样本实验揭示当前强化学习、视觉-语言-动作模型及模块化方法在跨场景适应上的显著瓶颈，其中端到端VLA模型成功率最高，模块化方法安全性最优，而地点导航任务显示从室外到室内-室外场景的性能明显下降。
