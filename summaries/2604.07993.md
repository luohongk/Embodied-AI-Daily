## 0. 概览（Concise Summary）

- **主题（Topic）**：全尺寸双足人形机器人的全身协调操作（Whole-Body Manipulation）。
- **问题（Problem）**：现有的视觉-语言-动作（VLA）模型在处理人形机器人控制时，往往将身体各部分（如手臂、腿）视为独立单元，缺乏对全身协调、动态平衡和时序一致性的显式建模，导致在高自由度、需要快速反应和长周期任务中表现不佳。
- **方法（Method）**：提出了一个名为 **HEX** 的框架，包含三个核心组件：
    1.  **人形对齐通用状态表示（Humanoid-Aligned Universal State Representation）**：将不同形态机器人的本体感受状态映射到一组规范的身体部件（如左臂、右腿）的共享潜在空间。
    2.  **基于混合专家模型（MoE）的统一本体感受预测器（Unified Proprioceptive Predictor, UPP）**：在共享潜在空间中，预测未来短时间内的全身状态动态，并利用MoE模块处理不同身体部件和不同机器人形态的差异。
    3.  **带有自适应融合的动作专家（Action Expert with Adaptive Fusion）**：通过残差门控融合机制，将视觉-语言上下文与预测的未来本体感受状态动态地结合起来，用于生成平滑、协调的动作。
- **创新（Innovation）**：
    1.  首次提出针对全尺寸双足人形机器人的全身VLA框架。
    2.  提出了跨形态的人形对齐状态表示和预测性本体感受建模，实现了可扩展的全身预训练。
    3.  引入了“回顾-预测”（review-and-forecast）范式，通过轻量级历史令牌总结视觉上下文，并结合未来状态预测进行动作生成。
- **意义（Significance）**：解决了人形机器人实现复杂、协调的全身操作（如一边走路一边搬运物体）的关键难题。通过显式建模身体部件间的相互作用和时序动态，显著提升了机器人在快速反应和长周期任务中的成功率和鲁棒性，为人形机器人在复杂人类环境中的部署提供了有效方案。
- **一句话总结**：HEX通过引入人形对齐的预测性本体感受建模和自适应多模态融合，实现了全尺寸双足人形机器人协调、稳定的全身操作，在真实世界任务中达到了最先进的性能。

## 1. 研究动机（Motivation）

- **解决什么问题**：现有的人形机器人研究主要分为两类：**运动控制**（如行走、跑步）和**手部操作**（如抓取、放置）。然而，人类在执行复杂任务时（如搬运箱子、整理桌子）需要同时协调全身运动（行走、弯腰、伸手）。当前的人形机器人研究在**全身协调操作**方面严重不足。
- **为什么重要**：实现全身协调操作是人形机器人真正融入人类环境（如家庭、学校）的关键。它能让机器人执行更复杂、更自然的任务，而不仅仅是孤立地行走或操作。
- **挑战在哪里**：
    1.  **高维度和紧耦合**：机器人需要在保持动态平衡的同时，产生高维度、紧密耦合的多肢体运动。
    2.  **现有方法的局限**：
        - **分解式方法**：将运动控制和操作控制分开，依赖人工先验，在复杂任务中脆弱，且难以实现如“行走中操作”这样的紧耦合行为。
        - **层次化方法**：高层模块（如VLA）产生任务指令，低层控制器执行。但现有VLA方法往往没有显式建模身体部件如何通过共享的平衡和姿态进行交互，导致生成的全身行为不协调，尤其在快速反应和长周期任务中。
- **核心洞察**：有效的全身操作需要同时具备**具身感知的预测性动态**（embodiment-aware predictive dynamics）和**时间上连贯的场景理解**（temporally grounded scene understanding）。

## 2. 创新点（Innovation）

1.  **首个全身VLA框架**：据作者所知，这是第一个专门为全尺寸双足人形机器人设计的全身VLA框架，将视觉-语言理解与全身协调控制统一起来。
2.  **跨形态人形对齐状态表示与预测性建模**：
    - **创新点**：提出了一种通用状态表示，将不同形态（如不同关节配置、有无手部触觉）的人形机器人的本体感受状态，映射到一组固定的、规范的身体部件（如左臂、右腿）的共享潜在空间。
    - **与现有方案的不同**：现有方法通常需要为每种新形态重新训练状态编码器。HEX通过可重用的部件级编码器和动态专家路由，高效地利用了跨形态数据。
    - **为什么能奏效**：这种表示方式使得一个统一的预测器（UPP）可以在共享空间内学习通用的全身动态，而无需关心底层硬件的具体差异，从而实现了可扩展的跨形态学习。
3.  **“回顾-预测”范式与自适应多模态融合**：
    - **创新点**：提出了一个“回顾-预测”的动作生成范式。
        - **回顾**：通过轻量级历史令牌缓存（History Query Feature Cache）高效地总结过去的视觉-语言上下文，避免了重复编码历史图像。
        - **预测**：通过UPP预测未来的本体感受状态，为动作生成提供“状态前瞻”。
        - **融合**：使用残差门控融合机制，自适应地整合视觉-语言特征（提供语义指导）和预测的本体感受特征（提供动态协调指导）。
    - **与现有方案的不同**：现有方法通常直接融合视觉和语言特征，或简单地将状态作为额外输入。HEX则通过预测未来状态，让动作生成过程能够“看到”未来，从而产生更平滑、更协调的全身行为。
    - **为什么能奏效**：视觉上下文帮助理解场景和任务意图，而预测的未来状态则提供了身体即将如何运动的动态线索。两者的自适应融合（通过门控机制）让模型能够根据当前情况动态调整视觉和状态信息的权重，从而生成更鲁棒的动作。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

HEX采用**层次化架构**：
- **高层VLA策略**：负责理解任务、感知环境、生成任务相关的动作（如手臂和手的动作指令）。
- **低层RL全身控制器**：以高频运行，接收高层指令，并生成保持平衡、动态可行的全身运动（如腿部运动以保持稳定）。

高层VLA策略是核心，包含三个模块：
1.  **视觉-语言模型（VLM）模块**：编码当前视觉、语言指令和历史视觉上下文。
2.  **统一本体感受预测器（UPP）**：建模人形对齐的状态动态，预测未来状态。
3.  **HEX动作专家**：通过自适应融合，整合VLM和UPP的输出，生成最终的高层动作。

### 3.2 关键算法与数学推导

**1. VLM与历史查询特征缓存（History Query Feature Cache）**

- 在每个时间步 $t$，将语言令牌 $L$、视觉令牌 $V_t$ 和查询令牌 $Q_t$ 输入VLM：
  $$[L'_t, V'_t, Q'_t] = f_{vlm}([L, V_t, Q_t])$$
- 将得到的查询特征 $Q'_t$ 存储在固定长度的缓存中：
  $$M^{vl}_t = \{Q'_{t-\tau_v}, \dots, Q'_{t-1}, Q'_t\}$$
  其中 $\tau_v$ 是视觉历史窗口长度（实验中设为2）。
- **作用**：缓存只存储紧凑的查询特征，而非原始图像，避免了重复编码历史图像的高昂计算成本，实现了高效的“视觉回顾”。

**2. 基于形态学MoE的统一本体感受预测器（UPP）**

- **状态对齐**：将原始本体感受状态 $s^{(e)}_t$ 映射到 $P$ 个规范身体部件槽的潜在空间，得到结构化部件潜在表示 $P^{(e)}_t \in \mathbb{R}^{P \times d}$。缺失部件用可学习的“缺失令牌”填充。
- **时空令牌序列**：将当前部件令牌 $P_t$ 和可学习的未来查询令牌 $N_{t+1:t+\tau_p}$ 拼接，并加上时间和部件位置编码，形成共享的时空令牌表示。
- **形态学MoE**：在共享Transformer骨干网络的前后各放置一个MoE模块。每个时空令牌通过一个top-k门控网络路由到少数几个专家，同时还有一个共享专家提供通用变换。这使得不同身体部件和时间步可以根据其局部动态和形态特异性选择不同的专家。
- **预测未来状态**：UPP基于当前本体感受潜在状态 $P_t$、语言特征 $L'_t$、视觉特征 $V'_t$ 和历史缓存 $M^{vl}_t$，预测未来 $\tau_p$ 步的本体感受潜在状态：
  $$P'_{t+1:t+\tau_p} = f_{upp}(P_t, L'_t, V'_t, M^{vl}_t)$$

**3. 带有自适应融合的动作专家（Action Expert）**

- 采用流匹配（Flow-Matching）目标进行动作生成。
- **双交叉注意力**：动作隐藏状态 $X_t$ 作为查询，分别与视觉-语言特征 $H^{vl}_t$ 和预测的本体感受特征 $H^p_t$ 进行交叉注意力计算：
  $$\tilde{H}^{vl}_t = Attn_{vl}(\hat{X}_t, H^{vl}), \quad \tilde{H}^p_t = Attn_{p}(\hat{X}_t, H^p)$$
- **残差门控融合**：通过一个门控机制自适应地融合两个分支的信息：
  $$g_t = \sigma(W_g[\tilde{H}^{vl}_t; \tilde{H}^p_t; \hat{X}_t])$$
  $$F_t = \tilde{H}^{vl}_t + g_t \odot \tilde{H}^p_t$$
  $$X'_t = X_t + F_t$$
  其中 $\sigma$ 是sigmoid函数，$g_t$ 是门控权重，控制本体感受分支的注入强度。

### 3.3 模型、训练与数据集

- **模型**：
    - **VLM**：Qwen3-VL-2B-Instruct
    - **UPP**：4层Transformer，隐藏层大小768，预测未来50步状态。MoE模块有16个路由专家和2个共享专家。
    - **动作专家**：16层DiT-B，隐藏层大小1024，预测未来100步动作。
- **训练**：
    - **预训练**：在超过1200万帧的跨形态数据上训练200k步。优化器为AdamW，使用余弦学习率调度。
    - **微调**：在每个具体任务上微调20k步。
    - **损失函数**：联合优化动作生成损失 $L_a$ 和未来状态预测损失 $L_s$。
        $$L_a = \|D_a(Z^a_t) - Vel_{t:t+\tau_a}\|_2^2$$
        $$L_s = \|D_s(P'_{t+1:t+\tau_p}) - s_{t+1:t+\tau_p}\|_2^2$$
        $$L = L_a + \alpha L_s$$
- **数据集**：包含来自7种人形机器人形态的超过1200万帧数据，包括自采数据（天工2.0/3.0、天轶）、Humanoid Everyday、AgiBot World Colosseo和RoboCOIN等公开数据集。

### 3.4 实验设置与结果

- **硬件**：天工2.0（Tienkung 2.0）和天工3.0（Tienkung 3.0）全尺寸双足人形机器人。
- **基线**：ACT、SwitchVLA、GR00T N1.5、π0.5。
- **任务**：7个真实世界任务，涵盖全身控制、快速反应和长周期操作。
- **主要结果**：
    - **已见场景（Seen Scenarios）**：HEX在7个任务的平均成功率（79.8%）上优于所有基线，尤其在需要全身协调的任务上表现突出（见表1）。
    - **长周期场景（Long-Horizon Scenarios）**：在“长距离搬运箱子”任务中，HEX在所有阶段（尤其是最后的“放置箱子”阶段）都取得了最佳成功率，显示出更强的抗误差累积能力（见表2）。
    - **泛化性（Generalization）**：在8个未见过的场景变体（如快速切换姿势、视觉干扰物、光照变化）中，HEX的平均成功率（61.8%）大幅领先于其他方法（见图6）。
    - **消融实验（Ablation）**：验证了预训练、历史缓存、UPP和MoE模块各自的重要性。UPP的贡献最大（见图7）。
    - **分析**：失败模式分析显示HEX的失败更集中，鲁棒性更强；MoE路由分析显示，Transformer后的MoE能根据任务阶段动态选择专家；延迟分析显示HEX在保持最高成功率的同时，延迟（73.34ms）低于π0.5，具有实用价值。

### 3.5 技术细节

- **低层控制器**：针对不同姿态（站立、行走、半跪）训练了不同的RL策略。站立和行走使用DeepMimic风格，半跪使用AMP风格。
- **数据收集**：采用模块化遥操作管线，头部、手臂、手、腰部和腿部由不同设备控制。
- **推理优化**：对预测的手臂和手部动作进行线性插值，以提高运动平滑度。

## 4. 意义与影响（Significance and Impact）

- **解决了什么问题**：解决了现有VLA模型在人形机器人全身操作中缺乏对**身体部件间结构化交互**和**时序动态**建模的核心问题。HEX通过显式建模这些依赖关系，使得生成的全身行为更加协调、稳定和鲁棒。
- **贡献**：
    1.  提出了首个针对全尺寸双足人形机器人的全身VLA框架。
    2.  提出了一种可扩展的跨形态学习范式，通过人形对齐状态表示和预测性建模，有效利用了异构数据。
    3.  引入了“回顾-预测”和自适应融合的创新范式，显著提升了动作质量和任务成功率。
- **对未来研究/应用的影响**：
    - 为人形机器人全身操作的研究提供了一个新的、有效的基准框架。
    - 其跨形态学习能力为构建通用人形机器人基础模型铺平了道路。
    - 在家庭服务、工业制造、物流搬运等需要复杂全身协调的应用场景中具有巨大潜力。
- **局限与开放问题**（作者提及）：
    - 预训练在单任务微调场景下的最终性能提升有限，主要优势在于优化效率和样本效率。如何更好地利用预训练提升最终性能是未来方向。
    - 论文未明确讨论在更复杂、更动态的环境（如非结构化户外环境）中的表现。
    - 低层控制器是针对特定姿态训练的，如何实现一个统一的、更通用的低层全身控制器是未来的挑战。

## 5. 澄清与简化（Clarifications and Simplifications）

- **人形对齐通用状态表示**：可以想象成给所有不同型号的机器人（比如高矮胖瘦、关节数量不同）都穿上一件“标准化的紧身衣”。这件紧身衣上标记了固定的身体部位（左臂、右腿等）。无论机器人内部结构如何，我们只关心这件“紧身衣”上各个点的运动状态。这样，不同机器人的运动数据就可以在同一个“坐标系”下进行比较和学习。
- **统一本体感受预测器（UPP）**：可以理解为机器人的“运动直觉”。它不只看当前身体是什么姿势，还能根据当前动作和视觉信息，预测出接下来0.5秒内身体各部位（手臂、腿、腰）会如何协调运动。这个“运动直觉”是通过学习大量人类和机器人的运动数据得来的。
- **混合专家模型（MoE）**：可以想象成一个由多个“小专家”组成的团队。当处理“左臂”的运动时，系统会优先激活那些擅长处理手臂运动的“小专家”；当处理“右腿”时，则激活腿部专家。这样，每个“小专家”可以专注于自己擅长的领域，整体效果更好，也更容易适应不同机器人的身体结构。
- **“回顾-预测”范式**：就像人类开车时，既要看后视镜（回顾过去的视觉信息，比如刚才有没有车），也要预测前方路况（预测未来状态，比如前面路口会不会有车出来）。HEX的“回顾”是通过轻量级的历史令牌缓存实现的，而“预测”则是通过UPP实现的。最后，它把“看后视镜”和“预测路况”的信息融合起来，决定如何打方向盘（生成动作）。
