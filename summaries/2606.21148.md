# Pose-Agnostic Robotic Functional Grasping via Observation-Action Canonicalization

- arXiv: https://arxiv.org/abs/2606.21148v1
- arXiv ID: 2606.21148

---

🔍 **核心问题**：机器人功能性抓取中，面对不同几何形状和姿态（如竖直或倒置放置的杯子）的物体时，现有方法难以兼顾泛化性与任务所需的接触精度，尤其是薄把手结构的感知与控制易受姿态变化影响。

🛠 **方法**：提出AnyMug框架，采用**观测-动作规范化**技术，将深度观测和预测的末端执行器动作统一转换到以物体为中心的坐标系中，使策略在不同杯子姿态下看到一致的视角并输出规范化动作；结合**把手感知奖励**、姿态课程学习与域随机化，在仿真中训练闭环策略并零样本迁移到真实机器人。

✨ **主要贡献**：在仿真中对未见过的竖直和倒置杯子均达到93%以上成功率，零样本迁移到真实Franka Panda机器人后，在5个真实杯子上（涵盖两种姿态）达到80%成功率，显著提升了功能性抓取在姿态变化下的泛化能力。
