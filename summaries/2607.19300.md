# LLM Detection as an Intervention: Downstream Impact under Strategic User Behavior

- arXiv: https://arxiv.org/abs/2607.19300v1
- arXiv ID: 2607.19300

---

🔍 **核心问题**：探讨不完美的LLM检测工具在用户策略性行为下，如何对LLM使用率和输出质量等下游指标产生反直觉的影响。

🛠 **方法**：构建了一个简化模型，模拟用户如何策略性调整LLM使用程度及后处理内容以规避检测，并分析检测器对下游指标的扭曲效应。

✨ **主要贡献**：揭示了LLM检测可能反而导致用户增加LLM使用率，并降低输出质量；同时发现检测属性呈现“先升后降”的规律，并在arXiv摘要的词频数据中得到实证验证。
