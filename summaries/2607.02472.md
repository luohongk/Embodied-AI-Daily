# Learning Agile Intruder Interception using Differentiable Quadrotor Dynamics

- arXiv: https://arxiv.org/abs/2607.02472v1
- arXiv ID: 2607.02472

---

🔍 **核心问题**：如何在仅依赖3D方向向量（而非相对位置或距离）的条件下，实现高速（10 m/s）四旋翼飞行器对入侵目标的敏捷拦截控制。

🛠 **方法**：利用可微四旋翼动力学模型，通过解析策略梯度方法直接学习控制策略，避免传统深度强化学习对相对位置信息的依赖。

✨ **主要贡献**：提出一种仅需方向向量即可实现敏捷拦截的学习框架，相比基于简化质点动力学的基线方法，拦截成功率平均提升30%。
