## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人操作臂的网络安全，具体针对虚假数据注入攻击（FDIA）的弹性控制。
- **问题（Problem）**：传统的基于残差的被动异常检测系统（如χ²检验）无法有效检测针对反馈线性化操作臂的“隐蔽”虚假数据注入攻击。攻击者可以利用系统的“积分器脆弱性”，在绕过检测的同时，持续地操纵末端执行器偏离预定轨迹。
- **方法（Method）**：提出一种主动防御策略——“异常感知指令缩放”。该方法利用一个“无测量、仅基于驱动”的状态预测器生成异常分数，并基于该分数通过一个单调递减函数来衰减控制输入指令。
- **创新（Innovation）**：
    1.  将被动检测升级为主动防御，通过修改控制律来限制攻击效果。
    2.  提出了一个新颖的、不受传感器直接污染的“驱动投影状态预测器”来生成异常分数。
    3.  为防御提供了概率性保证（在无攻击时，名义控制性能损失可控）和闭环稳定性证明。
    4.  从攻击者角度，将最优隐蔽攻击建模为一个凸的二次约束二次规划（QCQP）问题。
- **意义（Significance）**：这项工作为机器人操作臂提供了一种实用且轻量级的主动防御机制，将异常检测从单纯的“监控”角色转变为“弹性”机制，显著提升了系统在遭受隐蔽网络攻击时的安全性和可靠性。
- **一句话总结**：本文针对机器人操作臂的隐蔽传感器攻击，提出了一种基于异常感知指令缩放的主动防御方法，通过修改控制律而非仅依赖检测，有效限制了攻击对末端执行器轨迹的操纵，并提供了理论保证。

## 1. 研究动机（Motivation）

- **问题与空白**：机器人操作臂在开放网络环境中易受虚假数据注入攻击（FDIA），攻击者通过篡改传感器信号来误导控制器。传统的被动异常检测系统（ADS），如基于卡尔曼滤波器新息的χ²检验，虽然能检测到明显的异常，但对于精心设计的“隐蔽”攻击却无能为力。这类攻击可以巧妙地使攻击信号保持在检测阈值以下，从而在不触发警报的情况下持续影响系统行为。
- **为什么重要**：对于执行精密任务（如装配、人机协作）的机器人，隐蔽攻击可能导致严重的任务失败、财产损失甚至人身伤害。因此，仅依赖被动检测是不够的，需要更主动的防御机制来提升系统的弹性。
- **挑战在哪里**：
    1.  **积分器脆弱性**：反馈线性化将复杂的机器人动力学简化为双积分器模型。这种结构存在一个固有的脆弱性：持续的传感器偏差会在积分过程中累积，导致状态估计和实际状态之间产生巨大漂移，而基于新息的检测器在稳态下可能无法察觉这种缓慢的累积偏差。
    2.  **检测与防御的分离**：传统ADS仅负责“检测”并发出警报，不参与控制策略的修改。这使得系统在攻击发生时，其控制结构保持不变，攻击者可以持续利用系统动力学来达到其恶意目的。
    3.  **攻击的隐蔽性与最优性**：攻击者可以设计最优的攻击序列，在满足隐蔽性约束（即不触发警报）的前提下，最大化其对系统状态的操纵能力。

## 2. 创新点（Innovation）

- **关键创新**：本文的核心创新在于提出了“异常感知指令缩放”（anomaly-aware command scaling）这一主动防御机制。它将异常检测的结果直接反馈到控制回路中，通过动态调整控制输入的大小来主动对抗攻击。
- **与现有方案的不同**：
    - **从被动到主动**：现有工作多集中于设计更好的被动检测器（如自适应阈值、CUSUM等），而本文直接修改控制律，使系统在检测到异常时能主动“削弱”攻击的影响。
    - **新颖的异常分数**：提出了一个基于“驱动投影状态预测器”的异常分数 $\tilde{z}_k$。这个预测器完全忽略传感器测量，仅依靠模型和控制输入来预测状态，因此其预测误差（与卡尔曼估计的差异）能直接反映传感器被攻击的程度，而不受攻击信号本身的污染。这与传统的基于新息的分数 $z_k$ 有本质区别。
    - **理论保证**：论文不仅提出了方法，还提供了严格的数学证明，包括：
        1.  **概率性保证**：在无攻击时，控制输入被衰减到名义值一定比例以下的概率是可控的（Theorem 3, 1）。
        2.  **稳定性保证**：证明了在冻结增益情况下，即使指令被衰减，闭环系统仍然是Schur稳定的（Theorem 3, 2）。
- **为什么能奏效**：该防御机制创造了一个“Stackelberg博弈”式的动态。攻击者为了达到其目标，需要增加注入量，但这会提高异常分数 $\tilde{z}_k$，进而触发更强的指令衰减。这迫使攻击者在“增大攻击效果”和“避免指令被衰减”之间做出权衡，最终限制了其能实现的最大破坏力。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文提出的系统架构如图1所示，包含以下几个关键部分：

1.  **被动检测层**：一个标准的闭环系统，包含反馈线性化的操作臂（Plant）、稳态卡尔曼滤波器（Kalman Filter）和基于χ²检验的异常检测系统（ADS）。攻击者向传感器信号注入 $a_k$。
2.  **主动防御层**：这是本文的核心贡献。它包含两个新模块：
    - **驱动投影状态预测器**：一个独立于传感器测量的开环状态预测器。它只接收控制输入 $u_k$，并利用系统模型（$A, B$）来预测状态 $\tilde{x}_k$。该预测器会定期与卡尔曼估计 $\hat{x}_k$ 同步以重置漂移。
    - **异常感知指令缩放**：计算预测器状态与卡尔曼估计之间的差异 $\tilde{r}_k = \hat{x}_k - \tilde{x}_k$，并由此生成一个异常分数 $\tilde{z}_k$。然后，通过一个单调递减函数 $f(\tilde{z}_k)$ 来缩放名义控制输入 $u_k^{nom}$，得到最终的执行指令 $u_k = f(\tilde{z}_k) u_k^{nom}$。

### 3.2 关键算法与数学推导

1.  **系统模型**：
    - 离散时间LTI系统：$x_{k+1} = A x_k + B u_k + w_k$, $y_k = C x_k + v_k$。
    - 受攻击的测量值：$\tilde{y}_k = y_k + a_k$。
    - 稳态卡尔曼滤波器：$\hat{x}_{k+1} = A \hat{x}_k + B u_k + L r_k$, $r_k = \tilde{y}_k - C \hat{x}_k$。其中 $L$ 是稳态增益，$r_k$ 是新息，其协方差为 $\Sigma$。

2.  **被动检测**：
    - 马氏距离：$z_k = r_k^\top \Sigma^{-1} r_k \sim \chi^2(p)$。
    - 窗口统计量：$w_k = \sum_{i=k-W+1}^{k} z_i \sim \chi^2(pW)$。
    - 阈值设定：$\tau = F^{-1}_{\chi^2(pW)}(1-\alpha)$，其中 $\alpha$ 是期望的误报率。

3.  **主动防御核心**：
    - **驱动投影状态预测器**：
        $$\tilde{x}_{k+1} = A \tilde{x}_k + B u_k, \quad \tilde{x}_0 = \hat{x}_0$$
    - **驱动投影残差**：
        $$\tilde{r}_k := \hat{x}_k - \tilde{x}_k$$
    - **残差协方差**（Theorem 1）：$\tilde{r}_k$ 的协方差 $\Sigma_{\tilde{r}, k}$ 可以通过一个增广系统的李雅普诺夫方程迭代计算得到：
        $$P_{z, k+1} = F P_{z, k} F^\top + \Pi$$
        其中 $F$ 和 $\Pi$ 由系统矩阵、卡尔曼增益和噪声协方差定义。
    - **异常分数**：
        $$\tilde{z}_k = \tilde{r}_k^\top \Sigma_{\tilde{r}, k}^{-1} \tilde{r}_k \sim \chi^2(n)$$
        其中 $n = 2p$ 是状态维度。**关键点**：$\tilde{z}_k$ 在无攻击时服从 $\chi^2(n)$ 分布，这使得我们可以为其设定一个统计阈值 $z_x$。

4.  **指令缩放函数**：
    $$f(\tilde{z}) = \exp\left(-\left(\frac{\tilde{z}}{z_{scale}}\right)^\gamma\right)$$
    其中 $z_{scale} = \frac{z_x}{(-\ln(\beta))^{1/\gamma}}$。该函数满足 $f(0) = 1$, $f(z_x) = \beta$, 且当 $\tilde{z} \to \infty$ 时 $f(\tilde{z}) \to 0$。
    - **概率性保证**（Theorem 3, 1）：选择 $z_x = F^{-1}_{\chi^2(n)}(\psi)$，则至少有 $\psi$ 的概率，缩放后的指令 $u_k$ 的幅值不低于名义指令的 $\beta$ 倍：$\|u_k\| \ge \beta \|u_k^{nom}\|$。
    - **稳定性保证**（Theorem 3, 2）：对于冻结的缩放因子 $\bar{f} \in (0, 1]$，闭环系统矩阵 $A_{cl}(\bar{f}) = A_j - \bar{f} B_j K$ 是Schur稳定的（即所有特征值在单位圆内），因此系统指数稳定。

5.  **最优隐蔽攻击（QCQP）**：
    - 攻击者目标：最小化末端执行器加速度与期望目标加速度的误差，同时满足隐蔽性约束。
    - **增量攻击模型**：$a_k = a_{k-1} + \Delta_k$。
    - **线性化**：对攻击效果进行一阶泰勒展开：$\ddot{p}^{SIM}_{k+2} \approx \pi_{\ddot{p}}(Z_{k,2}(a_{k-1})) + Z_k \Delta_k$，其中 $Z_k$ 是雅可比矩阵。
    - **隐蔽约束**：$(\Delta_k + c_k)^\top \Sigma^{-1} (\Delta_k + c_k) \le \tau'$，其中 $c_k$ 是基线新息，$\tau'$ 是每步的异常预算。
    - **凸QCQP问题**（Theorem 4）：
        $$\min_{\Delta_k} \frac{1}{2} \Delta_k^\top H \Delta_k + g^\top \Delta_k$$
        $$\text{s.t.} \quad \Delta_k^\top O \Delta_k + b^\top \Delta_k + c \le 0$$
        其中 $H = Z_k^\top Z_k + \zeta I \succ 0$, $O = \Sigma^{-1} \succ 0$，保证了问题的凸性和全局最优解的存在。

### 3.3 模型、训练与数据集

- **模型**：一个6自由度（6-DOF）平面操作臂，其动力学通过反馈线性化被简化为双积分器模型。
- **训练**：本文不涉及传统的机器学习训练。控制器的增益（$K_{pp}, K_{dp}$等）通过离散时间LQR方法在双积分器模型上综合得到。卡尔曼滤波器是稳态的，其增益 $L$ 通过求解离散代数黎卡提方程（DARE）得到。
- **数据集**：没有使用真实世界的数据集。所有实验都是基于一个公开的Matlab仿真实现进行的。攻击者的参考轨迹（从起始点到目标点）由五次多项式插值生成。

### 3.4 实验设置与结果

- **实验设置**：
    - **操作臂**：6-DOF平面操作臂，连杆长度固定。
    - **任务**：保持末端执行器在固定位姿。
    - **攻击者**：试图将末端执行器从初始位置 $[1.48, 1.81]^T$ 米移动到目标位置 $[-2, 1]^T$ 米。
    - **对比方法**：
        - **U (Undefended)**：无任何防御。
        - **PO (Passive Only)**：仅有被动χ²检测器。
        - **D (Active Defence)**：采用本文提出的主动防御。
    - **指标**：任务偏差（dev_max, dev_RMS）和控制努力（mean control effort）。

- **主要结果**（Table I, Fig. 2, Fig. 3, Fig. 4）：
    - **U**：攻击者几乎完美地跟踪恶意轨迹，偏差极小。
    - **PO**：被动检测器迫使攻击者满足隐蔽约束，但攻击者仍能成功到达目标位置，偏差略有增加。检测器从未触发警报。
    - **D**：主动防御显著降低了攻击效果。
        - 在攻击初期（$k<250$），异常分数 $\tilde{z}_k$ 较小，缩放因子 $f \approx 1$，性能与PO类似。
        - 随着攻击进行，$\tilde{z}_k$ 增大，$f$ 开始减小，指令 $u_k$ 被大幅衰减。
        - 攻击者试图通过增大注入 $\Delta_k$ 来对抗，但这进一步推高了 $\tilde{z}_k$，形成负反馈。
        - 最终，$\tilde{z}_k$ 稳定在一个平衡点，指令幅值被显著抑制，末端执行器的实际位移远小于攻击者的目标（图2中D轨迹远未到达目标点）。
        - 从Table I看，主动防御（D）的 `dev_max` 和 `dev_RMS` 相对于攻击者参考轨迹（$\bar{p}^A$）很大，说明攻击失败；而相对于名义任务（$\bar{p}$）的偏差也小于PO，说明任务保持得更好。控制努力 `mean` 也大幅降低。

### 3.5 技术细节

- **反馈线性化**：通过逆动力学补偿 $\tau = M(q) u_{nom} + \nu(q, \dot{q})$，将非线性动力学转化为线性双积分器 $\ddot{q} = u_{nom}$。
- **任务空间控制**：使用PD+前馈控制律计算期望的任务空间加速度 $u_c^k$，然后通过雅可比伪逆 $J^\dagger(q)$ 映射到关节空间加速度 $u_{nom}^k$。
- **驱动投影预测器同步**：预测器 $\tilde{x}_k$ 会定期与卡尔曼估计 $\hat{x}_k$ 同步（论文中假设在 $k=0$ 时同步），以防止开环预测的长期漂移。
- **QCQP求解**：攻击者每步需要求解一个凸QCQP问题，论文中使用Gurobi求解器，每步耗时约40ms，表明该攻击策略在计算上是可行的。

## 4. 意义与影响（Significance and Impact）

- **解决了什么问题**：解决了传统被动异常检测系统无法有效防御针对反馈线性化机器人操作臂的隐蔽FDIA攻击的问题。它揭示了“积分器脆弱性”这一关键漏洞，并提出了一个切实可行的主动防御方案。
- **贡献是什么**：
    1.  **理论贡献**：形式化了隐蔽FDIA攻击对反馈线性化操作臂的威胁，并证明了最优攻击策略是一个凸QCQP问题。
    2.  **方法贡献**：提出了“异常感知指令缩放”这一新颖的主动防御框架，并提供了概率性性能保证和稳定性证明。
    3.  **实践贡献**：通过仿真实验验证了该防御方法的有效性，表明其能显著限制攻击者的破坏力，同时在无攻击时对名义任务性能影响极小。
- **对未来研究/应用的影响**：
    - 为机器人系统的网络安全设计提供了新思路，即从被动检测转向主动弹性控制。
    - 提出的“驱动投影状态预测器”和“异常感知缩放”框架可以推广到其他具有类似动力学特性的CPS系统。
    - 论文中建立的Stackelberg博弈模型为未来研究攻防双方的最优策略提供了基准。
- **局限与开放问题**：
    - **时变稳定性**：论文仅证明了冻结增益下的稳定性，而实际中 $f(\tilde{z}_k)$ 是时变的。作者提到未来工作可以研究低通滤波、概率下界或慢时变系统稳定性理论来扩展证明。
    - **安全性**：虽然概率性保证表明无攻击时性能损失极小，但统计波动仍可能触发不必要的衰减。如何设计更鲁棒的缩放函数是一个开放问题。

## 5. 澄清与简化（Clarifications and Simplifications）

- **积分器脆弱性（Integrator Vulnerability）**：想象一个积分器，你持续给它一个很小的正输入，它的输出会不断累积，最终变得很大。在机器人控制中，如果攻击者持续注入一个微小的传感器偏差，这个偏差经过控制器中的积分环节（例如，位置误差的积分）会不断累积，导致末端执行器产生巨大的位置漂移。而传统的χ²检测器检查的是“新息”（即预测与测量的差异），如果偏差是缓慢注入的，这个差异可能始终很小，从而躲过检测。这就是“积分器脆弱性”。
- **驱动投影状态预测器**：可以把它想象成一个“盲人”预测器。它不看传感器（盲），只凭自己的“记忆”（系统模型）和“动作”（控制输入）来猜测机器人当前的状态。如果传感器被攻击，卡尔曼滤波器（“正常人”）的状态估计会被带偏。此时，“盲人”预测器的状态（因为没有受到攻击影响）和“正常人”的估计之间就会出现差异。这个差异就是检测攻击的绝佳信号，因为它直接反映了传感器被污染的程度。
- **异常感知指令缩放**：可以理解为一种“怀疑机制”。当系统“怀疑”自己可能被攻击时（即
