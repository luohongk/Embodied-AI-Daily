# FF-ProCams: Feed-Forward Gaussian Splatting for Projector-Camera System

- arXiv: https://arxiv.org/abs/2607.17803v1
- arXiv ID: 2607.17803

---

🔍 **核心问题**：投影仪-相机系统（ProCams）在空间增强现实、投影映射等应用中需要高保真渲染，但现有方法要么依赖耗时的逐场景优化，要么无法适应投影仪光照变化，难以兼顾精度与效率。

🛠 **方法**：提出FF-ProCams框架，采用混合Mamba2-Transformer编码器从稀疏多视角观测中聚合几何与光度线索，通过轻量级预测头一次性输出可重光照的高斯表示，并设计投影仪感知的可微分渲染器合成任意主动光照下的相机图像。

✨ **主要贡献**：仅需8个输入视角即可实现高保真投影感知渲染，优于使用297视角的优化基线方法，测试时重建速度达0.13秒（加速3-5个数量级），且能泛化到未见光照模式并支持新视角合成。
