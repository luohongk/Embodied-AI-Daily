# UI-MOPD: Multi-Platform On-Policy Distillation for Continual GUI Agent Learning

- arXiv: https://arxiv.org/abs/2607.04425v1
- arXiv ID: 2607.04425

---

🔍 **核心问题**：多平台GUI代理在持续学习中面临跨平台交互轨迹稀缺、平台特有行为模式混淆及灾难性遗忘等问题，导致跨平台能力退化。

🛠 **方法**：提出UI-MOPD方法，通过构建高质量跨平台GUI数据集Uni-GUI，并首次将多教师在线策略蒸馏融入持续学习，动态选择平台专属教师，通过平台条件蒸馏将行为先验迁移至共享策略。

✨ **主要贡献**：在OSWorld和MobileWorld上分别取得38.2%和12.0%的任务成功率，有效平衡了跨平台能力保留与新平台适应，验证了方法的有效性。
