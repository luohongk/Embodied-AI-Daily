# Forget to Improve: On-Device LLM-Agent Continual Learning via Budget-Curated Memory

- arXiv: https://arxiv.org/abs/2606.25115v1
- arXiv ID: 2606.25115

---

🔍 **核心问题**：设备端语言模型代理在持续学习时，面临内存预算有限、上行带宽不足以及易受恶意注入攻击的问题，现有方法未能统一解决这些挑战。

🛠 **方法**：提出一个名为“净价值每字节”的评分机制，通过预算作为策展者，统一决定内存中条目的保留、共享和信任，具体包括低价值条目驱逐、高价值洞察分享以及基于来源的信任门控。

✨ **主要贡献**：在任务漂移基准和真实异构Jetson测试平台上，该方法将内存占用减少2.7倍、上行流量减少2.4倍，将注入攻击成功率从0.75降至零，同时提升受污染或过时内存影响的准确率，证明通过净价值遗忘能增强而非削弱代理性能。
