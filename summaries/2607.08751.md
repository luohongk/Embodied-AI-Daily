# DexVerse: A Modular Benchmark for Multi-Task, Multi-Embodiment Dexterous Manipulation

- arXiv: https://arxiv.org/abs/2607.08751v1
- arXiv ID: 2607.08751

---

🔍 **核心问题**：现有灵巧操作基准在任务多样性、数据丰富度、机器人形态覆盖和视觉变化可控性方面存在局限，难以系统评估策略的跨任务和跨形态泛化能力。

🛠 **方法**：提出DexVerse模块化基准，包含100个任务（如抓取、工具使用、双手协调等）、3种机械臂和6种灵巧手，并提供可配置的视觉变化（纹理、背景、光照、视角）及VR遥操作接口与3180条演示数据。

✨ **主要贡献**：构建了大规模、可扩展的灵巧操作测试平台，通过19个任务的代表性方法（如Diffusion Policy、DP3、OpenVLA等）基准测试，揭示了任务泛化和视觉运动鲁棒性方面的显著挑战。
