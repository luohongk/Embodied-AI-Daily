# MOPD: Multi-Teacher On-Policy Distillation for Capability Integration in LLM Post-Training

- arXiv: https://arxiv.org/abs/2606.30406v1
- arXiv ID: 2606.30406

---

🔍 **核心问题**：大语言模型后训练中，如何高效整合多个领域的能力，避免现有方法（如离线微调、混合强化学习）效率低或性能损失的问题。

🛠 **方法**：提出多教师在线蒸馏（MOPD）范式，先对每个领域进行专门的强化学习训练获得领域教师，再让学生模型基于自身生成轨迹进行在线蒸馏，从而消除暴露偏差并提供密集优化信号。

✨ **主要贡献**：在Qwen3-30B-A3B上，MOPD优于混合强化学习、级联强化学习、离线微调和参数融合等基线，几乎完整继承每位教师的能力；支持领域教师并行独立开发，消除跨领域耦合，并已成功部署于工业级模型MiMo-V2-Flash的后训练中。
