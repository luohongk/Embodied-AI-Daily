# Tunable MAGMAX: Preference-Aware Model Merging for Continual Learning

- arXiv: https://arxiv.org/abs/2605.20803v2
- arXiv ID: 2605.20803

---

🔍 **核心问题**：现有持续学习方法主要关注所有任务的平均性能，无法根据部署环境或用户偏好灵活调整各任务的表现。

🛠 **方法**：提出Tunable MAGMAX框架，通过引入偏好向量控制模型合并时从每个任务向量中选取的元素数量，并利用少量目标环境数据自动构建合适的偏好向量。

✨ **主要贡献**：实现了对持续学习模型各任务性能的偏好感知控制，在基准测试中达到或超越现有方法，为不同部署环境下的模型适配提供了实用方案。
