# RoboSSM: Scalable In-context Imitation Learning via State-Space Models

- arXiv: https://arxiv.org/abs/2509.19658v2
- arXiv ID: 2509.19658

---

🔍 **核心问题**：现有基于Transformer的上下文模仿学习方法在处理长提示时存在计算效率低和泛化能力不足的问题。  
🛠 **方法**：提出RoboSSM，用状态空间模型Longhorn替代Transformer，实现线性时间推理和强外推能力，支持长上下文提示。  
✨ **主要贡献**：首次证明状态空间模型可作为上下文模仿学习的高效可扩展骨干网络，在LIBERO基准上对未见任务和长时任务均实现优于Transformer方法的泛化性能。
