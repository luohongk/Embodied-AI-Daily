# Deep Reinforcement Learning-Empowered Wireless Sensor Networking for 6G Closed-Loop Controls

- arXiv: https://arxiv.org/abs/2607.08272v1
- arXiv ID: 2607.08272

---

🔍 **核心问题**：针对6G闭环控制系统中，远程或危险环境下机器人依赖传感器和边缘信息枢纽（EIH）进行状态感知与控制时，如何优化传感器到EIH的带宽分配以最小化线性二次型调节器（LQR）控制成本。

🛠 **方法**：基于互信息理论建模通信数据率受限导致的失真噪声，将控制过程建模为部分可观测马尔可夫决策过程（POMDP），并采用深度强化学习（DRL）中的近端策略优化（PPO）算法训练智能体，实现传感器到EIH的带宽分配方案。

✨ **主要贡献**：首次将DRL应用于6G闭环控制中的带宽分配问题，通过PPO算法优化LQR控制成本，仿真结果验证了所提方案相比传统方法的优越性。
