# Synthetic-to-Real Pipeline for Safe Landing Zone Detection

- arXiv: https://arxiv.org/abs/2606.14767v1
- arXiv ID: 2606.14767

---

🔍 **核心问题**：无人机在非合作、非结构化环境中实现安全自主降落时，缺乏高精度标注的航拍数据集，导致语义分割和障碍物检测困难。

🛠 **方法**：提出一个合成到真实的感知与数据生成流程，包括通过域随机化生成逼真城市环境的程序化合成数据引擎，并基于合成数据微调Transformer架构的OneFormer模型；同时利用欧几里得距离变换和动态推理逻辑确定最大内接安全降落区域。

✨ **主要贡献**：在UAVid数据集上验证了鲁棒的语义分割性能，并在真实无人机视频中成功识别无碰撞降落点，证明高保真程序化仿真可替代人工标注，实现边缘部署的自主降落感知能力。
