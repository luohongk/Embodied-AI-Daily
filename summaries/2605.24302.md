# Cross-Modal Action Recognition in Egocentric Video Using Mamba: Integrating RGB and Hand Skeleton Streams via CLS Token Fusion Strategies

- arXiv: https://arxiv.org/abs/2605.24302v2
- arXiv ID: 2605.24302

---

🔍 **核心问题**：解决第一人称视角视频中动作识别因相机运动剧烈、手部频繁遮挡及视觉表征不稳定而面临的挑战。  
🛠 **方法**：提出基于Mamba的跨模态架构，融合RGB视频流与手部骨架时序数据，通过四种CLS令牌混合策略（朴素、平均、加权、上下文）实现多模态融合。  
✨ **主要贡献**：设计并评估了四种CLS令牌融合策略，其中平均策略在H2O数据集上相比VideoMamba基线，Tiny配置下Top-1准确率提升超10%，Small配置下提升2%。
