# SafeDojo: Safe Reinforcement Learning for VLA via Interactive World Model

- arXiv: https://arxiv.org/abs/2606.20698v1
- arXiv ID: 2606.20698

---

🔍 **核心问题**：现有安全强化学习方法依赖真实世界探索或手工设计的安全函数，难以扩展到开放世界中视觉-语言-动作（VLA）模型的安全控制。

🛠 **方法**：提出SafeDojo框架，基于交互式视频世界模型进行在线强化学习，通过世界模型生成动作条件未来预测，并利用ResNet成功分类器和轻量级安全头分别评估任务进度与轨迹安全，结合拉格朗日约束的GRPO目标平衡任务奖励与安全成本。

✨ **主要贡献**：在SafeLIBERO基准上实现最优任务成功率、安全成功率及执行效率，Level I安全成功率比最强基线提升8.25个百分点；真实Franka机器人实验中，五项任务的平均任务成功率和安全成功率均达最佳，验证了基于世界模型的安全强化学习在具身智能中的可扩展性与泛化性。
