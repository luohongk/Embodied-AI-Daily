# UniLM-Nav: A Unified Framework for Zero-Shot Last-Mile Navigation

- arXiv: https://arxiv.org/abs/2607.06537v1
- arXiv ID: 2607.06537

---

🔍 **核心问题**：解决移动机器人在零样本开放词汇场景下的“最后一英里导航”问题，即到达目标附近后无法确保机械臂操作所需的精确基座姿态。

🛠 **方法**：提出UniLM-Nav框架，通过共享多模态大语言模型（MLLM）依次完成参考视角选择、任务条件化可操作点定位和几何感知基座姿态推理，将2D视觉信息映射到机器人坐标系。

✨ **主要贡献**：在OVMM基准上超越先前最优方法MoTo 3.13个百分点，验证了各模块的关键性及MLLM选择的影响，并在Unitree B2四足机器人上成功部署，证明其真实世界移动操作任务的适用性。
