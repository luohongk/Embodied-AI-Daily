# ROVER: Routing Object-Centric Visual Evidence for Grounded Multi-Image Reasoning

- arXiv: https://arxiv.org/abs/2605.27959v2
- arXiv ID: 2605.27959

---

🔍 **核心问题**：现有基于视觉定位的多模态大模型在处理多图像推理时，要么因注入裁剪图像块或区域特征而削弱全局场景理解与对象间关系，要么因自适应特征选择需要精细监督或复杂启发式规则，导致计算成本高且效率低。

🛠 **方法**：提出ROVER，一种轻量级可学习插件，通过在每个对象定位预测时注入步骤特定的令牌三元组，利用对象中心差分注意力聚合推理上下文、提炼图像内线索，并在视觉工作空间中跨对象和图像路由历史感知证据，集成到Qwen2.5-VL-7B中并采用SFT到GRPO的交替训练流程。

✨ **主要贡献**：在MM-GCoT上实现答案准确率提升4.8%、定位准确率提升14.6%，在VideoEspresso上答案准确率提升8.6%，且VideoEspresso训练模型展现出强迁移性，在多个基准上平均提升4.7%。
