# TimEE: End-to-end Time Series Classification via In-Context Learning

- arXiv: https://arxiv.org/abs/2607.07500v1
- arXiv ID: 2607.07500

---

🔍 **核心问题**：传统时间序列分类（TSC）采用两阶段范式（先训练特征编码器，再拟合分类器），导致表示学习与分类目标脱节、需针对每个数据集训练，且无法在推理时利用标签信息。

🛠 **方法**：提出TimEE，一个仅4.5M参数的端到端基础模型，基于先验数据拟合网络（PFN）框架，通过纯合成时间序列任务进行元训练，在推理时直接对带标签的支持集和查询序列输出类别分布，无需针对特定数据集训练。

✨ **主要贡献**：首次证明纯合成预训练模型可在UCR基准上达到SOTA性能（ROC AUC排名第一，准确率第三），为TSC开辟了端到端上下文学习这一极具潜力但尚未充分探索的方向。
