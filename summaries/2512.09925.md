# GAINS: Gaussian-based Inverse Rendering from Sparse Multi-View Captures

- arXiv: https://arxiv.org/abs/2512.09925v2
- arXiv ID: 2512.09925

---

🔍 **核心问题**：解决基于高斯泼溅的逆渲染方法在稀疏多视角输入下，因观测不足导致几何、反射率和光照估计存在严重歧义的问题。

🛠 **方法**：提出GAINS两阶段逆渲染框架，第一阶段利用单目深度、法线和扩散先验优化几何，第二阶段结合分割、内在图像分解（IID）和扩散先验正则化材质恢复，统一基础模型先验与物理基表示。

✨ **主要贡献**：在稀疏视角（4-32相机）下显著提升材质参数精度、重光照质量和新视角合成效果，尤其在高歧义场景中优于现有高斯逆渲染方法。
