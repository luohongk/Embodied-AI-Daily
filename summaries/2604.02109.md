# ROS 2-Based LiDAR Perception Framework for Mobile Robots in Dynamic Production Environments, Utilizing Synthetic Data Generation, Transformation-Equivariant 3D Detection and Multi-Object Tracking

- arXiv: https://arxiv.org/abs/2604.02109v1
- arXiv ID: 2604.02109

---

🔍 **核心问题**：动态生产环境中移动机器人对6D姿态估计与多目标跟踪的鲁棒感知需求，以及真实数据依赖、噪声鲁棒性和时空一致性的局限性。

🛠 **方法**：提出基于ROS 2的LiDAR感知框架，结合合成数据训练的变换等变3D检测与基于中心姿态的多目标跟踪技术。

✨ **主要贡献**：在72个场景中验证，独立姿态估计的交并比达62.6%，集成多目标跟踪后提升至83.12%，高阶跟踪精度达91.12%，显著增强了工业移动机器人感知系统的鲁棒性与通用性。
