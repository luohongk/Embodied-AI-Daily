# RAVEN: Reinforcement-Adaptive Visibility-Graph Planning for Robust Humanoid Navigation with Collision-Free MPC

- arXiv: https://arxiv.org/abs/2607.15701v1
- arXiv ID: 2607.15701

---

🔍 **核心问题**：人形机器人在动态环境中导航时，需兼顾长期路径规划与短期动态安全约束，但传统几何规划器依赖手动调参，易受控制延迟、状态估计噪声和运动不确定性影响，导致轨迹超调或约束违反。

🛠 **方法**：提出RAVEN框架，采用分层强化学习（RL）自适应调整可见性图规划器的障碍物膨胀参数和图形结构，以重塑自由空间几何并补偿延迟与跟踪误差；随后由无碰撞模型预测控制（MPC）层跟踪路径，显式执行速度与避障约束。

✨ **主要贡献**：通过RL自适应修改全局路径拓扑，结合约束MPC，在延迟和噪声下显著减少障碍物附近超调、提升窄通道鲁棒性，相比手动调参基线或纯RL策略更可靠，且保持可解释的几何规划与优化特性。
