# Traj-VLN: Learning Pixel-Space Interaction via Autoregressive Trajectory Generation

- arXiv: https://arxiv.org/abs/2607.10744v1
- arXiv ID: 2607.10744

---

🔍 **核心问题**：在连续环境视觉语言导航（VLN-CE）中，如何让基于视觉语言模型（VLM）的智能体有效理解并执行涉及深度感知的三维空间交互指令，而无需引入VLM预训练中罕见的深度或三维几何信息。

🛠 **方法**：提出Traj-VLN方法，通过自回归轨迹生成直接在二维像素空间中学习导航交互。模型根据语言指令和历史观测，依次预测一系列像素坐标，在当前观测的底部中心绘制一条轨迹，从而将导航任务分解为像素级子任务。

✨ **主要贡献**：首次证明像素空间轨迹监督能显著提升VLN性能，且该方法在有限计算资源和训练数据下达到最先进水平，避免了VLM对深度信息的依赖，实现了跨场景的泛化能力。
