# Discriminative Barrier Functions for Safe Adversarial Imitation Learning from Observation

- arXiv: https://arxiv.org/abs/2607.13938v1
- arXiv ID: 2607.13938

---

🔍 **核心问题**：逆强化学习（IRL）在探索过程中缺乏安全性保障，而控制障碍函数（CBF）的设计复杂且耗时，两者难以在实际部署中兼顾安全与效率。

🛠 **方法**：提出将IRL中的奖励函数限制在CBF空间内，通过无标签专家观测数据直接学习障碍函数，并利用该函数约束在线控制行为，实现安全探索与持续优化。

✨ **主要贡献**：首次实现从无标签数据中恢复鲁棒的障碍函数，即使面对专家数据中未出现的危险状态也能保证安全；在仿真导航任务中相比标准IRL方法显著提升安全性，并在真实避障任务中验证了规划型与策略型IRL方法的权衡。
