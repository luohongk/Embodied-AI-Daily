# KungfuBot: Physics-Based Humanoid Whole-Body Control for Learning Highly-Dynamic Skills

- arXiv: https://arxiv.org/abs/2506.12851v3
- arXiv ID: 2506.12851

---

🔍 **核心问题**：现有算法仅能跟踪平滑、低速的人类动作，难以让仿人机器人掌握如功夫、舞蹈等高动态技能。

🛠 **方法**：提出基于物理的仿人全身控制框架，包含多步运动处理管道（提取、滤波、修正、重定向）和自适应运动跟踪机制，通过双层优化动态调整跟踪精度容差，并采用非对称演员-评论家架构进行策略训练。

✨ **主要贡献**：在多种高动态动作上实现显著更低的跟踪误差，成功部署于Unitree G1机器人，展现出稳定且富有表现力的行为。
