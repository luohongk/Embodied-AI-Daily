# Data Alchemy: Mitigating Cross-Site Model Variability Through Test Time Data Calibration

- arXiv: https://arxiv.org/abs/2407.13632v2
- arXiv ID: 2407.13632

---

🔍 **核心问题**：解决深度学习临床影像工具在不同医疗站点部署时，因数据域偏移（如染色差异）导致的模型性能下降问题，且避免站点特定微调带来的监管负担。

🛠 **方法**：提出Data Alchemy框架，结合可解释的染色归一化方法与模板学习框架，在测试时对数据进行校准，无需修改模型权重即可减少跨站点差异。

✨ **主要贡献**：在肿瘤分类任务中，将AUPR从0.545提升至0.852（先提升0.165至0.710，再额外提升0.142），显著缩小多站点分类域差距，且能以最小操作开销实现预训练工具的无缝跨站点集成。
