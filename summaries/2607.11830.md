# MicroCharNet: Less is More for License Plate Character Detection

- arXiv: https://arxiv.org/abs/2607.11830v1
- arXiv ID: 2607.11830

---

🔍 **核心问题**：解决车牌字符检测中高精度模型计算开销大、难以在资源受限设备上实时部署的问题。  
🛠 **方法**：提出超轻量级模型MicroCharNet，采用C2f块紧凑骨干网络、CoordAtt模块增强特征提取、C3k2轻量颈部融合多级特征，以及单层无锚点检测头实现端到端预测。  
✨ **主要贡献**：在UFPR-ALPR数据集上仅用0.08M参数和0.096 GFLOPs即达到竞争性检测精度，优于多个YOLO基线，并在边缘设备上验证了实时部署效率。
