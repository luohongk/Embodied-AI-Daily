# PCFootprint: A Large-Scale Dataset and Benchmark for Vectorized Building Footprint Extraction from Aerial LiDAR Point Clouds

- arXiv: https://arxiv.org/abs/2606.20455v1
- arXiv ID: 2606.20455

---

🔍 **核心问题**：现有基于光学影像的建筑足迹提取方法易受遮挡、透视畸变和缺乏高程信息的影响，导致提取结果不完整或错位，且难以直接用于精细建筑建模。

🛠 **方法**：构建了首个大规模航空激光雷达点云建筑足迹提取数据集PCFootprint，包含33000个来自爱沙尼亚的128×128米瓦片，并配套系统对齐的矢量化足迹，同时设立3000个瓦片的跨域测试集以评估泛化能力。

✨ **主要贡献**：提供了公开可用的基准数据集和评估框架，通过主流方法实验揭示了复杂地理环境中类内方差高、数据不平衡和噪声等关键挑战，为建筑建模、城市场景理解及地理空间分析研究奠定基础。
