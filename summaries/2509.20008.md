# Learning Robust Penetration Testing Policies under Partial Observability: A systematic evaluation

- arXiv: https://arxiv.org/abs/2509.20008v2
- arXiv ID: 2509.20008

---

🔍 **核心问题**：解决渗透测试中部分可观测性导致的马尔可夫性质失效问题，提升强化学习策略在真实复杂网络环境下的鲁棒性和泛化能力。  
🛠 **方法**：以PPO算法为基线，对比多种缓解部分可观测性的变体（如帧堆叠、历史观测增强、LSTM和TrXL架构），并在不同规模主机网络上进行系统实验。  
✨ **主要贡献**：发现历史聚合方法（如LSTM/TrXL）可使策略收敛速度提升4倍，且通过人工策略分析揭示了超越量化指标的算法行为差异。
