# LIPP: Load-Aware Informative Path Planning with Physical Sampling

- arXiv: https://arxiv.org/abs/2603.06924v2
- arXiv ID: 2603.06924

---

🔍 **核心问题**：传统信息路径规划（C-IPP）假设机器人采集数据时物理状态不变，忽略了物理采样中样本质量增加导致后续运动能耗上升的问题，导致规划结果在能量预算下采集样本数量不足。

🛠 **方法**：提出负载感知信息路径规划（LIPP），将其建模为混合整数二次规划（MIQP），联合优化访问位置、顺序及每个位置的采样次数，以在能量预算下最大化信息获取。

✨ **主要贡献**：首次严格推广C-IPP以显式建模信息增益与负载相关能耗的耦合，理论推导了LIPP相比C-IPP的路径长度增加上界，并通过2000个任务场景的仿真验证了LIPP在样本质量增加时单位能量下的不确定性降低效果更优。
