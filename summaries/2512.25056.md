## 0. 概览（Concise Summary）

- **主题（Topic）**：在线数据同化、贝叶斯参数-状态联合估计、变分推断。
- **问题（Problem）**：在动态系统中，如何利用带有噪声且不完整的观测数据，在线（实时）且高效地联合估计未知的系统参数和系统状态，并同时提供可靠的量化不确定性。
- **方法（Method）**：提出一个名为**基于分解的在线变分推断（FBOVI）** 的框架。该框架将联合后验分布分解为参数的边缘分布和给定参数下的状态条件分布，并通过一个两阶段过程进行递归更新：第一阶段使用变分推断（VI）近似参数后验；第二阶段基于近似得到的参数后验，使用高斯滤波方法（如卡尔曼滤波、无迹卡尔曼滤波、集合卡尔曼滤波）计算状态的条件分布。
- **创新（Innovation）**：
    1.  提出了一种灵活、可扩展的在线变分推断框架，避免了传统方法（如联合高斯滤波）对后验分布形式的强假设（如高斯假设）。
    2.  为所提出的分解策略提供了**理论保证**（定理1和定理2），给出了近似误差的上界，为算法设计提供了理论指导。
    3.  框架可以灵活地集成多种高斯滤波方法（包括可扩展的集合卡尔曼滤波EnKF），使其能够处理高维状态空间问题。
- **意义（Significance）**：该方法为数字孪生等需要在线、可靠且具有不确定性量化的应用提供了强大的工具。它在低维问题上能达到与联合粒子滤波相当的精度，同时在高维问题上比联合集合卡尔曼滤波表现更好，并且具有理论支撑，增强了可信度。
- **一句话总结**：本文提出了一种名为FBOVI的、有理论保证的在线变分推断框架，通过分解联合后验并采用两阶段递归更新策略，高效且准确地解决了动态系统中带噪声和不完整观测数据的参数-状态联合估计问题。

## 1. 研究动机（Motivation）

- **核心问题**：在许多数据驱动的应用中，如数字孪生（Digital Twins, DTs），需要**在线**地、持续地更新对动态系统模型参数和状态的认知，并同时提供**不确定性量化**。然而，真实世界的观测数据通常是**有噪声**且**不完整**的，并且模型本身可能与真实系统存在**失配**，这引入了巨大的不确定性。
- **现有方法的不足**：
    - **联合高斯滤波器**（如联合UKF、联合EnKF）：假设联合后验是高斯分布，这在参数和状态关系高度非线性时会产生较大误差。
    - **联合粒子滤波器（PF）**：精度高，但静态参数的存在会加剧粒子权重退化问题，导致所需粒子数随状态和参数维度急剧增长，难以扩展到高维或中等规模问题。
    - **SMC²**：一种嵌套滤波方法，但其每步计算成本随时间增长，不适合在线应用。
    - **现有在线变分推断方法**：大多针对线性系统设计，且通常假设参数和状态独立（均值场近似），忽略了它们之间的依赖关系，导致近似不准确。
    - **缺乏理论保证**：很少有在线方法能提供近似精度的理论保证，而理论是建立可信赖系统的基础。
- **挑战**：设计一个既能提供高精度近似，又能扩展到高维问题，同时具备理论保证的在线参数-状态联合估计算法。

## 2. 创新点（Innovation）

1.  **提出FBOVI框架**：一个灵活、可扩展的在线变分推断框架。其核心创新在于将近似联合后验 $q_k(X_k, \theta)$ 分解为参数边缘分布 $\nu_k(\theta)$ 和给定参数下的状态条件分布 $\rho_k(X_k|\theta)$。这种分解巧妙地保留了参数和状态之间的依赖关系，避免了均值场近似的缺陷。
2.  **两阶段递归更新策略**：基于上述分解，提出一个两阶段策略：
    - **阶段1**：通过最大化证据下界（ELBO）来最小化 $\nu_k(\theta)$ 与“在线参数后验” $\nu^*_k(\theta)$ 之间的KL散度，从而更新参数后验。
    - **阶段2**：基于阶段1得到的 $\nu_k(\theta)$，通过最小化期望KL散度来更新状态条件分布 $\rho_k(X_k|\theta)$，这可以通过监督学习（用神经网络拟合目标均值和协方差）或直接使用高斯滤波实现。
3.  **理论保证**：提出了**定理1**，建立了真实联合后验 $p(X_k, \theta|\mathcal{Y}_k)$ 与近似联合后验 $q_k(X_k, \theta)$ 之间误差的上界。该上界由每个时间步两个KL散度项（$\text{KL}(\nu_j \| \nu^*_j)$ 和 $\mathbb{E}_{\theta \sim \nu_j}[\text{KL}(\rho_j \| \rho^*_j)]$）的累积和决定。这为两阶段策略的有效性提供了理论依据。此外，**定理2**针对线性系统给出了一个可计算的误差上界。
4.  **灵活性与可扩展性**：框架可以集成多种高斯滤波方法（KF, UKF, EnKF），特别是支持EnKF，使其能够扩展到高维状态空间问题。同时，$\nu_k(\theta)$ 的分布族也可以灵活选择。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

FBOVI框架的核心思想是递归地近似联合后验 $p(X_k, \theta|\mathcal{Y}_k)$。由于直接计算不可行，框架引入了“在线目标”的概念：
- **在线参数后验** $\nu^*_k(\theta)$：使用前一步的近似 $\rho_{k-1}$ 和 $\nu_{k-1}$ 替代真实分布后，通过贝叶斯公式计算得到的参数后验。
- **在线条件滤波分布** $\rho^*_k(X_k|\theta)$：使用前一步的近似 $\rho_{k-1}$ 替代真实分布后，通过预测-更新步骤得到的条件状态后验。

框架的目标是让 $\nu_k(\theta)$ 逼近 $\nu^*_k(\theta)$，让 $\rho_k(X_k|\theta)$ 逼近 $\rho^*_k(X_k|\theta)$。定理1保证了这种逼近策略的有效性。

**两阶段递归更新（每个时间步 $k$）**：
1.  **阶段1：更新参数后验 $\nu_k(\theta)$**
    - 通过最小化 $\text{KL}(\nu_k \| \nu^*_k)$ 来实现。这等价于最大化一个证据下界（ELBO） $\mathcal{L}_k(\nu_k)$。
    - $\mathcal{L}_k(\nu_k) = \mathbb{E}_{\theta \sim \nu_k}[\log I(\theta)] - \text{KL}(\nu_k \| \nu_{k-1})$，其中 $I(\theta) = \int p_{\mathcal{N}}(y_k; h(X_k;\theta), \Gamma(\theta)) \rho^{*-}_k(X_k|\theta) dX_k$ 是边缘似然。
    - $\rho^{*-}_k(X_k|\theta)$ 是在线条件预测分布，由 $\rho_{k-1}$ 通过系统动力学传播得到。对于线性系统，它是高斯分布，$I(\theta)$ 有闭式解。对于非线性系统，用高斯分布近似 $\rho^{*-}_k$，然后通过无迹变换或高斯求积计算 $I(\theta)$。

2.  **阶段2：更新状态条件分布 $\rho_k(X_k|\theta)$**
    - 目标是找到 $\rho_k(X_k|\theta)$ 来最小化 $\mathbb{E}_{\theta \sim \nu_k}[\text{KL}(\rho_k \| \rho^*_k)]$。
    - 对于线性系统，$\rho^*_k$ 是高斯分布，其均值和协方差可由卡尔曼滤波公式给出。为了保持计算成本不随时间增长，使用**神经网络**来参数化 $\rho_k$ 的均值 $m_k(\theta)$ 和协方差 $C_k(\theta)$，并通过监督学习（最小化与目标 $m^*_k(\theta), C^*_k(\theta)$ 的误差）来训练。
    - 对于非线性系统，首先使用高斯滤波（如UKF）来近似 $\rho^*_k$ 为一个高斯分布，得到目标 $m^*_k(\theta), C^*_k(\theta)$，然后同样使用神经网络进行监督学习。
    - **集成EnKF**：为了处理高维问题，框架支持使用EnKF。通过“每步重采样”策略，从 $\rho_{k-1}$ 中采样粒子，用EnKF更新，然后计算样本均值和协方差作为目标 $m^*_k(\theta), C^*_k(\theta)$。

### 3.2 关键算法与数学推导

**1. 问题形式化（公式1）**：
$$X_{k+1} = \Phi(X_k; \theta) + W_k, \quad W_k \stackrel{i.i.d.}{\sim} \mathcal{N}(0, \Sigma(\theta))$$
$$Y_k = h(X_k; \theta) + V_k, \quad V_k \stackrel{i.i.d.}{\sim} \mathcal{N}(0, \Gamma(\theta))$$

**2. 证据下界（ELBO）推导（公式10-11）**：
从 $\nu^*_k(\theta)$ 的表达式出发，推导出 $\text{KL}(\nu_k \| \nu^*_k)$ 的表达式，并提取出ELBO：
$$\mathcal{L}_k(\nu_k) := \mathbb{E}_{\theta \sim \nu_k(\cdot)} \left[ \log I(\theta) \right] - \text{KL}(\nu_k \| \nu_{k-1})$$
其中 $I(\theta) = \int p_{\mathcal{N}}(y_k; h(X_k;\theta), \Gamma(\theta)) \rho^{*-}_k(X_k|\theta) dX_k$。

**3. 线性系统下的简化（公式12-14）**：
当模型为线性时：
$$X_{k+1} = A(\theta) X_k + W_k, \quad Y_k = H(\theta) X_k + V_k$$
预测分布和 $I(\theta)$ 都有闭式解：
$$I(\theta) = p_{\mathcal{N}}\left( y_k; H(\theta) m^{*-}_k(\theta), H(\theta) C^{*-}_k(\theta) H(\theta)^T + \Gamma(\theta) \right)$$

**4. 定理1（近似误差上界）**：
定理1给出了真实后验与近似后验之间总变差距离和Hellinger距离的上界：
$$d(p(X_k, \theta|\mathcal{Y}_k), q_k(X_k, \theta)) \leq \sum_{j=1}^{k-1} C(\mathcal{Y}_k, \tilde{C}, j, k) \sqrt{ \mathbb{E}_{\theta \sim \nu_j}[\text{KL}(\rho_j \| \rho^*_j)] + \text{KL}(\nu_j \| \nu^*_j) } + \frac{1}{\sqrt{2}} \sqrt{ \mathbb{E}_{\theta \sim \nu_k}[\text{KL}(\rho_k \| \rho^*_k)] + \text{KL}(\nu_k \| \nu^*_k) }$$
这个定理表明，要减小最终误差，关键在于在每个时间步 $j$ 都减小两个KL散度项，这正是两阶段策略的理论基础。

**5. 定理2（线性系统下的可计算误差上界）**：
对于线性系统，定理2给出了一个更具体的、可以通过FBOVI的输出（$m_i(\theta), C_i(\theta), \nu_i(\theta)$）来估计的误差上界。这为实际应用中的误差监控提供了可能。

**6. 训练策略（公式25-26）**：
阶段2中，训练神经网络来拟合 $m_k(\theta)$ 和 $C_k(\theta)$ 的目标函数是期望KL散度：
$$\mathbb{E}_{\theta \sim \nu_k}[\text{KL}(\rho_k \| \rho^*_k)] \approx \frac{1}{2N} \sum_{i=1}^N \left[ \log |C^*_k(\theta^{(i)})| - \log |C_k(\theta^{(i)})| - n + \text{Tr}\left( (C^*_k(\theta^{(i)}))^{-1} C_k(\theta^{(i)}) \right) + \delta(\theta^{(i)})^T (C^*_k(\theta^{(i)}))^{-1} \delta(\theta^{(i)}) \right]$$
为了计算效率，也可以使用更简单的均方误差（MSE）作为替代损失函数（公式26）。

### 3.3 模型、训练与数据集

- **模型**：动态系统模型由公式(1)定义，可以是线性的或非线性的。观测模型也可以是线性的或非线性的。
- **训练**：
    - **阶段1（VI）**：优化参数后验 $\nu_k(\theta)$ 的参数（例如，如果 $\nu_k$ 是高斯分布，则优化其均值和协方差）。优化目标是最大化ELBO $\mathcal{L}_k(\nu_k)$。使用Adam优化器和反向模式自动微分计算梯度。
    - **阶段2（监督学习）**：训练两个神经网络，分别用于表示均值函数 $m_k(\theta)$ 和协方差矩阵的Cholesky因子 $c_k(\theta)$。训练数据由 $\{ \theta^{(i)}, m^*_k(\theta^{(i)}), C^*_k(\theta^{(i)}) \}_{i=1}^M$ 组成，其中 $\theta^{(i)} \sim \nu_k(\theta)$，$m^*_k, C^*_k$ 由高斯滤波（KF, UKF, EnKF）计算得到。损失函数可以是公式(25)的KL散度或其MSE近似。
- **数据集**：论文没有使用固定的公开数据集。实验数据是通过对给定的动态系统模型（如Lorenz'96、对流-扩散方程）进行仿真生成的，并添加了人工噪声。观测数据是部分可观测的。

### 3.4 实验设置与结果

论文通过三个数值实验验证了FBOVI的性能：

1.  **低维线性系统**：
    - **设置**：一个简单的线性系统，状态和参数维度都很低。
    - **对比方法**：联合PF（作为基准），联合UKF。
    - **结果**：FBOVI（集成KF）的估计精度与联合PF几乎相同，远优于联合UKF。验证了其在低维问题上的高精度。

2.  **混沌非线性系统（Lorenz'96）**：
    - **设置**：40维的Lorenz'96系统，观测是部分且带噪声的，并且存在模型失配（观测模型参数未知）。
    - **对比方法**：联合EnKF。
    - **结果**：FBOVI（集成UKF）在参数和状态估计上均优于联合EnKF，尤其是在不确定性量化方面（例如，预测区间更可靠）。展示了其在非线性、模型失配和混沌动力学下的鲁棒性。

3.  **高维偏微分方程系统（对流-扩散方程）**：
    - **设置**：一个由对流-扩散方程空间离散化产生的高维（100维）状态空间系统。
    - **对比方法**：联合EnKF。
    - **结果**：FBOVI（集成EnKF）在状态估计精度上显著优于联合EnKF，并且能够提供更合理的不确定性估计。证明了其在高维问题上的可扩展性和有效性。

### 3.5 技术细节

- **参数后验 $\nu_k(\theta)$ 的表示**：论文主要使用高斯分布，但框架支持任意变分族。
- **状态条件分布 $\rho_k(X_k|\theta)$ 的表示**：使用条件高斯分布 $\mathcal{N}(X_k; m_k(\theta), C_k(\theta))$。协方差矩阵 $C_k(\theta)$ 通过其Cholesky因子 $c_k(\theta)$ 来表示，以确保正定性。
- **神经网络**：用于参数化 $m_k(\theta)$ 和 $c_k(\theta)$。网络结构是固定的，不随时间步增长，从而保证了在线计算的常数复杂度。
- **EnKF集成**：在阶段2使用EnKF时，采用了“每步重采样”策略（Algorithm 1），即每个 $\theta^{(i)}$ 都从当前的 $\rho_{k-1}$ 中重新采样一组粒子，然后进行EnKF更新。这避免了粒子退化问题，但增加了计算量。
- **计算复杂度分析**：论文在表1中详细分析了FBOVI（集成KF/UKF/EnKF）、联合PF、联合UKF和联合EnKF的每步计算复杂度。FBOVI的复杂度与神经网络大小、优化步数、样本数等因素相关，用户可以根据精度和效率需求进行权衡。
