# ECoSim: Data Efficient Fine-Tuning for Controllable Traffic Simulation

- arXiv: https://arxiv.org/abs/2607.00545v1
- arXiv ID: 2607.00545

---

🔍 **核心问题**：现有可控交通仿真方法需用大量标注数据重新训练生成模型，缺乏轻量级控制适配方案。  
🛠 **方法**：通过身份初始化的FiLM层调节预训练扩散/自回归模型的中间特征，仅用不到1%的配对数据即可新增草图、潜在行为编码和文本等多模态控制能力。  
✨ **主要贡献**：在Waymo仿真挑战赛中实现高效可控性，支持反事实场景生成和长尾数据合成，同时保持闭环驾驶的真实性与安全性。
