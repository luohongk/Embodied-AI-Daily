# TOLiD: Bridging the Architecture Gap in Vision Foundation Model to LiDAR Pretraining via Token Lifting for Distillation

- arXiv: https://arxiv.org/abs/2607.10762v1
- arXiv ID: 2607.10762

---

🔍 **核心问题**：现有视觉基础模型（VFM）到LiDAR骨干的跨模态蒸馏存在架构差异——密集ViT令牌表示与稀疏3D编码器不兼容，导致学生模型难以弥合模态与架构双重鸿沟。

🛠 **方法**：提出TOLiD方法，将LiDAR骨干与从冻结VFM教师初始化的学生ViT耦合，通过视锥池化与视锥注意力将图像块内的点特征转换为令牌，并采用可见性掩码进行令牌级蒸馏；部署时通过掩码双线性采样将令牌特征回传至逐点表示。

✨ **主要贡献**：在5个异构LiDAR数据集和4个跨传感器适应任务上验证了TOLiD的有效性，使用冻结骨干和轻量头即可实现更优的迁移性能，显著缩小了架构差异带来的性能损失。
