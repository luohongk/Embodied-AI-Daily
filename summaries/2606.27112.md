# Heavy-Ball Q-Learning with Residual Weighting Correction

- arXiv: https://arxiv.org/abs/2606.27112v1
- arXiv ID: 2606.27112

---

🔍 **核心问题**：标准Q-learning收敛速度较慢，尤其在复杂强化学习任务中，需要更高效的加速方法。

🛠 **方法**：提出一种带残差加权修正的重球动量Q-learning算法，并扩展至线性函数逼近场景；通过切换线性系统（SLS）表示和联合谱半径（JSR）分析收敛性与加速条件。

✨ **主要贡献**：首次从SLS视角理论证明重球动量可加速Q-learning，并给出收敛性保证及加速条件，为强化学习优化提供了新分析框架。
