# Toward Personalized Digital Twins for Cognitive Decline Assessment: A Multimodal, Uncertainty-Aware Framework

- arXiv: https://arxiv.org/abs/2604.27217v1
- arXiv ID: 2604.27217

---

🔍 **核心问题**：针对认知衰退的个体异质性，解决从稀疏、不规则纵向数据中建模个性化疾病轨迹的难题。  
🛠 **方法**：提出PCD-DT框架，融合潜在状态空间模型、多模态特征（临床、生物标志物、影像）及不确定性感知验证与自适应更新，并利用条件生成模型增强数据。  
✨ **主要贡献**：在TADPOLE数据上验证了认知正常与阿尔茨海默病组的显著分离，多模态LSTM模型（认知+MRI）在ADAS13和脑室体积预测中取得最低标准化RMSE（0.4419和0.5842），为临床可部署的不确定性感知数字孪生系统奠定基础。
