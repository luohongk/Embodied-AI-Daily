# Wavelet Policy: Imitation Learning in the Scale Domain with World Prior Memory

- arXiv: https://arxiv.org/abs/2504.04991v5
- arXiv ID: 2504.04991

---

🔍 **核心问题**：传统模仿学习在时间域直接预测动作，缺乏对物理场景的持久感知和有效记忆，导致在复杂机器人操作任务中表现受限。

🛠 **方法**：提出Wavelet Policy框架，通过世界先验记忆（WPM）从静态背景图像中编码场景结构为紧凑记忆令牌，并融合到编码器中；再对小波域中的潜在动作令牌进行多尺度分解，采用单编码器多解码器（SE2MD）架构建模不同时间尺度的潜在成分，最后通过逆小波变换重构为可执行动作。

✨ **主要贡献**：引入小波域多尺度动作建模与场景记忆结合，显著提升机器人操作的场景感知和记忆能力；在4个模拟和6个真实机器人操作任务中一致优于强基线方法，验证了其有效性和轻量性。
