# Hetu v2: A General and Scalable Deep Learning System with Hierarchical and Heterogeneous Single Program Multiple Data Annotations

- arXiv: https://arxiv.org/abs/2504.20490v2
- arXiv ID: 2504.20490

---

🔍 **核心问题**：现有SPMD范式假设分布式深度学习训练中工作负载对称划分，无法有效应对由设备异构（如混合硬件、故障）和数据变化（如序列长度不均）导致的时空异构性。

🛠 **方法**：提出HSPMD扩展SPMD声明式注解，支持非对称分片与分层通信原语，通过渐进式图特化处理空间异构，并利用动态图切换应对时间异构。

✨ **主要贡献**：在异构设备、不稳定设备和混合长度数据场景中，HSPMD性能匹配或超越专用系统，为现代分布式深度学习训练提供了灵活高效的通用解决方案。
