# Industrial3D: A Water-Treatment TLS Point Cloud Dataset and Cross-Paradigm Benchmark for MEP Scene Understanding

- arXiv: https://arxiv.org/abs/2603.28660v2
- arXiv ID: 2603.28660

---

🔍 **核心问题**：工业水处理设施的TLS点云数据存在几何模糊、严重遮挡和极端类别不平衡问题，现有建筑基准数据集无法充分代表此类场景，导致语义理解困难。

🛠 **方法**：构建了包含6.127亿个专家标注点的Industrial3D数据集（6mm分辨率，覆盖20个房间场景和7个运营水处理设施），并建立了跨范式基准测试，涵盖全监督、弱监督、无监督和基础模型四种方法。

✨ **主要贡献**：提供了最大规模的工业MEP场景理解测试平台（规模为同类数据集的6.6倍），量化了工业TLS数据的域迁移差距（最佳监督方法mIoU为55.74%，零样本Point-SAM仅15.79%），并揭示了尾部类别的统计稀有性（215:1）与圆柱形几何模糊性导致的“双重危机”。
