# SignSparK: Efficient Multilingual Sign Language Production via Sparse Keyframe Learning

- arXiv: https://arxiv.org/abs/2603.10446v4
- arXiv ID: 2603.10446

---

🔍 **核心问题**：手语生成面临直接文本到姿态模型导致的回归到均值效应，以及字典检索方法产生的动作过渡不连贯问题。

🛠 **方法**：提出SignSparK框架，利用稀疏关键帧捕捉手语运动分布，通过条件流匹配生成密集动作序列；并引入FAST高效分割模型自动提取精确时间边界。

✨ **主要贡献**：首次实现跨四种手语的多语言手语生成框架，支持关键帧到姿态的精确时空编辑，结合3D高斯泼溅实现逼真渲染，在多项基准测试中达到最优性能。
