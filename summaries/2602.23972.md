## 0. 概览（Concise Summary）

- **主题（Topic）**：微型飞艇机器人（MBR）的倒立姿态控制。
- **问题（Problem）**：MBR具有复杂、欠驱动且参数高度动态的非线性动力学特性，使得实现和维持不稳定的倒立姿态（gondola在上，气囊在下）极具挑战性。现有的基于模型的控制方法（如能量整形控制器）对模型参数变化和外部扰动非常敏感，鲁棒性不足。
- **方法（Method）**：提出一个基于深度强化学习（DRL）的三阶段框架：1）构建并校准一个高保真度的3D Unity仿真环境；2）采用改进的TD3算法（结合多缓冲区经验回放和梯度裁剪）与物理信息域随机化策略，在仿真中训练鲁棒的控制策略；3）设计一个映射层（Mapping Layer）来弥合仿真到现实（sim-to-real）的差距，实现策略在真实MBR上的部署。
- **创新（Innovation）**：1）首次为MBR倒立控制构建专用的Unity 3D仿真环境；2）提出结合物理信息域随机化和改进TD3算法的鲁棒控制策略学习框架；3）设计了一个有效的sim-to-real映射层，无需额外训练即可将仿真策略迁移到真实机器人上。
- **意义（Significance）**：这项工作首次成功实现了基于DRL的MBR倒立姿态鲁棒控制，显著提升了MBR的机动性和敏捷性，为其在娱乐、仓储、环境监测等领域的应用解锁了新的可能性。同时，提出的框架为其他欠驱动、动力学复杂的机器人系统的鲁棒控制提供了参考。
- **一句话总结**：本文通过构建高保真仿真环境、结合域随机化与改进TD3算法，并设计有效的sim-to-real映射层，成功学习并部署了能使微型飞艇机器人可靠地实现并维持倒立姿态的鲁棒控制策略。

## 1. 研究动机（Motivation）

- **解决的问题**：微型飞艇机器人（MBR）实现全向敏捷飞行的关键障碍——倒立姿态（Inverted Pose）的获取与维持。MBR的“gondola-气囊”结构使其具有稳定的“直立”和不稳定的“倒立”两种平衡点。倒立姿态是一个不稳定的平衡状态，需要持续、精确的控制才能维持。
- **为什么重要**：掌握倒立姿态控制是解锁MBR全部敏捷性的基石。它能使MBR实现快速姿态转换和大范围位置调整，从而在娱乐表演、仓库盘点、室内环境监测、基础设施检查等应用中发挥更大潜力。目前，MBR的控制研究多局限于小角度调整和低速导航，缺乏实现这种高难度机动动作的先进控制策略。
- **挑战在哪里**：
    1.  **独特的动力学特性**：与依赖高转速螺旋桨的无人机（UAV）不同，MBR依靠浮力抵消大部分重力，推力较弱。同时，其巨大的气囊导致空气阻力占主导地位。这种“弱推力、强阻力”的特性使得传统的UAV控制策略（如PID、MPC）难以直接应用。
    2.  **高度非线性和欠驱动**：MBR的动力学模型（公式1）高度非线性，且系统是欠驱动的（控制输入少于自由度）。
    3.  **参数高度动态**：MBR的模型参数（如重心位置、浮力、电机增益等）在实际运行中会因气体泄漏、负载变化、环境扰动等因素而动态变化，导致基于固定模型的控制方法（如能量整形控制器）性能下降甚至失效。
    4.  **缺乏DRL研究**：虽然DRL在UAV敏捷控制上取得了成功，但在MBR倒立控制这一特定问题上，相关研究几乎是空白。

## 2. 创新点（Innovation）

- **关键创新**：
    1.  **首个针对MBR倒立控制的Unity 3D仿真环境**：论文构建了一个高保真度的仿真环境（图4），该环境不仅精确实现了MBR的动力学模型（包括空气阻力、恢复力/力矩、附加质量效应等），还通过实验数据校准了电机模型（公式9），并修改了MBR结构（图5）以利于倒立控制训练。这为策略学习提供了安全、高效且可重复的试验场。
    2.  **物理信息域随机化（Physics-informed Domain Randomization）**：不同于盲目的参数随机化，该方法基于对MBR动力学起主导作用的物理量——三个关键中心点（浮心$c_b$、重心$c_g$、推力中心$c_t$）之间的距离——进行扰动。通过调整参数$m_w$（额外配重）和$\lambda$（配重分布比例），在保持物理一致性的前提下，系统地改变动力学特性，从而训练出对参数变化鲁棒的策略。
    3.  **改进的TD3算法（TD3 with Multi-buffer and Clipping）**：在标准TD3算法基础上，引入了两个关键改进：
        - **多缓冲区经验回放（Multi-buffer Experience Replay）**：使用$N$个独立的经验回放缓冲区，分别存储在不同动力学配置（不同$\lambda$值）下生成的轨迹。这迫使策略从更多样化的数据中学习，学习更通用的特征，从而提升鲁棒性。
        - **梯度裁剪（Gradient Clipping）**：借鉴PPO算法，对Actor和Critic网络的梯度进行裁剪，防止梯度爆炸，提高训练稳定性。
    4.  **有效的Sim-to-Real映射层**：设计了一个简单的线性映射层（公式14），用于补偿仿真与真实环境之间的动力学差异。该映射层仅在倒立转换阶段（$\Delta\phi < \varrho$）生效，通过调整一个对角矩阵$M_0$的参数，即可实现策略从仿真到真实机器人的零样本迁移（zero-shot transfer），无需在真实数据上重新训练。

- **与现有方案的不同**：与[17]中基于模型的能量整形控制器相比，本文的DRL方法不依赖于精确的、时不变的模型，而是通过域随机化学习到对参数变化和扰动具有内在鲁棒性的策略。与[18]和[19]中结合PID或$H_\infty$的DRL方法相比，本文专注于更具挑战性的倒立姿态控制，并提出了更完整的从仿真到部署的解决方案。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文提出的框架是一个三阶段流水线（图3）：

1.  **阶段1：3D仿真环境创建**：在Unity中构建一个高保真度的MBR动力学仿真环境（图4）。该环境精确实现了公式(1)描述的动力学，并包含一个经过实验校准的电机模型（公式9）。为了便于倒立控制训练，对MBR的虚拟结构进行了修改（图5），将额外质量分解为$m_{w1}$和$m_{w2}$。
2.  **阶段2：鲁棒控制策略训练**：在仿真环境中，使用改进的TD3算法（算法1和2）训练一个鲁棒的倒立控制策略。训练过程中，采用物理信息域随机化策略，通过改变参数$m_w$和$\lambda$来生成多样化的动力学环境。改进的TD3算法使用多缓冲区（$N=10$）来存储不同动力学配置下的经验，并应用梯度裁剪来稳定训练。
3.  **阶段3：Sim-to-Real策略部署**：将训练好的策略部署到真实MBR上。为了弥合仿真与现实的差距，设计了一个映射层（公式14），将策略输出的期望力矩$\mathbf{a}^p$映射为物理力矩命令$\mathbf{\tau}^p$。该映射层仅在MBR接近倒立姿态时激活，通过调整映射矩阵$M_0$的参数，可以补偿动力学差异，无需重新训练策略。

### 3.2 关键算法与数学推导

- **MBR动力学模型**：
  $$(M_{rb} + M_a)\dot{\nu}_{b/e}^b + (C_{rb} + C_a)\nu_{b/e}^b + \Gamma_d + \Gamma_{gb} + \Gamma_e = \Gamma_t$$
  其中，$\Gamma_d$是空气阻力，$\Gamma_t$是推力，$\Gamma_e$是环境力/力矩，$\Gamma_{gb}$是恢复力/力矩。

- **控制目标**：
  $$\lim_{t \to \infty} ||\Theta_d - \Theta_t||_{M_\Theta} = 0$$
  其中$\Theta = [\phi, \theta, \psi]^T$是姿态角（滚转、俯仰、偏航），$\Theta_d$是目标倒立姿态。

- **强化学习目标**：
  $$\max_\alpha G_t, \quad G_t = \sum_{k=0}^{\infty} \gamma^k r_{t+k+1}$$
  其中$\alpha$是策略参数，$\gamma$是折扣因子，$r_t$是奖励函数。

- **奖励函数**：
  $$r_{total} = r_{rot} + r_\omega + r_a$$
  - 姿态奖励：$r_{rot} = \exp(-\text{clip}(g_\phi|e_\phi| + g_\theta|e_\theta| + g_\psi|e_\psi|, 0, g_n)) + \mathbb{I}_{\{\phi < \zeta\}} \cdot (1 - \frac{\phi}{\zeta})$
    其中$e_\phi, e_\theta, e_\psi$是基于轴角表示的姿态误差。
  - 角速度惩罚：$r_\omega = -\sum_{i \in \{x,y,z\}} g_{\omega i} \frac{|\omega_i|}{\omega_{max}}$
  - 动作惩罚：$r_a = -\sum_{i \in \{x,y,z\}} g_{ai} |\tau_i|$

- **Sim-to-Real映射层**：
  $$\tau^p = M_0 \mathbf{a}^p, \quad \Delta\phi < \varrho$$
  其中$\tau^p$是物理力矩命令，$\mathbf{a}^p$是策略输出的期望力矩，$M_0 = \text{diag}(m_\phi, m_\theta, m_\psi)$是映射矩阵，$\varrho$是切换阈值。

- **改进的TD3算法（算法1和2）**：
  - **算法1（环境交互）**：使用$N$个不同的$\lambda$值，为每个$\lambda$值分配一个经验回放缓冲区$B_k$。在每个episode中，随机选择一个$\lambda_k$，更新MBR参数，然后使用带噪声的策略与环境交互，并将经验存储到对应的缓冲区$B_k$中。
  - **算法2（策略优化）**：从所有$N$个缓冲区中采样批次数据，计算目标Q值$Q_d$（对所有缓冲区的目标Q值取平均）。然后，使用梯度裁剪更新Critic网络。每隔$d_p$步，使用梯度裁剪更新Actor网络。最后，软更新目标网络。

### 3.3 模型、训练与数据集

- **模型**：策略网络$\pi_\alpha$和Q值网络$Q_{\beta}$都是全连接神经网络，包含2个隐藏层，每层256个神经元。隐藏层使用Leaky ReLU激活函数，策略网络输出层使用Tanh激活函数。
- **状态空间**：由旋转矩阵$R$和角速度向量$\omega$组成。
- **动作空间**：三个旋转轴上的期望控制力矩，通过功能模块`Fcdl`映射为电机命令。
- **训练**：使用改进的TD3算法，在Unity仿真环境中进行训练。训练超参数见表II。使用$N=10$个经验回放缓冲区，对应$\lambda \in [0.6, 1.0]$。训练约100个episode后收敛（图10）。
- **数据集**：训练数据完全由策略在仿真环境中与环境交互生成，存储在10个不同的经验回放缓冲区中。没有使用真实世界数据。

### 3.4 实验设置与结果

- **仿真实验**：
  - **基线**：能量整形控制器[17]。
  - **评估指标**：倒立姿态（$\phi$从0到$\pi$）的成功率。
  - **结果**：
    - **对$m_w$的鲁棒性（表III，图6）**：当$m_w$从5g变化到25g时，基线控制器仅在$m_w=25g$（其标称调谐条件）下成功，而学习策略在$m_w=10g$到25g范围内均成功。
    - **对$\lambda$的鲁棒性（表IV，图7）**：基线控制器仅在$\lambda=1.0$时成功，而学习策略在所有$\lambda \in [0.6, 1.0]$下均成功。
    - **对$g_m$的鲁棒性（表V，图8）**：两者在$g_m \ge 1.0$时均成功，在$g_m=0.5$时均失败。
    - **联合变化（表VI，图9）**：当$m_w, \lambda, g_m$同时变化时，基线控制器在所有5个测试案例中均失败，而学习策略全部成功，展现了强大的鲁棒性。
  - **消融实验（图10）**：证明了多缓冲区策略和梯度裁剪对提升训练稳定性和样本效率的贡献。完整方法（多缓冲区+梯度裁剪）收敛最快（约100 episodes），而去掉梯度裁剪或使用单缓冲区都会显著增加收敛时间。

- **真实世界实验**：
  - **设置**：将训练好的策略部署到真实MBR上，通过调整映射层参数$m_\phi$进行实验。
  - **结果**：
    - **映射层有效性（图11）**：当$m_\phi$在0.5到0.8之间变化时，除$m_\phi=0.8$外，策略均能成功驱动MBR达到并维持倒立姿态。图12展示了成功的动作序列。
    - **对额外配重的鲁棒性（表VII，图13）**：在5种不同的$m_{w1}$和$m_{w2}$配置下，策略均成功实现了倒立姿态，且过渡时间的变化趋势与仿真结果一致，验证了方法的有效性。

### 3.5 技术细节

- **域随机化**：核心是扰动$h_g$（重心$c_g$到推力中心$c_t$的距离），通过公式(10)和(11)可知，改变$\lambda$（配重分布）可以在保持总配重$m_w$不变的情况下改变$h_g$，从而改变动力学特性。
- **多缓冲区**：算法1中，每个episode开始时，会从$N$个$\lambda$值中循环选择一个，并重置MBR参数。产生的经验被存储到对应的缓冲区。算法2中，Critic和Actor的更新梯度是分别从所有$N$个缓冲区中采样的批次数据计算出的梯度的平均值。这相当于让策略在多个不同的“世界”中学习，从而学到更通用的控制能力。
- **映射层设计**：映射层$M_0$是一个简单的对角矩阵，其物理意义是对策略输出的期望力矩进行缩放。论文假设仿真和真实环境之间的主要差异在于力矩的“效率”不同，因此通过一个线性缩放来补偿。该映射只在MBR接近倒立姿态（$\Delta\phi < \varrho$）时生效，因为此时动力学最不稳定，对模型误差最敏感。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **首次实现**：首次成功证明了基于DRL的方法可以解决MBR的倒立姿态控制这一极具挑战性的问题，填补了该领域的研究空白。
    2.  **鲁棒性**：提出的方法对MBR的参数变化（如重心位置、电机增益）和环境扰动具有显著的鲁棒性，远超传统的基于模型的控制方法。
    3.  **实用性**：通过设计有效的sim-to-real映射层，实现了策略的零样本迁移，大大降低了将DRL策略部署到真实机器人上的成本和难度，提升了方法的实用价值。
    4.  **方法论贡献**：提出的“物理信息域随机化 + 多缓冲区TD3 + 映射层”框架，为其他具有类似复杂、欠驱动、参数不确定特性的机器人系统的鲁棒控制提供了可借鉴的范式。

- **对未来研究/应用的影响**：
    - **解锁MBR的敏捷性**：倒立姿态控制是MBR实现全向敏捷飞行的关键一步。未来可以在此基础上研究更复杂的机动动作，如翻滚、快速转向、穿越狭窄空间等。
    - **推动MBR的实际应用**：更强的鲁棒性和敏捷性将使MBR在动态、非结构化的真实环境中（如仓库、室内场馆）的应用更加可靠。
    - **启发其他机器人系统**：该框架可以推广到其他水下机器人、软体机器人等具有类似“弱驱动、强阻力”特性的系统。

- **局限与开放问题**：
    - **映射层的局限性**：作者明确指出，当前的线性映射层虽然有效，但限制了策略的性能，无法完全弥合sim-to-real差距。未来需要更精确地分析和量化这个差距，可能需要更复杂的非线性映射或在线自适应方法。
    - **状态估计**：论文假设在真实实验中可以准确获取姿态和角速度。在实际应用中，如何通过机载传感器（如IMU）进行鲁棒的状态估计是一个需要解决的问题。
    - **计算资源**：训练过程需要仿真环境，对计算资源有一定要求。未来可以探索更高效的训练方法或模型压缩技术。

## 5. 澄清与简化（Clarifications and Simplifications）

- **什么是“倒立姿态”？**：想象一个气球下面挂着一个篮子。正常状态是篮子在下，气球在上（直立）。倒立姿态就是让篮子跑到气球上面去。这就像让一个不倒翁头朝下立着，非常不稳定，需要持续施加控制力才能保持。
- **为什么MBR的动力学这么难控制？**：无人机（如四旋翼）靠强大的螺旋桨推力来克服重力，推力远大于空气阻力。而MBR靠浮力抵消了大部分重力，只需要很小的推力就能
