# Planning with Transformers: Chain of Computation and Structured Context Windows

- arXiv: https://arxiv.org/abs/2607.17710v1
- arXiv ID: 2607.17710

---

🔍 **核心问题**：大型语言模型（LLMs）在理论上具有图灵完备的计算能力，但在实际规划任务中表现不佳，存在理论与实证性能之间的差距。

🛠 **方法**：提出链式计算（COC）架构，将基于Transformer的语言模型置于迭代循环中，并采用结构化上下文窗口（SCW）来模拟图灵机磁带，使模型能学习规划策略、预测世界模型并执行算术运算。

✨ **主要贡献**：在BlocksWorld和Pancake puzzle任务中实现超过99.89%的成功率；通过符号算术支持或确定性下推自动机（PDA）形式的SCW，成功解决需百万步动作的20盘汉诺塔问题，且大幅减少训练数据需求。
