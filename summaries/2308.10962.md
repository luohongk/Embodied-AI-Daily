# Humanoid Robot Co-Design: Coupling Hardware Design with Gait Generation via Hybrid Zero Dynamics

- arXiv: https://arxiv.org/abs/2308.10962v1
- arXiv ID: 2308.10962

---

🔍 **核心问题**：双足机器人硬件设计与步态控制之间存在强耦合，传统迭代设计流程耗时且昂贵，难以评估非线性混合动力学下的系统性能。  
🛠 **方法**：基于混合零动力学（HZD）框架，将机器人设计参数编码为“设计虚拟约束”，与执行器虚拟约束共同纳入优化问题，同步求解最优设计参数和稳定行走步态。  
✨ **主要贡献**：提出系统化的协同设计框架，通过耦合硬件参数与步态生成，实现能量效率最大化；并成功应用于新型人形机器人ADAM的大腿和小腿设计，验证了方法的有效性。
