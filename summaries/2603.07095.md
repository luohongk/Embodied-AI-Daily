# ACLM: ADMM-Based Distributed Model Predictive Control for Collaborative Loco-Manipulation

- arXiv: https://arxiv.org/abs/2603.07095v1
- arXiv ID: 2603.07095

---

🔍 **核心问题**：解决多足机器人团队在复杂非结构化环境中协作搬运重物时，集中式规划方法扩展性差、实时性不足，而分散式方法忽略动态耦合导致行为保守的问题。

🛠 **方法**：提出基于交替方向乘子法（ADMM）的分布式模型预测控制框架，利用负载引起的耦合结构将全局优化问题分解为并行个体机器人子问题，并采用滚动时域规划与力感知全身控制器执行轨迹。

✨ **主要贡献**：实现仅需少量ADMM迭代即可快速收敛的分布式规划，支持多达四台机器人的实时协作，并通过仿真验证了可扩展性、实时性及对模型不确定性的鲁棒性。
