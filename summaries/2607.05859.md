# AVA-VLM: Adaptive Visual Attention-Vision Language Model for In-the-Wild Construction Site Monitoring

- arXiv: https://arxiv.org/abs/2607.05859v1
- arXiv ID: 2607.05859

---

🔍 **核心问题**：现有建筑工地视觉语言模型（VLM）在野外部署时，存在远距离监控可靠性差、低分辨率输入下性能下降以及推理效率低的问题。

🛠 **方法**：提出AVA-VLM模型，采用类人“由粗到细”的推理策略——先分析低分辨率全局图像，仅在需要细节检查时请求高分辨率局部裁剪，并引入区域感知的思维链数据集训练模型判断何时检查、何处裁剪及如何利用局部证据。

✨ **主要贡献**：在远距离和低分辨率条件下显著提升模型可靠性，同时大幅减少视觉令牌使用量，实现高效推理。
