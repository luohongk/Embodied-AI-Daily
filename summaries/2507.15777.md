# Label tree semantic losses for rich multi-class medical image segmentation

- arXiv: https://arxiv.org/abs/2507.15777v4
- arXiv ID: 2507.15777

---

🔍 **核心问题**：现有医学图像分割方法对所有分类错误一视同仁，未能利用标签间的语义层次关系，尤其在标签类别丰富且细微差异时效果不佳。

🛠 **方法**：提出两种基于标签层次树的语义损失函数，并融入稀疏、无背景标注的训练框架，以利用标签的层级结构提升分割性能。

✨ **主要贡献**：在头部MRI全脑分区（全监督）和神经外科高光谱图像场景理解（稀疏标注）两个任务上，所提方法持续优于基线，其中基于Wasserstein的复合损失在全脑分区中表现最佳，层次加权顶层监督在稀疏标注场景中效果最优。
