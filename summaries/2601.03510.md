# G2P: Gaussian-to-Point Attribute Alignment for Boundary-Aware 3D Segmentation

- arXiv: https://arxiv.org/abs/2601.03510v3
- arXiv ID: 2601.03510

---

🔍 **核心问题**：点云分割中，稀疏且不规则的点分布导致几何特征不足，难以区分形状相似但外观不同的物体（如颜色、纹理、材质）。

🛠 **方法**：提出Gaussian-to-Point（G2P）方法，通过将3D高斯泼溅中的高斯属性（如不透明度和尺度）对齐到点云，并利用不透明度导出的可见性线索和尺度属性增强边界感知。

✨ **主要贡献**：在不依赖预训练2D特征或语言监督的情况下，显著提升了几何挑战性类别的分割精度，在标准基准上取得竞争性表现，并实现了外观一致的边界定位。
