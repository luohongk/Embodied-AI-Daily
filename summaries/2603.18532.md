# Scaling Sim-to-Real Reinforcement Learning for Robot VLAs with Generative 3D Worlds

- arXiv: https://arxiv.org/abs/2603.18532v2
- arXiv ID: 2603.18532

---

🔍 **核心问题**：解决机器人视觉-语言-动作（VLA）模型在真实世界强化学习（RL）中因场景和物体多样性不足导致的过拟合和泛化能力下降问题。

🛠 **方法**：利用3D世界生成模型结合语言驱动的场景设计器，自动生成数百个包含独特物体和背景的多样化交互场景，在仿真环境中对预训练的VLA模型进行RL微调，并通过域随机化实现仿真到现实的迁移。

✨ **主要贡献**：将仿真成功率从9.7%提升至79.8%，任务完成速度提升1.25倍；真实世界成功率从21.7%提升至75%，速度提升1.13倍；通过消融实验证明增加场景多样性可直接改善零样本泛化能力。
