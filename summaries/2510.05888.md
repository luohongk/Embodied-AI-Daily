# BioAutoML-NAS: An End-to-End AutoML Framework for Multimodal Insect Classification via Neural Architecture Search on Large-Scale Biodiversity Data

- arXiv: https://arxiv.org/abs/2510.05888v2
- arXiv ID: 2510.05888

---

🔍 **核心问题**：解决昆虫分类中因昆虫特征复杂、类别不平衡及大规模数据集带来的挑战。

🛠 **方法**：提出BioAutoML-NAS框架，利用多模态数据（图像与元数据），通过神经架构搜索自动学习每个细胞内的最优操作，堆叠细胞形成完整网络，并采用多模态融合模块结合图像与元数据特征，使用交替双层优化策略联合更新网络权重与架构参数，通过零操作移除冗余连接。

✨ **主要贡献**：在BIOSCAN-5M数据集上达到96.81%准确率、97.05% F1分数，在Insects-1M数据集上获得93.25%准确率，性能显著优于现有迁移学习、Transformer、AutoML及NAS方法，为可持续农业提供高精度昆虫分类支持。
