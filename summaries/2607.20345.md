# Closing the Lab-to-Store Gap: A Data-Efficient Post-Training and Experience-Driven Learning VLA Framework for Retail Humanoids

- arXiv: https://arxiv.org/abs/2607.20345v1
- arXiv ID: 2607.20345

---

🔍 **核心问题**：解决视觉-语言-动作（VLA）人形机器人在真实零售场景中从实验室到实际部署的差距，包括执行错误、分布偏移和环境变化等挑战。

🛠 **方法**：提出DEED框架，包含三个关键组件：数据高效的后训练流程（控制频率对齐、数据筛选、视觉高亮和降低VLA依赖）、基于经验驱动的真实世界优化（通过文本优势前缀和视觉-语言价值函数改进RECAP方法）、以及用于分析分布内外行为的潜在空间分析工具。

✨ **主要贡献**：证明弥合实验室到商店的差距主要是系统集成挑战而非架构问题；通过精心数据设计和针对性后训练，仅用单GPU即可将原本失败的策略转化为可靠的现实系统，并在超市芯片补货任务中验证有效性。
