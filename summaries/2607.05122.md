# Green for Go, Red for No: Visual Grounding via Semantic Segmentation for VLA Navigation Policies

- arXiv: https://arxiv.org/abs/2607.05122v1
- arXiv ID: 2607.05122

---

🔍 **核心问题**：视觉-语言-动作（VLA）导航模型容易受到感知干扰和场景歧义的影响，导致导航精度下降。

🛠 **方法**：提出一种基于实时语义分割的视觉定位方法，利用SegFormer将可通行区域标记为绿色、不可通行区域标记为红色，并评估了仅观测分割与观测-目标联合增强两种变体。

✨ **主要贡献**：在Grand Tour数据集上使用OmniVLA模型，视觉定位使最远航点的平均航点误差降低27-44%，尤其对长指令效果显著；该方法无需重新训练模型，计算成本低，但主要作为轨迹长度正则化器，无法补偿分布外指令的缺失训练信号。
