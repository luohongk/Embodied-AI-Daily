# SurgAM: Surgical Affordance Map Prediction with Multimodal Feature Fusion for Robot Autonomy

- arXiv: https://arxiv.org/abs/2607.04378v1
- arXiv ID: 2607.04378

---

🔍 **核心问题**：解决手术自动化中视觉场景理解与自主动作规划之间的鸿沟，特别是从视觉数据预测可操作区域（即手术功能区域）的问题。  
🛠 **方法**：提出自适应特征融合框架，结合自监督视觉Transformer编码器（语义理解）和大规模生成模型编码器（空间感知），并引入层级提示学习机制适应不同手术场景，最后通过场景引导注意力解码器聚焦关键区域。  
✨ **主要贡献**：首次提出手术功能区域预测任务，构建包含三种基本手术动作（抽吸、夹闭、牵拉）的标注数据集，方法在公开数据集上达到最优性能，并在真实肺部和前列腺模型上验证了其支持自主手术动作的可行性。
