# Adaptive Time Series Reasoning via Segment Selection

- arXiv: https://arxiv.org/abs/2602.18645v3
- arXiv ID: 2602.18645

---

🔍 **核心问题**：现有时间序列推理方法通常将整个序列编码为固定表示，忽略了证据可能仅出现在少数关键区间，导致对无关信息的冗余处理。  
🛠 **方法**：提出ARTIST框架，将时间序列推理建模为序列决策问题，采用控制器-推理器架构，通过强化学习训练控制器选择信息片段，推理器生成基于片段的推理轨迹和答案。  
✨ **主要贡献**：在六个基准测试中平均准确率提升6.46个百分点，尤其在罕见事件定位和多片段推理任务上表现突出；验证了选择性数据使用和分层策略优化对推理有效性的提升。
