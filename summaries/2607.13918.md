# Partially Correlated Verifier Cascades in LLM Harnesses: Concave Log-Odds, Polynomial Reliability, and Blind-Spot Ceilings

- arXiv: https://arxiv.org/abs/2607.13918v1
- arXiv ID: 2607.13918

---

🔍 **核心问题**：针对大语言模型（LLM）验证级联中验证器部分相关时，后验对数几率增长规律与可靠性边界尚不明确的问题，建立严格理论。

🛠 **方法**：将每个实例的假接受率建模为潜在变量α（服从分布G），利用de Finetti定理推导级联后验对数几率ℓ_k = ℓ₀ - ln m_k（m_k为G的k阶矩），并引入Beta分布刻画相关性，同时考虑真接受率β的变异。

✨ **主要贡献**：证明ℓ_k对任意非退化G呈凹性（Odds Law为其上界）；发现Beta潜伏变量下失败率呈多项式衰减（1-r_k ∝ k^{-b}），相关性参数ρ_v可测；揭示盲点原子质量1-π导致证据上限为-ln(1-π) nats；提出验证器效果的三分法（始终有益/平台/有害），由G和H的上尾指数决定；通过R次重复判决可识别G的前R阶矩，相关性拟合在合成测试中比独立性外推误差小20倍（k=5）至3000倍（k=10），实际杠杆在于去相关而非增加门数。
