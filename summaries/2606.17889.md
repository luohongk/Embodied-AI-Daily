# Dimensionality Controls When Modularity Helps in Continual Learning

- arXiv: https://arxiv.org/abs/2606.17889v1
- arXiv ID: 2606.17889

---

🔍 **核心问题**：研究在持续学习中，模块化架构、任务相似性与表征维度如何共同影响系统的可塑性与稳定性平衡。

🛠 **方法**：采用序列A-B-A学习范式，对比任务分区循环网络与单基线网络，通过权重尺度操作诱导高维“惰性”与低维“丰富”两种表征机制。

✨ **主要贡献**：发现初始化尺度诱导的表征维度是决定模块化结构何时有益的关键因素：高维下模块化作用微弱，低维下模块化显著提升任务特异性子空间的组织性与可解释性。
