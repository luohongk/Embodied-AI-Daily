# Technical Report for ICRA 2026 GOOSE 2D Fine-Grained Semantic Segmentation Challenge: Leveraging DINOv3 for Robust Outdoor Scene Understanding in Field Robotics

- arXiv: https://arxiv.org/abs/2606.18582v1
- arXiv ID: 2606.18582

---

🔍 **核心问题**：解决野外机器人场景中细粒度语义分割的挑战，针对64类精细类别和11类粗粒度类别的越野图像进行密集语义分割。

🛠 **方法**：采用自监督DINOv3 ViT-L/16骨干网络、ViT-Adapter和Mask2Former掩码分类解码器，结合粗类别辅助损失；推理时使用多尺度与水平翻转测试增强，并集成Codabench评分最高的三个检查点。

✨ **主要贡献**：在ICRA 2026 GOOSE挑战赛中取得第一名，官方综合得分76.57%（精细类mIoU 69.32%，粗类mIoU 83.81%），验证了DINOv3在野外场景理解中的鲁棒性。
