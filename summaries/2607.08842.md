# L2-Bench: An Evaluation Benchmark for Measuring LLM Capabilities in Second Language Education

- arXiv: https://arxiv.org/abs/2607.08842v2
- arXiv ID: 2607.08842

---

🔍 **核心问题**：当前AI在教育领域的应用缺乏严格评估，尤其在第二语言（L2）教育中，亟需一个以教学法为导向的基准来评测大语言模型（LLM）的实际能力。

🛠 **方法**：构建了包含1000多个任务-响应对的开源基准L2-Bench，采用基于12项能力与31项子能力的专家验证分类法，并设计了一套基于评分标准的评估方法，以衡量模型在应用学习体验设计原则上的表现。

✨ **主要贡献**：提供了经200多名专家验证的高效分类体系（任务真实性4.42/5.00，标准充分性4.18/5.00）；提出了可推广至类似开放学科的可信评估方法；揭示了模型在L2教育场景中的优势、弱点及情境鲁棒性（如Claude Opus 4.7总体最佳达85.5%，但困难任务性能降至69.9%-73.4%），助力教育利益相关者做出更明智的AI采用决策。
