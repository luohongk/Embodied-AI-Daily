# Chat2Scenic: An Iterative RAG-Based Framework for Scenario Generation in Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.14387v1
- arXiv ID: 2607.14387

---

🔍 **核心问题**：从法规描述自动生成自动驾驶测试场景的可执行脚本，现有方法在编译成功率和可扩展性之间存在权衡。

🛠 **方法**：提出Chat2Scenic，首个基于迭代检索增强生成（RAG）的框架，通过聊天机器人界面支持交互式场景优化，并利用RAG将场景生成与法规知识和领域特定语言（DSL）语法对齐。

✨ **主要贡献**：在包含123个场景的基准测试中，Chat2Scenic实现76.42%的编译成功率和58.17%的框架准确率，显著优于现有方法（最高30.08%编译成功率），并开源代码。
