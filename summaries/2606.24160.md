# An Introduction to Causal Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.24160v1
- arXiv ID: 2606.24160

---

🔍 **核心问题**：因果推断与强化学习长期独立发展，缺乏统一框架来利用因果结构提升策略学习效率，尤其在反事实推理和机制不变性方面存在空白。

🛠 **方法**：将强化学习环境分解为具有因果不变性的自主机制集合，并用结构因果模型形式化建模，从而统一在线学习、离线策略学习与因果演算学习，并引入广义策略学习、模仿学习与反事实学习等新范式。

✨ **主要贡献**：提出因果强化学习（CRL）概念，揭示因果推断与强化学习的本质联系，拓展了反事实学习的维度，为结合因果推理与试错学习开辟了新研究方向。
