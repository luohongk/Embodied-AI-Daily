# Causally Debiased Latent Action Model for Embodied Action Conditioned World Models

- arXiv: https://arxiv.org/abs/2607.09185v1
- arXiv ID: 2607.09185

---

🔍 **核心问题**：现有潜在动作模型（LAM）在训练时仅依赖重建目标，导致动作相关动态与背景、未触碰物体等无关视觉因素纠缠，限制了具身动作条件世界模型（ACWM）的可控性。

🛠 **方法**：提出因果去偏框架CD-LAM，通过三种高效微调目标——具身中心重建、动作中心对比学习、潜在空间校准——来鼓励模型关注具身主体、感知动作并防止潜在空间坍塌。

✨ **主要贡献**：在2B和14B规模的ACWM骨干上，CD-LAM仅需6k步微调，且下游机器人动作适应更新次数比基线减少12倍以上，显著提升了潜在动作可控性、机器人动作跟随能力、视觉保真度和适应效率。
