# Geometry-Aware Reinforcement Learning for 2D Irregular Nesting

- arXiv: https://arxiv.org/abs/2606.10611v1
- arXiv ID: 2606.10611

---

## 0. 概览（Concise Summary）

- **主题（Topic）**：二维不规则嵌套问题（2D Irregular Nesting Problem），这是一个组合优化问题，旨在将一组不规则、非凸的多边形无重叠地放置在一个最小边界框内，以最大化面积利用率。
- **问题（Problem）**：传统启发式求解器（如Sparrow）在处理该问题时，对多边形几何形状“视而不见”，依赖引导式暴力搜索来探索连续放置空间，缺乏有效的几何先验知识，导致搜索效率低下。同时，现有机器学习方法多依赖光栅化，精度不足且难以复现。
- **方法（Method）**：提出一个基于强化学习（RL）的框架，核心是名为**多边形变换器（Polygons Transformer, PoT）**的新型神经网络架构。PoT能直接编码连续的二维矢量几何图形，并支持跨多边形注意力机制。该架构与一个“学习改进”（Learning to Improve, L2I）的强化学习训练框架相结合，用于优化多边形布局。
- **创新（Innovation）**：1）首次将组合优化强化学习（CORL）成功应用于无约束的连续二维不规则嵌套问题。2）提出了PoT架构，这是首个能直接处理多边形的矢量几何表示并学习其空间关系的Transformer。3）创建并开源了基于真实地理轮廓的训练数据集和评估基准。
- **意义（Significance）**：证明了数据驱动的神经策略可以自动从原始数据中发现并利用复杂的几何先验，在4个多边形的小规模实例上达到了与当前最先进的启发式求解器Sparrow相媲美的性能，为工业级高精度空间优化问题开辟了新的路径。
- **一句话总结**：本文通过提出一个几何感知的强化学习框架（含新型Polygons Transformer架构），成功解决了二维不规则嵌套问题，在小型实例上达到了与顶尖启发式算法同等的性能，验证了神经组合优化在连续空间任务中的潜力。

## 1. 研究动机（Motivation）

- **解决的问题**：传统的二维不规则嵌套问题求解器（如Sparrow）本质上是对几何形状“盲”的。它们依赖于预先计算的几何工具（如No-Fit-Polygon, NFP）来避免碰撞，但在搜索最优布局时，缺乏对形状几何特性的深层理解，只能进行引导式的暴力搜索。这导致在复杂的连续空间中，搜索效率低下，难以找到全局最优解。
- **为什么重要**：该问题直接源于工业制造，特别是印刷电路板（PCB）的拼板生产。即使将面积利用率提升零点几个百分点，在大规模生产中也能带来巨大的经济和生态效益。因此，为每个实例分配大量计算预算以追求最高精度是值得的。
- **挑战在哪里**：1）**几何复杂性**：处理非凸、不规则的多边形，并允许自由连续的旋转和平移，使得手工设计有效的几何先验极其困难。2）**状态表示**：环境状态是动态数量的多边形，每个多边形由可变长度的顶点序列定义，如何将这些矢量数据高效地编码为固定维度的潜在空间表示，是深度强化学习应用的关键。3）**连续动作空间**：放置动作（旋转角度和平移位置）是连续的，直接处理非常困难。

## 2. 创新点（Innovation）

- **新方法/技术**：
    1.  **Polygons Transformer (PoT) 架构**：这是核心创新。它专门设计用于编码多个多边形的矢量几何信息。通过轮廓重采样、局部几何块（Patch）标记化、循环位置嵌入（CPE）和多边形ID嵌入（PIE），以及跨多边形自注意力机制，PoT能够学习每个多边形的内部几何结构和多边形之间的空间关系。
    2.  **基于L2I的CORL训练框架**：将问题建模为一个“学习改进”的马尔可夫决策过程（MDP）。代理从一个初始布局开始，通过选择并移动一个多边形来逐步改进布局，从而获得密集的奖励信号，解决了“学习构建”（L2C）方法中奖励稀疏的问题。
    3.  **NFP引导的离散化动作空间**：利用No-Fit-Polygon（NFP）将二维的平移搜索问题简化为在一维边界上的搜索。通过均匀采样NFP边界，将连续动作空间离散化为有限个候选位置，保证了放置的合法性。
- **与现有方案的不同**：
    - 相比传统启发式（如Sparrow）：PoT是数据驱动的，能自动学习几何先验，而非依赖手工规则。
    - 相比现有ML方法（如UV Packing）：PoT直接处理矢量几何，避免了光栅化带来的精度损失；并且是开源的，保证了可复现性。
    - 相比其他CORL方法：首次将CORL成功应用于需要高精度几何感知的连续空间任务。
- **为什么能奏效**：PoT的跨多边形注意力机制允许模型“看到”不同形状之间的几何关系，例如一个形状的凹槽是否与另一个形状的凸起相匹配。L2I框架通过提供密集的、基于改进的奖励信号，使得代理能够学习到精细的调整策略，从而在有限的搜索预算内找到更优解。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

整体框架是一个“学习改进”（L2I）的强化学习代理。其工作流程如下：

1.  **状态观察**：代理观察当前面板上所有已放置多边形的布局。
2.  **两阶段动作**：
    - **阶段1（选择与旋转）**：代理选择一个多边形（`i`）并决定其新的旋转角度（`Δθ`）。这个决策基于PoT编码的当前布局状态。
    - **阶段2（放置）**：根据选定的多边形和旋转角度，环境动态计算该多边形相对于所有其他多边形的No-Fit-Polygon（NFP）边界。代理随后在这个一维的NFP边界上选择一个放置点（`j, s`）。
3.  **环境反馈**：执行动作后，布局更新，环境返回新的状态和一个奖励信号。奖励基于新布局的面积利用率是否超过了历史最佳记录（`max reward`）。
4.  **推理策略**：在推理阶段，维护一个高性能布局缓冲区，代理从缓冲区中采样初始状态进行改进，从而在有限预算内找到更优解。

### 3.2 关键算法与数学推导

- **问题形式化**：目标是最大化面积利用率 $O = \frac{\text{多边形总面积}}{\text{边界框面积}}$。
- **Max Reward 公式**：奖励函数定义为 $R(a_t, s_t) = \max(0, O_{\text{peak}<t} - O(s_{t+1}))$，其中 $O_{\text{peak}<t}$ 是当前回合中历史最佳面积利用率，$O(s_{t+1})$ 是执行动作后的新利用率。只有当新布局超越了历史最佳时，代理才获得正奖励。
- **动作空间**：动作 $a_t = (i, k, j, s)$，其中 $i$ 是多边形ID，$k$ 是旋转角度区间，$j$ 是NFP边界上的一个块（patch），$s$ 是该块内的一个子区域。
- **NFP引导的放置**：对于一个旋转后的多边形 $P_i^\theta$，其相对于其他多边形 $P_j$ 的可行放置区域是NFP的补集。最优放置点位于NFP的边界 $\partial F(i, \theta)$ 上，其中 $F(i, \theta) = \bigcup_{j \neq i} \text{NFP}(P_j, P_i^\theta)$。
- **联合动作概率**：两阶段动作的联合对数概率分解为：
  $$ \log \pi(a_t) = \log \pi_{\text{item}}(i) + \log \pi_{\text{rot}}(k|i) + \log \pi_{\text{place}}(j, s|i, k) $$
  这允许阶段1和阶段2使用不同的网络和输入，提高了计算效率。

### 3.3 模型、训练与数据集

- **模型**：核心是**Polygons Transformer (PoT)** 编码器。它包含：
    - **轮廓重采样与块标记化**：将多边形轮廓均匀重采样，然后分组为局部几何块（类似ViT的Patch）。
    - **结构嵌入**：为每个块添加**循环位置嵌入（CPE）** 以保持多边形闭环拓扑，以及**多边形ID嵌入（PIE）** 以区分不同形状。
    - **跨多边形注意力**：将所有多边形的块序列拼接，通过标准Transformer的自注意力层处理，使模型能学习多边形内部和之间的几何关系。
    - **灵活聚合**：为每个多边形添加一个`[CLS]`标记，通过注意力机制聚合该多边形的全局信息，输出固定维度的表示向量。
- **训练**：采用**PPO**算法，结合**优势过滤（Advantage Filtering）** 和**课程学习（Curriculum Learning）**。训练在8块RTX 6000 Blackwell GPU上进行，约360小时，总计200亿步。
- **数据集**：从**OpenStreetMap (OSM)** 的陆地轮廓数据中提取，经过简化、过滤和缩放，得到70万个独特的非凸、不规则多边形。这确保了训练数据的几何多样性。

### 3.4 实验设置与结果

- **基准**：将Sparrow（2DISPP的SOTA启发式）适配到无约束嵌套问题，通过网格搜索优化容器高度，作为基线。
- **评估**：在自建的基准上评估，包含4个多边形（P4）和8个多边形（P8）两个复杂度等级，各15个实例。每个实例运行3个随机种子。
- **主要结果**：
    - **P4（4个多边形）**：PoT与Sparrow性能非常接近。平均差距为0.01%，最佳差距为0.03%，胜/负比为9/6。这表明数据驱动策略在小型实例上达到了与顶尖启发式相当的水平。
    - **P8（8个多边形）**：Sparrow在所有实例上均优于PoT，平均差距为3.76%。这表明当前模型存在扩展性瓶颈，但为未来研究提供了基线。
- **训练动态分析**：图4显示，训练过程中的奖励与推理时的性能存在正相关，表明改进学习算法能直接提升最终性能。

### 3.5 技术细节

- **离散化**：旋转角度被离散化为720个区间（0.5°精度）。NFP边界被均匀采样为块（patch），每个块内再分为10个子区域。
- **两阶段网络**：阶段1网络处理PoT编码的布局状态，输出多边形选择和旋转角度的logits。阶段2网络处理选定的多边形表示、旋转角度嵌入和NFP块序列，输出放置位置的logits。
- **推理**：维护一个大小为N的顶级布局缓冲区。每次推理时，从缓冲区随机采样一个布局作为初始状态，然后让代理进行K步改进。最终，缓冲区中的最佳布局作为输出。

- **解决的问题与贡献**：

    2.  **提出了新的神经架构**：PoT是首个能有效编码多个矢量多边形并学习其空间关系的Transformer，为处理几何数据的深度学习提供了新工具。
    3.  **验证了CORL的潜力**：证明了强化学习可以成功应用于需要高精度几何感知的连续空间组合优化问题，并达到与顶尖启发式算法竞争的水平。
    4.  **提供了开源基准**：发布了基于真实数据的训练集和评估基准，为社区提供了可复现的研究平台，有望推动该领域的发展。
- **对未来研究/应用的影响**：
    - 为工业界提供了一种替代手工启发式算法的数据驱动方案，有望在PCB、服装裁剪、金属板材切割等领域实现自动化和更优的布局。
    - 启发了将Transformer等先进架构应用于其他几何相关的组合优化问题（如3D装箱、路径规划中的障碍物规避）。
- **局限与开放问题**：
    - **扩展性**：当前模型在处理8个多边形时性能下降，表明需要更大的模型容量、更先进的搜索策略或更好的课程学习机制来扩展到更大规模的实例。
    - **训练效率**：训练成本高昂（360小时/8 GPU），如何提高样本效率是一个开放问题。
    - **泛化能力**：虽然使用了多样化的地理数据，但模型在完全未见过的几何类型（如工业零件）上的泛化能力有待验证。

## 5. 澄清与简化（Clarifications and Simplifications）

- **什么是No-Fit-Polygon (NFP)？**
    想象你有一个形状A和一个形状B。你想把B放在A旁边，但不能重叠。NFP就是B的参考点（比如中心）所有可能位置的集合，当B的中心在这个集合内时，B就会与A重叠。NFP的**边界**就是B刚好与A接触但不重叠的位置。所以，寻找最优放置位置就变成了在这个一维边界上“散步”并找到最佳点，而不是在整个二维平面上乱找。这极大地简化了问题。

- **PoT是如何“看到”几何形状的？**
    可以把PoT想象成一个非常聪明的裁缝。它先把每个多边形（比如一块布）的轮廓用很多点均匀地描一遍（**轮廓重采样**）。然后，它把这些点分成一小段一小段的“布片”（**Patch**）。接着，它给每个“布片”贴上标签，告诉它自己属于哪块布（**多边形ID嵌入**），以及在这块布上的顺序（**循环位置嵌入**，比如“我是这块布的第3个布片，而且第1个和第100个布片是连在一起的”）。最后，它把所有“布片”的标签信息都放在一起，让它们互相“交流”（**跨多边形注意力**），这样，一个布片就能知道“我旁边是另一块布的布片，它们之间可能能拼合得很好”。最终，每块布会有一个总结性的标签（`[CLS]`），代表了这块布的整体形状和它与其他布的关系。

- **为什么“学习改进”（L2I）比“学习构建”（L2C）好？**
    - **L2C（学习构建）**：就像让一个新手厨师从零开始做一道复杂的菜。他每放一种材料，都不知道最终味道如何，只有做完整个菜才能尝到味道（**稀疏奖励**）。他很难知道是哪个步骤导致了失败。
    - **L2I（学习改进）**：就像让一个厨师去改进一道已经做好的菜。他每次只调整一种调料（比如加一点盐），然后立刻尝一口（**密集奖励**）。他能立刻知道这次调整是好是坏，从而快速学会如何微调。在嵌套问题中，L2I代理每次只移动一个多边形，并立即看到面积利用率的变化，学习效率更高。
