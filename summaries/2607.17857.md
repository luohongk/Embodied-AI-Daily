# Leveraging Dissimilarity Invariance as a Robust Anchor for Learning with Noisy Labels

- arXiv: https://arxiv.org/abs/2607.17857v1
- arXiv ID: 2607.17857

---

🔍 **核心问题**：解决深度学习模型在训练标签被噪声污染时性能严重下降的问题，现有方法难以学习准确相似性导致学习过程被误导。  
🛠 **方法**：提出NegScale框架，包含结构化负正交惩罚（SNOP）强制不相关样本子空间正交，以及基于不相似性校准的相似性调整（DCSA）利用不相似锚点抑制虚假相似性。  
✨ **主要贡献**：发现并理论证明了“不相似性不变性”现象，即噪声下不相关样本的语义不相似性保持稳定；NegScale在合成噪声CIFAR和真实数据集上均超越现有最优方法。
