# RELISH: LLM REgression with a Latent Iterative State Head

- arXiv: https://arxiv.org/abs/2604.01206v2
- arXiv ID: 2604.01206

---

🔍 **核心问题**：如何高效地利用大型语言模型（LLM）进行文本回归任务，避免传统方法（如将数值解码为文本或聚合多个生成结果）的低效和不稳定问题。

🛠 **方法**：提出RELISH架构，通过一个可学习的潜在迭代状态头，在冻结的LLM表示上执行交叉注意力，逐步优化潜在状态，最终用线性回归器映射为标量预测值。

✨ **主要贡献**：在6个数据集、4种LLM骨干和2种训练模式下，RELISH显著优于所有现有LLM回归方法（包括自回归解码、回归感知推理和预测头方法），且仅需约3.4-3.7M可训练参数（仅占模型参数的0.01-0.04%），远低于LoRA变体（0.26-0.42%）。
