# Fog Computing and Large Language Models: A vision for mutual beneficiaries

- arXiv: https://arxiv.org/abs/2606.29483v2
- arXiv ID: 2606.29483

---

🔍 **核心问题**：雾计算与大型语言模型（LLMs）各自面临挑战——雾计算需解决云中心物联网的延迟、网络负载和隐私问题，而LLMs依赖云端资源，同样受限于延迟和计算瓶颈，因此需要探索两者如何相互赋能。

🛠 **方法**：通过LLM优化技术（如参数量化、剪枝、低秩适配）使其适配雾基础设施，同时利用LLM的代码生成能力支持雾应用的动态部署。

✨ **主要贡献**：提出雾计算与LLMs的互利共生愿景，分析当前技术现状并指出未来研究方向，为边缘智能与云-雾协同提供新思路。
