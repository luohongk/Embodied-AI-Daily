# TranSplat: Instant Object Relighting in Gaussian Splatting via Spherical Harmonic Radiance Transfer

- arXiv: https://arxiv.org/abs/2503.22676v6
- arXiv ID: 2503.22676

---

🔍 **核心问题**：在3D高斯泼溅（Gaussian Splatting）框架中，如何实现快速、准确且无需昂贵逆渲染或材质估计的物体重光照。

🛠 **方法**：提出一种无BRDF的辐射传输策略，通过源和目标环境贴图计算每法线辐照度比率，解析地调制2D高斯面元的球谐外观系数；引入镜面感知双路径球谐传输，在反射域自适应调整高阶球谐波段；并设计轻量级球谐域自阴影模块，无需显式网格光线投射即可模拟物理真实遮挡。

✨ **主要贡献**：作为后处理步骤，TranSplat无需重新训练GS模型，即可在1秒内完成重光照，在合成与真实物体上达到最先进精度，超越现有逆渲染和扩散方法，尤其对高光复杂材质也能生成感知真实的渲染结果。
