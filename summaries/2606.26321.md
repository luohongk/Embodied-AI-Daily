# KRVF: A Source-Aware Semantic Voxel World Representation for Edge Mobile Manipulation

- arXiv: https://arxiv.org/abs/2606.26321v1
- arXiv ID: 2606.26321

---

🔍 **核心问题**：边缘计算条件下的移动操作机器人需要一种实时、可查询、语义丰富且轻量级的世界表示方法，但现有重建型地图主要优化全局几何精度，缺乏任务导向的语义与时效性支持。

🛠 **方法**：提出KRVF（源感知语义体素世界表示），将局部世界状态编码为任务导向的体素，包含占用、颜色、语义证据、时间新鲜度和来源信息，并分离测量占用与语义先验假设，同时通过渲染地图先验深度实现建图与感知的反馈闭环。

✨ **主要贡献**：实现了深度失效感知的物体推理而不破坏持久几何结构，提供了面向语义对象和抓取候选的任务级查询算子，并在ROS 2中验证了将在线RGB-D观测转化为任务导向机器人记忆的可行性。
