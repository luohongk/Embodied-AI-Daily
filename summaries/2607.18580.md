# STeP: Signal Temporal Logic for Precise Specifications for Action Generation with Vision Language Models

- arXiv: https://arxiv.org/abs/2607.18580v1
- arXiv ID: 2607.18580

---

🔍 **核心问题**：视觉-语言-动作（VLA）模型在遵循包含空间、时间和逻辑约束的自然语言指令时，缺乏可解释性和精确性。

🛠 **方法**：提出一个分层框架，利用信号时序逻辑（STL）作为共享表示，连接高层语言理解与低层机器人执行。高层策略通过VLM将语言指令分解为子任务并生成STL规范，低层策略选择STL引导的模型预测控制或学习策略来执行。

✨ **主要贡献**：将STL集成到计划验证、子任务监控和重规划中，实现了运行时基于形式化结构的计划检查、优化和修订，在真实桌面任务中提升了语言条件机器人规划的精度、可靠性和可解释性。
