# Compositional Semantic Communication for Physical AI: Category Theory Meets Game Theory

- arXiv: https://arxiv.org/abs/2607.18115v1
- arXiv ID: 2607.18115

---

🔍 **核心问题**：物理AI系统中分布式传感代理传输原始数据导致高通信开销、延迟和冗余，现有语义通信方法缺乏适应性、泛化性和可扩展性。

🛠 **方法**：提出组合语义通信（CSC）框架，利用范畴论度量组合语义、格罗滕迪克拓扑形式化语义组合，并将多设备协调建模为Stackelberg博弈，采用ADMM算法求解均衡策略。

✨ **主要贡献**：相比基线方法，实现高达17%带宽缩减和53%端到端延迟降低，同时在多种自动驾驶场景中保持85%推理准确率，并证明了均衡策略的存在性与帕累托最优性。
