# Enhancing Blood Cells Classification using Hybrid Quantum Neural Networks

- arXiv: https://arxiv.org/abs/2605.23324v2
- arXiv ID: 2605.23324

---

🔍 **核心问题**：解决显微镜血细胞分类中，传统深度学习模型因数据有限和细微差异而性能受限的问题。  
🛠 **方法**：提出混合量子-经典神经网络（HQNN），结合预训练ResNet-50主干网络、低维潜在瓶颈层和变分量子电路，并与纯经典变换模型及无中间变换的基线模型进行对比。  
✨ **主要贡献**：在血细胞数据集上，HQNN相比经典基线将宏F1分数提升最高3.7%，在8类高难度场景中F1分数从98.54%提升至98.69%，且在IBM量子硬件上仍保持鲁棒性，验证了量子特征变换在医学图像分类中的实用潜力。
