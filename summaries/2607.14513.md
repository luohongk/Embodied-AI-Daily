# Compression of 3D Gaussian Splatting Data Using GPU-friendly Graphics Texture Coding

- arXiv: https://arxiv.org/abs/2607.14513v1
- arXiv ID: 2607.14513

---

🔍 **核心问题**：3D高斯泼溅（3DGS）技术中，球谐系数（SH）数据量庞大，导致内存需求高，且现有压缩方法未充分利用现代GPU的并行解码能力。

🛠 **方法**：提出利用专为高效并行GPU解码设计的纹理压缩方案（如BC1和BC7格式），通过局部分组和按颜色重排序基元，更有效地压缩SH颜色系数，并引入支持随机访问的码率控制策略。

✨ **主要贡献**：实现了GPU端快速并行解压缩，在渲染3DGS场景时视觉质量几乎无退化或可忽略，同时保持了大规模并行化与渲染性能。
