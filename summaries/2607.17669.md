# Attention from Above: A Multimodal Model for Drone-Based Object Localization

- arXiv: https://arxiv.org/abs/2607.17669v1
- arXiv ID: 2607.17669

---

🔍 **核心问题**：提升无人机视角下小目标检测的准确性，解决基于文本提示的指定目标定位难题。  
🛠 **方法**：基于YOLO-World框架，将YOLOv8骨干网络中的C2f层替换为注意力机制的A2C2f层，结合并行处理结构增强局部特征表示。  
✨ **主要贡献**：在VisDrone数据集上，模型精度从43.0%提升至45.1%，召回率从32.8%升至35.0%，mAP@0.5提高2.7个百分点，验证了注意力机制对无人机目标检测的有效性。
