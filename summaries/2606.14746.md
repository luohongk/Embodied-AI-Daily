# Style-CCL: Content-Preserving Style Transfer via Curriculum Continual Learning

- arXiv: https://arxiv.org/abs/2606.14746v2
- arXiv ID: 2606.14746

---

🔍 **核心问题**：扩散变换器（DiTs）在风格迁移中因内容与风格特征纠缠，导致纹理风格学习受阻，内容保真度下降。  
🛠 **方法**：提出Style-CCL多阶段课程持续学习框架，按语义风格（简单）到纹理风格（困难）、从干净数据到合成数据逐步训练SC-DiT，并通过随机记忆回放防止灾难性遗忘。  
✨ **主要贡献**：在风格相似性、内容一致性和美学质量三项核心指标上达到最优性能，有效解耦风格与内容特征。
