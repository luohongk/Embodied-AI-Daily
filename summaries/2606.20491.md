# Fast Human Attention Prediction for Fixation-guided Active Perception in Autonomous Navigation

- arXiv: https://arxiv.org/abs/2606.20491v1
- arXiv ID: 2606.20491

---

🔍 **核心问题**：现有的人类注意力预测模型计算成本高，难以应用于机器人自主导航中的实时感知任务。

🛠 **方法**：提出GazeLNN模型，采用液态神经网络作为循环引擎，结合MobileNetV3进行特征提取，以自回归方式预测注视扫描路径热图。

✨ **主要贡献**：GazeLNN仅需0.61 GFLOPs，计算成本降低99.40%，推理速度提升6倍，在MIT低分辨率数据集上达到0.47 ScanMatch分数，超越现有基线；成功集成到强化学习训练的主动相机-机器人控制策略中，实现真实环境下的无人机自主导航。
