# Water Reflection Detection Using Symmetric Attention

- arXiv: https://arxiv.org/abs/2607.10749v1
- arXiv ID: 2607.10749

---

🔍 **核心问题**：自然水域中的倒影会导致目标检测和语义分割等视觉任务出现大量误检和漏检，现有深度学习模型难以区分物体与其镜像。

🛠 **方法**：提出SAWRD-Net，利用水面的非完美反射对称性，结合二面体群等变卷积与矩阵分解解码器，通过多尺度反射等变模块和对称注意力机制提取反射相关特征，并直接回归反射轴关键点。

✨ **主要贡献**：在最大公开水反射场景数据集上，SAWRD-Net的真阳性率达到0.890，超越所有现有水反射检测方法，显著提升了反射轴检测的可靠性。
