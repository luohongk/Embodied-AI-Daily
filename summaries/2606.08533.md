# Autonomous Aerial Manipulation via Contextual Contrastive Meta Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.08533v1
- arXiv ID: 2606.08533

---

🔍 **核心问题**：无人机在自主抓取和运输不同载荷时，因载荷导致飞行动力学变化，现有方法依赖预装载荷或专用夹具，无法实现无需人工校准的端到端自主递送。

🛠 **方法**：提出**Aco2**框架，结合上下文对比元强化学习，通过上下文观测编码器从近期交互历史推断潜在上下文，并引入对比学习目标优化上下文嵌入，使策略在线适应载荷变化，全程在仿真中训练并直接部署到真实无人机。

✨ **主要贡献**：首次实现四旋翼无人机在无人工干预下自主完成不同载荷的钩取、运输和递送任务，无需真实世界微调，显著提升对多样载荷的泛化能力。
