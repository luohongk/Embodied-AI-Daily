# AI-Driven Synthesis for High-Tech System Design: Automating Innovation

- arXiv: https://arxiv.org/abs/2606.28126v1
- arXiv ID: 2606.28126

---

🔍 **核心问题**：解决现代高科技系统设计中因组合复杂性导致的创新瓶颈，减少对人工设计的依赖。  
🛠 **方法**：提出自动化设计（AiD）范式，采用计算设计合成（CDS）框架，结合深度学习与生成式AI自动生成新型系统。  
✨ **主要贡献**：通过电驱动系统设计与空间尺寸问题两个案例验证了该方法，标志着工程从仿真优化向自主设计的根本性转变。
