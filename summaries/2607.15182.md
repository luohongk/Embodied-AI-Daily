# Stigmergic Graph Memory: An Environment-Aware Approach for Many-to-Many Multi-Agent Pickup and Delivery

- arXiv: https://arxiv.org/abs/2607.15182v1
- arXiv ID: 2607.15182

---

🔍 **核心问题**：在多对多多智能体拣选与配送（MAPD）任务中，如何通过感知环境动态信息来优化任务分配和路径规划，避免仓库拥堵并提升吞吐量。

🛠 **方法**：提出了一种名为“Stigmergic Graph Memory (SGM)”的有限、衰减记忆层，该层记录仓库节点和有向边上的近期执行信号，用于对可行端点进行排序并影响路径偏好，而不改变碰撞约束或规划器有效性。

✨ **主要贡献**：在五种布局、三种负载水平和25个随机种子条件下，SGM在所有15种地图-负载组合中均优于两种重建的多对多分配基线，吞吐量提升20.5%-36.7%，证明近期执行记忆可通过影响可行目标的选择而非仅路径规划来提升仓库效率。
