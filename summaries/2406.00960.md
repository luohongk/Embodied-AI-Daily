# PDP: Physics-Based Character Animation via Diffusion Policy

- arXiv: https://arxiv.org/abs/2406.00960v2
- arXiv ID: 2406.00960

---

🔍 **核心问题**：解决基于物理的角色动画中，扩散策略因累积误差导致高频欠驱动控制（如双足行走）不稳定的问题。  
🛠 **方法**：提出PDP方法，结合强化学习（RL）和行为克隆（BC），利用RL策略在次优状态下提供修正动作，增强扩散策略的鲁棒性。  
✨ **主要贡献**：实现了扰动恢复、通用运动跟踪和基于文本的物理运动合成，展示了扩散策略在物理角色动画中的有效性与多样性。
