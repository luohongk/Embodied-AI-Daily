# Learning User Simulators with Turing Rewards

- arXiv: https://arxiv.org/abs/2606.19336v1
- arXiv ID: 2606.19336

---

🔍 **核心问题**：现有用户模拟器训练方法通常要求模型匹配单一真实回复，难以生成与真实用户难以区分的多样化响应。

🛠 **方法**：提出Turing-RL方法，基于图灵测试的强化学习框架，使用LLM判别器作为奖励函数，评估生成回复与真实用户历史行为的一致性，并优化模型以生成不可区分的回复。

✨ **主要贡献**：在对话聊天和Reddit论坛两个领域，Turing-RL在LLM和人工评估指标上均优于基线方法，证明优化“不可区分性”而非直接匹配回复是训练用户模拟器的有效策略。
