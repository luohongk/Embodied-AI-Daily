# Efficient reduction of stellar contamination and noise in planetary transmission spectra using neural networks

- arXiv: https://arxiv.org/abs/2602.10330v3
- arXiv ID: 2602.10330

---

🔍 **核心问题**：JWST观测系外行星透射光谱时，恒星表面不均匀性（如黑子、耀斑）和仪器噪声会严重干扰大气参数反演的准确性，现有校正方法计算成本高或精度不足。

🛠 **方法**：利用去噪自编码器（DAE）神经网络，基于大量合成数据集（类地行星TRAPPIST-1e和亚海王星K2-18b的模拟光谱）训练模型，自动从污染光谱中重建无污染光谱，并与标准校正方法进行对比。

✨ **主要贡献**：DAE在保持与同步拟合恒星污染方法同等精度的前提下，将计算成本降低一个数量级；即使在低信噪比条件下也能保留关键分子特征，显著减少反演参数偏差，为未来系外行星大气表征流程提供了高效、准确的预处理方案。
