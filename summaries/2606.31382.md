# Revisiting Parameter Redundancy in Vision-Language-Action Models: Insights from VLM-to-VLA Adaptation

- arXiv: https://arxiv.org/abs/2606.31382v1
- arXiv ID: 2606.31382

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在参数剪枝后性能严重下降，通常需要微调恢复，但本文质疑被剪枝参数是否真正冗余，并探究剪枝导致性能损失的根源。

🛠 **方法**：通过分析VLM到VLA适配过程中参数差异的空间分布，引入无微调的控制剪枝作为诊断工具，建立参数差异与功能贡献的因果联系，并设计多模块联合剪枝方案。

✨ **主要贡献**：在LIBERO基准上，该方法在不进行任何后剪枝恢复的情况下，将OpenVLA和π₀.₅模型参数减少12%-30%，同时保持约90%原始性能，而现有剪枝标准在相同约束下会导致性能完全崩溃。
