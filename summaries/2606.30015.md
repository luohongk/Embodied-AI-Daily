# Parametric Skills

- arXiv: https://arxiv.org/abs/2606.30015v1
- arXiv ID: 2606.30015

---

🔍 **核心问题**：大语言模型在复杂长上下文场景中难以有效定位和遵循技能指令，限制了其利用文本技能的能力。

🛠 **方法**：提出ParametricSkills框架，通过构建大规模高质量技能库并合成轨迹数据，训练超网络将文本技能实时转换为LoRA适配器参数，实现无上下文依赖的技能调用。

✨ **主要贡献**：在六个软件工程子任务上平均超越上下文学习方法6.44个点，同时获得更高BERT Score和F1分数，并展示了参数化技能在测试时持续学习方面的潜力。
