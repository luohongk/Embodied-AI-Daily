# Globalized Constrained Stein Variational Inference for Diverse Feasible Robot Motion Planning

- arXiv: https://arxiv.org/abs/2607.12732v1
- arXiv ID: 2607.12732

---

🔍 **核心问题**：机器人运动规划需要生成多样且可行的运动方案，但传统方法通常只返回单一解，且难以在有限预算内确保所有样本满足严格约束（如避障、关节限制等）。

🛠 **方法**：提出SteinSQP方法，结合约束Stein变分推断与序列二次规划，通过核空间SQP子问题嵌入约束，并采用GPU友好的矩阵自由原-对偶算法进行批量粒子更新，同时引入全局化评价函数平衡目标、约束与多样性。

✨ **主要贡献**：在五个约束运动规划任务中，SteinSQP能生成完全可行的多样化运动集合，相比一阶约束Stein基线和串行多起点非线性规划，迭代收敛更快、粒子可行性更高，且在复杂机器人任务上实现了更快的批量求解时间。
