# MemoryWAM: Efficient World Action Modeling with Persistent Memory

- arXiv: https://arxiv.org/abs/2606.20562v1
- arXiv ID: 2606.20562

---

🔍 **核心问题**：现有世界动作模型（WAM）在非马尔可夫环境中难以兼顾长期记忆与高效推理，长序列处理导致计算成本随长度显著增长。

🛠 **方法**：提出MemoryWAM，采用混合记忆设计，结合近期帧、事件边界锚点帧和压缩长程历史的紧凑摘要令牌，并通过定制注意力机制同时检索短期细节与长期压缩上下文。

✨ **主要贡献**：在长时域、依赖记忆的模拟和真实操作任务中，MemoryWAM优于强基线模型（如VLA和WAM），同时保持较低的推理延迟和GPU内存占用。
