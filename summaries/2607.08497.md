# Cognitive-structured Multimodal Agent for Multimodal Understanding, Generation, and Editing

- arXiv: https://arxiv.org/abs/2607.08497v1
- arXiv ID: 2607.08497

---

🔍 **核心问题**：现有统一多模态模型在处理长时多模态对话时，因重复输入所有历史视觉和文本信息导致视觉标记爆炸和跨轮次引用不可靠，限制了长期交互能力。

🛠 **方法**：提出认知结构化多模态智能体，通过情景视觉记忆外部化视觉信息，并选择性激活相关记忆；包含感知抽象引擎、认知检索引擎和多模态执行控制器，并利用统一场景引擎生成带细粒度检索标注的多轮对话数据，通过强化学习优化策略。

✨ **主要贡献**：8B参数智能体在20轮对话中达到91.4%的检索准确率，超越32B基线模型8.2%，同时将每轮推理时间从23.1秒降至12.7秒；还提供了工具增强部署框架CMA-Harness，支持持久多模态记忆、网络访问和图像编辑等工具，展示了结构化记忆和模块化决策相比单纯扩大参数规模的更高效范式。
