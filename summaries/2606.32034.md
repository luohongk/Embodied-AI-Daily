# QVal: Cheaply Evaluating Dense Supervision Signals for Long-Horizon LLM Agents

- arXiv: https://arxiv.org/abs/2606.32034v1
- arXiv ID: 2606.32034

---

🔍 **核心问题**：现有密集监督信号评估方法依赖下游训练性能，成本高且难以跨方法比较，无法直接衡量信号质量。  
🛠 **方法**：提出QVal测试平台，通过计算信号评分与参考策略Q值的一致性（Q对齐性），无需训练即可直接评估密集监督信号。  
✨ **主要贡献**：在4个环境、21种方法、6种模型上验证，发现简单提示基线优于最新密集监督方法，且方法族内性能聚类显著，为无训练基准评估提供可扩展框架。
