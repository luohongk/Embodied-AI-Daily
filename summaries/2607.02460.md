# Neuron-Aware Data Selection for Annotation-Free LLM Self-Distillation

- arXiv: https://arxiv.org/abs/2607.02460v1
- arXiv ID: 2607.02460

---

🔍 **核心问题**：在缺乏人工标注或真实交互反馈的情况下，如何有效对大型语言模型（LLM）进行后训练，避免跨领域性能下降和校准误差膨胀。  
🛠 **方法**：提出Neuron-OPSD框架，利用模型内部神经元激活值指导训练数据选择和教师上下文构建，再通过在线策略蒸馏从教师分布中训练模型，全程无需真实标签。  
✨ **主要贡献**：在专业领域基准上提升域内任务性能，同时保持跨域泛化能力，并缓解校准崩溃问题，适用于无法获取在线交互或外部监督的场景。
