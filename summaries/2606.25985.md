# Action ControlNet: A Lightweight Delay-Aware Adapter for Smooth Asynchronous Control in Vision-Language-Action Models

- arXiv: https://arxiv.org/abs/2606.25985v1
- arXiv ID: 2606.25985

---

🔍 **核心问题**：视觉-语言-动作（VLA）模型在机器人操控中因推理延迟导致异步控制时动作不连贯、抖动及接触任务失败。

🛠 **方法**：提出轻量级延迟感知适配器Action ControlNet（ACNet），利用已执行动作片段作为残差条件，冻结预训练主干并仅微调少量参数，兼容扩散或流匹配等生成式动作头。

✨ **主要贡献**：在Kinetix、Meta-World MT50及真实SO-ARM101平台上，ACNet在推理延迟下提升鲁棒性，生成更平滑的异步轨迹，且比全延迟条件重训练更轻量。
