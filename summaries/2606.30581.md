# Realtime Wind Estimation using Low Cost Quadrotor Uncrewed Aerial Vehicles

- arXiv: https://arxiv.org/abs/2606.30581v1
- arXiv ID: 2606.30581

---

🔍 **核心问题**：现有基于扩展卡尔曼滤波（EKF）的四旋翼无人机风速估计方法，在强非线性场景（如大风条件）下精度受限，且多局限于悬停或近线性轨迹。

🛠 **方法**：采用无迹卡尔曼滤波（UKF）作为非线性估计器，结合SE(3)群上的四旋翼动力学模型与几何控制器，在维持飞行轨迹的同时实时估计风速。

✨ **主要贡献**：通过数值仿真证明，随着非线性程度增加，UKF始终优于EKF，能在保持轨迹微小偏差的同时提供更准确的风速估计，适用于高非线性环境。
