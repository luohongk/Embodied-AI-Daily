# TRACE: Turn-level Reward Assignment via Credit Estimation for Long-Horizon Agents

- arXiv: https://arxiv.org/abs/2607.13988v1
- arXiv ID: 2607.13988

---

🔍 **核心问题**：多轮智能体在长序列工具交互后生成最终答案，导致信用分配困难——传统结果奖励稀疏、方差大，且无法区分失败轨迹中的有效动作。

🛠 **方法**：提出TRACE方法，通过冻结参考模型获取黄金答案的对数概率，转化为对数比率状态值，再利用时序差分变化推导每步动作的密集奖励，无需额外评论器或过程标签训练。

✨ **主要贡献**：在长时域复杂搜索任务中，TRACE仅用纯强化学习显著提升基础模型的工具使用能力（如Qwen3-4B在BrowseComp-Plus上从7.2提升至35.6），无需冷启动监督微调或网络数据训练，且学习曲线显示更早改进和更快收敛。
