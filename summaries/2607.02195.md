# Bridge-WA: Predicting Where and How the World Changes for Robotic Action

- arXiv: https://arxiv.org/abs/2607.02195v1
- arXiv ID: 2607.02195

---

🔍 **核心问题**：现有世界-动作模型依赖大规模生成模型或密集未来帧预测，计算成本高且易受与控制弱相关的视觉细节干扰，导致泛化能力不足。

🛠 **方法**：提出Bridge-WA轻量框架，通过冻结的未来变化教师模型蒸馏出三种紧凑先验（未来令牌、变化图、运动流图），并利用WorldBridge模块以多源注意力记忆和时空偏置条件化动作Transformer，推理时移除教师模型。

✨ **主要贡献**：在VLABench、LIBERO-Plus等多项基准及真实机器人评估中，显著提升任务成功率与鲁棒性，尤其在外观分布偏移下表现突出，通过聚焦场景变化位置与方式抑制背景、光照等干扰，无需部署时生成密集未来图像。
