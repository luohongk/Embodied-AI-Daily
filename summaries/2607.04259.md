# Integrated Graph Search and Model Predictive Control for Smooth and Efficient Path Planning in Autonomous Vehicles

- arXiv: https://arxiv.org/abs/2607.04259v1
- arXiv ID: 2607.04259

---

🔍 **核心问题**：在自动驾驶路径规划中，如何同时保证路径的安全性、舒适性、动态可行性以及计算效率。  
🛠 **方法**：提出一种顺序路径规划框架，先通过图搜索（Dijkstra算法）生成粗略路径，再基于该路径构建空间变化的凸侧向安全走廊，将避障约束转化为连续可行性约束，最后通过模型预测控制（MPC）优化路径，并惩罚横向偏移的三阶空间导数以提升平滑性。  
✨ **主要贡献**：在直道和弯道多场景超车测试中，该方法相比多项式拟合与二次规划方法，显著降低了横向加速度、曲率和冲击度，同时计算成本在直道和弯道上分别降低28.08%和29.52%，验证了图搜索与MPC结合在结构化环境中平衡平滑性与效率的有效性。
