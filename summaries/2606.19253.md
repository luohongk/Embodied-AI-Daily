# OneCanvas: 3D Scene Understanding via Panoramic Reprojection

- arXiv: https://arxiv.org/abs/2606.19253v1
- arXiv ID: 2606.19253

---

🔍 **核心问题**：现有视觉语言模型在3D场景理解中依赖复杂的几何编码器或高昂训练成本，难以高效实现空间推理。

🛠 **方法**：提出OneCanvas，将多视角图像块特征通过深度和相机位姿投影到单一全景画布上，并添加3D位置嵌入保留深度信息，使预训练VLM能像处理普通图像一样直接使用该表示。

✨ **主要贡献**：无需融合或修改骨干网络即可统一空间坐标，支持任意视角的具身推理；通过程序化生成空间预训练任务减少推理捷径，在SQA3D、VSI-Bench和SPBench上达到最先进精度，且训练计算量比最强方法低一个数量级。
