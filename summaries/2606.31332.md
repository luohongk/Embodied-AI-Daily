# CryoACE: An Atom-centric Framework for Accurate and Automated Model Building in Cryo-EM

- arXiv: https://arxiv.org/abs/2606.31332v1
- arXiv ID: 2606.31332

---

🔍 **核心问题**：解决冷冻电镜密度图中蛋白质自动建模面临的物理化学有效性验证和构象异质性管理难题，现有方法局限于静态预测或依赖高计算成本的启发式搜索。

🛠 **方法**：提出端到端框架CryoACE，采用原子中心重建范式（直接在原子坐标采样密度特征并迭代循环优化，替代昂贵体素卷积）和无需训练的引导机制（利用预测的局部分辨率先验解决动态模糊性）。

✨ **主要贡献**：在新构建的高质量数据集上显著超越现有基线，首次在复杂真实数据集（如EMPIAR-10345）中揭示原子级动态构象，且无需依赖预构建的静态结构。
