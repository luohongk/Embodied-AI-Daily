# Lightweight Multi-Scale Anomaly Detection for Resource-Constrained Edge Devices

- arXiv: https://arxiv.org/abs/2607.12599v1
- arXiv ID: 2607.12599

---

🔍 **核心问题**：在资源受限的边缘设备（如IoT系统、传感器网络）上，现有时间序列异常检测模型因计算开销大、多尺度敏感性不足，难以兼顾高精度与低延迟、低功耗需求。

🛠 **方法**：提出轻量级多尺度自编码器（LMSAE），利用离散小波变换（DWT）提取多尺度特征，并设计多尺度损失函数增强对细微或隐藏异常的检测灵敏度。

✨ **主要贡献**：模型大小不足500 KB，参数极少，在基准数据集上达到竞争性或更优的检测性能；在NVIDIA Jetson Nano上推理延迟降低9倍、功耗降低2倍，适合边缘部署。
