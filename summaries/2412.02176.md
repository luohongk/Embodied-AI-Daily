# Self-Supervised Learning-Based Path Planning and Obstacle Avoidance Using PPO and B-Splines in Unknown Environments

- arXiv: https://arxiv.org/abs/2412.02176v2
- arXiv ID: 2412.02176

---

🔍 **核心问题**：解决未知环境中自主机器人的实时路径规划与避障问题，提升在复杂场景下的适应性和鲁棒性。

🛠 **方法**：提出SmartBSP框架，结合近端策略优化（PPO）、卷积神经网络（CNN）和Actor-Critic架构，处理有限激光雷达（LIDAR）输入，将感知场离散为网格并生成空间概率分布，同时通过最小化包含路径曲率、终点接近度和避障的代价函数进行自监督训练。

✨ **主要贡献**：通过仿真和基于机器人操作系统（ROS）的实时实验验证了算法在不同场景下的鲁棒性和适应性，实现了高效的自监督学习路径规划与避障。
