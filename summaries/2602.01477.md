# Density-Informed Pseudo-Counts for Calibrated Evidential Deep Learning

- arXiv: https://arxiv.org/abs/2602.01477v3
- arXiv ID: 2602.01477

---

🔍 **核心问题**：证据深度学习（EDL）在分布偏移下会混淆认知不确定性与偶然不确定性，导致对分布外（OOD）输入过度自信。  
🛠 **方法**：提出密度信息伪计数EDL（DIP-EDL），通过分别估计条件标签分布和边际协变量密度，解耦类别预测与不确定性幅度，在高密度区域保留证据，对OOD数据则收缩预测至均匀先验。  
✨ **主要贡献**：从理论上证明了DIP-EDL的渐近集中性，实验表明该方法能提升模型可解释性、鲁棒性及分布偏移下的不确定性校准效果。
