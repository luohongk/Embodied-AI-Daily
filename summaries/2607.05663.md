# Physics-Regularized Machine Learning for Proprioceptive Vehicle Localization Using Onboard Sensors

- arXiv: https://arxiv.org/abs/2607.05663v1
- arXiv ID: 2607.05663

---

🔍 **核心问题**：解决自动驾驶车辆在卫星信号中断时，基于IMU的自主定位精度下降的问题。

🛠 **方法**：提出PRML2混合框架，通过可微分卡尔曼滤波器实现机器学习模型的端到端训练，将物理规律（车辆运动模型）作为正则化约束，融合IMU等车载传感器数据进行定位。

✨ **主要贡献**：首次将物理正则化学习与可微分卡尔曼滤波结合，在公开数据集上实现优于传统方法的定位精度和实时性能，并发布了低摩擦条件下的新型定位数据集。
