# PhysisForcing: Physics Reinforced World Simulator for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.28128v1
- arXiv ID: 2606.28128

---

🔍 **核心问题**：现有视频生成模型在机器人操作仿真中常产生物理不合理的现象，如运动轨迹不连续、机器人与物体交互不一致，限制了其作为世界模拟器的可靠性。

🛠 **方法**：提出PhysisForcing训练框架，通过联合优化像素级轨迹对齐损失和语义级关系对齐损失，重点监督物理信息关键区域，增强视频生成的物理一致性。

✨ **主要贡献**：在多个基准上显著提升具身视频生成质量，R-Bench上提升22.3%和9.2%；作为世界模型，将闭环成功率从16.0%提升至24.0%，并改善下游策略性能。
