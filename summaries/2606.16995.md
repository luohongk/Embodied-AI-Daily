# When in Doubt, Plan It Out: Committed Small Language Model Deliberation for Reactive Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.16995v1
- arXiv ID: 2606.16995

---

🔍 **核心问题**：强化学习策略在陌生环境中因缺乏显式推理能力而性能下降。  
🛠 **方法**：提出PACT混合架构，将快速反应式RL策略与慢速推理式小语言模型（SLM）规划器结合，SLM异步生成并验证候选动作计划，通过仿真验证后直接执行，无需重训或修改RL策略。  
✨ **主要贡献**：在三个难度递增的FrozenLake任务中，PACT基于2B参数的SLM超越所有基线，证明推理规划与反应执行协同工作比单独使用任一方法更有效。
