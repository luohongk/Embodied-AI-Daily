# An approach with Visual and Tabular Mamba to multimodal medical data using Mixed Fusion

- arXiv: https://arxiv.org/abs/2606.20738v1
- arXiv ID: 2606.20738

---

🔍 **核心问题**：解决多模态医疗数据（图像与表格数据）在癌症分类中的融合与可解释性问题。  
🛠 **方法**：提出基于Mamba架构的混合融合方法（Mixed Fusion），分别用视觉Mamba处理图像、表格Mamba结合临床数据，并通过SHAP增强可解释性。  
✨ **主要贡献**：在PAD-UFES-20和NDB-UFES数据集上验证了Mamba模型在召回率上的显著提升，且混合融合架构支持SHAP解释，为多模态医疗分类提供了高敏感性的替代方案。
