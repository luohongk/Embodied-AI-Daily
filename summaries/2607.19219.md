# Beyond Score Prediction: LLM-Based Essay Scoring and Feedback Generation via Reinforcement Learning with Rubric Rewards

- arXiv: https://arxiv.org/abs/2607.19219v1
- arXiv ID: 2607.19219

---

🔍 **核心问题**：现有基于大语言模型的自动评分与反馈生成方法主要依赖提示工程或监督微调，缺乏对强化学习后训练及反馈质量自动化评估的系统研究。

🛠 **方法**：提出RLAES统一框架，通过强化学习联合优化评分与反馈生成；引入基于166个细粒度二元评分项的论文锚定反馈评估框架（RFE），并设计自适应门控反馈优化（AGFO）与相邻对比推理（ACR）分别提升反馈质量和评分校准。

✨ **主要贡献**：RFE框架能有效捕捉论文-反馈一致性，与专家偏好高度对齐；在ASAP基准上，RLAES-AGFO在基于LLM的方法中取得最佳评分性能（QWK=0.803），且反馈质量媲美GPT-5.5，避免了纯评分强化学习导致的反馈退化。
