# Soft-TransFormers for Continual Learning

- arXiv: https://arxiv.org/abs/2411.16073v4
- arXiv ID: 2411.16073

---

🔍 **核心问题**：解决持续学习中灾难性遗忘与任务推断不可靠的问题，同时保持预训练模型的高效适应。

🛠 **方法**：提出Soft-TransFormers框架，在冻结的预训练Transformer上为每个任务学习实值软掩码（仅作用于选定自注意力层的Q/K/V/O投影），通过掩码空间梯度下降实现参数漂移有界优化。

✨ **主要贡献**：理论证明掩码微调的收敛速率与参数漂移受预训练权重控制，从而结构消除遗忘并解耦任务推断误差；在多个类增量基准上超越提示、适配器及LoRA方法，且推理成本与原始模型相同。
