# Event-VLA: Action-Conditioned Event Fusion for Robust Vision-Language-Action Model

- arXiv: https://arxiv.org/abs/2606.29384v1
- arXiv ID: 2606.29384

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在光照变化（如低光、近暗环境）下，因RGB观测退化而导致机器人操作鲁棒性不足。

🛠 **方法**：提出Event-VLA框架，引入事件流作为光照鲁棒、运动敏感的补充观测，并通过动作查询路由路径和门控交叉注意力机制，选择性融合事件特征，保留预训练的RGB-语言语义先验。

✨ **主要贡献**：在正常光照下保持强操作性能，在低光及近暗真实场景中显著提升成功率，验证了事件增强对VLA模型鲁棒性的有效性。
