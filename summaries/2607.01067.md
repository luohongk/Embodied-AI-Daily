# Human-Centric Transferable Tactile Pre-Training for Dexterous Robotic Manipulation

- arXiv: https://arxiv.org/abs/2607.01067v1
- arXiv ID: 2607.01067

---

🔍 **核心问题**：现有触觉数据集规模小、接触覆盖窄，且视觉-语言-动作模型在触觉模态上缺乏动力学感知的后训练，限制了灵巧操作任务的性能上限。

🛠 **方法**：提出H-Tac大规模触觉-动作数据集（160小时第一人称视频，含300+任务和13.5万片段），并构建可迁移触觉预训练系统（TTP），通过统一触觉与动作空间实现人-机器人知识迁移，结合触觉专家模型预测未来触觉信号以显式建模接触动力学。

✨ **主要贡献**：首次实现基于人类触觉数据的大规模预训练向机器人迁移，在仿真和真实实验中验证了模型在细粒度操作和鲁棒泛化上的优越性能，为可扩展触觉预训练开辟新路径。
