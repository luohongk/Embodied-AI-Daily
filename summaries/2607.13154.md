# Worlds in One Demo: A Synthetic Data Engine for Learning Open-World Mobile Manipulation

- arXiv: https://arxiv.org/abs/2607.13154v1
- arXiv ID: 2607.13154

---

🔍 **核心问题**：如何从单次人工演示中高效生成大量合成数据，以训练开放世界移动操作策略，实现空间泛化、长程鲁棒性和跨场景泛化。

🛠 **方法**：提出WANDA合成数据引擎，先通过RGBD观测重建背景高斯泼溅和机器人-物体交互轨迹，再通过全身运动规划重组接触丰富的交互片段，并利用校正状态扩展增加状态多样性，最后在生成的多样化3D世界中合成新轨迹和逼真观测。

✨ **主要贡献**：仅需一次真实演示即可训练出具备长程鲁棒性、广泛空间泛化和跨场景泛化能力的移动操作策略，并支持跨形态机器人零样本部署。
