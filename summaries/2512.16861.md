# ReinforceGen: Hybrid Skill Policies with Automated Data Generation and Reinforcement Learning

- arXiv: https://arxiv.org/abs/2512.16861v2
- arXiv ID: 2512.16861

---

🔍 **核心问题**：解决机器人领域长期存在的长时域操作任务，即如何让机器人高效完成需要多步骤协调的复杂操作。

🛠 **方法**：提出ReinforceGen系统，先通过任务分解将长任务拆解为多个局部技能，并用运动规划连接；基于10次人类演示生成数据集进行模仿学习训练，再通过在线自适应和强化学习微调各组件。

✨ **主要贡献**：在Robosuite基准测试中，所有视觉运动控制任务达到80%成功率；消融实验表明微调方法使平均性能提升89%；真实环境评估也验证了显著改进效果。
