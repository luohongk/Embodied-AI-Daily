# Deep Reinforcement Learning for Dynamic Origin-Destination Matrix Estimation in Microscopic Traffic Simulations Considering Credit Assignment

- arXiv: https://arxiv.org/abs/2511.06229v3
- arXiv ID: 2511.06229

---

🔍 **核心问题**：解决微观交通仿真中动态OD矩阵估计（DODE）的信用分配难题，即难以精确确定每辆车对路段流量的贡献，导致OD矩阵与观测数据关系模糊。

🛠 **方法**：将DODE问题建模为马尔可夫决策过程（MDP），提出基于无模型深度强化学习（DRL）的框架，智能体通过与仿真环境直接交互学习最优策略，逐步生成OD矩阵。

✨ **主要贡献**：首次将DODE重构为序列决策问题，通过DRL有效应对信用分配挑战；在Nguyen-Dupuis网络和真实高速公路子网实验中，路段流量均方误差相比最强传统基线降低23.7%和59.2-88.3%，显著提升校准性能。
