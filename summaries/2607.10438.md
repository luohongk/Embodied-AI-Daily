# Large Language Model Enhanced Differentiable Trajectory Planning for IoT-Enabled Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.10438v1
- arXiv ID: 2607.10438

---

🔍 **核心问题**：解决基于模仿学习的自动驾驶规划器在复杂城市环境中存在的长尾交互覆盖不足、与下游约束优化一致性弱、以及实时条件下高层语义信息利用不充分的问题。

🛠 **方法**：提出一种大语言模型增强的可微分轨迹规划框架，包括：基于周围智能体中心的数据增强策略、复杂度感知的异步LLM语义增强模块，以及可微分优化模块。

✨ **主要贡献**：在nuPlan闭环基准测试中分别取得83.63和78.29的最佳总分，并通过CARLA-ROS测试验证了在线部署与实时闭环执行能力。
