# Self-Evolving World Models for LLM Agent Planning

- arXiv: https://arxiv.org/abs/2606.30639v1
- arXiv ID: 2606.30639

---

🔍 **核心问题**：解决长时域LLM智能体在规划时，世界模型预测不可靠导致下游决策被忽略或误用的问题。

🛠 **方法**：提出WorldEvolver框架，包含三个模块：基于检索的**情景记忆**利用真实动作转换；**语义记忆**从预测-观察不匹配中提取持久启发式规则；**选择性预见**过滤低置信度预测，所有模块在部署时动态调整上下文，不修改模型参数。

✨ **主要贡献**：在ALFWorld和ScienceWorld上，WorldEvolver在三个骨干模型上均取得最高预测准确率，并显著提升下游智能体任务成功率，证明测试时记忆修订能同时增强预测保真度和规划性能。
