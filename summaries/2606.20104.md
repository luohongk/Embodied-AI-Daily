# Sensorimotor World Models: Perception for Action via Inverse Dynamics

- arXiv: https://arxiv.org/abs/2606.20104v1
- arXiv ID: 2606.20104

---

🔍 **核心问题**：如何让视觉世界模型学习与动作相关的表征，避免表征坍塌，同时丢弃无关的干扰信息。  
🛠 **方法**：提出传感器运动世界模型（SMWM），通过逆动力学正则化（预测动作）来端到端训练潜在世界模型，强制潜在状态保留动作信息。  
✨ **主要贡献**：单一逆动力学正则化同时解决了表征坍塌和动作对齐问题，无需冻结编码器或复杂正则化，在2D/3D控制任务中实现紧凑、可解释的潜在空间和竞争性规划性能。
