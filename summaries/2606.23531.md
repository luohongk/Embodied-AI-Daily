# BiliVLA: Scene-Aware Vision-Language-Action Model with Reinforcement Learning for Autonomous Biliary Endoscopic Navigation

- arXiv: https://arxiv.org/abs/2606.23531v1
- arXiv ID: 2606.23531

---

🔍 **核心问题**：解决内镜逆行胰胆管造影（ERCP）中，因解剖变异、镜面反射、组织遮挡等视觉干扰导致的自主导航可靠性不足问题。

🛠 **方法**：提出BiliVLA框架，将胆道内镜导航建模为指令条件化的视觉运动学习问题，结合场景感知监督（增强语义目标一致性）和安全感知恢复监督（诱导退缩行为），并采用两阶段训练范式（基础监督微调+组相对策略优化）。

✨ **主要贡献**：在真实体模实验中，BiliVLA在三个ERCP子任务上达到91.96%的平均动作精度和84.85%的整体成功率，验证了语义定位、场景感知学习与奖励引导优化对提升感知-动作对齐及鲁棒自主导航的有效性。
