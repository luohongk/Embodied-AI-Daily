# PrismAD: Decoupled Planning via Semantic Mixture-of-Planners for End-to-End Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.10336v1
- arXiv ID: 2607.10336

---

🔍 **核心问题**：现有端到端自动驾驶规划器将异构场景特征耦合到统一表示空间，导致单一规划分支难以同时建模交互、几何和意图，削弱了特定因素的推理能力。  
🛠 **方法**：提出PrismAD框架，将场景令牌划分为交互、几何和意图三组，分别分配给独立参数的专家网络，并通过语义感知路由器自适应聚合专家预测，同时引入稀疏Top-K激活与噪声门控提升鲁棒性。  
✨ **主要贡献**：在nuScenes开放循环数据集和NeuroNCAP闭环基准上验证了竞争性能，通过解耦规划专家和语义路由机制，显著增强了不同规划线索的贡献可解释性。
