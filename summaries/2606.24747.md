# Scaling Laws for Task-Specific LLM Distillation

- arXiv: https://arxiv.org/abs/2606.24747v1
- arXiv ID: 2606.24747

---

🔍 **核心问题**：研究如何在大语言模型（LLM）的领域特定压缩中，量化模型性能与数据集大小、压缩比、监督格式等关键因素之间的缩放规律，以解决部署时延迟和成本受限的挑战。

🛠 **方法**：以量化金融为应用领域，采用迭代结构剪枝下的logit蒸馏和LoRA蒸馏，并引入混合思维链监督损失来稳定推理轨迹上的KL散度蒸馏过程。

✨ **主要贡献**：发现领域内任务质量在压缩下可预测地下降，而通用知识基准会提前崩溃；监督格式是权衡的关键，思维链监督能有效恢复剪枝丢失的通用知识；同时发布了FinHeadlineMix数据集和实用的缩放规律建议。
