# Fine-Tuning Regimes Define Distinct Continual Learning Problems

- arXiv: https://arxiv.org/abs/2604.21927v3
- arXiv ID: 2604.21927

---

🔍 **核心问题**：本文探讨在持续学习中，微调机制（即可训练参数子空间）是否会影响不同方法的性能比较结果，以及这种影响是否被现有评估体系忽视。

🛠 **方法**：通过将微调机制形式化为固定可训练子空间上的投影优化，在任务增量学习场景下，对五种不同训练深度、四种标准方法（在线EWC、LwF、SI、GEM）及五个基准数据集进行系统性比较实验。

✨ **主要贡献**：发现方法间的相对排名在不同微调机制下并不一致，且更深层的微调机制会导致更大的更新幅度、更高的遗忘率以及两者间更强的关联性，从而提出应将训练深度作为显式实验因素纳入评估协议。
