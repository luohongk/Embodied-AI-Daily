# HDDLGym: A Tool for Studying Multi-Agent Hierarchical Problems Defined in HDDL with OpenAI Gym

- arXiv: https://arxiv.org/abs/2505.22597v1
- arXiv ID: 2505.22597

---

🔍 **核心问题**：现有工具缺乏将层次化规划与强化学习无缝集成的能力，尤其是在多智能体场景中。  
🛠 **方法**：开发了基于Python的HDDLGym工具，自动将层次化领域定义语言（HDDL）描述的领域和问题转换为OpenAI Gym环境，支持强化学习与层次化规划的协同。  
✨ **主要贡献**：首次实现了HDDL与Gym接口的自动集成，支持多智能体协作规划，并通过Transport和Overcooked等经典领域验证了其实用性，为层次化规划中的强化学习研究提供了便捷工具。
