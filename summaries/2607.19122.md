# Code Division Modulation Layers Against Forgetting and Inference in Continual Gait Identification

- arXiv: https://arxiv.org/abs/2607.19122v1
- arXiv ID: 2607.19122

---

🔍 **核心问题**：在持续学习框架下，步态识别系统面临灾难性遗忘和成员推理攻击的隐私泄露风险。

🛠 **方法**：采用码分调制层（CDML）技术，在不依赖数据重放的情况下，对持续学习策略训练的步态识别模型进行保护。

✨ **主要贡献**：该方法在保持所有任务识别精度的同时，有效缓解了成员推理攻击，并消除了数据重放带来的额外开销。
