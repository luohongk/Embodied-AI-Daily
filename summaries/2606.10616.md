# Learning What to Remember: Observability-Safe Memory Retention via Constrained Optimization for Long-Horizon Language Agents

- arXiv: https://arxiv.org/abs/2606.10616v3
- arXiv ID: 2606.10616

---

🔍 **核心问题**：解决长时程语言智能体在固定记忆预算下，如何决定保留、丢弃或恢复哪些信息，以应对未来不可预见的检索需求，避免因记忆策略不当导致的性能下降。

🛠 **方法**：提出OSL-MR方法，将记忆保留建模为约束下的部分可观测随机优化问题，采用观测分离原则——在线决策仅依赖当前可观测信号，离线学习利用交互后揭示的证据信号，并训练证据预测器来优化记忆选择。

✨ **主要贡献**：证明了单步记忆优化是NP难问题，并开发了可部署的近似方法OSL-MR；在LoCoMo和LongMemEval上显著优于强基线，尤其在紧预算下表现稳健；在可精确求解的实例中，OSL-MR接近动态规划最优，而单步优化远非最优。
