# Semantic-Guided Progressive Object Removal with Gaussian Splatting

- arXiv: https://arxiv.org/abs/2607.04144v1
- arXiv ID: 2607.04144

---

🔍 **核心问题**：现有3D场景物体移除方法无法有效利用多视角语义信息，且一次性处理整个掩膜区域，导致复杂几何细节和纹理修复效果不佳。

🛠 **方法**：提出融合语义引导块匹配（SBM）与区域渐进细化（RPR）的框架，基于DINOv2编码多视角语义，通过最佳匹配令牌解码补全缺失区域，并分区域选择性优化低质量部分，以3D高斯泼溅（Gaussian Splatting）为基础实现高效重建。

✨ **主要贡献**：首次将语义引导与渐进式细化策略结合用于3D物体移除，在感知质量和跨视图一致性上显著优于现有高斯泼溅方法。
