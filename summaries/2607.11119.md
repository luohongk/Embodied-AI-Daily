# VIA: Visual Interface Agent for Robot Control

- arXiv: https://arxiv.org/abs/2607.11119v1
- arXiv ID: 2607.11119

---

🔍 **核心问题**：如何利用现有通用基础模型（FM）直接控制机器人，避免针对机器人任务进行微调，同时保留模型的通用推理和规划能力。

🛠 **方法**：提出VIA框架，将机器人控制转化为智能体任务——通过浏览器3D界面，让FM驱动的智能体截图观察、发出直观命令、观察结果并调整，无需机器人专用微调或特权状态信息。

✨ **主要贡献**：零样本解决多种桌面操作任务，在LIBERO-Goal任务中达96.7%成功率，长时程彩虹组装任务达100%成功率，证明前沿智能体通过合适界面可直接迁移至机器人控制。
