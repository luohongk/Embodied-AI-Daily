# Variational meta-learning inference for low dimensional neural system identification

- arXiv: https://arxiv.org/abs/2607.18965v1
- arXiv ID: 2607.18965

---

🔍 **核心问题**：解决深度神经网络在低数据场景下容易过拟合且缺乏可靠不确定性量化的问题。  
🛠 **方法**：提出基于摊销变分推断的完全概率化流形元学习框架，结合最大后验估计与拉普拉斯近似实现任务自适应。  
✨ **主要贡献**：在保持与确定性方法相当预测精度的同时，成功在极低数据条件下提供校准的不确定性边界。
