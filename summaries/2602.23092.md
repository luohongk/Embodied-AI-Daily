# Enhancing CVRP Solver through LLM-driven Automatic Heuristic Design

- arXiv: https://arxiv.org/abs/2602.23092v2
- arXiv ID: 2602.23092

---

🔍 **核心问题**：解决带容量约束的车辆路径问题（CVRP）在大规模实例中因NP-hard特性带来的计算挑战，提升求解效率与质量。  
🛠 **方法**：提出AILS-AHD方法，将进化搜索框架与大语言模型（LLM）结合，动态生成并优化破坏启发式策略，并引入LLM加速机制提升计算效率。  
✨ **主要贡献**：在CVRPLib大规模基准测试中，对10个实例中的8个取得了新的最优解，验证了LLM驱动的启发式设计在车辆路径优化领域的显著优势。
