# VLN-Cache: Enabling Token Caching for VLN Models with Visual/Semantic Dynamics Awareness

- arXiv: https://arxiv.org/abs/2603.07080v3
- arXiv ID: 2603.07080

---

🔍 **核心问题**：视觉与语言导航（VLN）中，大模型推理成本高，现有令牌缓存方法因忽略视觉动态（视角变化导致位置错位）和语义动态（任务阶段变化导致缓存失效）而无法直接应用。

🛠 **方法**：提出VLN-Cache框架，包含视角对齐重映射（恢复几何对应关系）和任务相关性显著性过滤器（在语义转换时阻止缓存复用），并采用层自适应熵策略平衡各层的缓存预算。

✨ **主要贡献**：在R2R-CE基准上实现最高1.52倍加速，同时保持有竞争力的导航成功率，首次将令牌缓存适配到VLN的动态场景中。
