# Coverage Path Planning: Classical Foundations, Recent Advances, and Future Directions

- arXiv: https://arxiv.org/abs/2607.10649v1
- arXiv ID: 2607.10649

---

🔍 **核心问题**：解决机器人全覆盖路径规划（CPP）问题，即在最小化路径长度、重叠、转弯次数和能耗等目标下，生成完全覆盖目标工作空间的机器人轨迹。

🛠 **方法**：系统综述了125篇代表性文献（主要发表于2015-2026年），将CPP方法分为六大类：单机器人CPP、多机器人CPP、3D CPP、约束CPP、基于学习的CPP和视觉CPP，并分析了环境知识、工作空间几何、机器人约束、感知目标和协调需求对问题的影响。

✨ **主要贡献**：提供了CPP领域从经典方法到最新进展的结构化综述，总结了各类方法的规划公式、代表性算法、优势与局限，并指出了可扩展在线规划、多机器人协调、3D与视觉覆盖、统一平台约束与资源感知覆盖、以及学习增强覆盖等未来研究方向。
