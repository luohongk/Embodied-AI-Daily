# Real-Time LiDAR Gaussian Splatting SLAM

- arXiv: https://arxiv.org/abs/2607.04127v1
- arXiv ID: 2607.04127

---

🔍 **核心问题**：现有LiDAR SLAM方法在实时大规模场景中难以同时实现高精度定位与稠密建图，且缺乏对几何结构的有效利用。

🛠 **方法**：提出实时LiDAR高斯溅射SLAM框架，通过快速G-ICP配准与基于球面光栅化的稠密建图紧密耦合，利用跟踪估计的局部协方差初始化高斯体并推导表面法向量，同时引入协方差几何分数驱动自适应剪枝与加密，并将优化后的高斯体与置信度反馈至跟踪模块。

✨ **主要贡献**：在Newer College数据集上以>20 FPS的实时速度实现86.78%的F-score，验证了方法的稳定性与可扩展性，首次将高斯溅射与LiDAR几何特性结合用于实时SLAM。
