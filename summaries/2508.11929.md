# No More Blind Spots: Learning Vision-Based Omnidirectional Bipedal Locomotion for Challenging Terrain

- arXiv: https://arxiv.org/abs/2508.11929v2
- arXiv ID: 2508.11929

---

🔍 **核心问题**：解决双足机器人在复杂地形（如杂乱室内或不平整地面）中实现全方位视觉导航与运动控制的问题，避免传统方法因渲染全方位深度图像计算成本过高而难以应用。

🛠 **方法**：提出一种结合鲁棒盲控制器与教师策略的学习框架，教师策略监督基于视觉的学生策略，并在噪声增强的地形数据上训练，避免强化学习中的渲染开销；同时引入数据增强技术加速学生训练。

✨ **主要贡献**：首次实现基于视觉的全方位双足机器人运动，在模拟和真实环境中验证了其适应多种地形的能力，训练速度相比传统方法提升最高10倍，且降低了对昂贵渲染的依赖。
