# Proximal Action Replacement for Behavior Cloning Actor-Critic in Offline Reinforcement Learning

- arXiv: https://arxiv.org/abs/2602.07441v2
- arXiv ID: 2602.07441

---

🔍 **核心问题**：在离线强化学习中，基于行为克隆（BC）正则化的演员-评论家方法会因盲目模仿次优数据集动作而引入性能上限，导致后期训练中演员无法充分利用价值函数推荐的更优动作。

🛠 **方法**：提出近端动作替换（PAR）技术，通过稳定目标策略生成更优动作，并基于动作价值函数的局部上升方向与价值不确定性约束，替换数据集中的次优动作样本，作为即插即用的训练样本替换器。

✨ **主要贡献**：理论分析了BC正则化方法的收敛局限性，并通过PAR打破性能上限；PAR兼容多种BC范式，在离线RL基准测试中与基础TD3+BC结合即可达到最先进水平。
