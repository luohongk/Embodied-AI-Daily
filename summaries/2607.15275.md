# RoboTTT: Context Scaling for Robot Policies

- arXiv: https://arxiv.org/abs/2607.15275v1
- arXiv ID: 2607.15275

---

🔍 **核心问题**：现有机器人基础模型仅能处理单步或短历史视觉运动上下文，无法有效利用长序列信息完成复杂任务。  
🛠 **方法**：提出RoboTTT，将测试时训练（Test-Time Training）集成到视觉-语言-动作策略中，通过梯度下降更新快速权重作为循环状态，结合序列动作强制与截断反向传播扩展训练上下文至8000步。  
✨ **主要贡献**：首次实现上下文长度从1K到8K的扩展，性能提升62%；支持单次人类视频模仿、在线策略改进和长时任务（如10阶段装配任务）的完整完成，整体性能比单步基线提升87%。
