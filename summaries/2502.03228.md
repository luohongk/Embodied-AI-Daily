# GARAD-SLAM: 3D GAussian splatting for Real-time Anti Dynamic SLAM

- arXiv: https://arxiv.org/abs/2502.03228v2
- arXiv ID: 2502.03228

---

🔍 **核心问题**：现有基于3D高斯泼溅的SLAM系统在动态场景中容易产生建图错误和跟踪漂移。  
🛠 **方法**：提出GARAD-SLAM，通过高斯金字塔网络对高斯体进行动态分割并映射回前端获取动态点标签，同时对动态标签的高斯体施加渲染惩罚而非简单剪枝。  
✨ **主要贡献**：在真实数据集上实现了更鲁棒的动态物体去除和跟踪，减少了渲染伪影并生成了更高质量的重建结果。
