# DEFLECT: Temporal Counterfactual Preference Learning for Delay-Robust Asynchronous VLAs

- arXiv: https://arxiv.org/abs/2605.19294v2
- arXiv ID: 2605.19294

---

🔍 **核心问题**：异步视觉-语言-动作（VLA）策略在推理时因大模型延迟导致预测与执行状态不匹配，即基于过时观测生成的动作在执行时与当前场景错位，现有方法无法直接解决这一“陈旧输入”问题。

🛠 **方法**：提出DEFLECT离线后训练框架，将延迟导致的错位转化为反事实偏好监督：冻结的参考VLA从未来执行时刻观测生成“偏好”动作块，从陈旧预测时刻观测生成“拒绝”动作块，可训练策略在部署时输入下对两者评分，学习偏好执行时刻对齐的动作，同时用监督微调锚点保留专家动作流形。

✨ **主要贡献**：无需人工偏好标签、奖励模型、在线机器人部署、架构修改或额外推理计算；在Kinetix、LIBERO及三个真实机器人任务中，相比强异步VLA基线，高延迟成功率提升最高6.4个百分点，最长延迟下在真实规模VLA上获得4.6个百分点增益。
