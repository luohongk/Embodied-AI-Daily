# DreamForge-World 0.1 Preview: A Low-Compute Real-Time Controllable World Model

- arXiv: https://arxiv.org/abs/2606.30292v1
- arXiv ID: 2606.30292

---

🔍 **核心问题**：如何在消费级GPU上实现低计算成本的实时可控世界模拟，以弥补前沿世界模拟器在交互性和部署门槛上的不足。

🛠 **方法**：基于Wan2.1-T2V-1.3B视频生成模型，采用LongLive 1自回归视频堆栈，并引入受Matrix-Game系列启发的残差动作路径，实现键盘鼠标实时控制与多模态交互。

✨ **主要贡献**：在单张RTX 4090上以低内存占用实现480p分辨率下14-15 FPS的实时交互模拟，支持多视角操作、中途重提示和分钟级滚动生成，为消费级GPU上的可控世界模型提供了高效实用的技术路线。
