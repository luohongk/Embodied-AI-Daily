# RSICCLLM: A Multimodal Large Language Model for Remote Sensing Image Change Captioning

- arXiv: https://arxiv.org/abs/2606.28266v1
- arXiv ID: 2606.28266

---

🔍 **核心问题**：现有遥感图像变化描述方法受限于传统深度学习架构的模型容量，且大模型后训练技术因数据稀缺和细粒度变化理解需求难以直接迁移。

🛠 **方法**：提出RSICCLLM框架，包括数据生成范式与指令数据集RSICI、差异感知监督微调显式提取变化表征，以及双负样本偏好优化（DNPO）通过互补负样本构建偏好数据集RSICP来优化模型。

✨ **主要贡献**：首次将大视觉语言模型后训练框架应用于遥感图像变化描述，仅用7B参数即超越更大规模模型，并公开代码与数据集。
