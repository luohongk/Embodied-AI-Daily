# WeaveLA: Event Driven Cross-Subtask Latent Memory Weaving for Repetitive Robot Manipulation

- arXiv: https://arxiv.org/abs/2606.17463v1
- arXiv ID: 2606.17463

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）策略在单步操作中表现优异，但缺乏跨子任务的信息传递机制，导致在需要依赖前序步骤的重复性操作中性能脆弱。

🛠 **方法**：提出WeaveLA，一种基于子目标完成事件触发的跨子任务记忆接口。它冻结VLA骨干网络，通过查询驱动的注意力池化将每个已完成子任务压缩为潜在记忆令牌，并直接注入下一子任务的动作生成路径。

✨ **主要贡献**：在RoboMME基准测试中，WeaveLA将最难的重复操作任务（SwingXtimes，N=3）成功率从0%提升至47.8%，且不影响单次执行任务。配对分析证实，性能提升仅出现在因果结构需要跨子任务信息的任务中。
