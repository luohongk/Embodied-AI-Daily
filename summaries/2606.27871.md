# LocalNav: Distilling Frontier VLMs and Embodied RL for On-Device Object Goal Navigation

- arXiv: https://arxiv.org/abs/2606.27871v1
- arXiv ID: 2606.27871

---

🔍 **核心问题**：解决视觉语言模型（VLM）在资源受限的机器人上本地部署时计算开销大、推理延迟高的问题，以实现低延迟的开放词汇目标导航（ObjectNav）。

🛠 **方法**：提出一种蒸馏策略，将大型前沿VLM（如Claude Sonnet 4.6）的复杂空间语义推理能力迁移至轻量级4B参数本地VLM（Qwen3.5-4B），并引入E-RLVR框架与Token生成（TG）正则化，结合量化技术压缩输出序列长度和推理延迟。

✨ **主要贡献**：在HM3D OVON基准上，基于场景图（SG）的云模型达到39.7%成功率；仅用500条推理轨迹微调的轻量模型实现34.5%成功率，接近云模型性能；通过TG正则化减少72.1% Token开销和71.8%延迟，结合量化累计降低82.8%推理延迟，为移动机器人本地低延迟VLM执行提供了可行方案。
