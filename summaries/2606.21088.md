# MV-WAM: Manifold-Aware World Action Model with Value Augmentation

- arXiv: https://arxiv.org/abs/2606.21088v1
- arXiv ID: 2606.21088

---

🔍 **核心问题**：现有世界动作模型在分布内场景表现良好，但在分布外场景中泛化能力不足，原因是视觉与动作模态的流形结构不匹配，导致联合优化时动作鲁棒性下降。

🛠 **方法**：提出MV-WAM端到端框架，通过跨模态因果掩码将动作生成与视频预测、价值函数分层对齐，采用流形感知优化方案处理模态结构异质性，并引入进度-价值调节机制检测执行偏差并实现价值引导回滚。

✨ **主要贡献**：在RoboTwin仿真中随机场景下平均成功率55.7%，超越最强基线29.3%；在双机械臂真实任务中平均成功率77.5%，证明流形感知的跨模态对齐是提升策略泛化鲁棒性的关键。
