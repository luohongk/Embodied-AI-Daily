# Are All Tokens Necessary for Visual Place Recognition? An Empirical Study of Token Reduction for Efficient Inference

- arXiv: https://arxiv.org/abs/2607.15563v1
- arXiv ID: 2607.15563

---

🔍 **核心问题**：视觉位置识别（VPR）中基于视觉Transformer的模型处理所有视觉令牌导致计算开销大，难以在实时和资源受限场景部署，因此探究是否所有令牌都是必要的。

🛠 **方法**：首次系统性地构建令牌缩减基准，评估多种令牌剪枝、合并及混合方法，在多个先进VPR模型和多样化数据集（城市、郊区、自然）上进行实验，并从识别性能、计算复杂度、推理速度、可视化及边缘设备部署等多角度分析。

✨ **主要贡献**：揭示了令牌缩减在VPR中的关键特性，例如可降低最高29%计算成本、提升最高44%吞吐量，同时识别精度下降不到1%，为高效VPR和视觉检索系统提供了实用权衡见解和全面研究基础。
