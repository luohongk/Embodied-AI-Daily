# Action Map Policy: Learning 3D Closed-loop Manipulation via Pixel Classification

- arXiv: https://arxiv.org/abs/2607.10706v1
- arXiv ID: 2607.10706

---

🔍 **核心问题**：机器人学习中的动作空间存在高维度、长时域和多模态最优解等挑战，传统方法在动作表示和损失函数设计上存在权衡。  
🛠 **方法**：提出Action Map Policy（AMP），将3D闭环操作策略学习转化为图像空间中的分类问题，通过将3D动作投影到相机图像平面，将每个像素位置作为离散类别，从而控制维度并保留多模态性。  
✨ **主要贡献**：支持高维动作的毫米级精度，无需庞大词汇表；单次前向传播即可预测完整动作序列，避免复杂噪声调度和迭代去噪，推理速度显著快于扩散策略；在多种操作任务中成功率、推理速度和空间推理能力均优于强基线方法。
