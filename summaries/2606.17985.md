# Gaussian Light Field Splatting: A Physical Prior-Driven Vision Transformer for Unsupervised Low-Light Image Enhancement

- arXiv: https://arxiv.org/abs/2606.17985v1
- arXiv ID: 2606.17985

---

🔍 **核心问题**：现有无监督低光图像增强方法在复杂非均匀光照下常出现局部曝光不均和色彩失真，且视觉Transformer缺乏对光照退化物理先验的显式建模。

🛠 **方法**：提出基于高斯光场溅射的视觉Transformer（GLFS），用各向异性高斯基函数叠加表示场景光照，并在自注意力中引入物理引导偏置以自适应推断空间增益场，同时设计色向量角损失和亮度边缘损失来减少色彩偏差与结构退化。

✨ **主要贡献**：首次将连续物理光照建模（高斯溅射）集成到Transformer架构中，实现了复杂光照下的精确均匀恢复；在光照校正和细节保持上达到最先进性能，为低光图像增强提供了新的表示范式。
