# OrbitAll: A Unified Quantum Mechanical Representation Deep Learning Framework for All Molecular Systems

- arXiv: https://arxiv.org/abs/2507.03853v2
- arXiv ID: 2507.03853

---

🔍 **核心问题**：现有AI模型在处理带电、开壳层及溶剂化分子系统时精度不足，且难以泛化到更大分子，计算成本高。

🛠 **方法**：提出OrbitAll框架，利用量子力学方法提取自旋极化轨道特征，结合SE(3)-等变图神经网络，实现任意电荷、自旋和环境效应的分子系统建模。

✨ **主要贡献**：在预测带电、开壳层和溶剂化分子时表现优异，仅需10倍少于竞争模型的数据即可达到化学精度，比密度泛函理论快10³-10⁴倍；在高度带电物种上超越UMA模型，且模型体积小50倍、数据少35倍；学习溶剂效应后，预测溶剂依赖反应路径的成本比UMA显式溶剂模拟低约100倍。
