# FORCE: Efficient VLA Reinforcement Fine-Tuning via Value-Calibrated Warm-up and Self-Distillation

- arXiv: https://arxiv.org/abs/2606.26006v1
- arXiv ID: 2606.26006

---

🔍 **核心问题**：VLA模型受限于次优数据的模仿天花板，强化学习微调虽能突破但样本效率低，主要源于Q函数不稳定导致的初始灾难性遗忘和低质量探索数据引发的低效策略更新。

🛠 **方法**：提出FORCE三阶段框架，先通过价值校准热身阶段利用在线策略回滚缓解Q函数分布偏移，再在在线阶段用校准后的Q函数过滤策略动作和专家数据，仅保留高价值动作用于策略更新。

✨ **主要贡献**：在仿真和真实任务中实现79%的成功率绝对提升，比先前RL方法高10%，训练速度加快32.5%，且无需人工干预即可避免成功率下降，推动自主机器人智能体部署。
