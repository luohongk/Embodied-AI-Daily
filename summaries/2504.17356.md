# Comprehend, Divide, and Conquer: Feature Subspace Exploration via Multi-Agent Hierarchical Reinforcement Learning

- arXiv: https://arxiv.org/abs/2504.17356v3
- arXiv ID: 2504.17356

---

🔍 **核心问题**：现有基于强化学习的特征选择方法在处理复杂数据集时，因采用“每特征单智能体”的低效范式而面临扩展性差和计算开销大的挑战。

🛠 **方法**：提出HRLFS方法，首先利用大语言模型（LLM）构建混合状态提取器，捕获每个特征的数学与语义特征；随后基于该信息对特征进行聚类，并为每个簇及子簇构建分层强化学习智能体，实现层次化特征子空间探索。

✨ **主要贡献**：通过减少智能体数量显著加速运行时间，同时提升下游机器学习性能；实验验证了该方法在效率、可扩展性和鲁棒性上优于传统及“每特征单智能体”的强化学习方法。
