# AIGS-Net: Compact Illumination Field Modeling via 2D Gaussian Splatting for Fast Low-Light Image Enhancement

- arXiv: https://arxiv.org/abs/2606.17998v1
- arXiv ID: 2606.17998

---

🔍 **核心问题**：现有低光图像增强方法在光照场建模的表达能力与计算复杂度之间存在瓶颈，难以兼顾高效性与高质量。

🛠 **方法**：提出AIGS-Net，一种超轻量级自适应2D高斯溅射网络，通过输入图像的相对亮度动态调制高斯基函数的不透明度，并利用有序Alpha合成渲染光照补偿；同时引入零参数非线性多尺度上下文编码模块提取低频结构与局部对比度，结合噪声掩膜估计、单通道Gamma映射及跨通道一致性正则化抑制噪声与色偏。

✨ **主要贡献**：仅需约40个可学习参数，在LOL和LSRW基准上显著提升细节恢复与色彩保真度，实现了增强质量与极端推理效率的有效平衡。
