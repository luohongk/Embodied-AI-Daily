# PointSplat: Compact Gaussian Splatting via Human-Centric Prediction

- arXiv: https://arxiv.org/abs/2606.32036v1
- arXiv ID: 2606.32036

---

🔍 **核心问题**：现有前馈式3D人体重建方法在多视角下重复编码相同内容，导致视角间冗余大，难以在有限算力和带宽下实现高保真与紧凑表示的平衡。

🛠 **方法**：提出PointSplat，通过输入点集直接预测高斯原语。先估计粗略几何代理并执行光线投射以剔除冗余点并建立2D-3D对应，再使用点-图像Transformer融合外观与几何特征，单次前向传播预测高斯属性。

✨ **主要贡献**：将预测限制在前景区域，显著减少高斯数量并提升新视角渲染质量。实验表明，该方法在多个数据集上实现更高效率与质量，并对视角数量与图像分辨率变化具有强鲁棒性。
