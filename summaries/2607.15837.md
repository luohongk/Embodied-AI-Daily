# A Numerically-safe Branch-Price-and-Cut Algorithm for the Length-Constrained Cycle Partition Problem

- arXiv: https://arxiv.org/abs/2607.15837v1
- arXiv ID: 2607.15837

---

🔍 **核心问题**：解决长度约束的循环划分问题（LCCP），即如何将节点集划分为最少数量的循环，且每个循环的长度不超过其中任意节点的临界时间。  
🛠 **方法**：将LCCP建模为集合划分问题，采用精确的分支定价切割算法，其中定价子问题通过动态规划实现双向搜索和对称性破缺，以高效生成改进循环。  
✨ **主要贡献**：线性规划松弛提供了极强对偶界，算法显著优于现有方法，能在更短时间内解决已有实例，并首次关闭14个未解实例（含76节点），突破此前52节点的限制。
