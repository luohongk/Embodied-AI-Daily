# TAPE: Tether-Aware Path Planning for Autonomous Exploration of Unknown 3D Cavities Using a Tangle-Compatible Tethered Aerial Robot

- arXiv: https://arxiv.org/abs/2606.30817v1
- arXiv ID: 2606.30817

---

🔍 **核心问题**：针对使用系留空中机器人在未知三维空腔中自主探索时，如何同时最小化飞行路径长度和系绳放出长度，并避免系绳缠绕。

🛠 **方法**：提出一种双层分层架构（TAPE），全局层基于前沿探索求解旅行商问题（TSP）以最小化路径距离，局部层通过可调决策函数平衡路径成本与系绳长度，实现系绳感知的路径规划。

✨ **主要贡献**：首次提出兼顾系绳管理的三维空腔自主探索方法，仿真与实地测试表明，相比纯TSP方案，本方法仅增加4.1%路径距离，但能将系绳长度始终控制在允许范围内（100%成功率 vs 53%）。
