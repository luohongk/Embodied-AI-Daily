# A Nonlinear Model Predictive Control Perspective on Gradient-Based Optimization: A New Efficient, Parameter-Free and Provably Stable Algorithm

- arXiv: https://arxiv.org/abs/2607.14600v1
- arXiv ID: 2607.14600

---

🔍 **核心问题**：解决梯度优化算法在非线性模型预测控制（NMPC）应用中参数敏感、收敛慢且缺乏稳定性保证的问题。  
🛠 **方法**：提出一种名为“搜索与加速”（SaA）的新算法，融合了新型线搜索、信赖域机制和自适应梯度加速方案。  
✨ **主要贡献**：该算法在600个箱约束优化问题基准测试中表现优异，参数鲁棒性强（默认值即可适用），且在NMPC中能缩短控制更新周期，同时保证稳定性。
