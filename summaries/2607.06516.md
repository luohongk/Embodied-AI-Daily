# Point as Skeleton: Accumulated Point Cloud Enhanced Autoregressive Generation for Closed-Loop Autonomous Driving Simulation

- arXiv: https://arxiv.org/abs/2607.06516v1
- arXiv ID: 2607.06516

---

🔍 **核心问题**：现有自动驾驶仿真方法难以同时兼顾闭环交互性（如CARLA）和真实视觉保真度（如nuScenes），导致端到端自动驾驶评估存在挑战。

🛠 **方法**：提出“Point as Skeleton”框架，利用点云骨架作为条件，通过自回归生成器逐步合成驾驶视频，并引入“Reset-and-Roll”机制适配滚动扩散推理，以支持闭环仿真中的状态更新。

✨ **主要贡献**：通过点云骨架解耦前景与背景，提供外观和几何线索，稳定自回归生成中的误差累积；在nuScenes和nuPlan上验证了该方法在闭环仿真中生成质量的提升，为视觉逼真的闭环驾驶仿真提供了新方案。
