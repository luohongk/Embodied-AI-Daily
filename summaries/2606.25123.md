# RGB: RL Guided Whole-Body MPPI for Humanoid Control

- arXiv: https://arxiv.org/abs/2606.25123v1
- arXiv ID: 2606.25123

---

🔍 **核心问题**：人形机器人在接触丰富的环境中需要既鲁棒又精确的全身控制器，但深度强化学习策略与训练目标和命令接口紧密耦合，难以在不重新训练的情况下添加新的反馈目标。

🛠 **方法**：提出一种基于强化学习引导的全身模型预测路径积分（MPPI）框架，将预训练的RL策略作为采样先验，通过MPPI成本项指定任务目标，并在线持续修正RL先验以满足这些目标。

✨ **主要贡献**：在29自由度Unitree G1人形机器人仿真中实现平均280Hz的稳定高速控制，相比纯RL基线提高了任务级精度，通过修正直行时的系统漂移和跟踪额外的全身参考信号实现。
