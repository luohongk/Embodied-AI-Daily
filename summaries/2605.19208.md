# Precision Physical Activity Prescription via Reinforcement Learning for Functional Actions

- arXiv: https://arxiv.org/abs/2605.19208v2
- arXiv ID: 2605.19208

---

🔍 **核心问题**：如何基于可穿戴设备记录的每日步数数据，为个体提供个性化的最优每日步数分布方案，以改善特定健康生物标志物（如心血管代谢风险）。  
🛠 **方法**：利用“All of Us”研究计划中的长期步数数据和重复测量的健康指标，开发了一种新的离线强化学习算法，将每日步数分布作为连续动作函数进行优化学习。  
✨ **主要贡献**：该算法在模拟实验中优于现有连续动作强化学习方法，并基于真实数据发现：最优策略通常建议增加每日步数并保持更稳定的活动模式，同时针对血糖、BMI、血压、年龄和性别等亚组提供个性化建议。
