# Trust Region Inverse Reinforcement Learning: Explicit Dual Ascent using Local Policy Updates

- arXiv: https://arxiv.org/abs/2605.11020v2
- arXiv ID: 2605.11020

---

🔍 **核心问题**：传统逆强化学习（IRL）需要每轮迭代完整求解强化学习问题，而对抗式方法虽降低计算成本却牺牲了稳定性和单调性能提升，本文旨在兼顾两者优势。

🛠 **方法**：提出信任域逆强化学习（TRIRL），通过理论证明信任域最优策略可对应同一方向上的更小全局最优更新，从而仅依赖当前策略的局部搜索即可显式优化对偶目标，避免完整求解RL问题。

✨ **主要贡献**：实现了奖励函数与策略的单调性能提升，同时保持训练稳定性；在多个复杂任务中，聚合四分位均值比现有最优模仿学习方法提升2.4倍，且学到的奖励函数能泛化至系统动力学变化场景。
