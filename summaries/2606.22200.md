# Neural Conjugate Aggregation: Identifiable Unsupervised Multi-Sensor Regression under Heterogeneous Sensor Bias

- arXiv: https://arxiv.org/abs/2606.22200v1
- arXiv ID: 2606.22200

---

🔍 **核心问题**：在无监督条件下，如何融合多个存在异构偏差和噪声的传感器数据，以准确估计潜在目标变量并量化不确定性。

🛠 **方法**：提出神经共轭聚合模型（NCAM），结合神经网络与共轭高斯推断，构建层次贝叶斯框架，通过传感器锚定和方差正则化解决非可辨识性，并集成局部自适应蒙特卡洛共形预测提供有限样本下的覆盖保证。

✨ **主要贡献**：首次在无监督多传感器回归中实现可辨识的偏差与可靠性学习，并分解认知与偶然不确定性；在合成和真实空气质量数据集上，预测精度和不确定性校准优于均值聚合、概率PCA和卡尔曼滤波等基线方法。
