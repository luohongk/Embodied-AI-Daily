# Grasp-Oriented Non-Prehensile Manipulation via Learning a Graspability Field

- arXiv: https://arxiv.org/abs/2606.30474v1
- arXiv ID: 2606.30474

---

🔍 **核心问题**：现有非抓取操作通常需要预设目标物体姿态，但实际中物体存在多种可抓取构型，且目标姿态未知。

🛠 **方法**：提出基于可抓取性场的学习方法，通过合成抓取构建可抓取集合，定义衡量物体构型抓取适宜性的标量场，并利用强化学习驱动闭环操作-抓取策略。

✨ **主要贡献**：无需预定义目标姿态或外部规划器，策略能自主将物体重排至可抓取状态并过渡到抓取；预测的可抓取距离与真实抓取成功率相关，验证了学习表征的有效性。
