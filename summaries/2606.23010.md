# A Novel Approach to Temporal QoS Estimation via Extended Kalman Filter-Incorporated Latent Feature Analysis

- arXiv: https://arxiv.org/abs/2606.23010v1
- arXiv ID: 2606.23010

---

🔍 **核心问题**：解决现有数据驱动方法在预测非平稳时序QoS数据时精度下降的问题，提升对动态波动模式的捕捉能力。  
🛠 **方法**：提出扩展卡尔曼滤波增强的潜在特征分析（EKL）模型，结合模型驱动的时序特征生成（基于扩展卡尔曼滤波）与数据驱动的时不变特征提取（基于交替最小二乘），并采用密度导向并行策略优化计算效率。  
✨ **主要贡献**：首次将双向模型-数据驱动学习框架用于时序QoS预测，理论证明了模型收敛性，实验表明其在预测精度和计算效率上均超越现有最优方法。
