# OWMDrive: Causality-Aware End-to-End Autonomous Driving via 4D Occupancy World Model

- arXiv: https://arxiv.org/abs/2606.30421v1
- arXiv ID: 2606.30421

---

🔍 **核心问题**：现有端到端自动驾驶系统缺乏对未来场景的显式建模和时序因果动态推理，导致在遮挡、突发等不确定性场景下规划不稳定或过于保守。

🛠 **方法**：提出OWMDrive框架，基于4D占用世界模型进行多步3D占用预测，将预测的未来状态作为条件先验，引导扩散模型迭代优化轨迹生成。

✨ **主要贡献**：通过显式建模场景未来演化，捕捉关键时空因果依赖，显著提升规划可靠性与安全性，尤其在部分可观测的复杂驾驶场景中表现优异。
