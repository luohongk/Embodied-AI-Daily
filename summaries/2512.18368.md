# Learning Semantic Atomic Skills for Multi-Task Robotic Manipulation

- arXiv: https://arxiv.org/abs/2512.18368v2
- arXiv ID: 2512.18368

---

🔍 **核心问题**：多任务机器人操作中，模仿学习面临次优演示、行为多模态和任务间干扰等挑战，现有技能方法缺乏语义对齐或偏向语言结构，限制了泛化能力。

🛠 **方法**：提出AtomSkill框架，通过语义对比技能对齐将演示分割为可变长度原子技能，并利用对比学习保证语义一致性与时间连贯性；同时引入关键位姿想象，让策略预测技能终止位姿和即时动作，支持平滑技能切换。

✨ **主要贡献**：构建了紧凑可复用的语义对齐原子技能库，并通过关键位姿想象实现鲁棒的长时程任务执行；在仿真和真实实验中，AtomSkill在性能上持续超越最先进的模仿学习和技能基线方法。
