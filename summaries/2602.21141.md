# SynthRender and IRIS: Open-Source Framework and Dataset for Bidirectional Sim-Real Transfer in Industrial Object Perception

- arXiv: https://arxiv.org/abs/2602.21141v2
- arXiv ID: 2602.21141

---

🔍 **核心问题**：工业场景中，专有零部件缺乏大规模标注数据集，导致基于深度学习的物体感知模型难以在非受控环境下实现鲁棒自动化部署。

🛠 **方法**：提出集成框架，利用2D到3D的真实到仿真技术从物理零件创建3D资产，并通过开源框架SynthRender实现程序化引导域随机化（GDR）生成合成图像；同时构建包含32类、19672个标注的工业真实-仿真图像数据集IRIS，支持双向仿真到真实迁移的基准测试。

✨ **主要贡献**：通过结构化消融实验量化渲染设计选择的影响，提供数据高效合成训练指南；在三个工业基准上达到高精度（公共机器人数据集99.1% mAP@50、汽车基准98.3% mAP@50、IRIS数据集95.3% mAP@50），并开源框架与数据集以促进工业物体感知研究。
