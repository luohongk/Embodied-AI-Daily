# DARIL: When Imitation Learning outperforms Reinforcement Learning in Surgical Action Planning

- arXiv: https://arxiv.org/abs/2507.05011v3
- arXiv ID: 2507.05011

---

🔍 **核心问题**：研究在手术动作规划任务中，模仿学习（IL）是否可能优于强化学习（RL），挑战RL在序列决策中更优的普遍假设。  
🛠 **方法**：提出双任务自回归模仿学习（DARIL）基线，并与三种RL变体（基于世界模型、直接视频、逆强化学习）在CholecT50数据集上进行对比。  
✨ **主要贡献**：DARIL在动作三元组识别（34.6% mAP）和下一帧预测（33.6% mAP）上显著优于所有RL方法（最高仅15.9% mAP），揭示专家标注测试集对IL的天然偏向，为手术AI开发提供关键见解。
