# Learning Gait-Aware Quadruped Locomotion with Temporal Logic Specifications

- arXiv: https://arxiv.org/abs/2607.00442v1
- arXiv ID: 2607.00442

---

🔍 **核心问题**：四足机器人强化学习依赖固定、手工设计的马尔可夫奖励函数，导致策略可解释性差且无法显式控制步态行为。

🛠 **方法**：提出基于信号时序逻辑（STL）的参数化约束框架，将步态规范（如安全边界、同步约束、速度跟踪）转化为稠密连续奖励信号，结合平滑鲁棒性近似与PPO算法进行策略优化，并在MuJoCo XLA中并行训练Barkour机器人。

✨ **主要贡献**：相比手工奖励基线，STL塑造的奖励实现了更精确的速度跟踪和更稳定的训练过程，同时通过参数化步态模板（行走-小跑、小跑、跳跃）增强了策略的可解释性与步态控制能力。
