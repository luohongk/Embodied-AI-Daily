## 0. 概览（Concise Summary）

- **主题（Topic）**：大型语言模型（LLM）的强化学习（RL）推理优化。
- **问题（Problem）**：传统的基于 token 级别的 RL 方法（如 GRPO）在优化 LLM 推理时，会导致“多样性崩溃”（diversity collapse），即模型收敛到少数几种推理模式，牺牲了解决方案的多样性，限制了 pass@k 性能。同时，token 级别的探索与语义级别的多样性之间存在根本性的不匹配。
- **方法（Method）**：提出 **LaDi-RL** 框架，将推理过程从离散的 token 空间转移到连续的潜在空间。具体地，使用一个**潜在扩散模型**作为策略（policy），在潜在空间中生成完整的推理轨迹。为了解决由此带来的信用分配难题，引入了**层次化潜在-文本展开**（hierarchical latent-text rollouts）机制，通过为每个潜在轨迹采样多个文本解码结果并聚合其奖励，来获得更干净的、低方差的潜在效用估计。
- **创新（Innovation）**：
    1.  **探索空间转移**：将 RL 的探索从 token 级别提升到轨迹级别，在连续的潜在空间中进行，直接操作语义层面的推理策略。
    2.  **扩散策略**：使用扩散模型作为策略，能够建模复杂、多模态的推理轨迹分布，并通过去噪过程进行结构化探索。
    3.  **层次化信用分配**：提出层次化展开机制，有效解耦了潜在推理质量与文本解码质量，为潜在策略和文本策略提供了互补且干净的学习信号。
    4.  **多样性引导**：在扩散去噪过程中引入排斥力（repulsion）机制，显式地鼓励潜在轨迹的多样性。
- **意义（Significance）**：LaDi-RL 提供了一种替代传统 token 级别 RL 的、有原则且有效的方法。它不仅显著提升了 pass@1 准确率，更重要的是，它成功缓解了多样性崩溃问题，使得模型在 pass@k 指标上也能超越基础模型，证明了在潜在空间中进行扩散式轨迹优化的巨大潜力。
- **一句话总结**：LaDi-RL 通过在连续潜在空间中使用扩散模型进行轨迹级别的强化学习，并配合层次化信用分配机制，有效解决了 LLM 推理优化中的多样性崩溃问题，同时提升了准确率和解决方案的多样性。

## 1. 研究动机（Motivation）

- **核心问题**：当前主流的 LLM 推理强化学习方法（如 GRPO）在 token 级别进行策略优化，这会导致**多样性崩溃**。模型在训练过程中会逐渐将概率质量集中在少数几个成功的推理模式上，而抑制其他可能同样有效的策略。
- **为什么重要**：多样性崩溃是 RL 优化的一个关键瓶颈。当模型输出分布变得集中时，探索到的轨迹多样性降低，导致 RL 的性能提升受限。这在 pass@k 指标上体现得尤为明显：经过 RL 调优的模型虽然 pass@1 提升了，但在较大的 k 值下，其性能可能反而不如未经过 RL 训练的基础模型，说明模型失去了产生多种正确解法的能力。
- **挑战在哪里**：
    1.  **探索-语义不匹配**：语义多样性是一个全局的、轨迹级别的属性，但 LLM 策略却通过局部的 token 级别扰动（如替换同义词）进行探索。这种探索方式效率低下，难以发现语义上截然不同的新策略（如从代数推导切换到几何论证）。
    2.  **信用分配难题**：当将策略从 token 空间转移到潜在空间时，问题变得更加复杂。策略在潜在空间中生成推理轨迹，但奖励只有在潜在轨迹被解码为文本并评估后才能获得。一个失败的答案可能源于糟糕的潜在推理，也可能源于对好的潜在轨迹的不完美文本解码。直接将单个解码结果的奖励分配给潜在轨迹，会混淆这两种因素，产生有噪声且可能误导的学习信号。

## 2. 创新点（Innovation）

- **关键创新**：提出了一个名为 **LaDi-RL** 的完整框架，其核心创新点在于：
    1.  **潜在扩散策略（Latent Diffusion Policy）**：将策略参数化为一个扩散模型，在由 LaDiR 框架提供的连续潜在空间中生成推理轨迹。这从根本上改变了探索的几何结构，允许模型在语义层面进行结构化、多模态的探索。
    2.  **层次化潜在-文本展开（Hierarchical Latent-Text Rollouts）**：为了解决信用分配问题，提出了一种 N×M 的展开策略。先采样 N 个潜在轨迹，再为每个潜在轨迹采样 M 个文本答案。通过聚合 M 个文本的奖励来估计潜在轨迹的效用，从而解耦潜在质量和解码噪声。
    3.  **多样性引导（Diversity Guidance）**：在扩散模型的去噪过程中，引入基于排斥力的多样性引导机制。该机制通过计算潜在轨迹之间的相互作用力，在早期去噪步骤中显式地推动它们相互远离，从而在潜在空间中鼓励更广泛的探索。
- **与现有方案的不同**：
    - 与**传统 token 级别 RL**（如 GRPO）相比，LaDi-RL 不是在离散的 token 空间进行局部扰动，而是在连续的潜在空间进行全局的轨迹级探索。
    - 与**其他潜在空间 RL 方法**（如 SofT-GRPO, Multiplex Thinking）相比，LaDi-RL 使用扩散模型作为策略，而不是简单的单步 VAE 采样或向 token 嵌入注入噪声。扩散模型能够更好地建模复杂、多模态的推理分布。
- **为什么能奏效**：
    - 潜在空间中的扰动直接改变整个推理轨迹的语义，使得探索更高效。
    - 扩散模型的多步去噪过程提供了结构化的随机性，能够自然地生成多样化的轨迹。
    - 层次化展开通过平均化文本解码噪声，为潜在策略提供了更稳定、更准确的奖励信号。
    - 排斥力引导机制显式地防止了潜在轨迹在训练过程中坍缩到同一个模式。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

LaDi-RL 的整体架构如图 3 所示，包含两个核心策略：

1.  **潜在扩散策略 (Latent Diffusion Policy, $p_\phi$)**：负责在连续潜在空间中生成推理轨迹。它基于 LaDiR 框架，将推理过程压缩成一个固定大小的潜在块（latent block）。该策略通过一个流匹配（Flow Matching）模型实现，并通过添加随机性（SDE）将确定性的 ODE 去噪过程转化为随机策略。
2.  **文本策略 (Text Policy, $p_\theta$)**：一个标准的自回归 LLM，负责将潜在扩散策略生成的最终潜在块解码为最终的文本答案。

**训练流程**：
- **层次化展开**：对于每个问题 Q，首先从潜在扩散策略中采样 N 个潜在轨迹 $\{z_n\}_{n=1}^N$。然后，对于每个潜在轨迹 $z_n$，从文本策略中采样 M 个文本答案 $\{x_{n,m}\}_{m=1}^M$。
- **奖励计算**：对每个文本答案 $x_{n,m}$ 计算任务奖励 $R(x_{n,m})$。
- **信用分配**：
    - 对于**潜在策略**，其优势函数 $\hat{A}_n$ 基于 N 个潜在轨迹的平均奖励 $\bar{R}_n = \frac{1}{M}\sum_{m=1}^M R(x_{n,m})$ 计算。这衡量了每个潜在轨迹的**平均效用**。
    - 对于**文本策略**，其优势函数 $\hat{A}_{n,m}$ 基于同一个潜在轨迹 $z_n$ 下的 M 个文本答案的奖励计算（即局部标准化）。这衡量了在给定潜在轨迹下，哪个文本实现**相对更好**。
- **联合优化**：两个策略通过一个联合损失函数 $L_{RL} = \alpha L_{latent} + (1-\alpha)L_{text}$ 进行优化。

### 3.2 关键算法与数学推导

**1. 潜在扩散策略损失（基于 Flow-GRPO）**：
潜在策略的优化遵循 GRPO 框架。其损失函数为：
$$L_{latent} = \frac{1}{N} \sum_{n=1}^N \sum_{t=1}^K \min\left( r_{n,t}(\phi) \hat{A}_n, \text{clip}(r_{n,t}(\phi), 1-\epsilon_z, 1+\epsilon_z) \hat{A}_n \right)$$
其中：
- $r_{n,t}(\phi) = \frac{p_\phi(z_t^n | z_{t-1}^n, Q)}{p_{\phi_{old}}(z_t^n | z_{t-1}^n, Q)}$ 是重要性采样比率。
- $\hat{A}_n$ 是第 n 个潜在轨迹的组相对优势，通过对 N 个平均奖励 $\{\bar{R}_n\}_{n=1}^N$ 进行标准化得到：$\hat{A}_n = \frac{\bar{R}_n - \text{mean}(\{\bar{R}_n\})}{\text{std}(\{\bar{R}_n\})}$。
- $\bar{R}_n = \frac{1}{M} \sum_{m=1}^M R(x_{n,m})$ 是第 n 个潜在轨迹的 M 个文本答案的平均奖励。

**2. 文本策略损失（局部 GRPO）**：
文本策略的优化使用一个“局部”的 GRPO 损失，其优势计算仅限于同一个潜在轨迹下的 M 个文本：
$$L_{text} = \frac{1}{NM} \sum_{n=1}^N \sum_{m=1}^M \sum_{j=1}^L \min\left( r_{n,m,j}(\theta) \hat{A}_{n,m}, \text{clip}(r_{n,m,j}(\theta), 1-\epsilon_x^l, 1+\epsilon_x^h) \hat{A}_{n,m} \right)$$
其中：
- $r_{n,m,j}(\theta)$ 是第 j 个 token 的重要性采样比率。
- $\hat{A}_{n,m} = \frac{R(y_{n,m}) - \text{mean}(\{R(y_{n,i})\}_{i=1}^M)}{\text{std}(\{R(y_{n,i})\}_{i=1}^M)}$ 是第 m 个文本在**其所在组内**的相对优势。

**3. 多样性引导（Repulsion Guidance）**：
在去噪过程的每一步 t，对一组 N 个潜在轨迹 $\{z_t^n\}_{n=1}^N$ 施加排斥力。首先计算带宽参数 $\sigma$：
$$\sigma = \text{median}_{n < n'} \|z_t^n - z_t^{n'}\|_2$$
然后，对每个潜在轨迹 $z_t^n$，计算其受到的排斥力 $F(z_t^n)$：
$$F(z_t^n) = \sum_{n' \neq n} 2 \left(1 - \frac{d_{nn'}^2}{2\sigma^2}\right) \exp\left(-\frac{d_{nn'}^2}{2\sigma^2}\right) (z_t^n - z_t^{n'})$$
其中 $d_{nn'}^2 = \|z_t^n - z_t^{n'}\|_2^2$。这个力鼓励相近的轨迹相互远离，而对距离较远的轨迹影响较小。
最终的更新步为：
$$\hat{z}_t^{n-1} = f_\phi(z_t^n, t, Q) + \gamma_t F(z_t^n)$$
其中 $f_\phi$ 是基础扩散模型的预测，$\gamma_t = \gamma_{\text{max}} \frac{t}{K}$ 是一个随时间衰减的缩放因子，确保在早期去噪步骤中施加更大的排斥力。

### 3.3 模型、训练与数据集

- **基础模型**：
    - 代码生成：Qwen3-Base (8B)
    - 数学推理：DeepSeek-R1-Distill-Qwen-7B
- **潜在推理模型**：采用 LaDiR 框架。使用一个 VAE 将短链式思考（CoT）压缩成一个固定大小的潜在块（例如 B=64 个 token）。然后训练一个流匹配模型来生成这些潜在块。LLM 部分则被训练来基于问题和潜在块生成最终答案。
- **数据集**：
    - **代码生成**：
        - SFT：Ling-Coder 数据集（1.4M Python 样本）
        - RL：从 AceCoder 和 KodCoder 中筛选的 24k 问题。
        - 评估：HumanEval, MBPP, HumanEval+, MBPP+, LiveCodeBench V6。
    - **数学推理**：
        - SFT：R1-distill 数据集。
        - RL：DeepScaleR-Preview-Dataset（约 40k 问题-答案对）。
        - 评估：AIME 2024, AIME 2025, AMC 2023, MATH-500, Minerva Math, OlympiadBench。

### 3.4 实验设置与结果

- **主要结果（表 1 和表 2）**：
    - **pass@1 提升**：LaDi-RL 在所有基准测试中均显著优于所有基线方法。
        - 在**数学推理**上，平均 pass@1 达到 43.45%，比基础模型提升 7.68%，优于最佳基线 Multiplex Thinking（41.55%）。
        - 在**代码生成**上，平均 pass@1 达到 77.08%，比基础模型提升 16.50%，大幅优于最佳基线 GRPO w/ Entropy Adv.（69.36%）。
    - **pass@k 性能（图 5）**：
        - 验证了多样性崩溃问题：AR Vanilla GRPO 在 k 值增大时，性能被基础模型反超。
        - LaDi-RL 在所有 k 值下都显著优于基础模型，并且随着 k 增大，性能差距持续扩大，有效缓解了多样性崩溃。
- **消融研究（表 3, 4, 5）**：
    - **探索空间对比（表 3）**：轨迹级别的探索（LaDi-RL）比 token 级别的探索（GRPO, SofT-GRPO）产生了更高的语义多样性（0.52 vs 0.18-0.22）和更好的 pass@k 性能。
    - **扩散 vs. VAE（表 3）**：在潜在空间中，扩散策略（LaDi-RL）远优于单步 VAE 策略（LaVAE-RL），证明了扩散模型建模复杂分布的能力。
    - **策略分解（表 4）**：潜在策略是性能提升的主要驱动力（Latent Only 70.3 vs Text Only 67.3），而联合训练两个策略能带来进一步的提升（76.4）。多样性引导机制也贡献了显著的性能增益。
    - **层次化展开（表 5）**：层次化展开（N×M）对 LaDi-RL 至关重要（+17.1 pass@1），而对 token 级别 GRPO 帮助不大（+2.2）。增加 N（探索更多潜在模式）比增加 M（为单个潜在模式解码更多文本）更有效，但两者结合效果最佳。

### 3.5 技术细节

- **Flow Matching**：LaDi-RL 的潜在策略基于流匹配模型。它将数据分布和噪声分布之间的概率路径建模为常微分方程（ODE）。为了引入随机性，将确定性 ODE 转化为等价的随机微分方程（SDE），并使用欧拉-丸山法（Euler-Maruyama scheme）进行离散化，从而得到一个高斯转移核（公式 2），使得可以计算似然比并与 GRPO 目标结合。
- **冷启动**：LaDi-RL 使用预训练的 LaDiR 模型作为初始化，而不是从头开始训练。
- **效率**：在展开过程中，使用较少的去噪步数（K=10）并禁用无分类器引导（CFG）以提高效率。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了 LLM 推理 RL 中的多样性崩溃问题**，这是该领域的一个核心挑战。
    2.  **提出了一个全新的范式**：将 RL 探索从离散、局部的 token 空间转移到连续、全局的潜在轨迹空间，为 LLM 推理优化开辟了新方向。
    3.  **提供了有效的技术方案**：扩散策略 + 层次化信用分配 + 多样性引导，这套组合拳被证明是有效且鲁棒的。
    4.  **实证了潜在扩散推理的优越性**：不仅在 pass@1 上取得领先，更重要的是在 pass@k 上突破了基础模型的上限，证明了其强大的泛化和探索能力。
- **对未来研究/应用的影响**：
    - 为设计更高效、更鲁棒的 LLM 推理 RL 算法提供了新的思路。
    - 潜在空间推理与扩散模型的结合可能在其他需要结构化探索和多样性的任务中（如对话生成、创意写作、药物分子设计）找到应用。
    - 层次化信用分配的思路可以推广到其他具有层次化结构的生成模型中。
- **局限与开放问题**（作者提及）：
    - 论文在结论部分未明确列出局限，但可以从实验和分析中推断出一些潜在问题：
        - **计算成本**：扩散模型的多步采样和 N×M 的层次化展开可能带来更高的计算开销。作者在附录中提到了效率分析，但未在正文中详细讨论。
        - **对 LaDiR 的依赖**：LaDi-RL 依赖于 LaDiR 框架来提供潜在空间表示。LaDiR 本身的质量（如 VAE 的压缩损失）可能会影响最终性能。
        - **超参数敏感性**：框架引入了多个超参数（如 N, M, K, $\alpha$, $\gamma_{max}$ 等），其调优可能较为复杂。
        - **泛化到更
