# PDAGENT-BENCH: Characterizing, Grounding, and Architecting LLM/VLM Agents for VLSI Physical Design

- arXiv: https://arxiv.org/abs/2606.17253v2
- arXiv ID: 2606.17253

---

🔍 **核心问题**：大型语言模型和视觉语言模型在超大规模集成电路前端设计表现出色，但在物理设计阶段因缺乏标准化基准而未被充分探索，该阶段需要高维、多阶段优化、与EDA工具协调交互及迭代优化。

🛠 **方法**：提出PDAGENT-BENCH基准，包含353个结合概念问题与工业实物的专家验证问题，并构建统一、符合人类习惯的智能体物理设计工作流框架，支持在真实EDA环境中进行闭环评估。

✨ **主要贡献**：建立了首个标准化、可复现的物理设计自动化评估基准，实验揭示当前模型在工具执行（如Innovus脚本生成仅42.2%准确率）和长程推理上存在显著局限，而人类技能增强的智能体工作流能有效提升端到端性能。
