# LLM Judges Can Be Too Generous When There Is No Reference Answer

- arXiv: https://arxiv.org/abs/2607.12885v1
- arXiv ID: 2607.12885

---

🔍 **核心问题**：研究在没有参考答案的情况下，LLM作为评估者是否可靠，以及参考答案的存在如何影响其判断准确性。

🛠 **方法**：采用两阶段流水线，包括校准实验（评估法官模型对任务的知识）和敏感性实验（分析参考答案存在与否及位置对模型判断的影响），并在三种语言上进行了测试。

✨ **主要贡献**：发现LLM法官在无参考答案时倾向于过度认可错误答案，而加入参考答案后判断翻转率最高达85%，且这些变化与人类判断一致；提出了在无参考评估前需进行参考感知校准的方法论。
