# HiFuzz: Hierarchical Reinforcement Learning for Semantic-Aware and Adaptive CPU Fuzzing

- arXiv: https://arxiv.org/abs/2607.06619v1
- arXiv ID: 2607.06619

---

🔍 **核心问题**：传统基于变异的CPU模糊测试难以高效探索处理器深层架构状态，导致覆盖率低且漏洞检测不足。

🛠 **方法**：提出HiFuzz，采用分层强化学习框架，通过程序代理（Program Agent）生成全局布局、基本块代理（Basic Block Agent）填充精确指令，并引入自适应覆盖奖励机制和语义感知基本块编码器解决奖励稀疏问题。

✨ **主要贡献**：在三个真实RISC-V处理器上，HiFuzz在覆盖率和漏洞检测方面显著优于现有最先进模糊测试工具。
