# ImprovedVBGS: Real-time Continual Variational Bayes Gaussian Splatting

- arXiv: https://arxiv.org/abs/2607.15542v1
- arXiv ID: 2607.15542

---

🔍 **核心问题**：针对机器人自主导航等实时应用中，现有VBGS方法因每帧对所有观测点进行迭代导致延迟过高（约84秒/帧），无法满足实时重建的严格内存与延迟要求。

🛠 **方法**：提出ImprovedVBGS加速框架，通过（i）空间截断变分推理限制局部更新范围，以及（ii）改进重分配策略（引入前向传播、截断并消除冗余动态重编译）来提升效率。

✨ **主要贡献**：在NeRF合成数据集上，将平均每帧延迟从约84.0秒降至约0.050秒（RTX 3070 Ti），实现1680倍加速，同时保持重建质量。
