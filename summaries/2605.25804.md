# Event-to-Video Reconstruction using Spatio-Temporal and Frequency-Enhanced Deep Neural Networks

- arXiv: https://arxiv.org/abs/2605.25804v1
- arXiv ID: 2605.25804

---

🔍 **核心问题**：事件相机缺乏密集强度帧，导致传统计算机视觉方法难以直接用于场景理解，现有事件到视频重建方法在恢复细节结构和抑制伪影方面表现不足。

🛠 **方法**：提出MSFET-E2V模型，采用跨域注意力模块融合时空特征与离散小波变换的频率信息，并设计轻量级小波增强跳跃块进行空间-频率联合处理。

✨ **主要贡献**：在多个真实事件数据集上实现优于现有方法的重建质量，同时显著减少参数量、GPU内存占用和推理时间。
