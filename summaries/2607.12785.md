# ExtraGS: Enhancing Endoscopic View Extrapolation via Diffusion-Guided 3D Gaussian Splatting

- arXiv: https://arxiv.org/abs/2607.12785v1
- arXiv ID: 2607.12785

---

🔍 **核心问题**：机器人辅助微创手术中，传统内窥镜视野有限，且现有神经渲染方法在超出训练轨迹的外推视角合成时会产生严重伪影。

🛠 **方法**：提出ExtraGS框架，基于扩散引导的3D高斯泼溅技术，通过不确定性引导的虚拟相机采样主动探索盲区，利用扩散模型优化渲染视图生成伪观测数据，并采用置信度加权微调策略避免可靠区域退化。

✨ **主要贡献**：显著减少外推伪影，在多个公开内窥镜数据集上实现最先进的新视角合成性能。
