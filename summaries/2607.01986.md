# Liquid Latent State Dynamics for Interpretable Turbofan Degradation Modeling

- arXiv: https://arxiv.org/abs/2607.01986v1
- arXiv ID: 2607.01986

---

🔍 **核心问题**：现有航空发动机退化建模的多变量时序模型虽能预测点值，但其内部状态缺乏对退化过程的可解释性，且难以分离健康演变与工况变化。

🛠 **方法**：提出基于液态神经网络的潜在动态模型，将历史窗口编码为潜在状态，通过液态转移模型演化状态并解码未来传感器观测；同时将潜在状态分解为退化与工况分量，并引入剩余寿命、单调风险、潜在一致性及解耦损失进行监督。

✨ **主要贡献**：在C-MAPSS基准上，该模型将传感器预测均方根误差从GRU基线的0.2438降至0.2266，尤其在多工况子集上提升显著；退化状态形成更清晰的退化时间轴，状态速度斯皮尔曼相关系数达0.5960，但直接剩余寿命回归仍弱于GRU，表明其更适合作为可解释的世界模型而非校准的寿命回归器。
