# Object Tokens as a Bridge Between Segmentation and Visual Question Answering in Robotic Surgery

- arXiv: https://arxiv.org/abs/2606.15861v1
- arXiv ID: 2606.15861

---

🔍 **核心问题**：现有手术视觉问答（VQA）方法依赖粗粒度的边界框视觉定位，无法捕捉手术对象的精细空间结构，限制了场景理解能力。

🛠 **方法**：提出统一框架，将视觉语言模型（VLM）与基于Segment Anything Model（SAM）的解码器结合，通过VLM生成的对象令牌（object tokens）同时引导答案预测和分割掩码生成，并联合优化分割与问答目标。

✨ **主要贡献**：在RAMIE和EndoVis18数据集上，该方法在手术VQA任务中一致优于基线模型，证明上下文感知的对象令牌能提升细粒度手术场景理解与视觉推理能力。
