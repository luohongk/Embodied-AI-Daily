# Explaining Attention with Program Synthesis

- arXiv: https://arxiv.org/abs/2606.19317v1
- arXiv ID: 2606.19317

---

🔍 **核心问题**：如何用人类可理解的符号化程序替代深度神经网络中不透明的计算过程，特别是Transformer语言模型中的注意力头。

🛠 **方法**：先计算注意力头在随机样本上的注意力矩阵，再用预训练语言模型根据矩阵摘要生成Python程序，最后通过重排序筛选出能最佳复现注意力模式的程序。

✨ **主要贡献**：用少于1000个程序即可高精度复现GPT-2等模型的注意力模式（平均IoU>75%），且替换25%的注意力头后仅增加16%的困惑度，同时保持下游任务性能，为神经网络的符号化透明化提供了可扩展方案。
