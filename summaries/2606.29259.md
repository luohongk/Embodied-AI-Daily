# PL-LIT: A LiDAR-Inertial-Thermal SLAM Using Point-Line Features and Thermographic Mapping

- arXiv: https://arxiv.org/abs/2606.29259v1
- arXiv ID: 2606.29259

---

🔍 **核心问题**：热成像在强光、低照度、雾天等恶劣环境下表现稳定，但热像仪自动增益控制导致图像全局对比度低、亮度恒常性假设失效，使得传统基于光流跟踪的视觉里程计性能下降。

🛠 **方法**：提出PL-LIT系统，融合LiDAR、IMU与热/可见光相机，采用在线光度标定模块与深度神经网络提取点线特征，并在误差状态迭代卡尔曼滤波器（ESIKF）中实现紧耦合状态估计，同时引入线特征约束方案与概率热强度体素地图。

✨ **主要贡献**：在可见光与热红外环境下均表现出通用性与鲁棒性，在长距离热红外数据集上达到最优性能，并基于热成像地图实现实时热异常检测功能，支持实际安全巡检应用。
