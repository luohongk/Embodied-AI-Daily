# Model-based controller assisted domain randomization for transient vibration suppression of nonlinear powertrain system with parametric uncertainty

- arXiv: https://arxiv.org/abs/2504.19715v2
- arXiv ID: 2504.19715

---

🔍 **核心问题**：解决车辆动力传动系统在非线性特性和参数不确定性下，控制策略从仿真迁移到实际系统时面临的鲁棒性不足问题。

🛠 **方法**：提出一种基于深度强化学习（DRL）的鲁棒控制方法，结合域随机化、长短期记忆（LSTM）网络和基于模型的控制器（MBC），通过潜在马尔可夫决策过程（LMDP）建模不确定性环境。

✨ **主要贡献**：首次将域随机化与模型辅助控制协同应用于非线性动力传动系统，在更紧凑的网络结构和更少训练数据下实现高泛化能力，并通过主动阻尼实验验证了其强鲁棒性。
