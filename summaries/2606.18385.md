# CaVe-VLM-CoT: An Interpretable Vision-Language Model Framework

- arXiv: https://arxiv.org/abs/2606.18385v1
- arXiv ID: 2606.18385

---

🔍 **核心问题**：视觉语言模型（VLM）容易产生幻觉，输出流畅但视觉上不忠实，现有方法无法强制步骤级引用验证或通过检索修正推理错误。

🛠 **方法**：提出CaVe-VLM-CoT框架，采用模块化反射式智能体RAG，通过五阶段闭环流程（提取器、检索器、求解器、引用注入器、验证器）实现证据驱动的推理，未验证的声明会触发结构化反馈以重新检索。

✨ **主要贡献**：首次提出包含23个组件指标的评估体系（核心为CaVeScore），无需修改架构或提示，在ScienceQA上达87.1%准确率和56.6% CaVeScore，在MMMU（30个学科）上达55.2%准确率和35.7% CaVeScore。
