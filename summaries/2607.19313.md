# Off-Context GRPO: Learning to Reason on Hard Problems using Privileged Information

- arXiv: https://arxiv.org/abs/2607.19313v1
- arXiv ID: 2607.19313

---

🔍 **核心问题**：标准强化学习方法在解决困难推理问题时失效，因为模型无法生成任何正确解答，导致学习信号为零，无法有效提升推理能力。

🛠 **方法**：提出Off-Context GRPO（OC-GRPO），在训练时利用特权信息（如解题前缀）生成引导式轨迹，并通过重要性校正目标函数，将更新方向拉回原始无引导目标，避免训练偏差。

✨ **主要贡献**：在数学推理基准上，相比标准GRPO实现平均3.9%的绝对提升（13.8%相对提升），且几乎不增加额外计算成本。
