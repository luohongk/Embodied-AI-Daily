# DAIN: Dynamic Agent-Based Interaction Network for Efficient and Collaborative Multimodal Reasoning

- arXiv: https://arxiv.org/abs/2606.30189v1
- arXiv ID: 2606.30189

---

🔍 **核心问题**：现有静态混合专家模型（MoE）在多模态融合中缺乏自适应性和高效协作推理能力，难以应对复杂真实场景。

🛠 **方法**：提出动态智能体交互网络（DAIN），通过上下文感知的元控制器动态调度稀疏激活的专用交互智能体，并压缩智能体间通信以达成共识，同时采用多目标损失函数联合优化任务精度、智能体专业性和运行效率。

✨ **主要贡献**：在ADNI、MIMIC-IV等五个基准上取得新最优结果（如ADNI准确率提升2.6%），消融实验验证了动态调度与智能体通信的关键作用，并通过稀疏激活和可解释的协作模式兼顾了计算效率与模型透明度。
