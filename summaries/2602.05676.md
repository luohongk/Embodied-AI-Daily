## 0. 概览（Concise Summary）

- **主题（Topic）**：3D 内容编辑，特别是基于图像条件（Image-Conditioned）的 3D 形状和纹理编辑。
- **问题（Problem）**：现有的 3D 编辑方法在**视觉可控性**、**几何一致性**和**可扩展性**之间存在难以调和的权衡。优化方法速度慢，多视图传播方法存在视觉漂移，而免训练方法受限于冻结的先验知识，无法从数据规模扩大中获益。
- **方法（Method）**：提出 **ShapeUP**，一个可扩展的、基于图像条件的 3D 编辑框架。它将编辑问题形式化为**原生 3D 表示空间中的有监督潜空间到潜空间（latent-to-latent）翻译**任务。该方法基于预训练的 3D 基础模型（Step1X-3D），通过 LoRA 微调，学习将源 3D 形状的潜编码和编辑图像的潜编码映射到目标编辑后 3D 形状的潜编码。
- **创新（Innovation）**：
    1.  **有监督的潜空间翻译范式**：将 3D 编辑从优化或传播问题转变为有监督学习问题，实现了快速、可扩展的编辑。
    2.  **图像作为提示（Image-as-Prompt）**：使用单张编辑后的 2D 图像作为编辑指令，提供了比文本更精细的视觉控制，并实现了隐式的、无需掩码的编辑区域定位。
    3.  **远距离帧运动（DFM）数据**：创新性地利用 3D 动画序列中时间上相距较远的帧来构建训练数据，有效支持全局姿态和形变编辑，同时保持物体身份。
- **意义（Significance）**：ShapeUP 提供了一个**鲁棒、可扩展且高效**的 3D 编辑新范式。它解决了现有方法在编辑保真度和源形状保持性之间的核心矛盾，在身份保持和编辑保真度方面均优于现有基线，为原生 3D 内容创作开辟了新的道路。
- **一句话总结**：ShapeUP 通过将 3D 编辑转化为在原生 3D 潜空间中的有监督翻译任务，并利用创新的运动数据，实现了快速、可控、可扩展且无需掩码的 3D 形状与纹理编辑。

## 1. 研究动机（Motivation）

- **解决的问题**：尽管 3D 生成模型取得了巨大成功，但如何**精确、高效地编辑**已有的 3D 资产仍然是一个重大挑战。用户不仅需要生成，更需要修改和操控现有模型。
- **为什么重要**：3D 编辑是 3D 内容创作流程中的关键环节，对于游戏、影视、AR/VR 等领域的实际应用至关重要。一个理想的编辑框架应同时满足四个关键标准：
    1.  **原生 3D 一致性**：编辑必须与 3D 表示整体融合，保持几何一致性，避免视角伪影。
    2.  **隐式定位**：编辑区域应能从条件信号中自动推断，无需手动标注掩码。
    3.  **精细控制**：通过视觉条件（如图像）提供比自然语言更精确的控制。
    4.  **内在可扩展性**：编辑框架应能随着数据和模型规模的增大而性能提升。
- **挑战在哪里**：现有方法无法同时满足以上所有标准。
    - **基于优化的方法**（如 SDS）：速度极慢（分钟到小时级），且容易出现多面问题（Janus problem）。
    - **多视图传播方法**（如 EditP23）：虽然更快，但在 2D 到 3D 的转换过程中会引入**配准伪影**和**身份漂移**。
    - **免训练方法**（如 Nano3D）：受限于冻结的模型权重，无法从更大规模的数据和模型容量中受益，违背了“苦涩的教训”（The Bitter Lesson）。

## 2. 创新点（Innovation）

1.  **新颖的问题形式化**：将 3D 编辑视为一个**有监督的潜空间翻译**问题。这是核心创新，它绕开了优化和 2D 传播的固有缺陷，直接在一个统一的原生 3D 潜空间内学习从“源形状”到“目标形状”的映射。这使得编辑过程是前馈的（feed-forward），速度极快。

2.  **图像作为提示（Image-as-Prompt）的统一接口**：使用单张编辑后的 2D 图像作为唯一的编辑指令。这提供了比文本更高的带宽和更精细的控制，并且自然地实现了**隐式、无需掩码的定位**。模型需要自己学习图像中的编辑意图并应用到整个 3D 形状上，而不是依赖用户提供的掩码。

3.  **远距离帧运动（DFM）数据生成策略**：为了训练全局编辑（如姿态变化、形变），作者创新性地从 3D 动画序列中提取**时间上相距较远的帧**作为编辑对。这些帧在保持物体身份的同时，自然地包含了连贯的姿态和形变变化，为监督全局编辑提供了高质量的训练信号。这是使模型能够进行全局编辑的关键。

4.  **轻量级微调策略**：基于强大的预训练 3D 基础模型（Step1X-3D），通过**LoRA** 进行轻量级微调。这种方法既利用了基础模型的强大生成先验，又通过少量参数调整使其适应编辑任务，实现了高效且可扩展的训练。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

ShapeUP 采用了两阶段架构，分别处理几何和纹理编辑，如图 2 所示。

- **几何编辑阶段**：
    1.  **输入**：源 3D 形状（$S_{src}$）和一张编辑后的 2D 图像（$I_{edit}$）。
    2.  **编码**：使用预训练的 **形状 VAE 编码器**将 $S_{src}$ 压缩为潜编码（$K=1024$ 个潜变量）。使用预训练的 **图像编码器**（CLIP + DINOv2）提取 $I_{edit}$ 的特征。
    3.  **翻译**：将源形状潜编码和图像特征拼接，作为条件输入到一个 **3D 扩散 Transformer (DiT)** 中。该 DiT 基于 Step1X-3D 的 MMDiT 架构，并通过 **LoRA** 进行微调。模型学习将源形状的潜编码翻译为编辑后形状的潜编码。
    4.  **解码**：使用预训练的 **形状 VAE 解码器**将编辑后的潜编码解码为最终的 3D 网格（$S_{edit}$）。

- **纹理编辑阶段**：
    1.  **输入**：编辑后的几何体（$S_{edit}$）、编辑图像（$I_{edit}$）以及源纹理的多视图渲染（$IMV_{src}$）。
    2.  **几何引导**：从 $S_{edit}$ 渲染出**位置图**和**法线图**，作为几何条件。
    3.  **纹理融合**：基于预训练的**图像到多视图扩散模型**（MV-Adapter）。源纹理的多视图特征通过**交叉注意力**机制注入，并添加**视角位置编码**以区分编辑提示和源纹理。几何条件作为**残差信号**注入。
    4.  **输出**：生成一组与编辑后几何体一致的、多视图纹理图像，然后通过纹理烘焙（texture baking）得到最终的带纹理网格。

### 3.2 关键算法与数学推导

- **几何编辑的 Classifier-Free Guidance (CFG)**：
    在采样过程中，使用以下公式进行引导：
    $$\tilde{\epsilon}^G_\theta = \epsilon^G(\emptyset, \emptyset) + s^G_i \left[ \epsilon^G(c_i, \emptyset) - \epsilon^G(\emptyset, \emptyset) \right] + s^G_s \left[ \epsilon^G(c_i, c_s) - \epsilon^G(\emptyset, \emptyset) \right]$$
    - $\epsilon^G(\emptyset, \emptyset)$：无条件预测。
    - $\epsilon^G(c_i, \emptyset)$：仅以编辑图像 $c_i$ 为条件的预测。
    - $\epsilon^G(c_i, c_s)$：同时以编辑图像 $c_i$ 和源形状 $c_s$ 为条件的预测。
    - $s^G_i$ 和 $s^G_s$ 分别是**图像引导尺度**和**形状引导尺度**，用于控制编辑保真度和身份保持之间的平衡。实验中分别设为 2.5 和 3.5。

- **纹理编辑的 Classifier-Free Guidance (CFG)**：
    类似地，纹理编辑的 CFG 公式为：
    $$\tilde{\epsilon}^T_\theta = \epsilon^T(\emptyset, \emptyset) + s^T_i \left[ \epsilon^T(c_i, \emptyset) - \epsilon^T(\emptyset, \emptyset) \right] + s^T_{mv} \left[ \epsilon^T(c_i, c_{mv}) - \epsilon^T(\emptyset, \emptyset) \right]$$
    - $s^T_i$ 是编辑图像引导尺度。
    - $s^T_{mv}$ 是**多视图源纹理引导尺度**，用于控制从源纹理中保留细节的程度。实验中分别设为 2.5 和 3.5。

### 3.3 模型、训练与数据集

- **基础模型**：基于 **Step1X-3D** 的图像到 3D 基础模型，包括其预训练的几何 DiT 和纹理 UNet（MV-Adapter）。
- **数据集**：
    - **来源**：Objaverse 数据集中的 7,430 个带纹理网格。
    - **Parts 样本**（6,870 个）：通过逐步移除物体的部件（part）来创建编辑对，用于训练局部编辑。
    - **DFM 样本**（560 个）：从 Objaverse 动画子集中提取，每个动画序列选取 3 个时间上相距较远的帧作为编辑对，用于训练全局编辑（姿态、形变）。如图 5 所示。
- **训练细节**：
    - **几何管道**：在单张 NVIDIA L40S GPU 上训练，LoRA 秩为 128，batch size 为 8，共 500K 次迭代。对 DFM 样本进行上采样，使其采样概率是 Parts 样本的 3 倍。
    - **纹理管道**：在单张 NVIDIA A100 GPU 上训练，batch size 为 4，共 27K 次迭代。同样对 DFM 样本进行上采样。

### 3.4 实验设置与结果

- **基准测试（BenchUp）**：作者构建了一个新的基准测试，包含 24 个多样化的网格和 100 个编辑条件，涵盖局部修改、全局形变、姿态变化和纹理/材质编辑。
- **评估指标**：
    - **条件对齐（Condition Alignment）**：衡量编辑后的网格与编辑图像的一致性。使用 SSIM、LPIPS、CLIP-I、DINO-I 和 CLIP-Dir。
    - **遮挡区域保真度（Occluded Region Fidelity）**：衡量在编辑图像中不可见或部分可见的区域是否被正确保留。使用 CLIP-I 和 DINO-I。
- **基线方法**：与 **EditP23**（多视图传播方法）和 **3DEditFormer**（有监督学习方法）进行比较。
- **定量结果**（表 1）：ShapeUP 在所有指标上均优于基线方法，尤其是在**遮挡区域保真度**上取得了显著提升（CLIP-I: 0.928 vs. 0.880, DINO-I: 0.878 vs. 0.748），表明其在保持源形状身份方面具有巨大优势。
- **定性结果**（图 6）：可视化比较显示，ShapeUP 能更好地执行编辑指令（如改变姿态、添加部件），同时更完整地保留源物体的身份和未编辑区域的细节。
- **用户研究**（图 7）：在 664 次比较中，82.5% 的用户偏好 ShapeUP 胜于 EditP23，74.6% 的用户偏好 ShapeUP 胜于 3DEditFormer，证明了其卓越的感知质量。
- **消融实验**（表 2，图 8，图 9）：
    - **几何编辑**：验证了使用 1024 个潜变量比 256 或 512 个效果更好。验证了 DFM 数据对于实现全局编辑（如姿态变化）至关重要，如图 8 所示。
    - **纹理编辑**：验证了通过交叉注意力注入源纹理比直接通道拼接（ConcatMV）效果更好，后者虽然条件对齐分数高，但遮挡区域保真度差。图 9 展示了 CFG 尺度可以控制编辑保真度和纹理保持之间的权衡。

### 3.5 技术细节

- **形状潜编码**：源形状 $S_{src}$ 被编码为 2048 个潜向量，但实验发现其高度冗余，因此仅采样 $K=1024$ 个作为条件，在保持性能的同时降低了计算量。
- **数据不平衡处理**：由于 Parts 样本远多于 DFM 样本，训练时对 DFM 样本进行上采样，使其有效采样概率达到 22%，以平衡模型对局部和全局编辑的学习能力。
- **纹理注入策略**：在纹理编辑阶段，源纹理的多视图特征通过交叉注意力注入，并添加了视角位置编码，使模型能够区分“编辑目标”和“源纹理细节”，从而更好地保留源纹理信息。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    - 解决了 3D 编辑领域长期存在的**可控性、一致性和可扩展性**三者不可兼得的难题。
    - 提出了一个**简单、优雅且有效**的范式：将编辑视为有监督的潜空间翻译，这为未来的 3D 编辑研究提供了新的思路。
    - 通过 **DFM 数据**，为训练全局 3D 编辑模型提供了一种实用且可扩展的方法。
    - 在**身份保持**和**编辑保真度**上取得了显著的性能提升，并通过用户研究验证了其优越性。

- **对未来研究/应用的影响**：
    - **内容创作**：ShapeUP 可以极大地加速 3D 艺术家的创作流程，允许他们通过简单的 2D 编辑快速迭代 3D 模型。
    - **研究方向**：证明了“苦涩的教训”在 3D 编辑领域同样适用，即利用大规模数据和可学习模型是通往成功的关键。这可能会引导更多研究转向基于基础模型的有监督 3D 编辑。
    - **技术路线**：其“图像作为提示”的接口比文本更直观、更精确，可能成为未来 3D 编辑工具的主流交互方式。

- **局限与开放问题**：
    - **VAE 瓶颈**：编辑质量受限于预训练形状 VAE 的表示能力，对于具有精细几何细节或非物体中心构图的资产，VAE 的重建损失会导致细节丢失（如图 10 所示）。
    - **数据集规模和偏差**：当前训练集相对较小，且偏向于封闭的、以物体为中心的资产。扩展到更开放、更复杂的场景（如室内场景）是一个重要的未来方向。
    - **编辑类型**：虽然支持多种编辑，但对于某些高度复杂的拓扑变化或语义编辑可能仍有挑战。

## 5. 澄清与简化（Clarifications and Simplifications）

- **“潜空间翻译”类比**：可以把 3D 形状想象成一种“语言”。传统的优化方法就像是用“造句”的方式（反复修改词语直到句子通顺）来编辑，非常慢。多视图传播方法就像把 3D 文章翻译成 2D 图片，在图片上修改，再翻译回 3D，过程中容易丢失原意。而 ShapeUP 的方法是直接学习一本“同义词典”，输入一个“源句子”和一张“目标图片”，模型就能直接输出一个符合图片意思的“目标句子”。这个过程是前馈的，非常快，而且因为是直接在 3D 语言内部操作，所以语义保持得更好。

- **“图像作为提示”如何实现隐式定位**：想象你有一张椅子的 3D 模型，你想给它加一个扶手。你不需要用笔在 3D 模型上画出一个区域（掩码），只需要找一张带扶手的椅子图片作为提示。ShapeUP 的模型通过学习大量的“椅子 + 扶手图片 -> 带扶手的椅子”这样的数据对，自己就学会了“扶手”应该加在哪里，以及如何与原有结构融合。这就是隐式定位，模型从图像中推断出编辑意图和区域。

- **DFM 数据的作用**：想象一个 3D 人物模型，你想让它从“站立”变成“挥手”。如果只用“移除部件”的数据训练，模型只会学习删除或添加东西，不知道怎么改变姿态。DFM 数据就像是从一段人物动画中截取“站立”和“挥手”两个关键帧，告诉模型：“看，同一个角色，只是姿态变了”。这样模型就学会了如何在不改变身份的前提下进行全局的、非局部的形变和姿态编辑。
