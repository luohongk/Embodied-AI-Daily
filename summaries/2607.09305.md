# From Classification to Localization and Clinical Validation: Large-Scale Development of a Deep Learning System for Thoracic Disease Detection on Chest Radiographs in Thailand

- arXiv: https://arxiv.org/abs/2607.09305v1
- arXiv ID: 2607.09305

---

🔍 **核心问题**：解决泰国及东南亚地区因放射科医生严重短缺而导致的胸部X光片（CXR）诊断瓶颈，并提升深度学习模型在本地人群中的准确性。

🛠 **方法**：开发了Inspectra CXR version 5系统，采用DenseNet-121骨干网络结合Attend-and-Compare Modules（ACM）和概率类激活图（PCAM）聚合层，实现多标签胸部疾病分类与弱监督病灶定位。

✨ **主要贡献**：在874,858张本地数据上训练，内测集平均AUROC达0.994，跨13家医院泛化集达0.970；病灶定位准确率77.9%，放射科医生使用评估中分类一致率93.6%，系统可用性评分89，证明该系统高精度、强泛化且获临床信任。
