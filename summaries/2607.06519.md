# FreqDepthKV: Frequency-Guided Depth Sharing for Robust KV Cache Compression in Long-Context LLM Inference

- arXiv: https://arxiv.org/abs/2607.06519v1
- arXiv ID: 2607.06519

---

🔍 **核心问题**：长上下文大语言模型推理中，KV缓存的内存和带宽成本日益成为瓶颈，而激进压缩会丢失检索和多步推理所需的逐层关键信息。

🛠 **方法**：提出FreqDepthKV，一种推理时缓存压缩方法，将相邻层的KV状态分解为共享的低频深度分量和稀疏的高频残差，并通过轻量在线探针根据注意力头对重建敏感logits的贡献，动态分配其到共享深度、残差深度或精确缓存模式。

✨ **主要贡献**：在长上下文问答、检索、摘要和代码生成任务中，FreqDepthKV在显著降低缓存预算下保持任务精度，例如在32k token预填窗口下达到58.3 Exact Match、63.0 F1、32.5 ROUGE-L和48.1 pass@1，接近全KV性能且优于先前压缩方法；同时将解码吞吐提升至70.4 tokens/s，TTFT降至2.06秒，峰值KV内存降至6.2 GB，实现3.9倍有效压缩比。
