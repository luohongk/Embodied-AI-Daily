# BiSCo-LLM: Lookup-Free Binary Spherical Coding for Extreme Low-Bit Large Language Model Compression

- arXiv: https://arxiv.org/abs/2607.08643v1
- arXiv ID: 2607.08643

---

🔍 **核心问题**：大型语言模型（LLM）在部署时面临内存、带宽和存储限制，现有低比特压缩方法在目标比特率接近2比特/权重时，标量量化表示能力不足，而向量量化需额外存储码本和索引。

🛠 **方法**：提出BiSCo-LLM，一种无码本的二进制球面编码框架。首先将局部权重块映射到单位超球面并二值化为紧凑球面码；其次通过残差BSQ阶段编码重构误差；最后对每个Transformer模块进行类别恢复蒸馏，并采用8位保护通道稳定敏感通道。

✨ **主要贡献**：实现了无需显式码本和索引查找的极端低比特压缩，通过球面编码和残差机制提升表示能力，同时保持高效存储，报告预算包括二进制码、神经解码器、保护通道、LoRA适配器和元数据。
