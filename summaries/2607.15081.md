# DataShield: Uncovering Risky Fine-Tuning Data Across LLMs Through Consensus Subspace Alignment

- arXiv: https://arxiv.org/abs/2607.15081v1
- arXiv ID: 2607.15081

---

🔍 **核心问题**：现有方法在识别导致大语言模型安全能力下降的微调数据时，依赖单一模型及其分词器计算的平均向量，导致风险评估效果有限且难以迁移。

🛠 **方法**：提出DataShield框架，通过多个安全对齐大语言模型的共识子空间对齐，提取联合安全关键语义空间中的安全与不安全子空间，并基于数据样本与这些子空间的相对对齐程度评估风险，支持样本级过滤和细粒度片段级掩码。

✨ **主要贡献**：相比现有方法，DataShield在样本过滤和片段掩码下分别将攻击成功率降低14.6%和32.3%，同时保持下游任务性能，且无需针对目标模型进行特定风险计算。
