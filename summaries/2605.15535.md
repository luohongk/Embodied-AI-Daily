# Learning Dynamic Structural Specialization for Underwater Salient Object Detection

- arXiv: https://arxiv.org/abs/2605.15535v1
- arXiv ID: 2605.15535

---

🔍 **核心问题**：水下显著目标检测（USOD）面临图像退化导致的定位不准、区域碎片化和边界预测粗糙等问题。  
🛠 **方法**：提出DSS-USOD方法，从单张水下图像提取共享基础表示，分解为边界敏感和区域一致两个分支，并通过空间协调模块动态调整其贡献，同时引入协同结构监督促进分支专业化。  
✨ **主要贡献**：在基准数据集上取得领先性能，并在水下机器人实际部署中验证了其对水下目标检测的有效性。
