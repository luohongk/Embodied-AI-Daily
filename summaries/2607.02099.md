# X-Splat: Gaussian Splatting for 3D CBCT Generation from Single Panoramic Radiograph

- arXiv: https://arxiv.org/abs/2607.02099v1
- arXiv ID: 2607.02099

---

🔍 **核心问题**：从单张全景X光片（PXR）生成3D牙科CBCT图像，解决因全景成像沿弯曲X射线路径将3D衰减信息压缩为2D图像而导致的深度信息缺失问题。

🛠 **方法**：提出X-Splat框架，利用已知全景采集几何结构初始化可学习各向异性高斯原语，通过Beer-Lambert重投影和多视角放射学训练监督进行单次前向调整，并辅以轻量级残差精炼器引入数据集级解剖先验。

✨ **主要贡献**：首次将高斯泼溅（Gaussian Splatting）用于PXR到CBCT的3D生成，引入基于分割的几何感知评估指标，在恢复牙齿、皮质边界、牙槽骨结构及下颌管方面显著优于NeRF和GAN基线方法。
