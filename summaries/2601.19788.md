# Knowledge-Aware Evolution for Task-Free Streaming Federated Continual Learning with Arbitrary Class Overlap

- arXiv: https://arxiv.org/abs/2601.19788v2
- arXiv ID: 2601.19788

---

🔍 **核心问题**：解决无任务标识、数据流式到达且类别任意重叠的联邦持续学习场景中，新旧知识混淆及本地推理能力不足的问题。

🛠 **方法**：提出FedKACE框架，包含自适应模型切换机制、基于梯度范数平方比的响应式平衡重放策略，以及保留高信息量和边界样本的全局缓冲区维护策略。

✨ **主要贡献**：通过理论分析和多场景实验验证，FedKACE能有效提升客户端在类别重叠下的持续学习性能，平衡新知识获取与旧知识保留。
