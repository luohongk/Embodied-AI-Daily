# Reflex: Real-Time VLA Control through Streaming Inference

- arXiv: https://arxiv.org/abs/2607.14695v1
- arXiv ID: 2607.14695

---

🔍 **核心问题**：流匹配视觉-语言-动作（VLA）模型在实时机器人控制中面临推理效率瓶颈——迭代去噪过程与KV缓存不兼容，导致计算复杂度高或缓存复用不准确。

🛠 **方法**：提出Reflex框架，利用“时间步不变性”将感知编码器与去噪循环解耦，通过划分注意力上下文为静态、滑动和动态区域实现O(1)增量缓存更新；引入AdaRMSNorm自适应归一化层防止数值崩溃，并采用异步流水线和算子融合提升吞吐量。

✨ **主要贡献**：在LIBERO和Kinetix基准上实现2.58倍推理加速和50Hz稳定流式推理，反应延迟降低54%，且无性能退化，为实时VLA控制提供了高效部署方案。
