# ISO: An RLVR-Native Optimization Stack

- arXiv: https://arxiv.org/abs/2607.19331v1
- arXiv ID: 2607.19331

---

🔍 **核心问题**：强化学习与可验证奖励（RLVR）在提升语言模型推理能力时，其优化层（将奖励反馈转化为权重更新）的机制尚不明确，缺乏系统性的理论解释和高效方法。

🛠 **方法**：提出等谱优化（ISO）框架，通过分析模型权重的奇异结构，发现RLVR可复用基模型的权重谱（spectrum），仅需优化输入/输出奇异帧（frames）。ISO包含离线（ISO-Merger）和在线（ISO-Optimizer）两种实现：前者合并共享基模型专家的帧变化，无需额外数据或蒸馏；后者在固定谱下用AdamW或Muon等优化器更新帧变量。

✨ **主要贡献**：ISO揭示了RLVR的“谱继承”机制，即保留基模型谱、优化帧结构。实验表明，ISO-Merger在无数据合并方法中性能最强；ISO-Optimizer在1.5B-8B参数模型上显著加速训练（如Qwen3-8B-Base上，ISO-AdamW用100步达到AdamW 270步的精度，210步后进一步提升）。
