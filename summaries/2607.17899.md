# SEAM-V: A Hybrid-Decoupled RISC-V Vector Processor with Backend-Visible EP Context for Sustained Vector Throughput

- arXiv: https://arxiv.org/abs/2607.17899v1
- arXiv ID: 2607.17899

---

🔍 **核心问题**：传统紧耦合RISC-V向量处理器在执行短向量、循环尾部和控制/内存交错阶段时，因向量指令供应间隙、标量侧延迟和保守依赖处理导致吞吐量下降。

🛠 **方法**：提出SEAM-V混合解耦向量执行架构，通过任务级解耦、本地指令供应和VLIW风格打包形成连续执行包（EP）流，并在动态向量后端保留EP上下文以实现候选冒险抑制和请求绑定预取，同时支持有限跨EP向量重叠。

✨ **主要贡献**：相比基于Ara的紧耦合实现，SEAM-V在17个代表性内核上实现1.34倍几何平均加速，其中一维变长AVL内核在AVL=32时加速比接近3倍。
