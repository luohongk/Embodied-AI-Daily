# From Collaboration to Regulation: Characterizing Governance Practice in Three Deep Learning Open Source Communities

- arXiv: https://arxiv.org/abs/2607.19022v1
- arXiv ID: 2607.19022

---

🔍 **核心问题**：大型深度学习开源社区（如PyTorch、TensorFlow、Paddle）面临协作与质量控制挑战，缺乏关于如何系统化制定、引入和修订治理规则的指导。

🛠 **方法**：采用混合方法实证研究，基于制度分析与发展框架分析109份治理文档，并追踪超过1700次代码提交，识别规则主题与演变模式。

✨ **主要贡献**：归纳出17个规则主题（分属7类），发现操作规则出现更早且修订频繁，结构规则出现较晚且变化较少；提炼出四项治理功能，并总结出33条可操作的治理实践。
