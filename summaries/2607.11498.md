# See like a Robot: Robot-Centric Pointmaps for Vision-Language-Action Models

- arXiv: https://arxiv.org/abs/2607.11498v1
- arXiv ID: 2607.11498

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在相机视角下观察场景，但动作定义在机器人自身3D坐标系中，导致视角不匹配，限制了模型在不同相机设置下的泛化能力。

🛠 **方法**：提出机器人中心点图（robot-centric pointmaps），将图像像素存储为机器人坐标系中的3D点坐标，在保留密集网格结构的同时提供机器人帧的3D几何信息，可无缝集成到现有2D VLA模型中。

✨ **主要贡献**：在RoboCasa数据集上，点图显著提升了pi0.5和SmolVLA的性能，优于相机视角和3D感知基线；真实机器人实验中，当相机移动到训练未见的视角时，点图相比纯RGB策略优势更明显。
