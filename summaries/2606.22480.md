# ARP: Enhancing Quantized Skill Abstractions via Visual Alignment and Iterative Refinement for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.22480v1
- arXiv ID: 2606.22480

---

🔍 **核心问题**：现有基于离散量化的技能模仿学习方法缺乏视觉-语义对齐，且离散化导致连续动作生成精度不足，影响长时域机器人操作任务的性能。

🛠 **方法**：提出对齐精炼策略（ARP），包含视觉-动作对比对齐模块，将视觉嵌入与预量化动作表示在共享空间中对齐，并保持状态无关的技能解码器；同时引入轻量级迭代残差头（IRH），通过两步精炼恢复细粒度控制。

✨ **主要贡献**：在LIBERO和Meta-World基准上达到最先进性能，并在Kuavo 4 Pro人形机器人平台的两个挑战性任务中验证了有效性，显著优于多个基线方法。
