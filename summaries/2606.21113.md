# Object-Centric Dataset Resources for Constrained-Data Image Generation and Augmentation

- arXiv: https://arxiv.org/abs/2606.21113v1
- arXiv ID: 2606.21113

---

🔍 **核心问题**：现有图像数据集主要面向分类、检测或场景理解，缺乏针对有限标注数据场景下可控的物体中心图像生成与增强的标准化资源。

🛠 **方法**：构建了三个物体中心数据集资源——Cityscapes-Pedestrian、TrafficSigns和COCO PottedPlant，统一提供256×256的物体中心裁剪图像和边界框标注，并附有脚本、清单、校验和及重建文档。

✨ **主要贡献**：发布了包含3009个TrafficSigns样本、2156个Cityscapes-Pedestrian记录和7679个COCO PottedPlant记录的数据集集合，支持标签检查、子集创建、数据重建及物体中心图像生成或合成数据增强方法的评估。
