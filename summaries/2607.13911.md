# How to Guide LLM Generation: Dual-Surrogate Guided Search for Automated Heuristic Design

- arXiv: https://arxiv.org/abs/2607.13911v1
- arXiv ID: 2607.13911

---

🔍 **核心问题**：在基于大语言模型的自动化启发式设计中，如何在有限的查询和评估预算下，高效选择父代启发式与生成算子，以提升搜索效率。

🛠 **方法**：提出双代理引导搜索方法，通过过渡代理预测算子-父代动作产生的子代表示分布，并结合实例条件效用代理估计子代性能，采用不确定性感知的采集规则选择生成动作。

✨ **主要贡献**：首次将双代理引导机制引入LLM的启发式设计，在多个任务上超越现有基线，且行为分析表明其超越了简单的存档排序或固定算子偏好。
