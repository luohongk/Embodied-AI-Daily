# Follow-Bench: A Unified Motion Planning Benchmark for Socially-Aware Robot Person Following

- arXiv: https://arxiv.org/abs/2509.10796v4
- arXiv ID: 2509.10796

---

🔍 **核心问题**：机器人跟随特定人员时，如何在动态环境中同时保障目标与周围人群的安全性和舒适性，现有研究缺乏统一评估框架。

🛠 **方法**：提出Follow-Bench基准测试平台，模拟多种目标轨迹模式、人群动态和环境布局，并重新实现8种代表性跟随规划器，系统性地整合安全与舒适性指标。

✨ **主要贡献**：首次对机器人人员跟随任务进行综合研究，通过仿真和真实实验量化了现有规划器的安全-舒适性权衡，揭示了实际部署中的挑战与未来方向。
