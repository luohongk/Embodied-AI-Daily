# RAGU: A Multi-Step GraphRAG Engine with a Compact Domain-Adapted LLM

- arXiv: https://arxiv.org/abs/2607.11683v1
- arXiv ID: 2607.11683

---

🔍 **核心问题**：现有GraphRAG系统通过单次提取构建知识图谱，导致实体噪声大、检索脆弱，且大模型在提取任务中过度依赖参数规模，效率低下。

🛠 **方法**：提出RAGU引擎，采用两阶段类型化提取、DBSCAN去重、LLM摘要和Leiden社区检测的分离式流水线；并训练了7B参数的Meno-Lite-0.1模型，专注于语言技能优化。

✨ **主要贡献**：Meno-Lite-0.1在知识图谱构建上超越Qwen2.5-32B（调和平均+12.5%），在GraphRAG-Bench医学任务中证据召回率达0.84（对比≤0.76），且RAGU在单GPU上即可运行，开源MIT协议。
