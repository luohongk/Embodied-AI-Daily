# Patch Policy: Efficient Embodied Control via Dense Visual Representations

- arXiv: https://arxiv.org/abs/2607.18236v1
- arXiv ID: 2607.18236

---

🔍 **核心问题**：现有机器人策略要么将视觉观测压缩为单个全局token，丢失空间细节，要么从头训练视觉骨干，无法利用大规模预训练视觉特征；而使用密集补丁特征的大型视觉-语言-动作模型（VLA）则计算开销巨大，不适合高频控制。

🛠 **方法**：提出Patch Policy，一种轻量级架构扩展，通过引入块因果注意力掩码，使基于Transformer的策略能直接处理预训练的密集补丁token，同时保持时间因果性并兼容其他状态信息，无需完整视觉-语言模型（VLM）的计算负担。

✨ **主要贡献**：在4个模拟和3个真实环境套件中，相比使用全局池化表示的最先进策略，性能相对提升40%；相比微调后的OpenVLA-OFT，性能提升18%，而参数量仅为其0.7%，实现了高效、快速且高精度的具身控制。
