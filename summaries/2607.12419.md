# DeGuNet: Depth-Guided Ultra-Compact Backbones for Efficient LiDAR-Camera 3D Detection

- arXiv: https://arxiv.org/abs/2607.12419v1
- arXiv ID: 2607.12419

---

🔍 **核心问题**：现有LiDAR-相机3D检测框架过度依赖基于2D语义任务预训练的庞大视觉骨干网络，导致参数冗余和结构错位，难以适应鸟瞰图几何中LiDAR投影的极端稀疏性。

🛠 **方法**：提出DeGuNet，一种超紧凑、即插即用的图像骨干网络，通过稀疏感知特征提取机制，将多视图图像与非结构化LiDAR深度对齐，并严格防止无效区域污染。

✨ **主要贡献**：在nuScenes数据集上，DeGuNet可即插即用于现有基线，消除架构冗余，GPU内存消耗降低高达66.5%，推理速度提升1.16倍，同时mAP绝对提升达6.20，开创了参数高效的多模态3D感知新范式。
