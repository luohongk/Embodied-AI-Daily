# SIVA-RL: Sensitivity-Invariance Visual Alignment for Multimodal Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.13931v1
- arXiv ID: 2607.13931

---

🔍 **核心问题**：现有基于可验证奖励的强化学习方法仅关注答案正确性，无法确保视觉语言模型真正依赖视觉证据进行推理，且基于干预类型分配监督的假设不成立，相同操作在不同样本中效果差异大。

🛠 **方法**：提出SIVA-RL框架，通过令牌对齐、距离约束的图内补丁交换构建局部干预，利用冻结审计策略对每对原始-干预图像评分，根据奖励下降程度分配软路由权重，实现样本级、结果条件化的敏感性与不变性对齐。

✨ **主要贡献**：在9个多模态推理基准上，SIVA-RL在3B和7B模型上全面超越匹配的强化学习基线，视觉依赖推理任务提升8.79个百分点，GRPO和DAPO配置下总体相对提升最高达14.9%。
