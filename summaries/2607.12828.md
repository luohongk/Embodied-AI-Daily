# Detecting Phishing in Ethereum Networks using Quantum Machine Learning

- arXiv: https://arxiv.org/abs/2607.12828v1
- arXiv ID: 2607.12828

---

🔍 **核心问题**：在以太坊交易网络中检测钓鱼行为，解决传统统计方法性能不足、而深度学习方法虽优但计算成本高的问题。

🛠 **方法**：提出混合量子-经典集成框架，结合量子支持向量机（QSVM）和变分量子分类器（VQC），并采用级联量子随机存取编码（QRAC）替代传统ZZ特征映射，在模拟器和IBM Heron量子处理器上验证。

✨ **主要贡献**：QSVM在集成或单独使用时均实现最低假阴性率和更高召回率；QRAC编码相比ZZ特征映射在VQC和QSVM上分别提升13%和3%的准确率；部分量子算法在IBM Heron处理器上接近模拟器性能，展示了量子机器学习在硬件限制下的潜力。
