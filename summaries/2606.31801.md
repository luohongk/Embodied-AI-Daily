# RAISE: LLM-based Automated Heuristic Design with Robust Adversary Instance Search

- arXiv: https://arxiv.org/abs/2606.31801v1
- arXiv ID: 2606.31801

---

🔍 **核心问题**：现有基于大语言模型的自动化启发式设计方法在固定训练实例集上优化，但在实际部署中因分布偏移导致性能严重下降（最多达19倍）。

🛠 **方法**：提出RAISE框架，将约束最坏情况实例搜索集成到LLM驱动的进化搜索循环中——外层用LLM算子进化启发式，内层通过无LLM的基分布参数化与边界投影，在训练实例集的ε邻域内高效识别困难实例。

✨ **主要贡献**：在在线装箱、在线作业车间调度和在线车辆路径问题的五个分布族实验中，RAISE在所有测试分布和问题规模下均保持稳定强性能，而现有方法在分布偏移时性能急剧退化。
