# Shared Selective Persistent Memory for Agentic LLM Systems

- arXiv: https://arxiv.org/abs/2607.09493v1
- arXiv ID: 2607.09493

---

🔍 **核心问题**：代理式大语言模型（LLM）系统在多轮工具交互中，每次会话从零开始，丢弃了之前会话中有效的配置、领域约束、数据模式等上下文，导致效率低下；而简单保留完整历史不仅消耗大量token，还会因无关上下文降低生成质量。

🛠 **方法**：提出共享选择性持久记忆架构，识别并保留四类可复用上下文（任务规范、数据模式、工具配置、输出约束），丢弃会话特定的推理痕迹；并通过工作区实现跨用户共享，结合基于角色的访问控制，以及零token数据刷新机制，使生成程序与运行时数据解耦。

✨ **主要贡献**：在三个企业场景中，该方法实现96%任务完成率（无记忆79%、完整历史71%）；零token刷新将重复更新任务时间减少14倍，摘要驱动生成将每次调用token成本降低97倍；在四个公开数据集上验证了泛化性，且完整历史持久化会因陈旧痕迹反而降低性能。
