# CMMR-VLN: Vision-and-Language Navigation via Continual Multimodal Memory Retrieval

- arXiv: https://arxiv.org/abs/2603.07997v1
- arXiv ID: 2603.07997

---

🔍 **核心问题**：现有基于大语言模型的视觉语言导航（VLN）方法缺乏选择性回忆和利用相关先验经验的能力，导致在长距离和陌生场景中导航性能受限。

🛠 **方法**：提出CMMR-VLN框架，通过构建由全景视觉图像和显著地标索引的多模态经验记忆库，结合检索增强生成管道模拟人类导航经验，并采用基于反思的记忆更新策略选择性存储成功路径与失败案例的关键初始错误。

✨ **主要贡献**：在仿真和真实测试中，相比NavGPT、MapGPT和DiscussNav，平均成功率分别提升52.9%、20.9%、20.9%和200%、50%、50%，展示了作为VLN基础框架的巨大潜力。
