# Robust Extended Kalman Filter for Land Navigation Using Massive Array of MEMS IMUs

- arXiv: https://arxiv.org/abs/2606.29271v1
- arXiv ID: 2606.29271

---

🔍 **核心问题**：如何利用数百个低成本MEMS惯性传感器组成阵列，在GNSS拒止环境下实现稳健的陆地导航，同时抑制突发性传感器偏差和漂移，且不增加计算负担。

🛠 **方法**：提出鲁棒惯性传感器阵列融合（RISAF）预滤波框架，在扩展卡尔曼滤波（EKF）预测步骤前，结合动态百分位门控与实时偏差跟踪，对传感器读数进行聚合与异常抑制。

✨ **主要贡献**：RISAF在仿真和实地测试中显著提升了方位角精度并减少了漂移累积，使低成本MEMS阵列的导航性能接近战术级惯性导航系统。
