# SC3-Eval: Evaluating Robot Foundation Models via Self-Consistent Video Generation

- arXiv: https://arxiv.org/abs/2606.18610v1
- arXiv ID: 2606.18610

---

🔍 **核心问题**：解决机器人基础模型在真实世界中评估成本高、速度慢、难以规模化的问题，尤其是视频生成中的累积误差、多视角不一致和分布外策略泛化挑战。

🛠 **方法**：提出SC3-Eval，通过三种自一致性约束适配预训练视频基础模型：前向-反向动力学一致性（联合预测帧与恢复动作）、跨视角一致性（多相机视图相互补全）、测试时一致性（利用反向动力学提供不确定性信号终止漂移生成）。

✨ **主要贡献**：在七种真实世界视觉-语言-动作策略上，SC3-Eval达到0.929的闭环皮尔逊相关性和0.119的MMRV，超越三种强基线，并能复现策略的失败模式，支持细粒度诊断而非仅聚合排名。
