# A neural signed configuration distance function for path planning of picking manipulators

- arXiv: https://arxiv.org/abs/2502.16205v3
- arXiv ID: 2502.16205

---

🔍 **核心问题**：针对工业拣取机械臂路径规划中碰撞检测计算量大、难以在线应用的问题，提出一种高效的多查询路径规划方法。

🛠 **方法**：提出神经符号配置距离函数（nSCDF），将障碍物隐式表示为配置空间中的无碰撞球体，并基于球体而非点构建多查询路径规划器的图结构，最终通过凸规划生成优化路径。

✨ **主要贡献**：在显著减少计算时间的同时，生成的路径质量接近渐近最优路径规划器，为在线路径规划提供了高效解决方案。
