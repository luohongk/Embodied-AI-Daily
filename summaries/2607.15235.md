# Adaptive Sampling for Spatiotemporal Anomaly Monitoring in Wireless Sensor Networks

- arXiv: https://arxiv.org/abs/2607.15235v1
- arXiv ID: 2607.15235

---

🔍 **核心问题**：无线传感器网络在长期环境监测中采用稀疏采样以延长寿命，但容易错过短暂、局部的异常事件。

🛠 **方法**：提出哨兵辅助自适应采样框架，正常时期基于卡尔曼滤波预测不确定性进行稀疏采样，异常时期由连续采样的哨兵节点执行混合广义似然比检测，并触发一跳邻域唤醒与恢复感知警报控制。

✨ **主要贡献**：在Intel Berkeley实验室温度数据集上，将异常窗口采样率从0.439提升至0.933，相比AAS和Adapted e-Sampling方法分别降低总成本15.4%和2.1%，实现了异常可见性与采样成本的有效平衡。
