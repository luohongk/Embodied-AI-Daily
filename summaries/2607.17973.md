# SAGE: Subgoal-Conditioned Action Generation for Latent World Model Planning

- arXiv: https://arxiv.org/abs/2607.17973v1
- arXiv ID: 2607.17973

---

🔍 **核心问题**：在潜在世界模型规划中，随着规划时间跨度增长，固定候选动作预算难以在高维动作空间中搜索到高质量的未来动作序列，导致长时域规划性能下降。

🛠 **方法**：提出一种基于先验条件的规划器，通过目标条件生成器预测可到达的潜在子目标，并以此指导候选动作序列的生成，同时利用不同时间跨度的子目标作为先验，平衡局部控制与长期进展。

✨ **主要贡献**：在PushT和OGBench Cube任务上，该方法显著提升了长时域规划的成功率（如目标偏移150时，PushT成功率从12.7%提升至64.7%，OGBench Cube从26.7%提升至67.3%），同时保持了良好的短时域性能。
