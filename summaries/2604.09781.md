# Text-Guided 6D Object Pose Rearrangement via Closed-Loop VLM Agents

- arXiv: https://arxiv.org/abs/2604.09781v2
- arXiv ID: 2604.09781

---

🔍 **核心问题**：解决视觉-语言模型（VLM）在3D场景中难以根据文本指令推断目标物体6D姿态的问题。

🛠 **方法**：通过闭环VLM代理系统，结合RGB-D图像或3D网格场景，循环执行“观察-评估-更新-渲染”步骤，并引入多视角推理、物体中心坐标系可视化和单轴旋转预测三种推理技术。

✨ **主要贡献**：无需微调或新增模块，即可显著提升VLM在文本引导的6D姿态预测精度，且兼容开源与闭源模型；结合简单机器人运动规划后，性能优于最新的视觉-语言-动作模型（VLA）。
