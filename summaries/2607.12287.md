# Reducing Temporal Redundancy for Efficient Vision-Language-Action Inference

- arXiv: https://arxiv.org/abs/2607.12287v1
- arXiv ID: 2607.12287

---

🔍 **核心问题**：视觉-语言-动作（VLA）模型在机器人操作中泛化能力强，但高推理延迟限制了其实时部署，主要源于连续帧的重复视觉编码和扩散策略的多步迭代采样。

🛠 **方法**：提出系统级加速策略，在感知端仅增量更新动态场景区域的令牌而非重新编码整帧，在动作端通过效率导向训练将扩散采样压缩为紧凑的2步调度，同时保持动作精度。

✨ **主要贡献**：在Libero、RobotWin和真实机器人平台上实现超过2倍加速，同时保持高性能，在通用操作基准上成功率高达98%。
