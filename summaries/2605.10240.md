# MARGIN: Margin-Aware Regularized Geometry for Imbalanced Vulnerability Detection

- arXiv: https://arxiv.org/abs/2605.10240v3
- arXiv ID: 2605.10240

---

🔍 **核心问题**：现实软件漏洞检测中，数据存在频率不平衡和难度不平衡，导致超球面表示空间出现几何畸变，影响模型性能。

🛠 **方法**：提出MARGIN框架，通过自适应边界度量学习和超球面原型建模，利用von Mises-Fisher浓度估计分布结构，动态调整几何正则化以对齐嵌入分布与Voronoi单元。

✨ **主要贡献**：在公开漏洞数据集上显著优于强基线方法，尤其在困难不平衡数据上提升分类与检测性能，同时生成更结构化的嵌入几何，增强鲁棒性、可解释性和泛化能力。
