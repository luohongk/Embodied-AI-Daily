# FlowRL: A Taxonomy and Modular Framework for Reinforcement Learning with Diffusion Policies

- arXiv: https://arxiv.org/abs/2603.27450v2
- arXiv ID: 2603.27450

---

🔍 **核心问题**：扩散模型和流模型作为策略表示具有灵活性，但缺乏显式对数概率，导致基于策略梯度的强化学习效率低下，且现有方法缺乏统一视角。

🛠 **方法**：提出一种针对扩散/流策略的强化学习算法分类体系（FlowRL），并构建基于JAX的模块化开源代码库，利用JIT编译实现高吞吐量训练。

✨ **主要贡献**：首次系统化分类扩散策略的强化学习方法，提供跨Gym-Locomotion、DeepMind Control Suite和IsaacLab的标准化基准测试，为算法选择提供实践指南，并开源高效工具包。
