# Learn2Decompose: Learning Problem Decomposition for Efficient Sequential Multi-object Manipulation Planning

- arXiv: https://arxiv.org/abs/2408.06843v5
- arXiv ID: 2408.06843

---

🔍 **核心问题**：在动态环境中进行顺序多物体操作时，传统任务与运动规划（TAMP）方法的规划时间会随物体数量和规划步长指数增长，难以满足实时应用需求。

🛠 **方法**：提出Learn2Decompose框架，通过从演示中学习问题分解来加速TAMP求解，包含三个关键组件：目标分解学习（识别子目标序列）、计算距离学习（预测状态间计算复杂度）和物体缩减（最小化重规划中考虑的活跃物体集）。

✨ **主要贡献**：在三个基准测试上验证了该方法能显著提升动态环境中顺序多物体操作任务的重规划效率，有效克服了传统TAMP的指数级时间增长瓶颈。
