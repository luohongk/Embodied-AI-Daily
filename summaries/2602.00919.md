# Green-VLA: Staged Vision-Language-Action Model for Generalist Robots

- arXiv: https://arxiv.org/abs/2602.00919v2
- arXiv ID: 2602.00919

---

🔍 **核心问题**：如何构建一个既能部署在特定人形机器人（Green）上，又能泛化到多种机器人形态的通用视觉-语言-动作（VLA）模型。

🛠 **方法**：提出Green-VLA框架，采用五阶段训练课程（从基础VLM到强化学习策略对齐），结合可扩展的数据处理流水线（3000小时演示数据）、时间对齐与质量过滤，以及统一的、感知机器人形态的动作接口。

✨ **主要贡献**：实现了跨多种机器人形态（人形、移动操作臂、固定基座臂）的单一策略控制；通过强化学习对齐显著提升成功率、鲁棒性和长程任务效率；在仿真和真实机器人实验中验证了强泛化能力。
