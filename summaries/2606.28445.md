# LoRA-Tuned Large Language Models for Dementia Detection via Multi-View Speech-Derived Features

- arXiv: https://arxiv.org/abs/2606.28445v1
- arXiv ID: 2606.28445

---

🔍 **核心问题**：早期检测痴呆症，利用自发性语音作为非侵入性筛查手段，解决传统方法仅关注单一表征维度（如声学特征、ASR文本等）而缺乏对异质性认知症状的整合推理问题。

🛠 **方法**：提出基于低秩适配（LoRA）微调的大语言模型（LLM），通过统一提示编码四种互补的语音衍生信号（含停顿标记的ASR文本、话语级主题线索、时间流畅性统计和音韵序列），实现结构化多视角推理，无需专用编码器或后期融合。

✨ **主要贡献**：在ADReSSo数据集上达到90.14%的F1分数，消融实验证实每种视角的互补贡献，展示了单一LLM通过多视图提示学习连贯决策函数的有效性。
