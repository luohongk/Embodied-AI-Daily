# Hierarchical Chain-of-Thought: Enhancing LLM Reasoning Performance and Efficiency

- arXiv: https://arxiv.org/abs/2604.00130v2
- arXiv ID: 2604.00130

---

🔍 **核心问题**：传统链式思维（CoT）提示方法在处理复杂多步推理时存在冗余和逻辑连贯性不足的问题。

🛠 **方法**：提出分层链式思维（Hi-CoT），通过交替进行指令规划与逐步执行，将推理过程分解为层次化子步骤。

✨ **主要贡献**：在多种大语言模型和数学推理基准上，Hi-CoT平均准确率提升6.2%（最高达61.4%），同时推理轨迹长度减少13.9%，且严格遵循层次结构时效果最优。
