# NASimJax: GPU-Accelerated Policy Learning Framework for Penetration Testing

- arXiv: https://arxiv.org/abs/2603.19864v1
- arXiv ID: 2603.19864

---

🔍 **核心问题**：现有渗透测试模拟器速度太慢，无法在真实规模的网络场景上高效训练强化学习策略，导致策略泛化能力差。

🛠 **方法**：基于JAX框架重新实现网络攻击模拟器NASim，实现整个训练流程在硬件加速器上运行，并提出上下文POMDP建模、网络生成流水线、两阶段动作分解（2SAS）等技术。

✨ **主要贡献**：环境吞吐量提升高达100倍，支持多达40台主机的网络实验；发现优先级别回放结合稀疏拓扑训练能提升零样本泛化能力；提出的2SAS方法在动作空间线性增长时显著优于传统掩码方法。
