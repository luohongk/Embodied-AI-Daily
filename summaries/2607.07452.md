# GeoGS-SLAM: Geometry-Only Gaussian Splatting for Dense Monocular SLAM

- arXiv: https://arxiv.org/abs/2607.07452v1
- arXiv ID: 2607.07452

---

🔍 **核心问题**：现有基于3D高斯泼溅（3DGS）的密集SLAM方法同时建模外观和几何，但下游机器人任务（如导航）更依赖精确的几何结构，而非逼真渲染。因此，本文探索是否可以在不进行外观建模的情况下，仅用3DGS实现几何重建。

🛠 **方法**：提出GeoGS-SLAM，仅保留高斯原语的空间参数（减少80%以上参数），通过单视图和多视图的几何与光度监督优化原语，并采用局部平面驱动的初始化加速几何收敛。同时，引入针对闭环的全局地图更新策略，避免因视角不一致的位姿修正导致地图撕裂。

✨ **主要贡献**：在合成和真实基准上，该方法在在线建图效率和几何重建质量上均超越现有最先进方法，显著减少了高斯原语数量，提升了光照变化的鲁棒性，并有效解决了闭环中的地图一致性问题。
