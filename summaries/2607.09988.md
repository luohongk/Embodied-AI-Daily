# An LLM-powered Agentic Recommendation System for Connected TV Content Discovery

- arXiv: https://arxiv.org/abs/2607.09988v2
- arXiv ID: 2607.09988

---

🔍 **核心问题**：传统推荐系统难以自然整合非结构化或异构的上下文信号（如热点话题、文化事件等），需要大量人工特征工程和数据管道设计。

🛠 **方法**：提出基于LLM的智能体推荐系统，采用混合架构，由LLM负责推理与整合多样化信号，同时将检索、个性化等子任务交由传统机器学习模型处理，以平衡灵活性与性能。

✨ **主要贡献**：通过工程化方法成功克服LLM在推荐系统中的实际限制（尤其是推理延迟），并分享了构建混合系统的经验、权衡与教训。
