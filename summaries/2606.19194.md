# Invertible Neural Network Adapter for One-Step Flow Matching in Robot Manipulation

- arXiv: https://arxiv.org/abs/2606.19194v1
- arXiv ID: 2606.19194

---

🔍 **核心问题**：现有机器人操作策略依赖多步迭代的流匹配过程，导致推理延迟高，难以满足实时控制需求。

🛠 **方法**：提出一种可逆神经网络适配器，基于流匹配公式将动作生成轨迹约束在可逆潜空间中，通过单步去噪过程生成高维动作。

✨ **主要贡献**：将推理延迟从110毫秒降至61毫秒，在仿真和真实机器人实验中保持或达到最先进的任务性能，显著提升视觉-语言-动作模型的推理效率。
