# Securing Autonomous Vehicle Systems via Twin-Aware Federated Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.08137v1
- arXiv ID: 2607.08137

---

🔍 **核心问题**：联邦强化学习（FRL）在自动驾驶等安全关键系统中易受投毒攻击，恶意参数注入会破坏全局控制模型，而现有研究对此威胁的防御不足。

🛠 **方法**：提出防御框架SecAgg，结合数字孪生进行排练式学习，并利用历史聚合参数与选定中心梯度，确保仅聚合良性数据，从而抑制恶意智能体的影响。

✨ **主要贡献**：为FRL系统在投毒攻击下的收敛性能提供理论保证，并通过模拟真实高速公路环境的数字孪生实验，验证了该方法在对抗条件下对自动驾驶车辆控制的有效性。
