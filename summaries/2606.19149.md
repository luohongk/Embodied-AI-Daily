# OpenAnt: LLM-Powered Vulnerability Discovery Through Code Decomposition, Adversarial Verification, and Dynamic Testing

- arXiv: https://arxiv.org/abs/2606.19149v1
- arXiv ID: 2606.19149

---

🔍 **核心问题**：大型代码库的自动化漏洞发现面临高误报率（静态分析）和高基础设施成本（动态测试如模糊测试）的挑战，现有方法难以兼顾可扩展性与准确性。

🛠 **方法**：提出OpenAnt系统，采用多阶段流水线，结合静态程序分析与大语言模型（LLM）推理。首先通过代码分解和可达性过滤将分析范围缩小97%，然后利用对抗性验证模拟攻击者能力评估漏洞可利用性，最后通过自动生成沙盒环境进行动态测试验证。

✨ **主要贡献**：在OpenSSL、WordPress等真实项目中成功发现未知漏洞，显著降低误报率并保持可控分析成本；提出闭环漏洞发现流程（语义推理+漏洞验证），为可扩展的自动化安全分析提供实用路径；系统以Apache 2.0协议开源。
