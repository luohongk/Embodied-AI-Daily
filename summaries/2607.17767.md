# VLN-AVP: Zero-Shot Vision-Language Navigation with Hybrid Long-Short-Term Memory for Autonomous Valet Parking

- arXiv: https://arxiv.org/abs/2607.17767v1
- arXiv ID: 2607.17767

---

🔍 **核心问题**：现有自动代客泊车（AVP）方法依赖预建地图，难以适应未知环境和开放词汇目标，缺乏零样本导航能力。

🛠 **方法**：提出VLN-AVP框架，结合鸟瞰图（BEV）模型的精确空间感知与视觉语言模型（VLM）的通用智能，并引入混合长短期记忆系统（短期感知记忆追踪语义视觉线索，长期拓扑记忆从过往经验中学习稳定策略）。

✨ **主要贡献**：首次实现零样本AVP导航，无需预建地图；构建了包含10个高保真停车场场景和超1000个导航片段的VLN-AVP数据集（首个地下停车场VLN基准）；在仿真中成功率比VLN方法提升超25%，比自动驾驶方法提升超15%，并在真实车辆实验中验证了实用性。
