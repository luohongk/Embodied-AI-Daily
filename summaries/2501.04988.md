# Intelligent Sailing Model for Open Sea Navigation

- arXiv: https://arxiv.org/abs/2501.04988v2
- arXiv ID: 2501.04988

---

🔍 **核心问题**：当前缺乏标准化仿真环境来测试自主航行船舶在开放海域的交互行为，尤其是如何模拟符合国际海上避碰规则（COLREGS）的智能船舶行为。

🛠 **方法**：提出首个智能航行模型（ISM），通过实时监测适用规则、生成合规航路点，并采用模型预测控制（MPC）跟踪航路点，实现船舶在遵守规则的同时完成路径规划任务。

✨ **主要贡献**：首次实现可扩展、规则合规的多类型船舶交互仿真，在4049个关键场景测试中无碰撞发生，且目标到达率约达97%，验证了模型在纯ISM船舶及混合交通环境中的有效性。
