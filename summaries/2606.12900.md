# Zero-source LLM Hallucination Detection with Human-like Criteria Probing

- arXiv: https://arxiv.org/abs/2606.12900v1
- arXiv ID: 2606.12900

---

🔍 **核心问题**：在零源约束下（无模型内部信息或外部参考），仅凭文本问答对检测大语言模型生成的幻觉内容。  
🛠 **方法**：提出HCPD范式，通过类人标准探测（HCP）机制让LLM自适应分解判断为加权可解释标准，并聚合得分评估真实性；采用基于语义一致性的弱监督奖励对齐训练，推理时结合多采样聚合策略。  
✨ **主要贡献**：首次实现零源场景下完全可解释的幻觉检测，理论分析保障可靠性，实验表明HCPD在多个基准上持续超越现有最优方法。
