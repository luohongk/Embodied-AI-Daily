# CoFL-S: Spatially Queryable Sector Flow Fields for Local Language-Conditioned Navigation

- arXiv: https://arxiv.org/abs/2607.02222v1
- arXiv ID: 2607.02222

---

🔍 **核心问题**：现有视觉语言导航（VLN）研究多聚焦于高层指令推理与全局地图构建，而低层动作表示（如连续轨迹生成）仍未被充分探索。  
🛠 **方法**：提出CoFL-S框架，通过预测机器人局部可见区域的**语言条件流场**（sector flow fields），并滚动展开该流场生成连续轨迹；同时将完整导航片段转换为帧级局部监督，包含子指令、动作、轨迹和密集流场目标。  
✨ **主要贡献**：在连续时间Habitat基准测试中，CoFL-S在不同规划频率下均优于动作令牌和动作块基线；零样本真实世界闭环部署进一步验证了其超越仿真的实际优势。
