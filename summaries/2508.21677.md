# Robust Convex Model Predictive Control with collision avoidance guarantees for robot manipulators

- arXiv: https://arxiv.org/abs/2508.21677v3
- arXiv ID: 2508.21677

---

🔍 **核心问题**：工业机械臂在杂乱环境中运动时，模型不确定性会导致安全运动规划困难，传统方法需降低速度来保证安全，影响效率。

🛠 **方法**：提出一种鲁棒管式模型预测控制（Tube MPC）与走廊规划算法相结合的方法，将问题转化为凸优化形式，实现快速求解。

✨ **主要贡献**：在6自由度工业机械臂仿真中，该方法能容忍更高的模型不确定性，同时实现更快的安全运动，优于现有基准方法。
