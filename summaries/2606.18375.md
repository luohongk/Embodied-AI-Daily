# PAIWorld: A 3D-Consistent World Foundation Model for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.18375v1
- arXiv ID: 2606.18375

---

🔍 **核心问题**：现有世界基础模型（WFMs）在机器人操作中缺乏多视图3D一致性，导致跨视图物体漂移、深度不一致和纹理错位，无法满足多摄像头策略学习的需求。

🛠 **方法**：提出PAIWorld框架，在扩散-Transformer世界模型中引入三个核心组件：几何感知跨视图注意力模块（显式建立视图间通信）、几何旋转位置编码（将相机射线方向和外部姿态编码到注意力机制）、以及潜在3D-REPA（从冻结的3D基础模型中蒸馏3D感知特征）。

✨ **主要贡献**：在机器人操作基准上实现最先进的多视图3D一致性，在WorldArena排行榜上排名第一，AgiBot-Challenge2026排行榜上排名第二，并支持基于模型的规划、世界动作模型和多视图策略后训练等下游应用。
