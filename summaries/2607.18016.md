# Closing the Loop in Humanoid VLA: Persistent 3D Object Tokens for Verifiable Loco-Manipulation

- arXiv: https://arxiv.org/abs/2607.18016v1
- arXiv ID: 2607.18016

---

🔍 **核心问题**：人形机器人在长时间操作任务中，用于生成动作的物体状态与用于验证动作是否成功的物体状态可能不一致，导致任务执行不可靠。

🛠 **方法**：提出持久化物体标记化（POT），从RGB-D观测中维护角色索引的3D物体记录，并将其转化为物体标记，用于条件化全身动作生成，同时支持几何谓词检查，形成闭环执行系统。

✨ **主要贡献**：在Unitree G1机器人上，POT-VLA将匹配基线GR00T-N1.7的成功率从39/80提升至71/80；在外部基准测试中，以44/50的成绩超过Being-0报告的37/50。关键创新在于将物体状态同时用于动作生成与验证，显著提升了需要维持3D关系的任务表现。
