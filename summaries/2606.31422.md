# Ask the World Before Acting: Budgeted Environment Probing for World-Model Calibration

- arXiv: https://arxiv.org/abs/2606.31422v1
- arXiv ID: 2606.31422

---

🔍 **核心问题**：长时程语言智能体在决策过程中，其内部世界模型可能发生漂移，导致后续行动失败，而现有方法未能在行动前有效校准模型。

🛠 **方法**：提出一种预算受限的环境探测方法（\method），允许智能体在行动前有选择地向环境询问特定信念字段，并将反馈写回世界模型，从而将环境交互视为稀缺的校准资源。

✨ **主要贡献**：通过类型分层分析形式化了探测-行动边界，实验表明，遵循任务结构的探测策略能显著降低终端世界模型误差，并揭示了程序性信念（如工具依赖）与空间信念（如物体位置）的不同修复特性。
