# Sensoformer: Robust Sim-to-Real Inference on Variable-Geometry Sensor Sets via Physics-Structured Randomization

- arXiv: https://arxiv.org/abs/2601.06320v3
- arXiv ID: 2601.06320

---

🔍 **核心问题**：解决从稀疏、不规则且数量可变的传感器阵列中推断高维物理状态时，因传感器几何结构多变及物理异质性导致的模拟到现实分布偏移问题。

🛠 **方法**：提出Sensoformer框架，结合集合注意力机制与物理结构化域随机化（PSDR），通过随机化底层物理动力学（如传播介质、极端噪声、网络可用性丢失）而非视觉特征，强制学习域不变的物理算子。

✨ **主要贡献**：在地震源反演真实场景中，Sensoformer达到最先进精度，优于消息传递神经网络和神经算子；其注意力机制能自主发现最优实验设计原则，动态优先选择稀疏正交传感器以克服信息瓶颈。
