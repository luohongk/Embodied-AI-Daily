# MSPL: Multi-Step Pseudo-Labeling for Open-Vocabulary Object Detection

- arXiv: https://arxiv.org/abs/2510.14792v4
- arXiv ID: 2510.14792

---

🔍 **核心问题**：开放词汇目标检测（OVD）中，现有方法依赖单步图像-文本匹配生成伪标签，无法处理拥挤、遮挡等复杂语义场景的中间推理步骤。

🛠 **方法**：提出MSPL框架，将多步视觉推理引入伪标签生成过程，分解为三个可解释步骤——目标定位、类别识别和背景理解，利用中间推理状态作为丰富监督源。

✨ **主要贡献**：在OV-COCO数据集上，新类AP50提升9.4；在OV-LVIS数据集上，框和掩码APr分别提升3.2和2.2，达到最先进性能，且伪标签效率更高。
