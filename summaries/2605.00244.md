# Lucid-XR: An Extended-Reality Data Engine for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2605.00244v1
- arXiv ID: 2605.00244

---

🔍 **核心问题**：解决机器人操作任务中真实世界多模态数据获取成本高、多样性不足的问题，并实现从合成数据到真实环境的零样本迁移。

🛠 **方法**：提出Lucid-XR数据引擎，包含基于XR头显的Web物理仿真环境vuer，集成设备端物理模拟与人-机器人姿态重定向，并通过自然语言驱动的物理引导视频生成管线增强数据。

✨ **主要贡献**：首次实现完全基于合成数据训练的机器人视觉策略零样本迁移至未见过的杂乱、弱光真实环境，涵盖软材料、松散颗粒和刚体接触等灵巧操作任务。
