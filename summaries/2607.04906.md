# RL-Ballast: Ship Ballast Water Path Planning and Clog Prediction via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.04906v1
- arXiv ID: 2607.04906

---

🔍 **核心问题**：解决船舶压载水系统中因阀门故障或管道堵塞导致的液压异常问题，传统规则或手动方法适应性差且依赖密集传感器诊断。

🛠 **方法**：提出RL-Ballast框架，基于图论和深度优先搜索将阀门排列问题转化为54条可行路径，利用帧堆叠的液位和动作结果近似部分可观测环境，结合深度强化学习实现自适应路径规划，并通过失败动作记忆和动态掩码避免重复无效操作。

✨ **主要贡献**：在蒙特卡洛模拟中，RL-Ballast能应对所有单堵塞场景，平均决策步数从61.0降至41.5；故障历史评分方案在堵塞不可区分条件下达到100% Top-3命中率、66.7%严格Top-1命中率及83.3% Top-1平局命中率，实现低传感器依赖下的自适应重路由和堵塞诊断。
