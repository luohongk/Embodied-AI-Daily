# Scaling Short-Term Memory of Visuomotor Policies for Long-Horizon Tasks

- arXiv: https://arxiv.org/abs/2606.16178v1
- arXiv ID: 2606.16178

---

🔍 **核心问题**：现有视触觉策略在模仿学习中仅依赖即时感官输入，缺乏利用短期记忆来指导长时程任务决策的能力。

🛠 **方法**：提出PRISM架构，基于Transformer设计，包含门控注意力机制（过滤历史信息中的无关细节）和分层架构（先压缩局部信息为紧凑令牌，再整合捕捉长程依赖），以高效扩展短期记忆至两分钟。

✨ **主要贡献**：在ReMemBench基准（含8项家务任务、4类短期记忆）上，PRISM相比最强基线提升5%-12%，在RoboCasa和LIBERO上比无记忆变体及微调VLA模型（如GR00T-N1-3B、OpenVLA）提升11%-15%，且无需大规模预训练，为长时程任务中的记忆增强视触觉策略提供了评估基础。
