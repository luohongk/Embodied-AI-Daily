# LLM-Empowered Multimodal Fusion Framework for Autonomous Driving: Semantic Enhancement and Channel-Adaptive Design

- arXiv: https://arxiv.org/abs/2607.01772v1
- arXiv ID: 2607.01772

---

🔍 **核心问题**：解决自动驾驶中视觉-雷达融合因遮挡、恶劣天气和信道噪声导致的输入质量动态变化问题，实现鲁棒的感知融合。

🛠 **方法**：提出LM-SCIP框架，以大型语言模型（LLM）为核心推理中枢，结合信道自适应语义模块（CASM）将链路指标映射为“信道提示”动态调控雷达特征，并通过LoRA微调LLM与异构混合专家模型（H-MoE）协同处理视觉与雷达信息。

✨ **主要贡献**：在nuScenes数据集上，相比纯视觉基线，定位均方根误差降低40.0%；在VIRAT数据集上实现0.214m定位误差和0.179m最小最终位移误差（k=1），验证了低信噪比下视觉主导回退与高信噪比下协同融合的鲁棒性。
