# APVI-SLAM: Real-Time Acoustic-Pressure-Visual-Inertial Localization and Photorealistic Mapping System in Complex Underwater Environment

- arXiv: https://arxiv.org/abs/2607.06222v1
- arXiv ID: 2607.06222

---

🔍 **核心问题**：水下视觉-惯性SLAM在极端环境中常因特征退化导致定位失败，现有方法在视觉失效时难以实现鲁棒的多传感器融合。

🛠 **方法**：提出APVI-SLAM系统，包含可靠性感知的定位框架（动态重加权传感器估计器+滑动窗口冻结策略）和高效四叉树引导的建图模块（增量式水介质建模与3D高斯优化）。

✨ **主要贡献**：在复杂水下环境中实现实时高精度定位与逼真场景重建，并贡献了首个多模态珊瑚礁测绘数据集，在公开和自建基准上达到最先进的定位与重建质量。
