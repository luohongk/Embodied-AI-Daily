# Joint Reward Modeling: Internalizing Chain-of-Thought for Efficient Visual Reward Models

- arXiv: https://arxiv.org/abs/2602.07533v3
- arXiv ID: 2602.07533

---

🔍 **核心问题**：现有奖励模型在图像编辑等复杂任务中难以兼顾语义理解与推理效率——判别式模型缺乏推理监督，生成式模型推理成本高且难以对齐人类偏好。

🛠 **方法**：提出联合奖励建模（JRM），在共享视觉-语言骨干网络上联合优化偏好学习与语言建模，将生成式模型的语义推理能力内化到高效的判别式表示中。

✨ **主要贡献**：在MMRB2和EditReward-Bench上取得最优结果，并在下游在线强化学习中显著提升稳定性与性能，有效弥合了奖励模型中效率与语义理解之间的鸿沟。
