# EventVLA: Event-Driven Visual Evidence Memory for Long-Horizon Vision-Language-Action Policies

- arXiv: https://arxiv.org/abs/2606.20092v1
- arXiv ID: 2606.20092

---

🔍 **核心问题**：解决长时程机器人操作中，标准视觉-语言-动作策略因任务关键线索随时间被遮挡或不可观测而导致的记忆瓶颈问题。

🛠 **方法**：提出EventVLA端到端框架，包含基础视觉锚点（保留初始和短期上下文）和动态关键帧证据记忆模块（KEM），后者通过从VLA潜在嵌入中预测未来关键帧概率，自主捕获并存储稀疏的任务关键视觉事件。

✨ **主要贡献**：在17个模拟任务和4个真实双臂任务中，EventVLA平均成功率比现有最优记忆增强VLA提升40%，并提出了专门评估非马尔可夫操作任务的诊断基准RoboTwin-MeM。
