# Semantic Pareto-DQN: A Multi-Objective Reinforcement Learning Framework for Financial Anomaly Detection

- arXiv: https://arxiv.org/abs/2607.09641v1
- arXiv ID: 2607.09641

---

🔍 **核心问题**：金融异常检测面临严重的类别不平衡问题，传统单目标算法容易陷入“欺诈崩溃”，即默认预测为多数类，无法在拦截异常与减少客户摩擦之间取得平衡。

🛠 **方法**：提出语义Pareto-DQN多目标强化学习框架，将异构交易特征合成为连贯的自然语言描述，由大语言模型编码为鲁棒的状态表示；同时优化包含金融效能、操作摩擦和语义发现三个维度的向量化奖励，并通过连续Pareto前沿动态权衡漏报与误报成本。

✨ **主要贡献**：在电商欺诈和UCI信用数据集上，该方法成功突破了零召回陷阱，相比标量化基线显著提升了少数类召回率，为在有限操作摩擦下实现金融异常发现提供了新途径。
