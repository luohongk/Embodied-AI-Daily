# GaP: A Graph-as-Policy Multi-Agent Self-Learning Harness For Variational Automation Tasks

- arXiv: https://arxiv.org/abs/2607.05369v1
- arXiv ID: 2607.05369

---

🔍 **核心问题**：针对工业与商业应用中“变体自动化”任务（物体几何与姿态变化较大）的可靠性不足问题，现有无模型策略难以稳定执行。

🛠 **方法**：提出GaP框架，基于模块化开放机器人技能库生成有向计算图（含感知、规划、控制节点），并通过内部仿真环境并行排练不同图结构，迭代优化图结构与参数。

✨ **主要贡献**：在8个新变体自动化任务基准（4个仿真+4个真实场景）中，GaP的成功率显著优于基线方法，实现了可解释编程与开放世界适应性的结合。
