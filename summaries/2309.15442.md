# Template Model Inspired Task Space Learning for Robust Bipedal Locomotion

- arXiv: https://arxiv.org/abs/2309.15442v1
- arXiv ID: 2309.15442

---

🔍 **核心问题**：解决双足机器人在复杂环境下鲁棒行走的难题，传统端到端学习方法数据效率低且泛化性差，而纯模型方法难以适应多变条件。

🛠 **方法**：提出分层框架，上层采用强化学习策略在线生成任务空间指令，下层使用基于模型的控制器跟踪轨迹；策略设计借鉴角动量线性倒立摆模型，优化马尔可夫决策过程的观测与动作空间。

✨ **主要贡献**：通过低维状态与任务空间输出的巧妙映射，显著提升数据效率和鲁棒性；框架与下层控制器解耦，可泛化至不同双足机器人（如Rabbit、Walker2D、Digit），在对抗条件下仍能保持稳定行走并自然模仿人类步态。
