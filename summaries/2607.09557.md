# CORAL-AUV: CFD Oriented Reinforcement Learning for Autonomous Underwater Vehicles

- arXiv: https://arxiv.org/abs/2607.09557v1
- arXiv ID: 2607.09557

---

🔍 **核心问题**：解决自主水下航行器（AUV）传统控制方法劳动密集、对配置或环境变化鲁棒性差，以及强化学习因仿真物理模型不精确（尤其是阻力模型）导致模拟到现实差距大的问题。

🛠 **方法**：利用计算流体动力学（CFD）数据训练替代阻力模型（SDM），将其集成到强化学习管道中实现快速推理，并采用域随机化（DR）增强策略鲁棒性。

✨ **主要贡献**：首次在六自由度AUV上成功部署零样本强化学习策略，基于SDM的控制器相比简化物理模型控制器，能耗降低31%，航点间穿越速度提升11%，误差减少19%，且对奖励函数设计更鲁棒，在域随机化任务中唯一成功迁移。
