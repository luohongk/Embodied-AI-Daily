# One-Step Gradient Delay is Not a Barrier for Large-Scale Asynchronous Pipeline Parallel LLM Pretraining

- arXiv: https://arxiv.org/abs/2606.30634v1
- arXiv ID: 2606.30634

---

🔍 **核心问题**：大规模LLM预训练中，异步流水线并行虽能消除同步训练中的GPU空闲气泡，但梯度延迟（尤其是一步延迟）被认为会导致训练不稳定，限制了PipeDream-2BW等异步调度方案的实际应用。

🛠 **方法**：通过实验分析发现，一步梯度延迟的负面影响高度依赖优化器选择（AdamW严重退化，而Muon表现鲁棒），并引入一种与优化器无关的误差反馈（Error Feedback）修正机制来进一步缓解延迟效应，同时提供理论收敛性分析。

✨ **主要贡献**：首次全面实证表明一步延迟并非固有障碍，通过优化器选择与误差反馈修正可弥合与同步训练的性能差距；在高达100亿参数的模型上验证了异步流水线并行在大规模场景下的实用潜力。
