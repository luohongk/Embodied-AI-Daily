# Beyond Text-to-SQL: An Agentic LLM System for Governed Enterprise Analytics APIs

- arXiv: https://arxiv.org/abs/2605.21027v2
- arXiv ID: 2605.21027

---

🔍 **核心问题**：传统Text-to-SQL系统在企业场景中无法直接访问原始数据库，而企业分析依赖受治理的API来确保一致性、可审计性和安全性，但将数学或聚合逻辑交给LLM会带来可靠性与合规风险。

🛠 **方法**：提出Analytic Agent，一个基于LLM的智能体系统，通过多步推理和策略感知编排，将自然语言意图转化为与受治理企业分析API的安全交互，包括验证权限、执行查询并生成合规可视化。

✨ **主要贡献**：在90个由领域专家构建的真实企业用例上验证，该系统能可靠解释用户目标、执行受治理查询并生成合规可视化，解决了企业分析中自然语言接口与API治理之间的关键矛盾。
