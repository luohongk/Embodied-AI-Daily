# Comprehensive Robustness Analysis of LiDAR-based 3D Object Detection in Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.02074v1
- arXiv ID: 2607.02074

---

🔍 **核心问题**：现有基于LiDAR的3D目标检测模型在基准数据集上精度提升，但缺乏对其对抗鲁棒性的系统评估，且现有评估框架仅依赖mAP，忽略了结构性和预测性因素。

🛠 **方法**：提出一个综合评估框架，利用点云密度、点云定位两个结构性因素，以及误分类、定位误差、自车距离三个预测性因素，对近期和经典的LiDAR模型进行对抗攻击实验和关键分析。

✨ **主要贡献**：发现高容量体素检测器比柱状检测器更易受结构化坐标扰动影响，非锚点检测器鲁棒性差，且近期模型与早期模型同样脆弱，呼吁改进评估基准以同时考虑检测精度和对抗鲁棒性。
