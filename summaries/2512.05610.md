# NormalView: tree species classification from backpack and aerial lidar data using geometric projections

- arXiv: https://arxiv.org/abs/2512.05610v2
- arXiv ID: 2512.05610

---

🔍 **核心问题**：如何利用激光雷达点云数据（包括移动激光扫描MLS和机载激光扫描ALS）实现高精度的树种分类。

🛠 **方法**：提出NormalView方法，将点云的局部几何信息（法向量估计）嵌入二维投影中，并输入图像分类网络YOLOv11进行树种识别；同时探究多光谱辐射强度信息对分类性能的影响。

✨ **主要贡献**：在MLS数据上达到95.5%的总体准确率（宏平均94.8%），在ALS数据上达到91.8%（宏平均79.1%）；发现多通道强度信息可提升分类效果；公开了包含1915个样本的MLS数据集。
