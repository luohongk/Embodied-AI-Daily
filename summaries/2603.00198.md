# Stateful Token Reduction for Long-Video Hybrid VLMs

- arXiv: https://arxiv.org/abs/2603.00198v2
- arXiv ID: 2603.00198

---

🔍 **核心问题**：针对混合Mamba-Transformer架构的长视频视觉语言模型（VLM），现有令牌缩减方法仅适用于纯Transformer模型，未能有效处理Mamba层的状态依赖特性，导致早期激进剪枝不可靠。

🛠 **方法**：提出一种混合感知的令牌缩减框架，采用渐进式低到高调度策略，并设计统一查询条件重要性评分，同时针对Mamba层排除位置依赖衰减以增强选择信号。

✨ **主要贡献**：在25%令牌预算下实现3.8-4.2倍预填充加速，且保持接近基线的精度；混合模型在激进缩减下同时提升效率与精度，优于纯Transformer的折衷表现；在相同混合骨干网络上超越先前基线，并能与视觉冗余缩减方法有效结合。
