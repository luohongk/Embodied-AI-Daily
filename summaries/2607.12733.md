# LLMs Can See the Smoke but not the Fire: Evaluating Abductive Reasoning with Elenchos

- arXiv: https://arxiv.org/abs/2607.12733v1
- arXiv ID: 2607.12733

---

🔍 **核心问题**：评估大型语言模型（LLMs）在反事实推理（溯因推理）中的能力，即从观察到的行为差异推断隐藏的规则变化。  
🛠 **方法**：提出Elenchos框架，基于形式系统（如λ演算）设计生成式评估任务，要求模型判断系统是否被修改并识别具体规则变化。  
✨ **主要贡献**：发现模型存在“检测-归因分离”——能察觉系统被改动但难以定位具体突变；交互突变时性能显著下降，且增加推理时间带来的收益有限。
