# AR Forcing: Towards Long-Horizon Robot Navigation World Model

- arXiv: https://arxiv.org/abs/2605.31314v1
- arXiv ID: 2605.31314

---

🔍 **核心问题**：基于扩散的机器人导航世界模型在训练时使用并行监督，但推理时采用自回归方式，导致长时域预测中训练与推理的分布偏移，性能不稳定。

🛠 **方法**：提出AR Forcing自回归训练策略，将标准扩散损失融入自回归训练循环，每一步使用模型自身预测更新上下文并优化单步噪声预测目标，使模型在训练中暴露于推理状态分布。

✨ **主要贡献**：无需额外判别器或分布匹配损失，保留原始扩散框架和采样器，易于集成；在多领域导航数据集上，显著提升长时域导航中生成图像的一致性和预测轨迹的准确性，增强模型在复杂已知/未知环境中的鲁棒性。
