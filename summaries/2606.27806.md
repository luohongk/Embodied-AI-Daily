# Grounded Iterative Language Planning: How Parameterized World Models Reduce Hallucination Propagation in LLM Agents

- arXiv: https://arxiv.org/abs/2606.27806v1
- arXiv ID: 2606.27806

---

🔍 **核心问题**：解决基于LLM的语言智能体在规划任务中因幻觉导致状态错误传播的问题，特别是纯语言智能体难以量化错误，而参数化模型规划能力较弱。

🛠 **方法**：提出**Grounded Iterative Language Planning (GILP)**，训练一个小型参数化世界模型作为骨干，与LLM API推理结合。参数化模型提供有效动作、状态变化预测、风险和价值，LLM生成动作和想象的状态变化，通过一致性门控机制在两者不一致时要求修订。

✨ **主要贡献**：在真实GPT-4o-mini调用中，将幻觉状态率从0.176降至0.035；在模拟消融实验中，成功率从0.668提升至0.838，仅增加约22%的额外LLM调用。
