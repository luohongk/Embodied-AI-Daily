# Safe Reinforcement Learning using Ideas from Model Predictive Control

- arXiv: https://arxiv.org/abs/2607.07252v1
- arXiv ID: 2607.07252

---

🔍 **核心问题**：在强化学习的主动探索阶段，如何确保物理系统（如机器人）严格满足硬性安全约束，避免因违反机械极限造成不可逆损伤。

🛠 **方法**：提出一个通用框架，将深度强化学习（DRL）的自适应性与模型预测控制（MPC）的形式化安全保证相结合。通过离线MPC计算定义可行的状态-动作空间，并在训练和部署时利用安全滤波器将RL智能体的即时动作投影到该全局验证的安全集合内。

✨ **主要贡献**：在非线性单自由度实验室平台上进行了系统评估，成功实现了物理硬件上的安全探索和稳定的策略收敛，验证了该方法在保证严格安全约束的同时保持高性能控制的有效性。
