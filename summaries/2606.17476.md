# Multi-Adapter PPO: A Cross-Attention Enhanced Wavelength Selection Framework for LIBS Quantitative Analysis

- arXiv: https://arxiv.org/abs/2606.17476v1
- arXiv ID: 2606.17476

---

🔍 **核心问题**：激光诱导击穿光谱（LIBS）定量分析中，高维光谱数据导致波长选择困难，且预测精度与特征效率之间存在根本性权衡。

🛠 **方法**：提出Multi-Adapter PPO框架，将波长选择转化为强化学习问题，利用交叉注意力机制和多个专用适配器捕捉复杂光谱关系。

✨ **主要贡献**：在钢铁和煤炭数据集上，综合得分平均比传统粒子群优化（PSO）提升28.4%，预测精度提升45.2%，在平衡预测精度与特征效率方面达到最优，同时保持可解释性和计算效率。
