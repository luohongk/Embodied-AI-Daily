# LLMs for Qualitative and Mixed-Methods Social Network Analysis

- arXiv: https://arxiv.org/abs/2607.14045v1
- arXiv ID: 2607.14045

---

🔍 **核心问题**：探讨如何将大型语言模型（LLMs）整合到定性与混合方法的社会网络分析（SNA）中，以增强研究深度和严谨性，而非替代人类研究者。

🛠 **方法**：提出LLMs作为辅助工具，用于数据收集、编码、理论构建和溯因推理，同时强调保持定性SNA的核心原则（如理解关系意义、叙事和关系身份）。

✨ **主要贡献**：明确了LLMs在定性SNA中的增强角色，指出了偏差、幻觉和反思性等伦理挑战，并提供了负责任整合LLMs的研究设计建议。
