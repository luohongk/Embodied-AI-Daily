## 0. 概览（Concise Summary）

- **主题（Topic）**：该论文属于**强化学习（Reinforcement Learning, RL）** 领域，具体关注**非马尔可夫奖励决策过程（Non-Markovian Reward Decision Processes, NMRDPs）** 中的**基于模型的强化学习（Model-Based RL）** 问题。
- **问题（Problem）**：在NMRDPs中，奖励依赖于整个历史轨迹，而非当前状态。现有的方法（如将NMRDP转化为标准MDP）在理论上缺乏高效的样本复杂度保证，且在实践中样本效率低下，因为它们忽略了环境动态（马尔可夫性）与奖励动态（非马尔可夫性）之间的结构性差异，导致冗余探索。
- **方法（Method）**：提出了一种名为 **QR-Max** 的新型基于模型的算法。该算法通过**奖励机（Reward Machines, RMs）** 将马尔可夫的环境转移学习与非马尔可夫的奖励处理进行**解耦（Factorization）**。具体来说，它分别学习环境转移模型 $P(s'|s, a)$ 和自动机状态转移模型 $P(q'|s', q)$，并利用乐观探索（optimistic exploration）策略进行规划。针对连续状态空间，论文进一步提出了 **Bucket-QR-Max**，它使用 **SimHash** 进行离散化，并保留了相同的解耦结构。
- **创新（Innovation）**：
    1.  **结构解耦**：首次在离散动作的NMRDP中，明确地将环境动态（马尔可夫）与奖励动态（非马尔可夫）解耦，并利用此解耦设计算法。
    2.  **PAC-MDP保证**：证明了 QR-Max 是 PAC-MDP 的，其样本复杂度界限是**分解式**的（$U = |S||A|m_E + |S||Q|m_Q$），而非传统方法在乘积空间上的**乘积式**（$|S||Q||A|m$），从而显著降低了样本复杂度。
    3.  **连续状态扩展**：提出了 Bucket-QR-Max，将解耦思想扩展到连续状态空间，无需手动网格划分或函数逼近，并给出了条件性的 PAC-MDP 保证。
- **意义（Significance）**：该工作为处理具有复杂时序依赖奖励的决策问题提供了一种理论上高效、实践中样本效率高的解决方案。它证明了利用问题结构（环境与奖励的解耦）可以显著提升强化学习算法的性能，为未来在机器人、游戏等领域处理复杂任务提供了新的思路。
- **一句话总结**：本文提出了一种名为 QR-Max 的基于模型的强化学习算法，通过将环境动态与奖励动态解耦，在非马尔可夫奖励决策问题上首次实现了具有多项式样本复杂度的 PAC 收敛保证，并显著提升了样本效率。

## 1. 研究动机（Motivation）

在许多实际决策问题中，任务的成功与否取决于整个历史过程，而不仅仅是达到某个具有特定属性的状态。例如，一个机器人需要“先拿钥匙，再开门，最后取包裹”，其奖励取决于是否按顺序完成了这些子目标。这种奖励依赖于历史轨迹的特性被称为**非马尔可夫性**。

传统的**马尔可夫强化学习（Markovian RL）** 假设奖励仅依赖于当前状态（或当前状态-动作对），因此难以直接处理这类任务。虽然可以将非马尔可夫奖励决策过程（NMRDP）转化为一个状态空间更大的马尔可夫决策过程（MDP），但这种转化会带来以下关键挑战和空白：

1.  **样本效率低下**：转化后的MDP状态是环境状态 $s$ 和自动机状态 $q$ 的笛卡尔积 $(s, q)$。传统基于模型的方法（如 R-Max）会在这个乘积空间上学习一个统一的转移模型 $P(s', q' | s, q, a)$。这忽略了环境动态 $P(s'|s, a)$ 本身是马尔可夫的，与 $q$ 无关这一事实。因此，算法会为每个不同的 $q$ 重复学习相同的环境转移，导致大量冗余的探索，样本效率极低。
2.  **缺乏理论保证**：现有的针对NMRDP的模型无关方法（如 QRM）或模型基方法（如直接应用 R-Max）要么缺乏样本复杂度的理论保证，要么其保证是基于乘积空间的，不够优化。

因此，本文的核心动机是：**如何设计一种算法，既能利用NMRDP的结构（环境动态与奖励动态的解耦）来避免冗余探索，又能提供严格的、更优的样本复杂度理论保证？**

## 2. 创新点（Innovation）

本文的关键创新在于**结构解耦**和**基于此解耦的算法设计**，具体体现在以下两点：

1.  **提出 QR-Max 算法**：这是第一个在离散动作NMRDP中，明确利用环境动态和奖励动态解耦的基于模型的RL算法。其核心创新在于：
    - **分解式学习**：不学习乘积空间 $(s, q)$ 上的联合转移模型，而是分别学习**环境转移模型** $\hat{P}(s'|s, a)$ 和**自动机转移模型** $\hat{P}(q'|s', q)$。环境转移模型对所有自动机状态 $q$ 是共享的，一旦 $(s, a)$ 被学习为“已知”，其知识可以立即用于所有 $q$ 状态，无需重复探索。
    - **分解式乐观探索**：为环境组件和自动机组件分别设置“已知”阈值（$t_E$ 和 $t_Q$）。只有当某个组件被充分探索后，才将其视为“已知”并用于价值迭代。这避免了在乘积空间上对整个 $(s, q, a)$ 三元组进行探索。
    - **PAC-MDP 保证**：证明了这种分解式学习方式可以导出更优的样本复杂度界限。其未知组件数量 $U = |S||A|m_E + |S||Q|m_Q$，远小于直接应用 R-Max 在乘积空间上的 $|S||Q||A|m$。这是理论上的关键突破。

2.  **提出 Bucket-QR-Max 算法**：将 QR-Max 的思想扩展到连续状态空间。
    - **SimHash 离散化**：使用局部敏感哈希（Locality-Sensitive Hashing, LSH）中的 SimHash 方法，将连续状态 $s_{cont}$ 映射到离散的“桶”（bucket）$b$ 中。这种方法无需手动划分网格，且计算高效。
    - **保留解耦结构**：在离散化的桶空间 $B$ 上，依然保持环境动态和自动机动态的解耦学习，从而继承了 QR-Max 在样本效率上的优势。
    - **条件性 PAC 保证**：在标准的有界离散化假设下（即桶直径小且动态和奖励是 Lipschitz 连续的），证明了 Bucket-QR-Max 在原始连续 MDP 上具有条件性的 PAC-MDP 保证。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文的核心架构是**解耦**。它将一个 NMRDP 的转移和奖励函数分解为两个独立的部分：

1.  **环境动态（Markovian）**：$P(s'|s, a)$ 和 $R_E(s, a, s')$。这部分只依赖于当前环境状态 $s$ 和动作 $a$，与历史无关。
2.  **奖励动态（Non-Markovian）**：$P(q'|s', q)$ 和 $R_A(q, s', q')$。这部分由一个确定性的有限状态自动机（DFA）或奖励机（RM）来描述。自动机状态 $q$ 编码了历史信息，其转移 $q' = \eta(q, s')$ 取决于当前自动机状态 $q$ 和下一个环境状态 $s'$。

**QR-Max** 算法正是基于这个解耦架构设计的。它维护两组独立的计数器：
- **环境计数器**：$n_E(s, a)$，用于统计在环境状态 $s$ 执行动作 $a$ 的次数。当 $n_E(s, a) \ge t_E$ 时，认为 $(s, a)$ 是“已知”的。
- **自动机计数器**：$n_Q(q, s')$，用于统计从自动机状态 $q$ 观察到环境状态 $s'$ 的次数。当 $n_Q(q, s') \ge t_Q$ 时，认为 $(q, s')$ 是“已知”的。

算法流程（Algorithm 1）：
1.  **初始化**：将所有 Q 值初始化为乐观值 $R_{max}/(1-\gamma)$。初始化所有计数器为0。
2.  **交互**：在每个时间步，根据当前状态 $(s, q)$ 选择 Q 值最大的动作 $\bar{a}$。
3.  **更新**：执行动作后，观察到 $(s', q', r_E, r_A)$。
    - 如果 $(s, \bar{a})$ 未知，则更新环境计数器 $n_E$ 和转移/奖励估计。
    - 如果 $(q, s')$ 未知，则更新自动机计数器 $n_Q$ 和转移/奖励估计。
4.  **触发价值迭代**：如果任何环境或自动机组件达到“已知”阈值，则触发一次价值迭代（VI），使用当前估计的模型更新 Q 表。
5.  **重置**：如果 episode 结束，将终止状态的 Q 值设为0，并重置环境。

**Bucket-QR-Max** 的架构与 QR-Max 类似，只是将离散状态 $s$ 替换为通过 SimHash 映射得到的桶 $b$。它维护四组计数器（环境转移、环境奖励、自动机转移、自动机奖励），只有当所有四组计数器都达到阈值时，才认为 $(b, a, q)$ 是“已知”的。

### 3.2 关键算法与数学推导

**1. 解耦的转移模型（公式1）**：
$$
P(s', q' | s, q, a) = P(s' | s, a) P(q' | s', q)
$$
这个公式是整个方法的基础。它表明，联合转移概率可以分解为环境转移概率和自动机转移概率的乘积。由于自动机是确定性的，$P(q' | s', q)$ 要么是0，要么是1。

**2. 解耦的奖励函数（公式2）**：
$$
R(s, q, a, s', q') = R_E(s, a, s') + R_A(q, s', q')
$$
总奖励是环境奖励和自动机奖励之和。

**3. 价值迭代的 Bellman 备份（公式3）**：
$$
Q_{k+1}(s, q, a) = \sum_{s'} \hat{R}_E(s, a, s') + \sum_{s'} \hat{P}(s'|s, a) \left[ \sum_{q'} \hat{R}_Q(q, s', q') + \gamma \sum_{q'} \hat{P}(q'|s', q) \max_{a'} Q_k(s', q', a') \right]
$$
其中 $\hat{R}_E(s, a, s') = \hat{P}(s'|s, a) \hat{r}_E(s, a, s')$ 和 $\hat{R}_Q(q, s', q') = \hat{P}(q'|s', q) \hat{r}_Q(q, s', q')$ 是“奖励质量”估计。这个备份公式清晰地体现了解耦思想：它首先对环境转移和奖励求和，然后对自动机转移和奖励求和。在价值迭代中，只有当 $(s, a)$ 和 $(q, s')$ 都是“已知”时，才使用估计值；否则，使用乐观初始值 $R_{max}/(1-\gamma)$。

**4. PAC-MDP 定理（Theorem 3）**：
> 定理：QR-Max 是 PAC-MDP 的。以至少 $1-\delta$ 的概率，其贪婪策略不是 $\epsilon$-最优的交互步数被 $O\left(\frac{HU}{p} \log \frac{U}{\delta}\right)$ 所界，其中 $U = |S||A|m_E + |S||Q|m_Q$。

这个定理的核心在于 $U$ 的定义。它表明，算法需要学习的“未知组件”数量是环境组件数 $|S||A|m_E$ 和自动机组件数 $|S||Q|m_Q$ 之和，而不是乘积空间上的 $|S||Q||A|m$。由于 $|Q|$ 通常远小于 $|S|$，这带来了巨大的样本效率提升。

### 3.3 模型、训练与数据集

- **模型**：算法本身不依赖于深度神经网络，是一种**表格型（tabular）** 的基于模型的方法。它维护了环境转移、自动机转移、环境奖励、自动机奖励的计数表和 Q 值表。
- **训练**：训练过程是标准的基于模型的 RL 流程：与环境交互 -> 更新模型 -> 规划（价值迭代）-> 选择动作。算法使用**乐观探索**策略，即对未知状态-动作对赋予最大可能的 Q 值，鼓励 agent 去探索。
- **数据集**：论文没有使用标准的数据集，而是构建了一个扩展的 **Office Gridworld** 基准测试环境。该环境包含多个地图（map0, map1, map4）和多个不同的奖励机（Reward Machines），用于测试算法在不同空间和时间复杂度下的表现。环境具有随机性（随机动作结果）。

### 3.4 实验设置与结果

- **对比算法**：
    - **模型基方法**：R-Max（在乘积空间上）、UCBVI（及其变体）、PSRL、OPSRL。
    - **模型无关方法**：QRM（一种针对 NMRDP 的 Q-learning 变体）。
- **评估指标**：
    - **样本效率**：达到与最优策略（通过 Value Iteration 计算）统计上无显著差异所需的训练步数。
    - **成功率**：在训练过程中，当前策略在测试环境中的成功率。
- **主要结果**：
    1.  **样本效率大幅提升**：在所有配置中，QR-Max 和 QR-MaxRM 达到最优策略所需的训练步数比 R-Max、UCBVI、PSRL、OPSRL 和 QRM 少 **1 到 2 个数量级**。例如，在配置2（map1 exp5）中，QR-Max 需要约 24,000 步，而 UCBVI 需要超过 500,000 步（见表1）。
    2.  **鲁棒性更强**：在配置3（map4 exp6）中，QR-Max 能够成功找到取“好咖啡”的最优策略，而 QRM 和 UCBVI 等算法则陷入取“普通咖啡”的局部最优解，无法跳出。
    3.  **连续状态空间有效**：在连续状态的 Frozen Lake 和 Office World 任务中，Bucket-QR-Max 在约 50 个 episode 内达到近最优性能，而 DQN 需要超过 500 个 episode，且方差更小。
    4.  **超参数易调**：QR-Max 的主要超参数是阈值 $t_E$ 和 $t_Q$，其性能对 $t_E$ 的敏感性较低，且 $t_Q=1$ 对于确定性 RM 是自然的选择。相比之下，其他算法（如 UCBVI 的 H，PSRL 的采样数）的调参难度更大。

### 3.5 技术细节

- **已知性阈值**：$t_E$ 和 $t_Q$ 的选择基于 Hoeffding 不等式和 Weissman 不等式，通过集中不等式和联合界（union bound）来确定，以保证以高概率估计误差在可控范围内。对于确定性 RM，$t_Q = 1$ 就足够了。
- **价值迭代**：在 QR-Max 中，价值迭代只在有新的环境或自动机组件变为“已知”时触发，而不是每步都运行，从而节省了计算资源。
- **QR-MaxRM 变体**：当奖励机（RM）的结构（即 $\eta$ 函数）已知时，QR-MaxRM 可以利用这个知识进行**反事实更新**，即对于未观察到的 $(q, s')$ 组合，也能推断出 $q'$ 并更新模型，从而进一步加速学习。
- **SimHash 离散化**：Bucket-QR-Max 使用 $L$ 个独立的 SimHash 函数将连续状态映射到 $L$ 维的哈希码，形成一个桶。桶的数量是动态增长的。

## 4. 意义与影响（Significance and Impact）

- **解决了什么问题**：
    - 解决了在非马尔可夫奖励决策问题中，现有方法样本效率低下且缺乏高效理论保证的问题。
    - 通过结构解耦，解决了在乘积空间上学习导致的冗余探索问题。
    - 为连续状态空间的非马尔可夫奖励问题提供了一种无需手动特征工程或神经网络的实用解决方案。

- **贡献是什么**：
    - 提出了第一个在离散动作 NMRDP 中具有 PAC 保证且样本复杂度界限更优的基于模型的 RL 算法（QR-Max）。
    - 提出了一种将解耦思想扩展到连续状态空间的实用方法（Bucket-QR-Max）。
    - 通过大量实验证明了所提方法在样本效率和鲁棒性上相对于现有方法的显著优势。

- **对未来研究/应用的影响**：
    - **理论方向**：为设计更高效、更具结构意识的 RL 算法提供了新的思路。未来可以探索将这种解耦思想扩展到随机奖励机、部分可观测环境或深度 RL 中。
    - **应用方向**：在机器人、自动驾驶、游戏 AI 等需要完成复杂时序任务（如“先做A，再做B，最后做C”）的领域，该方法有望大幅降低训练成本和时间。论文也提到，其最优离散动作策略可以用于指导连续动作策略
