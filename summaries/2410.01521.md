# MiraGe: Editable 2D Images using Gaussian Splatting

- arXiv: https://arxiv.org/abs/2410.01521v3
- arXiv ID: 2410.01521

---

🔍 **核心问题**：现有基于高斯函数的2D图像表示方法（如GaussianImage）无法支持图像编辑，而传统隐式神经表示虽能编码图像但缺乏灵活修改能力。

🛠 **方法**：提出MiraGe方法，利用镜面反射将2D图像感知为3D空间中的对象，并通过扁平化控制的高斯函数（flat-controlled Gaussians）实现精确的2D图像编辑，同时可结合物理引擎进行基于物理的修改。

✨ **主要贡献**：首次实现基于高斯函数的可编辑2D图像表示，在提升渲染质量的同时支持类人3D感知的逼真图像修改，并展示了与物理引擎结合实现物理驱动的图像编辑能力。
