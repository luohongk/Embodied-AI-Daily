# Preferential Multi-Target Search in Indoor Environments using Semantic SLAM

- arXiv: https://arxiv.org/abs/2309.14063v3
- arXiv ID: 2309.14063

---

🔍 **核心问题**：解决服务机器人在室内环境中高效搜索多个目标物体（如“找齐我的物品”）的难题，尤其需应对障碍物遮挡导致传统导航失效的情况。

🛠 **方法**：提出一种基于语义SLAM的启发式搜索策略，融合语义物体识别与几何数据构建多层地图，并结合概率先验进行高效推理，同时兼容多种物体检测器、RGB-D SLAM和局部导航规划器。

✨ **主要贡献**：首次实现考虑用户偏好的多目标优先搜索，在真实室内环境中验证了方法的有效性，显著提升机器人在复杂场景下定位、抓取和运输物体的鲁棒性。
