# PISA: A Pragmatic Psych-Inspired Unified Memory System for Enhanced AI Agency

- arXiv: https://arxiv.org/abs/2510.15966v2
- arXiv ID: 2510.15966

---

🔍 **核心问题**：现有AI代理的记忆系统缺乏任务适应性，且忽视了记忆的建构性和任务导向性功能。  
🛠 **方法**：基于皮亚杰认知发展理论，提出PISA统一记忆系统，包含三模态适应机制（模式更新、演化、创建）和混合记忆访问架构（符号推理与神经检索结合）。  
✨ **主要贡献**：在LOCOMO和新提出的AggQA基准上达到最优性能，显著提升了AI代理的适应性和长期知识保持能力。
