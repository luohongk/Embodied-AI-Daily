# Logic-Guided Vector Fields for Constrained Generative Modeling

- arXiv: https://arxiv.org/abs/2602.02009v2
- arXiv ID: 2602.02009

---

🔍 **核心问题**：生成模型在生成过程中缺乏机制来强制执行声明性约束，导致输出可能违反逻辑规则或可行性条件。  
🛠 **方法**：提出逻辑引导向量场（LGVF），将逻辑约束的可微松弛注入流匹配生成模型，通过训练时逻辑损失（沿连续轨迹惩罚违规）和推理时约束梯度调整（轻量级修正采样路径）相结合。  
✨ **主要贡献**：在三种约束生成任务中，LGVF将约束违规减少59-82%，达到最低违规率；在部分场景中提升分布保真度（MMD），并展现出无需显式路径规划的避障行为。
