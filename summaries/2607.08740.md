# Workflow as Knowledge: Semantic Persistence for LLM-Mediated Workflows

- arXiv: https://arxiv.org/abs/2607.08740v1
- arXiv ID: 2607.08740

---

🔍 **核心问题**：现有LLM应用的工作流系统主要关注执行效率，但缺乏将工作流本身作为可检查、可恢复、可审查的知识对象进行持久化表示的概念模型。

🛠 **方法**：受Lisp启发，提出一种语言无关的概念模型，通过符号形式、对象标识和实时映像思维，将工作流定义、实例、推理记录、上下文快照和依赖关系表示为共享知识基板中的持久知识对象，并区分确定性计算（derive）与LLM推理（infer）。

✨ **主要贡献**：首次提出“语义持久性”概念，使工作流不仅产生知识痕迹，其自身也能作为可检查、可恢复和可审查的知识对象存在，为未来形式化转换语义研究奠定基础。
