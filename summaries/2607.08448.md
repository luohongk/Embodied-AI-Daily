# Harness VLA: Steering Frozen VLAs into Reliable Manipulation Primitives via Memory-Guided Agents

- arXiv: https://arxiv.org/abs/2607.08448v1
- arXiv ID: 2607.08448

---

🔍 **核心问题**：解决端到端视觉-语言-动作（VLA）模型在部署时因语义重定向、目标重绑定、空间布局变化及局部接触不稳定等扰动而失效的问题，同时弥补LLM编码代理在复杂接触操作（如不规则抓取、铰接物体交互）中的不足。

🛠 **方法**：提出Harness VLA框架，将冻结的VLA模型作为可重试的接触密集型原语，与少量固定分析原语（如接地、运输、导航）组合，并通过记忆增强代理学习任务执行轨迹、全局成功规则和失败模型，从而在不微调VLA的情况下扩展其操作范围。

✨ **主要贡献**：在LIBERO-Pro和RoboCasa365上分别提升38.6和25.4个百分点，在RoboTwin C2R上达到58.4%的成功率，显著优于最强基线，且无需扩展技能库或微调预训练模型。
