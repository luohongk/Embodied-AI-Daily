# QPILOTS: Efficient Test-Time Q-Steering for Flow Policies

- arXiv: https://arxiv.org/abs/2606.14801v1
- arXiv ID: 2606.14801

---

🔍 **核心问题**：解决流匹配与扩散策略在时序差分强化学习中难以优化的问题，尤其是通过多步去噪过程反向传播评论家梯度时存在数值不稳定性。

🛠 **方法**：提出QPILOTS方法，在推理时保持原始策略不变，通过将每个去噪步骤中的噪声中间状态投影到最终干净动作的估计值，并在该估计值上计算评论家梯度来引导去噪过程。包含两种变体：QPILOTS-U使用快速单点近似，QPILOTS-M通过学习的辅助网络生成可微分的后验样本。

✨ **主要贡献**：在离线到在线强化学习基准上，QPILOTS在50个任务中平均成功率达90%，取得最佳综合性能；同时成功应用于引导大型预训练视觉-语言-动作基础模型，在六项模拟操作任务中优于或匹配现有推理时方法。
