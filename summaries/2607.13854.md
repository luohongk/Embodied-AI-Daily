# SPyCE: Skill-Policy Co-evolution for Multimodal Agents

- arXiv: https://arxiv.org/abs/2607.13854v1
- arXiv ID: 2607.13854

---

🔍 **核心问题**：现有强化学习方法将多模态推理轨迹简化为标量奖励，导致策略需从零发现可复用的工具使用模式；而基于记忆的方法仅依赖测试时检索，未更新策略以吸收可复用模式。

🛠 **方法**：提出SPyCE框架，在强化学习过程中将轨迹蒸馏为分层技能库（包括局部视觉操作的执行技能和编排工具使用的高层工作流技能），并通过策略与技能库的协同进化形成闭环——策略利用检索技能指导轨迹生成，技能库则从策略生成的有效轨迹中更新。

✨ **主要贡献**：在八个基准测试中，SPyCE一致优于基于强化学习和基于记忆的基线方法，且分层技能设计与协同进化机制均被验证为关键因素，表明技能-策略联合优化是构建多模态智能体的有效范式。
