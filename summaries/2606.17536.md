# OmniDrive: An LLM-Choreographed Multi-Agent World Model with Unified Latent Co-Compression for Multi-View Driving Video Generation

- arXiv: https://arxiv.org/abs/2606.17536v1
- arXiv ID: 2606.17536

---

🔍 **核心问题**：自动驾驶生成式世界模型面临两大难题：异构控制信号（如自然语言、高精地图、轨迹等）无法在统一表示空间内对齐，以及多视角潜变量缺乏全局3D几何编码，导致跨视角融合效果差。

🛠 **方法**：提出DRIVE-CHOREO，一种由大语言模型（Qwen2.5-VL）编排的多智能体世界模型。通过三个智能体（Director解析意图生成WorldScript，Cartographer映射为空间布局令牌，Auditor提供跨视角反馈）共同生成位置感知令牌序列，并与多视角视频通过视图-时间排列进行联合潜压缩，在3D VAE的卷积感受野内强制编码相机间几何关系。

✨ **主要贡献**：在nuScenes数据集上，多视角一致性和BEV mAP（21.6）达到新最优，FVD（45.7）具竞争力；仅用合成数据训练的检测器在真实验证集上NDS提升+2.4，验证了下游实用性。
