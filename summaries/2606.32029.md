# When LLMs Read Tables Carelessly: Measuring and Reducing Data Referencing Errors

- arXiv: https://arxiv.org/abs/2606.32029v1
- arXiv ID: 2606.32029

---

🔍 **核心问题**：大语言模型在表格任务中常出现数据引用错误（DREs），即错误引用或遗漏表格数值，影响推理步骤的准确性和可靠性，而此前缺乏系统性评估。

🛠 **方法**：首次系统评估不同模型和任务中的DREs，并提出基于数据引用批评机制的方案，通过批评过滤和拒绝采样提升答案准确性，同时训练轻量级批评模型检测DREs。

✨ **主要贡献**：发现所有测试模型（1.7B至20B参数）均存在DREs；引入批评机制后答案准确率最高提升12.0%；训练的4B参数批评模型在分布内和分布外DREs检测中平均F1达78.2%，有效辅助大模型推理。
