# Data-Free Reservoir Features for Efficient Long-Horizon Cold-Start Continual Learning

- arXiv: https://arxiv.org/abs/2606.27095v1
- arXiv ID: 2606.27095

---

🔍 **核心问题**：解决冷启动无样本类增量学习中，现有方法在长任务序列下计算成本高或性能弱的问题，特别是避免回放、外部预训练或大初始任务的需求。

🛠 **方法**：提出CIRCLE方法，使用固定的双向二维储层特征（BiRC2D）作为特征提取器，结合流式线性判别分析（SLDA）分类头，并通过多随机储层实例集成和软输出平均来平衡偏差-方差。

✨ **主要贡献**：在CIFAR-100、TinyImageNet等数据集上，CIRCLE在10-20任务分割时表现竞争力，在50、100、500任务分割时大幅超越强基线，且训练速度远快于基于骨干网络漂移补偿的方法。消融实验证实BiRC2D提取器、SLDA头和集成策略均贡献显著。
