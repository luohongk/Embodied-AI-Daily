# Regularized Reward-Punishment Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.28152v1
- arXiv ID: 2606.28152

---

🔍 **核心问题**：现有奖励-惩罚强化学习方法中，追求奖励和规避惩罚的策略优化过程相互独立，缺乏有效协调，导致安全性和学习稳定性不足。

🛠 **方法**：提出KL耦合策略正则化（KCPR）框架，通过将同伴策略视为动态学习的先验，实现奖励与惩罚策略的直接交互；并推导出KL耦合软最优性（KCSO），结合KL正则化贝尔曼算子，使奖励和惩罚信息共同影响价值传播。

✨ **主要贡献**：引入同伴先验软化机制和分离经验回放设计，在网格世界和Gazebo机器人导航任务中，klDMP方法相比DQN、SQL和softDMP，显著提升了安全性和学习稳定性，同时保持竞争性任务性能，验证了策略级协调对整合多行为目标的有效性。
