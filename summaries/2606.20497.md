# Interpretable Meta-Learning for Multi-Objective Chemical Search

- arXiv: https://arxiv.org/abs/2606.20497v1
- arXiv ID: 2606.20497

---

🔍 **核心问题**：在合成化学分子的大规模搜索中，现有深度学习模型难以在量子化学计算约束下同时满足可解释性和多目标优化需求。

🛠 **方法**：提出一种模块化流水线，结合可解释的线性元学习模型与自适应置信度不确定性量化，集成到高效全局优化框架中，用于多目标分子发现。

✨ **主要贡献**：首次将线性元学习应用于多目标化学搜索，通过跨化学目标和廉价辅助属性训练，实现快速适应新目标的迁移学习；同时引入动态置信度调优算法，在主动搜索中重新校准探索-利用平衡，使性能在帕累托前沿上比基线提升78%，且动态调优在超过50%的静态校准前沿上占据优势。
