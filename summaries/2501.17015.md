# UniMM: A Unified Mixture Model Framework for Multi-Agent Simulation

- arXiv: https://arxiv.org/abs/2501.17015v2
- arXiv ID: 2501.17015

---

🔍 **核心问题**：解决自动驾驶多智能体仿真中行为多模态性及闭环分布偏移问题。  
🛠 **方法**：提出统一混合模型（UniMM）框架，整合回归混合模型与离散NTP模型，并设计闭环样本生成策略及时间解耦对齐机制。  
✨ **主要贡献**：系统分析了模型与数据配置的影响，通过闭环样本缓解分布偏移，在WOSAC基准上离散、无锚点及锚点变体均达最优性能。
