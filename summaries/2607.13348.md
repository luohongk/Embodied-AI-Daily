# Safe Overtaking for Autonomous Racing Using Hierarchical Optimization and Learning-Based Control

- arXiv: https://arxiv.org/abs/2607.13348v1
- arXiv ID: 2607.13348

---

🔍 **核心问题**：在自动驾驶赛车超车场景中，如何在保证安全的前提下提升超车性能，避免传统固定衰减控制屏障函数（CBF）因过于保守而限制表现，且需手动调整参数适应不同赛道。

🛠 **方法**：提出分层超车框架，高层使用混合整数二次规划（MIQP）选择超车拓扑，低层采用非线性Frenet帧模型预测控制（MPC）嵌入离散时间CBF约束，并通过强化学习策略在线自适应调整CBF衰减参数。

✨ **主要贡献**：通过分层设计分离决策与轨迹优化，结合自适应CBF参数学习，在无需逐赛道调参的情况下，实现安全与性能的均衡，仿真与硬件实验验证其鲁棒性和高成功率。
