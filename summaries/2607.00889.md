# DeWorldSG: Depth-Aware 3D Semantic Scene Graph Generation via World-Model Priors

- arXiv: https://arxiv.org/abs/2607.00889v1
- arXiv ID: 2607.00889

---

🔍 **核心问题**：现有3D语义场景图生成方法因3D物体表示不稳定和逐帧推理导致关系缺失，难以构建时空鲁棒的场景结构。

🛠 **方法**：提出DeWorldSG框架，通过深度引导滤波估计实例级几何3D高斯分布，将每个物体表示为概率性3D节点；并跨帧聚合时空证据，利用世界模型（V-JEPA 2）的上下文先验优化关系。

✨ **主要贡献**：在3DSSG和ReplicaSSG数据集上实现物体与谓词预测的SOTA性能，三元组召回率提升77.4%，谓词召回率提升23.2%，生成时空一致的场景结构，适用于机器人操作和AR应用。
