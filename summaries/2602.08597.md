# An Attention Mechanism for Robust Multimodal Integration in a Global Workspace Architecture

- arXiv: https://arxiv.org/abs/2602.08597v3
- arXiv ID: 2602.08597

---

🔍 **核心问题**：在多模态系统中，当某些模态存在噪声或不可靠时，如何在不依赖端到端联合训练的情况下，实现鲁棒的模态选择与融合。

🛠 **方法**：基于全局工作空间理论，设计了一个轻量级的自上而下模态选择器，该选择器在冻结的多模态全局工作空间之上运行，通过注意力机制动态选择可靠模态。

✨ **主要贡献**：该选择器在仅使用远少于端到端注意力基线的可训练参数下，提升了多模态系统的鲁棒性；其学习到的选择策略能更好地迁移到不同下游任务、噪声场景甚至未见过的模态，并在MM-IMDb 1.0基准上取得了有竞争力的性能。
