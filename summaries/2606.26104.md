# Assert, don't describe: Linguistic features that shift LLM reasoning about animal welfare

- arXiv: https://arxiv.org/abs/2606.26104v2
- arXiv ID: 2606.26104

---

🔍 **核心问题**：研究哪些语言特征能改变大语言模型（LLM）在动物福利议题上的推理倾向，为动物福利倡导者提供写作指导。  
🛠 **方法**：使用词汇匹配的立场对比探针，在保留的动物福利基准上，通过微调数据测试10种语言特征对Llama-3.2-1B模型立场的影响。  
✨ **主要贡献**：发现8种特征显著改变模型立场，其中7种（如断言性肯定、道德词汇、情感词等）增强动物福利支持，2种（如模糊语言、具体感官描述）削弱立场；建议写作时明确主张而非中性描述。
