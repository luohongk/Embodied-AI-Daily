# AC-VLA: Robust Out-of-Distribution Action Execution via Compositional Learning

- arXiv: https://arxiv.org/abs/2607.15714v1
- arXiv ID: 2607.15714

---

🔍 **核心问题**：解决视觉-语言-动作（VLA）模型在组合式分布外（OOD）任务中泛化能力差的问题，具体表现为轨迹过拟合和感知捷径两种失效模式。

🛠 **方法**：提出AC-VLA框架，包含两个组件：一是利用LLM驱动的指令分解器和本体感知轨迹对齐器生成子任务监督，通过混合训练增强组合泛化；二是状态条件非对称掩码策略，在夹爪闭合阶段抑制腕部视角输入，强制模型依赖全局空间信息。

✨ **主要贡献**：在LIBERO-OOD基准上，AC-VLA在组合OOD任务上实现约28%的绝对性能提升，同时保持接近完美的分布内任务表现，且无需修改模型架构即可直接集成到任何VLA骨干网络中。
