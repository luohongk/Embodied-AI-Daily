# Metric-Guided Synthetic Image Data Rendering for Deep Learning compatible with Agentic AI

- arXiv: https://arxiv.org/abs/2607.12874v1
- arXiv ID: 2607.12874

---

🔍 **核心问题**：深度学习计算机视觉在科学应用中需要大量人工标注的真实图像数据，过程繁琐、昂贵且易出错，而合成数据生成虽能简化流程，但缺乏系统性的量化指标来指导缩小真实与合成图像之间的领域差距。

🛠 **方法**：开发了Python包GraNatPy，通过引入量化指标（如梯度相似性）来指导渲染场景的改进，并利用程序化渲染与真实数据混合策略优化小目标检测性能；同时将程序化数据渲染转化为智能体技能SynthClaw，实现参数自动优化。

✨ **主要贡献**：证明了渲染数据集的真实感、多样性和尺寸的量化提升与视觉感知改善及目标检测模型零样本性能正相关；通过病毒噬斑实验照片验证了梯度相似性对小目标检测的影响，并展示了混合真实与合成数据的有效性；首次将合成数据渲染流程封装为可自主优化的智能体技能。
