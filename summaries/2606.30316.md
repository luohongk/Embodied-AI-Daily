# Toward an Energy-Optimized Operation of Data Centers Located in Wind Farms Using Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.30316v1
- arXiv ID: 2606.30316

---

🔍 **核心问题**：如何利用强化学习在线优化位于风电场的数据中心的能耗，解决风能利用不足和信用分配问题。  
🛠 **方法**：采用强化学习（PPO和SAC变体）作为在线控制器，结合模仿学习和奖励塑形两种互补策略，在合成风能和电价信号的固定日模拟框架中进行训练和测试。  
✨ **主要贡献**：验证了强化学习在风能感知工作负载迁移中的有效性，发现模仿学习和奖励塑形能改善性能，并提供了可复现的基准框架，为扩展到多站点和连续时间场景奠定基础。
