# HiL-ResRL: A Model-Agnostic Finetuning Adapter via Human-in-the-loop Residual Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.22860v1
- arXiv ID: 2606.22860

---

🔍 **核心问题**：现有基于行为克隆的机器人操作模型存在累积误差和分布偏移问题，导致在工业实际部署中效果有限。

🛠 **方法**：提出一种模型无关的微调适配器HiL-ResRL，在视觉-语言-动作模型生成的动作为统一接口上训练残差策略，并引入人在回路引导以实现安全探索和高效训练。

✨ **主要贡献**：在真实机器人场景中仅需1.5小时在线强化学习训练，平均成功率超过95%，为行为克隆模型在工业场景中的实用化部署提供了高效解决方案。
