# Optimizing Feature Extraction for On-device Model Inference with User Behavior Sequences

- arXiv: https://arxiv.org/abs/2603.21508v1
- arXiv ID: 2603.21508

---

🔍 **核心问题**：现有研究主要关注加速模型推理本身，但忽略了从原始应用日志中提取输入特征这一实际瓶颈，导致端到端模型执行延迟较高。

🛠 **方法**：提出AutoFeature自动化特征提取引擎，通过图抽象将不同特征的提取流程构建为有向无环图，利用图优化融合冗余操作节点，并采用高效缓存减少连续推理间的重复数据处理。

✨ **主要贡献**：在搜索、视频和电商等五个工业移动服务中验证，AutoFeature将端到端设备端模型执行延迟降低1.33倍至4.53倍，且不牺牲模型推理精度。
