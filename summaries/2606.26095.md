# Learning Action Priors for Cross-embodiment Robot Manipulation

- arXiv: https://arxiv.org/abs/2606.26095v1
- arXiv ID: 2606.26095

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型缺乏显式的动作先验，导致动作模块需从零学习物理运动，在跨实体场景中难以同时捕捉时间动作动态与跨模态对齐。

🛠 **方法**：提出两阶段训练框架。第一阶段，使用基于流匹配的轻量级编码器-解码器动作模块，仅从无条件动作轨迹中学习时间运动结构；第二阶段，通过解码器复用和早期潜在蒸馏，将学到的先验迁移至VLA训练，同时支持端到端策略优化。

✨ **主要贡献**：在13个跨实体任务（模拟与真实平台）上，该方法相比无动作先验的VLA训练，实现更快收敛、更高成功率，尤其在数据稀缺的真实任务中表现显著提升；且第一阶段动作数据扩展可产生更泛化的动作先验，直接改善下游VLA性能。
