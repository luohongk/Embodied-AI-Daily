# Which Values Do LLMs Confuse? A Schwartz-Based Recognition Study

- arXiv: https://arxiv.org/abs/2607.20270v1
- arXiv ID: 2607.20270

---

🔍 **核心问题**：研究大型语言模型能否准确识别具体情境中表达的施瓦茨基本价值观，以及模型在价值观识别中存在的系统性混淆模式。

🛠 **方法**：构建包含1000个俄语情境文本的评估集，覆盖施瓦茨的十种基本价值观，由两名人工标注者独立标注；采用固定排序响应协议评估21个指令微调的大语言模型，分析其top-1和top-3识别准确率及错误类型。

✨ **主要贡献**：发现模型在价值观识别中准确率有限（Acc@1=0.683），但能定位正确动机区域（Acc@3=0.892）；识别出八种跨模型复现的定向混淆模式（如普世主义→仁慈、传统→遵从），且严重程度因模型而异，可能影响高阶价值观评估的偏差。
