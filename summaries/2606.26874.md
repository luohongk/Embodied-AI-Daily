# TAVR-VLM: Risk-Conditioned Causal Grounding for Hallucination-Resistant Report Generation

- arXiv: https://arxiv.org/abs/2606.26874v1
- arXiv ID: 2606.26874

---

🔍 **核心问题**：多模态大语言模型在TAVR手术规划中生成文本时，常出现缺乏解剖学依据的诊断性幻觉，影响高风险医疗场景的可靠性。

🛠 **方法**：提出TAVR-VLM框架，核心是风险条件因果注意力机制，通过“风险→区域→词”的结构化因果路径，将多模态输入压缩为全局风险掩码，并在自回归生成中利用支持投影因果一致性约束来确保词级定位。

✨ **主要贡献**：在1482名患者数据集上，AUROC达0.896，CIDEr提升至0.936，幻觉率降至8.1%，显著提升可解释性，为循证手术AI建立新标杆。
