# Rethinking Code Performance Benchmarks for LLMs

- arXiv: https://arxiv.org/abs/2607.07619v1
- arXiv ID: 2607.07619

---

🔍 **核心问题**：现有代码性能基准测试未能有效评估大语言模型（LLM）生成代码的执行效率，因为多数测试中LLM生成代码与标准解法在运行时间上无显著差异。

🛠 **方法**：提出基于LLM的多智能体框架，通过三个独立智能体（生成、诊断、修复）自动生成面向性能的测试用例，在保持功能正确性的同时更有效地暴露运行时差异。

✨ **主要贡献**：在1,345个原测试无显著差异的任务中，该框架使用DeepSeek-v3.1和GPT-4o分别使24.01%和25.43%的任务显示出统计显著的性能提升，优于现有最先进的LLM性能测试生成方法。
