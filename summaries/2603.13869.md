# TransDex: Pre-training Visuo-Tactile Policy with Point Cloud Reconstruction for Dexterous Manipulation of Transparent Objects

- arXiv: https://arxiv.org/abs/2603.13869v2
- arXiv ID: 2603.13869

---

🔍 **核心问题**：灵巧手在操作透明物体时面临自遮挡、深度噪声大和深度信息丢失的问题，导致操作困难。

🛠 **方法**：提出TransDex，一种基于点云重建预训练的3D视觉-触觉融合运动策略，采用自监督点云重建预训练（基于Transformer）和细粒度分层感知编码，结合多轮注意力机制自适应融合机械臂与灵巧手特征。

✨ **主要贡献**：在真实机器人系统上验证了TransDex在透明物体操作中优于现有基线方法，并证明了其泛化能力和各模块的有效性。
