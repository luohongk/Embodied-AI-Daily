# Spatial Attention: Adapting Execution Horizons for Diffusion Policies via Observation Sensitivity

- arXiv: https://arxiv.org/abs/2607.04739v1
- arXiv ID: 2607.04739

---

🔍 **核心问题**：现有扩散策略在机器人模仿学习中固定执行动作块的时间长度，难以在响应速度和计算成本之间取得平衡。

🛠 **方法**：提出“空间注意力”指标（动作对数似然对观测梯度的期望平方范数），用于衡量策略对观测变化的敏感度，并据此动态调整动作块的执行时长——高敏感阶段缩短执行周期，低敏感阶段延长执行周期。

✨ **主要贡献**：在固定采样预算下，通过预测空间注意力值动态分配执行时长，在仿真和真实机器人实验中显著提升任务成功率，同时保持平均执行时长与基线方法相当。
