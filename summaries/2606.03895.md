# Agent libOS: A Runtime Substrate for Capability-Controlled Self-Evolving LLM Agents

- arXiv: https://arxiv.org/abs/2606.03895v2
- arXiv ID: 2606.03895

---

🔍 **核心问题**：LLM智能体在自我进化过程中，若动作暴露即授予权限，会导致权限提升的安全漏洞，需要一种机制在允许能力进化的同时严格控制资源权限变更。

🛠 **方法**：提出Agent libOS，一种智能体原生库操作系统，通过将智能体建模为具有进程标识、对象内存、消息队列、工具表、技能、JIT工具、子进程、预算、检查点和显式能力的AgentProcess，并利用进程本地命名空间、系统调用中介、可信运行时模块等机制，确保模型可见的能力可进化，但资源权限仅通过显式审计的原语变更。

✨ **主要贡献**：实现了能力控制与安全隔离的统一，在27项确定性任务中完成所有任务计划并阻止所有模拟的未授权副作用，保守误拒率仅7.0%，而简单包装和沙箱基线虽保持任务完成但多数安全检查失败。
