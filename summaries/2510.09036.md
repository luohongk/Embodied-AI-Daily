# RoDyn: Taming Interactive Robot-Dynamic 2.5D World Model for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2510.09036v2
- arXiv ID: 2510.09036

---

🔍 **核心问题**：现有2D视频世界模型缺乏空间与运动推理能力，难以支持机器人操作任务中的物理交互。

🛠 **方法**：提出RoDyn模型，通过机器人动态分词器将语义视觉与空间、智能体先验耦合，并采用掩码引导的自回归架构聚焦机器人-物体交互区域。

✨ **主要贡献**：在大规模数据集上实现最先进的生成保真度，将预测能力转化为下游任务提升，使模型强化学习加速，真实世界模仿学习成功率较纯2D基线提升42%。
