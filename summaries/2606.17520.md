# GASE: Gaussian Splatting-Based Automated System for Reconstructing Embodied-Simulation Environments

- arXiv: https://arxiv.org/abs/2606.17520v1
- arXiv ID: 2606.17520

---

🔍 **核心问题**：解决在真实世界中训练具身智能体成本高、效率低的问题，提出快速构建高保真仿真环境以缩小仿真与现实差距的方法。

🛠 **方法**：提出GASE系统，利用全景相机阵列的多视角视频流快速扫描环境，通过基于相机姿态的策略在2D域稳健提取前景物体，并实现高保真场景修复，随后独立重建前景物体与静态背景，无缝导入物理仿真器用于策略训练。

✨ **主要贡献**：在分割精度上超越现有3D高斯方法超10%，修复质量达到最优；在真实机器人操作与导航任务中，策略性能与纯真实数据训练差距小于10%，验证了系统高效缩小仿真与现实差距的能力。
