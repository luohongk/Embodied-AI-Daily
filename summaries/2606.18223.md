# Learning Red Agent Policy from Observations for Neurosymbolic Autonomous Cyber Agents

- arXiv: https://arxiv.org/abs/2606.18223v1
- arXiv ID: 2606.18223

---

🔍 **核心问题**：在部分可观测的自主网络环境中，防御者无法直接观察攻击者（红方智能体）的动作，导致难以预测其行为、学习其策略或评估入侵程度。  
🛠 **方法**：提出一种基于模仿学习的策略学习技术，用于部分可观测的离散状态和离散动作强化学习智能体，通过网络观测和防御者动作来预测红方智能体的行为。  
✨ **主要贡献**：将该方法集成到神经符号网络防御智能体中，能有效处理多种红方策略，并在不同模拟场景中实现高预测准确率。
