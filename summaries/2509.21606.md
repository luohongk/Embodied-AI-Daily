# Task-Agnostic Federated Continual Learning via Replay-Free Gradient Projection

- arXiv: https://arxiv.org/abs/2509.21606v3
- arXiv ID: 2509.21606

---

🔍 **核心问题**：联邦持续学习（FCL）中，分布式客户端在顺序到达的任务上协作训练时，常因灾难性遗忘、依赖可能违反隐私的回放缓冲区或生成模型，以及推理时需已知任务身份而受限。

🛠 **方法**：提出FedProTIP框架，客户端通过随机奇异值分解提取中间激活的低秩核心基，上传至服务器聚合为全局任务子空间；训练时将更新投影到先前子空间的正交补上以减少干扰，推理时通过子空间相关性估计任务身份。

✨ **主要贡献**：无需回放或任务标签，在CIFAR100、ImageNet-R和DomainNet上持续优于现有联邦持续学习方法，同时降低训练时间、内存占用和通信成本。
