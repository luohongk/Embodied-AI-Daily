# SplatSearch: Instance Image Goal Navigation for Mobile Robots using 3D Gaussian Splatting and Diffusion Models

- arXiv: https://arxiv.org/abs/2511.12972v2
- arXiv ID: 2511.12972

---

🔍 **核心问题**：解决移动机器人在未知环境中仅凭单张目标参考图像（任意视角）搜索特定物体或人的实例图像目标导航（IIN）问题，尤其应对稀疏视角场景重建的挑战。

🛠 **方法**：提出SplatSearch架构，利用稀疏视角的3D高斯泼溅（3DGS）在线重建地图，渲染候选物体周围多视角图像，并通过多视角扩散模型补全缺失区域，实现与目标图像的鲁棒特征匹配；同时引入基于合成视角视觉上下文与目标图像语义上下文的前沿探索策略，优先探索语义和视觉相关的区域。

✨ **主要贡献**：在逼真家庭场景和真实环境中，SplatSearch在成功率与成功路径长度上显著优于现有方法；消融实验验证了其设计选择的有效性。
