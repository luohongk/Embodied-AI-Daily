# Copy Less, Ground More: Overcoming Repetitive Copying in Long-Context Reasoning via Evidence-Aware Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.19345v1
- arXiv ID: 2607.19345

---

🔍 **核心问题**：长上下文推理中，大语言模型会从输入中大量复制无关文本，而非基于关键证据进行有效推理，导致回答错误且推理效率低下。

🛠 **方法**：提出GEAR（基于证据感知的奖励塑造方法），在准确率奖励基础上，增加对关键证据重叠的正面奖励和对无关上下文重叠的惩罚，并构建自动化流水线生成带证据标注的训练数据。

✨ **主要贡献**：在多个模型规模和基准上，GEAR相比标准强化学习（仅基于准确率奖励）平均提升+4.6分，尤其在长上下文场景下效果更显著，同时减少了重复复制和推理长度。
