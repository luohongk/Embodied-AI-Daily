# PROTECT-90: A Fault Dataset for Power System Protection

- arXiv: https://arxiv.org/abs/2606.24298v1
- arXiv ID: 2606.24298

---

🔍 **核心问题**：当前电力系统保护领域缺乏标准化、公开可用的高压波形数据集，导致数据驱动方法的评估缺乏透明性和可重复性。  
🛠 **方法**：通过电磁暂态仿真，在标准化90 kV双线拓扑上生成9,022个物理一致的短路故障场景，并记录8个测量点的三相电压/电流波形及结构化元数据。  
✨ **主要贡献**：首次公开提供包含系统化域随机化、平衡场景覆盖的开放基准数据集PROTECT-90，为保护导向的信号处理与学习方法建立可重复评估的标准化基础。
