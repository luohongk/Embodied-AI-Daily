# On-Policy Delta Distillation

- arXiv: https://arxiv.org/abs/2607.15161v1
- arXiv ID: 2607.15161

---

🔍 **核心问题**：现有强化学习中的同策略蒸馏方法直接模仿教师模型的输出分布，未能有效区分教师模型在推理调优前后产生的关键变化，导致推理能力迁移效率较低。

🛠 **方法**：提出“Delta信号”作为新的蒸馏奖励，即教师模型与其基础模型（未经过推理指令调优的版本）在输出分布上的差异，并基于此设计同策略Delta蒸馏（OPD²）方法。

✨ **主要贡献**：通过数学、科学和代码推理基准实验证明，OPD²显著优于传统同策略蒸馏，使推理型大语言模型在仅需短时后训练的情况下即可达到强性能。
