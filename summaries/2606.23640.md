# Learning Process Rewards via Success Visitation Matching for Efficient RL

- arXiv: https://arxiv.org/abs/2606.23640v1
- arXiv ID: 2606.23640

---

🔍 **核心问题**：强化学习中稀疏奖励（仅任务完成时+1，其余为0）导致信用分配困难，训练效率低下。

🛠 **方法**：通过训练一个判别器区分历史成功与失败轨迹，生成密集过程奖励，激励当前策略匹配成功轨迹的状态-动作访问分布，同时避开失败轨迹。

✨ **主要贡献**：提出“成功访问匹配”方法，在不改变最优策略的前提下提供密集反馈，显著加速模拟和真实机器人操作任务的微调效率。
