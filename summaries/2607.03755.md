# EvoEye: Self-Evolving Runtime Monitoring for Autonomous Driving Systems

- arXiv: https://arxiv.org/abs/2607.03755v2
- arXiv ID: 2607.03755

---

🔍 **核心问题**：现有自动驾驶系统运行时监控器存在检测能力固定、依赖初始训练数据且预测误差较大的问题，无法自适应提升检测性能。

🛠 **方法**：提出EvoEye框架，通过FusionMonitor学习跨模块时序交互进行碰撞预测，并利用BlindSpotEvolver将当前预测误差转化为搜索引导，结合密度感知变异生成信息丰富的执行场景，实现监控器的自我进化。

✨ **主要贡献**：在Baidu Apollo与CARLA仿真中，FusionMonitor在0.05假阳性率下帧级召回率提升最高37.8个百分点，预警中位时间2.8-4.2秒；BlindSpotEvolver在相同预算下对遗漏危险场景的F1分数比均匀采样和违规导向采样最高提升13.2点。
