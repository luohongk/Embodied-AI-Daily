# Learning More from Less: Reinforcement Learning from Hindsight

- arXiv: https://arxiv.org/abs/2607.09042v1
- arXiv ID: 2607.09042

---

🔍 **核心问题**：强化学习后训练视觉-语言-动作模型时，机器人交互数据采集成本高、效率低，且稀疏奖励导致早期训练中失败轨迹无法提供有效学习信号。

🛠 **方法**：提出“事后学习”（LfH）方法，通过视觉-语言模型为失败轨迹重新标注事后指令和奖励，将失败转化为其他任务的成功，并与原始轨迹联合训练策略。

✨ **主要贡献**：在LIBERO-PRO任务上实现5倍样本效率提升，超越密集进度奖励基线，并在不同VLA骨干网络和实体Franka机器人上验证了有效性。
