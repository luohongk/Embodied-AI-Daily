# GeoProp: Grounding Robot State in Vision for Generalist Manipulation

- arXiv: https://arxiv.org/abs/2607.07101v1
- arXiv ID: 2607.07101

---

🔍 **核心问题**：机器人操作中，本体感知（如关节角度）常被孤立处理，缺乏与视觉特征的显式对齐，导致策略难以将机器人状态与场景关联，性能甚至不如纯视觉方法。

🛠 **方法**：提出GeoProp，一种轻量级即插即用适配器，通过几何投影将机器人状态映射到图像平面，采样局部视觉特征构建接地状态令牌，并利用FiLM调制注入空间先验；同时基于短期运动学预测坐标采样特征，提供前瞻视觉上下文。

✨ **主要贡献**：在67个任务中，GeoProp使Diffusion Policy提升8.7%、pi_0提升4.0%，真实世界平均增益10.6%，仅增加2-3%参数量，证明其作为通用具身策略的简单高效归纳偏置。
