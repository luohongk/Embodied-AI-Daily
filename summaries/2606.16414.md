# Instance-Aware Knowledge Distillation for Semi-Supervised Learning of an On-Board Multi-Task Dense Prediction Model for Collision Avoidance System

- arXiv: https://arxiv.org/abs/2606.16414v1
- arXiv ID: 2606.16414

---

🔍 **核心问题**：在计算资源受限的边缘环境（如乡村俱乐部）中，如何以低标注成本部署基于摄像头的碰撞避免系统，并实现多任务密集预测的实时运行。

🛠 **方法**：提出实例感知知识蒸馏框架，利用教师模型的领域先验和基础模型的实例中心知识生成伪标签，以减轻教师偏差，并通过半监督学习训练轻量级学生模型，部署于碰撞避免系统中执行多任务密集预测。

✨ **主要贡献**：学生模型在实例分割上超越大型教师模型，同时缓解了单目深度估计的性能下降；相比教师模型，计算量降低22.68倍，参数量减少14.33倍，在低成本边缘设备上达到6.46 FPS，并通过构建大规模乡村俱乐部数据集和现场验证证明了系统有效性。
