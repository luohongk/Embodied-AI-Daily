# Real-Time Source-Free Object Detection

- arXiv: https://arxiv.org/abs/2606.31834v1
- arXiv ID: 2606.31834

---

🔍 **核心问题**：现有无源目标检测（SFOD）方法在自动驾驶、监控等实时场景中，因依赖重型架构而牺牲了速度与内存效率，无法同时满足高精度与低延迟需求。

🛠 **方法**：基于无NMS的双头检测器YOLOv10，提出RT-SFOD方法。通过双头伪标签融合（DHF）策略选择性整合单头与多头预测，以及多尺度自适应表示多样化（MARD）损失函数，增强特征判别性，且仅在训练阶段引入额外模块。

✨ **主要贡献**：在域迁移基准上，相比现有最优SFOD方法，mAP提升1.4-3.5%，吞吐量提高1.3倍，参数量减少约2倍，首次在速度、精度与模型大小间实现更优的帕累托前沿，并验证了在多种双头检测器上的泛化性。
