# GenerativeMPC: VLM-RAG-guided Whole-Body MPC with Virtual Impedance for Bimanual Mobile Manipulation

- arXiv: https://arxiv.org/abs/2604.19522v1
- arXiv ID: 2604.19522

---

🔍 **核心问题**：解决双臂移动机器人在人机协作中，高层语义推理与安全、柔顺物理交互之间的脱节问题，即端到端模型不透明且经典控制器缺乏上下文感知能力。

🛠 **方法**：提出GenerativeMPC框架，采用视觉-语言模型结合检索增强生成（VLM-RAG）将视觉和语言上下文转化为动态速度限制和安全边界等控制约束，并调制虚拟阻抗参数，通过全身模型预测控制器（MPC）和统一阻抗-导纳控制器实现上下文感知的柔顺控制。

✨ **主要贡献**：首次将VLM-RAG与全身MPC及虚拟阻抗控制显式结合，实现语义到物理参数的可解释映射；在仿真和真实平台上验证了靠近人类时速度降低60%，并实现了安全、社交感知的导航与操作，无需重新训练模型。
