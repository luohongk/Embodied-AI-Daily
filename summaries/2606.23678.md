# AIR: Adaptive Interleaved Reasoning with Code in MLLMs

- arXiv: https://arxiv.org/abs/2606.23678v1
- arXiv ID: 2606.23678

---

🔍 **核心问题**：现有多模态大模型在视觉任务中依赖预定义规则，无法处理需要代码辅助的复杂数值计算问题。  
🛠 **方法**：提出AIR框架，包含两阶段冷启动数据构建、强化学习数据过滤策略，以及基于组约束奖励函数的自适应工具调用机制。  
✨ **主要贡献**：通过强化学习训练，模型在评估基准上平均提升6.1个百分点，交错推理样本准确率提升9.9个百分点，工具调用成功率超95%。
