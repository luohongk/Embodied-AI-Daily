# From Pixels to Shelf: An Integrated Robotic System for Autonomous Supermarket Stocking with a Mobile Manipulator

- arXiv: https://arxiv.org/abs/2509.11740v2
- arXiv ID: 2509.11740

---

🔍 **核心问题**：解决超市环境中机器人自主补货面临的动态人机交互、空间受限及商品形状多样等挑战。  
🛠 **方法**：采用模块化系统架构，整合商用硬件与ROS2框架，利用行为树（BTs）规划任务、微调视觉模型检测物体，并通过基于ArUco标记的两步模型预测控制（MPC）实现精准货架导航。  
✨ **主要贡献**：在700多次补货实验中实现超98%的抓取成功率，但对比表明当前系统在性能和成本效益上仍逊于人类工人，明确了商业化部署前需改进的关键方向。
