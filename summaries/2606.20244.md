# SPOT-E: Test-Time Entropy Shaping with Visual Spotlights for Frozen VLMs

- arXiv: https://arxiv.org/abs/2606.20244v1
- arXiv ID: 2606.20244

---

🔍 **核心问题**：视觉语言模型（VLM）在处理需要密集视觉证据的任务时，常因关键视觉线索微小、局部化而难以被有效提取，导致推理能力虽完整但证据读取失败。

🛠 **方法**：提出SPOT-E方法，利用答案跨度预测熵作为模型内部反馈信号，通过低熵锚点和熵塑造目标减少答案不确定性，并基于GRPO进行轻量级逐实例优化，生成问题条件化的视觉聚光灯。

✨ **主要贡献**：首次将熵塑造与视觉聚光灯结合，实现无需重训练的测试时干预，在多个基准和不同VLM家族上取得一致性能提升，并增强了对视觉噪声的鲁棒性。
