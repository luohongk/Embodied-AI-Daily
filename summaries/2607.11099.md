# Desc++: Efficient Descriptor Enhancement for Data Association in Existing Visual SLAM Systems

- arXiv: https://arxiv.org/abs/2607.11099v1
- arXiv ID: 2607.11099

---

🔍 **核心问题**：现有视觉SLAM系统在光照和视角变化下，手工设计的描述子匹配质量下降，而基于学习的方法虽能改善但计算开销大，且现有描述子增强方法因上下文建模能力有限而匹配精度不足。

🛠 **方法**：提出Desc++轻量级增强模块，通过混合架构联合编码描述子表示与关键点几何信息，结合全局注意力（顺序无关）和几何感知序列建模（线性时间），在不改变原始描述子维度和匹配接口的前提下提升性能。

✨ **主要贡献**：在描述子匹配、对应分析和四个不同V-SLAM系统的基准测试中，Desc++相比现有最优增强方法显著提升匹配精度，并转化为更准确稳定的轨迹估计，同时在实际实时SLAM流水线中实现了精度与效率的良好平衡。
