# LLM agent safety, multi-turn red-teaming, jailbreak benchmarks, adversarial robustness, safety-critical systems

- arXiv: https://arxiv.org/abs/2606.20408v1
- arXiv ID: 2606.20408

---

🔍 **核心问题**：大型语言模型（LLM）代理在安全关键系统中作为监督组件时，面对持续自适应对抗攻击的鲁棒性尚未得到充分研究。

🛠 **方法**：提出NRT-Bench基准，在模拟核电站控制室中构建多轮红队攻击场景，由五个LLM代理组成操作团队，通过四个信道注入对抗消息，并以安全功能丧失作为客观危害信号。

✨ **主要贡献**：发现自适应多轮攻击可可靠地使操作团队越过安全极限（8.7%-12.1%攻击成功），且不同模型的漏洞几乎不重叠；防御效果高度依赖模型，同一防御可能对某模型有效却对另一模型有害。
