# HERB: Human-augmented Efficient Reinforcement learning for Bin-packing

- arXiv: https://arxiv.org/abs/2504.16595v2
- arXiv ID: 2504.16595

---

🔍 **核心问题**：解决高度多样化的三维物体（如家庭或杂货物品）的装箱问题，现有方法（如启发式或纯强化学习）训练时间长、延迟高，且忽略物体特性（如易碎性、可变形性）和人类偏好。

🛠 **方法**：提出HERB框架，一种结合人类演示的强化学习方法，利用人类专家展示的装箱策略（隐含空间优化、稳定性等难以显式建模的因素），并与强化学习探索相结合，以确定每个物体在容器中的放置位置。

✨ **主要贡献**：首次探索从人类演示中学习以解决不规则物体装箱任务；在装箱效率和延迟上优于启发式、纯强化学习和模仿学习方法；生成的装箱策略更稳定、更接近人类布局，并在机器人系统上验证了实际可行性。
