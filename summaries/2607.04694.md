# Solve the Missing First Step: Can VLMs Standardize Raw Heterogeneous Medical Data?

- arXiv: https://arxiv.org/abs/2607.04694v1
- arXiv ID: 2607.04694

---

🔍 **核心问题**：现有医疗AI基准测试假设数据已标准化，但真实临床中原始医疗数据格式混乱、来源异构，导致视觉语言模型（VLM）无法直接应用。本文聚焦这一被忽视的“缺失步骤”——原始医疗数据的标准化。

🛠 **方法**：构建医疗数据标准化基准（MDS-Bench），包含1,939个手动标注的原始数据标准化任务，要求VLM识别源格式、转换图像、提取文本并生成结构化图文对，以评估其端到端处理能力。

✨ **主要贡献**：揭示了原始数据标准化是医疗AI临床落地的关键瓶颈——即使最佳模型（Gemini 3 Flash）的端到端成功率也仅达48.6%，为未来研究指明了方向。
