# When Prompts Ignore Structure: Graph-Based Attribute Reasoning for Calibrated VLMs

- arXiv: https://arxiv.org/abs/2607.07395v1
- arXiv ID: 2607.07395

---

🔍 **核心问题**：视觉-语言模型（VLM）在测试时自适应中，基于熵的提示调优虽提升零样本准确率，却因过度自信导致校准退化，且现有方法忽略属性间的结构关系。

🛠 **方法**：提出ARGTCA框架，将（类别，属性）对构建为符号属性图节点，利用图注意力网络（GAT）通过对比学习生成结构感知嵌入，并设计两种属性选择策略：ARGTCA-DIV（类内多样性）和ARGTCA-DISC（类间区分性）。

✨ **主要贡献**：在9个基准上，ARGTCA-DIV将平均期望校准误差（ECE）降低约37%，ARGTCA-DISC降低约17%，证明建模属性符号交互能有效提升VLM测试时自适应的可靠性。
