# LPCAN: Lightweight Pyramid Cross-Attention Network for Rail Surface Defect Detection Using RGB-D Data

- arXiv: https://arxiv.org/abs/2601.09118v2
- arXiv ID: 2601.09118

---

🔍 **核心问题**：现有基于视觉的钢轨缺陷检测方法存在计算复杂度高、参数过多和精度不足的问题，难以兼顾效率与准确性。

🛠 **方法**：提出轻量级金字塔交叉注意力网络（LPCANet），以MobileNetv2为骨干提取RGB特征，结合轻量金字塔模块（LPM）处理深度数据，通过交叉注意力机制（CAM）进行多模态融合，并引入空间特征提取器（SFE）增强结构分析。

✨ **主要贡献**：在三个无监督RGB-D钢轨数据集上，LPCANet仅用990万参数、2.50 G FLOPs和162.60 fps推理速度即达到最优性能，相比最佳基线提升S_α 1.48%、IOU 0.86%、MAE 1.77%，并在非钢轨数据集上验证了泛化能力。
