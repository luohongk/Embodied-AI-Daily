# Learning More With Less: Sample Efficient Model-Based RL for Loco-Manipulation

- arXiv: https://arxiv.org/abs/2501.10499v3
- arXiv ID: 2501.10499

---

🔍 **核心问题**：解决四足机器人（如波士顿动力Spot）在全身操控任务中，因系统复杂性和商业平台黑箱特性导致的动力学模型难以精确建模、控制策略鲁棒性不足的问题。

🛠 **方法**：提出基于模型的强化学习框架，先构建平台的手工运动学模型作为物理先验，再结合贝叶斯神经网络从少量数据中高效学习精确的动力学模型，最后利用该模型通过强化学习推导操控控制策略。

✨ **主要贡献**：在低数据量条件下仍能实现高精度的动态末端执行器轨迹跟踪，并在波士顿动力Spot平台上验证了方法的有效性，显著提升了样本效率。
