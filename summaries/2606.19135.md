# A Technical Taxonomy of LLM Agent Communication Protocols

- arXiv: https://arxiv.org/abs/2606.19135v1
- arXiv ID: 2606.19135

---

🔍 **核心问题**：当前大语言模型（LLM）智能体通信协议种类繁多且碎片化，缺乏统一分类标准，导致多智能体系统面临互操作性挑战。

🛠 **方法**：采用迭代式分类法，经过五次迭代（三次从经验到概念、两次从概念到经验），对九个活跃维护的开源协议进行系统分析，构建了包含五个维度的技术分类体系。

✨ **主要贡献**：提出了覆盖对手方、载荷、交互状态、发现机制和模式灵活性的五维分类框架，揭示了智能体协议中混合载荷与会话持久性等共性模式，并预测了协议向分层联邦架构演进的趋势。
