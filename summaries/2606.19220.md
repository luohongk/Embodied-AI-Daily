# Machine Unlearning for the XGBoost Model with Network Intrusion Datasets

- arXiv: https://arxiv.org/abs/2606.19220v1
- arXiv ID: 2606.19220

---

🔍 **核心问题**：现有机器学习遗忘（MU）研究主要针对深度学习和图像数据，缺乏对基于表格数据的网络入侵检测场景的支持，尤其是XGBoost模型的遗忘方法。

🛠 **方法**：提出XGBoost-Forget方法，针对XGBoost模型设计遗忘机制，在IoT-23和GeNIS两个网络入侵表格数据集上，通过多项指标评估模型性能、遗忘效率与遗忘质量。

✨ **主要贡献**：XGBoost-Forget在保持与原始模型相近预测性能的同时，显著提升了遗忘速度，验证了其在表格型网络入侵数据中实现高效遗忘的潜力。
