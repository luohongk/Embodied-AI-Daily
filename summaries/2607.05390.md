# Deform360: A Massive Multi-view Visuotactile Dataset for Deformable World Models

- arXiv: https://arxiv.org/abs/2607.05390v1
- arXiv ID: 2607.05390

---

🔍 **核心问题**：机器人操作中可变形物体的动力学建模面临高维状态空间和复杂材料特性的挑战，现有2D像素空间与3D几何空间的世界模型缺乏大规模真实数据的系统性对比。

🛠 **方法**：构建包含198个日常物体、1980个交互序列及超215小时观测数据的Deform360多模态数据集，通过41个环绕相机和双手触觉夹爪采集全局运动与局部形变，并采用无标记视觉-触觉3D追踪管道提取密集几何与运动信息。

✨ **主要贡献**：系统评估了2D视频模型与3D粒子模型的性能差异，揭示了结构先验与可扩展性之间的权衡，并通过机器人规划任务验证了数据集的实际应用价值，为可变形物体世界建模提供了基准。
