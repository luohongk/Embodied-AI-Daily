# Kolmogorov Regression for Robust Diffusion Policies

- arXiv: https://arxiv.org/abs/2606.18186v1
- arXiv ID: 2606.18186

---

🔍 **核心问题**：有限维扩散策略因离散化伪影导致时间漂移，影响长期部署性能。  
🛠 **方法**：引入向后Kolmogorov方程将扩散策略提升至Cameron-Martin空间，用确定性边界值PDE替代随机分数匹配，并设计精度加权的Cameron-Martin损失与Kolmogorov残差诊断。  
✨ **主要贡献**：实现收敛性保证（依赖核有效秩而非动作维度）、通过谱加权提升轨迹规则性、无需奖励信号的确定性故障检测；在PushT基准上最大回合奖励提升17%，在制造线控制中RMSE降低28.4%，死锁事件减少96%。
