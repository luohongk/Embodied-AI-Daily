# Gold Points Sniper: Self-guided Visual Reasoning in VLM for Fine-grained Action Understanding

- arXiv: https://arxiv.org/abs/2606.22409v1
- arXiv ID: 2606.22409

---

🔍 **核心问题**：现有机器人系统难以从宽视角图像中准确理解人类细粒度动作、意图和上下文线索，而当前视觉语言模型（VLM）在信息丰富性与事实准确性之间存在权衡，无法满足可靠的人机交互需求。

🛠 **方法**：提出Gold Points Sniper（GPS）框架，包含三个模块：Gold Points Extractor（提取关键动作细节）、Selective Socratic Questioner（通过选择性自问验证并优化细节）、Semantic Entailment Evaluator（利用语义蕴含分类量化事实一致性），从而赋予轻量级VLM自引导的多模态推理能力。

✨ **主要贡献**：在CAP基准数据集上，GPS增强的轻量级VLM在细粒度动作理解任务中性能显著提升，部分模型达到与GPT-4o相当的水平，同时保持更高的事实准确性，为家庭机器人安全理解人类行为提供了可靠基础。
