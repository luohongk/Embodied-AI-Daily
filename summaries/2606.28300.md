# CacheMPC: Certified Cached Model Predictive Control for Quadruped Locomotion

- arXiv: https://arxiv.org/abs/2606.28300v1
- arXiv ID: 2606.28300

---

🔍 **核心问题**：四足机器人分层控制器中，模型预测控制（MPC）每周期求解二次规划（QP）耗时过长，限制了嵌入式处理器的更新速率。

🛠 **方法**：提出Certified CacheMPC，通过局部敏感哈希缓存地平线接触力轨迹，按接触模式分区，查询时仅接受满足后验可行性证书和拉格朗日对偶间隙上界的缓存解，并采用结合Top-K检索、截止时间QP求解和最后认证回退的有界预算控制器调度。

✨ **主要贡献**：在Unitree Go2机器人上，缓存方案在仿真中实现25倍中位求解加速，硬件上达18.7倍，且闭环稳定率与无缓存基线无显著差异，但证书对闭环安全性的贡献在当前样本量下未得到验证。
