# Feeling the Unexpected: ResTacVLA for Contact-Rich Manipulation via Residual Tactile Representation

- arXiv: https://arxiv.org/abs/2607.03387v2
- arXiv ID: 2607.03387

---

🔍 **核心问题**：在接触丰富的机器人操作任务中，将触觉感知集成到视觉-语言-动作（VLA）模型时，高带宽的视觉特征常压制稀疏的触觉信号，导致模态坍塌，无法有效利用触觉信息。

🛠 **方法**：提出ResTacVLA，受预测编码启发，将触觉数据重构为残差触觉表示，即视觉先验与物理感觉之间的差异；通过向量量化（VQ）瓶颈将残差离散化为潜在接触基元，并利用视觉先验的不确定性自适应门控触觉集成，优先在视觉不可靠阶段引入触觉残差。

✨ **主要贡献**：该方法在多种接触密集操作任务中持续优于所有基线，并对意外动态干扰保持鲁棒性，有效解决了视觉主导下的触觉信息利用问题。
