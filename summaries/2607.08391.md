# On Exploring Input Resolution Scaling For Anytime LiDAR Object Detection

- arXiv: https://arxiv.org/abs/2607.08391v1
- arXiv ID: 2607.08391

---

🔍 **核心问题**：如何在LiDAR点云3D目标检测中实现动态计算延迟与结果质量的权衡（即任意时间计算），以适应自动驾驶系统实时性要求的变化。

🛠 **方法**：提出一种支持多分辨率输入推理的单一DNN模型，通过将点云处理为柱体或体素实现动态分辨率缩放；并设计一个基于执行时间预测的截止时间感知调度器，在运行时选择最高可行分辨率。

✨ **主要贡献**：仅需部署单个模型即可实现多分辨率推理，避免多模型冗余；在nuScenes数据集上显著优于现有任意时间计算方法；在模拟自动驾驶系统中实现无碰撞导航，同时避免因环境复杂导致的不必要停车。
