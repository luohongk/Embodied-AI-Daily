# GGD-SLAM: Monocular 3DGS SLAM Powered by Generalizable Motion Model for Dynamic Environments

- arXiv: https://arxiv.org/abs/2604.12837v1
- arXiv ID: 2604.12837

---

🔍 **核心问题**：解决单目视觉SLAM在动态环境中因运动物体干扰导致的定位精度下降和稠密地图重建失败问题。

🛠 **方法**：提出GGD-SLAM框架，通过先进先出队列管理帧序列，结合时序注意力机制提取动态语义特征，并利用动态特征增强器分离静态与动态成分；同时设计遮挡区域填充方法和自适应SSIM损失函数，减少动态干扰对静态建模的影响。

✨ **主要贡献**：无需预定义语义标注或深度输入，在真实动态数据集上实现了相机位姿估计与稠密重建的最优性能，显著提升了系统在动态场景下的鲁棒性。
