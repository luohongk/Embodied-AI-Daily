# An Intelligent-Cloud Edge Multimodal Interaction System for Robots

- arXiv: https://arxiv.org/abs/2607.14675v1
- arXiv ID: 2607.14675

---

🔍 **核心问题**：在计算资源受限的机器人平台上，实现复杂环境下的鲁棒人机交互，包括准确的手势感知、场景理解和任务规划。

🛠 **方法**：提出云-边多模态交互框架，集成改进的YOLO手势检测器（加入CBAM注意力模块和DIoU损失函数）与LLM/VLM智能体；云端负责检测、融合与规划，TonyPi机器人本地处理数据采集与执行。

✨ **主要贡献**：手势检测在公开和自定义数据集上分别达到98.9%和95.0%的精度，系统任务成功率最高达95%；30人评估平均满意度3.69/5，验证了轻量级手势检测与多模态智能体结合在资源受限机器人上的可行性。
