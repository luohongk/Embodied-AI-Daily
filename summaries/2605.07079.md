# Learning Visual Feature-Based World Models via Residual Latent Action

- arXiv: https://arxiv.org/abs/2605.07079v1
- arXiv ID: 2605.07079

---

## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人学习中的视觉特征基世界模型（Visual Feature-Based World Models），用于未来状态预测和策略学习。
- **问题（Problem）**：现有世界模型主要依赖视频生成（如扩散模型），计算开销大、易产生幻觉；而基于视觉特征（如DINO token）的直接回归方法在复杂3D交互中预测模糊或崩溃。此外，如何利用世界模型高效改进下游策略（如从无动作视频中学习、在模型内进行强化学习）仍是挑战。
- **方法（Method）**：提出**残差潜在动作（Residual Latent Action, RLA）**，一种从DINO token残差中学习的紧凑潜在表示。基于RLA，构建**RLA世界模型（RLA-WM）**，通过流匹配（Flow Matching）在紧凑的RLA空间中预测未来状态，而非直接预测高维特征。并基于此开发了两种机器人学习技术：最小化世界动作模型（WAM）和完全在RLA-WM内的视觉强化学习（WMRL）。
- **创新（Innovation）**：1）提出RLA，一种新颖的、从DINO残差中学习的潜在动作表示，具有预测充分性、泛化性和时间拓扑性。2）RLA-WM在紧凑的RLA空间中进行流匹配，实现了高效且高保真的未来状态预测，显著优于现有特征基和视频扩散世界模型。3）首次展示了完全在从离线视频学习的特征基世界模型内进行视觉强化学习，无需在线交互或手工奖励。
- **意义（Significance）**：RLA-WM为机器人学习提供了一种高效、准确且可扩展的世界模型范式，降低了计算成本并减少了幻觉。其应用（WAM和WMRL）展示了从无动作视频中学习策略和进行无奖励的模型内强化学习的潜力，为构建更通用、数据高效的机器人学习系统铺平了道路。
- **一句话总结**：本文提出残差潜在动作（RLA）及其世界模型RLA-WM，通过在高维视觉特征空间中学习紧凑的动力学表示，实现了高效、准确的未来预测，并成功应用于从无动作视频学习策略和完全在模型内进行视觉强化学习。

## 1. 研究动机（Motivation）

当前世界模型的研究主要分为两大范式：基于视频生成的和基于视觉特征的。

1.  **基于视频生成的世界模型的局限性**：
    - **计算开销巨大**：视频扩散模型（如Vid2World）需要大量的计算资源（FLOPs高达1.1P），限制了其在实际机器人系统中的应用。
    - **易产生幻觉**：生成模型在预测未来帧时，可能会产生物理上不真实或与输入动作不一致的内容（hallucination），导致预测不可靠。
    - **下游应用受限**：由于计算和可靠性问题，这类模型主要局限于开环数据生成或特定任务的规划，难以用于高效的闭环策略学习。

2.  **基于视觉特征的世界模型的挑战**：
    - **直接回归导致模糊/崩溃**：现有方法（如DINO-WM）通过直接回归来预测未来帧的DINO token。虽然在简单2D任务中有效，但在复杂的3D交互中，这种回归到均值（regression-to-the-mean）的方法会导致预测结果模糊甚至完全崩溃。
    - **高维特征空间生成困难**：一个直观的解决方案是在特征空间中使用生成模型（如扩散模型）。然而，DINO token等视觉特征的维度远高于图像生成中使用的VAE潜变量（例如，512x512图像，VAE潜变量约16k维，而DINOv3-L token约1M维）。这种“维度灾难”使得在高维特征空间中进行生成建模极其困难，且会丧失特征基模型的计算效率优势。

因此，本文的核心动机是解决以下两个关键问题：
1.  如何开发一个既高效又准确、且能扩展到复杂3D操作的视觉特征基世界模型？
2.  如何利用这样的世界模型来有效改进下游的机器人策略学习？

作者认为，虽然视觉特征是高维的，但有效的物理状态转移的流形本质上是低维的。因此，学习一个紧凑的低维动力学表示是解决上述问题的关键。

## 2. 创新点（Innovation）

本文的核心创新在于提出了**残差潜在动作（Residual Latent Action, RLA）**，并基于此构建了高效的**RLA世界模型（RLA-WM）**。具体创新点如下：

1.  **提出残差潜在动作（RLA）**：
    - **新颖的表示**：RLA是一种从两帧DINO token的残差（$s_{t+h} - s_t$）中学习到的紧凑潜在向量。它通过一个简单的自编码器（编码器$f_{enc}$和解码器$f_{dec}$）进行训练，仅使用单一回归损失来重建$s_{t+h}$。
    - **三大关键特性**：
        - **预测充分性**：与以往仅作为弱条件标签的潜在动作不同，RLA解码器$f_{dec}$在给定当前状态$s_t$和RLA $z$后，能通过单次前向传播高保真地重建未来状态$s_{t+h}$，无需迭代生成。
        - **泛化性**：在任务无关数据上训练的RLA自编码器，可以泛化到未见过的场景和运动模式，用于任务相关的无动作视频。
        - **时间拓扑性**：RLA的潜在空间自然地编码了时间进程。对高斯噪声和RLA进行线性插值，解码后能得到近似于中间帧的结果。

2.  **提出RLA世界模型（RLA-WM）**：
    - **紧凑空间中的动力学学习**：RLA-WM不直接预测高维的$s_{t+h}$，而是预测紧凑的RLA $z$。这符合经典物理模拟中建模相对位移的思想，使模型专注于学习状态演化而非绝对状态。
    - **轻量级流匹配**：RLA-WM使用流匹配（Flow Matching）在紧凑的RLA空间中生成$z$。由于条件网络只需执行一次，且迭代生成过程在低维空间中进行，因此计算非常高效（FLOPs仅为3.5T，远低于视频扩散模型的1.1P）。

3.  **开发两种基于RLA-WM的机器人学习技术**：
    - **最小化世界动作模型（WAM）**：通过一个简单的线性层，将标准的行为克隆（BC）策略扩展为世界动作模型。该线性层预测RLA，使策略能从无动作视频中学习，且不增加推理成本。
    - **模型内视觉强化学习（WMRL）**：首次展示了完全在一个从离线视频学习的特征基世界模型内进行视觉强化学习。该方法无需在线交互、手工奖励或辅助BC损失，通过提出的视频对齐奖励（VAR）和PPO算法优化策略。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

本文的整体框架如图1所示，包含三个主要部分：

1.  **RLA自编码器学习（图1a）**：从任务无关的离线视频中学习RLA。输入是两帧DINO token $s_t$和$s_{t+h}$，计算残差$s_{t+h} - s_t$。该残差与可学习查询一起输入编码器$f_{enc}$，输出紧凑的RLA $z$。解码器$f_{dec}$接收$z$和$s_t$，重建$s_{t+h}$。训练目标是最小化重建的$s_{t+h}$与真实$s_{t+h}$之间的MSE损失。

2.  **RLA世界模型（RLA-WM）动力学学习（图1b & 图2）**：从离线视频和动作数据中学习动力学。给定当前状态$s_t$和动作序列$a_{t:t+h}$，RLA-WM首先通过一个条件网络（由MLP和自注意力层组成）处理这些信息，生成条件token。然后，在流匹配过程中，从高斯噪声$z_0 = \epsilon$开始，以条件token为引导，通过求解ODE逐步去噪得到预测的RLA $\hat{z}_1$。最后，解码器$f_{dec}$根据$s_t$和$\hat{z}_1$生成预测的未来状态$\hat{s}_{t+h}$。

3.  **应用（图1c）**：
    - **从无动作视频中学习**：将RLA自编码器固定，在BC策略网络上添加一个线性层来预测RLA。该线性层以策略网络提取的视觉特征为输入，输出预测的RLA $\hat{z}$，其目标值由RLA编码器从视频帧对$(s_t, s_{t+h})$中提取。
    - **RLA-WM内的视觉强化学习**：将预训练的BC策略作为初始化，添加LoRA适配器和残差动作头。策略输出动作块$a_{t:t+h}$给RLA-WM，RLA-WM预测未来状态$\hat{s}_{t+h}$。通过UNet解码$\hat{s}_{t+h}$为RGB图像，作为下一时刻的观测。奖励由视频对齐奖励（VAR）提供，即预测的DINO token与参考视频中真实DINO token的负L1距离。策略通过PPO在RLA-WM生成的轨迹上进行优化。

### 3.2 关键算法与数学推导

**1. RLA自编码器学习**

目标是从帧对$(s_t, s_{t+h})$中学习一个紧凑的潜在表示$z$，该表示编码了状态变化。

- **编码**：$z = f_{enc}(s_{t+h} - s_t)$，其中$f_{enc}$是一个包含自注意力的神经网络。
- **解码**：$\hat{s}_{t+h} = f_{dec}(z, s_t)$，其中$f_{dec}$也是一个神经网络。
- **损失函数**：$\mathcal{L}_{RLA} = \| \hat{s}_{t+h} - s_{t+h} \|_2^2$

**2. RLA世界模型（RLA-WM）中的流匹配**

RLA-WM的目标是学习一个条件概率路径，从噪声分布$p_0 = \mathcal{N}(0, I)$过渡到数据分布$p_1$（即RLA $z$的分布），条件为当前状态$s_t$和动作$a_{t:t+h}$。

- **条件网络**：$\text{cond} = g(s_t, a_{t:t+h})$，其中$g$是一个包含MLP和自注意力的网络。
- **流匹配训练**：在训练时，采样一个时间步$\tau \in [0, 1]$，噪声$\epsilon \sim \mathcal{N}(0, I)$，以及真实RLA $z$。构造带噪样本$z_\tau = \tau z + (1-\tau)\epsilon$。目标速度场为$v^* = z - \epsilon$。流匹配网络$v_\theta$以$z_\tau$和条件$\text{cond}$为输入，预测速度$\hat{v} = v_\theta(z_\tau, \text{cond})$。
- **损失函数**：$\mathcal{L}_{FM} = \mathbb{E}_{\tau, \epsilon, z} \left[ \| v_\theta(z_\tau, \text{cond}) - (z - \epsilon) \|_2^2 \right]$
- **推理（生成）**：从$z_0 = \epsilon \sim \mathcal{N}(0, I)$开始，使用欧拉法求解ODE：
  $$z_{\tau + \Delta\tau} = z_\tau + \Delta\tau \cdot v_\theta(z_\tau, \text{cond})$$
  从$\tau=0$迭代到$\tau=1$，得到最终的预测RLA $z_1$。然后通过解码器得到预测的未来状态：$\hat{s}_{t+h} = f_{dec}(z_1, s_t)$。

**3. 视频对齐奖励（VAR）**

用于WMRL，奖励函数定义为预测的DINO token与参考视频中真实DINO token的负L1距离：
$$r_t = -\| \hat{s}_t - s_t \|_1$$
其中$\hat{s}_t$是RLA-WM在时间步$t$预测的DINO token，$s_t$是参考离线视频中对应时间步的真实DINO token。

### 3.3 模型、训练与数据集

- **模型架构**：
    - **RLA自编码器**：主要由自注意力层构成。编码器$f_{enc}$将DINO残差和可学习查询映射到低维RLA空间。解码器$f_{dec}$将RLA和当前状态$s_t$映射回未来状态$\hat{s}_{t+h}$。
    - **RLA-WM**：包含一个条件网络（MLP + 自注意力）和一个流匹配网络。条件网络处理$s_t$和$a_{t:t+h}$。流匹配网络在RLA空间中进行迭代生成。
    - **WAM**：基于ResNet-18的BC策略，额外添加一个线性层来预测RLA。
    - **WMRL策略**：基于预训练的BC-ResNet，添加LoRA适配器和残差动作头（预测高斯动作分布的均值和标准差）。

- **训练**：
    - **RLA自编码器**：在任务无关的“play”数据上训练，使用MSE损失。
    - **RLA-WM**：在任务相关的离线视频和动作数据上训练。训练时随机采样帧对$(s_t, s_{t+h})$，其中$h \in [1, 15]$。使用流匹配损失训练。
    - **WAM**：联合训练BC损失（对有动作的视频）和RLA预测损失（对所有视频）。对于无动作视频，动作损失被屏蔽，仅使用RLA损失训练共享骨干网络。
    - **WMRL**：使用PPO算法在RLA-WM生成的轨迹上优化策略。奖励由VAR提供。

- **数据集**：
    - **ManiSkill**：模拟环境，包含5个任务（Pull Cube, Pull Cube with Tool, Roll Ball, Push T, Poke Cube）和3种机器人（Panda, XArm, UR10e）。收集了每个任务1000个成功和500个失败轨迹，以及每个机器人3000个任务无关的play视频。
    - **IWS**：真实世界数据集，包含3个任务（Push T, Rope Manipulation, Open Box），使用双臂ALOHA机器人，每个任务超过600个人类遥操作演示。

### 3.4 实验设置与结果

**实验1：未来帧预测质量评估（表1 & 图3）**

- **设置**：在ManiSkill和IWS数据集上，给定初始帧和动作块，模型自回归地预测未来30步（ManiSkill）或60步（IWS）。评估指标包括LPIPS（越低越好）、SSIM（越高越好）和DINO L1距离（越低越好），以及计算开销FLOPs。
- **基线**：DINO-WM（直接回归）、RAE（DINO token扩散）、FM-WM（DINO token流匹配）、Vid2World（视频扩散）。
- **结果**：
    - RLA-WM在所有指标上显著优于所有基线，包括特征基和视频扩散模型。
    - 视频扩散模型Vid2World虽然视觉上清晰，但存在幻觉，物理预测不准确，且计算开销巨大（1.1P FLOPs）。
    - RLA-WM在保持高保真预测的同时，计算效率极高（3.5T FLOPs），仅次于直接回归的DINO-WM。
    - 定性结果（图3）显示，DINO-WM预测模糊，RAE和FM-WM效果差，Vid2World产生幻觉，而RLA-WM的预测最接近真实情况。

**实验2：从无动作视频中学习（表2）**

- **设置**：在ManiSkill上，仅使用5%的视频包含动作标签（PushT为15%），其余为无动作视频。训练一个WAM，评估其动作头的成功率。
- **基线**：BC-ResNet（仅使用有动作视频）、DINO CLS、UniVLA、AdaWorld（使用不同潜在动作表示）。
- **结果**：RLA作为潜在动作表示，在所有任务上取得了最高的平均成功率（35.6%），显著优于BC基线（27.2%）和第二好的AdaWorld（33.7%）。在最具挑战性的Push-T任务上，RLA的提升最大（15.2% vs 3.6%）。

**实验3：RLA世界模型内的视觉强化学习（图6 & 表3）**

- **设置**：在ManiSkill上，对预训练的BC策略应用WMRL。在RLA-WM内进行15次独立运行（种子1-15），每次2400步PPO更新。评估最终策略在50个标准种子（42-91）上的成功率。
- **基线**：BC-ResNet。
- **结果**：
    - 图6显示，WMRL在所有5个任务上都一致地提升了最佳BC模型的性能。
    - 表3显示，在1500个种子的更大规模评估中，WMRL在XArm和UR10e机器人上取得了显著的性能提升。虽然在Panda机器人上略有下降，但平均提升了+1.1%。这表明WMRL能够在不使用额外数据的情况下，通过额外的计算来改进BC策略。

### 3.5 技术细节

- **DINO特征**：使用DINOv3-L模型提取patch tokens，维度为$L \times C$，其中$L$是patch数量，$C=1024$是特征维度。
- **动作嵌入**：机器人动作$a_{t:t+h}$通过MLP嵌入，并与DINO token $s_t$和可学习查询拼接。
- **流匹配**：使用欧拉法求解ODE，步长$\Delta\tau$固定。条件网络只执行一次，迭代生成在紧凑的RLA空间（维度远低于DINO token空间）中进行。
- **UNet解码**：为了计算图像空间指标（LPIPS, SSIM），需要使用一个预训练的UNet将预测的DINO token
