# C2E: Boosting Ego-Only 3D Object Detection via Multi-Teacher Contrastive Knowledge Distillation

- arXiv: https://arxiv.org/abs/2607.01827v1
- arXiv ID: 2607.01827

---

🔍 **核心问题**：解决传统单车感知（Eo-Perception）在复杂户外环境中因视角受限和遮挡导致的3D目标检测性能瓶颈，同时避免多车协同感知（Co-Perception）的高通信成本和位姿误差问题。

🛠 **方法**：提出C2E范式下的多教师对比知识蒸馏框架（M2S），包含多级特征增强模块、辅助点云重建机制和多教师对比蒸馏，将协同感知的优越性能迁移至单车感知模型。

✨ **主要贡献**：在V2XSet等数据集上，M2S框架与先进检测器结合后，3D mAP提升最高达8.64%，且无需引入额外通信成本，有效兼顾协同感知的高性能与单车感知的低开销。
