# CL-CLIP: CLIP-Based Continual Learning Framework with Cost-Volume Category Decoupling for Object Detection

- arXiv: https://arxiv.org/abs/2606.06978v1
- arXiv ID: 2606.06978

---

🔍 **核心问题**：解决持续目标检测（COD）中，基于CLIP的开放词汇检测器在持续学习新类别时出现的灾难性遗忘问题，即无法同时保留旧类别检测能力并适应新类别。

🛠 **方法**：提出CL-CLIP框架，通过计算CLIP图像-文本相似度成本体积（cost volume），生成密集的类别响应图，将共享区域特征解耦为类别特定路径，并利用多专家RoI头（Multi-Expert RoI head）处理这些解耦特征，实现持续学习。

✨ **主要贡献**：在PASCAL VOC和MS-COCO数据集上，CL-CLIP显著提升了F-ViT基线在持续微调下的性能，与现有COD方法相比，在适应新类别的同时保持了有竞争力的基类检测能力。
