# Hyperdimensional Probe: Decoding LLM Representations via Vector Symbolic Architectures

- arXiv: https://arxiv.org/abs/2509.25045v3
- arXiv ID: 2509.25045

---

🔍 **核心问题**：现有大语言模型（LLM）可解释性方法难以同时整合输入特征提取（如监督探针、稀疏自编码器）与输出分布分析（如logit方法），导致对模型内部向量空间的语义理解不完整。

🛠 **方法**：提出超维探针（Hyperdimensional Probe），一种结合向量符号架构（VSA）与超向量代数的混合监督探针，将监督探针的顶层可解释性、稀疏自编码器的稀疏代理空间以及logit导向分析统一在一个框架中。

✨ **主要贡献**：该方法在输入补全和问答文本生成任务中，能跨不同LLM和配置一致提取有意义的语义信息，克服了logit分析受限于词汇表的缺陷，并缓解了稀疏自编码器在有限概念特征空间中的噪声问题，实现了输入-输出特征的联合分析。
