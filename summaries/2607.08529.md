# Log-Insight: Automating Microservice Incident Diagnosis via Neuro-Symbolic Log Analysis

- arXiv: https://arxiv.org/abs/2607.08529v1
- arXiv ID: 2607.08529

---

🔍 **核心问题**：大规模微服务系统生产环境故障诊断中，日志数据量远超大语言模型上下文窗口（如30分钟内产生12亿字符），现有方法（模板解析、深度学习检测、LLM管道）存在语义推理缺失、黑盒信号或上下文溢出等问题。

🛠 **方法**：提出Log-Insight系统，模拟SRE手动排查流程，通过符号化阶段（采样、模式理解、聚类、统计异常排序）将原始日志压缩1000-7000倍，再将精简证据摘要输入LLM生成假设报告。

✨ **主要贡献**：在11个历史生产事故中（110次运行），MRR达0.790，90%以上运行将正确根因排在前3位，延迟低于1分钟；其“法医证据”模块（精确日志模板与偏差统计）被运维人员视为关键采纳因素，使系统从黑盒工具转变为可交互的调查助手。
