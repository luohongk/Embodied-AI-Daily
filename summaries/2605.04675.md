# Physical Adversarial Clothing Evades Visible-Thermal Detectors via Non-Overlapping RGB-T Pattern

- arXiv: https://arxiv.org/abs/2605.04675v1
- arXiv ID: 2605.04675

---

🔍 **核心问题**：现有研究忽视了可见-热红外（RGB-T）多模态目标检测器在物理世界中的安全性，尤其是针对人体目标的对抗攻击方法缺失。

🛠 **方法**：提出非重叠RGB-T图案（NORP）的对抗服装，通过空间离散-连续优化（SDCO）方法优化图案，并利用3D RGB-T人体模型模拟全视角攻击。

✨ **主要贡献**：首次实现物理世界中对RGB-T检测器的全视角对抗攻击，NORP设计避免了重叠图案的光衰减问题，且融合阶段集成方法增强了攻击在不同融合架构检测器间的可迁移性。
