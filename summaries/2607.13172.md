# Learning Safe Agent Behaviour from Human Preferences and Justifications via World Models

- arXiv: https://arxiv.org/abs/2607.13172v1
- arXiv ID: 2607.13172

---

🔍 **核心问题**：在环境动态未知且缺乏合适奖励函数的安全关键场景中，如何安全地训练和部署智能体策略。

🛠 **方法**：提出DROPJ方法，先利用真实轨迹数据学习世界模型（模拟器），让人类在模拟器中生成信息丰富的轨迹；然后让人类对轨迹片段进行偏好比较并提供安全理由，据此训练奖励模型；最终结合世界模型与模型预测控制直接部署智能体。

✨ **主要贡献**：通过人类在模拟器中生成轨迹显著降低训练计算成本；偏好反馈比其它反馈类型更能提升部署性能；偏好附带的安全理由可有效增强部署时的安全性或优先满足用户指定的安全要求。
