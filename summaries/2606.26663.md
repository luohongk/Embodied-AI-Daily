# Tactile-WAM: Touch-Aware World Action Model with Tactile Asymmetric Attention

- arXiv: https://arxiv.org/abs/2606.26663v1
- arXiv ID: 2606.26663

---

🔍 **核心问题**：在接触丰富的机器人操作任务中，仅依赖视觉信息无法完整预测物理交互（如滑动、卡滞、接触法向等），而直接注入触觉令牌会导致“触觉污染”，破坏视频和动作预测的准确性。

🛠 **方法**：提出Tactile-WAM，包含触觉非对称注意力机制（TAAM），通过VideoClean掩码阻止视频查询访问触觉键/值令牌但保留动作查询访问，并引入基于预测触觉变化的触觉感知偏置来调节动作注意力。

✨ **主要贡献**：在ManiFeel基准上，整体平均成功率提升38.9%，接触密集型任务成功率提升86%，有效解决了触觉污染问题并提升了触觉-视觉联合预测的鲁棒性。
