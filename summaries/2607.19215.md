# HACO: Hedged Agent Computing for Reliable LLM Systems

- arXiv: https://arxiv.org/abs/2607.19215v1
- arXiv ID: 2607.19215

---

🔍 **核心问题**：解决大语言模型（LLM）代理在长期工作流中因角色到实例绑定边界（即任务请求需分配给具体代理实例）而出现的可靠性问题，现有系统常假设固定稳定环境，导致实际部署中延迟、失败率和输出质量不稳定。

🛠 **方法**：提出HACO（对冲代理计算）运行时控制方案，将每个角色请求视为在候选代理实例（包含角色类型、LLM和执行环境）上的可靠性约束选择问题。通过结合乐观排序（优先选择高质量、高可靠性和高信息不确定性的候选）与保守可靠性累积（仅在对冲集达到目标成功概率时停止选择）的自适应对冲集分配规则，并利用经验收集更新候选和链路档案。

✨ **主要贡献**：在多种基准测试和运行时退化研究中，HACO在动态部署条件下显著提升了鲁棒性和输出质量，同时相比穷举并行执行降低了令牌和延迟成本。
