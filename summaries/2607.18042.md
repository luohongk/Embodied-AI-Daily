# Anticipate Before Acting: Future-State-Conditioned Vision-Language Navigation

- arXiv: https://arxiv.org/abs/2607.18042v1
- arXiv ID: 2607.18042

---

🔍 **核心问题**：在视觉-语言导航（VLN）任务中，现有端到端模型仅监督下一步动作，未显式训练策略状态预测未来视觉信息，导致长距离导航性能受限。

🛠 **方法**：提出未来状态条件化VLN（FSC-VLN），在训练时引入未来查询令牌，将其隐藏状态与冻结的Δ步后视觉嵌入对齐，通过仅在训练阶段使用的目标分支实现未来信息监督，推理时移除该分支。

✨ **主要贡献**：在R2R val-unseen数据集上，FSC-VLN相比StreamVLN基线显著提升SR/OSR/SPL指标，尤其在长距离任务中增益更大；消融实验验证了分离未来查询与动作查询的双查询设计的有效性。
