# When Fewer Layers Break More Chains: Layer Pruning Harms Test-Time Scaling in LLMs

- arXiv: https://arxiv.org/abs/2510.22228v2
- arXiv ID: 2510.22228

---

🔍 **核心问题**：研究层剪枝技术对大型语言模型（LLMs）长链推理能力的影响，特别是其对测试时扩展（test-time scaling）的破坏作用。

🛠 **方法**：通过大量实验，在知识密集型、浅层推理和长链推理基准上评估层剪枝（即使仅剪掉1-2层）对模型性能的影响，并分析标准监督微调能否恢复受损的测试时扩展能力。

✨ **主要贡献**：发现层剪枝会严重损害测试时扩展，导致长链推理性能急剧下降，且标准微调无法修复；揭示了测试时扩展脆弱性的内在机制，呼吁重新思考层剪枝策略。
