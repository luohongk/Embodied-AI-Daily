# AnyPoC: Universal Proof-of-Concept Test Generation for Scalable LLM-Based Bug Detection

- arXiv: https://arxiv.org/abs/2604.11950v2
- arXiv ID: 2604.11950

---

🔍 **核心问题**：解决基于LLM的自动化漏洞检测中，候选漏洞报告缺乏可执行验证（即PoC测试）的问题，现有方法易产生虚假或无效的PoC。

🛠 **方法**：提出ANYPoC多智能体框架，包含三个核心步骤：1）分析并事实核查候选漏洞报告；2）迭代合成并执行PoC，同时收集执行轨迹；3）独立重新执行并审查PoC以消除幻觉和奖励黑客行为。此外，框架持续提取并演化PoC知识库以处理异构任务。

✨ **主要贡献**：在Firefox、Chromium等12个大型关键软件系统中，ANYPoC相比Claude Code等先进编码智能体，对真阳性漏洞报告生成的有效PoC多37%，对假阳性报告拒绝率高9.7倍；从2000多个噪声报告中发现了121个新漏洞，其中108个被开发者确认、92个已修复，46个PoC被采纳为官方回归测试。
