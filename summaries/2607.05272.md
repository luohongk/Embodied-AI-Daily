# Adaptive Inference Batching using Policy Gradients

- arXiv: https://arxiv.org/abs/2607.05272v1
- arXiv ID: 2607.05272

---

🔍 **核心问题**：推理服务系统在突发、异构工作负载下，如何自动平衡吞吐量与延迟，替代需要手动调优且无法适应流量变化的静态批处理策略。

🛠 **方法**：将问题建模为马尔可夫决策过程（MDP），使用REINFORCE和PPO强化学习智能体，在基于离散事件模拟器（经排队论和真实生产轨迹验证）上训练，学习自适应批处理与路由策略。

✨ **主要贡献**：发现强化学习在系统问题中的价值边界——单GPU场景下收益微小（+0.1%~1.0%），但在多GPU异构路由中，智能体学会工作负载隔离策略，消除队头阻塞，相比轮询策略提升3.5倍（348%），相比最强启发式基线（最短队列）提升48%，同时吞吐量提高60%、延迟降低25%，且策略能泛化到未见过的突发和真实流量。
