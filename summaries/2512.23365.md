# SpatialMosaic: A Multiview VLM Dataset for Partial Visibility

- arXiv: https://arxiv.org/abs/2512.23365v4
- arXiv ID: 2512.23365

---

🔍 **核心问题**：现有多模态大语言模型在真实场景中常因部分可见、遮挡或低重叠视角而难以进行有效的空间推理，缺乏针对此类挑战的专门数据集和评估基准。

🛠 **方法**：提出可扩展的多视图数据生成与标注流程，构建包含200万问答对的指令微调数据集SpatialMosaic，并设计含100万问答对的基准SpatialMosaic-Bench，同时引入几何编码器增强视觉语言模型的跨视图一致性。

✨ **主要贡献**：首次系统性地解决多视图部分可见场景下的空间推理难题，通过大规模高质量数据集和基准验证了所提方法在复杂真实环境中的有效性，为多视图空间理解提供了实用基线。
