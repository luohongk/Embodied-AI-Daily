# FootsiesGym: A Fighting Game Benchmark for Two-Player Zero-Sum Imperfect-Information Games

- arXiv: https://arxiv.org/abs/2607.06514v1
- arXiv ID: 2607.06514

---

🔍 **核心问题**：为双人零和不完美信息博弈（如格斗游戏中的“中立博弈”）提供一个简洁、可复现且高效的基准测试环境，以研究非传递性策略交互。

🛠 **方法**：基于HiFight的极简2D格斗游戏Footsies，构建了FootsiesGym环境，并开发了向量化模拟器，支持在标准硬件上进行高吞吐量的强化学习训练。

✨ **主要贡献**：开源了一个隔离格斗游戏核心循环策略交互的轻量级基准，提供了多种强化学习算法的性能基准测试，并指出了未来研究方向（如非传递性博弈、对抗性策略学习等）。
