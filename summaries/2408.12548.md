# Human-In-The-Loop Machine Learning for Safe and Ethical Autonomous Vehicles: Principles, Challenges, and Opportunities

- arXiv: https://arxiv.org/abs/2408.12548v3
- arXiv ID: 2408.12548

---

🔍 **核心问题**：解决自动驾驶在复杂场景下难以实现全自主性，以及数据标注成本高、效率低的问题，同时确保安全与伦理合规。

🛠 **方法**：采用人机协同的机器学习框架，包括课程学习（从简单到复杂任务训练）、人在回路强化学习（通过奖励设计、动作修正等）、人在回路大语言模型（协作与监督）、主动学习（优化数据标注）以及伦理原则的集成。

✨ **主要贡献**：系统综述了HITL-ML在自动驾驶中的关键技术与应用，涵盖感知、规划、控制等模块，并提出了提升学习效率、安全探索和实时干预的可行方案，同时强调透明度、问责制和人机监督等伦理要求。
