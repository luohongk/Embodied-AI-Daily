# MPC-Injection: Biasing Off-Policy Locomotion RL Toward Controller-Induced Behavior Basins

- arXiv: https://arxiv.org/abs/2606.26392v1
- arXiv ID: 2606.26392

---

🔍 **核心问题**：强化学习在机器人运动控制中常收敛于局部最优但不可部署的行为（如肢体抖动或躯干滑动），无法产生可用的步态。  
🛠 **方法**：提出MPC-Injection，通过将模型预测控制器（MPC）求解同一马尔可夫决策过程产生的状态转移插入到经验回放缓冲区，引导策略偏向控制器偏好的行为，无需修改奖励函数或添加对抗判别器。  
✨ **主要贡献**：在2D仿真步行器和Go2四足机器人的仿真到真实迁移中，仅用1-2项任务奖励即可产生与21项调优奖励或对抗运动先验方法质量相当的步态，并揭示了注入转移如何通过偏置演员-评论家更新使策略学习到纯强化学习在简单奖励下难以达到的行为。
