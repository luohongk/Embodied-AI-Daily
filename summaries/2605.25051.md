# A Decentralized LiDAR-SLAM System with Certifiably Optimal Pose Graph Optimization

- arXiv: https://arxiv.org/abs/2605.25051v2
- arXiv ID: 2605.25051

---

🔍 **核心问题**：现有分布式多机器人LiDAR-SLAM系统在全局一致性上存在不足，局部搜索或单次坐标对齐方法易导致次优收敛和长期不一致，尤其在大型或退化环境中。

🛠 **方法**：提出首个集成可认证最优位姿图优化（PGO）后端的分布式LiDAR-SLAM系统，采用黎曼块坐标下降（RBCD）算法，无需精确初始估计即可实现全局一致的轨迹估计。

✨ **主要贡献**：系统在鲁棒性上显著提升，相比当前最先进的DiSCo-SLAM，轨迹RMSE降低高达48.9%。
