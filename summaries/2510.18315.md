# Higher Embedding Dimension Creates a Stronger World Model for a Simple Sorting Task

- arXiv: https://arxiv.org/abs/2510.18315v2
- arXiv ID: 2510.18315

---

🔍 **核心问题**：研究嵌入维度如何影响Transformer在强化学习中形成内部“世界模型”的能力，以执行冒泡排序式的相邻交换任务。

🛠 **方法**：通过强化学习训练Transformer执行简单的排序任务，并系统性地改变嵌入维度，分析模型内部表征的准确性、一致性和鲁棒性。

✨ **主要贡献**：发现更高嵌入维度能增强结构化内部表征的形成，提升模型的可解释性，并揭示两种一致机制（注意力权重矩阵最后一行单调编码全局顺序、所选交换与编码值的最大相邻差异对齐），证明模型规模提升不仅优化最终性能，还改善表征质量。
