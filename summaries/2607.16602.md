# PAVXploreRL: Physical-Action-Visual World Model Reinforcement Learning with Action Exploration

- arXiv: https://arxiv.org/abs/2607.16602v2
- arXiv ID: 2607.16602

---

🔍 **核心问题**：现有基于动作条件的视频预测世界模型未能显式优化物理合理性、动作一致性和视觉保真度（PAV目标），且对分布外动作的泛化能力不足。

🛠 **方法**：提出PAVXploreRL框架，在预训练潜在世界模型基础上，通过强化学习奖励驱动训练，联合利用专家轨迹和噪声驱动的分布外动作探索，无需配对视频监督。

✨ **主要贡献**：在多个基准上平均提升5.6%，生成更高质量的PAV属性；作为策略评估器时，提供更可靠的性能估计，并减少如Ctrl-World等仅依赖专家数据的世界模型的过估计偏差。
