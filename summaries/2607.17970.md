# MEVION: Low-Cost Open-Source Data Collection System for Powerful and High-Speed Dual-Arm Manipulation

- arXiv: https://arxiv.org/abs/2607.17970v1
- arXiv ID: 2607.17970

---

🔍 **核心问题**：现有低成本开源双臂机器人数据采集系统（如ALOHA）在力和速度输出上有限，难以处理重物或执行快速操作任务。

🛠 **方法**：开发了MEVION系统，采用四台6自由度机械臂（含平行夹爪），通过广泛使用钣金焊接构建低成本大型机身，并在肘关节引入类似四足机器人的闭链机构，以降低远端质量并提升末端执行器的力和速度。

✨ **主要贡献**：实现了低成本（约14,000美元）下高力（最大扭矩60 Nm）和高速度的双臂操作，支持此前无法完成的物体操作数据采集，并兼容模仿学习运动生成；所有硬件和软件已开源。
