# DASIP: Dynamic Test-Time Compute Scaling for Robot Control with Stochastic Interpolant Policies

- arXiv: https://arxiv.org/abs/2511.20906v2
- arXiv ID: 2511.20906

---

🔍 **核心问题**：现有扩散/流策略在机器人控制中固定推理预算，导致简单任务计算浪费而复杂任务性能不足。  
🛠 **方法**：提出DA-SIP框架，通过难度分类器实时分析观测，动态选择积分步数、求解器变体及ODE/SDE模式，基于随机插值策略统一训练与推理配置。  
✨ **主要贡献**：在多种操作任务中，总计算时间减少2.6-4.4倍，同时保持与固定最大计算基线相当的任务成功率，实现自适应推理资源分配。
