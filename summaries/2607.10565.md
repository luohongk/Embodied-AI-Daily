# BucketKD: A Safety-Aware Bucket-Based Knowledge Distillation Framework for End-to-End Motion Planning

- arXiv: https://arxiv.org/abs/2607.10565v1
- arXiv ID: 2607.10565

---

🔍 **核心问题**：端到端运动规划模型因体积过大难以部署在资源受限平台，且现有知识蒸馏方法依赖简化的规划状态表示，缺乏对安全关键行为的保留。

🛠 **方法**：提出BucketKD框架，通过将关键环境变量离散化为自适应桶（buckets）以丰富场景语义，并设计安全感知路径点注意力机制，结合碰撞时间（TTC）评估风险，在蒸馏过程中保留安全行为。

✨ **主要贡献**：在CARLA和Bench2Drive数据集上，BucketKD在保持高压缩比的同时，显著提升了规划精度和安全性，优于现有最先进方法。
