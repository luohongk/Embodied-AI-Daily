# InCarEmo: A Multimodal Dataset for In-Cabin Emotion Recognition and Driver State Monitoring

- arXiv: https://arxiv.org/abs/2607.14683v1
- arXiv ID: 2607.14683

---

🔍 **核心问题**：现有车内情感计算数据集多局限于视觉模态，缺乏对话信息，难以捕捉驾驶员情绪中的语言和交互线索。  
🛠 **方法**：构建了InCarEmo多模态数据集，整合RGB/红外视频、车内音频和对话文本，覆盖多种光照与驾驶场景，并支持情感识别、疲劳检测和分心监测三项任务。  
✨ **主要贡献**：提供了统一基准和跨模态/跨语言评估结果，验证了多模态融合的优势，并揭示了真实噪声与低光条件下的挑战，为鲁棒、可解释的人车交互研究奠定基础。
