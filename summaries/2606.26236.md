# Rendering Novel Views of MRI Using 3D Gaussian Splatting

- arXiv: https://arxiv.org/abs/2606.26236v1
- arXiv ID: 2606.26236

---

🔍 **核心问题**：解决脊柱MRI扫描中因原始图像平面与目标解剖结构不对齐，导致放射学分级不准确的问题。

🛠 **方法**：将3D高斯泼溅技术（3D Gaussian Splatting）应用于稀疏各向异性MRI数据，构建体积重建，然后沿与临床评估相关的解剖平面对新视角进行采样和渲染。

✨ **主要贡献**：相比原始扫描和体素插值重采样方法，基于高斯泼溅的重采样图像在脊柱狭窄分级上取得了更准确的预测结果。
