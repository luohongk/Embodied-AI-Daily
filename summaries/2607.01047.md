# Conversable Complexity: Agentic LLM Collectives as Interpretable Substrates

- arXiv: https://arxiv.org/abs/2607.01047v1
- arXiv ID: 2607.01047

---

🔍 **核心问题**：复杂性与可解释性难以兼得——复杂系统难以理解，而透明系统又过于简单，无法产生复杂行为。本文探讨如何利用大语言模型（LLM）智能体集合作为人工生命（ALife）研究的可解释计算基质。

🛠 **方法**：通过让多个LLM智能体具备持久记忆、工具、共享技能及主动发起行动的能力，形成交互式集合体，并利用自然语言通信产生的文本痕迹直接询问智能体以解释其集体行为。

✨ **主要贡献**：提出“可对话复杂性”概念，证明LLM智能体集合能展现孤立模型所缺乏的涌现动态，同时通过自然语言交互实现直接可解释性，并综述了从受控实验到实际部署的现有实例。
