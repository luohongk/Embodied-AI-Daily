# SE(2) Navigation Mesh

- arXiv: https://arxiv.org/abs/2607.01454v1
- arXiv ID: 2607.01454

---

🔍 **核心问题**：现有导航网格（NavMesh）假设机器人可任意转向（偏航不变），无法处理非圆形机器人在狭窄空间中的通行性，导致复杂多层环境中全局导航效率低。

🛠 **方法**：提出SE(2)导航网格，通过足迹掩模评估不同朝向的通行性，构建多层图结构（含平移和旋转连接），并设计A*-String Pulling-A*（ASA）分层路径规划策略，同时支持从点云流在线增量更新。

✨ **主要贡献**：相比经典NavMesh，SE(2) NavMesh可捕获多50%以上可通行区域；SE(2) NavMesh+ASA方法在受限环境中优于采样基线；真实机器人实验验证了实时在线生成与多环境成功导航。
