# Towards Miniature Humanoid Tele-Loco-Manipulation Using Virtual Reality and Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.20399v1
- arXiv ID: 2607.20399

---

🔍 **核心问题**：针对全尺寸人形机器人成本高、研究社区难以普及的问题，开发适用于微型人形机器人的全身远程操控与移动操作方案。

🛠 **方法**：结合虚拟现实（VR）进行上半身远程操控，并利用强化学习（RL）控制下半身平衡与行走，在ROBOTIS OP3硬件上实现完整系统。

✨ **主要贡献**：在微型人形机器人上实现了独立于手臂运动的行走（最高0.45 m/s），并通过专家操作员在10分钟内完成40克立方体的搬运实验（总行走距离5米），验证了远程移动操作的有效性。
