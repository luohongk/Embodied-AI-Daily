# Beyond Time Series: Spatial Reasoning for Epidemic Forecasting via Multimodal Learning

- arXiv: https://arxiv.org/abs/2606.22171v1
- arXiv ID: 2606.22171

---

🔍 **核心问题**：传统流行病预测模型将行政区域视为原子单元，忽略了区域内空间结构对疾病传播的影响，且难以融合分辨率与结构不一致的多模态辅助信号。

🛠 **方法**：提出M-SPICE框架，通过注意力机制的多模态融合，对时间序列疾病动态与空间上下文进行联合推理，使空间信号能选择性调节不同预测时域的时间表征。

✨ **主要贡献**：在COVID-19、流感和ILI真实预测任务中，该方法持续优于最先进的多元时间序列、多模态和流行病学基线模型，并通过可解释性分析揭示了空间信号何时、何地及如何提升预测性能。
