# FlashRT: Agent Harness for Guiding Agents to Deploy Real-Time Multimodal Applications

- arXiv: https://arxiv.org/abs/2607.18171v1
- arXiv ID: 2607.18171

---

🔍 **核心问题**：实时多模态应用（如语音代理和交互式视频生成）需要将异构模型组合成流水线，但现有系统难以自动实现高效的部署优化，通常需要手动调整。

🛠 **方法**：提出FlashRT，一种代理框架，通过“链式程序”范式引导编码代理将开发者编写的参考实现逐步转换为中间表示（IR），验证数据依赖和持久状态，并进行静态分析，再通过测量驱动的优化循环迭代实现、验证和基准测试候选变换，最终生成高效的多GPU部署。

✨ **主要贡献**：在NVIDIA B200 GPU上实现高达70倍延迟降低和2.8倍吞吐量提升；在AMD MI355X GPU上延迟降低相同，吞吐量提升达3.6倍；在Qwen3-Omni文本到音频推理中，相比专家级vLLM-Omni实现，响应延迟降低65%，展示了代理驱动优化在成熟度较低的平台上更具可扩展性。
