# Real-time Whole-body Model Predictive Control for Bipedal Locomotion with a Novel Kino-dynamic Model and Warm-start Method

- arXiv: https://arxiv.org/abs/2505.19540v1
- arXiv ID: 2505.19540

---

🔍 **核心问题**：双足机器人因高自由度与模型复杂性，难以实现快速稳定的全身模型预测控制（WB-MPC）以满足实时性能要求。

🛠 **方法**：提出一种结合线性倒立摆+飞轮模型与全身运动学的新型运动学-动力学模型，利用零力矩点（ZMP）降低计算成本；并设计基于模块化多层感知器（MLP）的热启动策略，为每个控制周期提供优质初始解。

✨ **主要贡献**：通过ZMP替代传统接触力模型，显著减少基线计算量并保证接触状态切换时的低延迟；MLP热启动策略提升求解效率；实验证明该方法在抗扰动和实时行走控制中优于现有方案。
