# PAKE: Learning Whole-Body Loco-Manipulation with Partial Kinematic Embeddings

- arXiv: https://arxiv.org/abs/2607.11041v1
- arXiv ID: 2607.11041

---

🔍 **核心问题**：解决高自由度全身机器人在移动操作中高维动作控制、精确跟踪与冗余利用的挑战。  
🛠 **方法**：提出部分运动嵌入框架，通过运动学归一化流（KNF）模型生成多样可行的参考运动，结合高层控制器探索潜在空间冗余解与低层控制器执行物理可行运动。  
✨ **主要贡献**：在四足机器人（带六自由度臂）上，仿真中跟踪精度与可行工作空间覆盖显著优于现有方法；硬件实验中实现末端执行器位姿误差4.5 cm/0.14 rad，运动速度误差0.1 m/s/0.01 rad/s，验证了高精度泛化能力。
