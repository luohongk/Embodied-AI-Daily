# Scalable Trajectory Generation for Whole-Body Mobile Manipulation

- arXiv: https://arxiv.org/abs/2604.12565v1
- arXiv ID: 2604.12565

---

🔍 **核心问题**：现有方法难以高效生成大规模、物理有效且协调的全身移动操作轨迹数据，导致数据稀缺成为机器人策略学习的瓶颈。

🛠 **方法**：提出AutoMoMa框架，通过GPU加速统一AKR运动学模型（将基座、机械臂和物体运动学整合为单一链），并结合并行轨迹优化，实现高速数据生成。

✨ **主要贡献**：AutoMoMa实现每GPU小时生成5000条轨迹（比CPU基线快80倍以上），构建了包含50万条有效轨迹、覆盖330个场景和多种机器人的数据集，并证明数据规模而非算法限制是下游策略学习的关键瓶颈。
