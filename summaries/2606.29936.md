# OpenSPM: An Environment-Transferable Robotic Key Spatial Pose Memory and Closed-Loop High-Frequency Flow-Matching Action Generation Model

- arXiv: https://arxiv.org/abs/2606.29936v1
- arXiv ID: 2606.29936

---

🔍 **核心问题**：解决开放环境桌面机器人操作中，端到端VLA模型缺乏几何约束、难以实现高频动作生成和低成本部署的问题。

🛠 **方法**：提出OpenSPM框架，包含空间姿态记忆（利用语义3D感知和卡尔曼滤波追踪6D姿态，从人类演示中提取关键空间姿态）和流匹配动作生成模型（通过SE(3)变换迁移姿态至新场景，结合实时状态反馈和终端残差校正，生成高频动作块）。

✨ **主要贡献**：在LIBERO-GOAL十项任务中达到85.6%成功率与1033.3Hz等效控制频率，且推理算力需求极低；验证了结构化空间持久记忆与闭环残差校正对高频可靠操作的关键作用。
