# GRINQH: Graded Input-based Quantization Hierarchy for Efficient LLM Generation

- arXiv: https://arxiv.org/abs/2606.23419v1
- arXiv ID: 2606.23419

---

🔍 **核心问题**：大语言模型自回归解码受限于GPU内存带宽，尤其在边缘计算场景中，现有量化方法未区分计算密集的预填充阶段与内存密集的解码阶段。

🛠 **方法**：提出GRINQH框架，一种仅权重的训练后量化方法，通过激活值幅度作为计算重要性代理，动态分配权重通道到不同精度层级，并统一量化与稀疏化，利用层级嵌套内存布局实现多精度存储。

✨ **主要贡献**：在Llama3和Qwen3模型上，GRINQH在3位和4位设置下优于现有固定和混合精度基线，甚至支持有效2位生成，建立了生成质量与推理速度间动态权衡的新帕累托前沿。
