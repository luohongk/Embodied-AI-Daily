# Delta-JEPA: Learning Action-Sensitive World Models via Latent Difference Decoding

- arXiv: https://arxiv.org/abs/2606.31232v1
- arXiv ID: 2606.31232

---

🔍 **核心问题**：无重建的联合嵌入世界模型在规划任务中容易学到对动作不敏感的潜在表示，导致模型无法有效利用动作信息进行规划。

🛠 **方法**：提出Delta-JEPA，通过引入潜在差异动作解码器（LDAD），从连续观测之间的潜在位移中重建执行的动作，从而直接约束潜在转移的几何结构。

✨ **主要贡献**：在四个视觉连续控制任务上，Delta-JEPA的规划性能优于基于JEPA和表示学习的基线模型；消融实验表明位移级动作解码比端点拼接更有效，且模型展现出清晰的动作条件化潜在响应。
