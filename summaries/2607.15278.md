# Hierarchical Denoising For Multi-Step Visual Reasoning

- arXiv: https://arxiv.org/abs/2607.15278v1
- arXiv ID: 2607.15278

---

🔍 **核心问题**：现有视频模型在多步视觉推理任务中难以兼顾逻辑一致性与低延迟流式生成，密集帧级去噪导致高推理成本。

🛠 **方法**：提出HDR框架，将视频潜在表示组织为树状层次结构，通过粗到细的层次化去噪实现多步推理，并采用稀疏层次注意力模式（SHAP）降低时序注意力计算成本。

✨ **主要贡献**：在六项多步推理基准任务中，HDR将成功率从34.22提升至60.29（相对提升76.2%），平均进度从76.00提升至89.56；推理速度比双向扩散快54.2倍，且仅用2%训练数据即可保留82.9%性能，并在真实机器人实验中验证了物理交互潜力。
