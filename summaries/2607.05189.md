# When Claws Remember but Do Not Tell: Stealthy Memory Injection in Persistent Personal Agents

- arXiv: https://arxiv.org/abs/2607.05189v1
- arXiv ID: 2607.05189

---

🔍 **核心问题**：研究持久性个人代理在整合长期记忆与外部环境时，如何被远程黑盒攻击者通过单封邮件将恶意内容静默注入持久记忆，从而在用户无察觉的情况下长期操控代理行为。

🛠 **方法**：提出MemGhost框架，利用环境代理模拟持久代理执行，并通过目标代理将记忆采纳与对话隐蔽性转化为基于评分规则的密集奖励，结合监督微调与强化学习训练攻击策略，实现单次邮件投递的无反馈黑盒攻击。

✨ **主要贡献**：构建了包含108个案例的WhisperBench基准测试，覆盖五种风险类别；MemGhost在OpenClaw和Claude Code SDK上分别达到87.5%和71.4%的成功率，且能跨架构与记忆后端迁移，并有效规避多种防御机制，揭示了持久记忆可成为长期代理妥协的实用路径。
