# SPACE: Enabling Learning from Cross-Robot Data Toward Generalist Policies

- arXiv: https://arxiv.org/abs/2606.24049v1
- arXiv ID: 2606.24049

---

🔍 **核心问题**：不同机器人因动力学差异（如构型、硬件单元、运行状态变化）导致相同运动需要不同动作指令，阻碍了跨机器人数据的策略训练与部署。

🛠 **方法**：提出SPACE框架，采用笛卡尔状态增量作为通用动作表征，通过状态预测策略预测末端执行器位移，并由Action Adapter将其转换为各机器人特有的控制命令，处理三个层面的动力学变化。

✨ **主要贡献**：在跨构型与跨硬件单元数据训练中显著优于直接预测控制命令的策略，并在部署时对控制频率、负载重量、控制器增益等动力学变化保持鲁棒性。
