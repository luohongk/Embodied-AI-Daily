# TokenPilot: Cache-Efficient Context Management for LLM Agents

- arXiv: https://arxiv.org/abs/2606.17016v1
- arXiv ID: 2606.17016

---

🔍 **核心问题**：大语言模型智能体在长周期会话中，上下文积累导致推理成本激增，现有方法因序列突变引发前缀不匹配和缓存失效，难以平衡文本稀疏性与提示缓存连续性。

🛠 **方法**：提出TokenPilot双粒度上下文管理框架：全局上采用“摄入感知压缩”稳定提示前缀并过滤环境噪声；局部上通过“生命周期感知驱逐”监控上下文段残余效用，仅在任务相关性过期时保守卸载内容。

✨ **主要贡献**：在PinchBench和Claw-Eval基准测试中，TokenPilot在隔离模式下降低61%和56%成本，连续模式下降低61%和87%，同时保持竞争性能，并已集成至LightMem2开源项目。
