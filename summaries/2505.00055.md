# TinyMA-IEI-PPO: Exploration Incentive-Driven Multi-Agent DRL with Self-Adaptive Pruning for Vehicular Embodied AI Agent Twins Migration

- arXiv: https://arxiv.org/abs/2505.00055v1
- arXiv ID: 2505.00055

---

🔍 **核心问题**：解决车载具身AI网络中，因计算资源受限的RSU和车辆移动性导致的虚拟服务实时迁移效率低下问题。

🛠 **方法**：提出结合多领导者多追随者Stackelberg博弈激励机制的TinyMA-IEI-PPO算法，通过自适应动态结构化剪枝技术，在保留探索-利用性能的同时实现轻量化多智能体深度强化学习部署。

✨ **主要贡献**：首次将探索激励驱动的神经元重要性动态剪枝与博弈论激励模型结合，在计算受限车辆上实现接近Stackelberg均衡的迁移决策性能，且收敛速度与基线模型相当。
