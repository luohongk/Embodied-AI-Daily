# Next-Generation Agentic Reinforcement Learning Systems Enable Self-Evolving Agents

- arXiv: https://arxiv.org/abs/2607.01120v1
- arXiv ID: 2607.01120

---

🔍 **核心问题**：当前企业级LLM代理在部署后无法自我进化，依赖人工循环进行改进，缺乏从自身经验中持续学习的能力。

🛠 **方法**：提出下一代智能体强化学习系统，围绕三大支柱设计：标准化轨迹数据协议、企业级数据代理、统一进化控制平面，并通过AReaL2.0实例化在线RL循环。

✨ **主要贡献**：指出阻碍自进化代理部署的关键在于智能体在线RL系统而非算法，并给出了具体架构、案例和反论证，为构建持续学习的代理系统提供了系统化方案。
