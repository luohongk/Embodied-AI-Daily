# Precision Recall Controllable Radiology Report Generation via Hybrid Natural Language and Clinical Reward Learning

- arXiv: https://arxiv.org/abs/2606.21447v2
- arXiv ID: 2606.21447

---

🔍 **核心问题**：现有放射学报告生成方法主要优化语言流畅度，缺乏对临床精度和召回率等重要指标的灵活控制，导致生成的报告虽流畅但无法适应不同临床需求。

🛠 **方法**：提出基于强化学习的框架，通过引入控制参数在推理时显式调节临床精度与召回率的权衡，并设计临床奖励函数以提升临床有效性，同时采用组内相对训练策略稳定训练过程。

✨ **主要贡献**：在MIMIC-CXR数据集上，该方法在自然语言生成和临床有效性指标上均超越现有最优方法，并实现了对临床精度-召回率权衡的可靠控制。
