# Render-FM: Feedforward Model for Real-time Photorealistic Volumetric Rendering

- arXiv: https://arxiv.org/abs/2505.17338v2
- arXiv ID: 2505.17338

---

🔍 **核心问题**：现有神经渲染方法（如NeRF、3DGS）对CT扫描进行逼真体积渲染时，需逐扫描优化（耗时数十分钟至数小时），无法满足临床实时需求。

🛠 **方法**：提出前馈模型Render-FM，基于nnU-Net架构的3D U-Net，直接从CT体积回归6D高斯溅射参数，单次前向传播仅需2.8秒；并引入解剖引导初始化（AGP），利用分割掩膜和传递函数提供结构和外观先验。

✨ **主要贡献**：实现500倍加速（相比逐扫描优化），支持实时渲染、泛化至未见解剖结构和新传递函数，零额外准备时间即可组合器官可视化；可选89秒微调可超越逐扫描优化基线。
