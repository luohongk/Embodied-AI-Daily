# Signal Temporal Logic Motion Planning via Graphs of Convex Sets

- arXiv: https://arxiv.org/abs/2605.23240v1
- arXiv ID: 2605.23240

---

🔍 **核心问题**：解决在信号时序逻辑（STL）规范下，生成满足高层逻辑与时间约束、同时遵守低层运动限制的平滑机器人连续时间轨迹问题。

🛠 **方法**：提出将STL规范转化为时间自动机，并与配置空间的凸分解结合形成联合转移系统，进而将运动规划问题重构为凸集图（GCS）上的最短路径问题，通过求解得到平滑的贝塞尔样条轨迹。

✨ **主要贡献**：建立了方法的正确性证明，分析了计算复杂度（凸松弛随维度与贝塞尔阶数多项式增长），开发了针对表达性STL片段的紧凑时间自动机构建模板，并在低维基准、3D四旋翼、30自由度人形机器人和UR-3机械臂硬件实验中验证了高效性与轨迹平滑性。
