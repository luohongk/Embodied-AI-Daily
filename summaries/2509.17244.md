# Scalable Multi Agent Diffusion Policies for Coverage Control

- arXiv: https://arxiv.org/abs/2509.17244v2
- arXiv ID: 2509.17244

---

🔍 **核心问题**：解决分布式机器人群体在覆盖控制任务中，因动作空间高维且相互依赖而难以协同的问题。

🛠 **方法**：提出MADP方法，利用扩散模型从复杂高维动作分布中生成样本，每个机器人基于自身观测和同伴感知嵌入的融合表示来条件化策略采样，并通过空间变换器架构实现去中心化推理，采用模仿学习从全知专家策略中训练。

✨ **主要贡献**：首次将扩散模型应用于多机器人覆盖控制，模型能泛化至不同机器人密度和环境，在多种重要性密度函数配置下持续超越现有最优基线方法。
