# Words Speak Louder Than Code: Investigating Cognitive Heuristics in LLM-Based Code Vulnerability Detection

- arXiv: https://arxiv.org/abs/2606.30587v1
- arXiv ID: 2606.30587

---

🔍 **核心问题**：探究大型语言模型（LLM）在代码漏洞检测中是否受到认知启发（如光环效应、框架效应、锚定效应）的影响，以及这些启发是否可被利用来操纵检测结果。

🛠 **方法**：设计了一个控制框架，固定代码内容不变，仅通过改变上下文（如作者归属、任务目标、先前分析结果）来触发三种认知启发，并在8个LLM、3种编程语言上进行了定量和代码级分析。

✨ **主要贡献**：首次系统证明LLM在漏洞检测中普遍受认知启发影响，其中框架效应影响最大（33.2%），且需语义推理的漏洞更易受影响；还展示了黑盒认知攻击可抑制高达97%的已检测漏洞，揭示了认知脆弱性是可利用的模型属性。
