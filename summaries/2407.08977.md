# CURE: Privacy-Preserving Split Learning Done Right

- arXiv: https://arxiv.org/abs/2407.08977v2
- arXiv ID: 2407.08977

---

🔍 **核心问题**：在分布式深度学习训练中，分割学习（SL）虽能通过分离模型层降低隐私风险，但服务器与客户端交换的中间输出仍可能泄露客户端数据隐私，而现有同态加密（HE）方案计算开销过高。

🛠 **方法**：提出CURE系统，基于同态加密仅加密服务器端模型（可选加密数据），设计高效打包方案以支持MLP和卷积模型（如ResNet块），并引入新型估计器推荐最优服务器-客户端分割点。

✨ **主要贡献**：CURE在保持与明文SL相近精度的同时，相比现有隐私保护方案运行时效率提升最高达210倍，并首次实现HE在SL场景下的高效并行化与通信优化。
