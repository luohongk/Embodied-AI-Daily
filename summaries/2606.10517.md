# LAFP: Preserving Latent Action Structure in Latent Policy Learning via Flow Matching

- arXiv: https://arxiv.org/abs/2606.10517v1
- arXiv ID: 2606.10517

---

🔍 **核心问题**：现有基于大规模无标签视频学习潜在动作的方法，在训练动作解码器时因行为克隆导致多模态动作分布坍缩，且直接应用流匹配会因策略随机性造成潜在动作与物理动作的对齐偏差。

🛠 **方法**：提出潜在动作流策略（LAFP），利用流匹配进行潜在策略学习，并引入推理时插值机制来缓解随机性导致的对齐问题。

✨ **主要贡献**：在下游模仿学习任务中，LAFP一致优于先前方法，成功率提升10-15%，且额外推理开销不到1倍。
