## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人学中的安全性，具体针对视觉-语言-动作（VLA）模型的对抗性攻击。
- **问题（Problem）**：现有针对VLA模型的攻击主要关注于诱导单步的、低级别的错误动作，或者使这些错误动作在变化的图像中持续。然而，这些攻击未能控制机器人最终完成的**物理任务**。一个看似无害的文本扰动，能否在保持指令“看起来”正确的前提下，劫持整个闭环轨迹，使机器人完成一个完全不同的、由攻击者指定的物理目标？
- **方法（Method）**：作者形式化定义了“指令保持的轨迹重定向”威胁模型，并提出了一种**在策略（on-policy）的提示搜索方法**。该方法类似于DAgger算法，通过滚动执行候选对抗提示，收集其产生的状态分布，并利用冻结的VLA模型自身作为“教师”来提供目标任务的行动标签，从而优化出满足约束条件的对抗性提示。
- **创新（Innovation）**：
    1.  首次形式化定义了**轨迹级别的重定向攻击**，其目标是改变最终的物理结果，而非仅仅改变单步动作。
    2.  提出了一个**在策略的提示搜索算法**，该算法考虑了闭环控制中状态分布的变化，这是与以往基于固定数据集的攻击方法的关键区别。
    3.  在仿真和真实硬件上，对多种VLA架构（离散动作、流匹配、扩散等）进行了广泛评估，证明了该攻击的普遍性和有效性。
- **意义（Significance）**：这项工作揭示了VLA模型在指令理解上的一个根本性、轨迹级别的漏洞。它表明，仅仅对文本指令进行微小的、看似无害的修改，就能完全控制机器人的最终物理行为，这对VLA模型在现实世界中的安全部署构成了严峻挑战。它强调了语言理解必须被视为闭环控制系统的一部分，而不仅仅是输入预处理。
- **一句话总结**：本文发现并验证了VLA模型存在一个严重的轨迹级安全漏洞：通过在保持指令语义“看起来”不变的前提下进行微小文本扰动，可以劫持机器人的整个执行轨迹，使其完成攻击者指定的、与原始指令完全不同的物理任务。

## 1. 研究动机（Motivation）

- **问题与空白**：VLA模型（如OpenVLA, π0.5）通过自然语言指令进行闭环控制，这带来了新的安全风险。已有的攻击研究（如Jones et al., VLA-Fool）主要关注于：
    - **单步动作攻击**：通过对抗性后缀诱导模型在某个特定时刻输出错误动作。
    - **动作持续性攻击**：使一个错误动作在多个时间步中持续。
    - **视觉模态攻击**：通过对抗性补丁等视觉扰动来影响模型。
    - **核心空白**：这些攻击都没有解决一个更根本的问题：**控制机器人最终完成的物理任务**。一个复杂的操作任务（如“把碗放在盘子上”）需要一系列时序上结构化的动作（接近、抓取、抬起、运输、对齐、释放）。仅仅改变单步动作或让一个动作持续，并不等同于改变了整个任务轨迹的最终结果。攻击者真正想要的是让机器人完成一个**不同的、完整的任务**。
- **为什么重要**：随着VLA机器人从受控演示走向更广泛的部署，理解由这种持久性语言接口引入的故障模式成为一个具体的**安全问题**。如果攻击者可以通过一个看似无害的指令（如“把碗放在炉子上”，实际却是“把碗放在盘子上”）来操控机器人，后果可能非常严重。
- **挑战**：
    1.  **轨迹级目标**：攻击目标不是单步动作，而是最终的物理状态谓词（如“碗在盘子上”）。
    2.  **闭环状态分布偏移**：一个对抗性提示不仅改变当前动作，还会改变未来的观测。因此，在固定数据集上优化的提示，在闭环执行时可能因为遇到从未见过的状态而失效。攻击者必须考虑提示本身引起的状态分布变化。
    3.  **指令保持约束**：攻击提示必须看起来与原始良性指令非常接近，不能包含目标任务的任何关键词（如“盘子”），也不能包含任何覆盖或纠正性语言。这使得攻击更加隐蔽和危险。

## 2. 创新点（Innovation）

1.  **形式化定义“指令保持的轨迹重定向”**：这是本文最核心的创新。作者将攻击目标从“改变单步动作”提升到“改变最终物理结果”，并严格定义了攻击约束：攻击提示必须是良性指令的微小扰动，不能泄露目标指令，且必须保持原始指令的语义。这构成了一个更强大、更现实的威胁模型。

2.  **提出“在策略教师匹配提示搜索”方法**：这是解决上述挑战的关键算法创新。
    - **在策略（On-Policy）**：该方法的核心是**在策略数据聚合**。它不像传统方法那样在固定的、预收集的观测数据集上优化提示，而是**滚动执行**（rollout）候选提示，收集由该提示自身产生的观测数据。这解决了闭环控制中的状态分布偏移问题，因为优化是在攻击提示实际会遇到的“真实”状态上进行的。
    - **教师匹配（Teacher-Matching）**：利用冻结的VLA模型本身作为“教师”。对于每个观测，计算良性指令和目标指令下的动作（教师动作）。然后，优化候选提示，使其动作更接近于目标教师动作，同时远离良性教师动作。这为搜索提供了一个有效的梯度信号。
    - **类比DAgger**：作者明确指出，该方法是DAgger算法在提示搜索领域的对等物。DAgger通过聚合专家在策略产生的数据来改进模仿学习，而本文的方法通过聚合目标教师动作在攻击提示产生的状态上的数据来优化攻击提示。

3.  **广泛的实验验证**：在仿真（LIBERO）和真实硬件（SO-100机械臂）上，对9种不同的VLA架构（包括离散动作、流匹配、扩散、动作即文本等）进行了评估，证明了该攻击的**普遍性**和**有效性**（在7/9的架构上成功率超过90%）。这有力地证明了该漏洞是VLA模型的一个共性缺陷，而非特定模型的偶然问题。

4.  **因果分析**：通过因果追踪（Causal Trace）技术，作者揭示了攻击生效的**内部机制**：攻击主要集中在**被破坏的目标位置表示**上。例如，将“stove”改为“staove”，模型内部对目标位置的表示被破坏，从而引导动作头选择替代行为。这为理解攻击提供了深刻的洞察。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

本文提出的攻击框架是一个迭代式的搜索过程，其核心架构如图2所示。

1.  **初始化**：使用良性指令 $\tau_b$ 和目标指令 $\tau_t$ 分别进行一次闭环滚动，收集它们访问过的观测 $o$，并计算每个观测下对应的良性教师动作 $A_b(o)$ 和目标教师动作 $A_t(o)$，构建初始数据集 $D_0$。
2.  **候选提示生成**：基于当前数据集 $D_i$，使用各种文本扰动方法（如字符级编辑、同义词替换等）生成大量候选对抗提示 $\tau$。所有候选提示必须通过“指令保持”约束（公式2）的过滤。
3.  **候选提示评分**：对于每个通过过滤的候选提示 $\tau$，使用公式(5)计算其得分 $\hat{J}_i(\tau)$。该得分综合考虑了：
    - **目标-良性边际损失**：提示动作更接近目标教师还是良性教师。
    - **目标接近度**：提示动作与目标教师动作的绝对距离。
    - **文本扰动成本**：提示与原始指令的文本差异。
4.  **在策略滚动与选择**：从得分最高的候选提示中选出前 $M$ 个，在固定场景中进行完整的闭环滚动。根据滚动结果（是否达成攻击目标、是否失败基准任务、轨迹上的平均目标距离等），使用公式(6)计算滚动得分 $S_{roll}(\tau)$，并选出最优的提示集合 $B_i$。
5.  **数据聚合**：将选中的提示 $B_i$ 在滚动过程中产生的所有观测 $o_k^\tau$ 及其对应的教师动作标签，加入到数据集中，形成 $D_{i+1}$。这确保了后续的优化是在攻击提示自身产生的状态分布上进行的。
6.  **迭代与最小化**：重复步骤2-5，直到找到成功的攻击提示。最后，对成功提示进行最小化处理，移除不必要的扰动字符，同时保持其攻击有效性。

### 3.2 关键算法与数学推导

**1. 闭环VLA滚动 (Closed-Loop VLA Rollouts)**

VLA策略 $\Pi_\theta: \mathcal{T} \times \mathcal{O} \rightarrow \mathcal{A}^H$ 将文本指令 $\tau$ 和观测 $o$ 映射为长度为 $H$ 的动作块。在重规划步 $k$，机器人执行前 $m$ 个动作。由提示 $\tau$ 诱导的状态转移为：
$$s_{k+1}^\tau = F_m(s_k^\tau, U_m(\Pi_\theta(\tau, g(s_k^\tau)))), \quad s_0^\tau = s_0(e)$$
其中 $g$ 是观测函数，$U_m$ 是执行前缀操作符，$F_m$ 是环境动力学。整个轨迹记为 $\xi_\tau^e$。

**2. 指令保持的提示扰动 (Command-Preserving Prompt Perturbations)**

可接受的攻击提示集合 $\mathcal{T}_{cp}(\tau_b, \Gamma_e)$ 定义为：
$$\mathcal{T}_{cp}(\tau_b, \Gamma_e) = \{ \tau \in \mathcal{T}: C_{text}(\tau, \tau_b) \le \varepsilon, \text{Valid}(\tau) = 1, \text{Leak}(\tau; \Gamma_e) = 0, \text{Preserve}(\tau, \tau_b) = 1 \}$$
- $C_{text}$: 文本扰动成本（如编辑距离）。
- $\text{Valid}$: 提示可读性检查。
- $\text{Leak}$: 检查是否泄露了目标任务的关键词（如“plate”）。
- $\text{Preserve}$: 检查提示是否仍可被理解为原始指令。

**3. 攻击成功条件 (Attack Success)**

一个提示 $\tau$ 在回合 $e$ 上成功，当且仅当：
$$Success_e(\tau) = \mathbb{1}[ \tau \in \mathcal{T}_{cp}(\tau_b, \Gamma_e) \land T_e(\xi_\tau^e) = 1 \land B_e(\xi_\tau^e) = 0 ]$$
- $T_e$: 是否达成攻击者目标（如碗在盘子上）。
- $B_e$: 是否完成基准任务（如碗在炉子上）。

**4. 追踪界 (Tracking Bound)**

这是证明轨迹级攻击可行性的核心理论。设 $u^\star(s)$ 是一个能达成攻击者目标的参考控制器，$u^\tau(s)$ 是候选提示 $\tau$ 的动作。定义在状态 $s_k^\tau$ 上的动作不匹配度为 $\epsilon_k^\tau = \|u^\tau(s_k^\tau) - u^\star(s_k^\tau)\|$。假设环境动力学 $F_m$ 和参考控制器 $u^\star$ 是Lipschitz连续的，则最终状态差异 $\delta_K = \|s_K^\tau - s_K^\star\|$ 可以被追踪界约束：
$$\delta_K \le L_u \sum_{k=0}^{K-1} \alpha^{K-1-k} \epsilon_k^\tau$$
其中 $\alpha = L_s + L_u L_\star$，$L_s, L_u, L_\star$ 分别是 $F_m$ 和 $u^\star$ 的Lipschitz常数。
**含义**：如果攻击提示 $\tau$ 在其自身访问的状态序列上，其动作与一个能达成攻击者目标的参考行为足够接近（即 $\epsilon_k^\tau$ 足够小），那么它的最终物理状态 $s_K^\tau$ 也会与目标状态 $s_K^\star$ 足够接近。这个界还表明，早期的动作不匹配（$\epsilon_0^\tau$）由于系数 $\alpha^{K-1}$ 更大，对最终结果影响更大。

**5. 在策略教师匹配搜索目标 (On-Policy Teacher-Matching Search Objective)**

对于候选提示 $\tau$，在数据集 $D_i$ 上的评分函数为：
$$\hat{J}_i(\tau) = \frac{1}{\sum_r w_r} \sum_{(o_r, \cdot, \cdot, w_r) \in D_i} w_r \left[ \ell_{rank}(\tau, o_r) + \lambda_t d_t(\tau, o_r) \right] + \beta C_{text}(\tau, \tau_b)$$
其中：
- $\ell_{rank}(\tau, o) = \max\{0, d_t(\tau, o) - d_b(\tau, o) + \mu\}$ 是目标-良性边际损失，鼓励提示动作更接近目标教师。
- $d_t(\tau, o) = \ell_{pre}(A^\tau(o), A^t(o)) / \Delta(o)$ 是归一化的目标动作距离。
- $d_b(\tau, o) = \ell_{pre}(A^\tau(o), A^b(o)) / \Delta(o)$ 是归一化的良性动作距离。
- $\ell_{pre}(A, A') = \sum_{j=1}^m \alpha_j \|A_j - A'_j\|_2^2$ 是执行前缀损失。
- $\Delta(o) = \ell_{pre}(A^b(o), A^t(o)) + \eta$ 是归一化因子。

### 3.3 模型、训练与数据集

- **模型**：论文评估了9种不同的VLA架构，涵盖了当前主流的几种设计范式：
    - **离散动作Token预测**：OpenVLA, MolmoAct, GR00T-N1
    - **流匹配/扩散动作头**：π0.5, Octo, SmolVLA
    - **连续动作头**：OpenVLA-OFT, π0-FAST
    - **动作即文本**：VLA-0
- **训练**：所有模型均使用其公开的、在LIBERO数据集上微调过的版本。对于硬件实验，作者在SO-100数据集上对相应模型进行了微调。
- **数据集**：
    - **仿真**：LIBERO基准测试，包含Spatial, Object, Goal三个套件。
    - **硬件**：使用SO-100 6自由度机械臂，作者自己采集的数据集。

### 3.4 实验设置与结果

- **实验设置**：
    - **仿真**：在LIBERO环境中，固定一个场景和初始状态。选择一对“良性任务”（如“把碗放在炉子上”）和“攻击者目标任务”（如“把碗放在盘子上”），且两者在该场景下都是可行的。攻击者只能修改文本指令。
    - **硬件**：在SO-100机械臂上复现相同的协议。
    - **指标**：Clean SR（良性指令成功率）、Target-feasible SR（目标指令成功率）、**Attack ASR（攻击成功率）**、Bench fail（基准任务失败率）、Target final（最终状态达成目标率）、Edit（编辑距离）。
- **主要结果**：
    - **KF#1: 普遍性漏洞**：在LIBERO上，所有9种VLA架构都成功被攻击，其中7种的攻击成功率（Attack ASR）超过90%（见表1）。这表明该漏洞是VLA模型的一个共性缺陷。
    - **KF#2: 攻击由被破坏的目标位置表示承载**：因果追踪（图3）显示，攻击效果集中在被修改的目标词（如“staove”）的内部表示上。修复这些表示会消除攻击效果。
    - **KF#3: 攻击在自身产生的状态上保持目标导向**：图4显示，攻击提示不仅在初始帧匹配目标教师，而且在其自身滚动产生的整个轨迹上，其动作都更接近目标教师，尤其是在良性任务和目标任务需要不同动作的关键分支点。
    - **KF#4: 小扰动预算足够**：成功的攻击平均只需要3.4个字符的编辑（表1）。图5显示，更大的扰动预算可以降低搜索成本。
    - **KF#5: 攻击在真实机器人上有效**：硬件实验（图6）验证了攻击的有效性。在良性指令下，模型能可靠完成任务；但在对抗性指令下，原始任务的成功率急剧下降，机器人转而执行了攻击者的目标。
    - **KF#6: 防御需要指令级归一化**：表2显示，简单的文本预处理（如拼写纠正）效果有限，而将指令映射到一组规范的、经过验证的任务命令（Nearest-task canonicalization）可以最有效地防御此类攻击。

### 3.5 技术细节

- **候选提示生成**：结合了多种文本扰动方法，包括字符级编辑（插入、删除、替换）、基于梯度的搜索（如HotFlip）和基于遗传算法的变异。
- **指令保持约束的实现**：在附录B中详细说明了如何具体实现公式(2)中的四个约束，包括使用编辑距离作为 $C_{text}$，使用预定义的关键词列表和正则表达式来检测 $\text{Leak}$，以及通过人工标注或规则来判断 $\text{Preserve}$。
- **在策略聚合**：在每次迭代中，只有表现最好的几个候选提示的滚动数据被添加到数据集中。数据点的权重 $w_k$ 被设计为强调早期的重规划步，以及良性教师和目标教师动作差异大的观测。
- **提示最小化**：找到成功的攻击提示后，会尝试贪婪地移除扰动字符，并重新进行滚动测试，以找到最短的、仍然有效的攻击提示。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **提出了一个更强的
