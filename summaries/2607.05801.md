# TRIG: Trajectory-Rig Decoupled Metric Geometry Learning

- arXiv: https://arxiv.org/abs/2607.05801v1
- arXiv ID: 2607.05801

---

🔍 **核心问题**：现有视觉几何模型将多相机系统的位姿编码为耦合表示，无法有效利用车辆侧几何先验，限制了自动驾驶中度量几何与自运动估计的准确性。

🛠 **方法**：提出TRIG框架，将相机位姿分解为自车轨迹与相机刚性组件，通过解耦位姿编码与监督分别建模运动与静态拓扑，并采用稀疏时空注意力分离跨相机交互与时间聚合。

✨ **主要贡献**：在五个自动驾驶基准上实现位姿估计、度量深度预测和3D重建的最优性能，同时通过解耦设计降低全局注意力计算成本，提升几何推理一致性。
