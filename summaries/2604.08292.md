# EMMa: End-Effector Stability-Oriented Mobile Manipulation for Tracked Rescue Robots

- arXiv: https://arxiv.org/abs/2604.08292v1
- arXiv ID: 2604.08292

---

🔍 **核心问题**：履带式移动机械臂在救援任务中需兼顾运动可达性与安全性，但现有方法在规划和控制层面忽视了末端执行器的运动稳定性。

🛠 **方法**：提出运动生成框架，包括耦合末端与底盘状态的协调路径优化模型（通过紧凑代价/约束降低非线性与计算复杂度），以及带前馈补偿与反馈调节的隔离控制方案。

✨ **主要贡献**：在仿真与真实救援实验中，该方法在任务成功率与末端运动稳定性等关键指标上持续优于现有最优方法，验证了其在复杂移动操作中的有效性与鲁棒性。
