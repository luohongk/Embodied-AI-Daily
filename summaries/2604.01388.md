# LESV: Language Embedded Sparse Voxel Fusion for Open-Vocabulary 3D Scene Understanding

- arXiv: https://arxiv.org/abs/2604.01388v2
- arXiv ID: 2604.01388

---

🔍 **核心问题**：现有基于3D高斯泼溅（3DGS）的开放词汇3D场景理解方法存在空间模糊性（因高斯体非结构化重叠导致）和多级语义模糊性（因对象级掩码池化特征丢失细节）。

🛠 **方法**：提出一种新框架，采用稀疏体素光栅化（SVRaster）作为结构化、无重叠的几何表示，并利用单目深度和法线先验进行正则化，实现确定性、置信度感知的特征注册，同时利用AM-RADIO基础模型的密集对齐特性解决多级语义模糊。

✨ **主要贡献**：在开放词汇点云理解任务上达到最先进性能，并在3D和2D对象检索基准上取得极具竞争力的结果，有效抑制了3DGS中常见的语义渗漏伪影。
