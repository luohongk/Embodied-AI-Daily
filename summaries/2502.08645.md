# Re$^3$Sim: Generating High-Fidelity Simulation Data via 3D-Photorealistic Real-to-Sim for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2502.08645v4
- arXiv ID: 2502.08645

---

🔍 **核心问题**：解决机器人操作任务中真实数据采集成本高、仿真数据因几何与视觉差异难以迁移到真实场景的问题。  
🛠 **方法**：提出RE³SIM系统，结合3D重建与神经渲染技术，在物理仿真器中高保真复现真实场景，并利用特权信息高效收集专家演示数据，通过模仿学习训练机器人策略。  
✨ **主要贡献**：仅用仿真数据即可实现零样本的仿真到真实迁移，平均成功率超过58%；同时构建大规模仿真数据集，验证了策略对多种物体的泛化能力。
