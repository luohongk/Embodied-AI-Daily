# Select-to-Act: Hierarchical Reinforcement Learning via Adaptive Language Guidance

- arXiv: https://arxiv.org/abs/2606.22350v1
- arXiv ID: 2606.22350

---

🔍 **核心问题**：现有强化学习方法在利用自然语言指令时，往往将整个指令作为单一条件输入，忽略了语言指导在复杂环境中的阶段性依赖特性，导致样本效率低下。

🛠 **方法**：提出分层强化学习框架HRLLI，将指令分解为分段指导元素，通过高层语义策略动态选择与当前状态最相关的指令片段，并以此指导低层策略执行环境动作，两层策略联合训练以最大化累积期望回报。

✨ **主要贡献**：在指令密集型RTFM基准测试中，HRLLI显著优于强指令条件强化学习基线，证明了显式建模自适应指令选择能有效提升强化学习效果。
