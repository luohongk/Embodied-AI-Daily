# TransitNet: A Compact Attention-Augmented Deep Learning Framework for Low-SNR Transit Blind Searches

- arXiv: https://arxiv.org/abs/2606.18932v1
- arXiv ID: 2606.18932

---

🔍 **核心问题**：针对中等至长周期地球大小行星在低信噪比条件下观测不完整的问题，提出一种高效的凌星信号盲搜索方法。

🛠 **方法**：设计了一种紧凑的注意力增强深度学习框架TransitNet，并构建了统一的数据集构建、基准测试和阈值选择框架，以模拟真实盲搜索条件。

✨ **主要贡献**：在信噪比6-8的挑战性范围内达到95.2%的准确率，地球及亚地球大小凌星恢复率达93.0%，远超传统方法（TLS 63.1%、BLS 60.0%）；模型仅1.5 MB，推理速度比CPU-TLS快12-25倍，比CPU-BLS快4-5倍，且能准确估计凌星窗口和中心点。
