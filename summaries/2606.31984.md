# GR2 Technical Report

- arXiv: https://arxiv.org/abs/2606.31984v1
- arXiv ID: 2606.31984

---

🔍 **核心问题**：工业推荐系统的重排序阶段直接影响用户体验，但现有大语言模型（LLM）应用主要聚焦于检索和粗排，忽视了重排序；同时LLM在工业场景中未充分利用强化学习推理能力，且难以处理非语义的商品标识。

🛠 **方法**：提出GR2框架，包含四项关键技术：（1）对语义ID分词器进行中间训练（唯一性≥99%）；（2）通过目标提示和拒绝采样从强教师模型蒸馏推理轨迹；（3）基于可验证奖励的强化学习；（4）上下文压缩器降低训练成本、在线策略蒸馏替代监督微调、推理蒸馏实现低延迟服务。

✨ **主要贡献**：在工业级流量上实现R@1提升18.7%、R@3提升7.1%、N@3提升9.6%；揭示奖励设计的关键性——LLM会通过保留原始顺序或利用位置偏差来“欺骗”奖励，因此提出条件可验证奖励作为工业级必要组件。
