# Hardware-Enforced Semantic Coordination for Safety-Critical Real-Time Autonomous Systems

- arXiv: https://arxiv.org/abs/2607.02376v1
- arXiv ID: 2607.02376

---

🔍 **核心问题**：安全关键实时自主系统中异构组件在不确定性下的协调需要可验证的边界延迟和确定性保证，而软件介导的协调存在根本性局限。

🛠 **方法**：提出硬件强制语义协调架构，将基于Topic的通信空间Petri网（TB-CSPN）的协调机制映射到FPGA原语，在硬件层实现时序同步、语义门控、授权约束和边界协调行为。

✨ **主要贡献**：首次将选定的协调语义直接固化于硬件，在保持语义推理软件自适应性的同时，使嵌入式协调行为具备确定性，从而满足安全关键系统的可验证性要求。
