# The World Model Remembers, the Actor Forgets: Dream Rehearsal for Continual Model-Based RL

- arXiv: https://arxiv.org/abs/2607.19749v1
- arXiv ID: 2607.19749

---

🔍 **核心问题**：在基于模型的持续强化学习中，当智能体按任务序列训练时，即使保留所有过往经验，其行为策略仍会发生灾难性遗忘，而世界模型却几乎完全保留旧任务知识。研究旨在明确遗忘的根源并解决这一问题。

🛠 **方法**：通过组件级探针实验发现遗忘源于“通道问题”（行为策略而非记忆），提出“分级梦境排练”方法：冻结世界模型，利用其生成的带评分梦境轨迹进行监督式自我模仿训练，无需环境交互即可恢复遗忘技能。

✨ **主要贡献**：首次量化并定位持续强化学习中遗忘的组件（行为策略而非世界模型）；提出无需任务标签、参数固定的持续学习方法，在4任务和8任务链上均实现完全保留（基线方法完全失败）；引入梦境评分机制与离线选择标准，有效避免评分失败模式。
