# Scalable Behaviour Cloning on Browser Using via Skill Distillation

- arXiv: https://arxiv.org/abs/2606.32014v1
- arXiv ID: 2606.32014

---

🔍 **核心问题**：如何利用互联网用户的海量浏览器操作轨迹，规模化地训练浏览器智能体，解决其在信息不完整环境下的决策问题。

🛠 **方法**：通过技能蒸馏（skill distillation）将用户交互轨迹转化为紧凑的自然语言技能，并构建技能图（skill graph）实现技能的整合与复用，使智能体能够直接读取、检索、重用和组合这些技能。

✨ **主要贡献**：提出了一种可扩展的行为克隆方法，证明浏览器智能体的可扩展性可源于用户集体表达的技能而非人工设计任务，并开源了项目代码。
