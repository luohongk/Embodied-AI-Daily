# Signal Temporal Logic-Guided Model Predictive Control for Robust Bipedal Locomotion Resilient to Runtime External Perturbations

- arXiv: https://arxiv.org/abs/2310.11290v1
- arXiv ID: 2310.11290

---

🔍 **核心问题**：研究双足机器人在遭遇不可预知的外部扰动时，如何确保任务规范的正确性并最大化鲁棒性，尤其针对侧向扰动导致的腿部自碰撞风险。

🛠 **方法**：设计基于优化的任务与运动规划框架，核心是模型预测控制器（MPC），将信号时序逻辑（STL）任务规范编码为成本函数，并合成敏捷安全的交叉步态以应对侧向扰动。

✨ **主要贡献**：首次将STL的形式化保证引入双足机器人扰动恢复的轨迹优化中，通过仿真实验验证了框架在外部扰动下的有效性和鲁棒性。
