# Hybrid Cognitive IoT with Cooperative Caching and SWIPT-EH: A Hierarchical Reinforcement Learning Framework

- arXiv: https://arxiv.org/abs/2512.14488v1
- arXiv ID: 2512.14488

---

🔍 **核心问题**：针对混合底层-覆盖认知物联网（CIoT）网络，联合优化能量收集、混合频谱接入、功率分配与协作缓存，以最大化吞吐量和缓存命中率，同时最小化传输延迟。

🛠 **方法**：提出基于软演员-评论家（SAC）算法的三层分层深度强化学习框架（H-SAC），将混合离散-连续动作空间分解为高层（时间切换因子）、中层（频谱接入协调与缓存共享）和低层（发射功率与缓存决策）子问题。

✨ **主要贡献**：首次在统一框架中联合优化能量收集、混合接入、功率控制和缓存，H-SAC方法在平均和速率、延迟、缓存命中率和能效上显著优于基准策略，且对信道衰落和不确定性具有鲁棒性。
