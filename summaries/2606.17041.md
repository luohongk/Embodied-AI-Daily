# Benchmarking LLM Agents on Meta-Analysis Articles from Nature Portfolio

- arXiv: https://arxiv.org/abs/2606.17041v1
- arXiv ID: 2606.17041

---

🔍 **核心问题**：现有基准缺乏涵盖文献检索、筛选和统计聚合全流程的元分析评估标准，无法有效衡量LLM在系统性科学推理中的表现。

🛠 **方法**：构建MetaSyn数据集，包含442篇Nature Portfolio期刊的专家策展元分析，并设计12种流水线配置（9种RAG变体与1种协议驱动智能体）进行基准测试。

✨ **主要贡献**：揭示LLM在文献筛选环节存在关键瓶颈——即使检索召回率达90.9%，系统最多仅能恢复52.7%的真实纳入文献，且无法可靠区分PI/ECO合格研究与主题相似但不合格的干扰项。
