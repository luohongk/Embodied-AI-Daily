# CaT-GS: Efficient 3DGS Rendering for Large Scale Scenes via Inter-frame Caching and Tile Scheduling

- arXiv: https://arxiv.org/abs/2607.17842v1
- arXiv ID: 2607.17842

---

🔍 **核心问题**：针对大规模场景中3D高斯泼溅（3DGS）渲染因瓦片光栅化计算负担导致的性能严重下降，具体存在帧间预处理冗余、视点遮挡冗余和瓦片级负载不均衡三大效率瓶颈。

🛠 **方法**：提出CaT-GS渲染管线，通过推测性多帧预处理消除连续帧间的冗余计算，引入帧间缓存机制避免视点重复渲染，并重构光栅化任务以专用内核缓解瓦片负载不均衡，提升GPU利用率。

✨ **主要贡献**：相比原始3DGS实现最高10倍加速，较此前最优方法提升70%，为大规模场景的高保真实时渲染树立新基准。
