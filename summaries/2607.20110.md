# Extreme-RGMT: Continual Learning of Highly Dynamic Skills for Robust Generalist Humanoid Control

- arXiv: https://arxiv.org/abs/2607.20110v1
- arXiv ID: 2607.20110

---

🔍 **核心问题**：现有的人形机器人控制器难以兼顾通用运动跟踪能力与高动态技能的专业执行，通用策略在罕见高动态动作上表现差，而专业训练会损害已学技能。

🛠 **方法**：提出Extreme-RGMT，一种两阶段持续学习框架，先学习通用运动跟踪基策略，再通过非对称技能获取与能力巩固机制约束策略漂移，并结合难度感知采样与优势优先轨迹重采样，聚焦关键动态片段。

✨ **主要贡献**：实现了最先进的通用全身运动跟踪性能，显著提升高动态动作完成率，使控制器能直接执行未见的高动态运动，达到人类专家级水平。
