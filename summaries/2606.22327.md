# Geometry-Aware Online Scheduling for LLM Serving: From Theoretical Bound to System Practice

- arXiv: https://arxiv.org/abs/2606.22327v2
- arXiv ID: 2606.22327

---

🔍 **核心问题**：解决大语言模型推理中键值缓存动态内存占用导致的调度效率低下问题，现有时间中心调度策略（如最短作业优先）无法适应LLM推理特有的二维时空几何增长特性。

🛠 **方法**：提出几何感知在线调度算法——最小体积优先（SVF）及其高效变体1-bit SVF，通过引入体积证书证明理论，将最坏情况竞争比从48优化至3，并针对不同流量场景和信息可用性建立完整理论分类。

✨ **主要贡献**：首次将几何感知理论引入LLM调度，实现理论最优性突破（竞争比降至3）；在vLLM系统中无缝集成，实验表明SVF显著降低平均和尾部延迟，1-bit SVF仅需单比特信息即可达到竞争性吞吐量。
