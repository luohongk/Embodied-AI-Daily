# Extending LLM Context via Associative Recurrent Memory

- arXiv: https://arxiv.org/abs/2607.11614v1
- arXiv ID: 2607.11614

---

🔍 **核心问题**：标准Transformer模型在处理长文本时面临计算复杂度二次增长和内存线性扩展的瓶颈，难以高效支持超长上下文场景。

🛠 **方法**：提出关联递归记忆Transformer（ARMT），通过结合持续预训练、合成长上下文数据生成、课程学习以及选择性将关联记忆模块集成到特定模型层，实现长上下文处理。

✨ **主要贡献**：ARMT增强模型能处理远超原始上下文限制的输入且性能不降，对分布外上下文长度泛化更优，同时在原始上下文窗口内保持基线性能并减少30%的FLOPs。
