# Recursive Harness Self-Improvement

- arXiv: https://arxiv.org/abs/2607.15524v1
- arXiv ID: 2607.15524

---

🔍 **核心问题**：在模型与工具协同进化中，如何低成本、高效地优化用户构建的工具（harness），使其既能提升当前智能体性能，又能生成高质量的执行轨迹用于未来模型训练。

🛠 **方法**：提出递归工具自我改进（RHI），将工具表示为智能体循环的提示级规范，并利用其自身修订历史中的成对反馈进行迭代优化。

✨ **主要贡献**：在30个合成机器学习研究任务中，仅需少量RHI迭代即可显著提升低推理努力智能体的性能上限，超越高推理努力设置，同时降低高达60%的推理成本；并揭示了性能提升主要源于更有效的智能体间信息流，而非更长的推理轨迹。
