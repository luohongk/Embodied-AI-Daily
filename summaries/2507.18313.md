# Regression-aware Continual Learning for Android Malware Detection

- arXiv: https://arxiv.org/abs/2507.18313v2
- arXiv ID: 2507.18313

---

🔍 **核心问题**：在持续学习框架下，安卓恶意软件检测模型更新时会出现“安全回归”问题——即原本正确检测的恶意样本在更新后变为漏报，威胁用户信任，而传统遗忘指标无法捕捉这一风险。

🛠 **方法**：提出一种回归感知的持续学习框架，通过集成正一致训练（PCT）方法，与现有持续学习策略无缝结合，在模型更新时抑制恶意样本的预测反转。

✨ **主要贡献**：首次形式化定义并量化了持续学习中的安全回归问题（发现3-6%的恶意样本受影响），所提方法在ELSA、Tesseract和AZ-Class数据集上能将回归率减半，同时保持稳定的检测性能。
