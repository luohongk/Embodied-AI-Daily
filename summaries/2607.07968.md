# Soft Robotic Exogloves for Dexterous Mobility -- Towards Personalized Rehabilitation

- arXiv: https://arxiv.org/abs/2607.07968v1
- arXiv ID: 2607.07968

---

🔍 **核心问题**：传统软体机器人外手套依赖标准化尺寸，无法适应个体手部解剖差异，限制了精细操作和康复效果。

🛠 **方法**：通过拓扑扫描个性化定制手套，采用硅胶模具铸造制造，结合有限元分析（FEA）评估弯曲与接触力，并利用气动压力控制实现手指关节的精准运动。

✨ **主要贡献**：实现了从结构贴合、关节拓扑到人机接触力建模的全面个性化设计，并通过实验验证了压力控制对掌指关节和近端指间关节的精准驱动，为优化外手套以支持精细操作和神经肌肉康复奠定了基础。
