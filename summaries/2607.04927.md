# DSWAM: A Dual-System World Action Foundation Model for Fine-Grained Robot Manipulation

- arXiv: https://arxiv.org/abs/2607.04927v1
- arXiv ID: 2607.04927

---

🔍 **核心问题**：现有世界动作模型（WAM）缺乏语言级规划能力，难以将粗粒度指令分解为细粒度子任务，且缺乏与视觉-语言-动作（VLA）模型在真实机器人上的公平对比。

🛠 **方法**：提出DSWAM双系统框架，包含System 1的WAM执行器（默认控制路径）和System 2的视觉-语言子任务规划器（按需激活），通过动作预测与视频联合训练，并集成TensorRT加速、异步执行和实时分块技术提升实际部署效率。

✨ **主要贡献**：首次在DeMaVLA真实变形物体操作场景下，使用匹配的机器人平台、数据与评估标准，实现WAM与VLA的公平对比，验证了双系统设计在细粒度操作任务中的有效性。
