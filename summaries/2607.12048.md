# An Empirical Analysis of Continual Learning for Heterogeneous Medical Visual Question Answering

- arXiv: https://arxiv.org/abs/2607.12048v1
- arXiv ID: 2607.12048

---

🔍 **核心问题**：在真实临床环境中部署医学视觉问答（MedVQA）系统时，模型需要适应新任务而不遗忘旧知识，但现有持续学习方法在异构MedVQA任务（如分类、检测、计数、报告生成）中的表现尚未被充分研究。

🛠 **方法**：系统评估了多种持续学习方法在异构MedVQA任务上的性能，包括对灾难性遗忘的缓解能力、任务顺序的敏感性分析，以及低秩适应参数在不同方法下的权重漂移模式。

✨ **主要贡献**：揭示了现有持续学习方法在处理目标与监督格式差异大的任务时，难以维持稳定性-可塑性平衡，并提供了任务顺序影响性能保留与遗忘的实证分析。
