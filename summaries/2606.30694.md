# DSIP: A Dynamic Coordination Planner for Signal-Free Intersections using Diffusion-Model-Based Multi-Agent Motion Planning

- arXiv: https://arxiv.org/abs/2606.30694v1
- arXiv ID: 2606.30694

---

🔍 **核心问题**：传统信号灯控制导致车辆频繁启停，增加延误并降低交通效率，尤其在车流量大时问题更突出。  
🛠 **方法**：提出DSIP框架，基于扩散模型进行多智能体运动规划，将交叉口管理从离散相位控制转为连续多车轨迹优化。  
✨ **主要贡献**：在理想通信条件下，DSIP相比固定信号灯和强化学习方法显著降低平均延误并保持更高车速，为无信号交叉口管理提供可扩展、鲁棒的解决方案。
