# Optimal Base Station Placement for Beyond 5G Networks with Non-Convex Topology

- arXiv: https://arxiv.org/abs/2607.05210v1
- arXiv ID: 2607.05210

---

🔍 **核心问题**：在非凸拓扑的U形环境中，为超5G网络中的毫米波基站寻找最优部署位置，以最大化总速率和实现最大最小公平性，该问题为NP-hard且目标函数非凸非光滑。

🛠 **方法**：将基站部署问题建模为马尔可夫决策过程，并提出两种深度强化学习方法：一是将区域离散化为网格并用深度Q网络优化，二是将U形区域划分为连续子空间，每个子空间由深度确定性策略梯度智能体独立优化，最终选择最佳部署位置。

✨ **主要贡献**：最优部署实现了全覆盖和0.99的Jain指数；提出的分区多空间DDPG方法在获得优于DQN的解的同时，具有更低的计算复杂度。
