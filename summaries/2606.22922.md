# Hierarchical Reinforcement Learning for Sparse-Reward Search in Commutative Algebra

- arXiv: https://arxiv.org/abs/2606.22922v1
- arXiv ID: 2606.22922

---

🔍 **核心问题**：解决数学猜想验证中因奖励极度稀疏而难以应用机器学习的问题，具体以Kalai代数Hirsch猜想的反例构造为案例。  
🛠 **方法**：提出一种基于约束选项的分层强化学习框架，结合等变图神经网络策略，学习任务相关的时序抽象。  
✨ **主要贡献**：首次将分层强化学习应用于交换代数领域，在多种复杂度下显著优于经典强化学习算法和贪心搜索。
