# Reinforcement Learning for Versatile, Dynamic, and Robust Bipedal Locomotion Control

- arXiv: https://arxiv.org/abs/2401.16889v2
- arXiv ID: 2401.16889

---

🔍 **核心问题**：如何利用深度强化学习为双足机器人设计一个通用、动态且鲁棒的运动控制器，使其能够执行多种步态（如行走、跑步、跳跃和站立），并适应真实世界的复杂环境。

🛠 **方法**：提出一种基于强化学习的端到端控制方案，采用新颖的双历史架构（结合长期和短期机器人输入/输出历史），并通过任务随机化增强泛化能力和抗干扰性。

✨ **主要贡献**：在仿真和真实世界中，该控制器在多种动态技能上一致优于其他方法；成功部署于Cassie双足机器人，实现了鲁棒站立、灵活行走、400米短跑以及立定跳远和跳高等多样化技能，显著提升了双足机器人的敏捷性。
