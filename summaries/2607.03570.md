# Cross-Embodiment Robot Manipulation via a Unified Hand Action Space

- arXiv: https://arxiv.org/abs/2607.03570v1
- arXiv ID: 2607.03570

---

🔍 **核心问题**：现有机器人操作策略通常绑定特定机械手结构，难以在不同构型的机械手之间迁移学习到的行为。

🛠 **方法**：提出统一手部动作空间（UHAS），将机械手动作表示为标准球体的几何变形，并通过级联逆运动学（CIK）算法将共享表示映射到具体机械手的关节配置，利用强化学习在该空间中训练灵巧操作策略。

✨ **主要贡献**：在多种机械手（如Allegro、LEAP、Shadow Hand及MANO人手）上实现有效灵巧操作，支持零样本迁移到未见机械手、快速跨体微调，并在真实场景中成功部署，验证了UHAS表示能稳定控制并实现跨体策略迁移。
