# Causal Reward World Models: Zero-shot Reward Design for Automated Skill Generation

- arXiv: https://arxiv.org/abs/2606.23280v1
- arXiv ID: 2606.23280

---

🔍 **核心问题**：现有基于大语言模型的自动奖励设计方法依赖相关性驱动和迭代反馈，导致推理效率低且易引入虚假奖励成分，无法实现零样本奖励函数设计。

🛠 **方法**：提出因果奖励世界模型（CRWM），通过离线预训练在多任务交互数据中显式建模奖励组件与任务目标变量间的因果拓扑关系，并采用粗到细的预训练策略，结合显式机制解耦与置信度感知软融合模块，构建稳健可解释的因果骨架。

✨ **主要贡献**：首次将因果推理引入自动奖励设计，实现无需环境反馈的零样本奖励函数生成，在复杂连续控制任务中显著降低新技能获取延迟，性能匹配或超越现有方法，并展现出跨任务与跨机器人形态的强泛化能力。
