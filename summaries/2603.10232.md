# Hierarchical Task Model Predictive Control for Sequential Mobile Manipulation Tasks

- arXiv: https://arxiv.org/abs/2603.10232v1
- arXiv ID: 2603.10232

---

🔍 **核心问题**：如何让移动操作机器人高效、鲁棒地执行由高层任务规划器生成的连续任务序列，同时应对任务变化、奇异点和参考变化等挑战。

🛠 **方法**：提出一种基于非线性词典序优化的分层任务模型预测控制（Hierarchical-Task MPC）框架，通过有效利用机器人冗余性，实现与高层任务规划器的无缝衔接。

✨ **主要贡献**：相比现有任务优先级逆运动学控制方法，在任务变化、奇异点和参考变化场景下，分层轨迹跟踪性能平均提升42%；相比单任务架构，执行递送任务序列时路径更短，执行速度快2.3倍，并在9自由度移动操作机器人上通过真实实验验证。
