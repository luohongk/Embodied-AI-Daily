# SAMPLe: SAM-based Optimizer for Prompt Learning in VLMs

- arXiv: https://arxiv.org/abs/2607.05727v1
- arXiv ID: 2607.05727

---

🔍 **核心问题**：视觉-语言模型（如CLIP）的提示学习面临性能与泛化性的权衡——可学习提示在训练分布上过拟合，导致对未见数据的泛化能力下降。

🛠 **方法**：提出SAMPLe（锐度感知最小化提示学习），一种即插即用的优化器，通过考虑损失景观的锐度，在每一步动态平衡探索与利用，基于局部曲率和梯度特性调整优化状态。

✨ **主要贡献**：SAMPLe有效减少对已见分布的过拟合，提升对未见数据的适应性，集成到多种提示学习框架（如CoOp、CoCoOp等）后，一致优于现有优化器，成为鲁棒且模型无关的提示学习解决方案。
