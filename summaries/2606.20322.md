# Towards 3D karst underwater scene reconstruction from rotating sonar data

- arXiv: https://arxiv.org/abs/2606.20322v1
- arXiv ID: 2606.20322

---

🔍 **核心问题**：解决水下喀斯特环境因声纳数据稀疏、噪声大以及导航漂移导致的3D重建困难问题。

🛠 **方法**：采用连续时间SLAM方法校正轨迹漂移，并结合一种新型两阶段深度学习表面重建技术，生成可沉浸式导航的3D网格。

✨ **主要贡献**：提出完整的水下喀斯特管道重建流程，有效处理稀疏噪声数据与导航误差，为水文地质分析提供高精度、可交互的3D模型。
