# Parallel Dynamic Programming for Conic Linear Quadratic Control

- arXiv: https://arxiv.org/abs/2606.24632v1
- arXiv ID: 2606.24632

---

🔍 **核心问题**：线性二次型（LQ）控制问题在模型预测控制中至关重要，但传统求解方法本质上是串行的，无法充分利用现代多核CPU的并行计算能力，限制了实时可扩展性。

🛠 **方法**：提出一种基于“分而治之”策略的并行时间域方法，利用交替方向乘子法（ADMM）将锥形最优控制问题分解，并将ADMM的内部原始更新转化为LQ问题，沿时间轴拆分后，通过动态规划推导出Riccati递归的变体，实现各子问题的并行求解。

✨ **主要贡献**：在多核CPU硬件上，针对两个实际应用场景的数值基准测试表明，该方法相比现有相关方法实现了高达5倍的加速效果。
