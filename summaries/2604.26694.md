## 0. 概览（Concise Summary）

- **主题（Topic）**：具身智能（Embodied AI）中的统一世界动作模型（Unified World Action Model），旨在同时进行机器人动作预测和4D世界（视频+3D重建）生成。
- **问题（Problem）**：现有的统一世界模型（如UWM）局限于2D像素空间，缺乏3D几何感知能力，导致生成的未来世界在物理上不真实，且难以在动作执行效率和高质量世界生成之间取得平衡。
- **方法（Method）**：提出X-WAM框架，通过两个核心设计解决上述问题：1）**轻量级深度适应模块**：复制预训练视频扩散Transformer（DiT）的最后几层作为独立的深度预测分支，在不破坏预训练知识和不增加序列长度的情况下注入3D感知。2）**异步噪声采样（ANS）**：在训练和推理阶段，对视频和动作模态采用耦合的、非对称的去噪时间步长，使得动作可以快速解码（低延迟），而视频可以高质量生成。
- **创新（Innovation）**：
    1.  首次将显式的3D空间感知（深度预测）融入统一的视频-动作联合建模框架。
    2.  提出轻量级的“单边注意力”深度分支设计，高效且不破坏预训练模型。
    3.  提出异步噪声采样（ANS）策略，通过耦合训练分布与推理分布，同时优化了动作解码效率和视频生成质量。
- **意义（Significance）**：X-WAM证明了一个单一的框架可以同时作为高性能的策略模型、高保真的视频生成器和精确的3D重建系统，为构建更通用、更智能的具身智能体提供了新范式。它在RoboCasa和RoboTwin 2.0基准测试中取得了领先的成功率，并生成了高质量的4D重建结果。
- **一句话总结**：本文提出了X-WAM，一个通过轻量级深度分支和异步噪声采样技术，将2D视频-动作统一模型提升为具备3D空间感知能力的4D世界动作模型，在策略执行、视频生成和3D重建三个维度上均取得了显著提升。

## 1. 研究动机（Motivation）

当前具身智能领域的研究主要分为两个范式：
1.  **策略模型（Policy Models）**：如VLA模型，专注于从观察到动作的直接映射，擅长指令跟随和语义推理，但缺乏对动作如何在真实3D世界中连续展开的几何直觉和物理意识。
2.  **世界模型（World Models）**：专注于模拟未来的观测（如视频），能生成逼真的物理动态预测，但不能直接产生可执行的动作。

近期的工作尝试构建**统一世界动作模型（Unified World Action Models）**，如UWM，将视频生成和动作预测结合在单一框架中，取得了初步成功。然而，这些模型存在一个**关键空白**：它们仍然局限于**2D像素空间**，缺乏显式的空间感知和3D几何基础。由于物理世界本质上是三维的，这种限制导致模型会“幻觉”出物理上不合理的未来，并且无法进行几何上精确的3D重建。

因此，本文的研究动机是：**将统一世界动作模型从2D像素预测器提升为具备空间感知的4D动力学模拟器**，使其能够同时解决生成、重建和策略执行三个问题。这面临两个核心挑战：
1.  **如何在不破坏预训练知识或引入巨大计算开销的情况下，有效地将3D感知注入模型？** 简单地将深度图作为额外token拼接会加倍序列长度（注意力计算成本二次增长），而沿通道维度融合则会改变输入分布，增加学习难度。
2.  **如何解决联合生成高维视频轨迹和低维机器人动作时固有的模态不匹配问题？** 高质量视频需要大量去噪步，而低维动作只需少量步即可准确恢复。如何在训练和推理中协调这种差异，以同时实现快速动作解码和高保真视频生成？

## 2. 创新点（Innovation）

本文的核心创新点在于提出了一个统一的4D世界动作模型X-WAM，其关键创新包括：

1.  **轻量级深度适应模块（Lightweight Depth Adaptation）**：
    - **新意**：与以往将深度信息作为额外输入或输出token的方法不同，本文通过**复制预训练DiT的最后M个块**，构建一个独立的、与主分支**交错执行**的深度预测分支。
    - **不同之处**：该分支通过**单边注意力（Unilateral Attention）** 机制，只能从主分支读取信息，而主分支不受其影响。这严格保留了预训练权重的完整性，避免了序列长度爆炸或输入分布偏移的问题。
    - **为何奏效**：该设计基于一个基本假设：深度信息可以从RGB特征中推断出来，无需完全独立的生成过程。通过轻量级的微调，模型学会了从共享的RGB特征中提取几何信息，从而高效地实现了3D空间建模。

2.  **异步噪声采样（Asynchronous Noise Sampling, ANS）**：
    - **新意**：提出一种新的训练和推理噪声调度策略。在推理时，动作使用较少的去噪步数（$T_a$）快速解码，而视频使用更多的步数（$T_O$）进行高质量生成。在训练时，不同于以往对视频和动作进行**独立随机采样**，ANS进行**耦合采样**。
    - **不同之处**：以往的解耦采样（Decoupled Sampling）会导致训练时出现推理时永远不会出现的噪声组合（例如视频噪声水平低于动作噪声水平，$t_O < t_a$），造成训练效率低下。ANS通过从联合分布中采样，确保训练分布与推理分布一致。
    - **为何奏效**：通过耦合采样，模型在训练阶段就学会了在动作已经“干净”的情况下继续对视频进行去噪，这与推理时的异步去噪过程完美匹配，从而在实现4.5倍加速的同时，保持了与同步推理相当的生成质量。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

X-WAM的整体架构如图2(a)所示，其核心是一个从预训练视频扩散模型（Wan2.2-TI2V-5B）微调而来的Diffusion Transformer（DiT）。

- **输入**：语言指令 $c$，初始机器人状态 $s_0$，多视角初始RGB观测 $O_0$。
- **输出**：未来RGB视频 $O_{1:H}$，深度视频 $D_{1:H}$，未来机器人状态 $s_{1:H}$，未来动作 $a_{1:K}$。其中 $H=8$，$K=32$，动作频率是视频帧率的4倍。
- **编码**：
    - RGB视频通过预训练的因果VAE编码器 $E$ 编码为潜在表示 $z_O$。
    - 机器人状态和动作通过可学习的MLP投影到潜在空间：$z_s = \text{MLP}_s(s)$，$z_a = \text{MLP}_a(a)$。
- **统一序列**：三种模态的潜在表示被拼接成一个统一的去噪序列 $Z = [z_{O_0}, z_{O_{1:H}}, z_{s_0}, z_{s_{1:H}}, z_{a_{1:K}}]$，通过双向全注意力进行处理。初始观测和状态作为条件，其噪声时间步设为 $t=0$。
- **多视角处理**：为每个视角添加可学习的视角嵌入（View Embedding），以兼容多视角输入，同时不破坏预训练的位置编码。
- **相机位姿**：通过预测末端执行器位姿 $T_{ee}$，并结合固定的手眼标定矩阵 $T_{h2e}$ 推导出腕部相机位姿 $T_{wrist} = T_{ee} \cdot T_{h2e}$，从而将多视角RGB-D信息融合到统一的3D坐标系中。

### 3.2 关键算法与数学推导

**1. 轻量级深度适应模块（单边注意力）**

给定一个有 $N$ 个DiT块的模型，复制最后 $M$ 个块作为深度分支。前 $N-M$ 个共享块产生隐藏状态 $H$。深度分支和主分支以交错方式执行。对于第 $j$ 层 ($j \in \{1, \dots, M\}$)：

$$Z^{(j)}_D = \text{DepthBlock}_j \left( Z^{(j-1)}_D \mid Z^{(j-1)}_m \right)$$

$$Z^{(j)}_m = \text{DiTBlock}_{N-M+j} \left( Z^{(j-1)}_m \right)$$

其中，$\text{DepthBlock}_j$ 通过**交叉注意力**关注主分支的输入 $Z^{(j-1)}_m$，而主分支 $\text{DiTBlock}$ 不受深度token的影响。这种不对称的连接称为**单边注意力**。

**2. 异步噪声采样（ANS）**

- **异步推理**：分配 $T_a$ 步给状态和动作，$T_O$ 步给视频（$T_a < T_O$）。两者都从纯噪声开始，分别以 $1/T_a$ 和 $1/T_O$ 的步长去噪。$T_a$ 步后，动作被解码并执行。剩余 $T_O - T_a$ 步，动作作为干净模态，继续生成视频。
- **耦合噪声采样（训练）**：视频和动作的噪声水平 $(t_O, t_a)$ 从以下混合分布中采样：

$$(t_O, t_a) \sim \begin{cases}
t_a = 0, \ t_O \sim U(0,1) & \text{w.p. } p \\
t_a \sim U(0,1), \ t_O = t_a + (1 - t_a) \cdot b, \ b \sim \text{Beta}(1.5, 1) & \text{w.p. } 1-p
\end{cases}$$

    - 第一种情况（概率 $p$）：模拟动作条件视频生成（动作无噪声）。
    - 第二种情况（概率 $1-p$）：模拟异步联合生成。$t_O$ 被采样为**条件于** $t_a$，且偏向更高的噪声水平（因为视频需要更多去噪步），通过Beta分布实现。

**3. 训练目标**

模型使用流匹配（Flow Matching）框架训练，预测速度场 $v = \epsilon - z_0$。对于模态 $m \in \{O, s, a\}$，损失为：

$$\mathcal{L}_m = \left\| f^m_\theta(z^m_{t_m}, t_m) - (\epsilon^m - z^m_0) \right\|^2$$

深度分支使用MSE损失：$\mathcal{L}_{\text{depth}} = \left\| \hat{D} - D^* \right\|^2$

总损失：$\mathcal{L}_{\text{total}} = \mathcal{L}_O + \lambda_s \mathcal{L}_s + \lambda_a \mathcal{L}_a + \lambda_D \mathcal{L}_{\text{depth}}$

### 3.3 模型、训练与数据集

- **基础模型**：基于预训练的Wan2.2-TI2V-5B视频扩散模型。
- **训练数据**：超过5800小时的机器人数据，涵盖真实机器人和模拟数据集，包含多种操作任务。所有数据被预处理并统一到一致的坐标系和表示中。
- **状态与动作表示**：定义了通用的接口，将状态表示为双臂末端执行器的位姿和夹爪位置，动作表示为相应的变化量。对于单臂机器人，将单臂视为左臂。
- **推理**：使用UniPC多步调度器，为视频和状态/动作模态维护独立的调度器实例，遵循异步推理过程。

### 3.4 实验设置与结果

- **策略评估**：
    - **基准**：RoboCasa（24个任务）和RoboTwin 2.0（50个任务，含Clean和Randomized设置）。
    - **基线**：VLA模型（π0, GR00T-N1.5, π0.5）和WAM模型（UWM, DreamZero, Cosmos Policy, GigaWorld-Policy, Motus）。
    - **结果**：
        - **RoboCasa**：X-WAM平均成功率为**79.2%**，远超最强基线Cosmos Policy（67.1%）（表1）。
        - **RoboTwin 2.0**：X-WAM在Clean和Randomized设置下分别达到**89.8%** 和**90.7%**，均优于Motus（88.7%/87.0%）（表2）。

- **4D重建与生成评估**：
    - **基准**：在RoboCasa环境中，比较预测的多视角RGB-D观测与模拟器渲染的真值。
    - **指标**：视觉保真度（PSNR, SSIM, LPIPS），深度质量（AbsRel, δ1），点云质量（Chamfer Distance）。
    - **基线**：DreamZero + Depth Anything 3（DA3）的两阶段方法，以及Robot4DGen。
    - **结果**：
        - X-WAM在所有指标上均达到最佳（表3）。与DreamZero + DA3相比，PSNR提升2.34 dB，Chamfer Distance从0.0680降至**0.0049**，证明了端到端联合建模的优越性。

- **消融实验**：
    - **深度架构设计**（表4(a)）：对比了无深度、序列拼接、通道拼接和交错分支。交错分支在保持与无深度版本相同延迟（1033ms）的同时，显著提升了成功率和重建质量。无深度版本成功率下降至63.0%，证明了显式空间建模的重要性。
    - **噪声调度策略**（表4(b)）：对比了同步/异步推理与解耦/耦合训练。异步推理将延迟从4665ms降至1033ms（4.5倍加速）。ANS（耦合训练+异步推理）在相同延迟下取得了最高的成功率和最佳的重建质量，优于解耦训练方案。

### 3.5 技术细节

- **深度图处理**：单通道深度图被复制三份形成伪RGB图像，使用与RGB相同的因果VAE进行编解码。
- **状态与动作的时序对齐**：状态与视频帧在时间上对齐，以便进行逐帧的多视角RGB-D融合。动作在相同时间跨度内均匀分布，频率是视频帧率的4倍。
- **位置编码**：使用3D旋转位置编码（RoPE）。对于状态和动作token，应用与视频token相同的时间维度RoPE，模型通过位置邻近性推断它们之间的时间对应关系。
- **训练细节**：使用流匹配框架，并采用ANS策略采样噪声时间步。深度分支的权重系数 $\lambda_D$ 用于平衡损失。

## 5. 澄清与简化（Clarifications and Simplifications）

- **“单边注意力”的类比**：可以把主分支（RGB）想象成一个正在专心画画的大师，深度分支（Depth）是一个学徒。学徒可以随时偷看大师的画作（通过交叉注意力）来学习如何画好阴影（深度），但大师完全不受学徒的影响，继续专注于自己的创作。这样，大师的绘画风格（预训练知识）被完美保留，而学徒也高效地学会了新技能。
- **“异步噪声采样”的类比**：想象一个厨师同时做两道菜：一道是复杂的法国大餐（视频），需要很多步骤；一道是简单的沙拉（动作），几步就能完成。
    - **同步方式**：厨师必须等沙拉和法国大餐的每一步都完成后，才能进行下一步。这样沙拉做好了也得等着，效率低下。
    - **解耦方式**：厨师先按沙拉的节奏做，再按法国大餐的节奏做，但训练时是分开学的，导致他从未练习过“沙拉已经做好后，如何继续做法国大餐”这个场景。
    - **ANS方式**：厨师在训练时就练习“先快速做好沙拉，然后专心致志地完成法国大餐”这个完整流程。这样在真正服务时，他就能在沙拉做好后立刻上菜（执行动作），同时继续完美地完成法国大餐（生成视频）。这就是ANS的精髓：**训练和推理的流程保持一致**。
