# Consistent Bayesian Spatial Domain Partitioning Using Predictive Spanning Tree Methods

- arXiv: https://arxiv.org/abs/2508.08324v2
- arXiv ID: 2508.08324

---

🔍 **核心问题**：现有贝叶斯空间聚类方法在预测能力、子区域形状约束及后验聚类一致性理论上存在不足，尤其缺乏对无限聚类数下分区一致性的理论分析。

🛠 **方法**：提出空间域随机分区模型（Spat-RPM），基于生成树切割将空间域划分为不连续块，实现贝叶斯空间聚类回归，并在填充域渐近框架下引入新距离度量研究分区后验集中性。

✨ **主要贡献**：首次证明Spat-RPM在聚类数可趋于无穷时仍能一致估计域分区，推导分区、参数和预测的后验集中速率，并给出超参数选择的理论条件，通过模拟与大西洋海洋数据验证了模型有效性。
