# RMTL: Reinforced Micro-task Learning for Long-Horizon Manipulation with VLM Rewards

- arXiv: https://arxiv.org/abs/2606.26175v1
- arXiv ID: 2606.26175

---

🔍 **核心问题**：针对长时域机器人操作任务中，单一全局提示的视觉语言模型（VLM）奖励信号过于粗糙，导致智能体难以检测早期进展的问题。

🛠 **方法**：提出强化微任务学习（RMTL），将操作任务分解为多个语言描述的微任务，智能体通过分层策略在微任务间切换，并利用多视角VLM奖励和反向课程学习逐步提升难度。

✨ **主要贡献**：通过微任务分解和多视角奖励设计，提供更密集有效的奖励信号，显著加速学习过程，并实现从规则到完全学习的分层策略转换，提升语言引导强化学习的可扩展性。
