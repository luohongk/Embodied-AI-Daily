# Qwen-RobotManip Technical Report: Alignment Unlocks Scale for Robotic Manipulation Foundation Models

- arXiv: https://arxiv.org/abs/2606.17846v1
- arXiv ID: 2606.17846

---

🔍 **核心问题**：如何将语言和多模态领域通过统一对齐与规模化训练实现强泛化的方法，应用于机器人操作领域，以克服操作数据异构、昂贵且多样性不足的挑战。

🛠 **方法**：提出Qwen-RobotManip，基于Qwen-VL构建视觉-语言-动作基础模型，引入统一对齐框架（覆盖表征、运动和动作维度），使大规模多源训练协调一致；并设计人-机合成流水线将第一人称手部演示转化为跨15种平台的机器人轨迹，通过严格筛选构建约38,100小时的预训练语料。

✨ **主要贡献**：仅使用开源数据和人类视频，无需专有数据收集，即实现零样本指令跟随、扰动鲁棒性、错误恢复和跨本体迁移等涌现泛化能力；在RoboCasa365、LIBERO-Plus等多项OOD基准上显著超越先前最优模型（如π0.5），在RoboChallenge中排名第一且相对提升20%，并在AgileX ALOHA、Franka等真实机器人平台上验证有效。
