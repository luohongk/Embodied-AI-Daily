# Lost in Visual Translation: A VLM-Assisted Perceptual-Semantic Coherence Framework for EEG-to-Image Reconstruction

- arXiv: https://arxiv.org/abs/2607.12364v1
- arXiv ID: 2607.12364

---

🔍 **核心问题**：现有图像质量评估指标（如SSIM、LPIPS、CLIP）无法区分脑电信号重建图像中的视觉保真度与语义可恢复性，常错误惩罚语义正确但模糊的重建结果，或奖励视觉相似但语义错误的结果。

🛠 **方法**：提出BCI感知框架，利用四个视觉语言模型（VLMs）通过结构化问题评估图像对，生成容忍感知对齐分数（T-PAS）和容忍语义对齐分数（T-SAS），并蒸馏为紧凑评估器BCI一致性分数（BCS）。

✨ **主要贡献**：BCS在T-PAS上MAE为0.079（r=0.700），T-SAS上MAE为0.082（r=0.850），人类验证显示高度可靠的一致性判断（Cohen's kappa=0.882），强调感知-语义可恢复性而非通用视觉相似性。
