# Contour Errors: Ego-Centric Matching for 3D Multi-Object Tracking Performance Evaluation

- arXiv: https://arxiv.org/abs/2506.04122v4
- arXiv ID: 2506.04122

---

🔍 **核心问题**：现有3D多目标跟踪评估中，IoU对朝向误差过度敏感，而中心点距离完全忽略朝向，导致无法从自车视角有效惩罚平移、形状和朝向误差。

🛠 **方法**：提出Contour Errors（CE）作为自车中心匹配准则，利用Hausdorff型推理对稀疏边界框角点几何进行k近邻自车角点选择，实现介于IoU（过度惩罚）与CPD（忽略朝向）之间的分级朝向敏感性。

✨ **主要贡献**：在nuScenes数据集上，标准IoU车辆阈值下，47%的汽车和75%的行人CE有效匹配被IoU拒绝，而仅不到0.1%的IoU有效匹配未通过CE，证明CE能显著提升安全关键自动驾驶中开环感知评估的准确性。
