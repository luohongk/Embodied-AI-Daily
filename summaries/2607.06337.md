# OrchardBench: A Physically-Grounded, GPU-Parallel Apple-Orchard Simulation Benchmark for Agricultural Robotics

- arXiv: https://arxiv.org/abs/2607.06337v1
- arXiv ID: 2607.06337

---

🔍 **核心问题**：农业机器人领域的果园实地实验成本高、不可重复，且现有仿真缺乏物理真实性和树木模型，阻碍了自动化采摘技术的发展。

🛠 **方法**：提出OrchardBench，基于Newton物理引擎的GPU并行苹果园仿真平台。树木由随机L系统生长并实例化为全铰接体，树枝采用欧拉-伯努利梁理论的扭转弹簧阻尼器，苹果通过茎绳连接并依据文献拉力分离，树叶层可移动且密度可控，所有物理参数均源自公开文献。

✨ **主要贡献**：实现了在笔记本GPU上以交互速率运行多并行环境的物理仿真，定义了涵盖采摘完整性、吞吐量和植物损伤的评估指标，并提供了基线结果（约40%检测成功率，仅采摘约1/8可达果实），为新型自主方法留出显著提升空间。
