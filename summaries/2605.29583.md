# BitC-3DGS: High-Capacity 3D Gaussian Splatting Watermarking via Bit Compression

- arXiv: https://arxiv.org/abs/2605.29583v1
- arXiv ID: 2605.29583

---

🔍 **核心问题**：现有基于文本编码器的3D高斯泼溅水印方法受限于CLIP的77个token长度，无法嵌入超过77位的高容量消息（如128位），限制了大规模3D资产中的信息嵌入能力。

🛠 **方法**：提出BitC-3DGS框架，采用位压缩标记化方案将多个消息位编码为单个语义token，并设计双分支架构联合进行块解压缩和位解码，同时引入硬消息采样策略增强解码器训练时的组合覆盖。

✨ **主要贡献**：首次实现3D高斯泼溅中的高容量水印（支持128位消息），在Blender和LLFF数据集上达到与现有64位方法相当的恢复精度，同时保持高渲染保真度。
