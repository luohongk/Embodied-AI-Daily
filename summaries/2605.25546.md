# Safety-Critical Whole-Body Control for Humanoid Robots via Input-to-State Safe Control Barrier Functions

- arXiv: https://arxiv.org/abs/2605.25546v1
- arXiv ID: 2605.25546

---

🔍 **核心问题**：人形机器人在复杂人类环境中运行时，需满足关节限位、自碰撞避免、障碍物规避等物理安全约束，但现有方法在模型不确定、轨迹跟踪误差和外部扰动等未知干扰下，运动学安全保障会退化。

🛠 **方法**：提出基于输入到状态安全控制屏障函数（ISSf-CBF）的分层安全关键全身控制框架，集成运动学级全身控制器（KinWBC）、ISSf-CBF安全滤波器和动力学级全身控制器（DynWBC），通过保守调参将运动学安全保证传递至全阶人形动力学。

✨ **主要贡献**：在仿真和真实机器人实验中，该框架在模型失配下提升了安全裕度，并在行走、遥操作和单腿平衡等任务中实时可靠地强制执行多重安全约束。
