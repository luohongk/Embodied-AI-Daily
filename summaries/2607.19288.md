# No Training, Better Flights: Test-Time Scaled VLMs for UAV Navigation

- arXiv: https://arxiv.org/abs/2607.19288v1
- arXiv ID: 2607.19288

---

🔍 **核心问题**：现有无人机视觉语言导航（VLN）方法在复杂环境中单次推理易产生次优或不安全轨迹，缺乏自我修正能力。

🛠 **方法**：提出无需额外训练的测试时缩放策略，通过迭代优化过程增强导航推理：首先生成多个并行候选路径，再执行自修正步骤，并设计多准则评分函数（安全性、目标对齐、前进进度）评估候选方案。

✨ **主要贡献**：首次将测试时缩放应用于无人机VLN任务，使冻结的VLM模型能自我修正并生成更准确可靠的飞行计划，在该任务上达到最优性能（SOTA）。
