# ReWorld: Learning Better Representations for World Action Models

- arXiv: https://arxiv.org/abs/2606.27504v1
- arXiv ID: 2606.27504

---

🔍 **核心问题**：现有世界动作模型（WAMs）主要关注架构设计，但如何高效学习更好的世界表征以支持自动驾驶规划，这一问题尚未被充分探索。

🛠 **方法**：提出ReWorld框架，将中间表征作为直接优化目标，从三个维度进行塑造：对生成模块的视频DiT施加未来预测监督，对规划模块的动作DiT进行跨模态对齐与安全边界判别性增强。

✨ **主要贡献**：在nuScenes和NAVSIM上，ReWorld将视频生成FVD降低23.9%（81.3→61.9），闭环PDMS从89.1提升至90.4（无需后训练），并加速从头训练收敛约2倍。
