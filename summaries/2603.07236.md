# HY-WU (Part I): An Extensible Functional Neural Memory Framework and An Instantiation in Text-Guided Image Editing

- arXiv: https://arxiv.org/abs/2603.07236v2
- arXiv ID: 2603.07236

---

🔍 **核心问题**：现有模型在部署后面对动态变化的目标（如领域漂移、用户偏好演变和新任务出现）时，基于静态权重的适应方式会导致参数冲突、干扰或过度专业化，难以实现持续学习和即时个性化。

🛠 **方法**：提出HY-WU（Weight Unleashing）框架，采用记忆优先的适应策略，通过一个神经模块（即权重生成器）根据实例条件实时合成权重更新，生成实例特定的算子，无需测试时优化。

✨ **主要贡献**：将适应压力从覆盖单一共享参数点转移到功能级记忆模块，避免了传统方法中学习行为退化的问题，为持续学习和个性化提供了可扩展的架构基础。
