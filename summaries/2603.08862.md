# APPLV: Adaptive Planner Parameter Learning from Vision-Language-Action Model

- arXiv: https://arxiv.org/abs/2603.08862v2
- arXiv ID: 2603.08862

---

🔍 **核心问题**：移动机器人在高度受限环境中自主导航时，传统方法需手动调参且泛化性差，端到端学习缺乏精确控制，现有VLA模型存在推理延迟和精度不足的问题。

🛠 **方法**：提出APPLV框架，利用预训练视觉-语言模型结合回归头预测经典规划器的参数，而非直接输出动作；并设计监督学习微调（从导航轨迹学习）和强化学习微调（优化导航性能）两种训练策略。

✨ **主要贡献**：在模拟BARN数据集和真实机器人实验中，APPLV相比现有方法在导航性能和未知环境泛化能力上均表现更优，同时保留了经典规划器的安全性与精确控制优势。
