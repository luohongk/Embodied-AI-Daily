# WSA$_1$: a 3D-Centric World-Spatial-Action Model for Generalizable Robot Control

- arXiv: https://arxiv.org/abs/2607.03941v1
- arXiv ID: 2607.03941

---

🔍 **核心问题**：现有机器人基础模型缺乏对3D物理世界动态和因果效应的推理能力，导致2D视觉感知与3D具身交互之间存在根本性不匹配，严重限制了模型在真实任务中的泛化能力。

🛠 **方法**：提出WSA₁模型，基于3D-Centric World-Spatial-Action建模范式，学习3D世界感知的视觉思维以预测未来行为，并建模3D世界状态变化与机器人动作之间的相互约束。

✨ **主要贡献**：仅使用6千小时专家演示数据（其中仅1千小时来自真实机器人）即可实现高效预训练，在RoboTwin2.0仿真基准上达到93%成功率，并在真实机器人控制任务中平均性能提升20%以上，证明无需大规模真实数据即可实现可泛化的机器人基础模型。
