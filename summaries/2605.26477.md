# Variational Inference for Evidential Deep Learning

- arXiv: https://arxiv.org/abs/2605.26477v2
- arXiv ID: 2605.26477

---

🔍 **核心问题**：传统证据深度学习（EDL）存在两个缺陷：KL惩罚项仅抑制负类证据导致证据过高，且缺乏对Dirichlet参数设置的理论保证，从而削弱了模型量化不确定性的能力。

🛠 **方法**：提出变分推断证据深度学习（VI-EDL），通过变分推断重新构建证据学习框架，推导出证据下界（ELBO）来防止证据过度增长，并理论证明设置Dirichlet参数α=e+1可最小化该下界。

✨ **主要贡献**：建立了严格的泛化界，揭示了预测不确定性、特征与网络复杂度对泛化界的影响；在标准视觉和医学数据集上实现最先进性能，尤其在分布外检测、噪声检测和自动驾驶场景中表现优异。
