# Generative Skill Composition for LLM Agents

- arXiv: https://arxiv.org/abs/2606.32025v1
- arXiv ID: 2606.32025

---

🔍 **核心问题**：现有LLM智能体在技能库增长时难以联合选择技能的子集、数量和执行顺序，导致技能组合成为瓶颈。

🛠 **方法**：提出SkillComposer，将结构化技能组合形式化为任务条件技能序列预测，使用受约束的自回归解码器在单次解码中联合生成技能子集、数量和顺序。

✨ **主要贡献**：在GPT-5.2-Codex和Gemini-3-Pro-Preview上，SkillComposer将通过率分别提升23.1和18.2个百分点，超越top-3检索方法，以更低提示词成本达到黄金技能检索上限。
