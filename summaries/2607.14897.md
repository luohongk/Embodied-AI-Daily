# Selectivity Drives Efficiency: Dataset Pruning for Visual Place Recognition

- arXiv: https://arxiv.org/abs/2607.14897v1
- arXiv ID: 2607.14897

---

🔍 **核心问题**：视觉地点识别（VPR）依赖大规模数据集训练，导致存储和训练成本高昂，而现有数据集剪枝方法基于样本分类范式，忽略了VPR中图像对依赖的训练特性。

🛠 **方法**：提出面向VPR的地点级数据集剪枝框架，以每个地点为剪枝单元，引入两种互补指标——地点内多样性（IPD）和地点间相似性（IPS），联合评估地点的训练价值，从而构建紧凑且信息丰富的核心集。

✨ **主要贡献**：在多种剪枝比例下一致优于现有剪枝方法，并降低选择和训练成本；将约3.5倍GSV-Cities大小的合并数据集剪枝至可比规模后，核心集在MSLS-val和Nordland上分别达到94.5%和97.0%的R@1性能。
