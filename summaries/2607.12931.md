# ExToken: Structured Exploration for Efficient Vision-Language-Action Reinforcement Fine-tuning

- arXiv: https://arxiv.org/abs/2607.12931v1
- arXiv ID: 2607.12931

---

🔍 **核心问题**：当前视觉-语言-动作（VLA）模型在强化学习微调中面临探索停滞瓶颈，环境交互成本高昂，且轨迹多样性比数据量更影响样本效率。

🛠 **方法**：提出ExToken框架，通过将VLA策略条件化于从离线演示中提取的离散行为先验（即“探索令牌”），在轨迹收集时引导智能体探索不同行为模式；同时引入状态条件令牌选择器，在部署时自适应预测有效行为模式。

✨ **主要贡献**：在模拟和真实机器人操作任务中，ExToken显著加速收敛、提升任务性能，并在极低交互预算下保持强鲁棒性，有效缓解了VLA-RL的探索效率瓶颈。
