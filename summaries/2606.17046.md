# Geometric Action Model for Robot Policy Learning

- arXiv: https://arxiv.org/abs/2606.17046v1
- arXiv ID: 2606.17046

---

🔍 **核心问题**：现有机器人策略模型（如VLA、WAM）依赖2D图像或潜在空间，缺乏对3D几何结构的显式建模，难以处理需要精细接触的操控任务。

🛠 **方法**：提出几何动作模型（GAM），通过复用预训练的几何基础模型（GFM），在中间层插入因果未来预测器，将语言、本体感知和动作历史转化为未来潜在特征，再经GFM解码生成动作与几何信息。

✨ **主要贡献**：仅通过最小架构修改赋予GFM语言条件时序世界建模能力，在仿真和真实机器人操控基准测试中，相比现有基础模型基线，实现了更高精度、鲁棒性、速度和更轻量级的设计。
