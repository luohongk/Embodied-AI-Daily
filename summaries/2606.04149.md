# CoPark: Learning Reactive Parking via Self-Play

- arXiv: https://arxiv.org/abs/2606.04149v1
- arXiv ID: 2606.04149

---

🔍 **核心问题**：解决自动驾驶泊车中高精度几何目标与安全交互之间的冲突——单一策略难以同时兼顾精确到达车位和实时避让邻近车辆。

🛠 **方法**：提出CoPark，一种基于残差策略架构的多智能体自博弈强化学习方法。预计算离线计划提供固定动作先验，残差头学习反应式修正；通过伙伴威胁调制的通道非对称释放机制，纵向通道由残差头控制以让行，横向通道锚定参考以保持车位对准，并加入闭环细化层修正离散化误差。

✨ **主要贡献**：在六个停车场训练并在新基准上零样本测试，CoPark实现约70-85%成功率、仅3-6%碰撞率，显著优于经典、模仿学习和大规模RL基线，并涌现出倒车让行、中途让行、窄道通行和排队等交互行为。
