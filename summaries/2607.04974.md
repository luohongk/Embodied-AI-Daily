# A Comprehensive Study of Implementation Bugs in Multi-modal Agents

- arXiv: https://arxiv.org/abs/2607.04974v1
- arXiv ID: 2607.04974

---

🔍 **核心问题**：多模态智能体（M-agents）在动态、多模态环境中因实现缺陷（如感知过时、规划不可靠、执行不适用）可能导致严重事故或损失，但此前缺乏对其特有实现错误的系统性研究。

🛠 **方法**：收集34个代表性M-agents，从1,268份问题报告中筛选出158个特有缺陷，采用自上而下的策略构建分类体系（按全局症状、功能组件症状和根本原因分类），并开发自动化工具MATester，通过分析运行时组件间输出来识别缺陷。

✨ **主要贡献**：首次系统分类M-agent实现缺陷，提出完整分类法；MATester在12个额外M-agents上覆盖61.4%已知问题并发现31个新缺陷，为缺陷分类、预防和修复提供了实用指南。
