# FlowDec: Temporal Conditional Flow Decorruptor for Robust Continuous Vision-Language Navigation

- arXiv: https://arxiv.org/abs/2606.22424v1
- arXiv ID: 2606.22424

---

🔍 **核心问题**：在连续环境视觉语言导航（VLN-CE）中，现实世界的视觉干扰（如噪声、模糊等）会严重降低基于大语言模型的导航性能，而现有方法对此缺乏有效应对。

🛠 **方法**：提出FlowDec框架，通过混合时间条件策略将生成流路径与历史上下文对齐，并采用动作中心引导滤波动态评估和整合输出，实现鲁棒的图像恢复。

✨ **主要贡献**：FlowDec在导航准确率和生成延迟上均优于现有去干扰方法，为不可预测现实条件下的具身导航提供了高效、鲁棒的新范式。
