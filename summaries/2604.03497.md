# Sim2Real-AD: A Modular Sim-to-Real Framework for Deploying VLM-Guided Reinforcement Learning in Real-World Autonomous Driving

- arXiv: https://arxiv.org/abs/2604.03497v2
- arXiv ID: 2604.03497

---

🔍 **核心问题**：解决视觉语言模型引导的强化学习策略从仿真环境迁移到真实自动驾驶车辆时，因传感器、动力学和任务几何差异导致的部署鸿沟。

🛠 **方法**：提出Sim2Real-AD模块化框架，包含几何观测桥接、物理感知动作映射、两阶段渐进训练课程和实时部署管线，通过将真实感知与控制重投影到策略训练流形上，无需真实世界训练数据即可实现零样本迁移。

✨ **主要贡献**：首次在真实全尺寸电动货车（福特E-Transit）上零样本部署CARLA训练的VLM引导RL策略，成功完成跟车、避障和停车标志场景，并形式化证明了部署误差可由三个独立可控项约束。
