# Teaching Software Engineering with LLM and MCP Integration: From Classroom to Industry Practice

- arXiv: https://arxiv.org/abs/2606.19167v1
- arXiv ID: 2606.19167

---

🔍 **核心问题**：传统软件工程教育难以跟上工业界对LLM和MCP等新兴技术的需求，导致教学与行业实践脱节。  
🛠 **方法**：将LLM和MCP驱动的工具融入日常教学、代码辅助和工程模拟，并联合行业实习，构建与真实工程实践紧密相连的协作教学模式。  
✨ **主要贡献**：提出了一种将AI技术系统整合进软件工程课程的实际路径，显著提升了学生的编程能力、问题解决能力和智能工具使用水平，并加强了学术与职业实践的衔接。
