# Beyond Monotonic Progress: Retry-Supervised Value Learning for Robot Imitation

- arXiv: https://arxiv.org/abs/2606.24633v1
- arXiv ID: 2606.24633

---

🔍 **核心问题**：机器人模仿学习中的不完美演示数据（如抓取失误、重复尝试等）常被视为噪声，但现有方法依赖单调进展假设，无法捕捉局部错误与纠正行为，导致价值函数对执行质量不敏感。

🛠 **方法**：提出ReTVL框架，通过稀疏标注的“重试事件”作为监督信号，结合全局进展校准与局部成对偏好学习，构建对错误敏感的价值函数，并用于重加权演示片段以优化行为克隆。

✨ **主要贡献**：在真实机器人操作任务中，ReTVL相比基于进展的基线方法生成更细粒度的价值估计，有效提升从不完美演示中学习模仿策略的性能。
