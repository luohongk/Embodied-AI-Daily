# Gaga: Group Any Gaussians via 3D-aware Memory Bank

- arXiv: https://arxiv.org/abs/2404.07977v4
- arXiv ID: 2404.07977

---

🔍 **核心问题**：解决开放世界3D场景中，如何利用不一致的2D分割掩码实现精确的3D物体分割与重建，避免对连续视角或对比学习的依赖。

🛠 **方法**：提出Gaga框架，通过新颖的3D感知记忆库（3D-aware memory bank）关联不同相机视角下的物体掩码，有效利用空间信息，无需视频目标跟踪或对比学习。

✨ **主要贡献**：对稀疏采样图像和任意相机姿态具有鲁棒性，能兼容多种零样本2D分割模型，并在3D场景理解与操控任务中超越现有方法。
