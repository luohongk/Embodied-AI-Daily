# Z-1: Efficient Reinforcement Learning for Vision-Language-Action Models

- arXiv: https://arxiv.org/abs/2606.31846v1
- arXiv ID: 2606.31846

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型大多依赖行为克隆或监督微调（SFT），无法从自身失败中学习，限制了策略性能的提升。

🛠 **方法**：提出Z-1框架，基于流式VLA模型$π_{0.5}$，使用公开的RoboCasa演示进行SFT，然后应用任务级分组相对策略优化（GRPO）策略，结合共享前缀轨迹构建、树状轨迹分支、完成感知奖励校准及VLM与动作专家选择性联合训练。

✨ **主要贡献**：在24个RoboCasa任务中，Z-1平均成功率达80.6%，比SFT初始化提升13.2个百分点，超越现有最优模型，证明系统化GRPO后训练能显著提升流式VLA策略性能，且无需额外私有演示。
