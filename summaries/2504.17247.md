# OmegAMP: Targeted AMP Discovery via Biologically Informed Generation

- arXiv: https://arxiv.org/abs/2504.17247v3
- arXiv ID: 2504.17247

---

🔍 **核心问题**：解决深度学习在抗菌肽发现中存在的可控性差、抗菌特性表征效率低以及实验命中率低等关键挑战。

🛠 **方法**：提出OmegAMP框架，采用基于扩散的生成模型，通过新型条件机制实现对抗菌肽理化性质和物种特异性活性的精细控制，并结合生物信息编码空间与合成数据增强策略训练分类器进行过滤。

✨ **主要贡献**：在计算和湿实验阶段均取得最优性能，测试的25种候选肽中24种（96%）表现出抗菌活性，包括对多重耐药菌株有效，显著提升了抗菌肽发现成功率。
