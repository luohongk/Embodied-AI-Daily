# Mitigating Simplicity Bias in OOD Detection through Object Co-occurrence Analysis

- arXiv: https://arxiv.org/abs/2605.07821v2
- arXiv ID: 2605.07821

---

🔍 **核心问题**：现有OOD检测方法因模型存在简单性偏好，难以区分近分布外样本，且忽略了图像中丰富的上下文信息（如物体共现关系）。

🛠 **方法**：提出基于物体共现的OOD检测框架（OCO），通过预测测试样本的解耦表示来捕捉物体共现模式，根据训练数据中的共现模式将样本划分为三类场景，并采用分治策略进行检测。

✨ **主要贡献**：首次将物体共现分析引入OOD检测，有效利用语义上下文关系区分近分布外样本，在多种挑战性OOD设置下取得竞争性结果，并开源代码。
