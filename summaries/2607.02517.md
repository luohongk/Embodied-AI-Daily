# WorldDirector: Building Controllable World Simulators with Persistent Dynamic Memory

- arXiv: https://arxiv.org/abs/2607.02517v1
- arXiv ID: 2607.02517

---

🔍 **核心问题**：现有视频世界模型难以在长时间视角变化后保持动态物体的视觉一致性，且无法解耦物理运动与视觉渲染。

🛠 **方法**：提出WorldDirector框架，利用大语言模型协调3D轨迹与相机运动，将语义运动编排作为控制信号，解耦语义运动生成与视觉渲染过程。

✨ **主要贡献**：实现了对动态物体身份的长期稳定记忆（即使物体长时间离开视野后重新出现），并支持高度可控的复杂事件合成与自由视角探索。
