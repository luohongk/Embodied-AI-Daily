# Model Collapse Is Not a Bug but a Feature in Machine Unlearning for LLMs

- arXiv: https://arxiv.org/abs/2507.04219v5
- arXiv ID: 2507.04219

---

🔍 **核心问题**：现有的大语言模型机器遗忘方法需要将待删除的隐私数据加入微调数据中，这不仅可能强化模型对敏感信息的暴露，也违背了最小化使用隐私数据的原则。

🛠 **方法**：提出一种名为“部分模型崩溃”（Partial Model Collapse, PMC）的遗忘方法，通过故意触发模型在待删除数据上的生成分布崩溃，无需在遗忘目标中显式使用这些数据。

✨ **主要贡献**：理论上证明了PMC能收敛到期望的遗忘效果，实验表明该方法克服了现有方法的四个关键局限，更有效地移除模型输出中的隐私信息，同时保持通用模型性能。
