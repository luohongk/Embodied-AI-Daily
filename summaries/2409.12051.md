## 0. 概览（Concise Summary）

- **主题（Topic）**：视觉-惯性SLAM（VI-SLAM）与密集体素占据地图构建。这是一个结合了状态估计、深度学习和三维重建的机器人感知领域任务。
- **问题（Problem）**：现有的VI-SLAM系统通常使用稀疏或半稠密的地图（如特征点），这无法直接用于需要明确区分自由空间和占据空间的机器人下游任务（如导航、规划）。同时，现有的密集建图方法通常使用简单的启发式不确定性模型（如与深度平方成正比），无法准确反映深度神经网络预测的不确定性，尤其是在纹理稀疏或边缘区域。
- **方法（Method）**：提出了一种**完全概率化**的VI-SLAM框架，该框架紧密耦合了稀疏重投影误差、IMU预积分、相对位姿因子与密集体素占据地图。核心是：1）使用带有不确定性预测的深度神经网络（立体网络和MVS网络）；2）基于预测的不确定性，概率性地融合立体和运动立体（MVS）深度；3）将融合后的深度不确定性传播到占据概率更新和因子图优化中的“占据-点”因子中。
- **创新（Innovation）**：1）首次在VI-SLAM中，将深度神经网络预测的**像素级不确定性**系统地用于深度融合、占据地图更新和因子图优化，实现了全流程的不确定性感知。2）提出了一种基于不确定性的最优深度融合方法，结合了静态立体（短基线）和运动立体（长基线）的互补优势。3）通过“占据-点”因子将密集地图与稀疏SLAM后端紧密耦合，实现了全局一致的地图构建。
- **意义（Significance）**：该方法在定位和建图精度上均超越了现有技术水平，同时实时生成可直接用于机器人规划与控制的体素占据地图。这为机器人实现更安全、更可靠的自主导航提供了关键基础。
- **一句话总结**：本文提出了一种全概率化的不确定性感知视觉-惯性SLAM系统，通过融合带有不确定性的立体和MVS深度，紧密耦合了稀疏状态估计与密集占据地图构建，在精度和实用性上取得了显著提升。

## 1. 研究动机（Motivation）

- **解决的问题**：现有VI-SLAM系统存在两个主要问题：
    1.  **地图表示不适用于下游任务**：主流的VI-SLAM（如OKVIS2, ORB-SLAM3）使用稀疏或半稠密的地图（如特征点云），这种表示无法直接用于机器人导航、避障和路径规划，因为这些任务需要明确知道哪些空间是自由的、哪些是被占据的。
    2.  **深度不确定性建模不准确**：在密集建图中，通常使用一个简单的二次模型（$\sigma_d \propto d^2$）来近似深度测量的不确定性。这个模型假设了恒定的视差不确定性，但在实际中，深度神经网络在纹理缺失、边缘、遮挡等区域会产生高度不确定的预测，简单的模型无法捕捉这种像素级的变化，导致建图质量下降，甚至可能因错误信息导致机器人做出危险决策。
- **为什么重要**：实现高精度、高鲁棒性的定位与建图是机器人自主导航的基础。一个能够同时提供精确位姿和可靠、可直接用于规划的地图（如占据地图）的系统，对于无人机、自动驾驶汽车、服务机器人等应用至关重要。
- **挑战在哪里**：
    1.  **不确定性估计**：如何让深度神经网络不仅输出深度值，还能输出可靠的、像素级的深度不确定性。
    2.  **不确定性融合**：如何将来自不同来源（立体、MVS）的深度及其不确定性进行最优融合，以利用它们互补的特性（立体在近处、静态时可靠；MVS在远处、有运动时提供更大基线）。
    3.  **不确定性传播**：如何将深度不确定性系统地、概率性地传播到后续的占据地图更新和SLAM后端的因子图优化中，而不是将其作为一个独立的模块。
    4.  **紧密耦合**：如何将密集的占据地图与稀疏的视觉-惯性SLAM后端紧密耦合，使得地图信息能够反馈并提升定位精度，同时定位精度的提升又能改善地图的全局一致性。

## 2. 创新点（Innovation）

论文的关键创新点可以概括为以下三点：

1.  **全流程不确定性感知框架**：这是最核心的创新。与以往仅在深度融合或建图阶段使用启发式不确定性不同，本文提出了一种**端到端的不确定性感知**方法。从深度网络的训练（使用Laplacian损失学习不确定性），到深度融合（基于不确定性的加权平均），到占据地图更新（使用预测的$\sigma_d$作为逆传感器模型的输入），再到因子图优化（使用预测的$\sigma_d$和地图不确定性$\sigma_{map}$来加权“占据-点”因子），**深度不确定性被一致地传播并用于所有下游任务**。这使得整个系统对噪声和不可靠的深度测量具有更强的鲁棒性。

2.  **基于不确定性的最优深度融合**：本文巧妙地利用了**静态立体**（短基线，近处细节好）和**运动立体（MVS）**（长基线，远处精度高）的互补特性。通过让两个网络分别预测深度和不确定性，然后使用卡尔曼滤波的思想（公式14），根据各自的不确定性进行加权融合。这种融合方式在数学上是最优的（在独立高斯假设下），能够自动为每个像素选择更可靠的深度来源，从而显著提升建图精度，如图1和图3所示。

3.  **不确定性加权的“占据-点”因子**：在SLAM后端，本文引入了“占据-点”因子（公式6）来对齐子地图。其关键创新在于，该因子的权重矩阵（公式7）不仅包含了地图本身的不确定性（$\sigma_{map}$），还包含了**来自深度网络的像素级不确定性（$\sigma_d$）**。这与以往使用精确LiDAR点云的工作不同，因为深度相机的测量噪声大且不均匀。通过像素级加权，系统可以自动降低不可靠深度点对优化过程的贡献，从而在存在大量噪声的情况下，仍能实现鲁棒的子地图对齐和全局一致性。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

整个系统架构如图2所示，主要包含以下几个模块：

1.  **视觉-惯性状态估计器（Visual-inertial State estimator）**：基于OKVIS2，负责处理IMU数据和图像，通过因子图优化估计机器人位姿和速度、偏置等状态。
2.  **立体网络（Stereo network）**：基于Unimatch，输入左右立体图像对，输出视差图及其不确定性。
3.  **MVS网络（MVS network）**：基于Simplemapping，输入多张有重叠的、带有位姿的图像，输出参考帧的深度图及其不确定性。
4.  **深度融合（Depth fusion）**：根据两个网络预测的不确定性，对立体深度和MVS深度进行最优融合，得到融合后的深度图。
5.  **占据地图更新（Occupancy update）**：使用融合后的深度图及其不确定性，更新当前正在扩展的子地图的体素占据概率。
6.  **子地图管理（Submap management）**：当相机视野与当前子地图的重叠度低于阈值时，创建新的子地图。同时，通过共视关系检测，找到与当前子地图重叠度最高的旧子地图。
7.  **因子图优化**：构建包含重投影误差、IMU预积分误差、相对位姿误差和**占据-点因子**的代价函数（公式5）。占据-点因子分为**帧-地图因子**（使用立体深度）和**地图-地图因子**（使用融合深度），用于将当前帧或子地图与全局地图对齐。

### 3.2 关键算法与数学推导

1.  **深度不确定性学习**：
    - 立体网络使用Laplacian损失函数：
      $$L_{st}(\theta) = \sum_{i \in T} \left( \frac{|u_i - u_i^{gt}|}{\sigma_{u_i}} + \log \sigma_{u_i} \right)$$
      其中 $u$ 是视差，$\sigma_u$ 是预测的视差不确定性。通过最小化这个损失，网络学会了预测与预测误差相匹配的不确定性。
    - 视差不确定性到深度不确定性的传播（通过线性化）：
      $$\hat{d}_{st} = \frac{f_c b}{u}, \quad \sigma_{st} = \frac{f_c b}{u^2} \sigma_u$$
    - MVS网络类似，但学习的是对数深度的不确定性：
      $$L_{mvs}(\phi) = \sum_{i \in T} \left( \frac{|\log d_i - \log d_i^{gt}|}{\sigma_{l_i}} + \log \sigma_{l_i} \right)$$
      $$\hat{d}_{mvs} = \exp(\log d_i), \quad \sigma_{mvs} = \hat{d}_{mvs} \sigma_l$$

2.  **不确定性感知的深度融合**：
    假设两个深度估计独立且服从高斯分布，最优融合（最小化方差）为：
    $$\hat{d}_{fuse} = \sigma^2_{fuse} \left( \sigma^{-2}_{st} \hat{d}_{st} + \sigma^{-2}_{mvs} \hat{d}_{mvs} \right)$$
    $$\sigma^2_{fuse} = \left( \sigma^{-2}_{st} + \sigma^{-2}_{mvs} \right)^{-1}$$
    这个公式本质上是根据不确定性进行加权平均，不确定性越小的测量贡献越大。

3.  **不确定性感知的占据地图更新**：
    使用逆传感器模型（公式3）更新体素的对数几率。其中，模型的关键参数 $d_r$（点到测量表面的距离）的权重由深度不确定性 $\sigma_d$ 决定。例如，当 $d_r < -3\sigma_d$ 时，认为该点很可能是自由空间。这直接将网络预测的不确定性引入了建图过程。

4.  **不确定性加权的“占据-点”因子**：
    因子残差定义为：
    $$e_{o2p} = \frac{L(^{S_b}p)}{||\nabla L(^{S_b}p)||}$$
    其中 $L$ 是体素的对数几率，$^{S_b}p$ 是地图中的点。这个残差试图将点拉到占据概率为0.5（即 $L=0$）的等值面上。
    其权重为：
    $$w_{o2p_i} = \frac{1}{\sigma^2_{map_i} + \sigma^2_{d_i}}$$
    其中 $\sigma_{map}$ 是地图本身的不确定性（由 $l_{min}$ 和梯度计算），$\sigma_d$ 是网络预测的深度不确定性。这个权重确保了优化过程中，深度预测不可靠的点（$\sigma_d$ 大）对最终位姿估计的影响较小。

### 3.3 模型、训练与数据集

- **模型**：
    - **立体网络**：基于Unimatch。在预训练权重基础上，添加了一个不确定性解码器（来自MonoDepth2架构），并冻结其他权重，仅训练不确定性解码器。
    - **MVS网络**：基于Simplemapping。同样添加了与深度解码器结构相同的不确定性解码器，并在预训练权重基础上进行训练。
- **训练**：
    - 立体网络在**TartanAir**数据集上微调，使用Laplacian损失（公式8）。
    - MVS网络在**ScanNet**数据集上训练，使用对数深度Laplacian损失（公式12）。
    - 训练后，通过一个**校准步骤**：在验证集上计算平均马氏距离，并乘以一个常数增益，使其等于1，以确保不确定性预测的统计一致性。
- **数据集**：
    - **EuRoC MAV Dataset**：无人机在室内环境飞行的数据集，提供立体图像、IMU数据和毫米级精度的地面真值轨迹和点云。
    - **Hilti-Oxford Dataset**：手持设备在室内外环境采集的数据集，提供多相机图像、IMU数据和毫米级精度的地面真值点云。

### 3.5 技术细节

- **子地图管理**：通过跟踪当前深度视锥与最新子地图的重叠体积（采样点测试）来决定何时创建新子地图。通过共视稀疏地标来寻找与最新子地图重叠度最高的旧子地图（最老的共视子地图），用于构建地图-地图因子，实现几何闭环。
- **因子构建**：帧-地图因子使用来自立体网络的实时深度，可以在优化前构建。地图-地图因子使用已集成到子地图中的融合深度。为了控制计算量，对用于构建因子的点进行下采样，并优先选择不确定性最低的点。
- **优化器**：使用Ceres Solver进行非线性最小二乘优化。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了密集建图中不确定性建模不准确的问题**：提出了一个全流程的不确定性感知框架，将深度神经网络预测的像素级不确定性系统地用于深度融合、建图和SLAM优化，显著提升了建图和定位的鲁棒性与精度。
    2.  **解决了稀疏SLAM地图无法直接用于规划的问题**：通过紧密耦合，同时输出了精确的机器人位姿和可直接用于导航的体素占据地图，弥合了SLAM与下游机器人任务之间的鸿沟。
    3.  **提出了一种有效的多源深度融合策略**：利用立体和MVS的互补性，通过不确定性加权实现了最优融合，为多传感器融合提供了新思路。

- **对未来研究/应用的影响**：
    - **对机器人导航**：该方法为无人机、地面机器人等提供了一个强大且实用的感知前端，使其能够在复杂环境中进行更安全、更可靠的自主导航和探索。
    - **对SLAM研究**：展示了将深度学习不确定性融入传统优化框架的巨大潜力，可能会推动更多“学习+优化”混合方法的发展。
    - **对计算机视觉**：验证了不确定性学习在机器人学中的实际价值，鼓励开发更精确、更可靠的深度和不确定性预测网络。

- **局限与开放问题**：
    1.  **动态物体**：当前方法假设环境是静态的，无法处理动态物体。未来工作需要考虑动态物体检测和滤除。
    2.  **认知不确定性**：只考虑了偶然不确定性（数据噪声），未考虑认知不确定性（模型知识不足）。未来可以结合贝叶斯深度学习来建模认知不确定性。
    3.  **闭环控制**：目前仅停留在感知层面，未来可以探索如何将状态不确定性直接用于机器人导航和控制，实现“感知-规划-控制”的完整闭环。

## 5. 澄清与简化（Clarifications and Simplifications）

- **什么是“占据-点”因子？**
    - **通俗解释**：想象你有一个由很多小立方体（体素）组成的地图，每个小立方体都有一个“被占据”的概率。现在，你通过深度相机看到了一个点，这个点应该落在某个物体的表面上。这个“占据-点”因子就像一个**橡皮筋**，它的一端连着这个观测到的点，另一端连着地图中那个“被占据概率为50%”的等值面。优化过程就是调整机器人的位姿，让这个橡皮筋的拉力（即残差）最小，从而让观测到的点与地图表面完美对齐。本文的创新在于，这个橡皮筋的**弹性系数**（即权重）不是固定的，而是根据深度测量的不确定性动态调整的——测量越不准，橡皮筋越软，对位姿优化的影响就越小。

- **为什么深度融合能提升建图质量？**
    - **类比**：想象你用手电筒照一个物体。
        - **立体网络**就像**两个离得很近的手电筒**（短基线）。它们能照亮物体近处的细节，但照不远，远处的东西模糊不清。
        - **MVS网络**就像**一个手电筒在不同位置拍了很多张照片**（长基线）。它能通过多张照片合成出远处物体的清晰轮廓，但近处的细节可能不如立体网络。
    - 本文的方法就像一个**智能的摄影师**，它知道每个手电筒在哪个区域拍得最清楚（不确定性低），然后只取每个手电筒拍得最清楚的部分，合成一张全局最清晰的照片。这就是基于不确定性的最优融合。
