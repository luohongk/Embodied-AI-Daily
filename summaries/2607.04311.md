# Aura: Consistent Multi-Subject Video Generation via VLM-Grounded Semantic Alignment

- arXiv: https://arxiv.org/abs/2607.04311v1
- arXiv ID: 2607.04311

---

🔍 **核心问题**：现有视频生成方法难以在多主体场景中保持身份一致性，并有效建模主体间的复杂交互关系。

🛠 **方法**：提出Aura框架，利用视觉语言模型（VLM）提取多模态语义特征，并通过两阶段对齐策略将其映射到扩散Transformer（DiT）特征空间；引入主体感知RoPE-Shift机制和可学习令牌区分不同主体类型，同时使用记忆令牌平衡训练信号。

✨ **主要贡献**：在单主体和多主体视频生成任务上均达到最先进性能，通过AI导演级字幕和渐进式自适应提示引导（Progressive-APG）显著提升语义对齐与生成质量，并构建了高质量视频-主体图像数据集。
