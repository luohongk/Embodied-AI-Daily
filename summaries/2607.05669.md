# Uncertainty-Aware Velocity Correction for Proprioceptive Vehicle Localization using Evidential Mamba

- arXiv: https://arxiv.org/abs/2607.05669v1
- arXiv ID: 2607.05669

---

🔍 **核心问题**：在无GNSS信号环境中，智能车辆仅靠惯性导航系统会产生无界漂移，现有解决方案依赖专用基础设施、昂贵外部传感器或复杂多传感器融合，部署成本高。

🛠 **方法**：提出EVC-Mamba架构，利用Mamba选择性状态空间模型捕捉车辆运动时序动态，结合基于Normal-Inverse-Gamma分布的证据深度学习进行不确定性量化，将速度估计作为虚拟校正量输入误差状态扩展卡尔曼滤波器。

✨ **主要贡献**：仅用车载传感器数据实现IMU漂移校正，定位精度接近专用外部速度传感器（误差在10%以内），支持40Hz实时边缘部署，可在长时间GNSS中断下保持可靠定位。
