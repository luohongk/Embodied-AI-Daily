# Learning Probabilistic Prompt for Continual Learning

- arXiv: https://arxiv.org/abs/2607.04711v1
- arXiv ID: 2607.04711

---

🔍 **核心问题**：现有基于提示的持续学习方法存在“提示崩溃”问题，即不同提示之间高度相似，无法捕捉持续学习场景中多样化的数据分布。  
🛠 **方法**：提出概率提示框架，将每个提示建模为概率分布，并构建这些分布的混合模型，从中采样多样化的提示；同时引入分布正则化损失，防止训练过程中提示分布发生剧烈变化。  
✨ **主要贡献**：在ImageNet-R、CIFAR-100和CUB-200等标准基准上取得显著效果，有效缓解了提示崩溃问题，提升了持续学习中对多样化图像分布的捕获能力。
