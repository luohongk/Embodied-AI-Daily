# A Hybrid TGN-SEAL Model for Dynamic Graph Link Prediction

- arXiv: https://arxiv.org/abs/2602.14239v3
- arXiv ID: 2602.14239

---

🔍 **核心问题**：解决稀疏、持续演化的动态网络中链接预测的挑战，尤其是传统方法难以捕捉时间依赖性和处理数据稀疏与类别不平衡问题。  
🛠 **方法**：提出一种混合TGN-SEAL模型，在TGN框架中提取候选链接周围的封闭子图，联合学习结构信息与时间信息。  
✨ **主要贡献**：在稀疏的CDR、邮件和消息数据集上，该方法相比标准TGN将平均精度提升至少2%，验证了整合局部拓扑结构对动态网络鲁棒链接预测的有效性。
