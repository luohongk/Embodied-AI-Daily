# Machine Learning for Network Attacks Classification and Statistical Evaluation of Adversarial Learning Methodologies for Synthetic Data Generation

- arXiv: https://arxiv.org/abs/2603.17717v4
- arXiv ID: 2603.17717

---

🔍 **核心问题**：解决网络入侵检测系统（NIDS）中，利用机器学习对统一多模态数据集进行攻击分类，并评估对抗学习生成合成数据的质量。  
🛠 **方法**：采用分层交叉验证的机器学习算法进行攻击检测，并使用对抗学习算法生成合成数据，通过SDV框架、f-散度、可区分性及非参数统计检验评估其保真度、效用和隐私性。  
✨ **主要贡献**：构建了首个统一多模态NIDS数据集，实现了稳定的入侵检测模型，并验证了合成数据生成方法的高保真度和高实用性，结合TRTS/TSTR测试与统计评估提供了可靠基准。
