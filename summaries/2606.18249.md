# Unified Multimodal Autoregressive Modeling with Shared Context-Visual Tokenizer is Key to Unification

- arXiv: https://arxiv.org/abs/2606.18249v1
- arXiv ID: 2606.18249

---

🔍 **核心问题**：现有统一多模态模型通常使用两个独立的视觉分词器，导致表示空间分裂，难以实现真正的视觉理解与生成统一建模。

🛠 **方法**：提出UniAR框架，采用单一离散视觉分词器作为理解与生成的共享桥梁，结合多级特征融合与免查找比特量化方案，并通过并行比特预测和扩散视觉解码器实现高效图像生成。

✨ **主要贡献**：在图像生成和编辑任务上达到最先进性能，同时在多模态理解基准上保持竞争力，验证了共享分词器对统一建模的关键作用。
