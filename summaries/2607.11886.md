# Read It Back: Pretrained MLLMs Are Zero-Shot Reward Models for Text-to-Image Generation

- arXiv: https://arxiv.org/abs/2607.11886v1
- arXiv ID: 2607.11886

---

🔍 **核心问题**：如何在不依赖人工标注或额外训练的情况下，将预训练多模态大语言模型（MLLM）直接用作文本到图像生成任务的奖励模型，以提升生成质量。

🛠 **方法**：提出SpectraReward，通过计算生成图像条件下原始文本提示的平均对数似然作为奖励分数，无需微调或偏好标签；并进一步提出Self-SpectraReward，让统一多模态模型自身的理解分支作为生成分支的奖励模型，形成闭环自改进框架。

✨ **主要贡献**：实验证明SpectraReward和Self-SpectraReward在多种扩散模型、强化学习算法和MLLM骨干网络上显著提升生成性能，并超越先前基于MLLM的奖励训练方法；发现更大的奖励模型不一定更好，而奖励与策略的对齐是关键因素。
