# HyperNet-Adaptation for Diffusion-Based Test Case Generation

- arXiv: https://arxiv.org/abs/2601.15041v2
- arXiv ID: 2601.15041

---

🔍 **核心问题**：现有深度学习系统测试方法难以高效生成真实且可控的故障案例，传统对抗攻击仅评估鲁棒性，而生成式方法受限于简单数据集或计算成本高。

🛠 **方法**：提出HyNeA方法，利用超网络（hypernetwork）直接控制扩散模型的生成过程，无需依赖数据集微调或特定架构条件机制，并通过实例级调优策略识别故障触发案例。

✨ **主要贡献**：实现无数据集依赖的高效可控测试生成，相比搜索方法显著降低计算成本，同时提升测试多样性与领域泛化能力，在无故障标签数据场景下仍有效。
