# MVFusion-GS: Motion-Variance Guided Temporal Attention for High-Quality Dynamic Gaussian Splatting

- arXiv: https://arxiv.org/abs/2607.01578v2
- arXiv ID: 2607.01578

---

🔍 **核心问题**：现有动态3D高斯泼溅方法缺乏显式运动感知，无法准确捕捉长期运动强度与短期时间连贯性，导致前景变形不精确和背景出现伪静态残留。

🛠 **方法**：提出MVFusion-GS，包含两个互补模块：运动方差引导细化模块，通过聚合各高斯随时间变化的变形统计量估计运动方差，指导动态-静态分离；运动变换器时间注意力模块，利用Transformer自注意力机制建模相邻时间步的局部运动依赖，提升时间一致性。

✨ **主要贡献**：在动态场景重建和无干扰物重建基准上达到最优性能，证明显式运动感知能同时提升前景运动建模和静态背景重建质量。
