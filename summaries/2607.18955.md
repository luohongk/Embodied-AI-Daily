# H$^2$SD: Hybrid Hindsight Self-Distillation

- arXiv: https://arxiv.org/abs/2607.18955v2
- arXiv ID: 2607.18955

---

🔍 **核心问题**：在基于可验证奖励的强化学习（RLVR）中，标量轨迹奖励对语言模型推理的逐令牌指导不足，现有自蒸馏方法对成功与失败轨迹采用固定策略，导致成功行为不稳定或失败后缺乏有效修正。

🛠 **方法**：提出混合事后自蒸馏（H²SD），根据轨迹正确性动态调整教师上下文与更新策略——成功轨迹使用验证后的响应和改写指令作为教师上下文，仅重新评估原始令牌以细化信用分配；失败轨迹则通过验证器确认的参考提示进行反向KL散度蒸馏，提供修正性指导。

✨ **主要贡献**：通过结果条件路由和改写指令实现稳定优化，在挑战性推理基准上取得优于现有RLVR和自蒸馏方法的整体性能，并实现了准确性与效率的良好平衡。
