# VINE: Taming Generative Control Policies for Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.10369v1
- arXiv ID: 2607.10369

---

🔍 **核心问题**：流匹配策略在结合值梯度强化学习时存在训练不稳定性，现有方法错误地将此归因于迭代生成过程，并因此牺牲了端到端优化或策略表达能力。

🛠 **方法**：提出VINE采样方法，在每个去噪步骤中重建新的插值状态，而非沿单一流轨迹生成动作，从而为值梯度传播建立稳定的可微路径，同时保持与原始流匹配去噪过程的兼容性。

✨ **主要贡献**：首次证明不稳定性源于原始采样策略而非迭代生成本身；VINE在保留流匹配高表达力和迭代生成能力的同时，实现了稳定的端到端值梯度优化；在OGBench离线强化学习基准和真实机器人操作任务中持续优于现有最先进方法。
