# GSurf: Learning Signed Distance Fields from Splatting Opaque Gaussians for High-quality 3D Reconstruction

- arXiv: https://arxiv.org/abs/2411.15723v4
- arXiv ID: 2411.15723

---

🔍 **核心问题**：从多视角图像进行高质量3D表面重建时，现有方法在几何连续性与计算效率之间存在矛盾——神经隐式表面（如SDF）计算密集，而3D高斯泼溅（3DGS）缺乏几何连续性，易产生碎片化表面。

🛠 **方法**：提出GSurf框架，将符号距离场（SDF）直接集成到高斯泼溅管线中，利用SDF的连续性约束高斯基元，填补几何空洞并抑制稀疏点云噪声，同时通过泼溅的高效性避免混合方法中的体积采样开销。

✨ **主要贡献**：实现高质量表面重建，显著减少所需基元数量，在室内外场景中均获得更紧凑高效的表示，且收敛速度更快。
