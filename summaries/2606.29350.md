# Fast Enough to Act: Spatio-Temporal Visual Token Merging for Low-Latency Robotic VLMs and VLAs

- arXiv: https://arxiv.org/abs/2606.29350v1
- arXiv ID: 2606.29350

---

🔍 **核心问题**：视觉-语言模型处理高分辨率视频时产生大量视觉token，导致推理延迟极高，严重阻碍机器人实时控制。

🛠 **方法**：提出ST-Merge框架，在视觉编码阶段通过构建3D时空坐标、多队列并行匹配与加权聚合机制，直接融合冗余token，并引入后融合位置校正机制消除空间偏差。

✨ **主要贡献**：在Qwen2.5-VL上实现2倍推理加速且精度仅损失1%；在π₀.₅ VLA策略上，1024×1024分辨率下实现8.3倍加速并保持基线成功率，低分辨率下精度略有下降。
