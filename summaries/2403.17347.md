# Unified Path and Gait Planning for Safe Bipedal Robot Navigation

- arXiv: https://arxiv.org/abs/2403.17347v1
- arXiv ID: 2403.17347

---

🔍 **核心问题**：双足机器人在复杂环境中导航时，传统分层式路径与步态规划方法因忽略机器人物理约束（如动力学与运动学限制），可能导致不安全移动。

🛠 **方法**：提出一种基于模型预测控制（MPC）的统一路径与步态规划方法，采用线性倒立摆（LIP）模型表征双足运动，并利用离散时间控制障碍函数（CBF）实现避障，同时生成下一步落足位置。

✨ **主要贡献**：首次将路径与步态规划统一为单层优化框架，在20个随机环境中通过Digit机器人仿真验证，相比分层方法显著提升了导航的安全性与鲁棒性。
