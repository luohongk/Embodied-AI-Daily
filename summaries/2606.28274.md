# Parameter Efficient Hybrid Transformer (PEHT) for Network Traffic Prediction via Dynamic Urban Congestion Integration

- arXiv: https://arxiv.org/abs/2606.28274v1
- arXiv ID: 2606.28274

---

🔍 **核心问题**：解决动态城市蜂窝网络中，因复杂移动模式、拥堵动态和用户行为异质性导致的网络流量预测难题。

🛠 **方法**：提出参数高效混合Transformer（PEHT），将网络通信特征与城市移动特征分离，在Transformer编码器中引入低秩适配（LoRA）减少可训练参数，并通过多模态融合策略将外部移动和拥堵特征注入解码器。

✨ **主要贡献**：在米兰电信数据集和多种合成拥堵场景下，PEHT在RMSE、MAE和R²指标上均优于现有基线方法，且代码已开源。
