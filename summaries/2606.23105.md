# Compression and Retrieval: Implicit Memory Retrieval for Video World Models

- arXiv: https://arxiv.org/abs/2606.23105v1
- arXiv ID: 2606.23105

---

🔍 **核心问题**：视频世界模型在复杂相机轨迹下难以维持一致的长时记忆，现有方法依赖计算昂贵的上下文扩展或僵化的启发式检索机制，缺乏泛化能力。

🛠 **方法**：提出压缩与检索（CaR）机制，通过位置编码注入视角信息实现注意力驱动的隐式记忆检索，并引入轻量级上下文压缩网络以高效处理长序列。

✨ **主要贡献**：构建了大规模合成数据集SceneFly，在多个基准上取得最先进结果，并展现出对开放域场景的强泛化能力。
