# MAS-PromptBench: When Does Prompt Optimization Improve Multi-Agent LLM Systems?

- arXiv: https://arxiv.org/abs/2606.23664v1
- arXiv ID: 2606.23664

---

🔍 **核心问题**：在多智能体系统中，提示优化能否提升系统性能，以及在何种条件下效果显著，这是当前尚未明确的问题。

🛠 **方法**：系统性地研究了多种多智能体系统配置（包括任务、工作流、通信协议和团队规模），并扩展了两种先进的单智能体提示优化方法作为基准。

✨ **主要贡献**：揭示了提示优化在多智能体系统中具有显著性能提升潜力，同时指出了其面临的挑战，并明确了在不同系统设置下提示优化的有效性和提升幅度。
