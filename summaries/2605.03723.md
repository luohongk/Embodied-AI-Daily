# Segmenting Human-LLM Co-authored Text via Change Point Detection

- arXiv: https://arxiv.org/abs/2605.03723v2
- arXiv ID: 2605.03723

---

🔍 **核心问题**：现有LLM文本检测器仅能对整段文本进行二分类，无法定位人机协作文本中由人类或LLM撰写的具体片段。

🛠 **方法**：将文本分割任务类比为时间序列中的变点检测问题，提出加权算法和广义算法以适配检测分数异质性，并证明了方法的极小化最优性。

✨ **主要贡献**：首次将变点检测应用于LLM文本分割，在多种基线方法上展现出优异性能，并开源了Python实现。
