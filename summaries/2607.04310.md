# GPU-Accelerated Polygonal Signed Distance Functions for Real-Time Collision Avoidance

- arXiv: https://arxiv.org/abs/2607.04310v1
- arXiv ID: 2607.04310

---

🔍 **核心问题**：在障碍密集环境中，基于优化的局部规划与控制需要高效评估碰撞避免约束，但计算负载常成为控制周期瓶颈。  
🛠 **方法**：提出多边形符号距离函数（PSDF），通过无权重、无分支的张量化几何管线实现GPU批量执行与自动微分，并嵌入基于序列二次规划的模型预测控制（PSDF-MPC），分离CPU/GPU计算。  
✨ **主要贡献**：PSDF在几何精度与计算效率间取得平衡，微基准测试显示其符号距离查询性能优于基线；闭环仿真与真实导航实验证明PSDF-MPC在密集多边形环境中保持实时可行性与鲁棒碰撞避免。
