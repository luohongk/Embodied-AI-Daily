# Doomed from the Start: Early Abort of LLM Agent Episodes via a Recall-Controlled Probe Cascade

- arXiv: https://arxiv.org/abs/2607.06503v1
- arXiv ID: 2607.06503

---

🔍 **核心问题**：LLM智能体在多步任务中常陷入注定失败的轨迹，却仍消耗大量推理计算资源，而早期失败信号难以被行为观察捕捉。

🛠 **方法**：提出一种基于召回控制的探针级联方法，通过每轮轻量级探针分析智能体内部隐藏激活，结合无分布校准门控和全局召回预算搜索，在用户指定成功率下保留成功轨迹并提前中止失败轨迹。

✨ **主要贡献**：在TextCraft任务上，该方法在90%召回目标下节省47.1%（Qwen-2.5-7B）和37.2%（Llama-3.2-3B）推理计算，性能是单门控策略的1.6-1.7倍；且隐藏状态比行为特征更早预测失败，并提供了高召回目标认证的样本复杂度分析。
