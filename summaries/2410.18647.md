# Data Scaling Laws in Imitation Learning for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2410.18647v4
- arXiv ID: 2410.18647

---

🔍 **核心问题**：研究机器人模仿学习中是否存在类似自然语言处理和计算机视觉领域的数据缩放定律，即增加数据量能否使单任务机器人策略在同类物体和任意环境中实现零样本部署。

🛠 **方法**：通过大规模实证研究，在多种环境和物体上收集超过40,000次演示，并执行15,000次真实机器人测试，系统分析策略泛化性能随训练环境、物体数量和演示次数的变化规律。

✨ **主要贡献**：发现策略泛化性能与环境/物体数量呈幂律关系，且环境与物体的多样性远重要于演示总数；基于此提出高效数据收集策略，仅需四人工作一个下午即可使两个任务策略在新环境和新物体上达到约90%成功率。
