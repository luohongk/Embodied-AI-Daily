# DriveStack-VLA: Render-Teacher Alignment for BEV-Based DeepStack Vision-Language-Action Model

- arXiv: https://arxiv.org/abs/2606.24051v1
- arXiv ID: 2606.24051

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）驾驶模型缺乏面向驾驶的空间智能，其策略主要依赖透视图像和语言先验，导致在精确运动规划中缺乏度量几何、自上而下的场景结构以及对安全关键感知线索的关注。

🛠 **方法**：提出DriveStack-VLA框架，通过DeepStack式连接将鸟瞰图（BEV）表示注入大语言模型解码器，并引入渲染-教师对齐（Render-Teacher Alignment）技术，使真实图像与光栅化图像的感知焦点对齐；此外，设计基于头部的自我批评模块，用于排序和优化多模态轨迹选择。

✨ **主要贡献**：在NAVSIMv1上达到91.6 PDMS，NAVSIMv2上达到91.0 EPDMS，闭环Bench2Drive上取得79.49驾驶分数和56.36%成功率，显著提升了VLA驾驶模型的空间感知与规划能力。
