# TACO: A Test and Check Framework for Robust Pose Graph Optimization

- arXiv: https://arxiv.org/abs/2606.29851v1
- arXiv ID: 2606.29851

---

🔍 **核心问题**：解决位姿图优化（PGO）中因感知混淆导致的异常值（outliers）问题，这些异常值会严重降低SLAM轨迹估计的精度。

🛠 **方法**：提出TACO框架，包含两个互补组件：一是增量概率一致性（IPC）算法，在线评估每个新回环的一致性；二是可切换异常值清理模块，定期移除IPC可能误纳入的不一致测量。

✨ **主要贡献**：在2D和3D SLAM数据集上，TACO在高达50%的异常值率下仍保持90%以上（2D）和83%（3D）的成功率，平均收敛时间分别约45毫秒和100毫秒，兼具离线方法的鲁棒性和在线部署的计算效率。
