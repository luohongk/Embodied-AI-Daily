# Position: Modular Memory is the Key to Continual Learning Agents

- arXiv: https://arxiv.org/abs/2603.01761v2
- arXiv ID: 2603.01761

---

🔍 **核心问题**：解决持续学习智能体在连续运行中难以积累经验、实现个性化适应，且传统基于参数更新的方法（IWL）面临灾难性遗忘的挑战。

🛠 **方法**：提出模块化记忆架构，结合参数内学习（IWL）的稳定更新能力与上下文学习（ICL）的快速适应与知识积累能力，通过设计可扩展的记忆模块实现持续适应。

✨ **主要贡献**：首次系统论证模块化记忆是连接IWL与ICL的关键缺失环节，并给出面向持续学习智能体的概念框架与实践路线图，为大规模持续适应提供新范式。
