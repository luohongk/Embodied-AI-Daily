# Not Truly Multilingual: Script Consistency as a Missing Dimension in VLM Evaluation

- arXiv: https://arxiv.org/abs/2606.17188v1
- arXiv ID: 2606.17188

---

🔍 **核心问题**：当前多语言视觉语言模型（VLM）的评估假设语言与文字一一对应，忽略了多文字语言（如旁遮普语的三种文字）用户，导致模型在多文字场景下表现不一致。

🛠 **方法**：构建了PuMVR基准，包含1000个严格平行的图像-文本实例，覆盖旁遮普语的Gurmukhi、Shahmukhi和Roman三种文字，并评估了10个先进VLM的性能。

✨ **主要贡献**：揭示了VLM存在系统性“文字差距”，同一视觉任务在不同文字下准确率差异高达16%，且跨文字上下文迁移脆弱；提出了文字一致性率（SCR）作为强制评估指标，最低仅24.8%，表明当前“多语言”VLM并非真正多文字。
