# Spherical-GOF: Geometry-Aware Panoramic Gaussian Opacity Fields for 3D Scene Reconstruction

- arXiv: https://arxiv.org/abs/2603.08503v2
- arXiv ID: 2603.08503

---

🔍 **核心问题**：现有3D高斯泼溅（3DGS）方法针对透视相机设计，直接应用于全景相机时会产生几何畸变和不一致性。

🛠 **方法**：提出Spherical-GOF框架，在球面射线空间中对单位球进行高斯不透明度场（GOF）射线采样，并设计保守球面包围规则和球面滤波方案，实现高效、鲁棒的全景渲染。

✨ **主要贡献**：在全景基准数据集上，深度重投影误差降低57%，循环内点率提升21%，并引入真实机器人全景数据集OmniRob，验证了方法的泛化性和几何一致性。
