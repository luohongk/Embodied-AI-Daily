# Hybrid Machine Learning for Articulation Angle Estimation of Truck-Semitrailer Combinations

- arXiv: https://arxiv.org/abs/2607.16758v1
- arXiv ID: 2607.16758

---

🔍 **核心问题**：解决卡车-半挂车组合在自动驾驶和高级驾驶辅助系统中，无需手动初始化、额外传感器或挂车先验知识，即可准确估计铰接角的问题。

🛠 **方法**：提出多种基于学习的模型，从视觉和运动学输入直接估计铰接角，并与扩展卡尔曼滤波框架中的运动学模型集成，同时采用基于不确定性量化的自适应加权方案处理视觉测量。

✨ **主要贡献**：通过大量真实世界实验验证，该方法在不同挂车类型、颜色和光照条件下展现出鲁棒性和泛化能力，实现了准确可靠的铰接角估计，同时降低了部署要求，具有实际应用优势。
