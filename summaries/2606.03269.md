# Distilling Answer-Set Programming Rules from LLMs for Neurosymbolic Visual Question Answering

- arXiv: https://arxiv.org/abs/2606.03269v2
- arXiv ID: 2606.03269

---

🔍 **核心问题**：在神经符号视觉问答（VQA）中，如何自动生成可解释的逻辑规则，以减轻开发者手动调整推理规则的工作负担。

🛠 **方法**：提出从大型语言模型（LLM）中蒸馏答案集编程规则的方法，通过提示LLM扩展初始的ASP推理理论，并利用VQA数据集中的示例和ASP求解器的反馈来验证和修正规则。

✨ **主要贡献**：仅需少量示例即可从LLM中提取正确规则，实验表明该方法在多个VQA数据集上有效，为传统数据驱动的规则学习提供了有前景的替代方案。
