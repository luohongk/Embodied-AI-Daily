# Jetson-PI: Towards Onboard Real-Time Robot Control via Foresight-Aligned Asynchronous Inference

- arXiv: https://arxiv.org/abs/2607.12659v1
- arXiv ID: 2607.12659

---

🔍 **核心问题**：在低功耗机载设备（如Jetson Orin）上部署视觉-语言-动作（VLA）模型时，面临高计算复杂度导致的推理延迟大和控制频率低的问题，异步推理虽能部分掩盖延迟，却引发感知-执行错位和反应时间长两大挑战。

🛠 **方法**：提出Jetson-PI方法，通过训练轻量级未来校正模块预测未来环境表示以解决错位，并引入基于置信度的调度优化平衡VLM与动作专家调用，结合CUDA图复用、GPU常驻中间缓冲和流展开等系统级加速。

✨ **主要贡献**：在NVIDIA Jetson Orin上，控制频率相比PyTorch和vla.cpp分别提升8.66倍和5.41倍，在LIBERO基准上平均成功率比VLASH高14.8%，并开源异步算法代码和高效推理引擎。
