# StreamVLN: Streaming Vision-and-Language Navigation via SlowFast Context Modeling

- arXiv: https://arxiv.org/abs/2507.05240v2
- arXiv ID: 2507.05240

---

🔍 **核心问题**：现有基于视频大模型的视觉语言导航方法难以同时兼顾细粒度视觉理解、长期上下文建模与计算效率，导致实时性不足。

🛠 **方法**：提出StreamVLN框架，采用“慢-快”混合上下文建模策略——快速流对话上下文通过滑动窗口实现低延迟动作生成，慢速更新记忆上下文利用3D感知令牌剪枝压缩历史视觉状态，并通过KV缓存复用支持实时对话。

✨ **主要贡献**：在VLN-CE基准上实现最先进性能，同时保持低延迟，在真实部署中兼具鲁棒性与效率。
