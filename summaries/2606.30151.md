# AERIS: Aerial-Edge Role-Driven Intelligence at Runtime via Orchestrated Language-Model Swarm

- arXiv: https://arxiv.org/abs/2606.30151v1
- arXiv ID: 2606.30151

---

🔍 **核心问题**：如何在计算资源受限、心跳时序严格约束的无人机平台上，实现大语言模型的边缘部署与实时闭环控制。

🛠 **方法**：提出AERIS框架，将专用小语言模型与轻量级感知、控制模块组织为可运行时实例化的“角色”，并根据资源变化动态绑定到不同执行器；通过注意力-子目标对齐机制实现长程指令分解，在消息中标注当前活跃指令步骤以逐步逼近长期目标。

✨ **主要贡献**：在无人机视觉-语言导航基准上，AERIS在低频率规划器与高频率控制器之间维持稳定的感知-决策-控制闭环，支持实时运行；通过两项真实世界实验验证了其在规划与快速响应中的可部署性。
