# Continual Robot Policy Learning via Variational Neural Dynamics

- arXiv: https://arxiv.org/abs/2606.27353v1
- arXiv ID: 2606.27353

---

🔍 **核心问题**：机器人在真实环境中部署时，其动力学模型会因风、负载、电池、硬件磨损等因素持续变化，而现有学习控制器无法利用部署经验持续改进策略，导致性能下降。

🛠 **方法**：提出一种持续学习框架，结合解析物理先验与神经网络残差学习条件感知动力学模型；通过循环编码器从近期交互中推断隐藏条件，并基于该条件采样多样化动力学进行可微分仿真训练策略；部署时用在线推断的条件替换采样条件，通过识别而非残差重拟合恢复重复出现的动力学。

✨ **主要贡献**：在真实四旋翼轨迹跟踪实验中，该方法能在约1秒内从重复扰动中恢复，比在线残差重拟合快约5倍；相比最先进的在线自适应方法，大幅扰动下的悬停和跟踪误差分别降低65.7%和53.3%。
