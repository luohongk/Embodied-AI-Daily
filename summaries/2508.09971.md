# Vision-driven River Following of UAV via Safe Reinforcement Learning using Semantic Dynamics Model

- arXiv: https://arxiv.org/abs/2508.09971v2
- arXiv ID: 2508.09971

---

🔍 **核心问题**：无人机在GPS信号不可靠的密集河道环境中，需要安全地自主沿河飞行，但传统安全强化学习难以处理非马尔可夫（历史依赖）的奖励结构。

🛠 **方法**：提出三种技术——边际增益优势估计（MGAE）通过滑动窗口基线处理非马尔可夫奖励；基于语义掩码的语义动力学模型（SDM）实现可解释的短期状态预测；约束-演员-动力学估计器（CADE）架构整合演员、成本估计器和SDM，形成基于模型的安全强化学习框架。

✨ **主要贡献**：MGAE相比传统方法（如GAE）收敛更快且性能更优；SDM提供更准确的短期预测，帮助成本估计器更好预判违规；CADE通过拉格朗日方法在训练中软性平衡奖励与安全，并在推理时通过安全层硬性约束动作。
