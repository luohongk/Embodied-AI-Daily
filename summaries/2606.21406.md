# Robot Self-Improvement via Human-Video Dynamics Models

- arXiv: https://arxiv.org/abs/2606.21406v1
- arXiv ID: 2606.21406

---

🔍 **核心问题**：如何利用人类视频数据，使机器人能够像人类一样通过被动观察、自主实践和失败经验来提升技能，实现跨本体的自我改进。

🛠 **方法**：提出动力学引导的动作修正（DGAC）方法，从人类视频中学习与本体无关的动作、动力学和价值表示，并利用这些模型在无训练条件下对机器人失败状态进行修复和策略更新。

✨ **主要贡献**：在7个真实世界操作任务中，将多种策略的成功率从40%提升至81%，首次证明了人类视频先验可支持跨本体机器人的自主策略改进。
