# Automated Standardization of Legacy Biomedical Metadata Using an Ontology-Constrained LLM Agent

- arXiv: https://arxiv.org/abs/2604.08552v2
- arXiv ID: 2604.08552

---

🔍 **核心问题**：现有生物医学元数据常不完整且不符合社区标准，缺乏机器可操作表示，限制了数据的可发现性、互操作性和重用性。

🛠 **方法**：提出基于LLM的元数据标准化系统，通过实时查询标准报告指南和权威生物医学术语服务，结合本体约束与工具调用，自动标准化遗留元数据。

✨ **主要贡献**：在HuBMAP的839条元数据记录上，相比仅用LLM，实时工具增强方法在有无本体约束字段上均显著提升预测准确率，验证了自动化标准化生物医学元数据的实用路径。
