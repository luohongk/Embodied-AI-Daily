# Pixels to Proofs: Probabilistically-Safe Latent World Model Control via Parallel Conformal Robust MPC

- arXiv: https://arxiv.org/abs/2606.15594v1
- arXiv ID: 2606.15594

---

🔍 **核心问题**：在基于视觉的机器人控制中，如何利用学习到的潜在世界模型实现概率安全的反馈运动规划，同时应对潜在状态预测的不确定性。

🛠 **方法**：提出SLS²框架，通过训练动作条件联合嵌入世界模型获得紧凑马尔可夫潜在状态，结合GPU加速的系统级综合鲁棒模型预测控制（SLS-MPC），并利用共形预测校准潜在误差边界和约束集，同时学习并共形化潜在约束检查器。

✨ **主要贡献**：首次将共形预测与鲁棒MPC结合用于潜在世界模型控制，在视觉控制任务中相比基线方法显著提升了目标到达性能和安全约束满足能力。
