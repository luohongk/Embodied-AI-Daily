# An Additive MLP-GNN Framework for Characterizing Chemical and Structural Contributions to Aqueous Solubility

- arXiv: https://arxiv.org/abs/2607.02212v1
- arXiv ID: 2607.02212

---

🔍 **核心问题**：现有水溶性预测模型将化学描述符与分子结构信息融合为单一表示，难以区分预测结果是由全局化学性质、分子结构还是两者共同驱动。

🛠 **方法**：提出一种加性深度学习框架，使用多层感知机（化学分支）编码理化描述符，图神经网络（结构分支）编码分子图拓扑，仅在预测阶段通过加性模型（可选乘性交互）合并输出，并采用预训练-微调策略提升泛化性。

✨ **主要贡献**：首次实现化学与结构贡献的显式分解，通过线性投影、嵌入聚类和GNNExplainer等分析揭示了化学分支对应理化性质、结构分支捕捉拓扑与官能团模式，在保持高预测精度的同时增强了模型可解释性。
