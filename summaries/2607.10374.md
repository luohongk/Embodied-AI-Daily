# Navigating the Crowd: Non-linear MPC with Social Forces Dynamics for Human-Aware Robot Navigation

- arXiv: https://arxiv.org/abs/2607.10374v1
- arXiv ID: 2607.10374

---

🔍 **核心问题**：解决自主机器人在人群环境中安全且符合社交规范导航的挑战，需兼顾避障、预测人类运动并尊重个人空间。  
🛠 **方法**：提出SFM-NMPC框架，将社会力模型嵌入非线性模型预测控制的优化循环中，联合预测人与机器人轨迹，并设计社交成本函数引导行为。  
✨ **主要贡献**：在复杂模拟场景中，该方法以20Hz实时运行，在社交合规性指标上优于现有基线，并通过轨迹分析与消融实验验证了嵌入社会力模型和成本项的有效性。
