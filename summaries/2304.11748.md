# IDLS: Inverse Depth Line based Visual-Inertial SLAM

- arXiv: https://arxiv.org/abs/2304.11748v2
- arXiv ID: 2304.11748

---

🔍 **核心问题**：现有基于点线特征的视觉惯性SLAM方法在室内复杂环境中，使用Plücker矩阵或正交表示法描述直线，需要至少四个变量，导致计算负担重且精度受限。

🛠 **方法**：提出逆深度直线表示法，仅用两个变量（两个端点的逆深度）建模直线特征，并引入鲁棒的直线三角化方法、新型直线重投影误差模型以及两步优化法（先确定直线，再估计相机位姿）。

✨ **主要贡献**：IDLS方法在多个具有挑战性的室内数据集上，相比当前最先进的点线SLAM方法，精度更高、鲁棒性更强，且计算开销更低。
