# Feedback Manipulation Regularization: Enabling Offline Agent Alignment for Imitation Learning

- arXiv: https://arxiv.org/abs/2607.07859v1
- arXiv ID: 2607.07859

---

🔍 **核心问题**：现有方法主要将人类演示和反馈作为多阶段信号用于上下文赌博机框架的语言生成任务，缺乏在完全序列决策环境中利用这两种互补信号进行单阶段离线训练的研究。

🛠 **方法**：提出反馈操作正则化（FMR），一种算法无关的方法，将评估性反馈作为纠正信号，增强模仿学习策略的对齐能力。

✨ **主要贡献**：在改进的Safety Gymnasium环境中，FMR显著提升了对齐能力，最多减少98%的对齐偏差，且在数据稀缺、噪声演示等有限数据场景下仍保持鲁棒性。
