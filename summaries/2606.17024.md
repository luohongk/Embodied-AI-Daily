# ExpRL: Exploratory RL for LLM Mid-Training

- arXiv: https://arxiv.org/abs/2606.17024v1
- arXiv ID: 2606.17024

---

🔍 **核心问题**：针对大语言模型推理能力提升中，稀疏奖励强化学习依赖基础模型覆盖度，而人工设计中间训练策略成本高且难以应对复杂问题组合需求的问题。

🛠 **方法**：提出ExpRL方法，利用大规模人类问答数据中的参考答案作为“奖励支架”，通过LLM裁判将模型生成的推理轨迹与参考答案对比，构建问题级评分规则，并分配结果级或过程级密集奖励，从而强化部分进展、有效中间推理等稀疏奖励难以捕捉的行为。

✨ **主要贡献**：在数学推理任务上，ExpRL相比监督微调、稀疏奖励GRPO和自蒸馏方法，能提供更强的强化学习初始化效果，并为后续稀疏奖励强化学习奠定更优基础；混合领域实验表明其可扩展至数学以外的场景。
