# Wiki-R1: Incentivizing Multimodal Reasoning for Knowledge-based VQA via Data and Sampling Curriculum

- arXiv: https://arxiv.org/abs/2603.05256v2
- arXiv ID: 2603.05256

---

🔍 **核心问题**：知识型视觉问答（KB-VQA）中，多模态大语言模型（MLLM）因知识库检索噪声大、结构化百科知识与预训练分布存在差异，导致推理和领域适应困难。

🛠 **方法**：提出Wiki-R1框架，采用基于数据生成的课程强化学习，通过可控课程数据生成（调节检索器产生不同难度样本）和课程采样策略（基于奖励估计难度并传播至未观测样本），逐步引导模型从预训练分布过渡到KB-VQA目标分布。

✨ **主要贡献**：在Encyclopedic VQA和InfoSeek基准上取得新最优结果，准确率分别从35.5%提升至37.1%、从40.1%提升至44.1%，验证了课程强化学习对多模态推理的有效激励。
