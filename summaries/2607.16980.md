# Efficient Audio-Visual Event Recognition via Knowledge Distillation and Dynamic INT8 Quantization of a Hybrid Cross-Attention Network

- arXiv: https://arxiv.org/abs/2607.16980v1
- arXiv ID: 2607.16980

---

🔍 **核心问题**：基于Transformer的音视频事件识别模型计算复杂度和内存占用过高，难以部署在资源受限的边缘设备上。

🛠 **方法**：提出一种结合架构压缩、知识蒸馏和动态INT8量化的高效压缩框架，先构建高容量教师模型（VideoMAE+AST+混合交叉注意力），再设计轻量学生模型，通过蒸馏训练后进一步进行动态INT8后训练量化。

✨ **主要贡献**：在AVE数据集上，多模态融合模块参数量减少59.06%，分类精度仅下降2.14%；动态INT8量化将模型从10.71 MB压缩至2.04 MB，同时保持竞争性识别性能，实现了精度与效率的有效平衡。
