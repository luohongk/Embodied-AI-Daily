# Hierarchical Reinforcement Learning Empowered Task Offloading in V2I Networks

- arXiv: https://arxiv.org/abs/2405.11352v2
- arXiv ID: 2405.11352

---

🔍 **核心问题**：在V2I网络中，车辆将计算任务卸载到路边单元时，如何优化处理延迟、能耗和边缘计算成本，特别是任务包含相互依赖的子任务（DAG结构）时的卸载策略。

🛠 **方法**：提出一种分层强化学习卸载方案，利用图神经网络与注意力机制提取DAG子任务间的依赖关系，并开发参数化DRL算法处理包含离散和连续动作的分层动作空间。

✨ **主要贡献**：首次将分层强化学习与图注意力网络结合解决V2I任务卸载问题，在真实车速数据集上验证了该方法能有效降低系统总开销。
