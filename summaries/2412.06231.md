# A Scalable Decentralized Reinforcement Learning Framework for UAV Target Localization Using Recurrent PPO

- arXiv: https://arxiv.org/abs/2412.06231v1
- arXiv ID: 2412.06231

---

## 0. 概览（Concise Summary）
- **主题（Topic）**：无人机（UAV）在感知退化环境（如无GNSS/GPS信号）中的目标定位任务，采用多智能体强化学习。
- **问题（Problem）**：解决在缺乏GPS信号和能见度受限的复杂环境中，单个或多个无人机如何仅凭有限传感器信息高效、准确地定位辐射目标的问题。
- **方法（Method）**：提出一个基于循环PPO（Recurrent PPO）的分散式强化学习框架。该框架为每个无人机独立训练策略，通过共享搜索网格地图实现协同，并利用LSTM网络处理部分可观测环境下的时序依赖。
- **创新（Innovation）**：1) 将LSTM与PPO结合，提升模型在部分可观测马尔可夫决策过程（POMDP）中的性能；2) 提出一种可扩展的分散式多无人机协同定位框架，无需中心控制器；3) 设计了包含探索奖励、信号强度奖励和碰撞惩罚的奖励系统。
- **意义（Significance）**：为无人机在GPS拒止环境下的自主搜索与救援、环境监测等任务提供了一种高效、鲁棒的解决方案，并展示了多机协同在减少定位步数方面的优势。
- **一句话总结**：本文提出一种基于循环PPO的分散式强化学习框架，使单个或两个无人机能在无GPS信号的环境中，通过共享地图和有限传感器信息高效定位辐射目标。

## 1. 研究动机（Motivation）
- **问题与空白**：现有无人机目标定位方法严重依赖GPS信号和清晰视觉，但在室内、矿井、灾难现场等感知退化环境（GNSS/GPS信号缺失、能见度低）中，这些条件无法满足。传统方法（如基于GPS的导航）在这些场景下失效。
- **重要性**：搜索与救援（SAR）、环境监测（如核辐射源定位、火灾定位）等关键应用需要在上述恶劣环境中快速定位目标，直接关系到生命财产安全。
- **挑战**：无人机在部分可观测环境中（只能感知局部信息，无法获取全局地图或目标位置）需要做出决策；多机协同需要解决通信、碰撞避免和任务分配问题，同时保持可扩展性。

## 2. 创新点（Innovation）
- **核心创新**：提出一种**可扩展的分散式循环PPO框架**，用于无人机在感知退化环境中的目标定位。
- **与现有方案的不同**：
  - 现有工作（如Boiteau等人）使用POMDP和热成像相机，但本文仅使用更简单的信号传感器和检测传感器（如深度相机），硬件需求更低。
  - 相比标准PPO、DQN、DDPG，本文通过**集成LSTM层**（256个隐藏单元）构建循环PPO，使模型能够利用历史观测和动作的时序信息，更好地处理部分可观测性，避免陷入局部最优。
  - 多机方案采用**完全分散式**架构：每个无人机独立运行自己的策略，仅通过共享一个全局网格地图进行隐式协调，无需中心控制器或显式通信协议，因此易于扩展到更多无人机。
- **为何奏效**：LSTM的记忆能力让无人机能记住之前探索过的区域，避免重复搜索；分散式架构让多机可以并行探索不同区域，共享地图信息，从而减少总定位步数。

## 3. 主要内容（Main Content）【重点】
### 3.1 设计架构与方法
- **整体框架**：系统包括一个仿真环境（模拟室内布局）、无人机智能体（使用循环PPO策略）和奖励系统。无人机在环境中移动，感知局部信息（障碍物、地图状态、信号强度），并选择动作。
- **单无人机**：一个智能体独立探索并定位目标。
- **多无人机（两个）**：每个无人机独立运行相同的循环PPO策略，但共享一个全局网格地图（每个无人机更新后，地图对所有无人机可见）。奖励系统中增加了对邻近无人机的惩罚（-2）和碰撞惩罚（-50），以促进分散探索。

### 3.2 关键算法与数学推导
- **模型**：循环PPO（Recurrent PPO），其损失函数为：
  $$L^{CLIP+VF+S}(\theta) = \hat{E}_t \left[ L_t^{CLIP}(\theta) - c_1 L_t^{VF}(\theta) + c_2 S[\pi_\theta](s_t) \right]$$
  其中：
  - $L_t^{CLIP}(\theta)$ 是裁剪的替代目标函数：
    $$L^{CLIP}(\theta) = \hat{E}_t \left[ \min \left( r_t(\theta) \hat{A}_t, \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) \hat{A}_t \right) \right]$$
  - $r_t(\theta)$ 是新旧策略的概率比，但依赖于当前观测 $o_t$ 和LSTM隐藏状态 $h_t$：
    $$r_t(\theta) = \frac{\pi_\theta(a_t | o_t, h_t)}{\pi_{\theta_{old}}(a_t | o_t, h_t)}$$
  - $\hat{A}_t$ 是通过广义优势估计（GAE）计算的优势函数。
  - $L_t^{VF}(\theta)$ 是价值函数的平方误差损失。
  - $S[\pi_\theta](s_t)$ 是策略的熵奖励，鼓励探索。
  - $c_1, c_2$ 是系数。

### 3.3 模型、训练与数据集
- **模型架构**（图3）：输入层（17或21维观测）→ 全连接层 → LSTM层（256单元） → 全连接层 → 策略头（动作概率）和价值头（状态值）。
- **训练数据集**：通过8种不同的仿真环境（室内布局），改变无人机和目标位置，生成80张不同的地图。多无人机训练时放置两个无人机。每50次重置更换一次环境，训练数百万个时间步（epochs）。
- **观测空间**（表II）：
  - 单无人机：17维，包括8个方向障碍物、8个方向地图状态（表I：0未知/1障碍/2走过一次/3走过两次/4走过三次以上）、1个信号强度（分三档，表III）。
  - 多无人机：21维，额外增加4个方向是否有邻近无人机的信息。
- **动作空间**：8个方向的平移运动（简化，无旋转）。
- **奖励系统**（表IV）：
  - 探索奖励：移动到未知单元格（+2）。
  - 惩罚：撞障碍物（-50）、重复访问（-1或-4）。
  - 信号奖励：基于信号强度的连续奖励（0-250）。
  - 目标到达奖励：+1000。
  - 多无人机额外：邻近无人机（-2）、碰撞（-50）。

### 3.4 实验设置与结果
- **测试设置**：10种不同的室内布局地图，改变无人机和目标位置，生成100个独特测试场景。成功定义为到达目标。
- **评价指标**：
  - 成功率 $r = h / 100$，其中 $h$ 是成功定位次数。
  - 平均步数 $\bar{r} = T_q / s$，其中 $T_q$ 是成功定位的总步数，$s$ 是成功次数。
- **结果**（表V、VI）：
  - **单无人机**：在3.8百万次训练迭代后，成功率达**93%**，平均步数183.82。
  - **双无人机**：在4.15百万次训练迭代后，成功率达**86%**，平均步数162.89。虽然成功率略低，但平均步数显著减少（约12%），证明了协同效率。

  - 图5：单无人机评估图，显示成功率随训练步数上升并稳定在93%。

### 3.5 技术细节
- **仿真环境**：自定义网格世界，模拟室内布局，包含障碍物、未知区域和信号源。信号强度分为三个同心圆区域（表III）。
- **训练细节**：使用8种不同环境，每50次重置更换，训练数百万epochs。未提及具体超参数（如学习率、折扣因子、GAE参数等），但基于标准PPO实现。
- **可扩展性**：作者声称，将双无人机方案扩展到更多无人机只需最小调整，因为每个无人机独立运行，仅共享网格地图。

## 4. 意义与影响（Significance and Impact）
- **解决的问题与贡献**：
  - 提出了一种在GPS拒止环境下有效的无人机目标定位方法，仅需简单传感器（信号检测+深度/相机）。
  - 验证了循环PPO（LSTM+PPO）在部分可观测导航任务中的优越性。
  - 展示了分散式多无人机协同的潜力：虽然双无人机成功率（86%）略低于单机（93%），但平均步数减少，证明了并行探索的效率。
- **对未来研究/应用的影响**：
  - 为无人机群在搜索救援、环境监测、农业调查等领域的实际部署提供了理论基础和实验验证。
  - 分散式架构易于扩展到大规模无人机群，有望进一步缩短定位时间。
- **局限与开放问题**（作者提及）：
  - 性能有待提升（如双机成功率低于单机）。
  - 当前仅定位单个静态目标，未来需扩展到多个目标或移动目标。
  - 未在真实无人机上验证，仅仿真。
  - 未考虑障碍物对传感器读数的影响（如遮挡信号）。
  - 可进一步优化算法（如使用进化算法）并评估冗余路径等指标。

## 5. 澄清与简化（Clarifications and Simplifications）
- **部分可观测马尔可夫决策过程（POMDP）**：可以想象成无人机在玩一个“蒙眼寻宝”游戏。它只能看到自己脚下和周围一小块区域（观测），不知道整个地图和宝藏位置。因此，它必须记住自己走过的路（LSTM的作用），才能做出明智的决策，避免原地打转。
- **循环PPO（Recurrent PPO）**：标准PPO像一个“健忘”的决策者，只看当前情况做决定。循环PPO给决策者加了一个“短期记忆”（LSTM），让它能记住过去几步发生了什么，从而更好地理解当前情况（例如，知道“我刚刚向左走了，现在应该向右探索”）。
- **分散式 vs 集中式**：集中式像有一个“总指挥官”给所有无人机发指令，通信量大且单点故障风险高。分散式像每个无人机都是“独立士兵”，只通过共享“地图笔记”（网格地图）来协调，各自决定下一步去哪，更灵活、更鲁棒。
- **信号强度分档**：目标信号像Wi-Fi信号，离得近信号强（Section 1），中等距离信号中等（Section 2），远距离信号弱（Section 3），再远就收不到了。无人机通过信号强度大致判断距离，但不知道具体方向，所以需要探索。
