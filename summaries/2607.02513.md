# LACUNA: A Testbed for Evaluating Localization Precision for LLM Unlearning

- arXiv: https://arxiv.org/abs/2607.02513v1
- arXiv ID: 2607.02513

---

🔍 **核心问题**：现有的大语言模型（LLM）遗忘方法仅在输出层面评估效果，无法确认是否真正从模型参数中擦除知识，导致遗忘后的信息可能被重新提取（如重浮现攻击）。

🛠 **方法**：提出LACUNA测试平台，通过掩码连续预训练将合成个人身份信息（PII）注入OLMo模型（1B和7B参数）的预定义参数中，提供参数级别的真实定位基准，用于评估遗忘方法是否精准作用于存储知识的权重。

✨ **主要贡献**：首次实现遗忘方法的参数级定位评估，发现当前最先进方法虽在输出层面表现良好，但定位精度低且易受重浮现攻击；同时证明精准定位后，简单梯度方法也能实现强擦除和抗攻击能力，推动基于定位的鲁棒遗忘研究。
