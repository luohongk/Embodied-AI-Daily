# Supervised Deep Multimodal Matrix Factorization for Interpretable Brain Network Analysis

- arXiv: https://arxiv.org/abs/2605.13312v1
- arXiv ID: 2605.13312

---

🔍 **核心问题**：如何从多模态脑网络数据中提取可解释的、具有预测能力的特征，以克服传统无监督单图聚类方法的局限性。

🛠 **方法**：提出监督式深度多模态矩阵分解（SD3MF），将对称非负矩阵三分解从无监督单图聚类扩展为多模态图群体的监督预测，通过编码器-解码器联合优化图重构与监督预测，并利用自适应权重实现数据驱动的多模态融合。

✨ **主要贡献**：首次将深度层次化分解与多模态共享表示结合，在脑连接组数据集上显著优于CNN和GNN等强基线模型，同时通过社区级交互矩阵提供生物学可解释的洞察。
