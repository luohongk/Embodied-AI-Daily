# Can an Old Dog Be Taught New Tricks? Taking LLMs Beyond Sentence Level Translation

- arXiv: https://arxiv.org/abs/2607.14040v1
- arXiv ID: 2607.14040

---

🔍 **核心问题**：如何使大语言模型（LLM）超越逐句翻译的传统范式，实现基于整篇文档和语料库信息的语用重述翻译。

🛠 **方法**：提出PAT系统，基于检索增强生成（RAG）技术，结合用户配置的规范与可比语料库中的段落、章节和文档级示例，驱动LLM进行整篇文档生成。

✨ **主要贡献**：通过实验证明LLM能够被引导进行语用重述而非逐句翻译，但重述效果仍需改进；同时提供了关于自动翻译系统设计、语料库构建及翻译质量评估方法的深入讨论。
