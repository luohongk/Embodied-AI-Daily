# DIFF-IPPO: Diffusion-Based Informative Path Planning with Open-Vocabulary Belief Maps

- arXiv: https://arxiv.org/abs/2606.16780v1
- arXiv ID: 2606.16780

---

🔍 **核心问题**：在探索与目标搜索任务中，如何基于非高斯、多模态的语义信念图生成全局信息路径，以最大化目标检测概率或信息增益。

🛠 **方法**：提出DIFF-IPPO框架，结合开放词汇信念图生成器与基于扩散模型的规划器，在信念图上直接生成全局轨迹，使传感器覆盖集中于高信念区域。

✨ **主要贡献**：在不同数据集场景下实现81.49%–86.55%的归一化检测得分；在模拟搜救场景中，五架无人机通过批量信念图条件轨迹生成，在3.5分钟内首次检测到目标建筑。
