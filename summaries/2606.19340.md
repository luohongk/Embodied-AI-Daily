# Zero-Shot Long-Horizon Dexterous Manipulation via Multi-View 3D-Grounded VLM Reasoning

- arXiv: https://arxiv.org/abs/2606.19340v1
- arXiv ID: 2606.19340

---

🔍 **核心问题**：解决零样本条件下，基于语言指令的长时间跨度灵巧操作任务，无需预训练或微调策略，即可在未见过的物体和场景中执行抓取与工具使用。

🛠 **方法**：利用多视角校准RGB图像，通过视觉语言模型（VLM）生成参考帧任务基元和2D关键点，再经多视角融合（三角测量与射线投票）提升为3D关键点；对于工具使用，检索原子动作并对齐6D轨迹；对于灵巧执行，扩展抓取关键点为任务条件抓取区域，并生成抓取-运动对。

✨ **主要贡献**：提出零样本框架，通过多视角3D融合显著提升3D基元定位精度和执行可靠性，优于单视角RGB-D基元和微调VLA基线；支持闭环状态验证与重规划，实现长时间跨度灵巧操作，在真实实验中成功应用于未见物体和工具使用任务。
