# A time-series classification framework for individual-level absenteeism prediction under severe class imbalance

- arXiv: https://arxiv.org/abs/2606.31532v1
- arXiv ID: 2606.31532

---

🔍 **核心问题**：现有员工缺勤预测方法无法实现真正的未来预测，且面临严重的类别不平衡问题。  
🛠 **方法**：提出时间序列分类（TSC）框架，将历史出勤序列与未来缺勤标签分离；采用二元焦点损失（BFL）和几何平均损失（G-Mean）处理不平衡数据，并评估LSTM、CNN及LSTM-FCN三种深度学习架构。  
✨ **主要贡献**：发现BFL中平衡权重α主导性能（α≈0.023），G-Mean无需调参；LSTM-FCN在严重不平衡下实现约80%的平衡准确率，且窗口大小40-80天、批次≥64时性能稳定。
