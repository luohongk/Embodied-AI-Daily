# SPECTRA: Context-Conditioned Spectral Movement Primitives for Robot Skill Generalization

- arXiv: https://arxiv.org/abs/2607.06978v1
- arXiv ID: 2607.06978

---

🔍 **核心问题**：机器人模仿学习中，现有方法在生成符合任务几何特征的运动轨迹时，常因后处理（如滤波、平滑）扭曲末端执行器路径，且难以同时保证关节空间的动态可行性。

🛠 **方法**：提出频谱运动基元（SMP），将演示轨迹用截断傅里叶系数表示，通过低频任务带捕获主要几何特征，并利用上下文条件的高斯混合模型/回归（GMM/GMR）预测任务带系数；再通过相位耦合调节器限制关节速度与加速度，而不修改频谱系数。

✨ **主要贡献**：实现了紧凑的几何重建、跨未见任务帧的泛化、显著减少动态违规和加加速度，并在Franka Panda机器人上验证了末端路径的保持能力。
