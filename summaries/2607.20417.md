# ATSplat: Compact Feed-forward 3D Gaussian Splatting with Adaptive Token Expansion

- arXiv: https://arxiv.org/abs/2607.20417v1
- arXiv ID: 2607.20417

---

🔍 **核心问题**：现有前馈式3D高斯泼溅方法在生成高斯原语时依赖于像素对齐，导致原语数量与场景复杂度无关，产生大量冗余，失去了自适应分配能力。

🛠 **方法**：提出ATSplat框架，通过自适应3D令牌机制实现稀疏到自适应的原语分配。首先从图像中提取稀疏的锚点令牌，再将其回归为局部高斯并学习3D偏移，最后通过自适应令牌扩展模块根据渲染误差图选择性扩展高不确定性令牌。

✨ **主要贡献**：在保持高质量渲染的同时，将高斯数量减少5.7倍以上；在RealEstate10K和DL3DV数据集上达到最先进渲染质量；从12张512×960分辨率图像输入，单GPU可在1秒内完成重建，并以1136 FPS渲染新视角，仅需31.1万个高斯。
