# FlatManifold: Robust Continual Learning under Severe Label Noise and Domain Shifts via Intrinsic Manifold Flattening

- arXiv: https://arxiv.org/abs/2607.05201v1
- arXiv ID: 2607.05201

---

🔍 **核心问题**：在非平稳流式环境中，同时应对复杂的非线性域迁移和严重、未校准的标签噪声，避免灾难性遗忘和梯度破坏。  
🛠 **方法**：提出FlatManifold框架，利用基于核技巧的Nyström流形展平映射，将特征分布投影到正交化的再生核希尔伯特空间（RKHS）的固定正交目标拓扑上，并结合岭正则化与持续拓扑制动项（基于历史协方差矩阵）进行优化。  
✨ **主要贡献**：无需复杂样本过滤，通过流形结构线性化本身天然抵抗极端标签噪声（如40%对称噪声）和跨会话域迁移，在真实机器人多会话数据集上显著优于标准基线，证明结构线性化是抵御分布式标签破坏的数学屏障。
