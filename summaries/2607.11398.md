# GNSS Spoofing Detection in TDD Networks: A 3GPP Standards-Based Security Framework

- arXiv: https://arxiv.org/abs/2607.11398v1
- arXiv ID: 2607.11398

---

🔍 **核心问题**：时分双工（TDD）移动网络依赖GNSS授时实现±1.5μs同步精度，但GNSS欺骗攻击已成为实际威胁，而3GPP现有管理框架（SA5）和安全框架（SA3）缺乏标准化检测与报告机制。

🛠 **方法**：提出一个在现有3GPP管理结构内运行的检测监控框架，包括GNSS定时告警与性能计数器（符合TS 28.111/28.552）、拓扑感知关联机制（按主时钟分组gNB-DU分类异常），以及将故障管理与SECHAND事件处理（TR 33.894）桥接的安全事件机制。

✨ **主要贡献**：蒙特卡洛仿真表明，在良好PTP网络条件下，对高于0.5 ns/s的漂移率检测概率超过95%，误报率低于1%；框架无需新接口、代际无关，并通过场景分析验证了其区分欺骗与信号丢失、设备故障及维护瞬态的能力。
