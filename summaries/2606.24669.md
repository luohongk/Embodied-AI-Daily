# LaGO: Latent Action Guidance for Online Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.24669v1
- arXiv ID: 2606.24669

---

🔍 **核心问题**：现有方法将大语言模型（LLM）直接用作控制器，需要精确的动作生成，在实际应用中不可靠。  
🛠 **方法**：提出LaGO框架，利用预训练LLM作为潜在动作先验，以软引导方式辅助在线策略优化（如PPO），而非直接控制。  
✨ **主要贡献**：在CLEVR-Robot和Meta-World基准上，LaGO显著提升奖励和成功率（如成功率从15.1%升至27.2%），且更强的LLM能提供更有效的引导。
