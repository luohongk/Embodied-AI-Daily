# Calibration Without Comprehension: Diagnosing the Limits of Fine-Tuning LLMs for Vulnerability Detection in Systems Software

- arXiv: https://arxiv.org/abs/2606.20502v1
- arXiv ID: 2606.20502

---

🔍 **核心问题**：探究大型语言模型（LLM）在系统软件漏洞检测中是否真正具备安全推理能力，还是仅依赖数据污染进行模式匹配。

🛠 **方法**：构建CWE-Trace框架，包含834个手动整理的Linux内核样本（覆盖74个CWE），采用严格时间分割（2025年前历史集/无泄漏后截止集），并引入方向性失败指数（DFI）和层次距离与方向（HDD）两种诊断指标，评估8个原始LLM和15个LoRA微调变体。

✨ **主要贡献**：发现数据污染无显著优势（84%污染样本无可用记忆信号），微调仅调整输出阈值而未改变决策策略（校准无理解），最佳检测准确率仅52.1%，CWE分类Top-1准确率低于1.3%，证实当前LLM缺乏可靠的安全推理能力。
