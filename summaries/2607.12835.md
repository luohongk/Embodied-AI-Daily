# Can LLMs Write Reliable Rubrics? A Meta-Evaluation for Experiment Reproduction

- arXiv: https://arxiv.org/abs/2607.12835v1
- arXiv ID: 2607.12835

---

🔍 **核心问题**：当前基于论文复现的评估基准（如PaperBench）依赖专家人工构建评分标准（rubrics），成本高且难以规模化，亟需探索大语言模型（LLM）能否自动生成可靠的评分标准。

🛠 **方法**：将评分标准重构为清单格式，使用两种骨干模型在四种生成设置下自动生成评分标准，并通过语义相似性（内在评估）和与人工评分标准的一致性（外在评估）进行元评估。

✨ **主要贡献**：首次系统评估LLM生成评分标准的可靠性，发现增强生成设置能显著提升下游评估对齐度（接近人类基线），但LLM生成的评分标准存在粒度过细、偏向高分、领域适应性弱等局限。
