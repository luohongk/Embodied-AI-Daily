# Monocular 3D Occupancy Perception for Robots on Sidewalks via Hybrid 2D-3D Learning

- arXiv: https://arxiv.org/abs/2606.19122v1
- arXiv ID: 2606.19122

---

🔍 **核心问题**：针对人行道场景中机器人导航面临的拥挤、杂乱且非结构化环境，现有3D占用感知方法依赖昂贵的LiDAR-RGB配对数据和密集3D监督，且无法适应人行道特性。

🛠 **方法**：提出WalkOCC混合光线步进单目3D占用感知框架，通过LiDAR-RGB配对数据引导伪占用监督，并利用大规模未配对单目图像联合学习图像级表征，实现几何与语义的混合学习。

✨ **主要贡献**：无需昂贵3D标注即可提升预测精度，实现路缘、排水沟等细微城市结构的精细分割，对环境变化和跨本体迁移具有鲁棒性；同时发布包含多地点、多时段LiDAR-相机配对数据及3D语义占用标注的Sidewalk3D数据集。
