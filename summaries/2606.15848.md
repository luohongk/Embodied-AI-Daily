# EmoZone-Talker: Regional Semantic Control of Audio-Driven 3DGS Talking Heads via Facial Action Units

- arXiv: https://arxiv.org/abs/2606.15848v1
- arXiv ID: 2606.15848

---

🔍 **核心问题**：现有音频驱动说话头生成方法难以实现精细、可解释且可编辑的面部表情控制，主要由于语音驱动动态与显式表情信号之间存在冲突，导致空间耦合和时间不稳定。

🛠 **方法**：提出EmoZone-Talker框架，通过协同区域与优先注意力偏置（SZ-PAB）显式解耦不同模态贡献，并设计通道独立时间AU编码器（CIT-AE）建模时间连贯的面部动作单元动态，最终将表示集成到3D高斯变形中。

✨ **主要贡献**：实现了精确且可解释的面部表情控制，显著提升上半脸准确性和时间连贯性，同时保持高渲染质量和准确唇同步，代码将开源。
