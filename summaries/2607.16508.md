# Differentiable Reinforcement Learning for Path Tracking by an Agile Fish-Like Robot

- arXiv: https://arxiv.org/abs/2607.16508v1
- arXiv ID: 2607.16508

---

🔍 **核心问题**：仿生鱼形机器人的运动控制和路径跟踪因流体-结构交互建模困难及非线性欠驱动动力学而极具挑战，且缺乏兼具计算速度和精度的仿真环境。  
🛠 **方法**：开发了一种高效计算近似鱼形机器人运动的仿真平台，并采用PID控制结合可学习变增益，通过时间反向传播和课程训练进行强化学习优化。  
✨ **主要贡献**：在仿真中训练的策略成功迁移至物理机器人平台，实现了与仿真高度匹配的路径跟踪性能，验证了方法的有效性和跨平台泛化能力。
