# Beyond Bayer: Task-Optimal Sensor Co-Design for Robust Autonomous-Driving Segmentation

- arXiv: https://arxiv.org/abs/2606.24096v1
- arXiv ID: 2606.24096

---

🔍 **核心问题**：在自动驾驶感知任务中，如何通过优化相机传感器本身的设计（而非仅依赖模型规模或数据融合）来提升语义分割的鲁棒性。

🛠 **方法**：构建一个可微分的RAW图像到任务（语义分割）的端到端管线，联合优化传感器的三个自由度：光谱颜色滤波阵列（CFA）权重、点扩散函数（光学系统）和噪声参数。

✨ **主要贡献**：发现学习CFA权重是提升性能的关键（mIoU提升+0.017至+0.023），而优化点扩散函数反而有害（-0.020 mIoU），噪声优化效果微弱；增大CFA块尺寸会降低性能；最终提出简单有效的方案：学习2×2 CFA权重并保持单位点扩散函数。
