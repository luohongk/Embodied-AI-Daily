# Hierarchical Reinforcement Learning for Neural Network Compression (HiReLC): Pruning and Quantization

- arXiv: https://arxiv.org/abs/2606.26002v1
- arXiv ID: 2606.26002

---

🔍 **核心问题**：针对深度神经网络压缩中剪枝与量化联合优化搜索空间大、计算成本高的问题，提出自动化联合压缩方法。

🛠 **方法**：提出HiReLC分层强化学习框架，底层代理独立为每个块选择位宽、剪枝保留比等配置，高层代理基于Fisher信息灵敏度估计通过集成投票协调全局预算分配，并采用迭代主动学习循环结合轻量级MLP代理进行奖励塑造以降低策略评估成本。

✨ **主要贡献**：在Vision Transformer和CNN基准上实现5.99-6.72倍参数存储压缩比，在一种设置下准确率提升3.83%，其他设置下仅下降0.55-5.62%，验证了分层策略分解与灵敏度感知引导在联合压缩中的有效性。
