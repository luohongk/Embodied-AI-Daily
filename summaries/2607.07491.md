# Smooth Operator: A Real-Time Sampling-Based Algorithm for Kinematic Hand Retargeting

- arXiv: https://arxiv.org/abs/2607.07491v1
- arXiv ID: 2607.07491

---

🔍 **核心问题**：现有基于梯度的运动学手部重定向算法易陷入局部最优，导致输出抖动，影响遥操作数据质量和用户体验。

🛠 **方法**：提出采样重定向器（SBR），一种基于采样控制的无梯度重定向算法，专为低抖动、实时运动学重定向设计。

✨ **主要贡献**：在仿真和18名用户参与的复杂操作任务实验中，SBR实现了最高任务成功率（54.1%），并显著降低操作者认知疲劳（NASA-TLX评分36.4/100），为灵巧操作提供了高效直观的重定向方案及基准测试方法。
