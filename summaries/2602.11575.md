# ReaDy-Go: Real-to-Sim Dynamic 3D Gaussian Splatting Simulation for Environment-Specific Visual Navigation with Moving Obstacles

- arXiv: https://arxiv.org/abs/2602.11575v3
- arXiv ID: 2602.11575

---

🔍 **核心问题**：现有视觉导航模型在真实动态环境中面临模拟到现实（sim-to-real）的差距，且难以针对特定部署环境（如家庭、餐厅、工厂）训练策略，尤其是缺乏对动态障碍物（如行人）的逼真模拟。

🛠 **方法**：提出ReaDy-Go管道，通过将静态3D高斯泼溅（GS）场景与可动画化的人体GS模型结合，构建逼真的动态仿真环境，并利用该环境生成导航数据集，训练对动态障碍物和sim-to-real差距鲁棒的导航策略。

✨ **主要贡献**：① 首个集成静态场景GS与可动画人体GS的动态仿真器，支持从2D轨迹合成逼真人体运动；② 基于该仿真器的导航数据集生成框架，包含针对动态GS的专家规划器；③ 在仿真和真实实验中，ReaDy-Go在目标环境中超越基线，且零样本迁移到未见环境仍表现良好。
