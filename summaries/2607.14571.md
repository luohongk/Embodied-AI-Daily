# Gate-Zero Growth: A Geometric Framework for Function-Preserving Continual Learning

- arXiv: https://arxiv.org/abs/2607.14571v1
- arXiv ID: 2607.14571

---

🔍 **核心问题**：解决持续学习中新增网络容量时，如何在不破坏已有功能（函数保持）的前提下安全激活新参数，避免灾难性遗忘。

🛠 **方法**：提出“门控零增长”算子，通过零初始化门控添加残差块，利用横截性条件实现功能雅可比矩阵的秩分离，使旧方向不变、新权重方向平坦、仅门控方向提供一阶功能变化。

✨ **主要贡献**：在300M→857M Transformer上实现近零旧域遗忘（Δ_A<0.1），遗忘量比非函数保持方法低一个数量级；同时统一了LoRA、ReZero等方法的几何分析，确立门控零增长为安全容量激活的规范实例。
