# Bridging the 2D-3D Gap: A Hierarchical Semantic-Geometric Map for Vision Language Navigation

- arXiv: https://arxiv.org/abs/2606.00095v1
- arXiv ID: 2606.00095

---

🔍 **核心问题**：解决视觉语言导航中2D视觉语言模型与3D空间推理之间的语义-几何鸿沟，导致零样本导航不可靠的问题。

🛠 **方法**：提出分层语义-几何地图（HSGM），将3D几何信息转化为多通道俯视图，包含几何、语义和决策三个层级；结合视觉语言模型作为高层语义规划器与经典路径规划算法，实现语义推理与动作执行解耦。

✨ **主要贡献**：在R2R-CE和RxR-CE基准上，零样本框架达到最先进性能，甚至超越部分监督方法；通过指令分解缓解长程导航中的进度遗忘或幻觉问题。
