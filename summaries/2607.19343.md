# Masked Visual Actions for Unified World Modeling

- arXiv: https://arxiv.org/abs/2607.19343v1
- arXiv ID: 2607.19343

---

🔍 **核心问题**：如何让视频模型理解并执行物理世界中的机器人动作，同时保持其在视觉交互中学习到的丰富先验知识。

🛠 **方法**：提出“掩码视觉动作”（Masked Visual Actions），将动作表示为视频中任意实体轨迹的部分揭示，通过像素空间控制接口实现前向动力学建模（预测场景对动作的响应）和逆向建模（从期望物体运动生成机器人动作）。

✨ **主要贡献**：仅用15小时真实视频和仿真掩码示例微调，单个模型即可在多样场景和多实体形态下实现高视觉保真度与控制性；在操作任务中，模型生成的想象轨迹可评估策略、辅助基于模型的规划决策，并支持从物体运动逆向合成机器人动作。
