# History-Conditioned Spatio-Temporal Visual Token Pruning for Efficient Vision-Language Navigation

- arXiv: https://arxiv.org/abs/2603.06480v2
- arXiv ID: 2603.06480

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型在视觉-语言导航（VLN）任务中计算成本高、延迟大，难以满足机器人实时部署的需求。

🛠 **方法**：提出一种无需重新训练的时空视觉令牌剪枝框架，通过空间令牌选择处理当前视图，并结合时空压缩处理历史记忆，利用注意力令牌重要性和查询引导的时空过滤，保留导航关键信息并减少冗余计算。

✨ **主要贡献**：在标准VLN基准测试中显著优于现有剪枝策略，在极端剪枝场景下仍保持高导航精度和推理效率，并在Unitree Go2四足机器人上验证了低延迟指令跟随导航的可靠性。
