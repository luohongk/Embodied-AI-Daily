# Test-Time Scaling for World Action Models via Zero-Shot Geometric Evaluation

- arXiv: https://arxiv.org/abs/2607.17454v1
- arXiv ID: 2607.17454

---

🔍 **核心问题**：机器人控制中，如何在执行动作前判断是否需要额外计算资源来提升模型推理效果，避免无效计算。

🛠 **方法**：提出无需训练的“门控式选择性测试时扩展”框架，通过交叉视图深度重投影一致性评估预测未来帧的几何质量，并利用轻量级动作-未来一致性门控决定是否启用扩展采样。

✨ **主要贡献**：在多个基准测试中，固定预算方法稳定提升任务成功率（如RoboCasa提升2.1%），门控机制以仅26.2%的额外采样点恢复74.8%的增益，并揭示了高N值时性能饱和的失败模式。
