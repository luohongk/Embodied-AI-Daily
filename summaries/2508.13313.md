# Flow Matching for Efficient and Scalable Data Assimilation

- arXiv: https://arxiv.org/abs/2508.13313v4
- arXiv ID: 2508.13313

---

🔍 **核心问题**：数据同化（DA）在估计高维非线性动态系统状态时，现有生成模型（如EnSF）计算成本高，难以兼顾效率与可扩展性。

🛠 **方法**：提出无训练的流匹配框架EnFF，利用蒙特卡洛估计边际流场、局部引导观测同化，并基于贝叶斯DA公式设计新型流路径，可泛化粒子滤波与集合卡尔曼滤波。

✨ **主要贡献**：EnFF在保持高精度同时显著加速采样，通过改进成本-精度权衡实现高维基准测试的可扩展性，验证了流匹配在高效DA中的潜力。
