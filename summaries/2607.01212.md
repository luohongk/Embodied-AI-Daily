# FurnitureVLA: Learning Long-Horizon Bimanual Furniture Assembly with Vision-Language-Action Model

- arXiv: https://arxiv.org/abs/2607.01212v1
- arXiv ID: 2607.01212

---

🔍 **核心问题**：现有机器人家具组装研究多局限于玩具级场景或单臂操作，缺乏对真实尺寸双臂家具组装的长时程任务系统研究。

🛠 **方法**：提出FurnitureVLA框架，基于视觉-语言-动作模型（VLA），通过可扩展仿真管线生成专家数据，并利用VR遥操作系统收集真实双臂演示；引入进度增强机制，联合预测动作与连续进度信号，实现子任务自动切换并减少累积误差。

✨ **主要贡献**：首次系统研究真实尺寸双臂家具组装，在仿真中平均成功率从48%提升至80%，设计因素研究额外带来21%增益；在真实Kinova Gen3平台上验证，最难任务成功率仅下降16%。
