# Current World Models Lack a Persistent State Core

- arXiv: https://arxiv.org/abs/2606.20545v1
- arXiv ID: 2606.20545

---

🔍 **核心问题**：现有世界模型缺乏持续演化的内部状态核心，当观测中断（如相机移动）时，无法保持物理世界的连续演化，导致物体和事件在未被观察时停滞。

🛠 **方法**：提出 **WRBench** 诊断基准，将相机运动视为对可观测性的干预，通过人类校准的链条评估模型在观测中断后场景的连续性、目标一致性及事件推进能力，测试了23个模型共9600个视频。

✨ **主要贡献**：发现当前所有模型在观测中断后仅恢复目标被遗弃时的状态，而非推进事件演化；证明世界状态演化无法通过更清晰的图像、更精确的控制或更大的参数规模实现，需将物理状态内核的稳定性作为世界模型设计的首要目标。
