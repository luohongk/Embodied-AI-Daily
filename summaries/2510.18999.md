# OREN: Octree Residual Network for Real-Time Euclidean Signed Distance Mapping

- arXiv: https://arxiv.org/abs/2510.18999v3
- arXiv ID: 2510.18999

---

🔍 **核心问题**：从点云数据实时重建欧几里得符号距离函数（ESDF）时，现有离散体积方法缺乏连续性和可微性，而纯神经网络方法存在效率低、遗忘问题和内存限制，且通常局限于截断SDF。

🛠 **方法**：提出OREN混合方法，结合八叉树插值的显式先验与神经网络回归的隐式残差，实现非截断的欧几里得SDF重建。

✨ **主要贡献**：在计算和内存效率上媲美体积方法，同时具备神经网络方法的可微性和精度；实验表明OREN在准确性和效率上均超越现有最优方法，为机器人自主任务提供可扩展解决方案。
