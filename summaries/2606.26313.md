# Racing a Wheeled Quadruped: Active Load Transfer Mitigation via Model Predictive Control

- arXiv: https://arxiv.org/abs/2606.26313v1
- arXiv ID: 2606.26313

---

🔍 **核心问题**：解决轮式四足机器人在自主竞速中因横向载荷转移导致的侧翻风险与速度受限问题。

🛠 **方法**：提出分层控制框架，结合模型预测控制（MPC）与强化学习（RL），通过MPC规划器主动最小化横向载荷转移比（LTR），并利用腿部关节作为主动悬架产生抗侧倾力矩，底层由RL策略直接驱动16个执行器。

✨ **主要贡献**：在物理赛道实验中，主动侧倾控制使平均LTR降低44%，最快圈速提升8.7%，峰值侧向加速度提升21.3%至1.98 m/s²，并实现超越非倾斜基准控制器的高速稳定性。
