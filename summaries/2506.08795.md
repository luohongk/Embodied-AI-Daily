# Towards Biosignals-Free Autonomous Prosthetic Hand Control via Imitation Learning

- arXiv: https://arxiv.org/abs/2506.08795v2
- arXiv ID: 2506.08795

---

🔍 **核心问题**：传统肌电假肢控制需要用户持续产生肌电信号，操作负担重，本文旨在开发一种无需生物信号的自主假手控制系统，仅通过腕部摄像头自动完成抓取与释放动作。

🛠 **方法**：构建遥操作数据采集系统，收集人类演示数据，利用模仿学习训练假手控制模型，使模型能根据手部运动和环境自动调整抓取力度，并在物体靠近桌面时自动释放。

✨ **主要贡献**：仅基于单名参与者对有限物体的演示数据训练模型，即可实现高成功率抓取，并有效泛化至新用户和未见过的不同重量物体，显著降低用户操作难度与心理负担。
