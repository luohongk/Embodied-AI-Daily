# HEM: a margin-based loss for visual categorisation tasks

- arXiv: https://arxiv.org/abs/2501.12191v2
- arXiv ID: 2501.12191

---

🔍 **核心问题**：解决深度神经网络在分类任务中广泛使用的交叉熵损失函数在训练动态和特定视觉任务（如未知类拒绝、对抗鲁棒性、不平衡学习、持续学习和语义分割）上的局限性。

🛠 **方法**：提出一种基于边界的损失函数——高错误边际损失（HEM），通过修改损失设计来改善神经网络的训练动态，并在多种架构和基准数据集上使用文献中的标准超参数进行广泛评估。

✨ **主要贡献**：HEM在未知类拒绝、对抗鲁棒性、不平衡数据学习、持续学习和语义分割等任务上优于交叉熵损失，尤其在持续学习和语义分割上表现显著；与专用损失（如LogitNorm、Logit-adjusted loss、DICE）相比，HEM在多个任务上具有竞争力或更优性能，仅在平衡数据下的干净/损坏图像分类中略逊于交叉熵。
