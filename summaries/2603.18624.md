# REST: Receding Horizon Explorative Steiner Tree for Zero-Shot Object-Goal Navigation

- arXiv: https://arxiv.org/abs/2603.18624v2
- arXiv ID: 2603.18624

---

🔍 **核心问题**：零样本目标导航（ZSON）中，现有方法将子目标候选视为单一航点，忽略了路径上的信息增益和候选间的结构关系，导致决策效率低下。

🛠 **方法**：提出REST框架，通过在线RGB-D流构建开放词汇3D地图，基于采样规划生成以智能体为中心的路径树作为选项空间，并将每条分支文本化为空间叙事，利用链式思维大语言模型推理选择最优路径。

✨ **主要贡献**：首次将选项空间设计为路径树，通过共享分支实现从粗到细的LLM推理，显著提升导航效率；在Gibson、HM3D和HSSD基准上，成功率与路径效率均达到顶尖水平。
