# Task-Differentiated Atomic Skill Expansion and Routing for Continual Learning Across Highly Heterogeneous Tasks

- arXiv: https://arxiv.org/abs/2606.21307v1
- arXiv ID: 2606.21307

---

🔍 **核心问题**：针对高度异构任务（任务间推理模式和输入输出格式差异大）的持续学习场景，现有方法存在灾难性遗忘和容量分配低效的问题。

🛠 **方法**：提出TASER框架，包含三个关键技术：基于任务差异和模型不确定性的原子技能增量学习（动态扩展容量）、正交性增强的技能检测（保持技能语义独立）、以及轻量级任务条件门控的技能动态路由机制（组合任务相关技能）。

✨ **主要贡献**：构建了包含19个任务、9个认知维度的异构持续学习基准HeteroCLBench，实验表明TASER在提升可塑性和减少灾难性遗忘方面显著优于强基线方法。
