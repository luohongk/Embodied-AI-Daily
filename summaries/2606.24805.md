# DDStereo: Efficient Dual Decoder Transformers for Stereo 3D Road Anomaly Detection

- arXiv: https://arxiv.org/abs/2606.24805v2
- arXiv ID: 2606.24805

---

🔍 **核心问题**：解决自动驾驶中立体视觉3D障碍物感知在部署成本、检测精度和开放集适应性三者之间的不平衡问题，特别是对任意道路异常（OoD）的实时检测需求。

🛠 **方法**：提出DDStereo，一种基于立体视差几何先验的双解码器Transformer架构，通过解耦的编码策略和共享目标级查询，在轻量级双分支解码器中协同实现3D目标回归与开放集前景检测。

✨ **主要贡献**：在开放集和封闭集协议下均达到最先进精度，同时实现与单目3D检测基线相当的实时性能，为正常和异常类别障碍物感知提供了低成本高效方案。
