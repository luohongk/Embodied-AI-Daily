# KnowAct-GUIClaw: Know Deeply, Act Perfectly, Personal GUI Assistant with Self-Evolving Memory and Skill

- arXiv: https://arxiv.org/abs/2607.12625v1
- arXiv ID: 2607.12625

---

🔍 **核心问题**：OpenClaw框架在跨平台GUI交互支持不足，且缺乏有效的自我进化机制，导致其难以适应多样化设备生态，也无法通过持续学习提升性能。

🛠 **方法**：提出KnowAct-GUIClaw框架，基于“Know-Route-Act-Reflect”范式，通过主机代理进行长程任务分解与分配，并引入可插拔GUI子代理，结合经验归因记忆系统和自进化技能库，实现跨平台无缝迁移与快速路径集成。

✨ **主要贡献**：在Android、iOS、HarmonyOS和Windows平台上，KnowAct-GUIClaw在效率、准确性和跨平台适应性上表现优异；基于开源Kimi-2.6模型的GUIClaw在MobileWorld基准上达到64.1%的最佳性能，超越所有闭源模型；其知识记忆与执行技能可跨模型迁移，在Kimi-2.6上提升8.5%。
