# Residual-Conservative Model Predictive Path Integral Control

- arXiv: https://arxiv.org/abs/2607.06950v1
- arXiv ID: 2607.06950

---

🔍 **核心问题**：传统采样型模型预测控制（MPPI）在模型与实际系统不匹配时，使用固定约束惩罚无法自适应调整安全性，导致控制效果下降。

🛠 **方法**：提出残差保守型MPPI（RC-MPPI），通过在线利用预测-执行残差动态调节三个耦合机制：残差依赖的约束收紧、自适应安全-代价整形，以及通过探索收缩和温度松弛实现的残差自适应采样调制。

✨ **主要贡献**：理论推导了在Lipschitz动力学和次高斯扰动下约束违反的概率界，证明自适应机制随残差增大可降低违反概率；在LTI点质量和平面2R机械臂仿真中，相比原始MPPI显著提升了安全裕度、成功率和控制效率。
