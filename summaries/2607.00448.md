# Real-Time Hard Negative Sampling via LLM-based Clustering for Large-Scale Two-Tower Retrieval

- arXiv: https://arxiv.org/abs/2607.00448v2
- arXiv ID: 2607.00448

---

🔍 **核心问题**：大规模双塔推荐模型在训练时常用的负采样方法（如批内采样）容易产生简单负样本，无法有效提升模型的区分能力，导致推荐效果受限。

🛠 **方法**：提出一种基于大语言模型（LLM）的实时硬负采样技术，在训练过程中利用LLM对媒体表示进行聚类，从同一簇中生成更具挑战性的硬负样本，并设计为可无缝集成到生产系统的低计算复杂度框架。

✨ **主要贡献**：在公开数据集和大型在线系统中验证了该方法优于行业常用负采样技术，并能打破推荐中的反馈循环、显著降低流行度偏差。
