# Nowcasting PM2.5 in Beijing Using Synchronous Covariates and Lagged Features: Model Comparison and Variable Selection Stability

- arXiv: https://arxiv.org/abs/2607.07279v1
- arXiv ID: 2607.07279

---

🔍 **核心问题**：如何利用同步协变量和滞后特征实现北京PM2.5浓度的可靠临近预报，并比较不同模型在变量选择稳定性上的表现。

🛠 **方法**：对比了正则化回归方法（Ridge、Lasso、Elastic Net）与深度学习模型（MLP和LSTM），基于北京12个监测站2013-2017年的逐时数据，采用时间序列交叉验证和特征分组消融分析。

✨ **主要贡献**：MLP模型表现最优（RMSE=13.651，R²=0.972），比正则化回归模型RMSE降低约13%；发现CO、NO₂、PM10和PM2.5一阶滞后项构成稳健核心变量集，并揭示了模型误差的空间与季节异质性。
