# LLM-as-a-Verifier: A General-Purpose Verification Framework

- arXiv: https://arxiv.org/abs/2607.05391v1
- arXiv ID: 2607.05391

---

🔍 **核心问题**：现有大型语言模型（LLM）在智能体任务中缺乏通用的验证能力，难以精确判断解决方案的正确性并给出细粒度反馈。

🛠 **方法**：提出LLM-as-a-Verifier框架，通过计算评分令牌logits分布的期望值生成连续分数，替代传统的离散评分方式，并支持评分粒度、重复评估和标准分解三个维度的扩展。

✨ **主要贡献**：在Terminal-Bench V2等四个基准上达到最优性能（最高86.5%），其细粒度信号还可用于估计任务进度、改进强化学习样本效率，并已扩展为Claude Code的监控工具。
