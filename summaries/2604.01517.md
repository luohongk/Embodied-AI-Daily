# MorphoGuard: A Morphology-Based Whole-Body Interactive Motion Controller

- arXiv: https://arxiv.org/abs/2604.01517v1
- arXiv ID: 2604.01517

---

🔍 **核心问题**：机器人需要沿单一运动链处理动态多接触组合（如用肘部推门同时抓取物体）时，面临复杂接触表示和关节配置耦合的挑战。

🛠 **方法**：提出一种基于形态约束的全身控制网络（MorphoGuard），在自建双臂物理与仿真平台上训练，显式管理任意接触组合，并通过模型推荐实验优化骨干架构、融合策略和模型规模。

✨ **主要贡献**：在多目标交互任务中，该方法实现了约1厘米的接触点管理误差，验证了其在全身交互控制中的有效性。
