# Packet-Loss Robust 3D Gaussian Compression via Atomic Packaging and GNN-based Error Concealment

- arXiv: https://arxiv.org/abs/2607.17916v1
- arXiv ID: 2607.17916

---

🔍 **核心问题**：3D高斯泼溅（3DGS）在流传输中因网络丢包导致锚点属性损坏，引发渲染伪影，现有压缩方法缺乏鲁棒性。

🛠 **方法**：编码端采用锚点级原子封装，将每个锚点的所有属性联合打包，将属性损坏转化为锚点缺失；通过分层随机分组分散丢包空间分布。解码端提出上下文感知残差插值（CARI）分支，结合哈希网格先验与邻域残差构建基线，并用轻量级图神经网络（GNN）交叉注意力机制细化高频残差，同时引入属性置信度控制以在不可靠时回退插值。

✨ **主要贡献**：在20%随机丢包下，相比无错误隐藏方案显著提升渲染质量，平均PSNR退化仅约3 dB（相对于无损HAC++），在BungeeNeRF、Mip-NeRF 360和Tanks and Temples数据集上验证了有效性。
