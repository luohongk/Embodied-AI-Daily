# Gastroendoscopy View Synthesis: A New Real Dataset and Evaluation

- arXiv: https://arxiv.org/abs/2606.25427v1
- arXiv ID: 2606.25427

---

🔍 **核心问题**：现有数据集不足以评估神经辐射场（NeRF）和3D高斯泼溅（3DGS）等新视角合成方法在胃镜图像中的表现，限制了其在扩展视野、数字孪生等应用中的发展。  
🛠 **方法**：构建了首个真实胃镜新视角合成数据集GastroNVS，包含胃镜图像、相机位姿和点云数据，并基于多种3DGS方法进行性能评估。  
✨ **主要贡献**：提供了首个真实胃镜NVS基准数据集，揭示了当前方法在胃镜场景中的挑战，为未来算法优化和临床应用奠定基础。
