# Can LLMs Prove Robotic Path Planning Optimality? A Benchmark for Research-Level Algorithm Verification

- arXiv: https://arxiv.org/abs/2603.19464v2
- arXiv ID: 2603.19464

---

🔍 **核心问题**：评估大语言模型（LLMs）在机器人路径规划算法的最优性证明（近似比证明）中的能力，填补该领域缺乏标准化基准的空白。  
🛠 **方法**：构建包含34个研究级证明任务的基准，涵盖多种规划问题类型与复杂度，并通过上下文引理增强、思维链提示等策略测试LLMs的推理表现。  
✨ **主要贡献**：首次提出针对机器人路径规划最优性证明的LLM评估基准，发现当前最强模型仍需外部领域知识辅助，且任务特定上下文引理比通用提示更有效提升推理质量，并细粒度分析了错误类型及缓解策略。
