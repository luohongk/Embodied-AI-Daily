# Toward Anthropomorphic Dialogue: A Closed-Loop Framework for Human-Like Chat Generation, Evaluation, and Preference Alignment

- arXiv: https://arxiv.org/abs/2607.17191v1
- arXiv ID: 2607.17191

---

🔍 **核心问题**：如何让对话系统在私密聊天中不仅生成流畅回复，还能保持角色个性、关系、记忆、知识边界、时间感知和连贯的多轮对话结构。

🛠 **方法**：提出AnthroDial闭环框架，包含角色条件化调度对话运行时、可执行基准测试（含L0有效性门控及多维度评估）、以及基于认知诊断和最近发展区（ZPD）的奖励对齐训练（结合SFT和GRPO）。

✨ **主要贡献**：在55个角色、50个场景的基准测试中，最强非训练基线严格准确率仅32.00%，而Qwen3.6-27B-SFT+RL达到39.00%和98.5总分；在9B无思考设置下，SFT和RL将严格准确率从0.00%提升至18.37%，证明生成、评估与奖励共享相同行为维度可显著提升拟人化对话效果。
