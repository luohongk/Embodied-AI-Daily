# FlowAWR: Online Adaptive Flow Reinforcement via Advantage-Weighted Rectification

- arXiv: https://arxiv.org/abs/2606.30376v1
- arXiv ID: 2606.30376

---

🔍 **核心问题**：在线强化学习对齐连续空间上的生成流模型时，面临轨迹似然难以计算、训练与推理不一致以及依赖无分类器指导（CFG）的问题。

🛠 **方法**：提出FlowAWR，将连续生成策略优化转化为对理论最优速度场的监督回归，通过优势加权修正形式推导出最优速度场，无需随机微分方程（SDE）采样和CFG。

✨ **主要贡献**：实现SDE-free优化和CFG-free生成，在SD3.5-Medium上相比DiffusionNFT，对齐性能提升且收敛速度加快2-5倍（如1.2k步达24.12 PickScore），并在多奖励约束下保持生成质量和稳定域外性能。
