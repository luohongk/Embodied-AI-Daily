# DYNA : Dynamic Episodic Memory Networks for Augmenting Large Language Models with Temporal Knowledge Graphs in Continuous Learning

- arXiv: https://arxiv.org/abs/2606.15778v1
- arXiv ID: 2606.15778

---

🔍 **核心问题**：大型语言模型（LLM）在持续学习中难以融入新知识，容易发生灾难性遗忘，且重新训练成本高昂。

🛠 **方法**：提出DYNA框架，通过一个外部可更新的时间知识图谱（以事件为节点、带时间戳的有向边为关系）作为动态情景记忆，在查询时利用随机游走和中心性度量检索相关节点，增强冻结的LLM的响应。

✨ **主要贡献**：相比微调，DYNA将灾难性遗忘降低约7%；相比标准RAG，时间顺序推理提升约5%；揭示了图聚类系数与检索性能的正相关性，并实现了无需重训练的LLM持续知识增强。
