# ELASTIC: Efficiently Learning to Adaptively Scale Test-Time Compute for Generative Control Policies

- arXiv: https://arxiv.org/abs/2606.31132v1
- arXiv ID: 2606.31132

---

🔍 **核心问题**：生成式控制策略（如扩散策略）在机器人控制中面临测试时计算资源分配难题，即如何根据状态、任务和策略动态调整顺序缩放（细化动作）与并行缩放（探索多模态）的计算比例，以在提升任务成功率的同时最小化计算开销。

🛠 **方法**：提出ELASTIC算法，将计算分配建模为元马尔可夫决策过程，通过强化学习训练一个元策略，该策略与冻结的预训练机器人策略交互，在每个去噪迭代中自适应选择顺序步骤数和并行样本数，无需访问原策略的训练数据。

✨ **主要贡献**：在模拟操作基准测试中，ELASTIC在相同计算预算下帕累托优于固定或单轴缩放基线；在真实机器人操作任务中（使用π₀.₅视觉-语言-动作模型），ELASTIC在达到最佳10次采样成功率的同时，将实际延迟降低了34%。
