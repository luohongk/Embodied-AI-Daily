# UBP2: Uncertainty-Balanced Preference Planning for Efficient Preference-based Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.19328v1
- arXiv ID: 2606.19328

---

🔍 **核心问题**：解决基于偏好的强化学习中，因被动数据收集导致的样本效率低下问题，尤其在学习的早期阶段。  
🛠 **方法**：提出不确定性平衡偏好规划（UBP2），通过集成奖励、动力学和值函数模型，在规划时联合考虑期望奖励、终端值和认知不确定性，主动引导探索。  
✨ **主要贡献**：在标准假设下证明了有限时域和无限时域设置的亚线性遗憾界；在Meta-World基准上，UBP2的样本效率显著优于无模型偏好方法和非乐观模型基线。
