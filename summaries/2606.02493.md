# Not What, But How: A Framework for Auditing LLM Responses across Positioning, Generalization, Anthropomorphism, and Maxims

- arXiv: https://arxiv.org/abs/2606.02493v3
- arXiv ID: 2606.02493

---

🔍 **核心问题**：现有LLM评估主要关注事实正确性，忽略了主观信息查询中回答的沟通方式（如文化立场、语言风格等）对用户的影响。

🛠 **方法**：提出FRANZ框架，从文化定位、泛化语言使用、拟人化线索、对话准则遵循四个维度自动审计LLM回答；并构建SQUARE语料库（含37.6万条来自57个子论坛的主观问题，映射至7国19类）。

✨ **主要贡献**：发现LLM在回答中显著差异地使用不同沟通特征，且内部定位与拟人化呈正相关（耦合程度因国家而异），为诊断回答框架偏差提供了新视角。
