# 1000 Rallies: An Event-Camera Dataset and Real-Time Learned Ball-State Estimation for Robotic Table Tennis

- arXiv: https://arxiv.org/abs/2606.25620v1
- arXiv ID: 2606.25620

---

🔍 **核心问题**：在机器人乒乓球中，传统帧相机难以兼顾高帧率与低延迟，导致对快速运动球的实时状态估计存在盲点和计算负担。

🛠 **方法**：构建了首个大规模事件相机乒乓球数据集（含1000多次对打），并训练卷积神经网络从事件流中联合估计球在图像平面的位置和速度，再将预测速度作为卡尔曼滤波的额外测量值。

✨ **主要贡献**：数据集覆盖从业余到精英的多样化球员，事件驱动方法将弹跳点预测误差降低36%，并首次实现基于事件感知的实时人机乒乓球对打。
