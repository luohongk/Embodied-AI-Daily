# VT-WAM: Visual-Tactile World Action Model for Contact-Rich Manipulation

- arXiv: https://arxiv.org/abs/2607.02503v1
- arXiv ID: 2607.02503

---

🔍 **核心问题**：接触丰富的操作任务需要策略对局部变形、压力、滑移和摩擦等触觉线索做出反应，但这些线索在时间上稀疏且视觉观测中常不可见，现有视觉-触觉策略很少在动作生成过程中建模触觉变形动态。

🛠 **方法**：提出VT-WAM，一个视觉-触觉世界动作模型，在统一的流匹配框架中联合学习未来视觉预测、触觉变形预测和动作预测，采用非对称混合变换器注意力机制连接首帧视觉锚点与时间触觉动态，并通过接触门控动作-视觉-触觉注意力引导在接触阶段鼓励动作查询依赖触觉证据。

✨ **主要贡献**：在六个真实世界的接触丰富操作任务中，VT-WAM实现了71.67%的平均成功率，比Fast-WAM高出26.67%，比OmniVTLA高出35.84%；消融实验表明建模触觉变形动态和引导接触阶段触觉注意力对接触丰富任务均至关重要。
