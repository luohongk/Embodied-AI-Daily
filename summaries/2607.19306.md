# From Distances to Trajectories: Real-Time Signed Distance Function Mapping and Distance-Accelerated Motion Planning for UAVs

- arXiv: https://arxiv.org/abs/2607.19306v1
- arXiv ID: 2607.19306

---

🔍 **核心问题**：在杂乱环境中实现无人机自主飞行时，如何将地图构建与运动规划协同设计，以替代传统分离的二进制占据图方法，提升实时性与安全性。

🛠 **方法**：提出OREN（八叉树残差网络）在线重建符号距离函数（SDF），结合显式八叉树先验与隐式神经残差，兼顾效率与精度；同时开发Bubble$^\star$搜索规划器，利用距离信息生成最大无碰撞球体（bubbles），构建安全走廊进行轨迹优化。

✨ **主要贡献**：OREN将SDF估计精度提升22%；Bubble$^\star$在杂乱环境中1-3秒内规划约90米轨迹，而基线方法需10秒；集成系统在机载计算受限条件下实现实时室内导航。
