# Skill-Guided Continuation Distillation for GUI Agents

- arXiv: https://arxiv.org/abs/2606.18890v1
- arXiv ID: 2606.18890

---

🔍 **核心问题**：GUI智能体在闭环执行中因策略偏离专家轨迹，遇到未见过状态时缺乏有效监督，导致动作选择错误。

🛠 **方法**：提出Skill-Guided Continuation Distillation（SGCD）框架，先让基础策略无技能引导地运行几步到达真实偏离状态，再由技能引导策略完成后续任务并生成成功延续轨迹，与专家轨迹混合提供监督；技能从成功和失败轨迹中提取，包括延续计划、关键目标、失败陷阱和成功标准。

✨ **主要贡献**：在OSWorld-Verified上，SGCD将三种基础模型的成功率从30%左右提升至超过50%，验证了其有效性和通用性。
