# GASim: A Graph-Accelerated Hybrid Framework for Social Simulation

- arXiv: https://arxiv.org/abs/2605.07692v1
- arXiv ID: 2605.07692

---

🔍 **核心问题**：大规模社会模拟中，混合框架（结合LLM智能体与数值ABM）存在高延迟问题，主要源于昂贵的记忆检索和顺序化ABM执行。

🛠 **方法**：提出GASim框架，包含图优化记忆（GOM）用轻量图传播替代LLM检索，图消息传递（GMP）通过特征聚合和图注意力网络实现并行ABM更新，以及熵驱动分组（EDG）动态识别核心智能体。

✨ **主要贡献**：相比传统混合框架实现9.94倍端到端加速，令牌消耗降低至基线20%以下，同时保持与真实舆论趋势的高度一致性。
