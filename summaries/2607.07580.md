# Automatic Echocardiography Segmentation via Transition Probability Correlation for Stable Semantic Extraction

- arXiv: https://arxiv.org/abs/2607.07580v1
- arXiv ID: 2607.07580

---

🔍 **核心问题**：超声心动图因散斑噪声和低信噪比导致语义特征模糊、边界不连续，影响深度学习模型在复杂临床病例中的分割精度。

🛠 **方法**：设计了STLSF模块，包含基于窗口匹配的语义校正和语义引导的纹理增强，利用局部转移概率相关性校正语义；同时提出频率感知去噪预训练方法，帮助编码器适应超声成像先验。

✨ **主要贡献**：构建了兼具局部归纳偏置和长程依赖的卷积网络，在CAMUS和EchoNet-Dynamic数据集上分别达到93.87%和92.62%的Dice系数，HD95值分别为3.29mm和2.73mm，实现当前最优性能。
