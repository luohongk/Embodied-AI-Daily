# InSight: Self-Guided Skill Acquisition via Steerable VLAs

- arXiv: https://arxiv.org/abs/2606.24884v1
- arXiv ID: 2606.24884

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型的能力受限于训练数据中的技能，无法自主获取新技能来完成未见过的任务。

🛠 **方法**：提出InSight框架，通过两个阶段实现自主技能获取：（1）自动化分割流程，利用VLM规划分解和末端执行器位姿将演示数据分割为带标签的基本动作，使VLA具备基本动作可操控性；（2）VLM引导的数据飞轮，自动识别缺失的基本动作，通过VLM提出的底层控制自主尝试演示，并将成功的演示自动标注、存储并整合到VLA训练集中。

✨ **主要贡献**：在仿真和真实世界的多种操作任务（如翻转方块、关闭抽屉、清扫、扭转、倒水）中，无需任何人类演示即可学习新技能，且学到的基本动作可组合执行未见过的长时任务，证明了基本动作可操控性为VLA策略的持续技能获取提供了实用基础。
