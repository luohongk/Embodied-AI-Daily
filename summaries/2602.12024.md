# Adaptive-Horizon Conflict-Based Search for Closed-Loop Multi-Agent Path Finding

- arXiv: https://arxiv.org/abs/2602.12024v3
- arXiv ID: 2602.12024

---

🔍 **核心问题**：现有多智能体路径规划方法要么是开环规划，存在较大计算延迟，要么是闭环启发式方法，缺乏可靠性能保证，难以用于安全关键场景。

🛠 **方法**：提出一种基于有限时域冲突搜索的闭环算法ACCBS，借鉴模型预测控制中的迭代时域加深思想，动态调整规划时域以适应计算预算，并复用单一约束树实现时域无缝切换。

✨ **主要贡献**：ACCBS能快速生成高质量可行解，随预算增加渐近最优，具备任意时间行为；在计算效率、解质量和执行灵活性间取得良好平衡，并通过闭环形式自然适应在线扰动。
