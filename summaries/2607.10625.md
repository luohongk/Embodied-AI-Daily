# Dual-Process Atomic Skill Learning: Decoupling Semantic Reasoning and Real-Time Control

- arXiv: https://arxiv.org/abs/2607.10625v1
- arXiv ID: 2607.10625

---

🔍 **核心问题**：语言条件模仿学习在多步骤组合任务中泛化能力不足，现有分层方法因高层技能推理与低层动作生成紧密耦合，导致训练不稳定和码本崩溃。

🛠 **方法**：提出双过程原子技能学习框架（DASL），通过异步解耦慢速语义推理（低频策略，使用向量量化预测离散技能）与快速实时控制（高频策略，结合潜在扩散模型与决策Transformer生成动作），并利用扩散模型结构化潜在空间。

✨ **主要贡献**：缓解了联合训练中技能码本干扰问题，在仿真和实验中显著优于现有基线，在技能获取和对未见指令的组合泛化上表现突出。
