# PRISM: Personalized Robotic Dataset Generation via Image-based Scene and Motion Synthesis

- arXiv: https://arxiv.org/abs/2607.04880v1
- arXiv ID: 2607.04880

---

🔍 **核心问题**：机器人策略在用户特定环境中部署时泛化能力有限，需要针对目标环境收集定制数据集，但现有方法（如遥操作或仿真）难以同时兼顾数据质量、可扩展性和环境适配性。

🛠 **方法**：提出PRISM端到端流水线，从单张图像和自然语言指令出发，构建与用户环境语义和几何对齐但实例多样化的“数字孪生场景”，并自动合成可执行的演示轨迹，无需人工遥操作。

✨ **主要贡献**：在LIBERO和LIBERO-Plus基准上，基于PRISM数据训练的策略优于基线方法；在三个真实世界操作任务中成功率高达100%，且在训练时未见环境中仍保持更强性能。
