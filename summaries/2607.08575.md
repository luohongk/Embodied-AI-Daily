# FabriVLA: A Lightweight Vision-Language-Action Model for Precise Multi-Task Manipulation

- arXiv: https://arxiv.org/abs/2607.08575v1
- arXiv ID: 2607.08575

---

🔍 **核心问题**：解决多任务机器人操作中，现有视觉-语言-动作模型（VLA）依赖数十亿参数大模型、计算成本高的问题，实现轻量化精准操控。

🛠 **方法**：提出FabriVLA模型，采用1B参数的InternVL3.5视觉-语言骨干网络，结合流匹配动作头（含门控自注意力机制与浅层VLM特征融合），通过单阶段联合优化训练。

✨ **主要贡献**：在Meta-World MT50基准的50个任务上达到90.0%的平均成功率，证明紧凑型VLA（1B参数）无需依赖百亿级参数模型即可实现强性能。
