# Single-Frame Point-Pixel Registration via Supervised Cross-Modal Feature Matching

- arXiv: https://arxiv.org/abs/2506.22784v2
- arXiv ID: 2506.22784

---

🔍 **核心问题**：解决单帧LiDAR点云与相机图像之间因模态差异（稀疏点云 vs 密集图像）导致的点-像素匹配困难问题。

🛠 **方法**：提出基于探测器无关匹配框架的投影式方法，直接进行点云与图像的跨模态特征匹配，并引入可重复性评分机制作为软可见性先验，抑制低强度变化区域的不可靠匹配。

✨ **主要贡献**：在KITTI、nuScenes和MIAS-LCEC-TF70基准上达到最先进性能，尤其在nuScenes上超越依赖累积点云的先前方法，且仅使用单帧LiDAR输入。
