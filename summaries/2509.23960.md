# MAD-PINN: A Decentralized Physics-Informed Machine Learning Framework for Safe and Optimal Multi-Agent Control

- arXiv: https://arxiv.org/abs/2509.23960v2
- arXiv ID: 2509.23960

---

🔍 **核心问题**：在大规模多智能体系统中，如何同时保证安全性与性能优化，并克服现有方法（如多智能体强化学习、安全滤波或模型预测控制）在严格安全保证、保守性或可扩展性上的不足。

🛠 **方法**：提出MAD-PINN框架，利用epigraph重构将安全与性能联合建模，并通过物理信息神经网络近似求解；采用降阶系统训练值函数并分布式部署，结合基于HJ可达性的邻居选择策略和滚动时域执行方案。

✨ **主要贡献**：实现了安全与性能的优越权衡，在智能体数量增加时保持可扩展性，并在多智能体导航任务中持续优于现有基线方法。
