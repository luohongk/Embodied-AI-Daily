# AUSLUN: A Fixed-Hover UAV--USV System for GNSS-Denied Maritime Search and Navigation

- arXiv: https://arxiv.org/abs/2606.29875v1
- arXiv ID: 2606.29875

---

🔍 **核心问题**：在GNSS信号缺失的沿海环境中，无人水面艇（USV）无法远距离发现目标并保持全局参考导航。

🛠 **方法**：提出AUSLUN系统，利用固定悬停的无人机（UAV）通过视觉惯性里程计（VIO）自定位，作为远程感知与导航锚点；采用多边形感知环形吊舱扫描、模态感知方位-距离定位、目标相对USV引导及视觉丢失恢复三个耦合模块，并通过门控递归估计器融合激光测距（非合作目标）与数据链测距（合作USV）。

✨ **主要贡献**：自适应偏航边界扫描相比固定扇区扫描减少时间与冗余覆盖；门控递归估计器在定位精度上优于非递归基线方法；完整验证了从搜索到导航的流程，包括故意触发的视觉丢失恢复，证明了固定悬停无人机辅助USV在GNSS缺失环境中接近静止目标的可行性与操作边界。
