# Concept-as-Tree: A Controllable Synthetic Data Framework Makes Stronger Personalized VLMs

- arXiv: https://arxiv.org/abs/2503.12999v4
- arXiv ID: 2503.12999

---

🔍 **核心问题**：现有视觉语言模型（VLM）在个性化任务中面临用户提供的正样本稀缺和检索到的负样本质量低下的挑战，导致模型性能受限。  
🛠 **方法**：提出Concept-as-Tree（CaT）框架，将概念表示为树结构，从而可控地生成不同难度和多样性的正负样本，并配合数据过滤策略确保生成数据质量，支持多概念场景扩展。  
✨ **主要贡献**：首次提出用于VLM个性化的可控合成数据流水线，系统分析了样本数量与多样性对模型的影响，并通过实验证明CaT能显著提升VLM在个性化基准上的性能。
