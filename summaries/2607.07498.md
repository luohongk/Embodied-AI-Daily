# Reward-Adaptive Iterative Discovery: A Case Study on Automated Game Testing for NHL26

- arXiv: https://arxiv.org/abs/2607.07498v1
- arXiv ID: 2607.07498

---

🔍 **核心问题**：在游戏开发阶段，人工测试员需要反复测试冰球游戏《NHL 26》中守门员AI的行为漏洞，耗时耗力，亟需自动化方法减少重复测试工作。

🛠 **方法**：提出奖励自适应迭代发现（RAID）方法，基于迭代强化学习训练一群进球智能体，并在现有强化学习算法上引入简单扩展，使其能发现多种多样且高质量的漏洞策略。

✨ **主要贡献**：在单次实验中成功找到六种与人工测试员数小时手动测试结果定性相似的进球漏洞策略，显著降低了守门员AI重测的人力成本。
