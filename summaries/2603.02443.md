# Safe Whole-Body Loco-Manipulation via Combined Model and Learning-based Control

- arXiv: https://arxiv.org/abs/2603.02443v1
- arXiv ID: 2603.02443

---

🔍 **核心问题**：解决四足机器人在全身移动操作中，协调腿部运动与机械臂操作时面临的接触交互安全性与柔顺性问题。

🛠 **方法**：提出一种结合模型与学习的全身控制器，对机械臂采用基于模型的导纳控制，对腿部运动采用强化学习策略，并通过参考调节器保障安全，利用神经网络增强的卡尔曼滤波器提升鲁棒性。

✨ **主要贡献**：在Unitree Go2四足机器人上实现了交互驱动速度的精确跟踪、柔顺行为以及动态环境下的安全可靠性能，验证了混合控制方法的有效性。
