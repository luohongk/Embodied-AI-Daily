# ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.17011v1
- arXiv ID: 2606.17011

---

🔍 **核心问题**：如何利用人类干预信号对仿人机器人视觉-语言-动作（VLA）模型进行后训练，同时解决干预轨迹质量参差不齐（包含犹豫、低效甚至错误行为）的问题。

🛠 **方法**：提出ROVE框架，包含两个关键部分：1）构建人机协作流水线，收集仿人机器人的部署与干预数据；2）采用乐观价值估计（OVE）从混合质量轨迹中筛选高价值行为，并引入跨形态人类经验视频增强对长尾失败与恢复模式的价值估计，从而引导VLA模型聚焦于有效行为而非盲目模仿所有动作。

✨ **主要贡献**：在真实世界的接触密集型与精细操作任务中，ROVE显著优于基于经验学习的基线方法，并在多次迭代中持续提升性能，首次实现了利用不完美人类干预信号有效训练仿人机器人操作模型。
