# RSF-GLLM: Bridging the Semantic Gap in Multi-Hop Knowledge Graph QA via Recurrent Soft-Flow and Decoupled LLM Generation

- arXiv: https://arxiv.org/abs/2607.06527v1
- arXiv ID: 2607.06527

---

🔍 **核心问题**：多跳知识图谱问答中，传统检索-生成流程因不可微分导致检索器无法弥合语义鸿沟，即中间节点与查询缺乏词汇重叠时难以被正确识别。

🛠 **方法**：提出RSF-GLLM框架，包含两个模块：1）循环软流（RSF）模块，利用GRU引导的查询更新器传播连续相关性分数，并通过动态门控机制遍历结构线索；2）解耦的大语言模型（LLM）生成模块，将提取的推理路径文本化后微调LLM，确保生成基于事实拓扑。

✨ **主要贡献**：通过流稀疏正则化从理论上保证软概率收敛到离散推理路径；在WebQSP和CWQ数据集上取得与基于LLM的昂贵方法相当的竞争力，且推理效率显著更高。
