# Visual Place Recognition Using Rate-Encoded Spiking Neural Networks with Discrete STDP Learning

- arXiv: https://arxiv.org/abs/2607.13584v1
- arXiv ID: 2607.13584

---

🔍 **核心问题**：解决基于STDP学习的脉冲神经网络（SNN）在视觉位置识别（VPR）中召回率不足的问题，特别是无法达到自主导航所需的100%精度下的高召回率（R@100P）。

🛠 **方法**：提出一种离散化、张量原生的STDP-SNN-VPR实现，使用PyTorch和snnTorch，在Nordland数据集上训练15个独立网络，并引入确定性张量管道进行神经元分配、状态重置和速度补偿滑动窗口聚合。

✨ **主要贡献**：发现确定性神经元分配和状态重置显著提升R@100P，速度补偿滑动窗口（k=5）实现100%的R@100P，仅增加0.20毫秒延迟，揭示了推理阶段设计决策对召回精度的重要影响。
