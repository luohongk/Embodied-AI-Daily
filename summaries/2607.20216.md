# Small, Free, and Effective: Orchestrating Open-Weight Small Language Models to Outperform Single LLM for Malware Analysis

- arXiv: https://arxiv.org/abs/2607.20216v1
- arXiv ID: 2607.20216

---

🔍 **核心问题**：如何在资源受限场景下，利用开源轻量级小语言模型（SLM）的协作，在恶意软件分析任务中达到或超越单个大型语言模型（LLM）的性能。

🛠 **方法**：设计了四种编排架构，包括多智能体流水线、对抗辩论框架、分层咨询系统及混合架构，并基于Meta的CyberSecEval基准测试，对比了11个SLM、3个网络安全预训练模型和6个前沿LLM的表现。

✨ **主要贡献**：混合架构（Qwen3-4B + Foundation-Sec-8B）以35.30%的准确率超越最强网络安全基线（22.54%）和最强无证据前沿基线（34.77%），证明基于证据的编排能显著提升协作SLM在恶意软件报告解读中的性能。
