# Prompting Diffusion Models for Zero-Shot Instance Segmentation

- arXiv: https://arxiv.org/abs/2606.22660v1
- arXiv ID: 2606.22660

---

🔍 **核心问题**：现有基础模型在零样本实例分割中难以准确分割物体区域，易产生误检和过分割，且早期利用生成先验的方法仅在后期处理中使用提示，导致分割结果次优。

🛠 **方法**：提出Prompt2Seg，一种基于扩散模型的空间条件框架，通过添加条件分支将空间提示（如2D高斯或置信度图）作为显式输入，训练模型直接响应用户意图，并在有限类别数据集上微调后实现零样本泛化。

✨ **主要贡献**：在七个数据集（包括绘画、自我中心视角和X光等挑战性领域）上，Prompt2Seg一致优于原始扩散分割骨干网络，证明生成先验与空间条件结合能有效实现交互式分割的广泛泛化，无需大规模掩码监督。
