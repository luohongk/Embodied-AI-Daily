# 1D-Bench: A Benchmark for Iterative UI Code Generation with Visual Feedback in Real-World

- arXiv: https://arxiv.org/abs/2602.18548v2
- arXiv ID: 2602.18548

---

🔍 **核心问题**：设计稿到前端代码的转换任务缺乏统一、真实的基准测试，现有评估因数据集、工具链和协议不一致而难以比较模型性能。

🛠 **方法**：提出1D-Bench基准，基于真实电商工作流，提供参考渲染图和含潜在错误的中间表示作为输入，要求模型在固定工具链下生成可执行React代码，并支持多轮迭代编辑，利用执行反馈进行组件级修改。

✨ **主要贡献**：构建了首个结合真实缺陷中间表示和多轮视觉反馈的UI代码生成基准，实验表明迭代编辑能提升渲染成功率和视觉相似度，但基于强化学习的后训练效果有限且不稳定。
