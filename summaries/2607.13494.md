# A VAE-Driven Multi-Task Satellite-Aided Semantic Communication Framework for 6G-Enabled Connected Autonomous Vehicles

- arXiv: https://arxiv.org/abs/2607.13494v1
- arXiv ID: 2607.13494

---

🔍 **核心问题**：在6G车联网中，卫星通信带宽有限且传播损耗大，传统通信系统传输原始数据效率低，难以满足自动驾驶任务（如交通标志识别）对带宽、可靠性和低延迟的需求。

🛠 **方法**：提出一种基于变分自编码器（VAE）的多任务语义通信框架，利用概率潜在表示提取任务相关特征，通过端到端训练联合优化交通标志重建与分类任务，并在噪声信道中传输。

✨ **主要贡献**：相比确定性自编码器方法，该框架在信噪比变化下保持稳定性能，同时实现高达87.23%至98.17%的带宽压缩，显著提升卫星辅助自动驾驶的通信效率。
