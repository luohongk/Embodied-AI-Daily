# HERO: Hypothesis-Driven Evidence Retrieval from Omics for Multi-Task Breast Cancer Analysis

- arXiv: https://arxiv.org/abs/2606.21174v1
- arXiv ID: 2606.21174

---

🔍 **核心问题**：现有多组学与病理图像融合方法多将组学数据作为并行特征或文本上下文，缺乏显式的检索约束，导致生物标志物和预后预测性能受限。

🛠 **方法**：提出HERO框架，将DNA甲基化和miRNA通过稀疏通路-形态先验映射为意图向量，利用TF-IDF检索结构化病理描述选择相关区域，并通过余弦门控机制触发确定性缺陷修复，形成闭环验证。

✨ **主要贡献**：在TCGA-BRCA数据集上，HERO在ER、PR、HER2、亚型和风险预测任务中均达到最优性能，超越多模态融合和VLM基线，且每个检索和验证步骤均可词法审计。
