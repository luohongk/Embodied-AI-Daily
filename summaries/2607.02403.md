# ACID: Action Consistency via Inverse Dynamics for Planning with World Models

- arXiv: https://arxiv.org/abs/2607.02403v1
- arXiv ID: 2607.02403

---

🔍 **核心问题**：在基于世界模型的决策时规划中，标准规划成本仅检查预测终端状态与目标的接近程度，忽略了中间过渡的可实现性，导致预测轨迹与环境实际执行可能偏离。

🛠 **方法**：提出ACID框架，引入循环动作一致性——通过逆动力学模型从预测过渡中反向推断的动作应与原始条件动作一致，并将此逐步骤残差通过尺度不变自适应权重融入规划成本。

✨ **主要贡献**：在四种动作条件世界模型和六项任务（刚体/可变形操作、关节控制、视觉导航）中，ACID持续提升规划性能，并在显著减少规划计算量的情况下达到与基线相当的精度。
