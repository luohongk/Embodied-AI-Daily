# Optimizing RAG Rerankers with LLM Feedback via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2604.02091v2
- arXiv ID: 2604.02091

---

🔍 **核心问题**：现有重排序模型通常基于静态人工标注的相关性标签进行优化，与下游生成过程脱节，导致检索结果虽在主题上相关，却无法为LLM提供实际有用的上下文信息。

🛠 **方法**：提出ReRanking Preference Optimization（RRPO），一种基于强化学习的框架，将重排序建模为序列决策过程，利用LLM的生成质量反馈直接优化上下文效用，并引入参考锚定确定性基线以稳定训练。

✨ **主要贡献**：RRPO在知识密集型基准上显著优于强基线（如RankZephyr），且具备良好泛化性（可适配GPT-4o等不同读者）、与查询扩展模块（如Query2Doc）正交集成，并在使用噪声监督时仍保持鲁棒性。
