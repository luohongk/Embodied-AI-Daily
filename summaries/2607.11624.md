# SKooP: Symmetric Koopman Predictions for Faster and More Generalizable Legged Robot Locomotion with Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.11624v1
- arXiv ID: 2607.11624

---

🔍 **核心问题**：强化学习在机器人控制中样本效率低，且现有方法多局限于低维系统，难以处理高维复杂非线性机器人（如四足机器人）的步态学习。

🛠 **方法**：提出SKooP框架，通过自编码器学习系统动力学的Koopman模型，将其预测作为评论家的特权观测，同时在演员、评论家、编码器和解码器中融入形态对称性，构建高度等变的策略。

✨ **主要贡献**：首次将Koopman理论与对称性结合用于机器人强化学习，显著提升样本效率与收敛速度；在四足机器人多种双足步态任务中验证了策略的可迁移性，并实现更高的累积奖励。
