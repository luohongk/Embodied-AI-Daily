# A Camera-Cooperative ISAC Framework for Multimodal Non-Cooperative UAVs Sensing

- arXiv: https://arxiv.org/abs/2605.22090v1
- arXiv ID: 2605.22090

---

🔍 **核心问题**：解决单模态感知局限和通信与感知资源竞争导致的非合作无人机检测难题。  
🛠 **方法**：提出相机协同ISAC框架，利用相机粗粒度空域监控与ISAC细粒度高精度感知互补，并设计视觉-回波数据对齐模型和多模态融合估计模块。  
✨ **主要贡献**：在DeepSense 6G数据集上实现波束转向开销平均降低71%、跟踪开销降低1.69-11.15%，同时保持高角度估计精度，有效缓解资源竞争并释放系统资源用于通信任务。
