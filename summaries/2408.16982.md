# 2DGH: 2D Gaussian-Hermite Splatting for High-quality Rendering and Better Geometry Features

- arXiv: https://arxiv.org/abs/2408.16982v3
- arXiv ID: 2408.16982

---

🔍 **核心问题**：传统2D高斯溅射方法因高斯核缺乏各向异性和形变能力，导致物体边缘渲染模糊、几何重建质量受限。

🛠 **方法**：受量子物理启发，提出用高斯-埃尔米特核替代传统高斯核作为新基元，该核具有统一数学形式，将高斯函数作为零阶特例纳入广义框架。

✨ **主要贡献**：在DTU数据集上实现更精确的几何重建，在MipNeRF360和自定义Detail数据集上获得更优的新视角合成效果，验证了高斯-埃尔米特核在高质量3D重建与渲染中的潜力。
