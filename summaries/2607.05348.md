# Beyond Isolated Objects: Relationship-aware Open Vocabulary Scene Understanding via 3D Scene Graph Analysis

- arXiv: https://arxiv.org/abs/2607.05348v1
- arXiv ID: 2607.05348

---

🔍 **核心问题**：现有开放词汇3D场景理解方法依赖上下文无关的语义特征，忽略了物体间关系对语义细化的作用。

🛠 **方法**：提出RelGraphOV框架，通过3D场景图增强开放词汇理解，利用视觉-语言推理从多视角观测构建关系场景图，并采用自适应门控双流上下文图注意力网络（GAT）分离几何与语义特征，进行边引导消息传递与自适应融合。

✨ **主要贡献**：无需手动关系标注即可推断物体关系并修剪几何不合理连接；引入层次对比学习目标提升实例一致性与类别区分度；在ScanNetV2、ScanNet200等数据集上展现强性能与泛化能力。
