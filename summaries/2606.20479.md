# GroundControl: Anticipating Navigation Failures in Vision-Language Agents via Trajectory-Consistent Uncertainty Estimates

- arXiv: https://arxiv.org/abs/2606.20479v1
- arXiv ID: 2606.20479

---

🔍 **核心问题**：视觉语言导航智能体在基准任务中平均成功率尚可，但常因可预测的轨迹级故障（如振荡、停滞或低效绕路）而失败，现有不确定性信号仅反映瞬时动作熵，无法提前预警这类故障。

🛠 **方法**：提出轨迹一致性不确定性估计器GroundControl，通过恒定速度卡尔曼滤波器建模距离目标的动态变化，并结合归一化创新统计量与轨迹特征（进度、单调性、路径效率、振荡行为）来量化导航行为中的几何与时间不一致性。

✨ **主要贡献**：引入选择性风险-覆盖导航（SRCN）协议独立评估不确定性质量，实验表明GroundControl在五个导航分割上显著优于熵、共形和启发式基线，GPT-4o模型在成功选择性风险下加权平均E-AURC仅0.0024，接近理想排序，为预测导航故障提供了可解释且鲁棒的信号。
