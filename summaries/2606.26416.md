# Methane-Plume Segmentation From Hyperspectral Satellite Imagery Via Multimodal Deep Learning

- arXiv: https://arxiv.org/abs/2606.26416v1
- arXiv ID: 2606.26416

---

🔍 **核心问题**：如何从高光谱卫星图像中高效、准确地分割甲烷羽流，以支持大规模全球变暖监测。

🛠 **方法**：提出一种多模态深度学习模型，集成特征引导的甲烷增强（FGME）机制，在多个语义尺度上将物理意义的甲烷线索注入基于Transformer的RGB表示中。

✨ **主要贡献**：在MPDataset上，该方法以显著低于其他高性能架构的计算成本，实现了MIoU、MPrecision和Recall分别提升+0.92、+0.87和+1.01，展示了准确性与效率的优越平衡，为实际遥感应用中的可扩展甲烷监测提供了有效方案。
