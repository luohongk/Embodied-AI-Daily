# A Few Teacher Steps Go a Long Way: Cost-Efficient On-Policy Data Augmentation for Agent Post-Training

- arXiv: https://arxiv.org/abs/2607.04574v1
- arXiv ID: 2607.04574

---

🔍 **核心问题**：在LLM智能体的后训练中，如何以更低的成本分配教师监督资源，解决训练时教师引导的上下文与测试时学生自主生成的上下文之间的不匹配问题。

🛠 **方法**：将在线数据构建视为预算分配问题，通过控制“学生策略的上下文”、“切换教师的时间分布”、“教师延续长度”、“过滤规则”及“教师推理与监督成本”五个维度，在匹配预算下比较不同策略（如完整演示、长延续、结果过滤或覆盖学生上下文）的效果。

✨ **主要贡献**：在HotpotQA、ALFWorld和Terminal-Bench-Dev上，发现“在学生上下文中添加少量未过滤的教师步骤”比纯行为克隆更高效，且其效果可匹配或超越基于成功过滤或关键上下文过滤的复杂策略，表明低成本、短延续的教师监督是更经济的分配方式。
