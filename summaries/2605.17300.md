# HCLM: A Hierarchical Framework for Cooperative Loco-Manipulation with Dual Quadrupeds

- arXiv: https://arxiv.org/abs/2605.17300v1
- arXiv ID: 2605.17300

---

🔍 **核心问题**：解决双四足机器人协同操作中空间协调、鲁棒运动与闭环物理交互之间的矛盾，实现通用型协作操作。

🛠 **方法**：提出分层框架HCLM，高层采用联合扩散策略学习SE(3)不变的任务空间协调模式，低层通过任务中心混合全身控制器结合主动运动预测控制与反应式执行层，并集成协同导纳机制调节内力。

✨ **主要贡献**：在仿真中验证了协同搬运、打包和交接任务，并在真实场景成功部署，展示了任务可靠性、配置无关性及对强物理扰动的鲁棒性。
