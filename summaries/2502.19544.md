# Efficient Reinforcement Learning by Guiding World Models with Non-Curated Data

- arXiv: https://arxiv.org/abs/2502.19544v3
- arXiv ID: 2502.19544

---

🔍 **核心问题**：如何利用大量未经标注、无奖励信号、质量参差不齐且跨多种形态的离线数据，提升在线强化学习的样本效率。

🛠 **方法**：提出两种技术——经验回放（experience rehearsal）和执行引导（execution guidance），以缓解世界模型微调过程中离线与在线数据之间的分布偏移问题。

✨ **主要贡献**：在72个视觉运动任务（涵盖6种形态）上，该方法在有限样本预算下实现了近乎两倍于从头学习基线的累计得分，并在运动控制和机器人操作等挑战性任务上显著优于现有利用离线数据的方法。
