# Think Fast and Far: Long-Horizon Online POMDP Planning via Rapid State Sampling

- arXiv: https://arxiv.org/abs/2606.04355v1
- arXiv ID: 2606.04355

---

🔍 **核心问题**：解决长时域部分可观测马尔可夫决策过程（POMDP）中规划困难的问题，尤其是状态、动作和观测空间可能连续、离散或混合时的高维、长前瞻步数挑战。

🛠 **方法**：提出ROP-RAS3在线求解器，通过极快采样运动规划技术生成多样化宏动作，并利用这些动作偏置信念空间采样，避免穷举动作空间，从而高效推断近似最优策略。

✨ **主要贡献**：收敛速度仅依赖采样动作数量而非动作空间大小；在长达3000前瞻步、35维状态空间的长时域POMDP中，成功率较现有最优方法提升数倍，并成功应用于物理机器人演示。
