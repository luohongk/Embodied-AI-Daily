# Flow Sensitivity without Control Flow Graph: An Efficient Andersen-Style Flow-Sensitive Pointer Analysis

- arXiv: https://arxiv.org/abs/2508.01974v2
- arXiv ID: 2508.01974

---

🔍 **核心问题**：现有基于控制流图的流敏感指针分析因控制流图结构复杂，导致计算效率低下，难以高效处理大规模程序。

🛠 **方法**：提出CG-FSPTA方法，基于流敏感约束图（FSConsG）进行指针分析，利用集合约束图的结构优势，同时保持变量定义和使用的流敏感性，并集成图优化与动态求解技术。

✨ **主要贡献**：在保持流敏感分析精度的前提下，平均减少33.05%内存使用，加速7.27倍，显著提升大规模软件系统的分析可扩展性。
