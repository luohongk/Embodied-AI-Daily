# Beyond Reactivity: Measuring Proactive Problem Solving in LLM Agents

- arXiv: https://arxiv.org/abs/2510.19771v4
- arXiv ID: 2510.19771

---

🔍 **核心问题**：当前缺乏评估LLM智能体主动性问题解决能力的基准，现有方法局限于局部上下文，无法测试跨来源和长时段的推理能力。

🛠 **方法**：提出PROBE基准，将主动性分解为三个核心能力——搜索未明确问题、识别具体瓶颈、执行适当解决方案，并基于此评估主流LLM和智能体框架。

✨ **主要贡献**：发现即使最先进的模型（如GPT-5和Claude Opus-4.1）端到端性能仅达40%，揭示了当前自主行动的局限性，并指出了未来研究方向。
