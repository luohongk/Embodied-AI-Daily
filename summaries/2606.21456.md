# Technical Report for ICRA 2026 GOOSE 2D Fine-Grained Semantic Segmentation Challenge: Exploring Query-Based Segmentation and Increased Spatial Context for Outdoor Scene Understanding

- arXiv: https://arxiv.org/abs/2606.21456v1
- arXiv ID: 2606.21456

---

🔍 **核心问题**：解决户外场景中细粒度语义分割的挑战，特别是从多相机系统采集的复杂图像中区分56个精细类别。

🛠 **方法**：基于SegFormer基线，逐步改进为查询式Mask2Former架构，增大训练裁剪尺寸以保留更多空间上下文，并采用测试时增强技术。

✨ **主要贡献**：查询式分割显著优于基线，增大裁剪尺寸带来性能大幅提升；最终在测试集上达到69.6% mIoU，为户外细粒度分割提供了强基线，并公开代码与权重。
