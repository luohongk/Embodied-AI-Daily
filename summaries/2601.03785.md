# Membox: Weaving Topic Continuity into Long-Range Memory for LLM Agents

- arXiv: https://arxiv.org/abs/2601.03785v3
- arXiv ID: 2601.03785

---

🔍 **核心问题**：现有LLM智能体记忆系统将对话历史分解为孤立片段或固定大小块，依赖语义相似性检索，导致时间与因果组织弱化，无法有效维持话题连续性。

🛠 **方法**：提出Membox层次化记忆架构，包含Topic Loom（将对话流按局部话题组织为“盒子”）和Trace Weaver（跨盒子链接事件形成宏观话题轨迹）。

✨ **主要贡献**：在LoCoMo基准上，仅Topic Loom检索即比最佳基线提升13.00 F1（53.95 vs 40.95），加入轨迹扩展后F1达55.28；GPT-4o下达59.71 F1。DialSim实验证实跨盒子轨迹在多轮对话中同样有效。
