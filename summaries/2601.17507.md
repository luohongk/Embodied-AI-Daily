# MetaWorld: Skill Transfer and Composition in a Hierarchical World Model for Grounding High-Level Instructions

- arXiv: https://arxiv.org/abs/2601.17507v1
- arXiv ID: 2601.17507

---

🔍 **核心问题**：人形机器人操作任务面临语义与物理之间的鸿沟，现有方法在强化学习样本效率低、模仿学习泛化性差、视觉语言模型物理不一致性等方面存在局限。

🛠 **方法**：提出MetaWorld分层世界模型，通过专家策略迁移将任务解耦为VLM驱动的语义层和紧凑状态空间中的潜在动力学模型，并采用动态专家选择与运动先验融合机制，利用预训练多专家策略库实现高效在线适应。

✨ **主要贡献**：在Humanoid-Bench上超越基于世界模型的强化学习方法，在任务完成度和运动连贯性上取得显著提升；VLM作为语义接口直接映射指令到可执行技能，绕过了符号落地问题。
