# RainBalance: Alleviating Dual Imbalance in GNSS-based Precipitation Nowcasting via Continuous Probability Modeling

- arXiv: https://arxiv.org/abs/2601.06137v1
- arXiv ID: 2601.06137

---

🔍 **核心问题**：解决GNSS降水临近预报中两类不平衡问题——非降水事件占主导和极端降水样本稀缺，导致模型性能受限。

🛠 **方法**：提出RainBalance框架，通过聚类获取样本概率分布，并利用变分自编码器（VAE）将其映射到连续潜在空间，将任务从拟合单一不平衡标签转为建模连续概率标签分布。

✨ **主要贡献**：作为即插即用模块，RainBalance在多个先进模型中取得一致性能提升，统计分析与消融实验验证了其缓解不平衡问题的有效性。
