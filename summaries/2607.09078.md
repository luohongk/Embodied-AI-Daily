# Toward Active Object Detection for UAVs in the Wild: A Large-Scale Dataset, Benchmark and Method

- arXiv: https://arxiv.org/abs/2607.09078v1
- arXiv ID: 2607.09078

---

🔍 **核心问题**：针对无人机在复杂野外场景中因遮挡或目标像素稀疏导致目标检测困难的问题，提出主动目标检测（AOD）范式，但缺乏高质量无人机-地面主动目标检测（UGAOD）数据集和基准。

🛠 **方法**：构建首个大规模真实世界数据集ATRNet-LUDO（含121,000张多视角全景图像和121万张局部切片），并基于此建立评估基准；提出AOD-JEPA方法，利用联合嵌入预测架构（JEPA）构建世界模型以增强状态表征学习，并融入AOD先验知识。

✨ **主要贡献**：填补了UGAOD领域数据集和基准的空白，揭示了现有深度强化学习（DRL）策略的泛化差距，并通过AOD-JEPA验证了其有效性和优越性。
