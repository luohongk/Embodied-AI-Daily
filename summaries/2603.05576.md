# Task Parameter Extrapolation via Learning Inverse Tasks from Forward Demonstrations

- arXiv: https://arxiv.org/abs/2603.05576v3
- arXiv ID: 2603.05576

---

🔍 **核心问题**：解决机器人技能策略在陌生条件下泛化能力不足的问题，尤其是模仿学习局限于训练区域，而迁移学习在零样本泛化中精度低且数据需求大。

🛠 **方法**：提出一种联合学习框架，通过构建正向与逆向任务的共同表征，利用新配置下的正向演示数据（无需直接监督）来成功执行对应的逆向任务，实现知识的高效迁移。

✨ **主要贡献**：在模拟和真实环境中，针对复杂操作任务（如使用多样物体和工具），该方法在零样本泛化精度和效率上优于扩散模型和多模态VAE基线，并通过消融实验验证了其外推能力。
