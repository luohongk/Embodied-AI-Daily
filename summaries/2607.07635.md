# Unlearning to Protect: A Distilled Reinforcement Learning Framework with Privacy-Preserving Feature Unlearning and XAI for IoT Security

- arXiv: https://arxiv.org/abs/2607.07635v1
- arXiv ID: 2607.07635

---

🔍 **核心问题**：解决物联网设备因缺乏内置流量过滤而易受僵尸网络攻击的问题，同时克服现有AI方案部署过重、无法选择性遗忘敏感或过时特征的局限。

🛠 **方法**：提出DiRLU框架，采用A2C强化学习训练教师模型，并通过知识蒸馏将其压缩为轻量学生模型；结合事后遗忘机制修改权重以移除特定特征，并集成LIME可解释AI增强决策透明度。

✨ **主要贡献**：学生模型在BoT-IoT数据集（25%数据）上达99.60%准确率和99.80% F1分数，计算量仅需2370 FLOPS（比现有最优模型高效3.87倍），且支持隐私遗忘与可逆恢复，符合GDPR标准。
