# Themis: An explainable AI-enabled framework for Reinforcement Learning with Human Feedback

- arXiv: https://arxiv.org/abs/2606.24622v1
- arXiv ID: 2606.24622

---

🔍 **核心问题**：当前缺乏一个结合可解释人工智能（XAI）与人类反馈的公开框架，来提升强化学习系统的安全性和透明度。  
🛠 **方法**：提出Themis框架，整合XAI技术与人类反馈强化学习（RLHF），支持200多种环境，并提供云端平台用于收集人类偏好和实验管理。  
✨ **主要贡献**：Themis能训练出匹配或超越环境真实奖励信号的奖励模型，且支持大规模用户并发实验，在普通商用机器上可承载千名用户连续测试。
