# FlowPipe: LLM-Enhanced Conditional Generative Flow Networks for Data Preparation Pipeline Construction

- arXiv: https://arxiv.org/abs/2606.24679v1
- arXiv ID: 2606.24679

---

🔍 **核心问题**：自动构建数据预处理管道面临组合搜索空间大、端到端评估昂贵、现有方法在长期信用分配、数据集上下文注入和无效状态探索方面存在局限。

🛠 **方法**：提出FlowPipe框架，将管道合成建模为有向无环图上的条件概率流生成，采用条件生成流网络（C-GFlowNets）结合轨迹平衡目标，并通过特征线性调制（FiLM）注入大语言模型（LLM）导出的逻辑先验，同时引入失败感知机制避免无效状态。

✨ **主要贡献**：在74个真实数据集上，FlowPipe相比现有最优方法平均提升11.96%的准确率，训练收敛速度提升12.5倍，并开源代码。
