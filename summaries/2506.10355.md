# TreeLoRA: Efficient Continual Learning via Layer-Wise LoRAs Guided by a Hierarchical Gradient-Similarity Tree

- arXiv: https://arxiv.org/abs/2506.10355v2
- arXiv ID: 2506.10355

---

🔍 **核心问题**：解决大规模预训练模型在持续学习场景中面临的计算效率低下和灾难性遗忘问题。  
🛠 **方法**：提出TreeLoRA方法，通过构建层级梯度相似性树（K-D Tree）指导低秩适配器（LoRA）的分配，并利用基于置信下界的bandit算法高效探索任务结构，结合稀疏梯度更新优化参数。  
✨ **主要贡献**：在视觉Transformer和大型语言模型上验证了方法的有效性与效率，理论分析支持其合理性，显著降低了持续学习的计算开销并保持了模型性能。
