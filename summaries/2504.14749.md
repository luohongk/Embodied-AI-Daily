# PPO-EPO: Energy and Performance Optimization for O-RAN Using Reinforcement Learning

- arXiv: https://arxiv.org/abs/2504.14749v1
- arXiv ID: 2504.14749

---

🔍 **核心问题**：移动通信网络能耗高，影响资本与运营支出，需在保障服务质量的同时优化能效。  
🛠 **方法**：基于近端策略优化（PPO）算法的强化学习模型，通过流量感知的小区关闭策略，动态优化流量引导与能效。  
✨ **主要贡献**：在吞吐量、干扰和资源利用率约束下，实现自适应小区关闭决策，显著提升网络能效与下行吞吐量。
