# Infer and Adapt: Bipedal Locomotion Reward Learning from Demonstrations via Inverse Reinforcement Learning

- arXiv: https://arxiv.org/abs/2309.16074v1
- arXiv ID: 2309.16074

---

🔍 **核心问题**：解决双足机器人在高度不平、动态变化地形上行走时，因动力学复杂和环境交互困难而难以学习有效策略的问题，特别是缺乏对专家奖励函数的学习研究。

🛠 **方法**：采用先进的逆强化学习（IRL）技术，通过非线性函数逼近从专家演示中学习奖励函数，并基于推断的奖励函数训练双足行走策略。

✨ **主要贡献**：首次将IRL系统应用于双足运动，揭示了专家行走策略的深层含义，并实验证明学习到的奖励函数能显著提升机器人在未知地形上的行走性能和适应性。
