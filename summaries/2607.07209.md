# Continual Learning With Participation Privacy: An Auditable Buffering-Aggregation Recipe

- arXiv: https://arxiv.org/abs/2607.07209v1
- arXiv ID: 2607.07209

---

🔍 **核心问题**：在联邦学习和流式学习中，用户数据的一次插入或删除会导致后续所有模型更新偏移，传统基于汉明邻域的分析方法无法有效保护参与隐私，需要解决自适应交互下完整轨迹的隐私泄露问题。

🛠 **方法**：提出一种可审计的模块化方案，包括随机缓冲包装器（将用户更新流按大小$[U,2U]$分桶，转化为汉明风格的每桶更新流）和认证定理（证明非自适应汉明邻域差分隐私证明在特定条件下可扩展到自适应输入）。

✨ **主要贡献**：首次为单编辑用户流提供轨迹级别的$(\varepsilon,δ)$-差分隐私保证，通过参数$U$显式连接隐私与延迟，并兼容标准原语（如树前缀和），实现可审计的隐私保护。
