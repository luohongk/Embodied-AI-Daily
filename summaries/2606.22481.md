# Lighting-Consistent Object Transfer Across Radiance Fields

- arXiv: https://arxiv.org/abs/2606.22481v1
- arXiv ID: 2606.22481

---

🔍 **核心问题**：在3D高斯泼溅（3DGS）场景中，将物体从一个场景转移到另一个场景时，由于光照条件不一致，直接合成会导致视觉效果不真实。

🛠 **方法**：提出一种扩散模型，用于协调光照不一致的合成图像；模型使用包含合成、生成和真实数据的异构图像对进行训练，并通过后优化步骤将协调后的视图整合为3DGS表示。

✨ **主要贡献**：实现了光照一致的物体跨场景转移，显著提升了合成质量，使3DGS中的物体转移操作更易用且效果优于现有方法。
