## 0. 概览（Concise Summary）

- **主题（Topic）**：月球定位、导航与授时（Lunar PNT），具体研究地球GNSS信号在月球处受到的电离层和等离子层延迟特性。
- **问题（Problem）**：利用地球GNSS信号进行月球导航时，信号会穿过地球的电离层和等离子层，产生未被充分建模的延迟误差。特别是对于月球用户，信号以近乎水平的长路径穿过这些区域，其延迟特性与地球用户完全不同，现有模型不适用。
- **方法（Method）**：采用全球核心等离子体模型（GCPM）模拟电子密度分布，并提出一种低成本的射线追踪算法，该算法使用Nelder-Mead优化方法迭代求解弯曲信号路径，从而计算一阶、二阶、三阶群延迟和弯曲引起的额外路径延迟。
- **创新（Innovation）**：首次系统地针对月球轨道和月球表面的GNSS接收机，量化了电离层和等离子层延迟。提出了一个计算高效的射线追踪算法，并分析了延迟随信号频率、地磁活动（Kp指数）和太阳活动（R12指数）的变化规律。
- **意义（Significance）**：为设计鲁棒的月球PNT系统提供了关键的误差模型和定量依据。研究结果有助于指导信号选择（如L1 vs L5）、高度掩码设置以及延迟缓解策略的开发，对于实现高精度、高可靠性的月球GNSS导航至关重要。
- **一句话总结**：本文通过GCPM模型和自定义射线追踪算法，首次全面表征了地球GNSS信号在月球处受到的电离层和等离子层延迟，揭示了其与信号路径、太阳/地磁活动的强相关性，为月球PNT系统设计提供了关键误差预算。

## 1. 研究动机（Motivation）

- **问题与空白**：近年来，利用地球GNSS信号（包括弱旁瓣信号）进行月球定位已被验证可行（如LuGRE实验）。然而，信号从地球GNSS卫星到月球接收机的超长传播路径会穿过地球的电离层和等离子层。对于地球用户，信号路径近乎垂直，延迟模型（如Klobuchar模型）成熟。但对于月球用户，信号以近乎水平的“切线”方式穿过这些区域，路径更长，且等离子层（从1000km延伸到20000km以上）的贡献变得显著。目前，**没有研究系统地量化这种独特几何结构下的延迟特性**。
- **重要性**：月球PNT系统面临信号弱、几何精度因子差等挑战，任何误差源都会严重影响定位精度。电离层和等离子层延迟是主要的误差源之一。虽然双频接收机可以消除大部分一阶延迟，但会增加接收机噪声，这对于跟踪微弱信号是不利的。因此，**准确建模和预测这种延迟对于设计鲁棒的、低误差的月球导航算法至关重要**。
- **挑战**：挑战主要在于两点：1）需要能够准确描述等离子层电子密度分布的模型（GCPM）。2）需要能够处理长距离、大弯曲路径的射线追踪算法，该算法必须计算高效，因为需要模拟大量信号。

## 2. 创新点（Innovation）

- **首次系统性表征**：本文是**首次**针对月球轨道和月球表面用户，对地球GNSS信号的电离层和等离子层延迟进行全面的特性分析。这填补了该领域的研究空白。
- **计算高效的射线追踪算法**：提出了一种新颖的、低成本的射线追踪算法。其核心创新在于：
    1.  **路径分段**：将射线路径分为“等离子层内”（弯曲）和“等离子层外”（真空，直线）两部分，假设在距离地心4个地球半径（$4 R_E$）外，信号沿直线传播。这大大简化了计算。
    2.  **迭代优化**：使用Nelder-Mead单纯形算法（一种无需梯度信息的优化方法）来搜索初始射线方向，使得射线在穿过等离子层后能精确到达月球接收机。优化变量仅为初始射线的方位角和仰角，是一个二维优化问题。
    3.  **计算加速**：在优化迭代过程中，不重复调用GCPM模型计算电子密度，而是利用首次积分时存储的射线方向导数（$d\hat{s}/ds$）来快速计算新路径，显著提升了效率。
- **多维度参数分析**：系统地分析了延迟随**信号频率**（L1 vs L5）、**地磁活动**（Kp指数）和**太阳活动**（R12指数）的变化规律，为不同场景下的系统设计提供了量化指导。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

论文的整体架构是一个仿真框架，用于计算和表征月球GNSS接收机处的电离层和等离子层延迟。主要步骤包括：

1.  **环境建模**：使用GCPM v2.4模型，根据给定的日期、时间、Kp指数和R12指数，生成地球周围（特别是等离子层内）的三维电子密度分布。
2.  **信号源与接收机建模**：模拟GPS和Galileo卫星的轨道、天线方向图（包括旁瓣增益），以及月球轨道（LCRNS卫星）和月球南极用户的接收机天线方向图、灵敏度（C/N0阈值）。
3.  **射线追踪**：对于每一对可见的GNSS卫星-月球接收机，执行自定义的射线追踪算法，计算信号的实际弯曲路径。
4.  **延迟计算**：沿着计算出的弯曲路径，积分计算一阶、二阶、三阶群延迟和弯曲引起的额外路径延迟。
5.  **统计分析**：对所有信号样本进行统计分析，研究延迟与切线高度、频率、Kp、R12等参数的关系。

### 3.2 关键算法与数学推导

**1. 电离层/等离子层延迟公式**

对于右旋圆极化GNSS信号，相位折射率 $n$ 和群折射率 $n_{gr}$ 的近似表达式为：

$$n = 1 - \frac{f_p^2}{2f^2} - \frac{f_p^2 f_g \cos\theta}{2f^3} - \frac{f_p^2}{4f^4} \left[ \frac{f_p^2}{2} + f_g^2(1+\cos^2\theta) \right]$$

$$n_{gr} = 1 + \frac{f_p^2}{2f^2} + \frac{f_p^2 f_g \cos\theta}{f^3} + \frac{3f_p^2}{4f^4} \left[ \frac{f_p^2}{2} + f_g^2(1+\cos^2\theta) \right]$$

其中，$f_p = \sqrt{n_e e^2/(4\pi\epsilon_0 m)}$ 是等离子体频率，$f_g = eB/(2\pi m)$ 是回旋频率，$f$ 是信号频率，$\theta$ 是射线路径与磁场线的夹角。

群延迟 $dI_{gr}$ 和相位超前 $dI$ 沿路径的积分可表示为：

$$dI_{gr} = \int (n_{gr} - 1) ds = \frac{p}{f^2} + \frac{q}{f^3} + \frac{u}{f^4}$$

其中，一阶、二阶、三阶延迟项系数为：
- $p = 40.3 \int n_e ds = 40.3 \times TEC$
- $q = -2.2566 \times 10^{12} \int n_e B \cos\theta ds$
- $u = 2437 \int n_e^2 ds + 4.74 \times 10^{22} \int n_e B^2 (1+\cos^2\theta) ds$

**2. 射线追踪算法**

射线传播由矢量程函方程控制：
$$\frac{dr}{ds} = \hat{s}, \quad \frac{d\hat{s}}{ds} = \frac{\nabla n - \hat{s}(\hat{s} \cdot \nabla n)}{n}$$

算法核心是“打靶法”结合Nelder-Mead优化：
1.  **初始化**：猜测一个初始射线方向 $\hat{s}_0$。
2.  **前向积分**：从发射机位置开始，使用四阶龙格-库塔法积分上述方程，直到射线到达预定义的截止半径（$4 R_E$）。记录路径点位置和方向导数。
3.  **外推**：假设截止半径外为真空，射线沿 $\hat{s}_{exit}$ 直线传播，计算到达接收机平面的估计位置 $\bar{x}_{rx} = x_{exit} + \Delta s \hat{s}_{exit}$。
4.  **优化**：使用Nelder-Mead算法调整初始方向 $\hat{s}_0$（优化变量为方位角和仰角），以最小化估计位置 $\bar{x}_{rx}$ 与真实接收机位置 $x_{rx}$ 之间的距离。
5.  **迭代**：重复步骤1-4，直到位置误差小于阈值。

### 3.3 模型、训练与数据集

- **核心模型**：**Global Core Plasma Model (GCPM) v2.4**。这是一个经验模型，用于提供内磁层的电子密度。它集成了电离层（基于IRI）、等离子层、等离子层顶、磁层槽和极盖区的子模型。
- **训练**：本文不涉及模型训练，而是使用GCPM作为仿真工具。
- **数据集**：
    - **GNSS卫星**：使用2025年6月23日的SP3精密星历文件初始化GPS和Galileo卫星轨道。天线方向图来自公开的NASA ACE研究、Lockheed Martin等数据。
    - **月球用户**：模拟了5颗LCRNS月球轨道卫星（初始轨道参数见表1）和1个月球南极用户。LCRNS卫星轨道如图5所示。
    - **仿真场景**：基线场景为2025年1月1日12:00 UTC，Kp=3.0，R12=167.24（高太阳活动）。仿真时长为45小时。
    - **参数变化**：为分析影响，分别改变Kp指数（1, 3, 5, 6.667, 9）和R12指数（10, 50, 100, 150, 200）。

### 3.4 实验设置与结果

- **实验设置**：软件环境为Python/C++的LuPNT库与Fortran的GCPM v2.4接口。接收机假设峰值增益14 dBi，半功率波束宽度6度，能跟踪C/N0 ≥ 18 dB-Hz的信号。
- **主要结果**：
    1.  **基线结果（表2，图8，图9）**：
        - 平均群延迟通常在1米量级，但对于低切线高度（<500 km）的路径，平均延迟可达**50.59米**，极端情况下可超过100米。
        - 延迟随切线高度增加而迅速减小，高于10000 km时平均延迟小于0.2米。
        - 弯曲延迟（$d_{len} + d_{I1,B}$）通常远小于一阶延迟，但对于低切线高度（<1000 km）不可忽略，可超过1米。
        - 二阶、三阶延迟非常小，可忽略。
        - 存在**偏差-方差权衡**：低切线高度信号延迟大（偏差大），但C/N0高，接收机噪声小（方差小）；高切线高度信号则相反。
    2.  **地磁活动影响（图10，图11）**：
        - Kp指数越高，等离子层顶收缩（从4-6 $R_E$ 收缩到2-3 $R_E$），高密度等离子体区域减小。
        - 因此，**Kp指数越高，总群延迟越小**。
    3.  **太阳活动影响（图12，图13，表3）**：
        - R12指数越高，太阳活动越强，电离层和等离子层电子密度越大。
        - 因此，**R12指数越高，总群延迟越大**。例如，为将99%的延迟控制在5米以下，R12≤50时需2000 km高度掩码，而R12=200时需10000 km掩码。
    4.  **频率影响（表5，图14）**：
        - L1信号（1575.42 MHz）的群延迟约为L5信号（1176.45 MHz）的53-55%。
        - L5信号的接收机噪声仅为L1信号的7-8%。
        - 综合总UERE（User Equivalent Range Error）来看，在切线高度低于约5000 km时，**L1信号的总误差更小**（因为延迟误差占主导）。在更高高度，**L5信号的总误差更小**（因为接收机噪声占主导）。

### 3.5 技术细节

- **射线追踪步长自适应**：积分步长根据高度自适应，低于1000 km时步长10 km，1000-4000 km时20 km，高于4000 km时100 km，在保证精度的同时提高效率。
- **GCPM修改**：对原始GCPM代码进行了修改，使其能处理2015年后的历元，并接受自定义的R12指数作为输入。
- **收敛性**：射线追踪算法通常在3-6次迭代内收敛，终端位置误差可降至亚米级（L1）到十几米（L5），对应的延迟计算误差低于1毫米（图3）。

## 5. 澄清与简化（Clarifications and Simplifications）

- **什么是等离子层？**：可以把它想象成地球大气层最外层的“稀薄气体层”，但它是由带电粒子（等离子体）组成的。它从电离层顶部（约1000公里）一直延伸到数万公里。虽然密度比电离层低，但因为月球GNSS信号要斜着穿过它非常长的距离，所以累积的延迟效应不可忽视。
- **什么是“切线高度”？**：想象一下，从月球看地球，GNSS信号像一条射线擦过地球边缘。这条射线离地球表面最近的那个点的高度，就是切线高度。切线高度越低，信号穿过稠密大气层（电离层）的部分就越多，延迟就越大。图2清晰地展示了这个概念。
- **射线追踪算法为什么高效？**：传统方法需要在整个路径上反复求解复杂的微分方程。本文的聪明之处在于，它把问题分成了两步：1）只在稠密的等离子层内部（4个地球半径内）进行精确的弯曲路径计算；2）在更远的太空中，假设信号是直线传播。这样，需要复杂计算的部分大大缩短了。而且，在优化调整射线方向时，它复用了第一次计算的结果，避免了重复计算，就像用一张“地图”来导航，而不是每次都要重新画地图。
- **偏差-方差权衡是什么？**：这就像在“准”和“稳”之间做选择。低切线高度的信号（如L5）虽然延迟大（不准），但信号强，噪声小（稳定）。高切线高度的信号（如L1）延迟小（准），但信号弱，噪声大（不稳定）。系统设计者需要根据任务需求（是更看重精度还是稳定性）来选择合适的信号或组合。
