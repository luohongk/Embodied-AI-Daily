# Geometry-aware Depth-guided Representation Learning for Structure-preserving Low-light Image Enhancement

- arXiv: https://arxiv.org/abs/2607.05005v1
- arXiv ID: 2607.05005

---

🔍 **核心问题**：现有低光图像增强方法主要关注外观恢复，但未能充分利用场景几何信息来保持结构一致性。  
🛠 **方法**：提出深度引导的多尺度注意力网络（DMSA-Net），通过Retinex分解模块获取光照不变反射图，从中推断深度线索，并采用多尺度深度引导融合策略与深度感知注意力机制，在编码器-解码器架构中整合几何与外观特征。  
✨ **主要贡献**：在多个基准数据集上实现有效的低光恢复并提升结构保持性能；构建了深度增强的低光数据集LOL-D，推动几何感知低光视觉研究。
