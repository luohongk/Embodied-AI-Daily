# DecompRL: Solving Harder Problems by Learning Modular Code Generation

- arXiv: https://arxiv.org/abs/2607.02390v1
- arXiv ID: 2607.02390

---

🔍 **核心问题**：当大语言模型（LLM）的基础策略生成正确解的概率极低时，传统重复采样和强化学习（RL）均因搜索空间过大而失效，如何解决这类更难的问题？

🛠 **方法**：提出DecompRL算法，通过强化学习显式训练模型将复杂问题分解为多个可独立求解的子模块，并生成层次化代码结构，最终通过组合子模块实现（最多kⁿ个候选解）来提升求解效率。

✨ **主要贡献**：将计算瓶颈从昂贵的GPU推理转移到廉价的CPU评估，GPU令牌成本降低约50倍；在LiveCodeBench和CodeContests上，DecompRL在每问题超10⁵令牌时显著优于标准及多样性优化的RL基线，能解决传统方法无法触及的问题。
