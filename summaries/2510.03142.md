# MM-Nav: Multi-View VLA Model for Robust Visual Navigation via Multi-Expert Learning

- arXiv: https://arxiv.org/abs/2510.03142v2
- arXiv ID: 2510.03142

---

🔍 **核心问题**：解决视觉导航中光学信息难以像激光雷达或深度图那样显式建模，导致需要复杂模型和大规模数据的问题。

🛠 **方法**：提出MM-Nav多视图视觉-语言-动作（VLA）模型，基于预训练大语言模型和视觉基础模型，通过教师-学生范式从三个强化学习专家（分别擅长到达、挤压、避障）在线收集数据，并动态平衡训练比例以融合多种导航能力。

✨ **主要贡献**：在合成环境中验证了模型的强泛化能力，且学生VLA模型性能超越强化学习教师，展现了多能力协同的增效效果；真实世界实验进一步证实了方法的有效性。
