# Walking-by-Logic: Signal Temporal Logic-Guided Model Predictive Control for Bipedal Locomotion Resilient to External Perturbations

- arXiv: https://arxiv.org/abs/2309.13172v1
- arXiv ID: 2309.13172

---

🔍 **核心问题**：针对双足机器人在受到外部扰动时，现有恢复策略难以处理复杂任务逻辑推理和鲁棒性评估，容易因策略不当或鲁棒性不足而失败的问题。

🛠 **方法**：提出一种基于模型预测控制（MPC）的规划框架，引入信号时序逻辑（STL）规范来保证任务完成并量化鲁棒性，通过STL引导的轨迹优化生成最优且安全的恢复轨迹。

✨ **主要贡献**：首次将STL引导的轨迹优化应用于双足机器人受扰后的推挤恢复，在交叉腿等高难度场景中优于现有控制器，并展示了在踩踏石等任务中的通用性。
