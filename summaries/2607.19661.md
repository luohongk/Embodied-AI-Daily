# Leveraging ECRAM for Edge Continual Learning

- arXiv: https://arxiv.org/abs/2607.19661v1
- arXiv ID: 2607.19661

---

🔍 **核心问题**：边缘计算平台（如自动驾驶、智能传感设备）需要实时适应动态环境，但现有持续学习算法在CPU/GPU与内存间产生大量数据移动，导致能效低、不适合边缘部署；同时，存内计算（IMC）虽能减少数据移动，却面临计算噪声大、训练精度低及资源效率支持不足的挑战。

🛠 **方法**：提出CLASP系统，这是首个端到端支持持续学习的IMC加速平台。软硬件协同设计，基于自制的后道兼容ECRAM器件，通过软件可见的汇编级指令支持多种持续学习算法，并克服其他新兴存储器件在IMC训练中的噪声问题。

✨ **主要贡献**：在MNIST数据集上，CLASP结合ECRAM的持续学习精度接近GPU训练水平，同时实现67倍加速和132倍能耗节省，显著提升边缘持续学习的效率与可行性。
