# Think When It Matters: Conditional VLM Reasoning for Social Navigation with RL Policies

- arXiv: https://arxiv.org/abs/2607.10991v1
- arXiv ID: 2607.10991

---

🔍 **核心问题**：移动机器人在社交导航中需要平衡实时响应与语义推理能力，现有强化学习策略缺乏语义理解，而视觉语言模型计算成本高、推理慢，难以实时部署。

🛠 **方法**：提出HUMA混合架构，日常低密度导航由快速反应的强化学习策略处理，当人类进入机器人邻近敏感区域时，则触发后训练的视觉语言模型进行高级语义推理，实现动态切换。

✨ **主要贡献**：在Social-MP3D和Social-HM3D基准上分别提升任务成功率20%和3%，显著减少个人空间侵犯与碰撞；消融实验验证各组件有效性，并在Mirokaï机器人上完成实际部署验证。
