# A Standardized Benchmark for Skeleton-Based Rehabilitation Assessment Using Deep Learning

- arXiv: https://arxiv.org/abs/2507.21018v2
- arXiv ID: 2507.21018

---

🔍 **核心问题**：康复评估领域缺乏标准化基准、统一评估协议和可复现方法，导致不同研究间难以比较和推进。  
🛠 **方法**：整合现有康复数据集形成统一档案库Rehab-Pile，提出通用基准框架，并对多种深度学习架构在分类与回归任务上进行广泛基准测试。  
✨ **主要贡献**：建立了标准化评估平台，公开所有数据集、代码和结果，为自动化康复评估研究奠定可复现基础，推动可靠、个性化康复方案的发展。
