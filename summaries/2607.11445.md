# Event-based Neural Decoding for Neuroprosthetic Motor Control

- arXiv: https://arxiv.org/abs/2607.11445v1
- arXiv ID: 2607.11445

---

🔍 **核心问题**：解决神经假体运动控制中神经解码的高延迟、高能耗和空间限制问题，提升患者使用假肢的实时性和便携性。  
🛠 **方法**：提出基于事件的门控循环单元（Event-based Gated Recurrent Unit），通过分级脉冲生成稀疏通信模式，并结合高效训练与稀疏推理方法。  
✨ **主要贡献**：在保持高性能神经解码的同时，显著降低计算和通信开销，优于传统脉冲神经网络，为设备端实时解码提供新方案。
