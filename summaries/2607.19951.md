# OffNadirLoc: Benchmark and Framework for Challenging UAV-to-Satellite Geo-Localization under Large Off-Nadir Views

- arXiv: https://arxiv.org/abs/2607.19951v1
- arXiv ID: 2607.19951

---

🔍 **核心问题**：解决无人机与卫星图像在大倾斜视角下的跨视角地理定位难题，现有方法主要针对近天底场景，难以应对视角畸变、遮挡和外观差异等挑战。

🛠 **方法**：提出ONLoc框架，包含结构感知上下文加权机制以动态强调可靠局部特征并抑制模糊区域，以及视图一致学习策略，将多视角无人机图像与对应卫星图像视为语义组进行集合级监督训练。

✨ **主要贡献**：构建了首个大倾斜视角无人机-卫星地理定位基准OffNadirLoc，所提方法在OffNadirLoc及四个近天底数据集上均超越现有最优方法，并展现出强零样本泛化能力。
