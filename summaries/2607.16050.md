# DELUGE: Towards Continental-Scale Daily Pluvial Flood Damage Prediction via Interpretable Conditioning on Foundation Model Embeddings

- arXiv: https://arxiv.org/abs/2607.16050v1
- arXiv ID: 2607.16050

---

🔍 **核心问题**：针对美国全国范围内每日尺度、高分辨率（约1公里）的暴雨洪水损失预测难题，现有方法分辨率粗、区域局限或计算成本高，难以满足日常大陆尺度应用需求。

🛠 **方法**：提出DELUGE多模态深度学习框架，基于灾害、暴露度和脆弱性三要素，利用空间和时间校正后的NFIP索赔数据（2017-2022），并在水文气象分支中引入参数化模块（Value Modulator和Temporal Modulator），通过地形描述符和AlphaEarth基础模型嵌入进行条件化，实现可解释性设计。

✨ **主要贡献**：在空间块留出验证下，DELUGE在美元加权PR-AUC指标上比调优的随机森林、XGBoost和LightGBM基线提升9%至30%，尤其擅长预测高成本罕见事件；同时提出可迁移的条件化嵌入方案，为其他地理空间预测任务提供可解释性范式。
