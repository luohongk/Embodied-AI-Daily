# ABot-World-0: Infinite Interactive World Rollout on a Single Desktop GPU

- arXiv: https://arxiv.org/abs/2607.19191v1
- arXiv ID: 2607.19191

---

🔍 **核心问题**：如何在单张桌面级GPU上实现实时、长时程的闭环交互式视频世界模型，以支持可控的虚拟世界探索与角色交互。

🛠 **方法**：提出ABot-World-0，采用多源数据基础设施（AAA游戏、仿真引擎、互联网视频）训练动作条件视频世界模型；通过WorldExplorer进行代理驱动的数据收集，结合14项确定性质量检查与VLM评估；使用双向动作条件教师模型逐步蒸馏为因果学生模型，并引入LongForcing对齐长自回归轨迹；利用原始键盘动作作为统一控制接口，并设计参考角色记忆保持身份一致性；部署时采用流式推理栈，包含轻量VAE解码器、高效注意力机制、内存感知调度与低比特DiT推理。

✨ **主要贡献**：在单张NVIDIA RTX 5090 GPU上实现720P视频以最高16 FPS实时生成，动作到首帧延迟仅1.2秒，峰值显存约19GiB；在WorldRoamBench和扩展交互实验中展示了出色的可控性与长时程世界演化一致性。
