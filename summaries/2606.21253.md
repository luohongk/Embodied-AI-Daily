# Gradient-Free Warm-Start Library Recovery: an Amortized-Regret Separation

- arXiv: https://arxiv.org/abs/2606.21253v1
- arXiv ID: 2606.21253

---

🔍 **核心问题**：在无梯度、在线、仅追加的持续学习场景中，如何实现高效的热启动库恢复，以降低重复流数据下的计算成本。

🛠 **方法**：提出一种解耦机制，将识别已知流模式（成本与维度无关）与估计新模式（成本与维度成正比）分离，通过计数校准变体实现无超参数、无遗忘的在线学习。

✨ **主要贡献**：理论证明热启动学习器的摊销恢复成本远低于无记忆重估计器，且每个项均达到极小极大最优；实验验证了维度依赖的分离优势，并揭示了可识别模式数量受单纯形填充限制等边界条件。
