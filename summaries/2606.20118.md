# Pose6DAug: Physically Plausible Multi-view Object Swapping for Robot Data Augmentation

- arXiv: https://arxiv.org/abs/2606.20118v1
- arXiv ID: 2606.20118

---

🔍 **核心问题**：视觉-语言-动作（VLA）策略在处理外观或几何形状偏离训练分布的新物体时容易失败，而传统收集多视角遥操作数据的方法成本高、扩展性差。

🛠 **方法**：提出Pose6DAug框架，通过利用策略自身成功轨迹中的物理有效动作和校准多视角观测，仅替换操作物体并保持轨迹不变，在3D空间中用时间连贯的6D姿态轨迹驱动显式网格，生成几何一致的多视角渲染数据。

✨ **主要贡献**：无需新数据收集即可将成功轨迹转化为针对失败模式的演示，在新物体上相对最先进基线提升16.5%的成功率，同时保持分布内性能，验证了多视角物理一致增强对VLA泛化的实用价值。
