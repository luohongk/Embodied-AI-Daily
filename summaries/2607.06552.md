# MonoIR-RS: Infrared Remote Sensing Vision-Language Learning with CLIP and VLM Adaptation

- arXiv: https://arxiv.org/abs/2607.06552v1
- arXiv ID: 2607.06552

---

🔍 **核心问题**：红外遥感图像在视觉-语言理解中缺乏大规模数据集和模型适配，现有资源多聚焦可见光波段，导致红外模态的语义对齐研究不足。

🛠 **方法**：构建了大规模红外遥感视觉-语言数据集MonoIR-RS，通过CLIP对比学习适配和VLM指令微调，将红外图像作为模型输入模态，并基于灰度结构和红外对比度重写文本描述。

✨ **主要贡献**：合成红外图像在AVIID基准上更接近真实热成像；IR适配使CLIP平均召回率提升12.8点，VLM描述的红外线索覆盖率达100%，RGB颜色泄漏降至接近零，为红外遥感与语言对齐提供了可控、可复现的测试平台。
