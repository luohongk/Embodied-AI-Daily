# When LLMs Analyze Scars: From Images to Clinically-Meaningful Features

- arXiv: https://arxiv.org/abs/2606.18063v1
- arXiv ID: 2606.18063

---

🔍 **核心问题**：解决医学图像分类中因数据稀缺（如标注成本高、隐私限制、疾病罕见）导致深度学习模型性能下降的问题，尤其针对病理性疤痕（如瘢痕疙瘩与增生性疤痕）的细粒度分类。

🛠 **方法**：提出ScaFE框架，将大语言模型（LLM）作为知识驱动的特征工程师，通过提示LLM生成可执行Python代码，从图像中提取与临床评分系统（如温哥华疤痕量表）对齐的低维、可解释特征，而非直接进行端到端分类。

✨ **主要贡献**：在数据稀缺条件下，该方法显著优于端到端深度学习基线及将LLM作为黑箱分类器的方案，同时具备数据高效性（少量样本即可鲁棒）、隐私保护（本地处理图像）和临床可解释性（特征基于临床推理）。
