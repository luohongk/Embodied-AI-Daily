# DiffusionVS: A Generative Framework for Robust Visual Servoing Based on Diffusion Policy

- arXiv: https://arxiv.org/abs/2606.19397v1
- arXiv ID: 2606.19397

---

🔍 **核心问题**：传统基于回归的视觉伺服方法因噪声敏感的单步映射和分布偏移下的误差累积，导致轨迹抖动，鲁棒性不足。

🛠 **方法**：提出基于扩散策略的视觉伺服框架，以观测标签点的归一化图像坐标为输入，通过条件去噪生成相机速度；并采用在线训练范式，通过交互经验收集持续扩展训练数据多样性。

✨ **主要贡献**：在仿真中实现近100%成功率，物理实验中达93%；验证了扩散机制的通用性，集成该模块后现有视觉伺服网络性能均获提升。
