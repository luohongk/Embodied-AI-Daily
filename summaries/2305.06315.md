# NervePool: A Simplicial Pooling Layer

- arXiv: https://arxiv.org/abs/2305.06315v3
- arXiv ID: 2305.06315

---

🔍 **核心问题**：针对图结构数据上的深度学习问题，需要一种适用于单纯复形（图的广义形式，包含顶点和边之外的高维单纯形）的池化层，以实现下采样、降低计算成本和减少过拟合。

🛠 **方法**：提出名为NervePool的池化层，基于顶点分区构建单纯复形的层次化表示，通过学习顶点聚类分配，并以确定性方式扩展到高维单纯形的粗化，实际计算通过矩阵运算实现。

✨ **主要贡献**：首次为单纯复形结构数据定义了池化操作，其拓扑动机基于单纯形星形并集和神经复形的集合论构造，能灵活建模高阶关系并实现层次化学习。
