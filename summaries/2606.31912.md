# Learning Locomotion on Discrete Terrain via Minimal Proximity Sensing

- arXiv: https://arxiv.org/abs/2606.31912v1
- arXiv ID: 2606.31912

---

🔍 **核心问题**：解决四足机器人在离散地形（如间隙、踏脚石）上行走时，因缺乏对即将接触地面的预知能力而导致的鲁棒性不足问题，传统视觉感知存在延迟、遮挡和高计算成本，而本体感觉仅能事后反应。

🛠 **方法**：在四足机器人脚底嵌入低成本、高频率的红外接近传感器，提供“预接触”反馈，并结合强化学习框架，使机器人能提前感知地形不连续性，在仿真中训练后迁移到真实环境。

✨ **主要贡献**：首次证明稀疏的近场接近传感可有效替代或补充复杂全局视觉系统，显著提升机器人在离散地形上的穿越鲁棒性，且具有低功耗、低延迟的优势。
