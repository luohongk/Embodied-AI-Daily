# A2RL V\textsubscript{max}: The A2RL autonomous racing dataset for long-range, high-speed perception and multi-vehicle interaction

- arXiv: https://arxiv.org/abs/2607.17813v1
- arXiv ID: 2607.17813

---

🔍 **核心问题**：现有自动驾驶数据集多聚焦于结构化城市环境，缺乏针对高速自动驾驶和多车交互场景的感知数据集。  
🛠 **方法**：在2024年阿布扎比自动驾驶赛车联盟（A2RL）的亚斯码头F1赛道采集数据，包含单车、多车及四车决赛场景，提供近3万帧专业标注的LiDAR点云及RADAR点云，并采用开发者友好格式。  
✨ **主要贡献**：首次发布大规模自动驾驶赛车专业标注LiDAR点云数据集，支持基于深度学习的感知研究；提供现成3D检测与跟踪方法的基线评估，并指出高速场景需专用算法。
