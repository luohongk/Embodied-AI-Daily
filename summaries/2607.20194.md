# OLEDLM: A Unified Language Model for OLED Molecular Design

- arXiv: https://arxiv.org/abs/2607.20194v1
- arXiv ID: 2607.20194

---

🔍 **核心问题**：有机发光二极管（OLED）材料设计面临化学空间巨大、量子化学约束严格以及标注数据稀缺的挑战，现有模型难以高效生成满足特定光电性能的分子。

🛠 **方法**：提出基于因果语言模型的逆分子设计框架，采用多阶段策略：首先用LLaMA风格Transformer构建OLED专用基础化学语言模型，然后基于大规模OLED数据集预训练BERT属性预测器，最后通过强化学习优化SMILES生成。

✨ **主要贡献**：首次成功将大语言模型适配至OLED领域，实现从目标光电性质直接生成有效SMILES序列；通过DFT验证表明框架能高效探索OLED化学空间，生成结构有效且性能优化的新型候选分子。
