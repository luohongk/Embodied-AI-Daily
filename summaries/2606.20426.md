# TaCauchy: An Extensible FEM Framework for Vision-Based Tactile Simulation

- arXiv: https://arxiv.org/abs/2606.20426v1
- arXiv ID: 2606.20426

---

🔍 **核心问题**：现有视觉触觉传感器仿真方法难以在GPU加速的机器人平台上提供精确的力学应力场，限制了强化学习等下游任务的应用。

🛠 **方法**：提出TaCauchy框架，基于统一增量势接触求解器，通过超弹性本构方程直接计算柯西应力张量，并投影到接触面获取牵引力和压力分布，同时集成自动网格生成与模块化传感器接口。

✨ **主要贡献**：实现了33.40 FPS的单环境仿真与555 FPS的60并行环境吞吐量，应力提取开销低于1毫秒；在1.2556 N至4.7332 N力范围内，仿真与真实触觉响应的结构相似性指数超过0.93，验证了其提供精确物理力监督的能力。
