# FairCoder: Probing LLM Bias in High-Stakes Decision Making via Coding Tasks

- arXiv: https://arxiv.org/abs/2501.05396v3
- arXiv ID: 2501.05396

---

🔍 **核心问题**：大型语言模型在高风险决策（如招聘、大学录取）中可能隐含社会偏见，而现有方法难以检测通过代码推理过程泄露的隐性偏见。

🛠 **方法**：提出FairCoder基准，将决策任务转化为编码任务，覆盖就业、教育和医疗领域及多种公平性定义；并设计FairScore指标，同时捕捉模型拒绝回答行为和群体层面的结果多样性。

✨ **主要贡献**：通过实验揭示了LLM中一致且此前未被充分探索的偏见模式（如优先录取高收入家庭申请者），并提供了评估LLM作为决策代理风险的综合性框架。
