# Environmental Sound Deepfake Detection Using Deep-Learning Framework

- arXiv: https://arxiv.org/abs/2604.19652v3
- arXiv ID: 2604.19652

---

🔍 **核心问题**：解决环境声音深度伪造检测（ESDD）问题，即识别输入音频中的声音场景和声音事件是真实还是伪造的。

🛠 **方法**：提出一种深度学习框架，通过实验比较不同频谱图、网络架构和预训练模型的效果，并采用两阶段训练策略微调预训练BEATs模型。

✨ **主要贡献**：发现声音场景和声音事件的深度伪造检测应作为独立任务处理，微调预训练模型比从头训练更有效；最佳模型在EnvSDD测试集上达到Accuracy 0.98、F1 0.95、AUC 0.99，在跨数据集ESD-Challenge-TestSet上达到Accuracy 0.86、F1 0.80、AUC 0.93。
