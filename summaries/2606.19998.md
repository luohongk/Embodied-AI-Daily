# Tri-Info: Generalizable, Interpretable Failure Prediction for VLA Models via Information Theory

- arXiv: https://arxiv.org/abs/2606.19998v1
- arXiv ID: 2606.19998

---

🔍 **核心问题**：解决视觉-语言-动作（VLA）模型在物理交互中因黑箱特性导致的不可逆危害，需要一种可泛化且可解释的失败检测方法。

🛠 **方法**：基于信息论，将VLA控制形式化为闭环信息管道，推导出三重信息论信号（Tri-Info），用于捕捉动作多样性、时间一致性和与状态转移的耦合性。

✨ **主要贡献**：Tri-Info在域内匹配最强基线，且无需重新训练即可跨架构、环境和模拟到现实迁移，在真实任务中达到83%准确率（此前方法失效），同时提供可解释的失败模式诊断。
