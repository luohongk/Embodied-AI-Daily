# Compositional Skill Routing for LLM Agents: Decompose, Retrieve, and Compose

- arXiv: https://arxiv.org/abs/2606.18051v1
- arXiv ID: 2606.18051

---

🔍 **核心问题**：解决LLM代理在复杂任务中需要组合多个外部技能（工具）时，如何将用户查询分解为原子子任务、为每个子任务检索合适技能并生成可执行计划的问题。

🛠 **方法**：提出SkillWeaver框架，包含三个模块：LLM任务分解器（将查询分解为子任务）、基于双编码器和FAISS索引的技能检索器（为子任务匹配技能）、依赖感知的DAG规划器（组合技能为可执行计划）。针对分解质量瓶颈，进一步提出迭代技能感知分解（SAD），通过检索增强反馈循环逐步对齐分解与可用技能。

✨ **主要贡献**：构建了包含300个组合查询和2209个真实MCP服务器技能的基准CompSkillBench；发现任务分解质量是主要瓶颈（标准LLM分解步骤级类别召回率仅34.2%）；SAD将分解准确率从51.0%提升至67.7%（+32.7%）；SkillWeaver减少上下文窗口消耗超过99%，且跨领域迁移实验验证了泛化能力（目标类别不在检索池中时相对DA增益仍达35.6%）。
