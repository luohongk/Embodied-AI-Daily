# GNet: A scalable and flexible Gaussian process network with nonparametric neurons

- arXiv: https://arxiv.org/abs/2607.10735v1
- arXiv ID: 2607.10735

---

🔍 **核心问题**：解决传统高斯过程模型在大规模数据和高维输入场景下计算与存储成本过高的问题。  
🛠 **方法**：提出GNet框架，使用高斯过程作为非参数神经元，并引入联合逆卡尔曼滤波算法，避免显式构建协方差矩阵，同时提供梯度闭式解以加速训练与预测。  
✨ **主要贡献**：在非线性函数预测、真实数据非参数回归及高维物理建模（如经典密度泛函理论）中表现优异，显著降低计算与存储开销，具备良好的可扩展性与灵活性。
