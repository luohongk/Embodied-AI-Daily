# Time-Lag-Aware Deep Reinforcement Learning for Flexible Job-Shop Scheduling in PPVC Module Factories

- arXiv: https://arxiv.org/abs/2607.11725v1
- arXiv ID: 2607.11725

---

🔍 **核心问题**：在预制装配式建筑模块工厂的柔性作业车间调度中，混凝土养护、防水测试和油漆干燥等工序导致长时间滞后，使模块被阻塞而工作站空闲，传统调度方法忽略此问题会显著降低效率。

🛠 **方法**：基于最先进的双注意力深度强化学习求解器，提出三种可单独消融的扩展：滞后感知动态模型（含可接受奖励边界）、两个预测性滞后特征通道，以及活性掩码的操作与工位类型嵌入。

✨ **主要贡献**：在基于国家预制指南的基准实例上，学习策略达到约束规划参考解的约4%以内，优于所有调度规则和遗传算法元启发式，且能在容量竞争加剧时保持优势；无需求解器或模型即可在数秒内重新调度，并明确标出精确求解器作为质量上限的边界。
