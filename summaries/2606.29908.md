# Pondering the Way: Spatial-perceiving World Action Model for Embodied Navigation

- arXiv: https://arxiv.org/abs/2606.29908v1
- arXiv ID: 2606.29908

---

🔍 **核心问题**：现有基于世界模型的视觉导航规划器采用验证中心范式，存在候选路径依赖、计算开销大、采样动作与预测视觉不一致等问题。

🛠 **方法**：提出SWAM（空间感知世界动作模型），这是一个任务中心的联合观测-动作生成框架。它通过单次推理同时生成中间RGB-D序列和对应动作轨迹，并引入视觉引导动作精炼模块和轨迹尺度正则化损失。

✨ **主要贡献**：在成功率、轨迹精度和推理效率上显著超越现有最优两阶段规划器，且仅需单目RGB输入即可实现零样本泛化到未知环境。
