# PRO-LONG: Programmatic Memory Enables Long-Horizon Reasoning

- arXiv: https://arxiv.org/abs/2607.20064v1
- arXiv ID: 2607.20064

---

🔍 **核心问题**：解决大语言模型（LLM）在长周期任务中因观测序列过长而难以有效管理上下文信息、导致推理和探索能力受限的问题。

🛠 **方法**：提出PRO-LONG框架，通过程序化内存（programmatic memory）维护完整的结构化交互日志，并利用编码智能体高效搜索历史信息，从而在保留全部信息的同时提升检索效率。

✨ **主要贡献**：在ARC-AGI-3基准上，PRO-LONG使基础编码智能体平均提升18.0个百分点，以4.2-5.8倍更少的token达到或超越现有最优方法（最高76.1% pass@1）；在Fable 5任务中以97.4% best@2和仅$1,750成本实现卓越性能。
