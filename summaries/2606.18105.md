# OmniPlan: An Adaptive Framework for Timely and Near-Optimal Network Planning Optimization

- arXiv: https://arxiv.org/abs/2606.18105v1
- arXiv ID: 2606.18105

---

🔍 **核心问题**：现有网络规划优化框架无法有效适应多样化和动态的用户意图，导致执行时间与最优性之间存在权衡。

🛠 **方法**：提出OmniPlan框架，采用基于大语言模型的解释器将自然语言意图转化为统一偏好向量，通过混合专家架构动态选择MIP求解器、启发式算法和深度强化学习模型，并利用DRL模块微调优化权重以对齐用户偏好。

✨ **主要贡献**：在真实分布式机器学习推理任务测试中，OmniPlan实现了近最优且低延迟的卸载决策，延迟降低高达97.8%，网络设备资源消耗减少11.5%。
