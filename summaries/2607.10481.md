# ARMOR: Stabilizing On-Policy LLM RL with Off-Policy Anchor Samples

- arXiv: https://arxiv.org/abs/2607.10481v2
- arXiv ID: 2607.10481

---

🔍 **核心问题**：解决在线策略强化学习训练大语言模型时因“过度优化”导致的训练不稳定和泛化能力下降问题，传统反向KL正则化不足以抑制这种退化。

🛠 **方法**：提出ARMOR框架，包含两个组件：一是“锚点采样”，利用参考策略的离线数据保留已有解题模式；二是“混合优化”，重构策略目标函数以实现受控探索，无需额外辅助损失。

✨ **主要贡献**：有效缓解验证集性能崩溃，在推理基准测试中实现长期训练的持续性能提升，将强化学习范式从被动惩罚转向主动样本稳定。
