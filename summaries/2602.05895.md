# Residual Reinforcement Learning for Waste-Container Lifting Using Large-Scale Cranes with Underactuated Tools

- arXiv: https://arxiv.org/abs/2602.05895v1
- arXiv ID: 2602.05895

---

🔍 **核心问题**：解决城市环境中大型液压起重机在抓取和提升垃圾容器时，因欠驱动工具与容器挂钩间几何公差严格，导致轨迹跟踪精度不足和摆动难以抑制的问题。

🛠 **方法**：提出残差强化学习（RRL）方法，将标称笛卡尔控制器（含导纳控制轨迹跟踪与摆锤感知阻尼）与PPO训练的残差策略结合，在Isaac Lab仿真中通过随机初始化、域随机化（负载、执行器增益等）增强泛化能力。

✨ **主要贡献**：相比纯标称控制器，该方法显著提升了轨迹跟踪精度、减少了振荡，并提高了垃圾容器提升的成功率，且无需从头进行端到端学习。
