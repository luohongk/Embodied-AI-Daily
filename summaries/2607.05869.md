# GraspIT: A Dataset Bridging the Sim-to-Real gap and back for Validated Grasping SE(3) Pose Generation

- arXiv: https://arxiv.org/abs/2607.05869v1
- arXiv ID: 2607.05869

---

🔍 **核心问题**：现有数据集无法同时提供逼真的RGB-D观测、物理验证的抓取质量标注以及仿真与真实世界之间的可靠桥梁，导致机器人抓取泛化能力受限。

🛠 **方法**：在NVIDIA Isaac Sim中构建桌面场景，通过四阶段物理滑移测试对Franka Panda机械臂的抓取候选进行标注，生成轨迹可达性检查和连续质量分数，并利用Real↔Sim循环将标注反投影到100个真实场景中。

✨ **主要贡献**：发布了包含约31.6万标注RGB-D帧（覆盖1035个仿真和100个真实场景）的数据集，提供实例掩码、6-DoF姿态、物理属性及评分抓取，其中83%的候选通过“良好”阈值，17%的硬负例用于提升模型鲁棒性，所有工具开源并支持容器化部署。
