# Mitigating Covariate Shift in Imitation Learning for Autonomous Vehicles Using Latent Space Generative World Models

- arXiv: https://arxiv.org/abs/2409.16663v5
- arXiv ID: 2409.16663

---

🔍 **核心问题**：解决自动驾驶模仿学习中因训练与测试环境分布差异导致的协变量偏移问题，避免策略在偏离训练轨迹时失效。  
🛠 **方法**：利用潜在空间生成世界模型（World Model）预测下一状态，并在端到端训练中让策略学习从错误中恢复；同时提出基于Transformer的多视角交叉注意力感知编码器。  
✨ **主要贡献**：在CARLA和DRIVE Sim仿真器中，闭环测试性能显著超越现有方法，并能有效处理外部扰动，且无需大量训练数据。
