# A Continual Validation, Updating, and Decision-Making Framework for Self-Adaptive Digital Twins via Robust Model Predictive Control: A Case Study in Additive Manufacturing

- arXiv: https://arxiv.org/abs/2607.18164v1
- arXiv ID: 2607.18164

---

🔍 **核心问题**：数字孪生中的替代模型因运行条件变化而出现概念漂移，导致预测性能下降，且现有自适应框架缺乏有效的漂移检测、高效模型更新及更新效果验证机制。

🛠 **方法**：提出自适应数字孪生框架，集成基于Fisher分数的多变量漂移检测器、低秩适配（LoRA）参数高效持续学习，以及Mann-Whitney U检验用于在线统计验证。

✨ **主要贡献**：在随机线性系统和定向能量沉积增材制造案例中，该框架能快速检测分布偏移，通过微调少于1%的模型参数恢复预测精度与不确定性量化，为神经网络数字孪生在全生命周期内保持可信性提供了统计严谨且计算可行的方案。
