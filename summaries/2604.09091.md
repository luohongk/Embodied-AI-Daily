# Synthesizing real-world distributions from high-dimensional Gaussian Noise with Fully Connected Neural Network

- arXiv: https://arxiv.org/abs/2604.09091v2
- arXiv ID: 2604.09091

---

🔍 **核心问题**：如何高效地从高维高斯噪声中生成与真实表格数据分布相似的合成数据，同时兼顾隐私保护与生成速度。  
🛠 **方法**：提出基于全连接神经网络的生成方法，结合表格数据预处理、分布建模、PCA降维，并设计基于Wasserstein距离与特征协方差的随机损失函数，以及随机成对误差减少损失函数。  
✨ **主要贡献**：在25个真实表格数据集上，该方法在相似性、隐私保护方面达到与最先进生成方法相当的水平，且MMD评分计算速度比现代深度学习方法快数个数量级。
