# The Count Is There, but Misaligned: Understanding and Correcting Counting Failures in VLMs

- arXiv: https://arxiv.org/abs/2607.09544v1
- arXiv ID: 2607.09544

---

🔍 **核心问题**：视觉语言模型（VLM）在物体计数任务中表现不佳，研究旨在探究这是源于内部知识缺失，还是内部表征与输出之间存在脱节。

🛠 **方法**：通过在四个VLM的激活层上训练简单探针，结合SVCCA分析和因果干预，验证模型内部编码了正确计数但输出错误；并提出基于检测器的自纠正方法，仅在内部错误检测器预测失败时重新提示模型。

✨ **主要贡献**：发现非线性探针能可靠检测计数错误，证明内部知识与输出之间存在方向错位；提出的推理时自纠正方法无需参数更新，即可将计数准确率提升最多15.6个百分点。
