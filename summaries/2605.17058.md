# Learning Multi-Timescale Abstractions for Hierarchical Combinatorial Planning

- arXiv: https://arxiv.org/abs/2605.17058v1
- arXiv ID: 2605.17058

---

🔍 **核心问题**：在序列随机组合优化问题中，强化学习面临动作空间指数级增长、随机动态和长时域决策的挑战，而分层强化学习的高层策略在SMDP中动作时长可变，导致难以学习适合规划的世界模型。

🛠 **方法**：提出一种基于模型的分层框架，结合潜在空间树搜索规划器与SMDP感知的世界模型，通过多时间尺度目标结构潜在动态，使状态转换幅度反映抽象动作的有效时间尺度，并联合学习子目标条件预算策略以支持上下文感知的资源分配。

✨ **主要贡献**：在多个SSCO基准测试中，该方法显著优于强基线，有效解决了可变时长动作下的高效前瞻规划与资源分配问题。
