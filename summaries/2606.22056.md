# Provably Efficient Policy-Reward Co-Pretraining for Adversarial Imitation Learning

- arXiv: https://arxiv.org/abs/2606.22056v1
- arXiv ID: 2606.22056

---

🔍 **核心问题**：对抗性模仿学习（AIL）虽优于行为克隆（BC），但需要大量在线环境交互，且缺乏对预训练在AIL中作用的严格理论分析。  
🛠 **方法**：提出CoPT-AIL，一种基于奖励塑形分析的策略-奖励联合预训练方法，通过单一BC过程同时预训练策略和奖励函数。  
✨ **主要贡献**：首次从理论上证明预训练可提升AIL的模仿差距界限，实验验证CoPT-AIL性能优于现有AIL方法。
