# Hybrid quantum-classical algorithm for near-optimal planning in POMDPs

- arXiv: https://arxiv.org/abs/2507.18606v1
- arXiv ID: 2507.18606

---

🔍 **核心问题**：在部分可观测环境（POMDP）中，如何利用量子计算加速基于模型的强化学习规划，实现近最优决策。

🛠 **方法**：提出混合量子-经典算法QBRL，结合量子拒绝采样与振幅放大技术，对稀疏贝叶斯网络进行高效推理，并显式建模推理过程而非依赖黑箱量子预言机。

✨ **主要贡献**：在稀疏动态环境下，量子增强信念更新使近最优规划的计算速度达到亚二次加速；通过数值实验对比经典算法，揭示了量子优势在不同部署场景下的显著差异。
