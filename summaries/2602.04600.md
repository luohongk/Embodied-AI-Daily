# Act, Sense, Act: Learning Active Perception from Large-Scale Egocentric Human Data

- arXiv: https://arxiv.org/abs/2602.04600v2
- arXiv ID: 2602.04600

---

🔍 **核心问题**：解决机器人在非结构化环境中主动感知能力不足的问题，现有方法局限于有限的感知行为类型，难以适应复杂环境。

🛠 **方法**：提出CoMe-VLA框架，利用大规模人类第一人称数据学习探索与操作先验，集成认知辅助头实现子任务自主切换，并设计双轨记忆系统融合本体感知与视觉时序信息，在统一的第一人称动作空间中分三阶段训练模型。

✨ **主要贡献**：形式化主动感知为历史依赖的感知-动作循环，提出认知与记忆增强的视觉-语言-动作框架，在轮式人形机器人上验证了跨多种主动感知场景的长时域任务中具有强鲁棒性和适应性。
