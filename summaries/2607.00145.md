# Iterated Invariant EKF for 3D Landmark-Aided Inertial Navigation

- arXiv: https://arxiv.org/abs/2607.00145v1
- arXiv ID: 2607.00145

---

🔍 **核心问题**：解决三维地标辅助惯性导航中经典SO(3)-EKF存在的虚假可观性问题，即滤波器在不可观方向上过度自信导致估计性能下降。  
🛠 **方法**：首次将迭代不变扩展卡尔曼滤波（IterIEKF）应用于基于地标的3D惯性定位，通过确保低噪声条件下状态保持在观测流形上、不确定性限制在切空间内，改进IEKF的测量更新兼容性。  
✨ **主要贡献**：数值仿真表明，IterIEKF在估计精度和一致性上均优于经典SO(3)-EKF、迭代SO(3)-EKF和IEKF。
