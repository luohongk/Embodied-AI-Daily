# Disentangled Unsupervised Skill Discovery for Efficient Hierarchical Reinforcement Learning

- arXiv: https://arxiv.org/abs/2410.11251v2
- arXiv ID: 2410.11251

---

🔍 **核心问题**：现有无监督技能发现方法常学习到纠缠的技能，即一个技能变量同时影响环境中多个实体，导致下游技能链式组合极其困难。

🛠 **方法**：提出DUSDi方法，将技能分解为可解耦的组件，每个组件仅影响状态空间的一个因子，通过基于互信息的优化目标和值函数分解技术，实现组件间影响的解耦。

✨ **主要贡献**：在多个挑战性环境中，DUSDi成功学习到解耦技能，并在下游任务中显著优于以往技能发现方法，支持技能组件的并发组合与高效链式复用。
