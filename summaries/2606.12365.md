## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人模仿学习，特别是如何利用次优（suboptimal）或分布外（out-of-distribution, OOD）的数据来训练策略。
- **问题（Problem）**：高质量、任务特定的机器人数据昂贵且难以获取，而次优数据（如失败轨迹、仿真数据、不同任务的数据）虽然丰富，但直接用于训练会引入偏差，损害策略性能。现有方法（如数据过滤、协同训练）要么浪费数据，要么无法有效区分次优数据中的有用和有害特征。
- **方法（Method）**：提出一种名为“环境扩散策略”（Ambient Diffusion Policy）的简单且原则性的方法。其核心思想是在训练扩散策略（Diffusion Policy）时，根据噪声水平（diffusion time）来控制次优数据的使用：只允许次优数据在**高噪声**（学习全局规划）和**低噪声**（学习局部运动基元）的时间段参与训练，而在中间噪声水平（学习精细动作）时排除它们。
- **创新（Innovation）**：
    1.  首次将“噪声依赖的数据使用”（noise-dependent data usage）这一新维度引入机器人协同训练。
    2.  通过理论和实验证明，机器人动作数据具有**频谱幂律（spectral power law）** 特性，这导致了扩散策略的**全局到局部层次结构（global-to-local hierarchy）** 和**局部性（locality）**，为所提方法提供了坚实的理论基础。
    3.  方法极其简单，只需修改扩散策略的数据采样器（data sampler）即可实现。
- **意义（Significance）**：该方法极大地扩展了机器人模仿学习中可用数据源的范围，能够从任意来源的次优数据中提取有用信息，从而在数据稀缺的现实场景下显著提升策略性能。在Open X-Embodiment大规模数据集上，该方法相比基线方法性能提升高达33%。
- **一句话总结**：本文提出环境扩散策略，通过利用机器人动作数据的频谱特性，在扩散过程的不同噪声阶段有选择性地使用次优数据，从而高效、鲁棒地学习高质量策略。

## 1. 研究动机（Motivation）

- **问题与空白**：在机器人领域，收集高质量、任务特定的专家演示数据成本高昂（需要熟练的操作员和精调的低级控制器）。然而，次优数据（如非专家操作、仿真数据、不同任务或不同形态机器人的数据）却非常丰富。如何有效地利用这些海量但质量参差不齐的数据来提升策略性能，是当前机器人学习中的一个关键挑战和未被充分探索的领域。
- **为什么重要**：如果能有效利用次优数据，将大大降低机器人数据收集的门槛和成本，加速机器人技能的泛化和规模化学习。这对于构建通用机器人基础模型至关重要。
- **挑战在哪里**：
    1.  **数据浪费**：简单丢弃低质量数据（数据过滤）会浪费其中包含的有用学习信号。
    2.  **策略偏差**：将所有数据混合训练（协同训练）会使策略学习到次优数据中的有害特征（如抖动、错误逻辑），从而产生偏差。
    3.  **特征混淆**：次优数据中，有用的特征（如高层规划）和有害的特征（如低层抖动）是混杂在一起的，现有方法难以将它们分离。

## 2. 创新点（Innovation）

- **关键创新**：提出了**噪声依赖的数据使用（noise-dependent data usage）** 这一新范式。具体来说，就是利用扩散模型的特性，让次优数据**只在特定的噪声时间段**参与训练。
- **与现有方案的不同**：
    - **数据过滤**：完全丢弃次优数据。本文方法则保留并利用它们。
    - **协同训练**：在所有噪声水平上平等地使用所有数据。本文方法则根据噪声水平有选择性地使用。
    - **微调**：先在大数据上预训练，再在小的高质量数据上微调。本文方法可以与微调互补，且实验表明，基于本文方法训练的基座策略进行微调，效果优于基于协同训练的策略进行微调。
- **为什么能奏效**：
    1.  **理论基础**：论文发现机器人动作数据具有**频谱幂律**特性。这意味着动作信号的能量主要集中在低频（全局规划），高频（局部抖动、精细动作）能量较低。
    2.  **扩散过程的层次性**：由于频谱幂律，扩散策略在**高噪声**阶段主要学习**全局**、低频的规划信息；在**低噪声**阶段主要学习**局部**、高频的运动基元。这被称为“全局到局部层次结构”。
    3.  **局部性**：在低噪声阶段，最优去噪器具有“局部性”，即预测某个动作时，主要依赖其时间上的邻近动作，而非全局信息。
    4.  **方法设计**：基于以上两点，如果次优数据与目标数据在全局规划上一致（如都想去同一个目标），但在局部动作上存在差异（如抖动），那么就可以让次优数据只在**高噪声**阶段（学习全局规划）参与训练。反之，如果次优数据在局部运动基元上有用（如抓取动作），但全局任务逻辑错误，那么就可以让次优数据只在**低噪声**阶段（学习局部运动基元）参与训练。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

环境扩散策略（Ambient Diffusion Policy）包含两个阶段：

1.  **阶段一：数据标注（Phase 1: Dataset Annotation）**
    - 为次优数据集 $D_q$ 中的样本标注两个关键噪声时间参数：$t_{min}$ 和 $t_{max}$。
    - **$t_{min}$ 的确定**：找到最小的噪声水平 $\sigma_{t_{min}}$，使得加噪后的次优分布 $q_t$ 和目标分布 $p_t$ 变得难以区分。论文提出使用一个**分类器** $c_\phi(A_t, t)$ 来学习区分 $p_t$ 和 $q_t$。$t_{min}$ 就是分类器无法可靠区分两者时的最小噪声水平。
    - **$t_{max}$ 的确定**：找到最大的噪声水平 $\sigma_{t_{max}}$，使得在低噪声下，次优数据和目标数据在局部特征上一致。论文在实验中通过超参数搜索来确定 $t_{max}$。

2.  **阶段二：训练（Phase 2: Training）**
    - 训练过程与标准的扩散策略（Diffusion Policy）几乎相同，唯一的修改是**自定义数据采样器（custom data sampler）**。
    - 采样器首先随机采样一个扩散时间 $t \sim U[0, T]$。
    - 然后，从数据集中抽取样本：
        - 来自目标数据集 $D_p$ 的样本**始终**可以用于训练。
        - 来自次优数据集 $D_q$ 的样本**只有**在 $t \in [0, t_{max}) \cup (t_{min}, T]$ 时才被允许用于训练。
    - 推理过程与标准扩散策略完全相同。

**图1** 直观地展示了训练算法：高质量数据在所有扩散时间都可使用，而次优数据只在高噪声（$t > t_{min}$）和低噪声（$t < t_{max}$）区间使用。

### 3.2 关键算法与数学推导

- **扩散策略基础**：
    学习一个去噪器 $h_\theta(A_t, O, t)$ 来预测干净动作 $A_0$，其中 $A_t = A_0 + \sigma(t)Z, Z \sim \mathcal{N}(0, I)$。
    损失函数为：
    $$L(\theta) = \mathbb{E}_{t, (O, A_0), A_t|A_0} \left[ \| h_\theta(A_t, O, t) - A_0 \|^2 \right]$$
    最优去噪器是条件期望 $h^*_\theta(A_t, O, t) = \mathbb{E}[A_0 | A_t, O]$。

- **协同训练基线**：
    $$L_{\text{co-training}}(\theta) = \alpha L_{D_p}(\theta) + (1-\alpha) L_{D_q}(\theta)$$
    其中 $\alpha$ 是权重参数。

- **环境扩散策略的核心思想**：
    利用噪声作为“收缩机制”。对于任意分布 $p$ 和 $q$，其加噪后的分布 $p_t$ 和 $q_t$ 之间的总变差距离满足：
    $$d_{TV}(p_t, q_t) \le d_{TV}(p, q) \cdot \frac{D}{2\sigma_t}$$
    这意味着随着噪声 $\sigma_t$ 增大，两个分布会变得难以区分。

- **定理1（频谱幂律导致快速收缩）**：
    假设 $p_0$ 和 $q_0$ 是零均值平稳高斯分布，其功率谱在低频段一致，在高频段遵循幂律衰减（$S(f) \le C f^{-\alpha}$）。那么，当 $\sigma_t^2 \ge C (f^\star)^{-\alpha}$ 时，加噪后的分布 $p_t$ 和 $q_t$ 之间的KL散度和TV距离会以 $1/\sigma_t^4$ 和 $1/\sigma_t^2$ 的速度快速收缩，与原始分布的距离无关。
    $$d_{KL}(p_t \| q_t) \le \frac{4C^2}{(2\alpha-1)\sigma_t^4 (f^\star)^{2\alpha-1}}$$
    这解释了为什么 $t_{min}$ 可以很小，即次优数据可以很快地在高噪声阶段被安全使用。

- **定理2（频谱幂律导致局部性）**：
    假设 $X_0$ 是遵循频谱幂律的零均值平稳高斯分布。其最优去噪器 $h^\star_t(x) = \mathbb{E}[X_0 | X_t = x]$ 具有局部性。即，对于输入 $x$ 的局部掩码 $M_i$（只保留索引 $i$ 附近 $L$ 个邻域的值），去噪器在索引 $i$ 处的输出误差上界为：
    $$| (h^\star_t(x))_i - (h^\star_t(M_i x))_i | \le \frac{\alpha N \|x\|_\infty}{8L}$$
    这意味着在低噪声下，最优去噪器可以仅通过观察局部邻域来重建信号，而忽略全局结构。这解释了为什么可以在低噪声阶段安全地使用任务逻辑错误的次优数据来学习局部运动基元。

### 3.3 模型、训练与数据集

- **模型**：基于扩散策略（Diffusion Policy），使用条件去噪过程在动作空间进行采样。
- **训练**：训练过程与标准扩散策略一致，核心区别在于数据采样器。采样器根据扩散时间 $t$ 决定是否从次优数据集中采样。
- **数据集**：
    - **目标数据集 $D_p$**：高质量、任务特定的专家演示。
    - **次优数据集 $D_q$**：包含多种类型的次优数据：
        1.  **噪声轨迹**：使用RRT（快速随机搜索树）算法生成的抖动轨迹。
        2.  **仿真到现实差距**：在仿真环境中由运动规划器生成的轨迹。
        3.  **任务不匹配**：执行错误任务（如反向排序）的演示。
        4.  **大规模数据混合**：Open X-Embodiment (OXE) 数据集，包含70多个子数据集，质量和分布差异巨大。

### 3.4 实验设置与结果

    - **基线**：数据过滤（Data Filtering）、协同训练（Co-training）、微调（Finetuning）。
    - **结果**：

        - **仿真到现实差距**：环境扩散策略在所有变体上都优于协同训练（**表2**）。
        - **任务不匹配**：只有环境扩散策略（利用局部性）能同时完美地学习运动技能和任务逻辑，而协同训练则完全失败（**表3**）。**图7**展示了通过调整 $t_{max}$ 可以完美平衡逻辑和运动指标。
        - **微调比较**：基于环境扩散策略的基座模型进行微调，效果始终优于基于协同训练的基座模型（**图8**）。

- **大规模实验（Section 8）**：
    - **任务**：真实机器人上的桌面清理和搭积木。
    - **数据集**：$D_p$ 为少量任务演示，$D_q$ 为OXE数据集的两个子集（MS++和COXE）。
    - **结果**：
        - **桌面清理**：环境扩散策略在 $D_p$ 只有50个演示时，比协同训练高出12-15%；当 $D_p$ 增加到150个演示时，优势依然存在（**图9**）。
        - **搭积木**：环境扩散策略搭建的塔比数据过滤和协同训练分别高84%和33%（**图9**）。
        - **消融实验**：环境扩散策略对数据集重加权（re-weighting）的敏感性远低于协同训练。协同训练如果不进行重加权，策略会变得不安全而无法评估（**图10**）。

### 3.5 技术细节

- **频谱幂律的验证**：论文通过计算OXE数据集中动作的功率谱密度（PSD），发现其随频率的衰减符合幂律 $|f|^{-\alpha}$，$\alpha \approx 1.47$（**图3a**）。
- **局部性的验证**：通过可视化去噪器对输入动作的敏感性，发现低噪声下，对第8个动作的预测主要受其时间邻近动作的影响（**图3b**）。
- **$t_{min}$ 的自动标注**：训练一个分类器来区分 $p_t$ 和 $q_t$，$t_{min}$ 是分类器置信度低于阈值（如0.5+τ）时的最小噪声水平。
- **实现**：算法实现非常简单，只需修改扩散策略的数据加载器（data loader）中的采样逻辑。论文提供了伪代码（附录E.1）。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了机器人学习中的一个核心瓶颈**：如何有效利用海量但质量参差不齐的数据。该方法提供了一种简单、通用且理论扎实的解决方案。
    2.  **提出了新范式**：将“何时使用数据”这一新维度引入机器人协同训练，超越了传统的“使用哪些数据”和“如何加权”的思考方式。
    3.  **理论贡献**：证明了机器人动作数据的频谱幂律特性，并以此为基础推导出扩散策略的层次性和局部性，为环境扩散框架在机器人领域的应用提供了坚实的理论基础。
    4.  **实践价值**：在多种次优数据类型和大规模真实数据集上验证了方法的优越性，证明了其强大的泛化能力和实用性。

- **对未来研究/应用的影响**：
    - 该方法有望成为机器人基础模型预训练的标准组件，使得可以利用几乎所有可获取的数据进行训练。
    - 为数据质量评估和自动标注提供了新思路（如通过分类器自动确定 $t_{min}$）。
    - 理论部分为理解扩散模型在结构化数据（如机器人动作序列）上的行为提供了深刻见解。

- **局限与开放问题**：
    1.  **观测空间分布偏移**：当前方法未处理观测空间（如图像）的分布偏移。
    2.  **标注方法**：基于分类器的 $t_{min}$ 标注方法虽然理论扎实，但有时不如超参数搜索效果好。更好的标注方法有待探索。
    3.  **理论局限性**：理论部分仅限于高斯分布，扩展到更一般的分布是未来工作。
    4.  **计算成本**：由于使用了更多数据，计算成本高于数据过滤，但收益通常值得。
    5.  **数据划分**：如何最优地将数据划分为 $D_p$ 和 $D_q$ 是一个重要的设计决策，需要根据具体目标（任务特定 vs. 通用预训练）来确定。

## 5. 澄清与简化（Clarifications and Simplifications）

- **频谱幂律（Spectral Power Law）**：可以想象一首歌。低频部分（如鼓点）构成了歌曲的节奏和骨架（全局规划），而高频部分（如镲片声）则增添了细节和质感（局部动作）。机器人动作数据也是如此，其“骨架”（如从A点到B点的路径）的能量远大于“细节”（如路径上的微小抖动）。这个特性使得我们可以通过添加噪声来“抹去”细节，只保留骨架。

- **扩散过程的层次性**：想象一个画家画画。他首先会用大刷子画出物体的轮廓（高噪声，学习全局规划），然后再用小刷子精修细节（低噪声，学习局部动作）。环境扩散策略就是让次优数据只参与“画轮廓”或“精修细节”的阶段，而不参与“画中间层次”的阶段，从而避免学习到次优数据中不想要的“笔触风格”（如抖动）。

- **局部性（Locality）**：想象你在拼图。如果你要拼出图中一个人的眼睛，你只需要看眼睛周围那一小块区域的碎片，而不需要看整张图。在低噪声下，扩散策略的去噪器也是如此，它预测某个动作时，主要看它前后几个动作，而不关心整个轨迹的全局目标。因此，即使次优数据的全局目标（如“把积木放到右边箱子”）是错误的，我们仍然可以利用它的局部动作（如“抓取积木”）来学习抓取技能。

- **$t_{min}$ 和 $t_{max}$
