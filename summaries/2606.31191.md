# ISM:Self-Improving Strategy Memory for Continual Mathematical Reasoning

- arXiv: https://arxiv.org/abs/2606.31191v1
- arXiv ID: 2606.31191

---

🔍 **核心问题**：在连续学习场景下，如何让冻结参数的LLM在严格隔离的数学推理任务中持续提升性能，避免灾难性遗忘。

🛠 **方法**：提出智能模式记忆（ISM）系统，通过维护一个自优化的策略模式库，从成功和失败案例中学习，并利用符号工具验证中间步骤和最终答案，无需更新模型参数。

✨ **主要贡献**：在MATH-Hard和OlympiadBench上，ISM以比最强被动基线少64%和86%的策略模式，超越被动、检索和反思基线，证明小规模、主动维护且经验证的策略记忆能支持可靠的连续数学推理。
