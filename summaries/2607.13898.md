# Jack of All Scales: A Versatile FPGA Tensor Block for MXFP Precisions

- arXiv: https://arxiv.org/abs/2607.13898v1
- arXiv ID: 2607.13898

---

🔍 **核心问题**：当前FPGA的DSP模块对标准化MXFP低精度格式（如MXFP8、MXFP6、MXFP4）支持有限，无法高效实现所有精度类型的点积运算。

🛠 **方法**：提出对Altera Agilex-5 FPGA的DSP模块内部张量模式架构进行针对性修改，以原生支持所有MXFP精度，并通过开源ASAP7 PDK评估面积开销。

✨ **主要贡献**：修改后的DSP模块在仅增加36%的DSP单元面积（占FPGA总面积的1.8%）下，实现了对所有MXFP格式的全面支持，相比现有架构平均吞吐量提升4.2倍。
