# A cross-process welding penetration status prediction algorithm based on unsupervised domain adaptation in laser and TIG welding

- arXiv: https://arxiv.org/abs/2606.26078v1
- arXiv ID: 2606.26078

---

🔍 **核心问题**：解决不同焊接工艺（如TIG焊与激光焊）之间因物理机制差异导致的模型迁移性能下降问题，降低新工艺场景下的重新标注成本。

🛠 **方法**：提出一种基于无监督域适应（UDA）的框架，结合渐进式源域扩展（GSDE）策略，在无需目标域标签的情况下实现跨工艺焊接状态分类。

✨ **主要贡献**：在同工艺场景下平均准确率达90.65%和90.72%，跨工艺场景下达80.48%和81.13%，相比监督基线提升超35%；通过UMAP可视化验证模型学习到域不变特征并保持类别可分性。
