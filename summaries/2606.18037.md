# ProvenanceGuard: Source-Aware Factuality Verification for MCP-Based LLM Agents

- arXiv: https://arxiv.org/abs/2606.18037v1
- arXiv ID: 2606.18037

---

🔍 **核心问题**：针对基于MCP（模型上下文协议）的工具型LLM代理在回答时，可能出现“跨源混淆”错误——即答案虽被某来源支持，但被错误归因到其他来源，而现有事实性验证指标无法检测此类问题。

🛠 **方法**：提出ProvenanceGuard验证器，通过捕获MCP轨迹中的工具ID、来源ID和原始输出，将答案分解为原子声明，路由到特定来源进行NLI和令牌对齐验证，比较声明归属与路由来源，最终输出逐声明判决和答案级阻止/放行决策，并可结合检索增强修复后重新验证。

✨ **主要贡献**：在281条医疗领域MCP代理轨迹上，ProvenanceGuard在40条保留集上达到阻止F1值0.802和来源准确率0.858，显著优于无来源感知的基线；在50次临床混淆探测中成功检测所有注入的归因交换，证明来源归因是MCP代理事实性验证的独立维度。
