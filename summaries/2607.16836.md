# Beyond Storage: State as a Runtime Control Problem in Parallel and Distributed Systems

- arXiv: https://arxiv.org/abs/2607.16836v1
- arXiv ID: 2607.16836

---

🔍 **核心问题**：在流处理、服务、检索和持续学习等并行与分布式系统中，共享状态对性能和故障行为的影响日益显著，但现有研究往往将访问控制、硬件感知执行、内存管理和长期更新等问题割裂开来，缺乏统一视角。

🛠 **方法**：本文通过文献综述，围绕状态访问与调度、状态感知执行、状态演化与复用三个耦合维度，并采用“状态对象、控制面、耦合路径、评估边界、未解决契约”这一通用框架进行综合比较。

✨ **主要贡献**：识别出反复出现的反模式，提出了以契约为导向的设计蓝图和扰动感知的评估议程，最终将状态管理重新定义为运行时控制问题。
