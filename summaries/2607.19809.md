# Dreamer-CPC: Message Learning with World Models for Decentralized Multi-agent Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.19809v1
- arXiv ID: 2607.19809

---

🔍 **核心问题**：在部分可观测的多智能体强化学习中，现有基于表示学习的通信方法仅依赖当前观测，无法传递随时间累积的信息，导致智能体在观测缺失时协作失效。

🛠 **方法**：提出Dreamer-CPC，将基于集体预测编码（CPC）的消息学习集成到DreamerV3的世界模型中。每个智能体独立维护世界模型和消息模块，从反映历史观测与动作的潜在状态中推断并交换消息。

✨ **主要贡献**：在非协作信息共享任务Observer和新增的观测缺失任务CatchApple中，Dreamer-CPC均优于基于当前观测的IPPO-CPC和无通信基线。尤其在CatchApple中，其回合回报达到IPPO-CPC的4-5倍，证明基于世界模型潜在动态的通信能有效支持观测不足时的分散决策。
