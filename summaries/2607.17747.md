# Mobile Network Control with a World Model

- arXiv: https://arxiv.org/abs/2607.17747v1
- arXiv ID: 2607.17747

---

🔍 **核心问题**：移动网络日益复杂，需要智能且动态的控制策略以实现高效节能的管理，同时平衡服务质量与能耗。

🛠 **方法**：提出基于世界模型的方法，从历史数据中训练模型预测网络状态变化，控制器利用模型的不确定性估计鲁棒地调整关键参数，且优化目标可动态更改而无需重新训练。

✨ **主要贡献**：在模拟移动网络节能场景中，该方法相比传统方法和强化学习在节能与服务质量平衡上表现更优；并在真实网络数据上验证了世界模型性能，评估了不同吞吐量约束下的反事实动作。
