# Structured 4D Latent Predictive Model for Robot Planning

- arXiv: https://arxiv.org/abs/2607.01166v1
- arXiv ID: 2607.01166

---

🔍 **核心问题**：现有基于2D视频的机器人预测模型缺乏3D几何理解，导致空间推理和物理一致性不足，难以实现精准的长期规划。

🛠 **方法**：提出结构化4D潜在预测模型，在结构化潜在空间中预测场景3D结构的演化，并利用目标条件逆动力学模块将生成的未来场景转化为可执行动作。

✨ **主要贡献**：模型生成未来场景时具有强视觉质量、显著更优的3D一致性与多视角连贯性，在复杂操作任务上表现优越，且能泛化到新视觉条件并成功应用于真实机器人平台。
