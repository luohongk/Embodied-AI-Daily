# PAC-ACT: Post-training Actor-Critic for Action Chunking Transformers

- arXiv: https://arxiv.org/abs/2607.09590v1
- arXiv ID: 2607.09590

---

🔍 **核心问题**：在工业精密接触操作中，预训练的动作分块策略（如Action Chunking Transformer）因行为克隆导致分布偏移，难以应对位姿扰动和接触力约束。

🛠 **方法**：提出PAC-ACT，一种基于强化学习的后训练框架，在分块级别优化策略，构建ACT迁移的演员-评论家架构，并引入混合行为先验约束以保持预训练动作分布。

✨ **主要贡献**：在工业精密接触基准上，PAC-ACT显著提升任务成功率、接触稳定性和力安全性，同时保持低延迟和低GPU内存占用；在Contour任务中，峰值接触力降低，60 N以上力读数比例减少46倍；稀疏奖励实验证明行为先验约束能有效支持随机初始位姿下的探索。
