# CA-DGCL: Dynamic Graph Continual Learning via Condensation and Attachment

- arXiv: https://arxiv.org/abs/2607.11112v1
- arXiv ID: 2607.11112

---

🔍 **核心问题**：现有动态图持续学习方法未能充分利用图快照间的时间信息，导致灾难性遗忘问题。  
🛠 **方法**：提出CA-DGCL框架，先通过压缩历史图快照为紧凑语义表示，再构建跨时间节点链形成三阶张量并应用Tucker分解获取稳定节点特征，最后用这些特征生成新节点附加到当前图以回放历史信息。  
✨ **主要贡献**：引入改进的遗忘度量指标，实验证明该方法在抑制遗忘和保持竞争性准确率上优于基线，有效解决了动态图持续学习中的时间信息利用不足问题。
