# Improving path-tracking performance of an articulated tractor-trailer system using a non-linear kinematic model

- arXiv: https://arxiv.org/abs/2606.31889v1
- arXiv ID: 2606.31889

---

🔍 **核心问题**：解决铰接式拖拉机-拖车系统在路径跟踪任务中拖车位置偏差较大的问题，尤其针对精准农业中的小地块作业场景。

🛠 **方法**：提出一种新型非线性运动学模型，并将其嵌入非线性模型预测控制器（NMPC），结合后退时域技术，同时利用ROS框架和Gazebo模拟器进行仿真验证。

✨ **主要贡献**：通过考虑拖车运动学，显著降低了拖车在直线和地头转弯时的位置偏差；实现了对拖车位置（而非拖拉机位置）的直接控制，提升了路径跟踪精度。
