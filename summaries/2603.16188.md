# ECHO: Edge-Cloud Humanoid Orchestration for Language-to-Motion Control

- arXiv: https://arxiv.org/abs/2603.16188v1
- arXiv ID: 2603.16188

---

🔍 **核心问题**：如何实现基于自然语言指令的类人机器人全身运动控制，并解决云端生成与边缘执行之间的实时性与兼容性问题。

🛠 **方法**：提出ECHO框架，包含云端扩散模型（1D卷积UNet+CLIP文本编码）生成运动序列，以及边缘部署的强化学习跟踪器（教师-学生范式+证据适应模块）执行闭环控制，两者通过38维机器人原生运动表示桥接。

✨ **主要贡献**：在HumanML3D基准上达到FID 0.029、R-Precision Top-1 0.686的生成质量，并在Unitree G1机器人上零硬件调优实现多种文本指令的稳定执行，同时具备自主跌倒恢复能力。
