# Adaptive Human-AI Coordination via Hierarchical Action Disentanglement

- arXiv: https://arxiv.org/abs/2605.24343v1
- arXiv ID: 2605.24343

---

🔍 **核心问题**：现有的人机协作方法常陷入单一行为模式或技能对齐不良，难以适应多样化的合作伙伴行为与技能水平，尤其在面对未见过的合作伙伴时鲁棒性不足。

🛠 **方法**：提出内在动作解耦（IAD）框架，基于深度分层强化学习（DHRL），学习以高层潜在技能为条件的、区分于合作伙伴的低层动作序列，并通过内在奖励显式鼓励不同技能下的动作分布解耦。

✨ **主要贡献**：在Overcooked-AI环境中，IAD在多种布局及合作伙伴设置（包括未见过的模拟伙伴、基于人类游戏数据训练的代理模型及真实人类）下，均优于强基线方法，实现了更可靠、自适应的协调。
