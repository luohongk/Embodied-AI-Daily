# RedCoder: Automated Multi-Turn Red Teaming for Code LLMs

- arXiv: https://arxiv.org/abs/2507.22063v2
- arXiv ID: 2507.22063

---

🔍 **核心问题**：现有针对代码生成大模型的红队测试方法依赖大量人工，且忽略了真实编程场景中的多轮交互特性，难以高效、规模化地诱导模型生成漏洞代码。

🛠 **方法**：提出RedCoder，通过多智能体博弈模拟对抗交互，生成原型对话并构建可复用攻击策略库；随后微调LLM作为核心，使其能在多轮对话中动态检索策略，引导受害者模型输出易受攻击的代码。

✨ **主要贡献**：在多个代码LLM上实验表明，RedCoder在诱导漏洞生成方面显著优于现有单轮和多轮红队方法，为评估代码生成系统的安全边界提供了可扩展、高效的工具。
