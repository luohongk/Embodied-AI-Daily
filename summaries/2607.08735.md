# Learning Adaptive Solvers for Distributed Factor Graph Optimization on Matrix Lie Groups

- arXiv: https://arxiv.org/abs/2607.08735v1
- arXiv ID: 2607.08735

---

🔍 **核心问题**：现有分布式因子图优化求解器依赖手动调参，且主要针对刚体姿态图，难以适应一般矩阵李群上的大规模分布式几何优化问题。

🛠 **方法**：提出DeepCORD框架，通过将并行加速的黎曼优化器展开为可微迭代，学习自监督反馈策略，动态调整求解器参数以适应优化阶段和通信状态，支持同步与异步通信。

✨ **主要贡献**：在真实SE(3)姿态图优化和SL(4)投影子图对齐任务上，该方法在多数基准测试中取得比现有分布式基线更低的优化目标值，实现了矩阵李群上的自适应分布式优化。
