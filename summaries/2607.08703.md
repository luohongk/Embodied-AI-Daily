# MPFlow: Learning Budgeted Max-Flow Optimization on the Lightning Network with Deep Graph Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.08703v1
- arXiv ID: 2607.08703

---

🔍 **核心问题**：在比特币闪电网络中，给定固定预算，节点应选择开通哪些通道以最大化其路由容量（即最大流）。

🛠 **方法**：将该问题建模为图上的预算约束组合优化问题，通过深度图强化学习（结合消息传递策略网络、PPO算法和动作掩码）求解，并采用中心排除课程训练策略。

✨ **主要贡献**：在真实闪电网络快照上，该方法在最大流目标上持续优于强启发式基线，并已部署至生产环境，执行4640次通道开通决策，累计分配267.3 BTC（超1600万美元）。
