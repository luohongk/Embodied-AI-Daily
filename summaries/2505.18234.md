# A Robust PPO-optimized Tabular Transformer Framework for Intrusion Detection in Industrial IoT Systems

- arXiv: https://arxiv.org/abs/2505.18234v1
- arXiv ID: 2505.18234

---

🔍 **核心问题**：解决工业物联网（IIoT）环境中网络入侵检测面临的类别不平衡和少样本攻击（如中间人攻击）检测难题。

🛠 **方法**：提出一种结合TabTransformer（用于表格特征表示）与近端策略优化（PPO，用于策略学习优化分类决策）的强化学习增强型网络入侵检测系统。

✨ **主要贡献**：在TON_IoT基准上达到97.73%的宏F1分数和98.85%的准确率，对罕见攻击（如MITM）的F1分数达88.79%，并通过消融实验验证了TabTransformer与PPO在缓解类别不平衡和提升泛化能力上的互补作用。
