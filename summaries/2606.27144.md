# PAMAE: Phase-Aware-MoE Action Experts Towards Reliable Flow-Matching Vision-Language-Action Policies

- arXiv: https://arxiv.org/abs/2606.27144v1
- arXiv ID: 2606.27144

---

🔍 **核心问题**：多阶段机器人操作任务中，现有流匹配视觉-语言-动作（VLA）模型使用单一共享动作专家，难以捕捉不同执行阶段的特定控制模式，导致动作生成可靠性不足。

🛠 **方法**：提出即插即用的相位感知混合专家动作模块（PAMAE），用稀疏专家混合替换原始流匹配动作专家，通过相位感知路由器利用执行阶段线索分配专家，并采用两阶段训练（先标准流匹配损失预热，再优化相位一致路由）。

✨ **主要贡献**：在多阶段操作仿真任务中，PAMAE将任务成功率提升高达9.2%，消融实验证明相位监督路由和分阶段优化是关键，为流匹配VLA策略的可靠动作生成提供了有效机制。
