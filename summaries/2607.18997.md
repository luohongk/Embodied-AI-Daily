# Benchmarking Deep Learning Approaches for AEC Engineering Drawing Layout Detection and Information Extraction

- arXiv: https://arxiv.org/abs/2607.18997v1
- arXiv ID: 2607.18997

---

🔍 **核心问题**：从建筑、工程与施工（AEC）图纸中提取信息仍依赖人工，效率低下，而作为组织图形与文本层次的关键“中间件”——布局检测，尚未得到充分研究，通用文档布局模型在工程图纸上缺乏验证。

🛠 **方法**：构建了AEC专用的布局数据集，并基准测试了五种深度学习架构，包括RF-DETR和视觉语言模型Qwen3-VL。

✨ **主要贡献**：RF-DETR在布局检测上达到最优性能（mAP₅₀=0.949），Qwen3-VL在信息提取上取得领先F1分数（0.911）；同时发现通用文档预训练模型存在“领域干扰”，导致性能下降，为AEC自动化信息提取奠定了坚实技术基础。
