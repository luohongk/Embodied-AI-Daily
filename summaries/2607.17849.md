# AlphaOracle: Oracle bone script decipherment via human-workflow-inspired deep learning

- arXiv: https://arxiv.org/abs/2607.17849v1
- arXiv ID: 2607.17849

---

🔍 **核心问题**：约4500个甲骨文中仍有3000个未被破译，现有AI方法无法模拟专家结合字形、语义和文献考据的完整推理流程。

🛠 **方法**：提出AlphaOracle框架，模拟人类专家工作流，包含四阶段流水线：拓片解析、基于部首的形态分析与历时建模、上下文检索与语义对齐、以及基于古典文献的语文学验证，最终生成带置信度的可解释证据链。

✨ **主要贡献**：在多个测试字符上，AlphaOracle的解读与专家高度一致；在86位专家研究中，分析时间减少64%，79%参与者认为高度有用；成功破译“老”字为地名或氏族称谓，修正了商代行政与社会解释。
