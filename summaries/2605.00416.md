## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人学、强化学习（RL）、视觉-语言-动作（VLA）模型、大规模部署、离线到在线学习。
- **问题（Problem）**：预训练的通用机器人策略（VLA模型）在真实世界部署时，由于分布偏移、长尾故障和任务变化，性能不足。现有的离线数据无法覆盖所有部署场景，而现有的在线强化学习方法要么样本效率低，要么难以扩展到大规模、多任务的通用策略。
- **方法（Method）**：提出一个名为“边部署边学习”（LWD）的框架。它是一个大规模（车队级）的离线到在线强化学习系统。LWD通过结合**分布式隐式价值学习（DIVL）** 和**伴随匹配Q学习（QAM）**，在部署过程中，利用机器人车队自主收集的异构经验（包括成功、失败、干预等）来持续改进一个共享的通用VLA策略。
- **创新（Innovation）**：
    1.  **算法创新**：提出了DIVL，用分布式的价值函数替代标量价值函数，更好地处理异构数据中的多模态回报；并采用QAM来稳定地从价值函数中提取策略更新，避免了直接反向传播通过生成式策略的不稳定性。
    2.  **系统创新**：构建了一个完整的离线到在线RL数据飞轮系统，将预训练、车队部署、数据收集、策略更新和重新部署闭环，实现了通用策略在真实世界中的持续自我改进。
- **意义（Significance）**：LWD是首批在真实世界、大规模机器人车队上，成功实现通用VLA策略离线到在线RL闭环的系统之一。它证明了部署本身可以成为持续训练的数据来源，为大规模部署通用机器人提供了一条可行路径，即部署的机器人越多，共享策略越强。
- **一句话总结**：本文提出了LWD框架，通过一个结合分布式价值学习和稳定策略提取的离线到在线强化学习系统，让部署在真实世界的机器人车队能够从自身的交互经验中持续学习，从而显著提升通用VLA策略的性能。

## 1. 研究动机（Motivation）

- **解决的问题**：预训练的通用机器人策略（如VLA模型）在真实世界部署时性能不足。离线数据无法覆盖部署中遇到的所有情况，如新的物体、环境、用户指令、长尾故障和罕见的成功模式。
- **为什么重要**：真实世界的部署不是一个固定的测试分布。机器人需要能够从部署经验中持续学习，以适应不断变化的环境和任务。将部署从训练的终点转变为持续改进的源泉，是实现通用机器人规模化应用的关键。
- **挑战在哪里**：
    1.  **数据异构性**：车队部署产生的数据是异构的，包含成功、失败、恢复、部分进展、人工干预等，回报分布可能是多模态和重尾的。传统的标量价值函数会将这些信息平均化，丢失有价值的高回报模式。
    2.  **策略提取困难**：VLA策略通常是基于生成式模型（如流匹配），直接通过价值函数的梯度反向传播来更新策略，计算成本高且不稳定。
    3.  **离线到在线不匹配**：离线阶段训练的批评家（Critic）可能过于保守，无法很好地适应在线阶段的新数据分布，导致在线学习不稳定。
    4.  **可扩展性**：现有的RL方法大多局限于小规模、短时域或单一任务，缺乏一个能在大规模、多任务、真实世界车队中有效工作的通用后训练框架。

## 2. 创新点（Innovation）

- **关键创新**：提出了一个完整的、可扩展的离线到在线强化学习系统（LWD），用于对通用VLA策略进行持续后训练。
- **新方法/技术**：
    1.  **分布式隐式价值学习（DIVL）**：这是价值学习组件。它学习一个**分布式的价值模型** $V_\psi(s)$，该模型表示在状态 $s$ 下，数据集中动作价值的分布。然后，使用该分布的 $\tau$ 分位数作为TD目标。这比IQL中的标量期望值回归更强大，因为它能保留多模态回报信息，为策略改进提供更丰富的信号。同时，它还引入了一个基于熵的自适应 $\tau$ 策略，在不确定性高的状态下使用更保守的目标。
    2.  **伴随匹配Q学习（QAM）**：这是策略提取组件。它将批评家（Critic）在最终动作上的梯度，通过伴随状态（adjoint state）反向传播到流匹配（Flow Matching）策略的整个生成轨迹上，形成局部回归目标。这避免了直接反向传播通过多步去噪过程的不稳定性和高计算成本，实现了稳定且高效的价值引导策略更新。
    3.  **统一的离线到在线训练目标**：LWD在离线预训练和在线微调阶段使用**相同的RL目标函数**（DIVL + QAM）。这缓解了常见的离线到在线不匹配问题，因为离线阶段训练的批评家已经为在线阶段的优化做好了准备。
- **与现有方案的不同**：
    - 相比于纯离线方法（如RECAP），LWD能利用在线部署数据快速适应分布偏移。
    - 相比于在线模仿学习方法（如HG-DAgger），LWD能利用自主探索中的成功、失败和奖励信号，而不仅仅是模仿人类干预。
    - 相比于在线RL方法（如VLA-RL），LWD能高效地重用大型离线数据集和离策略的在线回放缓冲区，样本效率更高。
    - 相比于任务特定的RL方法，LWD训练的是一个跨多个任务的通用策略，并保持其通用性。
- **为什么能奏效**：
    - DIVL通过分布式的价值表示，更准确地捕捉了异构数据中的价值信息，为策略提供了更可靠的改进方向。
    - QAM提供了一种稳定且高效的方式，将价值函数的信息注入到生成式策略中，实现了有效的策略提升。
    - 统一的训练目标确保了离线到在线阶段的平滑过渡，避免了策略和批评家的重新适应。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

LWD的整体架构是一个数据飞轮，如图2(a)所示，分为两个阶段：

1.  **阶段1：离线RL预训练**：在一个静态的离线回放缓冲区 $B_{off}$ 上训练策略 $\pi_\theta$、批评家 $Q_\phi$ 和分布式价值模型 $V_\psi$。这个缓冲区包含人类演示、历史策略的自主rollout和人类引导的探索数据。
2.  **阶段2：持续在线后训练**：将预训练的策略部署到一个由16台机器人组成的车队上。机器人自主执行rollout，并将产生的经验（包括成功、失败和可选的人类干预）存入在线缓冲区 $B_{on}$。中央学习器从混合回放缓冲区 $B_{off} \cup B_{on}$ 中采样，使用DIVL和QAM更新策略和批评家。更新后的策略定期重新部署到机器人车队，形成闭环。

算法结构如图2(b)所示，包含两个核心组件：
- **价值学习（DIVL）**：训练一个共享的VLM骨干网络，其上连接一个用于状态表示 $z_t$ 的读取令牌。价值头 $V_\psi$ 基于 $z_t$ 预测一个分类分布，批评家 $Q_\phi$ 基于 $z_t$ 和动作块 $a_t$ 预测两个标量值（双Q网络）。
- **策略提取（QAM）**：策略 $\pi_\theta$ 是一个基于流匹配的VLA模型。在更新时，它使用QAM损失，该损失由批评家 $Q_\phi$ 提供的梯度引导。

### 3.2 关键算法与数学推导

**问题设定**：
- 马尔可夫决策过程 $M = (S, A, T, r, \gamma)$。
- 状态 $s_t = (o, \ell_k)$，包含观测 $o$ 和语言指令 $\ell_k$。
- 动作块 $a_t \equiv a_{t:t+H} = [a_t, a_{t+1}, ..., a_{t+H-1}] \sim \pi_\theta(\cdot | s_t)$。
- 块奖励 $r_t \equiv r_{t:t+H} = \sum_{i=0}^{H-1} \gamma^i r_{t+i}$。
- 稀疏二进制奖励：成功时 $r=1$，否则 $r=0$。

**1. 分布式隐式价值学习（DIVL）**

DIVL的核心是学习一个分布式的价值模型 $V_\psi(s_t)$，它表示在状态 $s_t$ 下，数据集中动作价值的分布：
$$p_\psi(v | s_t) = P(v = Q_{\bar{\phi}}(s_t, a_t) | a_t \sim \mathcal{D}(\cdot | s_t))$$
其中 $Q_{\bar{\phi}}$ 是目标批评家网络。

- **价值模型损失**：通过最小化负对数似然来拟合该分布：
  $$L_V(\psi) = \mathbb{E}_{(s_t, a_t) \sim \mathcal{D}} \left[ -\log p_\psi \left( Q_{\bar{\phi}}(s_t, a_t) | s_t \right) \right]$$
  在实践中，$p_\psi$ 被参数化为一个分类分布。

- **TD目标**：使用 $V_\psi$ 的 $\tau$ 分位数作为引导统计量：
  $$y^Q = r_t + \gamma^H \text{Quant}_\tau \left( V_\psi(s_{t+H}) \right)$$
  其中 $\text{Quant}_\tau(V_\psi(s)) \triangleq \inf\{v : F_\psi(v|s) \geq \tau\}$，$F_\psi$ 是累积分布函数。

- **批评家损失**：
  $$L_Q(\phi) = \mathbb{E}_{(s_t, a_t, r_t, s_{t+H}) \sim \mathcal{D}} \left[ \left( Q_\phi(s_t, a_t) - y^Q \right)^2 \right]$$

- **自适应 $\tau$ 策略**：使用价值分布 $p_\psi$ 的归一化熵作为不确定性信号，动态调整 $\tau$：
  $$\mathcal{H}(s_{t+H}) = -\frac{1}{\log C} \sum_{c=1}^{C} p_{\psi, c}(s_{t+H}) \log p_{\psi, c}(s_{t+H})$$
  $$\tau(s_{t+H}) = \text{clip} \left( \tau_{\text{base}} - \alpha \mathcal{H}(s_{t+H}), \tau_{\text{min}}, \tau_{\text{max}} \right)$$
  当分布熵高（不确定性大）时，使用更小的 $\tau$（更保守的目标）。

- **n步TD目标（离线阶段）**：对于长时域任务，使用n步目标加速稀疏奖励传播：
  $$y^Q = \sum_{i=0}^{n-1} \gamma^{iH} r_{t+iH} + \gamma^{nH} \text{Quant}_{\tau(s_{t+nH})} \left( V_\psi(s_{t+nH}) \right)$$

**2. 伴随匹配Q学习（QAM）**

QAM用于从价值函数中提取策略更新。给定一个参考流策略 $f_\beta$ 和批评家 $Q_\phi$，KL正则化的改进目标为：
$$\pi^*(a|s) \propto \pi_\beta(a|s) \exp(Q_\phi(s, a) / \lambda)$$

QAM将此目标转化为一个局部回归损失。首先，定义一个残差向量场 $f_\delta = f_\theta - f_\beta$。然后，QAM损失为：
$$L_{\text{QAM}}(\theta) = \mathbb{E} \left[ \int_0^1 \left\| \frac{2 f_\delta(s, a_w, w)}{\sigma_w} + \sigma_w \tilde{g}_w \right\|_2^2 dw \right]$$
其中：
- $a_w = (1-w)a_0 + w a_1$ 是参考流 $f_\beta$ 生成的插值轨迹。
- $\sigma_w = \sqrt{2(1-w)w}$。
- $\tilde{g}_w$ 是伴随状态，其终端条件由批评家梯度决定：
  $$\tilde{g}_1 = -\nabla_a \left[ Q_\phi(s, a_1) / \lambda \right]$$
  这个伴随状态通过求解一个反向常微分方程（ODE）从 $w=1$ 传播到 $w=0$，从而将终端梯度信息分布到整个生成轨迹上。

### 3.3 模型、训练与数据集

- **模型架构**：
    - **价值/批评家网络**：共享一个Gemma3-SigLIP VLM骨干网络。骨干网络输出一个读取令牌 $z_t$ 作为状态表示。价值头 $V_\psi$ 是一个分类头，预测固定支持上的logits。批评家 $Q_\phi$ 将 $z_t$ 和编码后的动作块 $a_t$ 拼接，输入到两个标量头（双Q网络）。
    - **策略网络**：采用 $\pi_{0.5}$ 流匹配VLA架构，包含一个PaliGemma VLM骨干网络和一个用于流匹配动作生成的Gemma-300M动作专家。
- **训练过程**：
    - **离线阶段**：策略、价值/批评家网络全部进行微调。
    - **在线阶段**：策略的VLM骨干网络被冻结，只更新动作专家。价值/批评家网络继续全参数微调。这保证了在线更新的效率，并保留了预训练的视觉-语言表示。
- **数据集**：
    - **离线缓冲区 $B_{off}$**：包含三种来源的数据：
        1.  **演示（Demonstrations）**：人类专家收集的成功轨迹。
        2.  **Rollout**：由历史策略生成的数据，包括成功和失败。
        3.  **Play数据**：人类引导的探索数据，用于覆盖失败模式。
    - **在线缓冲区 $B_{on}$**：由机器人车队在部署过程中自主收集的rollout数据，包括自主策略轨迹和人类干预轨迹。

### 3.4 实验设置与结果

- **实验设置**：
    - **任务**：8个真实世界操作任务，包括4个短时域任务（杂货补货）和4个长时域任务（泡功夫茶、榨果汁、调鸡尾酒、装鞋盒）。
    - **机器人**：16台Agibot G1双臂机器人。
    - **基线**：SFT（监督微调）、RECAP（离线RL）、HG-DAgger（在线模仿学习）。
    - **评估指标**：短时域任务用二值成功率；长时域任务用分步成功率（0, 0.5, 1）。同时报告循环时间。
    - **预算**：每个在线实验分配4小时墙钟时间。

- **主要结果（表I）**：
    - LWD（在线）在所有任务上取得了平均 **0.95** 的成功率，显著优于所有基线。
    - 在长时域任务上，LWD（在线）的优势尤为明显（平均0.91），远超SFT（0.68）、RECAP（0.77）和HG-DAgger（0.73）。
    - LWD（离线）也优于SFT，证明了离线RL阶段的有效性。
    - 图5显示，LWD不仅提高了成功率，还显著缩短了任务完成时间。

- **消融研究（表II）**：
    - **价值学习设计**：将DIVL与标量期望值回归（IQL风格）对比。DIVL在所有任务上表现更好，尤其是在长时域任务上，在线阶段提升了 **16.7%**。这证明了分布式价值表示在处理异构数据时的优势。
    - **自适应 $\tau$ 策略**：消融实验表明，自适应 $\tau$ 策略优于固定 $\tau$ 的基线，尤其是在长时域任务上。

### 3.5 技术细节

- **动作块**：策略输出一个动作块 $a_{t:t+H}$，执行后再重新规划。
- **双Q网络**：批评家使用双Q网络设计，取两个Q值中的最小值来构建TD目标，以减轻过估计。
- **n步目标**：在离线阶段，对于长时域任务使用n=10的n步TD目标，以加速稀疏奖励的传播。在线阶段使用1步目标，因为在线轨迹可能混合了自主和干预数据，长步数目标可能不准确。
- **策略同步**：中央学习器每50个训练步骤将更新后的策略发布到机器人车队。
- **人类干预**：当机器人需要纠正时，操作员可以干预。干预片段被存储为常规的在线回放转换，并分配与自主rollout相同的终端奖励。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    - 解决了通用机器人策略在真实世界部署中无法持续改进的关键问题。
    - 提出了一个完整的、可扩展的离线到在线RL系统（LWD），该系统在真实世界的机器人车队上得到了验证。
    - 算法上，DIVL和QAM的组合为处理异构数据和稳定更新生成式策略提供了有效方案。
    - 系统上，构建了一个数据飞轮，使得部署本身成为持续训练的动力源。

- **对未来研究/应用的影响**：
    - 为大规模部署通用机器人提供了一条可行的技术路线：部署的机器人越多，收集的数据越多，共享策略就越强，从而形成一个正反馈循环。
    - 推动了RL在真实世界机器人中的应用，证明了离线到在线RL框架在复杂、长时域任务上的有效性。
    - 为后续研究提供了新的方向，如更高效的策略提取方法、更鲁棒的价值学习算法，以及如何更好地利用人类干预数据。

- **局限与开放问题**：
    - 论文中未明确讨论，但可以推断的局限包括：对计算资源的需求较高（需要中央学习器和分布式车队）；
