# Inverse Design of Compact and Wideband Inverted Doherty Power Amplifiers Using Deep Learning

- arXiv: https://arxiv.org/abs/2606.27002v1
- arXiv ID: 2606.27002

---

🔍 **核心问题**：解决传统Doherty功率放大器设计复杂、带宽受限且难以同时实现紧凑结构与高性能的问题。  
🛠 **方法**：提出一种基于深度学习（卷积神经网络CNN）与遗传算法（GA）的逆向设计方法，自动生成像素化Doherty合路网络，将负载调制、阻抗匹配、功率合成与相位补偿集成于单一结构。  
✨ **主要贡献**：成功设计并制造了基于GaN HEMT的紧凑宽带倒置Doherty功率放大器，在1.9-2.5 GHz频段内实现峰值漏极效率51%-63%、6dB回退效率48%-54%，输出功率44±0.3 dBm，且经数字预失真后邻道泄漏比优于-53.2 dBc。
