# Foresight: Failure Detection for Long-Horizon Robotic Manipulation with Action-Conditioned World Model Latents

- arXiv: https://arxiv.org/abs/2606.23085v1
- arXiv ID: 2606.23085

---

🔍 **核心问题**：针对长时域机器人操作任务中故障检测困难的问题，尤其是故障起始模糊且缺乏密集时间标注的现实挑战。

🛠 **方法**：提出Foresight框架，利用动作条件世界模型的潜在表征监控操作轨迹，仅需最终任务级成功/失败标签进行训练，并结合功能共形预测自适应校准检测阈值。

✨ **主要贡献**：在多个仿真和真实机器人长时域任务上验证了该方法优于现有故障检测技术，表明动作条件世界模型嵌入为长时域操作提供了可扩展的可靠故障监控表征。
