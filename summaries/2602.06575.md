# Think Proprioceptively: State-Grounded Visual Token Selection for VLA Policies

- arXiv: https://arxiv.org/abs/2602.06575v2
- arXiv ID: 2602.06575

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型通常将本体感知仅作为后期条件信号输入，导致机器人状态无法参与指令理解或引导视觉注意力，限制了模型对动作相关信息的提取效率。

🛠 **方法**：提出ThinkProprio，将本体感知离散化为VLM词汇表中的token，并与指令共同作为主动查询，在VLM计算前筛选视觉patch，仅保留约12%的关键视觉token，同时丢弃冗余信息。

✨ **主要贡献**：实验表明，以token形式的本体感知作为主动查询能显著提升性能，在CALVIN、LIBERO及真实机器人任务中，该方法在降低端到端推理延迟的同时，超越了全token基线模型。
