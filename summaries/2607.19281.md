# A Reinforcement-Learning-Augmented Liquid-Fueled Reactor Network Model for Predicting Lean Blowout in Gas Turbine Combustors

- arXiv: https://arxiv.org/abs/2607.19281v1
- arXiv ID: 2607.19281

---

🔍 **核心问题**：提升燃气轮机燃烧室中稀薄吹熄（LBO）预测的准确性，解决现有聚类方法依赖人工经验或距离度量、未直接优化预测目标的问题。

🛠 **方法**：提出一种强化学习（RL）框架，采用多阶段聚类-分类策略：先用k-means生成大量均匀微簇，再通过演员-评论家RL智能体将微簇合并为最优反应器区域，以LBO预测精度为优化目标。

✨ **主要贡献**：相比传统k-means方法，RL框架在Jet-A燃料机理（119种组分、841个反应）验证中显著提高了LBO预测精度和趋势捕捉能力，同时大幅降低计算成本，为快速设计空间探索提供了高效降阶建模工具。
