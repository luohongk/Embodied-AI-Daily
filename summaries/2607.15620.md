# AEGIS: Assay-Aware Protocol Validation and Runtime Monitoring for Open-Source Liquid Handling Robots

- arXiv: https://arxiv.org/abs/2607.15620v1
- arXiv ID: 2607.15620

---

🔍 **核心问题**：开源液体处理机器人（如Opentrons OT-2）在运行中无法检测两类失败模式：违反实验特定规则的协议（如重复使用吸头导致污染）和物理执行错误（如部分分液、气泡、吸头缺失）。

🛠 **方法**：提出AEGIS双层监护系统。第一层结合机器可读实验规则库与大型语言模型（LLM），对OT-2 Python代码进行推理验证；第二层使用PCA世界模型分析YOLO裁剪的四帧移液轨迹，实现运行时视觉监控。

✨ **主要贡献**：第一层在24个协议基准测试中达到调整F1值0.97，且开源模型性能与最佳商业模型相当；第二层在无泄漏评估中平均精度0.89、F1值0.71，并成功在物理机器人上检测吸头缺失和部分分液故障，同时通过级联策略将视觉语言模型成本降至每板约1.63美元。这是首个统一预飞行协议验证与运行时视觉监控的开源系统。
