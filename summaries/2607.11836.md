# Cycle-World: Mitigating Error Accumulation in Long-term Video World Models via Reverse-Prediction Cycle Consistency

- arXiv: https://arxiv.org/abs/2607.11836v1
- arXiv ID: 2607.11836

---

🔍 **核心问题**：解决自回归扩散模型在长视频生成中因误差累积导致的生成漂移、结构崩溃和视觉退化问题。

🛠 **方法**：提出Cycle-World框架，通过训练阶段引入反向预测模型强制前向生成器产生可逆序列，并在推理阶段利用冻结的反向模型作为运行时校正器，通过梯度循环指导迭代优化潜在表示，抑制误差累积。

✨ **主要贡献**：在VBench基准上，Cycle-World的双阶段协同机制显著缓解了误差漂移，在60秒长视频合成中实现了最先进的整体生成质量和长时间一致性。
