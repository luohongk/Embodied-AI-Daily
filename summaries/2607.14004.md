# Do Agent Optimizers Compound? A Continual-Learning Evaluation on Terminal-Bench 2.0

- arXiv: https://arxiv.org/abs/2607.14004v1
- arXiv ID: 2607.14004

---

🔍 **核心问题**：研究智能体优化方法带来的性能提升是否能在持续学习场景中累积，即当新任务不断出现时，优化效果是否会因遗忘或退化而丧失。

🛠 **方法**：在Terminal-Bench 2.0的困难任务上构建两阶段持续学习评估，对比三种智能体优化方法（GEPA、Meta Harness、RELAI-VCL），在相同优化预算下测试其在新旧任务上的表现。

✨ **主要贡献**：发现只有内置回归控制机制的RELAI-VCL方法能实现优化增益的持续累积，在每阶段均达到最高通过率（终身平均76.4%），而其他方法要么迁移能力差，要么无法在新任务上进一步改进。
