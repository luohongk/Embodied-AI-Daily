# As It Was: Aligning LLM Search Evaluation with Historical User Preferences

- arXiv: https://arxiv.org/abs/2607.01040v1
- arXiv ID: 2607.01040

---

🔍 **核心问题**：大型搜索系统快速迭代时，人工评估难以覆盖长尾和多语言查询，而基于大语言模型的评估方法仅依赖语义相似性，可能偏离真实用户偏好，尤其在模糊查询场景下。

🛠 **方法**：提出一种行为锚定的大语言模型评估方法，为每个搜索结果项附加轻量级、可审计的行为先验（查询-相关性-印象卡片），该卡片汇总历史用户交互数据，供模型在评估时引用以消除歧义。

✨ **主要贡献**：在Spotify音乐搜索评估中，该方法与用户偏好的斯皮尔曼秩相关系数提升约5%，争议案例相对提升91%；在多语言人工标注数据集上相关性提升15%；在A/B测试中与获胜模型一致性更高，验证了轻量级行为锚定可提升大语言模型评估的可靠性和实用性。
