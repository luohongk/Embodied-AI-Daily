# Qwen-RobotNav Technical Report: A Scalable Navigation Model Designed for an Agentic Navigation System

- arXiv: https://arxiv.org/abs/2606.18112v1
- arXiv ID: 2606.18112

---

🔍 **核心问题**：现有导航模型在推理时无法灵活调整观测策略，难以同时适应指令跟随、目标搜索、目标跟踪和自动驾驶等不同任务的需求。

🛠 **方法**：提出Qwen-RobotNav，通过参数化接口引入两个互补维度——多任务模式（选择导航行为）和可控观测参数（如token预算、相机权重），并在训练时随机化所有参数，使模型对推理时的任意配置鲁棒，无需修改主干架构。

✨ **主要贡献**：在15.6M样本上训练，联合视觉-语言数据防止模型退化为反应式动作映射器；支持上层规划器在长时域场景中动态切换任务模式，实现复杂行为组合；在多个主流导航基准上达到新SOTA，参数从2B扩展到8B时性能持续提升，并展现出对真实世界机器人的强零样本泛化能力。
