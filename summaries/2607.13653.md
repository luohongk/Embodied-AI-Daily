# Exploratory, Communicative, and Deployable: Vision-Driven Embodied Agents for Open-World Mobile Manipulation

- arXiv: https://arxiv.org/abs/2607.13653v1
- arXiv ID: 2607.13653

---

🔍 **核心问题**：现有具身智能框架依赖模拟器特权状态或完整指令，无法应对开放世界移动操作中的主动探索、视觉定位和交互意图消歧等现实部署挑战。

🛠 **方法**：提出REAL框架，构建模拟-现实一致的API环境（无特权感知），集成模拟用户实现人机交互，通过多样化任务组合驱动数据收集、监督微调和在线强化学习，并设计REAL-Bench基准（241个任务）进行系统评估。

✨ **主要贡献**：在交互任务上以56.9%成功率超越商业闭源VLM；分层训练流程在长程探索中保持开放词汇推理能力；在真实双臂移动机器人上实现78.3%端到端成功率，验证了零样本迁移到未见家庭场景的鲁棒性。
