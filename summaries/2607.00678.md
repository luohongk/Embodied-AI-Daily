# ABot-M0.5: Unified Mobility-and-Manipulation World Action Model

- arXiv: https://arxiv.org/abs/2607.00678v1
- arXiv ID: 2607.00678

---

🔍 **核心问题**：移动操作机器人面临三大对齐挑战——时间粒度粗、导航与操作动作耦合、训练与推理条件不一致，导致现有世界动作模型在长程任务中误差累积、控制精度不足。

🛠 **方法**：提出ABot-M0.5模型，通过引入中间潜在动作对齐时间粒度，设计双级混合Transformer架构解耦动作空间，并采用梦想强制训练策略逐步在模型预测视频上训练逆动力学，实现训练-推理一致性。

✨ **主要贡献**：在移动操作和精细控制基准上取得最先进性能，验证了粒度对齐、动作解耦和推理一致的世界动作建模对长程任务成功率和控制精度的关键作用。
