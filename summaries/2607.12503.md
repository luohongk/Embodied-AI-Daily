# DynTrace: Tracking Dynamic Object Evidence for 4D Spatio-Temporal Reasoning in MLLMs

- arXiv: https://arxiv.org/abs/2607.12503v1
- arXiv ID: 2607.12503

---

🔍 **核心问题**：当前多模态大语言模型在连续动态场景中难以有效追踪动态物体证据，导致4D时空推理不连贯，且无法区分物体真实运动与相机引起的表观运动。

🛠 **方法**：提出无需训练的框架DynTrace，包含两个互补组件：动态轨迹可视化（DTV）将世界坐标轨迹重投影到图像平面，提供几何先验以分离真实运动与表观运动；动态追踪标记（DT-Token）通过动态追踪图（DTG）追踪物体级动态线索、演化过程及关键时刻，维持连续证据。

✨ **主要贡献**：在Dyn-Bench、VLM4D和DSI-Bench三个基准上取得最优结果，验证了追踪动态物体证据对鲁棒4D时空推理的关键作用，且能一致提升开源多模态大语言模型性能。
