# FakeVLM-R1: Internalizing Physical Laws via CoT for Synthetic Image Detection

- arXiv: https://arxiv.org/abs/2605.30062v1
- arXiv ID: 2605.30062

---

🔍 **核心问题**：现有基于大多模态模型的合成图像检测方法依赖大量伪造数据的模仿学习，缺乏因果推理能力，容易产生解释性幻觉，且对真实图像存在过度拒绝偏差。

🛠 **方法**：提出FakeVLM-R1框架，在监督微调基础上结合群体相对策略优化与批判性思维链机制，在推理时执行“双向辩证推理”——同时提出伪造假设并调用物理常识构建真实性反证。

✨ **主要贡献**：构建了FakeClue++高质量数据集，引入物理规律引导的标注；模型在多个基准上达到最先进性能，实现了高精度、逻辑可解释的检测，并解决了对真实图像的过度拒绝偏差，展现出良好的泛化性和抗扰动鲁棒性。
