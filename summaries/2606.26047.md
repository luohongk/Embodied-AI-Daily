# Learning Robot Visual Navigation in Crowds via Intention-Aware Scene Representations

- arXiv: https://arxiv.org/abs/2606.26047v1
- arXiv ID: 2606.26047

---

🔍 **核心问题**：现有机器人人群导航方法多将行人简化为2D点，忽略了视觉场景中的丰富信息，导致对行人意图和环境结构理解不足。

🛠 **方法**：提出iCrowdNav方法，采用时空编码器提取场景占用特征，并设计Intent-Interact Former模块通过编码人体姿态推断行人运动意图，将两者融合为紧凑状态嵌入以训练深度强化学习策略。

✨ **主要贡献**：在仿真和真实场景中均优于基线方法，首次实现了基于视觉的意图感知人群导航，验证了视觉信息在复杂动态环境中的有效性。
