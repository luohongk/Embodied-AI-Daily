## 0. 概览（Concise Summary）

- **主题（Topic）**：自主水下机器人（AUV）在三维（3D）杂乱环境中的实时避障导航。
- **问题（Problem）**：现有方法要么依赖昂贵且计算量大的3D声纳，要么使用视野有限、缺乏垂直信息的2D前视声纳（FLS），导致在部分可观测、浑浊、有遮挡的水下环境中导航不安全、效率低。
- **方法（Method）**：提出EROAS（高效反应式避障策略）框架。该框架通过一个**机械旋转机构**将标准的2D FLS升级为**2.5D声纳**，并结合三个核心模块：1）**SPD2C**（声纳轮廓引导的方向决策控制）用于快速检测间隙并生成参考指令；2）**SCG**（空间上下文生成器）用于维护短期障碍物记忆，缓解部分可观测性问题；3）**ST-CBF**（时空控制障碍函数）用于过滤参考指令，保证安全约束的向前不变性。
- **创新（Innovation）**：1）提出**2.5D声纳**概念，以低成本、低计算量的方式获取垂直方向信息。2）设计**SPD2C**算法，通过一系列逻辑判断（间隙检测、有界性、收敛性、垂直扫描）实现高效的反应式避障。3）将**SCG**与**ST-CBF**结合，利用短期记忆增强CBF在部分可观测环境下的安全保证能力。
- **意义（Significance）**：为资源受限的AUV提供了一种**轻量级、高效、安全**的3D避障解决方案。相比传统方法（DWA、APF），EROAS在路径长度、旅行时间、控制平滑度上均有显著提升，并成功在边缘计算设备上实现实时运行，具有很高的实际应用价值。
- **一句话总结**：本文提出EROAS，通过创新的2.5D声纳和轻量级算法框架，实现了AUV在杂乱水下环境中高效、平滑且安全的3D反应式避障。

## 1. 研究动机（Motivation）

**研究动机**：自主水下机器人（AUV）在海洋勘探、环境监测等任务中至关重要。然而，在复杂的水下环境中安全导航面临巨大挑战。

- **问题与空白**：
    1.  **传感器局限**：光学相机在浑浊水域性能差。2D前视声纳（FLS）虽然能在低能见度下工作，但**缺乏垂直方向的空间感知**，无法应对3D障碍物（如悬空的岩石、海底山脊）。全3D声纳能提供完整信息，但其**成本高、功耗大、数据处理复杂**，不适合资源受限的小型AUV。
    2.  **环境挑战**：水下环境存在**部分可观测性**（障碍物可能被遮挡）、**视野有限**（FLS的扇形区域）、**浑浊**和**遮挡**等问题，导致无法完整获取障碍物的几何形状和分布。
    3.  **现有方法不足**：传统的避障方法如动态窗口法（DWA）和人工势场法（APF）在复杂3D环境中表现不佳，容易陷入局部极小值或产生不平稳的轨迹。基于学习的方法（如深度强化学习）需要大量数据且缺乏形式化的安全保证。基于模型预测控制（MPC）的方法计算量大，难以在边缘设备上实时运行。

- **重要性**：解决这些问题对于提高AUV的任务成功率、避免昂贵的碰撞事故、延长其在水下的作业范围至关重要。特别是在没有先验地图的未知区域，一个高效、安全且计算资源友好的避障系统是AUV自主导航的核心。

- **核心挑战**：如何在**不增加硬件成本和计算负担**的前提下，利用**有限的2D声纳感知**，实现**鲁棒、安全且高效**的3D避障，同时应对**部分可观测性**带来的不确定性。

## 2. 创新点（Innovation）

论文提出了三个关键创新点，共同构成了EROAS框架：

1.  **2.5D声纳感知机制**：
    - **新意**：不同于直接使用昂贵的3D声纳，本文提出为标准的2D FLS增加一个**机械旋转机构**。通过主动控制声纳的俯仰角（pivot angle），可以在需要时快速扫描垂直方向，获取“按需”的垂直信息。这种设计将2D声纳升级为**2.5D声纳**，在几乎不增加成本和计算量的前提下，极大地扩展了环境感知的维度。
    - **为何奏效**：它巧妙地平衡了感知能力与资源消耗。在大多数情况下，AUV只需在水平面进行避障。只有当水平方向无路可走时，才触发垂直扫描，寻找垂直方向的可行通道。这种“按需”机制避免了持续处理大量3D点云数据，实现了高效性。

2.  **Sonar Profile-guided Directional Decision Control (SPD2C)**：
    - **新意**：这是一个**分层决策**的避障算法，它模拟了人类驾驶员的决策过程。它并非直接计算一个复杂的路径，而是通过一系列逻辑判断（**间隙检测 → 有界性检查 → 收敛性分析 → 垂直扫描**）来快速确定一个可行的运动方向。
    - **为何奏效**：这种结构化流程非常**高效**，因为它将复杂的避障问题分解为一系列简单的、确定性的规则。例如，当检测到障碍物时，它首先寻找通往目标方向的最大间隙；如果被完全包围，它会判断障碍物是向左还是向右延伸（有界性）；如果障碍物在两侧都无边无际，它会分析其形状是凸的还是凹的（收敛性），从而决定是沿着障碍物边缘走还是转向垂直方向。这种设计避免了复杂的全局路径搜索，实现了**反应式**的快速决策。

3.  **Spatial Context Generator (SCG) 与 Spatio-Temporal Control Barrier Function (ST-CBF) 的结合**：
    - **新意**：传统的CBF依赖于当前时刻的状态，在障碍物离开传感器视野后，其安全约束会立即失效，导致“健忘”问题。本文提出的SCG维护了一个**短期障碍物记忆**，将过去一段时间内检测到的障碍物点保留在一个局部地图中。ST-CBF则利用这个包含历史信息的局部地图来构建安全约束。
    - **为何奏效**：这种**时空耦合**的设计解决了部分可观测性问题。即使障碍物因为AUV的转向或自身遮挡而暂时离开FLS的视野，只要它还在SCG的记忆半径内，ST-CBF仍然会将其视为一个威胁，并持续施加安全约束。这防止了AUV在绕过障碍物后立即“忘记”它，从而做出危险动作，大大增强了系统的鲁棒性。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

EROAS框架的整体架构如图3所示，是一个模块化的流水线系统：

1.  **输入**：2D FLS的原始声纳数据。
2.  **SPD2C模块**：处理声纳数据，生成名义参考速度（$V_R$）和偏航率（$r_R$）。该模块包含间隙检测、有界性判断、收敛性分析和声纳旋转决策。
3.  **SCG模块**：并行运行，接收声纳数据，维护并更新一个以AUV为中心的局部障碍物地图（记忆半径$r_o$），并提供最近的障碍物点信息。
4.  **ST-CBF模块**：接收来自SPD2C的名义参考$V_R$和来自SCG的障碍物信息，通过求解一个二次规划（QP）问题，对$V_R$进行“过滤”，生成**安全参考速度**$V_R^S$。同时，对偏航率$r_R$进行限幅。
5.  **低层控制器**：一个PID控制器接收安全参考指令$V_R^S$和$r_R$，并将其转化为具体的推进器推力，驱动AUV运动。

### 3.2 关键算法与数学推导

**1. SPD2C - 间隙检测与目标波束选择**

- **障碍物自由波束集**：$B_F = \{b_i \in B \mid I_i < I_{thr}\}$，其中$I_i$是波束$b_i$的反射强度，$I_{thr}$是强度阈值。
- **候选间隙**：从$B_F$中提取长度为$L$的连续子集$G_i$。
- **目标波束**：$b_T$，对应全局目标方向在声纳视野中的投影。
- **选择的最优波束**：$b_{cl} = \arg\min_{b_M \in M} |b_M - b_T|$，即所有候选间隙的中间波束中，最接近目标方向的波束。

**2. SPD2C - 有界性判断**

根据障碍物波束在声纳视野中的起始和结束索引（$i_{min}, i_{max}$），将障碍物分为四类：
- **BO (Bounded Obstacle)**：障碍物在视野内，两侧都有自由空间。
- **LUBO (Left Unbounded Obstacle)**：障碍物从视野最左侧开始。
- **RUBO (Right Unbounded Obstacle)**：障碍物延伸到视野最右侧。
- **UBO (Unbounded Obstacle)**：障碍物完全覆盖整个视野。

**3. SPD2C - 收敛性判断**

当遇到UBO时，对障碍物点云进行二次多项式拟合：$f(x) = ax^2 + bx + c$。
- 如果曲率系数$a \ge C_{th}$（$C_{th}$为阈值），则障碍物是**凸的（Converging）**，AUV应转向拟合抛物线的目标侧。
- 如果$a < C_{th}$，则障碍物是**凹的/墙状（Diverging）**，表明水平方向无路可走，触发垂直扫描。

**4. SPD2C - 速度映射**

当找到可行间隙后，生成参考速度：
- 前向速度：$v_{x,R} = K_v (\psi_{max} - |\psi_R|)$，其中$\psi_R$是目标波束对应的偏航角。
- 垂直速度：$v_{z,R} = v_{x,R} \times \tan(\Theta_{cl})$，其中$\Theta_{cl}$是垂直方向上的最优俯仰角。
- 偏航率：$r_R = K_t * \psi_R$

**5. ST-CBF (Spatio-Temporal Control Barrier Function)**

- **CBF定义**：$h(p_v, t) = \|p_v(t) - p_{o,cl}(t)\|_2^2 - R_o^2$，其中$p_v$是AUV位置，$p_{o,cl}$是SCG提供的最近障碍物点，$R_o$是障碍物安全半径。
- **安全集**：$C(t) = \{p_v \mid h(p_v, t) \ge 0\}$。
- **安全控制律**：通过求解以下二次规划（QP）问题得到：
  $$ \min_{V} \frac{1}{2} \|V - V_R\|^2 $$
  $$ \text{subject to } \dot{h} \ge -\alpha(h) $$
  其中$V = [v_x, v_y, v_z]^T$是实际速度，$V_R$是SPD2C给出的参考速度。约束条件$\dot{h} \ge -\alpha(h)$（$\alpha$是$\mathcal{K}$类函数，如$\alpha(h) = kh$）保证了安全集$C(t)$的向前不变性，即只要系统从安全状态开始，就会一直保持安全。
- **代价函数**：$\|V - V_R\|^2 = (v_x - v_{x,R})^2 + a(v_y - v_{y,R})^2 + b(v_z - v_{z,R})^2$。其中$a, b \in \{0, 1\}$是选择器，当水平避障时$a=1, b=0$；垂直避障时$a=0, b=1$。这确保了QP只在当前活跃的平面上进行优化。

### 3.4 实验设置与结果

- **实验环境**：在Gazebo仿真环境中搭建了包含各种复杂形状静态障碍物的3D水下场景。
- **基线算法**：与**动态窗口法（DWA）** 和**人工势场法（APF）** 进行对比。
- **主要结果**：
    1.  **路径效率**：在杂乱走廊场景中（图5），EROAS的路径长度（112.30m）比DWA（125.9m）和APF（133.61m）更短，旅行时间（190s）也比DWA（220s）和APF（260s）更少。分别实现了10.8%和15.9%的路径长度缩减，以及13.6%和27%的时间缩减。
    2.  **控制平滑性**：EROAS的偏航率输出（图6）振幅更小，振荡更少。其平均角加速度（0.42 rad/s³）远低于DWA（1.02 rad/s³）和APF（2.42 rad/s³），分别降低了58.8%和82.4%，表明控制更平滑，对执行机构更友好。
    3.  **部分可观测性处理**：在狭窄隧道场景中（图7），DWA和APF因障碍物离开视野而失败，而EROAS凭借SCG的记忆功能成功导航。
    4.  **3D避障能力**：在非凸障碍物场景中（图8），EROAS成功触发垂直避障模式，AUV通过上浮绕过障碍物，然后下潜回到原路径。
    5.  **边缘设备部署**：在Jetson Orin NX和Xavier NX上成功运行，帧率达到8Hz（高于声纳采样率4Hz），功耗低，CPU占用率低（表II），证明了其轻量级和实时性。

### 3.5 技术细节

- **2.5D声纳实现**：通过机械旋转2D FLS来实现。当SPD2C判断水平方向无可行路径时，会命令声纳在垂直方向上进行一系列俯仰角扫描（公式16）。系统会寻找一个在中央波束区域（公式18）内所有波束都无遮挡的俯仰角范围（公式19-20），并选择最接近目标垂直方向的角度（公式23）。
- **SCG更新规则**：公式29展示了SCG的更新逻辑。它保留了上一时刻记忆半径内的点，并添加当前时刻新检测到的、且在记忆半径内的点。这确保了记忆的动态更新和有效性。
- **ST-CBF的QP求解**：这是一个典型的在线优化问题。由于代价函数是二次的，约束是线性的（对$\dot{h}$展开后），该QP问题可以非常高效地求解，适合实时应用。选择器$a, b$的引入使得QP的维度可以根据避障模式动态调整，进一步降低了计算量。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了AUV在资源受限条件下的3D避障难题**，提出了一种不依赖昂贵3D声纳的实用方案。
    2.  **提出了2.5D声纳的概念**，为低成本、高效率的3D感知提供了新思路。
    3.  **设计了SPD2C算法**，提供了一种快速、反应式的决策机制，避免了复杂的路径规划。
    4.  **将SCG与CBF结合**，创造性地解决了CBF在部分可观测环境下的“健忘”问题，增强了安全保证的鲁棒性。
    5.  通过详尽的仿真和硬件实验，**验证了框架的有效性和实用性**，性能全面超越DWA和APF。

- **对未来研究/应用的影响**：
    - **应用层面**：EROAS非常适合部署在**小型、低成本、计算资源有限**的AUV上，用于执行水下勘探、管道检查、环境监测等任务。其高效性可以延长AUV的续航和作业范围。
    - **研究层面**：1）2.5D感知的思想可以推广到其他机器人领域。2）SCG与CBF的结合为解决部分可观测性下的安全控制问题提供了一个有效的范式。3）SPD2C的分层决策逻辑为设计可解释的、反应式的机器人控制器提供了参考。

- **局限与开放问题**：
    1.  **局部最优**：作者指出，CBF的局部反应式特性可能在复杂环境中（尤其是存在动态障碍物时）导致次优的导航路径。
    2.  **声纳精度**：FLS固有的方位角引起的深度误差限制了导航的精度。
    3.  **未来方向**：作者计划通过集成**全局路径规划器**来克服局部最优问题，引入**实时SLAM**进行环境建图，以及开发**运动预测CBF**来处理动态障碍物。
