# Multi-Resolution Feature Stem for Diabetic Retinopathy lesion segmentation

- arXiv: https://arxiv.org/abs/2607.08679v1
- arXiv ID: 2607.08679

---

🔍 **核心问题**：糖尿病视网膜病变（DR）病灶大小差异极大（从微小动脉瘤到大面积出血），单一输入分辨率无法同时兼顾小病灶的细节捕捉和大病灶的上下文信息，导致模型性能冲突。

🛠 **方法**：提出一种多分辨率特征茎（Multi-Resolution Feature Stem），作为输入级金字塔结构与UNet++骨干网络结合，并行处理多个尺度的图像，在保持参数效率的同时平衡细节与全局信息。

✨ **主要贡献**：通过系统实验发现输入分辨率对不同病灶类型存在矛盾影响（高分辨率提升小病灶但降低大病灶性能），并设计出能有效解决这一权衡的实用架构。
