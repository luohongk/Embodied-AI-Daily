# Spatiotemporal Knowledge Graphs as Persistent Scene Memory for Embodied Question Answering

- arXiv: https://arxiv.org/abs/2510.01483v3
- arXiv ID: 2510.01483

---

🔍 **核心问题**：现有视觉-语言模型（VLM）在处理长时自我中心视频时，缺乏持久记忆和显式空间表示，导致每次查询都需要重新处理所有视频帧，延迟高且成本大。

🛠 **方法**：提出VL-KnG框架，无需训练即可从单目自我中心视频构建时空知识图谱，通过基于大语言模型的时空对象关联（STOA）维护持久对象身份，并利用图增强检索（GER）结合子图检索与视觉定位来回答问题。

✨ **主要贡献**：在OpenEQA、NaVQA和WalkieKnowledge三个具身问答基准上，VL-KnG达到与前沿VLM相当的准确率，同时显著降低查询延迟，并提供可解释的图推理过程；在真实机器人部署中，查询延迟不随观测历史增长而增加，为具身AI智能体提供了持久、可查询的场景记忆基础。
