# First Order Model-Based RL through Decoupled Backpropagation

- arXiv: https://arxiv.org/abs/2509.00215v2
- arXiv ID: 2509.00215

---

🔍 **核心问题**：在强化学习中，利用模拟器梯度可提升学习效率，但实际中模拟器梯度常因实现成本高或不可用而难以获取；同时，基于模型的强化学习方法因训练轨迹中的复合预测误差导致策略性能下降。

🛠 **方法**：提出一种解耦反向传播方法，将轨迹生成与梯度计算分离——轨迹通过真实模拟器展开，而梯度则通过一个可微分的模拟器学习模型进行反向传播计算。

✨ **主要贡献**：该方法在模拟器梯度不可用时仍能实现高效且稳定的一阶策略优化，兼具SHAC等专用优化器的样本效率与PPO等通用方法的广泛适用性，并在基准控制任务和真实Go2四足机器人（包括四足与双足运动任务）上验证了有效性。
