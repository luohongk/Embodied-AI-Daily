# CodeCoR: An LLM-Based Self-Reflective Multi-Agent Framework for Code Generation

- arXiv: https://arxiv.org/abs/2501.07811v2
- arXiv ID: 2501.07811

---

🔍 **核心问题**：现有基于LLM的多智能体代码生成框架中，各智能体顺序执行任务时容易因中间输出质量不佳导致错误累积，影响最终代码的语法和语义正确性。

🛠 **方法**：提出CodeCoR框架，包含提示生成、代码生成、测试用例生成和修复建议四个智能体，每个智能体生成多个输出并剪除低质量结果；未通过测试的代码由修复智能体提供建议后重新生成，最终返回通过测试用例最多的代码。

✨ **主要贡献**：在HumanEval、HumanEval-ET、MBPP和MBPP-ET四个数据集上，CodeCoR的平均Pass@1得分达77.13%，优于CodeCoT和MapCoder等基线方法，验证了通过剪枝中间输出减少错误传播的有效性。
