# HoloCount: A Holistic Visual Counting Benchmark for MLLMs

- arXiv: https://arxiv.org/abs/2607.06420v1
- arXiv ID: 2607.06420

---

🔍 **核心问题**：现有视觉计数基准仅关注简单场景下的基础感知，无法评估多模态大语言模型（MLLMs）在逻辑约束或对抗条件下的复杂计数失败模式（如数值幻觉）。

🛠 **方法**：提出HoloCount基准，基于三级分层分类法设计，包含语义计数（原子/属性枚举）、分析计数（空间/集合逻辑推理）和鲁棒性测试（高密度场景、语言偏见等对抗条件）。

✨ **主要贡献**：通过对20余种先进MLLMs的全面评估，揭示了从感知到复杂推理及对抗场景中模型性能显著下降的差距，为开发更可靠的多模态系统提供了系统性诊断框架和路线图。
