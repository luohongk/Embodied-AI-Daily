# Topological Neural Dynamics: A Neuron-wise Framework for Sequence Modeling

- arXiv: https://arxiv.org/abs/2606.21295v1
- arXiv ID: 2606.21295

---

🔍 **核心问题**：现有序列模型（如RNN、LSTM、Transformer）均采用层内神经元共享参数的层状动态机制，限制了神经元独立演化能力，难以模拟复杂系统中局部单元通过结构化连接涌现全局行为的特性。

🛠 **方法**：提出拓扑神经动力学（TND）框架，将计算从层状动态转向神经元级动态。TND将神经网络表示为有向神经元图、交互算子和局部动态函数，每个神经元独立演化，通过显式图拓扑实现集体计算。

✨ **主要贡献**：在单玩家Pong行为克隆任务中，TND相比RNN、LSTM、CfC和Transformer等基线模型，实现了最佳接球率（平均每轮17.47次连续接球，超过最强基线三倍以上），验证了神经元级动态作为序列建模有效归纳偏置的潜力。
