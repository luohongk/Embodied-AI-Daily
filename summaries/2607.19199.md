# Conservative Query and Adaptive Regularization for Offline RL Under Uncertainty Estimation

- arXiv: https://arxiv.org/abs/2607.19199v1
- arXiv ID: 2607.19199

---

🔍 **核心问题**：离线强化学习中，如何有效利用专家偏好反馈来提升策略性能，同时避免查询选择不准确和反馈利用不充分导致的策略更新不稳定问题。

🛠 **方法**：提出一种轻量级框架，使用Morse网络估计策略动作相对于离线数据集的不确定性，基于此设计保守查询策略（仅查询靠近数据集的动作）和不确定性感知的自适应正则化方法（动态调整策略优化中的数据约束），并集成到CQL算法中。

✨ **主要贡献**：在D4RL基准测试上，该方法在多种任务中取得了优越或具有竞争力的性能，有效提升了偏好查询的稳定性和反馈利用的效率。
