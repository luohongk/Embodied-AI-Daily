# Multi-Robot Motion Planning from Vision and Language using Heat-Inspired Diffusion

- arXiv: https://arxiv.org/abs/2512.13090v2
- arXiv ID: 2512.13090

---

🔍 **核心问题**：现有扩散模型在多机器人运动规划中难以灵活处理语言条件任务，且推理成本高、泛化能力差，缺乏对几何可达性的推理机制。

🛠 **方法**：提出语言条件热启发扩散（LHD）框架，结合CLIP视觉语言模型的语义先验与碰撞避免扩散核作为物理归纳偏置，在端到端视觉基础上生成语言条件且无碰撞的轨迹。

✨ **主要贡献**：LHD在无需显式障碍物信息的情况下，通过可达性引导实现语言命令的语义匹配，在真实世界地图和机器人实验中显著提升成功率并降低规划延迟。
