# Learning Terrain Aware Bipedal Locomotion via Reduced Dimensional Perceptual Representations

- arXiv: https://arxiv.org/abs/2512.12993v1
- arXiv ID: 2512.12993

---

🔍 **核心问题**：解决双足机器人在复杂地形上实时生成稳健步态时，传统端到端方法难以高效融合地形感知与低维状态表示的问题。

🛠 **方法**：提出分层强化学习框架，利用卷积变分自编码器（CNN-VAE）提取低维潜在地形编码，结合降阶机器人动力学模型，并通过历史观测序列增强鲁棒性；同时引入蒸馏方法从深度相机图像直接学习潜在表示。

✨ **主要贡献**：系统分析了潜在空间维度对学习效率与策略鲁棒性的影响；提出历史感知机制提升地形适应性；通过仿真与真实传感器数据对比验证了硬件部署可行性。
