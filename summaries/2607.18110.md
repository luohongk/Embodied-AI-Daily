# LLM-as-a-Coach: Experiential Learning for Non-Verifiable Tasks

- arXiv: https://arxiv.org/abs/2607.18110v1
- arXiv ID: 2607.18110

---

🔍 **核心问题**：针对开放型非可验证任务，传统强化学习将基于评分标准的评估压缩为标量奖励，丢失了丰富的文本反馈，且无法区分不同质量水平的回答。

🛠 **方法**：提出**体验式学习**，将“LLM作为裁判”中的反馈模型改造为“LLM作为教练”，教练将对每个策略生成回答的评估提炼为可迁移的体验知识，用于条件化教师模型，并通过策略内上下文蒸馏被策略内化。

✨ **主要贡献**：相比标量奖励，该方法提供更高带宽的密集监督，保留高质量回答间的细粒度偏好；在多种策略族上，始终优于基于评分标准的强化学习，尤其在分布外泛化上表现更佳，并能缓解奖励黑客问题。
