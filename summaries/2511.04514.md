# Linear Mode Connectivity under Data Shifts for Deep Ensembles of Image Classifiers

- arXiv: https://arxiv.org/abs/2511.04514v2
- arXiv ID: 2511.04514

---

🔍 **核心问题**：研究深度集成图像分类器在数据分布偏移下的线性模式连通性（LMC）现象，并探索缓解数据偏移对LMC影响的条件。  
🛠 **方法**：通过实验分析数据偏移作为随机梯度噪声的额外来源，采用小学习率和大批量大小来降低噪声，并观察模型收敛到相同局部最小值或不同损失景观区域的行为。  
✨ **主要贡献**：揭示了数据偏移下LMC的稳定性条件（小学习率、大批量），指出LMC模型虽易产生相似错误，但能平衡训练效率与集成多样性，为提升深度集成性能提供指导。
