# UniSkip-Mamba: A Frequency-Aware State Space Model for Audio-Visual Temporal Forgery Localization

- arXiv: https://arxiv.org/abs/2607.04498v1
- arXiv ID: 2607.04498

---

🔍 **核心问题**：现有音视频时序伪造定位方法对全频段信息不加区分地处理，导致模型过拟合高频噪声，在真实数据退化场景下鲁棒性不足。

🛠 **方法**：提出UniSkip-Mamba框架，包含统一多模态序列融合以保持跨模态相位关系，以及基于Skip-Scanning Mamba模块的频域感知正则化，通过Group-Scan-Merge机制引导模型聚焦低频（0-0.15）判别模式。

✨ **主要贡献**：在LAV-DF和AV-Deepfake1M数据集上分别实现63.4% AP@0.95（+9.8%）和63.58% mAP（+14.32%）的SOTA性能，推理速度提升6倍，并从信号处理角度理论解释了跳扫描机制提升准确性与鲁棒性的原因。
