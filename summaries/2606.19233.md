# Mobile Pedipulation for Object Sliding via Hierarchical Control on a Wheeled Bipedal Robot

- arXiv: https://arxiv.org/abs/2606.19233v1
- arXiv ID: 2606.19233

---

🔍 **核心问题**：如何让轮式双足机器人利用其轮式腿部，在保持稳定运动的同时完成平面物体滑动任务（如推拉物体）。

🛠 **方法**：提出一种分层控制框架，基于降阶三刚体动力学模型设计非线性模型预测控制器，同时调节机器人运动与交互力，并集成轨迹优化运动规划器处理地面-物体接触的粘滑转换。

✨ **主要贡献**：首次在真实硬件上验证了轮式双足机器人的两种腿部操作运动（“滑行”和“侧滑”），成功从桌下取出1kg物体，并通过滑行动作将4kg物体推移0.228米。
