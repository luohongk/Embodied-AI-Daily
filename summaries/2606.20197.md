# Stable Transformer-Actor-Critic Model Predictive Control: A Contraction Analysis Approach

- arXiv: https://arxiv.org/abs/2606.20197v1
- arXiv ID: 2606.20197

---

🔍 **核心问题**：Actor-Critic模型预测控制（MPC）能处理复杂非凸控制问题，但难以保证基于序列学习的闭环稳定性。

🛠 **方法**：提出一种Transformer-Actor-Critic MPC架构，通过证明Transformer网络满足全局增量输入-状态稳定性（δISS），并利用黎曼收缩理论分析物理系统与预测神经网络的互联动态，将理论界作为训练正则化项。

✨ **主要贡献**：首次为Transformer-Actor-Critic MPC提供形式化的鲁棒性保证，并在非线性3D无人机模型上验证了目标到达与避障任务的有效性。
