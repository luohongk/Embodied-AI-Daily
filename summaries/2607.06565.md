# ELSA3D: Elastic Semantic Anchoring for Unified 3D Understanding and Generation

- arXiv: https://arxiv.org/abs/2607.06565v1
- arXiv ID: 2607.06565

---

🔍 **核心问题**：现有统一3D基础模型将文本与3D标记简单拼接，导致粗粒度结构线索与细粒度几何细节混杂，跨模态交互不够精确。

🛠 **方法**：提出ELSA3D，采用尺度感知八叉树分词器表示几何，并引入锚点标记（Anchor Tokens）作为稀疏跨模态单元，通过轻量级逐块路由器弹性选择文本标记在合适几何尺度上实例化锚点，实现语义与几何的联合对齐。

✨ **主要贡献**：在图像到3D生成、文本到3D生成和3D描述任务上达到最先进性能，相比非弹性版本模型，计算量和推理延迟均减半。
