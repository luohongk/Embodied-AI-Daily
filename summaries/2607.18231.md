# FM-VLA: Force-based Memory for Vision-Language-Action Models in Contact-Rich Manipulation

- arXiv: https://arxiv.org/abs/2607.18231v1
- arXiv ID: 2607.18231

---

🔍 **核心问题**：解决接触丰富操作中，基于视觉的记忆方法在时间事件视觉模糊（如多次小幅度按按钮）时无法有效推理非马尔可夫任务的问题。

🛠 **方法**：提出FM-VLA模型，利用变分自编码器（VAE）将力历史编码为紧凑的力记忆令牌，并与短状态历史一起作为额外条件令牌注入动作专家模块，实现基于力的时间上下文推理。

✨ **主要贡献**：在隐藏块寻找、按钮按压和特定次数擦拭盘子三项任务中，轻量级力记忆以最小推理开销实现超过80%的成功率，显著优于基线方法。
