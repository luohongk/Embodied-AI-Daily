# SWAP: Symmetric Equivariant World-Model for Agile Robot Parkour

- arXiv: https://arxiv.org/abs/2606.19928v1
- arXiv ID: 2606.19928

---

🔍 **核心问题**：现有潜在世界模型在敏捷机器人跑酷任务中，因缺乏对称性先验而冗余编码左右对称交互，导致学习负担加重且难以捕捉几何规律，限制了潜在空间对下游策略的效率。

🛠 **方法**：提出SWAP框架，一种端到端等变对称世界模型，将对称性直接嵌入世界模型和演员-评论家网络中。

✨ **主要贡献**：在真实世界测试中，机器人跨越2.13米间隙并攀爬1.63米平台，创下四足机器人跑酷纪录；同时展现出对未见镜像地形的鲁棒几何泛化能力，以及跨多样户外环境的零样本迁移能力。
