# Multi-Modal, Multi-Environment Machine Teaching for Robust Reward Learning

- arXiv: https://arxiv.org/abs/2607.08647v1
- arXiv ID: 2607.08647

---

🔍 **核心问题**：如何通过多环境、多模态的机器教学，学习在不同动态环境中都能泛化的鲁棒奖励函数，避免因单一环境过拟合导致的部署失败。

🛠 **方法**：提出分层机器教学算法，先贪婪选择能暴露互补奖励约束的多个环境，再在这些环境中策略性地查询低成本反馈（如比较型反馈），以高效约束奖励函数。

✨ **主要贡献**：理论上证明比较型反馈在无限数据下提供最强全局约束；实验表明该方法在相同反馈预算下，比均匀教学基线显著降低遗憾值，并提升对未见环境的泛化能力。
