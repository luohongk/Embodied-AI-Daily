# Graph Optimization Foundation Model: Tokenizing Graph via A Language-Model Paradigm

- arXiv: https://arxiv.org/abs/2509.24256v2
- arXiv ID: 2509.24256

---

🔍 **核心问题**：解决图结构上的距离优化问题，克服语言模型统计灵活性与图组合约束之间的冲突，实现图优化问题的通用求解。

🛠 **方法**：提出图基础模型（GFM），采用类似大语言模型的自监督预训练范式，在随机游走生成的路径上进行训练，使模型内化图的拓扑与组合规则，并利用预训练模型作为基础，通过简单生成式启发方法解决多种优化任务。

✨ **主要贡献**：首次实现一个框架解决所有基于距离的图优化问题；在20至893个节点的网络上，GFM在多种优化任务中达到与专用求解器竞争的性能，同时推理速度显著更快；开创了将预训练-迁移范式应用于图优化的新路径。
