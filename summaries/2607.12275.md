# Flatness-Preserving Residual Learning for Real-Time Tight Quadrotor Formation Flight

- arXiv: https://arxiv.org/abs/2607.12275v1
- arXiv ID: 2607.12275

---

🔍 **核心问题**：四旋翼飞行器在紧密编队飞行时，受下洗气流等复杂空气动力学干扰影响，可能导致灾难性碰撞，现有模型难以实时补偿这些效应。

🛠 **方法**：提出一种物理信息引导的残差动力学学习框架，在捕捉复杂气动交互的同时保持多四旋翼系统的微分平坦性，并基于此设计计算高效的反馈线性化控制器，通过前馈补偿消除气动干扰。

✨ **主要贡献**：硬件实验显示平均跟踪误差降低31%，计算量比非线性模型预测控制（NMPC）低一个数量级，首次实现仅需30秒训练数据和5ms循环率的稳定紧密编队飞行。
