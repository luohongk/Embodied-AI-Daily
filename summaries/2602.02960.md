# Embodiment-Aware Generalist Specialist Distillation for Unified Humanoid Whole-Body Control

- arXiv: https://arxiv.org/abs/2602.02960v2
- arXiv ID: 2602.02960

---

🔍 **核心问题**：现有的人形机器人全身控制策略通常针对单一机器人设计，难以在不同动力学、自由度或运动学拓扑的异构人形机器人间迁移，且无法支持从行走、蹲下到倾斜等丰富行为。

🛠 **方法**：提出EAGLE框架，通过迭代的通用专家蒸馏过程，从当前通用策略分叉出针对特定机器人的专家，经各自机器人优化后，将新技能蒸馏回通用策略，并在多机器人集合上训练，循环直至性能收敛。

✨ **主要贡献**：首次实现无需逐机器人奖励调优的单一统一策略，可控制多种异构人形机器人（如Unitree H1、G1、Fourier N1），在仿真和真实环境中均展现出高跟踪精度和鲁棒性，推动了可扩展的机群级人形控制发展。
