# Towards Predictive, Aligned, and Scalable Robot Learning

- arXiv: https://arxiv.org/abs/2607.11270v1
- arXiv ID: 2607.11270

---

🔍 **核心问题**：现有机器人学习方法在动作生成中缺乏对物理动态的预测推理能力，且动作表示与下游控制性能之间存在对齐不足的问题。

🛠 **方法**：提出Lumo-2模型，通过潜在世界-动作模型在潜在空间推理世界动态生成动作，并采用多阶段模态预对齐策略，逐步将动作表示与潜在世界动态、视觉和语言对齐。

✨ **主要贡献**：揭示了潜在空间几何对动作生成质量的关键作用，通过结构化多模态对齐和预测推理显著提升机器人控制性能，在长时程和灵巧操作等复杂任务上超越现有基线模型。
