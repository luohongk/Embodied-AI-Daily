# MemLearner: Learning to Query Context memory for Video World Models

- arXiv: https://arxiv.org/abs/2606.31734v1
- arXiv ID: 2606.31734

---

🔍 **核心问题**：视频世界模型在长时间生成中缺乏记忆，导致场景不一致，尤其在物体遮挡和动态场景下表现不佳。

🛠 **方法**：提出MemLearner，一种基于学习的自适应上下文查询方法，利用查询令牌连接上下文与预测令牌，并借助视频生成模型自身的预训练视觉先验进行上下文检索，无需从头训练额外模块。

✨ **主要贡献**：在包含遮挡和动态物体的长视频数据集上，MemLearner显著提升了场景一致性和记忆能力，优于先前方法，并提出了结合标注渲染视频与未标注真实视频的多数据集训练策略。
