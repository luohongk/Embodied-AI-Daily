# MindDrive: A Vision-Language-Action Model for Autonomous Driving via Online Reinforcement Learning

- arXiv: https://arxiv.org/abs/2512.13636v4
- arXiv ID: 2512.13636

---

🔍 **核心问题**：当前自动驾驶中的视觉-语言-动作（VLA）模型主要依赖模仿学习，存在分布偏移和因果混淆问题，而在线强化学习在连续动作空间中探索效率低下，难以直接应用。

🛠 **方法**：提出MindDrive框架，使用一个大语言模型（LLM）搭配两组不同的LoRA参数，分别作为决策专家（进行场景推理和驾驶决策）和动作专家（将语言决策映射为可行轨迹），通过将轨迹级奖励反馈到推理空间，实现在离散语言决策上的试错学习，而非直接在连续动作空间中操作。

✨ **主要贡献**：首次证明在线强化学习在自动驾驶VLA模型中的有效性，在Bench2Drive基准上使用轻量级Qwen-0.5B LLM实现了78.04的驾驶得分和55.09%的成功率，并平衡了复杂场景下的最优决策、类人驾驶行为和高效探索。
