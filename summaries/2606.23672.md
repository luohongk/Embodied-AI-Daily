# Teaching LLMs String Matching, Backtracking, and Error Recovery to Deduce Bases and Truth Tables for the Combinatorially Exploding Bit Manipulation Puzzles

- arXiv: https://arxiv.org/abs/2606.23672v1
- arXiv ID: 2606.23672

---

🔍 **核心问题**：解决大语言模型在比特操作谜题中因布尔逻辑和算术模拟导致的幻觉问题，以及位运算组合爆炸带来的计算困难。

🛠 **方法**：放弃算术逻辑，采用字符串相似性、结构化搜索和自主错误恢复；通过基与真值表公式化、回溯深度优先搜索及错误恢复、比特分词与交互推理监督微调。

✨ **主要贡献**：提出基于字符串相似性的基选择方法，实现无算术的真值表推导；引入回溯搜索与错误恢复机制；通过比特分词和动态掩码训练模型自主推理，最终在验证集上达到96%以上准确率，获得竞赛第7名。
