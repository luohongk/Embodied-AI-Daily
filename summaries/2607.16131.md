# ToolSciVer: Multimodal Scientific Claim Verification with Visual Tool Augmented Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.16131v1
- arXiv ID: 2607.16131

---

🔍 **核心问题**：多模态科学声明验证（MSCV）中，现有方法难以准确定位论文中的视觉证据（如图表、表格），无法有效解析结构化科学图像，并整合多模态信息进行可靠推理。

🛠 **方法**：提出ToolSciVer框架，为视觉语言模型（VLM）配备三种类型感知的视觉工具（表格行列聚焦、图表结构解析、高分辨率区域缩放），并通过GRPO强化学习策略，结合答案正确性、格式有效性、工具使用效率等复合奖励进行训练。

✨ **主要贡献**：首次引入工具增强框架解决MSCV任务，在SciVer和MuSciClaims数据集上，基于Qwen、InternVL、Gemma三个模型家族的五个VLM实验表明，该方法在提示学习和强化学习基线中均取得最优性能，验证了类型感知工具学习对科学声明验证的有效性。
