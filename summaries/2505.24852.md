# Chameleon: A Multiplier-Free Temporal Convolutional Network Accelerator for End-to-End Few-Shot and Continual Learning from Sequential Data

- arXiv: https://arxiv.org/abs/2505.24852v3
- arXiv ID: 2505.24852

---

🔍 **核心问题**：在边缘设备上实现低功耗、端到端的片上学习，特别是从少量真实世界时序数据中进行少样本学习和持续学习，同时保持推理效率。

🛠 **方法**：提出Chameleon加速器，采用无乘法器的时序卷积网络（TCN）架构，通过统一的学习与推理设计（仅增加0.5%面积开销）和双模式计算阵列，支持端到端少样本学习、持续学习及16-kHz原始音频推理。

✨ **主要贡献**：在40nm CMOS工艺下，首次实现片上端到端时序数据少样本学习（Omniglot 5-way 1-shot达96.8%）和持续学习（250类10-shot最终准确率82.2%），同时推理功耗仅3.1μW，在12类语音命令数据集上保持93.3%准确率，且峰值性能比现有推理专用加速器高4.3倍。
