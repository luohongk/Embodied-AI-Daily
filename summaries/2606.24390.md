# Female-RHINO: A Real-Time Scanner-Integrated Framework for Automated Quantitative Uterine MRI Analysis and Structured Reporting

- arXiv: https://arxiv.org/abs/2606.24390v1
- arXiv ID: 2606.24390

---

🔍 **核心问题**：子宫MRI标准化评估因解剖变异、观察者依赖及缺乏自动化分析工具而面临挑战。  
🛠 **方法**：提出Female-RHINO框架，集成MRI扫描仪实时通信与深度学习模型，自动分割子宫、检测纤维瘤等病灶，并提取6个解剖标志点，生成结构化报告。  
✨ **主要贡献**：基于500+多中心数据集训练，子宫分割Dice系数0.82，纤维瘤0.80，标志点检测径向误差3.7mm，端到端处理<70秒，实现扫描期间实时、标准化分析，提升临床工作流效率。
