# Toward Autonomous Soft Robotic Endovascular Navigation via Imitation Learning

- arXiv: https://arxiv.org/abs/2510.09497v2
- arXiv ID: 2510.09497

---

🔍 **核心问题**：解决软体机器人导丝在血管内介入手术中自主导航的建模与控制难题，提升动脉瘤靶向任务的精准性和安全性。

🛠 **方法**：构建大规模2D投影环境，基于Transformer的模仿学习框架，结合目标条件、相对动作输出及自动造影剂注入，在36种模块化分叉几何体上训练647次演示，并在未见过的血管几何体上测试。

✨ **主要贡献**：在未见过几何体上实现83%的动脉瘤到达成功率，优于多个基线方法；在患者衍生几何体上达到75%成功率；通过消融实验验证了各设计模块的有效性。
