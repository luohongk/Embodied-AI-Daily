# Agora: Enhancing LLM Agent Reasoning Via Auction-Based Task Allocation

- arXiv: https://arxiv.org/abs/2607.09600v1
- arXiv ID: 2607.09600

---

🔍 **核心问题**：现有LLM智能体框架在调用专家模型和工具时，仅基于粗粒度的任务-功能匹配，忽略了性能差异和成本效率，导致关键推理步骤可能被分配给能力不足或过度自信的模型。

🛠 **方法**：提出Agora框架，引入激励相容的拍卖机制，将推理步骤视为可交易物品，让专家模型根据修正后的能力进行竞价，从而将关键逻辑路由到最合适的求解器。

✨ **主要贡献**：在五个基准测试上，Agora在可比候选池下优于单模型、路由和级联基线，并通过单一拍卖参数实现了可控的成本-质量权衡。
