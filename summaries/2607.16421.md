# When to Plan: Learning to Select Between Reactive Control and Deliberative Planning

- arXiv: https://arxiv.org/abs/2607.16421v1
- arXiv ID: 2607.16421

---

🔍 **核心问题**：如何让智能体学会在快速反应控制与慢速规划决策之间智能切换，以平衡计算效率与决策质量。

🛠 **方法**：提出一种基于强化学习的元推理方法，通过训练一个元策略，利用反应策略的不确定性评分来决定何时需要调用耗时的规划计算。

✨ **主要贡献**：首次将反应策略的不确定性作为元推理的决策依据，实验证明该方法能有效学习何时依赖反应控制、何时需要规划，并随反应策略改进自动向全反应控制过渡。
