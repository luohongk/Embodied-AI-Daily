# R2RDreamer: 3D-aware Data Augmentation for Spatially-generalized 2D Manipulation Policies

- arXiv: https://arxiv.org/abs/2606.17040v1
- arXiv ID: 2606.17040

---

🔍 **核心问题**：如何通过数据增强提升模仿学习策略在空间泛化（如物体姿态、机器人配置和相机视角变化）上的能力，同时避免复杂3D场景设置或仿真到现实的差距。

🛠 **方法**：提出R2RDreamer框架，先对真实演示中的不完整物体点云和末端执行器轨迹进行轻量级3D增强，再通过遮挡感知投影将编辑后的场景映射到掩码图像空间，最后用密集控制图像到视频模型生成时间连贯的RGB观测。

✨ **主要贡献**：实现了真实到真实的数据增强，无需完整3D场景解析或几何补全，在2D扩散策略和视觉-语言-动作策略上显著提升空间泛化能力，验证了3D编辑、遮挡感知投影和视频补全三个模块的有效性。
