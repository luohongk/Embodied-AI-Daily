# HoloAgent-0: A Unified Embodied Agent Framework with 3D Spatial Memory

- arXiv: https://arxiv.org/abs/2606.23565v1
- arXiv ID: 2606.23565

---

🔍 **核心问题**：将LLM智能体的闭环执行逻辑（推理-工具调用-反馈-修正）扩展到物理机器人时，面临物理执行连续、依赖具体形态、不确定性和安全约束等挑战，现有系统各能力模块松散耦合。

🛠 **方法**：提出HoloAgent-0统一具身智能体框架，包含三层耦合结构：Embodied AgentOS（闭环执行）、3D空间记忆（物理世界锚定）和具身技能（机器人动作），将语言指令转化为可执行技能图并调度资源。

✨ **主要贡献**：在真实硬件上部署并验证了空间记忆、长程导航、闭环执行能力，涵盖运动生成、物体搜索、跨机器人协调和移动操作等任务，实现了从数字环境到物理世界的统一智能体框架。
