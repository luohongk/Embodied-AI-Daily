## 0. 概览（Concise Summary）

- **主题（Topic）**：机器人学、假肢与康复工程、强化学习、软体机器人建模。具体为：利用强化学习控制一个包含柔性假肢的混合刚-柔耦合系统，模拟单侧小腿截肢者的跑步运动。
- **问题（Problem）**：运动假肢（如碳纤维叶片弹簧式假肢）的设计和选择通常依赖“试错法”，缺乏基于全身动力学和假肢形变交互的量化分析工具。现有研究要么简化假肢模型（如刚性多连杆），要么计算成本过高（如有限元法），难以在合理计算成本下进行全身动态仿真。
- **方法（Method）**：提出一个结合**混合刚-柔耦合系统（Hybrid-link System）** 和**强化学习（RL）** 的仿真框架。混合系统使用**分段恒定应变（PCS）模型**精确且高效地模拟假肢的柔性形变，并与刚性骨骼模型耦合。RL算法（PPO）通过**模仿学习**，基于运动捕捉数据训练控制策略，驱动骨骼模型运动，同时与柔性假肢动态交互。
- **创新（Innovation）**：
    1.  **模型创新**：首次将PCS模型与RL框架结合，用于全身跑步运动仿真，实现了对假肢形变的高保真、低计算成本建模。
    2.  **方法创新**：采用“预训练（刚性模型）+微调（混合模型）”的两阶段训练策略，显著提高了学习效率，解决了混合系统仿真计算成本高的问题。
    3.  **应用创新**：成功仿真了在不同虚拟假肢刚度条件下的自适应跑步运动，并分析了代谢成本（COT），其结果与文献报道一致，验证了该方法用于虚拟假肢参数评估的潜力。
- **意义（Significance）**：提供了一个强大的虚拟仿真平台，可以量化分析假肢参数（如刚度）对跑步性能（如能量消耗、地面反作用力）的影响。这有望为运动员提供个性化的假肢设计和选型指导，减少对昂贵且耗时的物理试错的依赖。
- **一句话总结**：本文提出了一种结合混合刚-柔耦合系统与强化学习的仿真方法，能够高效、逼真地模拟截肢者使用柔性运动假肢的跑步过程，并用于分析假肢刚度对运动表现的影响。

## 1. 研究动机（Motivation）

- **问题与空白**：运动假肢（如叶片弹簧式假肢）的设计和选择目前主要依赖运动员的“试错”和主观感受。这种方法效率低、成本高，且难以保证运动员总能找到最适合自己的假肢。虽然已有研究尝试通过仿真分析假肢影响，但存在明显不足：
    - **模型过于简化**：如将假肢简化为线性弹簧或刚性多连杆，无法准确捕捉其复杂的形变行为及其与人体运动的动态交互。
    - **计算成本过高**：如使用有限元法（FEM）虽然精度高，但计算量巨大，难以用于需要大量迭代的强化学习训练或长时间动态仿真。
    - **控制模型简单**：如使用弹簧倒立摆或PD控制，无法模拟人体复杂的运动控制系统，限制了仿真的逼真度和适应性。
- **重要性**：开发一个既能精确模拟假肢柔性形变，又能高效进行全身动态仿真，并能生成逼真运动控制的平台，对于理解假肢-人体交互机制、优化假肢设计、提升运动员表现至关重要。
- **挑战**：主要挑战在于如何平衡**模型精度**（假肢形变）与**计算效率**（RL训练需要大量仿真），以及如何生成**逼真且鲁棒**的运动控制策略，使其能适应不同的假肢参数。

## 2. 创新点（Innovation）

- **关键创新**：提出了一个**基于强化学习的混合刚-柔耦合系统仿真框架**。
- **与现有方案的不同**：
    1.  **假肢建模**：不同于以往使用刚性多连杆或线性弹簧的简化模型，本文采用**分段恒定应变（PCS）模型**。该模型源于软体机器人学，能够以较少的自由度（18 DOF）高效、准确地模拟叶片弹簧假肢的连续、非线性形变，在精度和计算成本之间取得了良好平衡。
    2.  **控制策略学习**：不同于传统的逆动力学（ID）或简单PD控制，本文采用**强化学习（RL）** 来学习控制策略。这使得模型能够生成更自然、更鲁棒的运动，并能通过模仿学习复现特定受试者的运动模式。
    3.  **训练策略**：提出**两阶段训练法**。先在计算成本低的**刚性假肢模型**上预训练控制策略，再在更精确的**混合系统（含PCS模型）** 上进行微调。这极大地加速了收敛，解决了直接在高成本混合系统上训练RL的难题。
- **为什么能奏效**：
    - PCS模型提供了物理上更真实的假肢动力学，使得RL能够学习到利用假肢弹性形变的运动策略，这是刚性模型无法做到的。
    - 两阶段训练策略巧妙地利用了刚性模型快速迭代的优势，为混合系统的微调提供了良好的初始策略，从而在保证最终模型精度的同时，大幅降低了总训练时间。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

整个框架分为两大核心部分：**混合刚-柔耦合系统模型**和**强化学习控制器**。

1.  **混合系统模型**：如图1(b)所示，该系统将人体骨骼建模为**刚性多连杆系统**（9 DOF），将运动假肢建模为**PCS柔性模型**（18 DOF）。两者通过动力学耦合，形成一个33 DOF的浮动基座系统。该模型能够计算假肢形变与人体运动之间的相互作用力。
2.  **强化学习控制器**：如图2所示，RL智能体（Policy）接收当前**状态**（$s_t$，包括基座位姿、速度、关节角度、角速度），输出**动作**（$a_t$，即目标关节角度）。该动作通过一个底层PD控制器转换为关节力矩，驱动刚性骨骼模型运动。骨骼运动与柔性假肢交互，产生新的状态和奖励。RL的目标是最大化累积奖励，该奖励函数设计为模仿运动捕捉数据。

### 3.2 关键算法与数学推导

1.  **PCS模型**：
    - 假肢的构型由齐次变换矩阵 $H(s) \in SE(3)$ 表示，其中 $s$ 是沿中心轴的弧长坐标。
    - **应变向量** $\xi(s) \in \mathbb{R}^6$ 定义为：
      $$[\xi(s)\times] := H^{-1} \frac{\partial H}{\partial s} \in \mathfrak{se}(3)$$
    - 将假肢分为 $N$ 段，假设每段内应变恒定。第 $i$ 段的应变为 $\xi_i$。
    - 给定 $\xi_i$，第 $i$ 段的构型为：
      $$H(s) = H_{i-1} \exp\{(s - L_{i-1})[\xi_i\times]\} \quad (L_{i-1} \leq s < L_i)$$
    - PCS模型的广义坐标为所有段的应变向量组合：$q_S = [\xi_1^T, \xi_2^T, \cdots, \xi_N^T]^T$。

2.  **混合系统动力学**：
    - 浮动基座系统的运动方程为：
      $$M \ddot{q} + b = \tau + \sum_i J_{C,i}^T f_{C,i}$$
      其中 $M$ 是惯性矩阵，$b$ 是偏置项（科里奥利力、离心力、重力），$\tau$ 是广义力，$J_{C,i}$ 和 $f_{C,i}$ 是接触点的雅可比矩阵和接触力。
    - 假肢的广义力 $\tau_S$ 由被动粘弹性产生：
      $$\tau_S = K(q_{S,eq} - q_S) - D \dot{q}_S$$
      其中 $K$ 和 $D$ 是刚度和阻尼矩阵，$q_{S,eq}$ 是无载荷时的应变。

3.  **强化学习（PPO）**：
    - 目标是最大化累积折扣奖励 $J(\theta) = \mathbb{E}_{\pi_\theta}[\sum_{t=0}^\infty \gamma^t r_t]$。
    - 策略 $\pi_\theta(a_t|s_t)$ 是一个高斯分布，由两个隐藏层（各64单元）的MLP实现。
    - 使用PPO算法进行稳定更新。

4.  **奖励函数（模仿学习）**：
    - 总奖励 $r_t = \sum_{i=\{q,v,e,0\}} w_i r_i$。
    - 定义指数函数 $r(\beta, x_{ref}, x) = \exp(-\beta \|x_{ref} - x\|^2)$。
    - $r_q$ 和 $r_v$：模仿关节角度 $q_R$ 和角速度 $\dot{q}_R$。
    - $r_e$：模仿末端执行器（手、脚、头）的位置 $p$。
    - $r_0$：模仿基座（骨盆）的位姿 $H_0$ 和速度。

### 3.3 模型、训练与数据集

- **模型**：
    - **骨骼模型**：33 DOF，其中9个为关节（髋、膝、踝的屈伸），6个为浮动基座，18个为假肢PCS模型。骨盆被约束在矢状面内。
    - **假肢模型**：基于Runner 1E91假肢，使用PCS模型划分为6段，共18 DOF。其粘弹性参数（$K, D$）通过运动捕捉和地面反作用力数据估计得到。
- **训练**：
    - **算法**：PPO。
    - **策略**：两阶段训练。第一阶段在**刚性假肢模型**上预训练82,000个episode（33小时）。第二阶段在**混合系统模型**上微调88,000个episode（102.5小时）。
    - **并行计算**：使用多进程（刚性模型8进程，混合模型16进程）收集经验数据，加速训练。
    - **超参数**：buffer size=4096, learning rate=0.0003, discount factor=0.95。
- **数据集**：
    - 使用光学运动捕捉系统（200 Hz）和测力台（1 kHz）采集一名15岁男性单侧小腿截肢者（175cm, 53kg）的步行和跑步数据。
    - 标记点设置遵循Helen Hayes模型，并在假肢上额外放置了24个标记点。
    - 通过逆运动学求解得到关节轨迹，作为模仿学习的参考数据。

### 3.4 实验设置与结果

- **实验1：基础运动仿真**（图3，图4，图5，表III）
    - **设置**：训练RL策略，生成步行（1.2 m/s）、跑步（3.0 m/s）和冲刺（5.0 m/s）运动。
    - **结果**：
        - 成功生成了利用假肢弹性的全身运动（图3）。
        - 混合系统模型在跑步任务上的最终回报值低于刚性模型，表明刚性模型无法准确复现跑步运动（图4）。
        - 仿真结果与实验数据在关节角度、假肢应变和地面反作用力（GRF）上趋势一致。RMSE分别为9.2%（关节角度）、16.1%（应变）、10.9%（GRF）（图5，表III）。这表明即使假肢应变未直接包含在状态和奖励中，模型也能学习到正确的形变模式。

- **实验2：不同假肢刚度的影响**（图6，图7，表IV）
    - **设置**：定义三种刚度条件：Nominal（受试者使用的标准刚度）、Compliant（-10%）、Stiff（+10%）。在Nominal条件下训练的策略基础上，对其他两种条件进行微调。
    - **结果**：
        - **运动学**：不同刚度下，关节角度轨迹保持相似，表明策略具有适应性（图6a）。
        - **假肢形变**：Compliant假肢形变更大（比Stiff大32%）（图6b）。
        - **地面反作用力**：Compliant假肢的GRF峰值最低（比Stiff低35%）（图6c）。
        - **能量消耗**：Nominal条件下的代谢成本（CoT）最低（4.3 J/kg·m），且所有条件下的CoT值均落在文献[1]报告的实验测量范围内（图7）。这表明存在一个最优刚度，既能避免过软导致形变过大、能量损耗，也能避免过硬导致冲击力过大、需要更多推进力。

- **实验3：策略鲁棒性分析**（图8）
    - **设置**：在Nominal策略下，将假肢刚度从-40%变化到+50%，观察回报值和episode长度。
    - **结果**：在刚度变化约±20%范围内，策略表现稳定。超出此范围，性能下降。这对应于约±20kg体重变化，覆盖了多个假肢型号的适用范围。

### 3.5 技术细节

- **接触模型**：使用**完全非弹性碰撞模型**结合**弹簧-阻尼模型**计算足地接触力。弹簧系数为8.0 kN/m，阻尼为Rayleigh阻尼。
- **数值积分**：采用**隐式数值积分法**，时间步长为 $2 \times 10^{-4}$ s，以保证混合系统仿真的稳定性。
- **PD控制器**：底层PD控制器频率为30 Hz，与人类运动神经元放电频率一致。增益 $K_P = 100E, K_D = E$。
- **实现**：RL使用PyTorch和Stable-Baselines3，混合系统动力学计算使用C++实现并通过Pybind11与Python连接。
- **附录分析**：
    - **假肢模型简化影响**：将PCS模型简化为刚性多连杆，会导致GRF（尤其是侧向力Fy）的估计误差显著增大（图9）。
    - **骨骼模型精细化影响**：增加髋关节内收/外展自由度（从9 DOF到11 DOF），对GRF估计精度提升有限（图10）。
    - **接触参数敏感性**：改变弹簧-阻尼模型的弹簧系数（±10%, ±20%），对GRF和支撑时间影响不大（图11），表明模型对该参数不敏感。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **方法论贡献**：首次成功将PCS模型与RL结合，为**柔性体与刚性体耦合系统的动态仿真与控制**提供了一个高效、可扩展的范例。其“预训练+微调”策略对解决高计算成本仿真问题具有普适意义。
    2.  **工具价值**：提供了一个**虚拟假肢测试平台**，能够量化分析假肢参数（如刚度）对跑步性能（如CoT、GRF、关节功率）的影响。这为假肢的**用户个性化设计**和**科学选型**提供了理论依据和仿真工具，有望替代部分物理试错过程。
    3.  **科学发现**：仿真结果揭示了假肢刚度与跑步能量消耗之间的**非线性关系**，存在一个最优刚度值，这与实验观测一致，验证了方法的有效性。

- **对未来研究/应用的影响**：
    - **假肢设计**：未来可利用该框架优化假肢的形状、材料、安装角度等更多参数，甚至设计出针对特定运动员的定制化假肢。
    - **康复训练**：可用于模拟不同假肢参数下的运动效果，为康复医师和运动员提供训练指导。
    - **控制领域**：该方法可推广到其他需要模拟柔性体与刚性体交互的领域，如外骨骼机器人、软体机器人控制等。

- **局限与开放问题**：
    1.  **肌肉模型缺失**：当前使用关节力矩驱动，而非肌肉模型。这导致能量消耗计算（CoT）存在正偏差（高估），因为忽略了肌肉的负功、共收缩和多关节能量传递等生理机制。未来工作应扩展到**肌肉骨骼模型**。
    2.  **单受试者数据**：所有仿真基于一名受试者的数据，限制了结果的普适性。未来需要在**多受试者**上进行验证。
    3.  **运动范围限制**：当前模型将骨盆约束在矢状面，限制了三维运动的分析。未来应扩展到**全三维运动**。
    4.  **策略泛化范围**：策略在刚度变化超过±20%时性能下降，未来需要研究如何训练更鲁棒的策略，或开发自适应方法。

## 5. 澄清与简化（Clarifications and Simplifications）

- **PCS模型（分段恒定应变模型）**：可以想象一根弯曲的钓鱼竿。为了描述它的形状，PCS模型不把它当作一个整体，而是切成很多小段。假设每一小段内部的弯曲程度（应变）是恒定的。这样，只要知道每一小段的弯曲程度，就能像搭积木一样，把整根鱼竿的形状拼出来。相比于用有限元法（FEM）计算每个点的应力应变，PCS模型计算量小得多；相比于用几根刚性杆连接，它又能更真实地模拟鱼竿的连续弯曲。
- **混合刚-柔耦合系统**：可以想象一个“半机器人”运动员。他的上半身（骨骼）是刚性的，由强化学习控制的“大脑”发出指令驱动。但他的“假肢腿”是柔性的，像一根弹簧片。这个系统就是要把刚性身体和柔性假肢的动力学方程耦合在一起，精确计算它们之间如何相互作用、传递力量。
- **两阶段训练策略**：这就像学开车。先在一个**简单的模拟器**（刚性假肢模型）上练习，熟悉基本操作（油门、刹车、转向）。等熟练了，再上**高仿真的模拟器**（混合系统模型）进行进阶训练，
