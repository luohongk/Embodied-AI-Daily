# ROAD-Waymo: A Large-Scale Action Awareness Dataset for Autonomous Driving

- arXiv: https://arxiv.org/abs/2411.01683v3
- arXiv ID: 2411.01683

---

🔍 **核心问题**：自动驾驶系统需要理解道路场景中其他交通参与者的行为（如动作、位置和事件），而现有数据集缺乏大规模、多城市的动作感知标注数据。

🛠 **方法**：在Waymo Open数据集基础上，通过新增标注层构建ROAD-Waymo数据集，并设计自动化标注流水线，用于检测和修正标注中的违规问题。

✨ **主要贡献**：提供了目前最大、最具挑战性的道路动作感知数据集（198k帧、54k轨迹、390万边界框、1240万标签），并兼容原有ROAD数据集，支持跨国家场景的领域适应基准测试（ROAD++）。
