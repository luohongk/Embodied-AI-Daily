# Multimodal Assessment of Pancreatic Cancer Resectability Using Deep Learning

- arXiv: https://arxiv.org/abs/2607.13826v1
- arXiv ID: 2607.13826

---

🔍 **核心问题**：解决胰腺导管腺癌（PDAC）可切除性评估中，专家对CT影像判断存在显著差异的问题，实现自动化、标准化的分类。

🛠 **方法**：提出全自动多模态深度学习框架，使用Swin-UNETR骨干网络对3D增强CT进行胰腺、肿瘤和血管的辅助分割，提取解剖感知特征，并与17项临床结构化信息融合，通过轻量分类头按NCCN标准分为三类。训练采用动态多任务目标，根据肿瘤分割Dice性能自适应平衡分割与分类任务。

✨ **主要贡献**：首次实现完全自动化的PDAC可切除性多模态评估，通过解剖感知特征与临床信息融合提升分类准确性，动态多任务学习策略有效促进特征表达既保留解剖信息又具备判别能力。
