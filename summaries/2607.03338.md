# Closed-loop vs. Open-loop Kalman Filter Architectures in Airborne Aided Inertial Navigation

- arXiv: https://arxiv.org/abs/2607.03338v1
- arXiv ID: 2607.03338

---

🔍 **核心问题**：在机载辅助惯性导航中，闭环（反馈）与开环（前馈）卡尔曼滤波器架构的性能差异，尤其是在安全关键航空应用中如何权衡系统安全性与导航精度。  
🛠 **方法**：采用标准地理坐标系下的惯性机械编排和直接位置辅助，通过仿真对比不同惯性传感器误差特性（从消费级到导航级）下两种架构的表现。  
✨ **主要贡献**：揭示了开环滤波器对高端IMU的平滑信息融合优势与闭环架构的长期稳定性之间的权衡，为航空导航系统架构选择提供了量化依据。
