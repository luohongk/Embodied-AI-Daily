# CORE: Common Outcome Regularities from Action-Free Visual Demonstrations for Robot Manipulation

- arXiv: https://arxiv.org/abs/2606.29517v1
- arXiv ID: 2606.29517

---

🔍 **核心问题**：机器人模仿学习依赖昂贵的机器人演示数据，而丰富的无动作视觉演示（如人类视频）因缺乏可执行动作且存在具身差异而难以利用。

🛠 **方法**：提出CORE框架，从视觉演示中提取“共同结果规律”——先训练终端状态编码器，聚合成功轨迹的终端嵌入为视觉目标原型，再将其作为全局目标条件注入机器人策略。

✨ **主要贡献**：在Meta-World、RoboTwin 2.0和真实世界操作中，CORE将策略骨干的平均成功率分别提升最多+3.9、+11.1和+17.0个百分点，且优于文本条件变体。
