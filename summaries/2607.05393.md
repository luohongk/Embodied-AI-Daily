# Interpretable Human-Label-Free Deep Learning for Real-Bogus Classification with Uncertainty Quantification

- arXiv: https://arxiv.org/abs/2607.05393v1
- arXiv ID: 2607.05393

---

🔍 **核心问题**：时域巡天中真实与虚假瞬变源分类依赖昂贵的人工标注或噪声较大的社区标签，且现有方法难以应对强标签污染和缺乏不确定性量化。  
🛠 **方法**：提出一种无需人工标签的弱监督框架，通过模拟瞬变源注入与污染的真实巡天数据训练双网络模型，采用非对称协同教学处理不同噪声水平的类别，并引入混合不确定性量化策略（结合MC Dropout与深度集成）。  
✨ **主要贡献**：在无人工标注下实现稳健的Real-Bogus分类，对强标签污染具有鲁棒性，提供校准的不确定性估计；通过潜空间分析揭示虚假源子类结构，且方法可迁移至未来巡天项目。
