# Do as the Romans Do: Learning Universal Behaviors from Heterogeneous Agents

- arXiv: https://arxiv.org/abs/2606.18537v1
- arXiv ID: 2606.18537

---

🔍 **核心问题**：如何从一群追求不同目标的异构演示者中，提取出普遍有用的行为，避免因行为信号冲突而难以判断哪些行为值得模仿。

🛠 **方法**：提出GRID（通用奖励推断与解耦）方法，将每个演示者的奖励函数分解为通用奖励（捕捉所有智能体共享的行为）和特定奖励（捕捉个体偏好与目标），并仅基于通用奖励进行训练。

✨ **主要贡献**：实现了通用预训练的新范式，生成能内化环境通用能力（如安全性和基本任务熟练度）的通用智能体，避免了标准模仿学习中的模式平均偏差，并在下游任务微调中表现更优、更稳定，实验验证了其在多个场景中的语义解耦和性能提升。
