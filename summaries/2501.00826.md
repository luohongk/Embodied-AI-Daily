# LLM-Powered Multi-Agent System for Automated Crypto Portfolio Management

- arXiv: https://arxiv.org/abs/2501.00826v3
- arXiv ID: 2501.00826

---

🔍 **核心问题**：加密货币投资组合管理需要融合多模态信号（价格、链上数据、新闻文本、技术指标），但现有深度学习模型缺乏可解释性，单一大型语言模型（LLM）难以处理多样化的模态输入。

🛠 **方法**：提出一个多智能体系统（MAS）框架，包含三个模态专用智能体——市场动态智能体、新闻情感智能体和交易智能体，通过层级、协作和辩论三种通信架构分解任务，并评估零样本、思维链、检索增强生成和技能增强四种配置。

✨ **主要贡献**：在2025年全年回测中，最佳配置（层级+技能增强）实现133.52%累计收益率和1.502夏普比率，优于单智能体、被动基准和深度学习基线；消融实验表明市场动态智能体最关键，跨模型对比验证了多智能体协调的模型无关优势，且所有决策可追溯至显式推理，兼具可解释性与有效性。
