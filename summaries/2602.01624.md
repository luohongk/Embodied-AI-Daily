# PISCES: Annotation-free Text-to-Video Post-Training via Optimal Transport-Aligned Rewards

- arXiv: https://arxiv.org/abs/2602.01624v2
- arXiv ID: 2602.01624

---

🔍 **核心问题**：文本生成视频（T2V）任务中，现有基于奖励的后训练方法要么依赖大规模人工偏好标注，要么使用预训练视觉语言模型中的不对齐嵌入，导致可扩展性差或监督信号不准确。

🛠 **方法**：提出无标注后训练算法PISCES，通过双最优传输（OT）对齐奖励模块，在分布级和离散令牌级对齐文本与视频嵌入，分别实现质量奖励（评估视觉质量与时序一致性）和语义奖励（确保文本-视频的时空语义对应）。

✨ **主要贡献**：首次将最优传输引入生成式后训练的无标注奖励监督，在短/长视频生成任务中，VBench质量与语义分数均超越基于标注和无标注方法，人类偏好研究进一步验证有效性，且该奖励模块兼容直接反向传播和强化学习微调。
