# A VLM-Enhanced Framework for Comprehensive Traffic Sign Condition Assessment Integrating Daytime Visual Performance and Nighttime Retroreflectivity Evaluation

- arXiv: https://arxiv.org/abs/2607.06478v1
- arXiv ID: 2607.06478

---

🔍 **核心问题**：传统交通标志评估依赖人工检查，主观性强、成本高且存在安全隐患，现有研究未能同时整合日间视觉性能与夜间反光性能的全面评估。

🛠 **方法**：提出一个融合日间与夜间评估的框架，使用三个微调的视觉语言模型（VLMs）评估日间视觉因素（可读性、颜色、表面完整性、环境条件），并通过情感分析和CLIP评分转化为数值；夜间反光性能则利用LiDAR数据按校准流程评估，最终整合为综合标志状况指数（SCI）。

✨ **主要贡献**：LLaVA和Qwen模型在双向余弦相似度上表现优异（0.67-0.76），在462个验证标志中识别出68个需立即更换，提供了一种低成本、全面的交通标志评估替代方案。
