# A Realistic Protocol for Evaluation of Weakly Supervised Object Localization

- arXiv: https://arxiv.org/abs/2404.10034v3
- arXiv ID: 2404.10034

---

🔍 **核心问题**：弱监督目标定位（WSOL）在训练时仅使用类别标签，但现有方法依赖验证集和测试集的人工标注边界框（bbox）进行模型选择和阈值估计，这与实际场景中缺乏bbox标注的情况不符。

🛠 **方法**：提出一种新的WSOL评估协议，利用预训练的区域提议方法（如Selective Search、CLIP、RPN）生成带噪声的伪边界框，用于模型选择和从定位图中估计阈值，从而避免人工bbox标注。

✨ **主要贡献**：在自然图像和医学图像数据集上的实验表明，使用伪边界框进行模型选择和阈值估计，其定位性能与使用真实边界框相当，且优于仅依赖类别标签和定位图动态阈值的方法。
