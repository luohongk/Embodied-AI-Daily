# Visually Grounded Self-Reflection for Vision-Language Models via Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.02490v1
- arXiv ID: 2607.02490

---

🔍 **核心问题**：大型视觉语言模型在链式思维推理中缺乏有效的视觉自反思能力，导致在分布偏移场景下无法基于视觉反馈纠正早期错误。

🛠 **方法**：提出VRRL强化学习框架，通过随机遮蔽轨迹前缀迫使模型从错误中间状态恢复，并引入经验回放缓冲区的滚动输入，使模型学习修正多样化的失败状态。

✨ **主要贡献**：在表格/图表视觉定位和空间导航任务中，该方法显著提升了分布偏移下的平均准确率，优于标准强化学习和面向反思的微调基线。
