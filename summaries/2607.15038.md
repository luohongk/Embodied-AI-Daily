# Video = World + Event Stream

- arXiv: https://arxiv.org/abs/2607.15038v1
- arXiv ID: 2607.15038

---

🔍 **核心问题**：如何将视频内容分解为稳定世界与动态事件流，并以此为基础构建通用的实时交互模型。

🛠 **方法**：提出Wan-Streamer v0.3，将视频建模为“世界+事件流”框架，通过大规模真实视频预训练，学习基于世界上下文预测事件流变化的能力，并针对实时全双工视听交互任务进行微调。

✨ **主要贡献**：首次将视频交互统一为“世界+事件流”视角，实现多模态输入到语言形式语音与行为动作的映射，在640×368分辨率、25 FPS、160毫秒流单元、约200毫秒模型响应延迟及约550毫秒总交互延迟下保持高效性能。
