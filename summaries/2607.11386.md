# From Sketch Prior to Trajectories: A Mission-Oriented Coordinated Navigation Framework for Indoor UAV Swarm

- arXiv: https://arxiv.org/abs/2607.11386v1
- arXiv ID: 2607.11386

---

🔍 **核心问题**：针对室内无人机集群在执行巡检、安防等任务时，需按预定顺序访问任务区域，而现有方法难以有效利用先验草图信息（如楼层平面图）实现协同导航的问题。

🛠 **方法**：提出一种任务导向的协同导航框架，通过机载观测与草图先验进行拓扑对齐，融合构建任务导向的可通行性表示；并设计分层2D-3D导航机制，由2D路径规划生成引导路径，再驱动3D轨迹优化生成动态可行、无碰撞的轨迹。

✨ **主要贡献**：在结构化多房间室内环境中验证了框架的有效性，展示了其在有/无通信条件下的协同导航能力，并通过多楼层仿真证明了系统对分层结构的可扩展性。
