# CloudDiffusion: Diffusion-Based Scene Completion in the Point Cloud Domain

- arXiv: https://arxiv.org/abs/2606.16048v2
- arXiv ID: 2606.16048

---

🔍 **核心问题**：从稀疏LiDAR点云中重建稠密3D场景（LiDAR场景补全）时，现有扩散模型存在全局表征不稳定、地面真值受里程计漂移污染、多步推理延迟过高的问题。

🛠 **方法**：提出CloudDiffusion，包含三个独立组件：多令牌高斯VAE（带交叉注意力池化）实现稳定场景级LiDAR压缩；基于锚点的ICP地面真值精炼管道消除训练噪声；单步x0扩散教师模型直接在坐标空间中完成场景补全。

✨ **主要贡献**：在SemanticKITTI序列08上，无需修改模型即可将平方Chamfer距离降低约16倍（0.396→0.024 m²）；推理延迟仅209ms/帧，比迭代扩散基线快65-138倍；证明数据质量在此任务中比模型设计更重要。
