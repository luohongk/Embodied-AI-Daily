# LARAD: Layout-Aware Road Anomaly Detection via Spatial-Logic Reasoning

- arXiv: https://arxiv.org/abs/2607.12858v1
- arXiv ID: 2607.12858

---

🔍 **核心问题**：现有道路异常检测方法过度依赖纹理新颖性识别分布外物体，忽略空间逻辑上下文，导致误报率高，且常需级联多个视觉模型，推理延迟大。

🛠 **方法**：提出LARAD方法，包含两个关键技术：一是空间逻辑违反合成（SLVS）管线，生成纹理一致但空间无效的训练样本；二是在标准闭集分割网络中加入轻量级OoD引导注意力分支。

✨ **主要贡献**：将异常检测范式从外观匹配转向空间逻辑推理，显著提升对逻辑异常的鲁棒性，在保持单模型高效性的同时达到新最优性能。
