# Simple-to-Complex Structured Demonstrations for Vision-Language-Action Learning

- arXiv: https://arxiv.org/abs/2607.04591v1
- arXiv ID: 2607.04591

---

🔍 **核心问题**：现有视觉-语言-动作（VLA）模型研究主要关注模型架构、训练策略和数据集规模，忽视了演示数据的组织和收集方式对模仿学习效率、训练稳定性和策略泛化能力的影响。

🛠 **方法**：提出一种简单到复杂的结构化演示收集策略，通过分解复杂任务为可逐步学习的子技能、标准化交互环境以减少不必要变化、以及按任务复杂度递增组织演示数据，使VLA模型先学习基础操作技能再逐步掌握复杂任务组合。

✨ **主要贡献**：在积木抓取与分类、毛巾折叠两项机器人操作任务中，该方法相比直接收集端到端完整任务轨迹的基线方法，显著提升了任务成功率和训练稳定性，揭示了演示组织是VLA学习中此前被忽视但至关重要的因素。
