# A Stereo Visual SLAM System Using Object-Level Motion Estimation and Geometric Filtering Based on Cross Disparity

- arXiv: https://arxiv.org/abs/2607.02005v1
- arXiv ID: 2607.02005

---

🔍 **核心问题**：解决动态场景中视觉SLAM系统因静态世界假设而无法处理运动物体，导致位姿估计和建图失败的问题。

🛠 **方法**：提出OCD SLAM框架，在ORB-SLAM2基础上引入基于视差与“交叉视差”差异的几何滤波方法，结合SMOKE 3D目标检测与卡尔曼滤波目标跟踪，实现特征级与物体级的运动分类。

✨ **主要贡献**：在KITTI数据集上显著提升轨迹精度，优于ORB-SLAM2及多种动态SLAM方法；消融实验验证了交叉视差模块能有效检测3D目标检测遗漏的动态特征。
