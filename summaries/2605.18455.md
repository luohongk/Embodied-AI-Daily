# OrganicHAR: Towards Activity Discovery in Organic Settings for Privacy Preserving Sensors Using Efficient Video Analysis

- arXiv: https://arxiv.org/abs/2605.18455v1
- arXiv ID: 2605.18455

---

🔍 **核心问题**：家庭环境中人体活动识别（HAR）难以部署，因为传感器信号在不同房屋、人和时间下差异大，需现场采集数据训练，且现有方法依赖摄像头定义活动，导致传感器无法可靠区分预设活动。

🛠 **方法**：提出OrganicHAR框架，以隐私保护传感器（如雷达、LiDAR、热阵列）的能力为中心，自动发现自然信号模式；仅在关键时刻调用视觉语言模型（VLM）理解场景，生成传感器可可靠区分的活动标签。

✨ **主要贡献**：在12名参与者实验中，仅用基本环境传感器实现粗粒度活动（4-5类）79%准确率，添加可穿戴传感器后细粒度活动（8-9类）达73%准确率；平均准确率77%，发现每位用户4-8类活动（共15类），并将VLM查询减少90%，实现隐私保护且实用的自然场景活动识别。
