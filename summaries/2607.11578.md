# DiffEEG: A Self-Supervised Denoising Diffusion Model for Learning EEG Generic Representations

- arXiv: https://arxiv.org/abs/2607.11578v1
- arXiv ID: 2607.11578

---

🔍 **核心问题**：解决基于EEG的癫痫检测中标注数据严重不足和类别极度不平衡（发作事件占比<10%）的问题。

🛠 **方法**：提出DiffEEG模型，包含970万参数，采用去噪扩散自监督预训练（在130万无标签EEG片段上训练1D U-Net+多头自注意力），以及基于强化学习（策略梯度优化）的微调，直接最大化F1分数以优先检测罕见发作事件。

✨ **主要贡献**：在279名患者的严格评估下，四类癫痫亚型分类达61%准确率和59% F1分数，二分类检测达81%准确率和85%加权F1，在极度不平衡（6.7%发作率）下仍保持临床可行的59%发作召回率；段级评估显示97.6%准确率上限，证明扩散预训练结合强化学习可显著降低对标注数据的需求。
