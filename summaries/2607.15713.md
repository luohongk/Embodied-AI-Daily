# Map as a Prompt: Learning Multi-Modal Spatial-Signal Foundation Models for Cross-scenario Wireless Localization

- arXiv: https://arxiv.org/abs/2607.15713v1
- arXiv ID: 2607.15713

---

🔍 **核心问题**：现有无线定位方法在跨场景环境下泛化能力差，需要大量标注数据且难以适应新环境。

🛠 **方法**：提出SigMap多模态基础模型，包含循环自适应掩码策略（利用信道周期性学习鲁棒表征）和“地图即提示”框架（通过轻量软提示融合3D地理信息实现跨场景适应）。

✨ **主要贡献**：在多个定位任务上达到最优性能，并在未见环境中展现出强大的零样本泛化能力，显著超越监督和自监督基线方法。
