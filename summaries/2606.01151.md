# Lagrangian Perturbation Diffusion Steering: Latent Reinforcement Learning for Generative Policies

- arXiv: https://arxiv.org/abs/2606.01151v1
- arXiv ID: 2606.01151

---

🔍 **核心问题**：高容量生成策略的行为克隆受限于演示覆盖不足和分布偏移，直接强化学习微调大动作解码器不稳定且样本效率低。

🛠 **方法**：提出拉格朗日扰动扩散引导（LP-DS），在冻结的生成策略解码前，学习紧凑的噪声空间扰动，并通过拉格朗日信任域目标优化，提升下游价值同时约束潜在先验偏差。

✨ **主要贡献**：在多个基准（如RoboMimic、OpenAI Gym、Adroit）上，LP-DS显著提升样本效率、成功率和回报（最高提升25%），保持更高动作空间熵，并验证了在流匹配骨干、大型视觉-语言-动作模型及物理机器人部署中的通用性。
