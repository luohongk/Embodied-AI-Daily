# 4DR360: State Reasoning for Joint 3D Detection and Occupancy Prediction in 4D Radar-Camera Full-Scene Perception

- arXiv: https://arxiv.org/abs/2607.09629v1
- arXiv ID: 2607.09629

---

🔍 **核心问题**：现有4D雷达-相机融合方法主要优化目标检测，而双任务系统（检测+占用预测）中两者交互有限，缺乏对全场景语义布局的联合建模。

🛠 **方法**：提出4DR360框架，将语义占用建模为持续传播的场景状态，通过状态引导的BEV增强（SBE）和Doppler引导的时间融合（DTF）实现跨模态粗到细的特征聚合。

✨ **主要贡献**：首次在4D雷达-相机系统中实现360°全场景联合检测与占用预测，并扩展ManTruckScenes数据集生成卫星地图标注的占用标签，建立统一的跨数据集多任务评估协议。
