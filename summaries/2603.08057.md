# See and Switch: Vision-Based Branching for Interactive Robot-Skill Programming

- arXiv: https://arxiv.org/abs/2603.08057v2
- arXiv ID: 2603.08057

---

🔍 **核心问题**：如何让非专家用户通过演示编程（PbD）使机器人能够根据视觉输入在线选择合适任务变体，并处理真实环境中的可变性。

🛠 **方法**：提出See & Switch框架，将任务表示为由决策状态连接的技能部分图，通过眼在手视觉系统（Switcher）在回放时选择后继技能部分，并检测需要新演示的异常情况；支持执行中通过动觉教学、摇杆和手势进行恢复演示。

✨ **主要贡献**：在8名新手用户的3项灵巧操作任务中收集约900次真实机器人执行数据，离线评估显示分支选择准确率最高达90.6%，79个决策状态中47个异常检测准确率>90%，验证了基于视觉输入的条件机器人技能编程的可靠性。
