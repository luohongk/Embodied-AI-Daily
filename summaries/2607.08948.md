# SplatCtrl: Perception-Action Coupling via Gaussian Scene Representations and Reactive Robot Control

- arXiv: https://arxiv.org/abs/2607.08948v1
- arXiv ID: 2607.08948

---

🔍 **核心问题**：解决机器人在未知和动态环境中实时避障与运动生成的问题，克服传统方法在非结构化场景中的局限性。

🛠 **方法**：基于3D高斯泼溅（3D-GS）提出混合体素滤波与动态高斯重定位策略，实现RGB-D流的实时场景重建；同时从各向同性高斯中推导连续有符号距离函数，并将其融入控制障碍函数，构建感知-动作耦合框架。

✨ **主要贡献**：首次将3D高斯场景表示与反应式控制结合，实现动态环境下的实时场景重建与平滑避障运动生成，并在仿真、实体机器人及人机协作场景中验证了有效性。
