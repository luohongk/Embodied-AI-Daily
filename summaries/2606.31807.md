# Reinforcement Learning-Based Control for an Inline Skating Humanoid Robot

- arXiv: https://arxiv.org/abs/2606.31807v1
- arXiv ID: 2606.31807

---

🔍 **核心问题**：解决被动轮式直排轮滑人形机器人的复杂欠驱动控制问题，实现高速、节能的动态滑行。  
🛠 **方法**：基于强化学习训练控制策略，采用球形和椭球形轮模型处理接触伪影，结合成功导向指令课程和滚动奖励机制，无需人类数据或运动先验。  
✨ **主要贡献**：策略零样本迁移至真实Booster T1机器人，实现动态平衡、抗扰动和高速转弯，运输成本较步行降低50%。
