# DSWorld: A Data Science World Model for Efficient Autonomous Agents

- arXiv: https://arxiv.org/abs/2607.15901v1
- arXiv ID: 2607.15901

---

🔍 **核心问题**：当前自主数据科学智能体依赖试错工作流，导致计算成本高昂，亟需一种能预测数据科学操作效果的世界模型来减少不必要的执行开销。

🛠 **方法**：提出DSWorld框架，结合结构化状态构建、成本感知路由、轻量级真实执行以及基于大语言模型的昂贵操作模拟器；并采用反思式世界模型优化（一种基于错误感知的强化学习策略）来提升状态转移预测能力。

✨ **主要贡献**：构建了8K规模的转移轨迹数据集；DSWorld将强化学习智能体训练加速约14倍，搜索式推理加速3-6倍，同时保持竞争性能；在转移预测任务上比最强的大语言模型基线高出35.6%。
