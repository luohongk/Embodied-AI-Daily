# Moving Beyond Diversity: Visual Token Pruning as Subspace Reconstruction for Efficient VLMs

- arXiv: https://arxiv.org/abs/2606.18681v1
- arXiv ID: 2606.18681

---

🔍 **核心问题**：视觉语言模型（VLM）因大量视觉令牌导致计算开销大，现有基于余弦相似度的多样性最大化剪枝方法忽略幅度信息，无法准确近似原始特征表示，尤其在组合多技能推理任务上表现不佳。

🛠 **方法**：提出SPARE方法，将令牌剪枝重构为子空间重建问题（列子集选择），通过迭代选择投影残差大的令牌进行重建驱动剪枝，并引入反相关性机制优先保留低图像-文本相关性但能更好保存上下文信息的令牌。

✨ **主要贡献**：在LLaVA上可移除94%视觉令牌且保持95%基线性能（无需训练），在组合推理任务上显著优于现有方法，揭示了反相关性现象并验证其有效性。
