# A Multi-Fidelity Convolutional Autoencoder-Transfer Learning Framework for Guided-Wave-Based Damage Diagnosis Using Large Simulated and Limited Experimental Datasets

- arXiv: https://arxiv.org/abs/2606.27304v1
- arXiv ID: 2606.27304

---

🔍 **核心问题**：解决导波结构健康监测中深度学习模型因实验标记数据稀缺和高保真仿真计算成本高而难以实际部署的问题。

🛠 **方法**：提出多保真迁移学习框架，结合轻量物理仿真生成大量合成数据、卷积自编码器（CAE）进行深度特征学习、前馈神经网络，并利用少量实验数据进行迁移学习。

✨ **主要贡献**：CAE迁移学习框架在损伤定位精度上显著优于CNN方案，损伤定位和尺寸预测的R²分别超过0.93和0.99，且对未见损伤场景具有高泛化能力，实现了准确、高效且实用的导波损伤诊断。
