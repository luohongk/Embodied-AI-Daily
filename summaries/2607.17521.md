# GeoWorldAD: Geometry World Action Model for Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.17521v1
- arXiv ID: 2607.17521

---

🔍 **核心问题**：现有自动驾驶视觉/视频动作模型缺乏显式几何空间约束和未来场景感知，难以在动态3D环境中平衡安全避障与高效行驶。

🛠 **方法**：提出GeoWorldAD模型，在自车对齐的3D空间中建立轨迹规划的几何世界，并通过潜在未来几何令牌预测短时场景演化，结合多尺度当前几何与未来几何信息进行迭代轨迹优化。

✨ **主要贡献**：首次将显式3D几何空间建模与未来几何世界预测引入动作学习，在NAVSIM v1和v2上取得最优性能，验证了几何引导对安全高效自动驾驶的有效性。
