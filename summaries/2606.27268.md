# E-TTS: A New Embodied Test-Time Scaling Framework for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.27268v1
- arXiv ID: 2606.27268

---

🔍 **核心问题**：现有具身任务中的测试时扩展方法存在两个不足：一是推理过程的扩展机制研究不足，二是缺乏对历史信息的有效利用，导致仅依赖当前观测进行动作扩展效果有限。

🛠 **方法**：提出E-TTS框架，通过历史感知的迭代优化与视觉语言验证器，统一实现推理与动作的联合测试时扩展；采用成对采样与评分机制，并引入历史缓冲区存储上下文信息，结合反馈生成形成闭环迭代优化。

✨ **主要贡献**：模块化即插即用设计，无需额外数据或重训练即可提升性能；在4个基准、6个环境、3种实体和4种基础模型上验证，模拟环境提升达33.14%，真实场景提升达26.62%。
