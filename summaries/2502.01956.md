# DHP: Discrete Hierarchical Planning for Hierarchical Reinforcement Learning Agents

- arXiv: https://arxiv.org/abs/2502.01956v3
- arXiv ID: 2502.01956

---

🔍 **核心问题**：分层强化学习（HRL）智能体在长时程视觉规划中，因依赖易出错的连续距离度量而表现不佳。

🛠 **方法**：提出离散分层规划（DHP），用离散的可达性检查替代连续距离估计，递归构建树状计划将长期目标分解为简单子任务，并采用新型优势估计策略奖励更短计划。

✨ **主要贡献**：在25房间导航任务中实现100%成功率（基线90%）；离线变体在OGBench基准上取得最优结果，在巨型HumanoidMaze任务中绝对提升达71%；方法架构无关，可泛化至动量控制任务，重规划仅需log N步。
