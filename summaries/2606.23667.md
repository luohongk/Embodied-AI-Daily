# The Table Says Otherwise: Testing LLMs with Counterfactual Relational Data

- arXiv: https://arxiv.org/abs/2606.23667v1
- arXiv ID: 2606.23667

---

🔍 **核心问题**：评估大语言模型在回答结构化数据问题时，是依赖表格中的真实数据，还是调用预训练阶段学到的先验知识，尤其是在表格内容与常识冲突时。  
🛠 **方法**：构建了ContraTable基准，包含原始数据库和对应的反事实数据库（保持结构不变但修改关键属性），并设计214个跨单表、多表及时间推理的问题进行测试。  
✨ **主要贡献**：发现强指令调优模型在直接查表时表现良好，但涉及连接、比较和时间推理时可靠性下降；原始与反事实准确率的差距表明模型会回退到先验知识，强调表格问答评估需同时关注准确性和对提供数据的忠实度。
