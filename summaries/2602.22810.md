# Multi-agent imitation learning with function approximation: Linear Markov games and beyond

- arXiv: https://arxiv.org/abs/2602.22810v2
- arXiv ID: 2602.22810

---

🔍 **核心问题**：在多智能体模仿学习中，现有理论分析依赖于状态-动作层面的“所有策略偏差集中系数”，该系数可能非常大，且缺乏高效交互式算法。

🛠 **方法**：利用线性马尔可夫博弈的结构（转移动态和奖励函数均为特征线性），将集中系数降至特征层面；并设计首个计算高效的交互式多智能体模仿学习算法，其样本复杂度仅与特征维度d相关。

✨ **主要贡献**：理论上证明了特征层面集中系数可远小于状态-动作层面系数，并提出了首个交互式算法（包括深度版本），在井字棋和四子棋等游戏中显著优于行为克隆（BC）。
