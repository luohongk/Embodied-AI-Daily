# Promoting Generalization for Exact Solvers via Adversarial Instance Augmentation

- arXiv: https://arxiv.org/abs/2310.14161v2
- arXiv ID: 2310.14161

---

🔍 **核心问题**：基于学习的混合整数线性规划（MILP）求解器在遇到未见过的实例（尤其是大规模扰动环境下的实例）时，因训练数据多样性不足而性能严重下降。

🛠 **方法**：提出对抗性实例增强方法（AdaSolver），将MILP实例表示为二分图，通过学习增强策略扰动图结构生成多样化实例，并将非可微的实例增强建模为上下文赌博机问题，与求解器进行对抗训练。

✨ **主要贡献**：首次提出通用框架AdaSolver，能同时提升基于模仿学习和强化学习的分支定界求解器的泛化能力，实验表明其在多种分布上显著提高求解效率。
