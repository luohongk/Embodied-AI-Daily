# CLASH: Collaborative Large-Small Hierarchical Framework for Continuous Vision-and-Language Navigation

- arXiv: https://arxiv.org/abs/2512.10360v2
- arXiv ID: 2512.10360

---

🔍 **核心问题**：解决连续视觉语言导航（VLN-CE）中，大型视觉语言模型推理能力强但导航性能不如专用全景小模型的问题。

🛠 **方法**：提出CLASH框架，包含反应式小模型规划器（RSMP，采用因果学习双分支架构）和反思式大模型推理器（RLMR，结合全景视觉提示与思维链推理），并通过不确定性感知协作机制（UCM）自适应融合两者决策；在仿真中用可学习点目标策略替代规则控制器，在真实部署中设计基于LiDAR的聚类模块与在线SLAM局部控制器。

✨ **主要贡献**：在VLN-CE排行榜上取得第一名（SoTA），在测试未见集上显著提升SR和SPL指标；真实实验验证了强鲁棒性，证明其在仿真与部署场景中的有效性。
