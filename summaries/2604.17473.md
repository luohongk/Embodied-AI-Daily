# Dual-Anchoring: Addressing State Drift in Vision-Language Navigation

- arXiv: https://arxiv.org/abs/2604.17473v4
- arXiv ID: 2604.17473

---

🔍 **核心问题**：解决视觉-语言导航（VLN）中长程任务的状态漂移问题，包括进度漂移（无法区分已完成与未完成子目标）和记忆漂移（历史表征退化导致丢失已访问地标信息）。

🛠 **方法**：提出双锚定框架，包含指令进度锚定（监督模型生成结构化文本标记区分子目标状态）和记忆地标锚定（利用地标中心世界模型预测分割模型提取的物体中心嵌入，强制验证历史观测并保留地标独特表征）。

✨ **主要贡献**：构建360万进度描述样本和93.7万地标验证数据，在仿真和真实环境中成功率提升15.2%，长程轨迹成功率提升24.7%。
