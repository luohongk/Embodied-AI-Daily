# Eval-PPO: Building an Efficient Threat Evaluator Using Proximal Policy Optimization

- arXiv: https://arxiv.org/abs/2503.12098v2
- arXiv ID: 2503.12098

---

🔍 **核心问题**：在游戏场景中，从多个敌方单位中固定数量目标进行威胁评估时，传统规则方法难以处理复杂特征关系，且监督学习缺乏明确标签。  
🛠 **方法**：将威胁评估问题转化为强化学习任务，基于近端策略优化（PPO）算法提出Eval-PPO，通过系统训练融合多维敌方特征与友方状态信息。  
✨ **主要贡献**：相比规则方法，Eval-PPO在平均成功率上提升17.84%，实现了更高效的威胁评估。
