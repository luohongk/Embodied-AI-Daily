# A Heterogeneous Dual-Network Framework for Emergency Delivery UAVs: Communication Assurance and Path Planning Coordination

- arXiv: https://arxiv.org/abs/2604.12501v2
- arXiv ID: 2604.12501

---

🔍 **核心问题**：在灾后复杂环境中，无人机执行紧急物资投递时，如何确保指挥控制（C2）链路的持续可靠，避免因链路失稳导致失控或救援延误。

🛠 **方法**：提出异构双网络框架（HDNF），通过悬停无人机基站形成应急通信支持网络（ECSN），与快速移动的投递无人机路径网络（DPN）紧密耦合；采用分层策略，包括多层C2服务模型、3D覆盖感知多智能体强化学习算法和3D通信感知A*路径规划器，联合优化任务分配、基站部署与路径规划。

✨ **主要贡献**：显著提升C2可靠性，消除关键阶段通信中断，保持高任务成功率，同时降低硬件部署成本；解决了高维搜索空间和轨迹-覆盖不匹配等难题。
