# MuViSeg: Multi-View Segment Correspondences from Dense Geometry Priors

- arXiv: https://arxiv.org/abs/2607.17938v1
- arXiv ID: 2607.17938

---

🔍 **核心问题**：传统图像匹配方法（稀疏关键点或密集像素）与下游任务（如物体级建图、拓扑导航）之间存在语义鸿沟，后者需要直接理解整个物体实例。

🛠 **方法**：提出MuViSeg框架，基于类无关分割器提取实例片段，利用大规模3D基础模型（如MASt3R、VGGT）的池化特征生成片段描述符，并设计三种学习型匹配头：LightGlue风格注意力头（含DoubleSoftmax评分）、DPT风格多尺度融合模块，以及核心的多视图联合自注意力机制，实现跨视图的传递性匹配。

✨ **主要贡献**：在零样本协议下，LightGlue风格头在Replica和Virtual KITTI 2上分别提升AUPRC达+4.85和+25.9；多视图变体在HM3D导航基准中无需重新训练，将成功率从50%提升至70%，SPL从45.7提升至59.1。
