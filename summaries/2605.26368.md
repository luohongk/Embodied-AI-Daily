## 0. 概览（Concise Summary）

- **主题（Topic）**：全景图像（Panoramic Image）的几何估计，包括深度、表面法线和天空分割。
- **问题（Problem）**：现有的单目几何估计基础模型（如Depth Anything 3）主要针对透视图像设计，无法直接处理全景图像。全景图像存在严重的几何畸变（如等距柱状投影的极点拉伸），且高质量的全景标注数据集稀缺，导致现有全景几何估计方法泛化能力差、精度有限。
- **方法（Method）**：提出 **PaGeR** (Panoramic Geometry Reconstruction) 框架。核心思想是将全景图表示为**立方体贴图（Cubemap）**，从而将全景几何估计问题转化为一个结构化的**多视角透视问题**。然后，利用预训练的透视几何基础模型（DA3）作为骨干网络，通过引入跨面有效填充（Cross-face Valid Padding）和混合透视/全景联合训练策略，将强大的3D先验知识无缝迁移到全景领域。
- **创新（Innovation）**：
    1.  **新颖的迁移策略**：利用立方体贴图表示，将全景问题转化为多视角问题，从而复用强大的透视基础模型，避免了为全景设计专用架构。
    2.  **统一的多任务框架**：PaGeR 在单次前向传播中同时预测尺度不变深度、度量深度、表面法线和天空掩码，实现了对360°场景的全面几何理解。
    3.  **混合训练策略**：通过混合合成全景数据和真实透视数据进行训练，既适应了360°场景，又保留了基础模型的真实世界先验，防止过拟合。
    4.  **新数据集**：引入了 **ZüriPano**，一个用于评估室外远距离场景的真实世界全景数据集。
- **意义（Significance）**：PaGeR 首次成功地将最先进的透视几何基础模型应用于全景领域，在室内外多个基准上取得了最先进的性能，特别是在零样本泛化方面表现卓越。这项工作为从单张全景图进行高质量、全面的3D重建开辟了新路径，对VR/AR、机器人、自动驾驶等领域具有重要价值。
- **一句话总结**：PaGeR 通过将全景图巧妙地转化为立方体贴图，并复用透视基础模型，实现了从单张全景图中高效、统一且高精度地估计深度、法线和天空掩码，在零样本泛化上取得了突破性进展。

## 1. 研究动机（Motivation）

- **解决的问题**：如何从单张全景RGB图像中，准确、全面地恢复出整个360°场景的3D几何结构（深度、表面法线）。
- **为什么重要**：全景图能提供完整的周围环境信息，对于需要全局空间理解的VR/AR、机器人导航、自动驾驶等应用至关重要。从单张图像恢复3D结构可以避免多视角拍摄和复杂的相机位姿估计，极大简化了流程。
- **挑战在哪里**：
    1.  **数据稀缺**：高质量、带有精确度量深度和法线标注的全景数据集非常难以获取，导致现有模型容易在小数据集上过拟合，泛化能力差。
    2.  **几何畸变**：全景图常用的等距柱状投影（ERP）会在极点附近产生严重的非线性畸变，这与为透视图像设计的、依赖平移不变性的卷积神经网络（CNN）或Transformer架构不兼容。
    3.  **先验迁移困难**：现有的强大3D基础模型（如DA3）都是在海量透视图像上训练的，其学到的3D先验知识难以直接迁移到畸变的全景域。

## 2. 创新点（Innovation）

- **关键创新**：提出了一种将透视基础模型迁移到全景域的**通用策略**，并基于此构建了统一的**多任务全景几何估计框架PaGeR**。
- **与现有方案的不同**：
    - 不同于为全景设计专用架构（如PanoFormer），PaGeR 通过**立方体贴图**表示，将问题“降维”为模型熟悉的透视多视角问题，从而直接复用强大的透视基础模型。
    - 不同于直接在全景ERP图上微调模型，PaGeR 通过**跨面有效填充**和**混合训练**，在保留透视先验的同时，学习生成几何一致的全景输出。
    - 不同于仅预测单一模态（如深度），PaGeR 是一个**统一的多任务框架**，能同时输出深度、法线和天空掩码，实现了更全面的场景理解。
- **为什么能奏效**：
    - **立方体贴图**：将连续的球面分解为6个标准的90°视场角透视图像，完美匹配了透视模型的输入格式和归纳偏置。
    - **跨面有效填充**：解决了立方体贴图在面边界处的几何和光度不连续问题，保证了全局一致性。
    - **混合训练**：合成全景数据提供了完整的360°监督信号，而真实透视数据则作为“锚点”，防止模型在合成数据上过拟合，并保留了基础模型从真实世界中习得的丰富先验。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

PaGeR 的整体架构如图2所示，主要包含以下部分：

1.  **全景输入表示**：将输入的全景ERP图转换为6个面的**立方体贴图**。每个面都是一个90°视场角的透视图像。
2.  **共享的几何Transformer骨干**：采用预训练的 **Depth Anything 3 (DA3)** 作为骨干。DA3由一个ViT编码器和一个密集预测Transformer解码器组成。编码器通过自注意力和交叉注意力机制处理6个透视面，并利用固定的相机参数（内参K和外参E）来引导跨视角的空间推理。
3.  **全景适配层**：
    - **隐式编码器同步**：微调ViT编码器，使其通过交叉注意力层自然地学习在立方体贴图相邻面之间传递和同步特征。
    - **球面感知解码器填充**：在解码器的卷积和上采样操作中，用**跨面有效填充**替代标准的零填充。该层会动态地从几何相邻的立方体贴图面提取特征，确保边界处的无缝过渡。
4.  **多任务几何解码**：在共享骨干之上，添加多个专用的预测头，实现单次前向传播的多任务输出。
    - **尺度不变深度头**：预测对数空间的尺度不变深度。
    - **度量尺度头**：一个并行的、低分辨率的解码器，预测粗略的度量对数深度，然后通过中值对齐的方式，为尺度不变深度提供一个全局尺度因子，从而恢复绝对度量深度。
    - **表面法线头**：一个独立的解码分支，输出三维单位向量。
    - **天空分割头**：一个轻量级分支，从中间特征和语义token中读取信息，输出二值天空概率图，用于掩码深度和法线中的无限远区域。

### 3.2 关键算法与数学推导

- **等距柱状投影 (ERP)**:
  $$u = \frac{\theta}{2\pi} + 0.5, \quad v = \frac{\phi}{\pi} + 0.5$$
  其中 $\theta \in [-\pi, \pi]$ 是经度，$\phi \in [-\pi/2, \pi/2]$ 是纬度。

- **尺度不变深度损失**:
  $$L_{\text{depth}} = \lambda_{L1} L_1 + \lambda_{\text{grad}} L_{\text{grad}} + \lambda_{\text{norm}} L_{\text{norm}}$$
  其中：
  - $L_1 = \frac{1}{N} \sum_p \left( c_p |\hat{z}_{\text{aligned},p} - z^*_p| - \lambda_c \log c_p \right)$：置信度加权的L1损失。$\hat{z}_{\text{aligned}} = \hat{z}_{SI} + \beta^*$ 是最优对齐后的预测对数深度，$\beta^* = \arg\min_\beta \|(\hat{z}_{SI} + \beta) - z^*\|_2^2$。
  - $L_{\text{grad}} = \frac{1}{N} \sum_p \sum_{i \in \{x,y\}} |\partial_i \hat{z}_{\text{aligned},p} - \partial_i z^*_p|$：边缘感知的梯度损失，用于保持边界锐利度。
  - $L_{\text{norm}} = \frac{1}{N} \sum_p \left( 1 - \hat{\mathbf{n}}_p \cdot \mathbf{n}^*_p \right)$：法线一致性损失，通过预测深度的梯度计算法线 $\hat{\mathbf{n}}$，并与真实法线 $\mathbf{n}^*$ 比较余弦相似度。

- **度量尺度恢复**:
  从粗略的度量对数深度图 $\hat{z}_m$ 和尺度不变深度图 $\hat{z}_{SI}$ 中推断全局尺度因子 $\hat{\beta}$：
  $$\hat{\beta} = \text{median}_a \left[ \hat{z}_m(a) - \text{pool}[\hat{z}_{SI}](a) \right]$$
  其中 $a$ 是低分辨率网格上的空间锚点。最终的绝对度量深度为 $\hat{Z}_m = \exp(\hat{\beta}) \hat{Z}_{SI}$。

- **表面法线损失**:
  $$L_{\text{normal}} = \lambda_{\text{cos}} L_{\text{cos}} + \lambda_{\text{perc}} L_{\text{perc}}$$
  其中 $L_{\text{cos}}$ 是像素级的余弦相似度损失，$L_{\text{perc}}$ 是基于VGG网络的感知损失，用于防止过度平滑。

### 3.3 模型、训练与数据集

- **模型**：基于预训练的 **Depth Anything 3 (DA3)**，其骨干是 **DINOv2**。
- **训练策略**：两阶段训练。
    1.  **第一阶段**：联合训练尺度不变深度头和表面法线头，同时微调骨干网络。
    2.  **第二阶段**：冻结第一阶段的所有组件，独立训练度量尺度头和天空分割头。
- **优化器**：AdamW，学习率 $3 \times 10^{-4}$，指数移动平均衰减为0.999。
- **训练数据**：
    - **合成全景数据**：80k张来自 **Structured3D**（室内）和作者自建的 **PanoInfinigen**（室内外）数据集。
    - **真实透视数据**：10k张来自 **ScanNet++** 和 **ARKitScenes** 数据集。这些图像被裁剪为90°视场角，并随机分配一个立方体贴图面的外参，用于混合训练。
- **推理**：处理一张2K分辨率的全景图仅需0.5秒，显存消耗12.8GB。

### 3.4 实验设置与结果

- **评估数据集**：
    - **室内**：Matterport3D360, Stanford2D3DS (用于深度)，Structured3D (用于法线)。
    - **室外**：作者新提出的 **ZüriPano** (用于深度)。
- **评估指标**：
    - **深度**：绝对相对误差 (AbsRel)，均方根误差 (RMSE)，阈值精度 ($\delta_1$)。
    - **法线**：平均角度误差 (Mean)，均方误差 (MSE)，阈值精度 ($\delta_{5^\circ}$, $\delta_{22.5^\circ}$)。
- **主要结果**：
    - **表1 (尺度不变深度)**：PaGeR 在所有三个数据集上均取得了最优结果。在室外 ZüriPano 数据集上，AbsRel 从 18.27 (RPG360) 降至 **9.36**，几乎减半。
    - **表2 (度量深度)**：PaGeR 同样表现最佳，尤其是在室外 ZüriPano 上，RMSE 从 716.38 (DepthAnyCamera) 降至 **530.85**。
    - **表3 (表面法线)**：在 Structured3D 上，PaGeR 的 Mean Angular Error 达到 **5.49°**，MSE 为 **174.9**，显著优于之前的最优方法 PanoNormal (MSE 246.6)。
- **消融实验 (表4, 5)**：
    - **表4a**：与原始DA3相比，PaGeR 在跨面一致性（SDD, SP, SS）上提升巨大，同时保持了深度精度。
    - **表5a**：验证了相机条件、对数空间深度、天空分割头、跨面有效填充和混合训练等组件的重要性。
    - **表5b**：验证了度量尺度头使用密集监督（F=4）优于全局标量或过度的下采样。

### 3.5 技术细节

- **立方体贴图分辨率**：每个面 504x504 像素，组合成约2K的全景图。
- **跨面有效填充**：在解码器的卷积和插值操作中，当计算边界像素时，不是用0填充，而是从立方体贴图几何相邻的面上提取对应的像素值。这确保了特征在面边界处的连续流动。
- **混合训练实现**：在训练批次中，一部分是完整的6面立方体贴图（来自合成数据），另一部分是单张透视图像（来自真实数据）。对于透视样本，跨面填充被禁用，退化为标准零填充。这种交替训练让模型既能学习360°结构，又能保持透视先验。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    - 成功弥合了透视基础模型与全景几何估计之间的鸿沟，提出了一种通用且高效的迁移学习范式。
    - 构建了一个统一的多任务全景几何估计框架，能够从单张全景图中同时恢复深度、法线和天空掩码，性能达到最先进水平。
    - 提出了 ZüriPano 数据集，为室外全景几何估计的零样本评估提供了新的基准。
- **对未来研究/应用的影响**：
    - 为利用其他强大的透视基础模型（如VGGT, DUSt3R）解决全景问题提供了蓝图。
    - 推动了从单张全景图进行高质量、全方位3D重建的实用化进程，对VR/AR内容创建、机器人环境感知、自动驾驶的环视系统等有直接推动作用。
- **局限与开放问题**：
    - 对镜面、反射和透明表面的预测不可靠。
    - 在结构极其复杂的场景中，立方体贴图的面边界处仍可能出现微小的几何或光度错位。
    - 未来的工作可以探索引入显式的数学连续性约束来彻底解决边界问题。

## 5. 澄清与简化（Clarifications and Simplifications）

- **为什么用立方体贴图而不是直接处理ERP图？**
    - **类比**：想象一下，你要用一张世界地图（ERP）来理解地球的3D形状。地图在南北极附近被严重拉伸，导致格陵兰岛看起来和非洲一样大。一个训练有素的“透视专家”（AI模型）看到这张扭曲的地图会感到困惑。立方体贴图就像把地球仪的表面展开成6张独立的、没有畸变的方形地图。每张地图都是标准的90°视角，这个“透视专家”一看就懂。PaGeR 就是让这个“专家”同时看这6张地图，并教会它如何把这6张图无缝地拼回一个完整的球体。

- **“跨面有效填充”在做什么？**
    - **类比**：想象你在拼一个立方体的纸模型。当你把6个面粘在一起时，在棱（边界）处，一个面上的图案需要和相邻面上的图案完美对齐。标准零填充就像在棱上留了空白，导致图案断裂。而“跨面有效填充”就像在粘合时，把相邻面上的图案“借”过来，填充到棱上，从而保证了图案的连续性。在PaGeR中，这个操作是在神经网络内部完成的，确保了深度和法线在立方体贴图边界处是平滑过渡的。

- **“混合训练”为什么有效？**
    - **类比**：你想教一个学生（模型）理解360°全景世界。你给了他很多高质量的3D合成全景图（课本知识），但他可能只学会了课本上的例子，遇到真实世界就懵了（过拟合）。同时，你也给了他很多真实世界的普通照片（真实案例）。通过混合训练，他既从课本上学到了360°的完整结构，又从真实案例中学到了真实世界的纹理、光照和统计规律。这样，他就能更好地将课本知识应用到真实世界中，实现“零样本”泛化。

## 6. 补充说明（Additional Notes）

- **相关工作关联**：
    - **透视几何基础模型**：本文建立在 **Depth Anything 3 (DA3)** 之上，DA3本身是 **Depth Anything** 系列的延续，代表了单目深度估计的最前沿。其他如 **VGGT**、**DUSt3R** 等也是同类模型。
    - **全景几何估计**：早期工作如 **Bifuse++**、**PanoFormer** 设计了专用架构。近期工作如 **Depth Any Panoramas (DAP)** 和 **DA2** 尝试微调透视模型，但受限于ERP畸变或训练成本。**DreamCube** 和 **RPG360** 也使用了立方体贴图，但PaGeR通过混合训练和跨面填充实现了更好的效果。
    - **多任务学习**：与 **MTL** 等工作类似，PaGeR也探索了联合学习深度和法线，但PaGeR的优势在于其强大的基础模型和创新的全景适配策略。
