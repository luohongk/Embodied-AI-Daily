# LiteResearcher: A Scalable Agentic RL Training Framework for Deep Research Agent

- arXiv: https://arxiv.org/abs/2604.17931v3
- arXiv ID: 2604.17931

---

🔍 **核心问题**：现有强化学习训练框架因依赖手工合成数据或真实搜索环境，导致成本高、不稳定，难以规模化应用于深度研究智能体。

🛠 **方法**：提出LiteResearcher框架，通过构建一个模拟真实搜索动态的轻量虚拟世界，实现持续改进的训练流程，使小型搜索智能体在强化学习中高效学习。

✨ **主要贡献**：仅4B参数的LiteResearcher在GAIA和Xbench基准上分别达到71.3%和78.0%的开放源码最优结果，性能超越通义DeepResearch和Claude-4.5 Sonnet等大规模模型，证明可扩展的强化学习训练是深度研究智能体的关键推动力。
