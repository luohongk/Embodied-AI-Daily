# Neural Subspace Reallocation: Continual Learning as Retrieval-Based Subspace Memory Management

- arXiv: https://arxiv.org/abs/2606.30067v1
- arXiv ID: 2606.30067

---

🔍 **核心问题**：解决持续学习中灾难性遗忘和循环任务恢复效率低下的问题，特别是在固定容量下如何高效管理参数子空间。

🛠 **方法**：提出神经子空间重分配（NSR），将低秩适配（LoRA）模块视为可压缩、可检索的记忆单元，通过SVD压缩、任务知识库存储、嵌入相似性检索和子空间重分配循环，并辅以知识蒸馏保护旧任务。

✨ **主要贡献**：理论证明循环环境中无记忆分配策略存在累积遗憾；实验上在Split-CIFAR-100上循环恢复时间减少10倍，在5-Datasets基准上达到最高准确率和最少遗忘，并揭示记忆机制（压缩与相似性检索）而非学习型分配策略是性能关键，且压缩知识库仅需每任务0.29 MB参数。
