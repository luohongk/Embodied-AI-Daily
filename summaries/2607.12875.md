# MetaInfer: A Knowledge Only LLM Inference Engine Generator SKILL Toolbox

- arXiv: https://arxiv.org/abs/2607.12875v1
- arXiv ID: 2607.12875

---

🔍 **核心问题**：通用大语言模型推理框架因支持多种模型、硬件和优化策略而代码复杂、维护成本高，且传统抽象层可能引入额外性能开销。

🛠 **方法**：提出“LLM-as-Compiler”方法MetaInfer，用户仅需指定推理程序的运行时约束，由LLM驱动的多智能体协作系统结合契约知识库自动生成紧凑的定制化推理框架。

✨ **主要贡献**：通过将生成约束、验证反馈和知识整合形成闭环，MetaInfer能在零参考约束下为已知目标生成可运行的定制推理引擎，并支持知识库对新模型和平台场景的演化，实验验证了其有效性和性能。
