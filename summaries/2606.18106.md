# Deep Reinforcement Learning for Minimum Zero-Forcing Sets

- arXiv: https://arxiv.org/abs/2606.18106v1
- arXiv ID: 2606.18106

---

🔍 **核心问题**：研究如何在无向图中找到最小零强制集（ZFS），这是一个NP难的图着色问题，在网络科学、控制及逻辑电路设计中有重要应用。  
🛠 **方法**：提出基于强化学习的框架SD-ZFS，通过改进S2V-DQN架构来适配零强制集问题，并在不同结构的图数据集上训练和评估模型。  
✨ **主要贡献**：展示了该框架在求解最小零强制集时优于贪心启发式算法，接近最优解，并分析了模型在不同网络类型上的泛化、扩展和迁移能力，揭示了网络结构对问题求解的影响。
