# Learning Stable In-Grasp Manipulation in a Non-Dropping Action Space

- arXiv: https://arxiv.org/abs/2606.28196v1
- arXiv ID: 2606.28196

---

🔍 **核心问题**：传统灵巧操作控制器依赖强假设的解析模型，而强化学习方法因学习目标冲突和隐性不稳定性导致效率低下，难以高效探索稳定且精确的抓取内操作技能。

🛠 **方法**：将灵巧操作技能分解为多个更简单、可分析的组件，每个组件在经典物理学和控制理论的约束与指导下进行学习，从而在非掉落动作空间中实现稳定抓取内的重定位/重定向。

✨ **主要贡献**：通过融合理论先验知识，显著提升了技能学习的效率和稳定性，在多种物体、传感器/电机噪声、延迟及摩擦条件下均能实现稳定抓取内的操作。
