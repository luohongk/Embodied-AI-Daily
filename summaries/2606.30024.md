# IBRSteG: Learning a Generalizable Steganography Framework for 3D Gaussian Splatting

- arXiv: https://arxiv.org/abs/2606.30024v1
- arXiv ID: 2606.30024

---

🔍 **核心问题**：现有3D高斯泼溅（3DGS）隐写方法无法泛化到不同场景，参数生成与特定场景强耦合，导致无法直接嵌入和隐藏任意3D场景内容。

🛠 **方法**：提出IBRSteG框架，通过GAS（高斯属性隐写器）网络学习场景无关的嵌入函数，将秘密3D高斯点的属性注入到封面场景中，直接重建隐写场景，无需逐场景微调或优化。

✨ **主要贡献**：首次实现可泛化的3DGS隐写，支持高质量、高容量的秘密场景隐藏，且安全性强；在多个数据集上验证了优越性能，代码已开源。
