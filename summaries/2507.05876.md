# Olaf: Programmable Data Plane Acceleration for Asynchronous Distributed Reinforcement Learning

- arXiv: https://arxiv.org/abs/2507.05876v2
- arXiv ID: 2507.05876

---

🔍 **核心问题**：异步分布式强化学习（DRL）在大规模训练中因网络拥塞和丢包导致模型更新过时，收敛性能下降。  
🛠 **方法**：提出Olaf——一种可编程数据平面加速引擎，在数据平面队列中动态聚合异步生成的模型更新，并设计新型队列结构（适用于未来机架顶交换机），同时结合主机端传输控制与模型过时度量（Age-of-Model, AoM）及形式化验证器来优化系统。  
✨ **主要贡献**：显著降低模型过时度和队列拥塞，提升异步DRL工作负载的整体收敛速度，并通过AoM度量实现多DRL集群场景下的系统级目标推理。
