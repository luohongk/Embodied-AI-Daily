# The Ebb and Flow of Multimodal Focus: Scheduling Visual Relay Windows for Grounded VLM Reasoning

- arXiv: https://arxiv.org/abs/2607.11436v1
- arXiv ID: 2607.11436

---

🔍 **核心问题**：视觉-语言模型在多模态推理中，视觉证据进入语言堆栈后变得不稳定，削弱了基于证据的推理能力。

🛠 **方法**：提出TRACE框架，通过轻量训练模块在推理时自适应控制视觉中继窗口（VRW）的分配，在预填充阶段重塑中继分配，并在解码阶段保持视觉支持。

✨ **主要贡献**：发现VLM内部存在稳定的三阶段注意力重分布模式，其中视觉中继窗口是关键阶段；TRACE在7个基准测试上平均提升4.33分，最高达6.6分，同时改善推理密集型任务。
