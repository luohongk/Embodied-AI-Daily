# Phase-Preserving Trimodal Transformer for Tropical Forest Biomass Estimation Using Optical and PolInSAR Data

- arXiv: https://arxiv.org/abs/2607.03663v3
- arXiv ID: 2607.03663

---

🔍 **核心问题**：解决热带成熟森林地上生物量（AGB）估算中SAR信号在高密度区域饱和及光学影像受云层遮挡的物理限制问题。

🛠 **方法**：提出三模态相干共注意力Transformer（TCCT），融合Landsat-5光学数据与P/L波段复值PolInSAR数据，采用复值编码器保留相位相干性，并通过动态共注意力机制自适应降低云污染光学像素权重，同时利用Levenberg-Marquardt优化进行局部空间异速校准。

✨ **主要贡献**：在亚马逊Paracou区域，TCCT实现冠层高度模型RMSE为3.78米（R²=0.33），AGB相对RMSE为4.51%（>50 Mg/ha区域），满足ESA BIOMASS任务误差<20%的要求；消融实验证实相位相干性可缓解深冠层信号饱和问题。
