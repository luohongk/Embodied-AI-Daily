# DPNeXt: A Lightweight Multi-Scale Feature Fusion Framework for Efficient ViT-Based Multi-Task Dense Prediction

- arXiv: https://arxiv.org/abs/2607.16012v1
- arXiv ID: 2607.16012

---

🔍 **核心问题**：针对机器人感知系统中多任务学习（如语义分割与深度估计）的密集预测任务，现有视觉基础模型（VFM）的解码策略存在计算瓶颈，且任务间负迁移影响性能。

🛠 **方法**：提出DPNeXt轻量级多尺度特征融合解码器，采用双深度可分离倒置瓶颈结构优化冻结VFM的特征利用，并引入多任务边界引导（MTBG）策略，通过对称边界监督增强几何一致性，无需额外标注或推理成本。

✨ **主要贡献**：在Cityscapes和NYUv2数据集上达到最优多任务性能，相比标准DPT，DPNeXt-S参数量减少78.6%，在资源受限设备上推理速度最快，且MTBG有效缓解任务负迁移。
