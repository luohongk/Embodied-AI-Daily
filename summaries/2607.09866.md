# Robo-ValueRL: Reliable Value Estimation for Offline-to-Online Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.09866v1
- arXiv ID: 2607.09866

---

🔍 **核心问题**：在离线到在线强化学习中，价值函数估计的可靠性如何影响策略优化，这一问题尚未被充分探索。

🛠 **方法**：提出Robo-ValueRL框架，通过历史条件价值估计器结合全局进度与局部偏好指标评估可靠性，并将可靠价值估计用于质量条件一致性策略预训练和在线残差适应模块。

✨ **主要贡献**：实验证明价值可靠性显著影响下游性能，可靠价值函数能更有效指导离线预训练与在线改进，在精密芯片插入任务达86%成功率，在通用化模块拆卸任务达84%成功率。
