# M$^\text{4}$World: A Multi-view Multimodal Driving World Model for Interactive Object Manipulation and Minute-long Streaming

- arXiv: https://arxiv.org/abs/2607.14005v1
- arXiv ID: 2607.14005

---

🔍 **核心问题**：现有驾驶世界模型在物体级别的可控性和长时间生成的稳定性上存在不足，难以支持交互式操作和分钟级连续仿真。

🛠 **方法**：提出M⁴World多视图多模态驾驶世界模型，通过灵活的条件控制接口实现物体空间布局与外观的精细操控，并采用多阶段训练框架，仅需四次去噪步骤即可实现稳定的在线因果生成，支持分钟级流式输出。

✨ **主要贡献**：首次实现交互式物体操控与分钟级稳定流式生成的结合；引入高效少样本后训练及视觉参考条件生成模型，提升长尾场景定制能力；提出基于VLM的自动化评估流程，从场景、视图和跨视图一致性三个维度衡量可控性。
