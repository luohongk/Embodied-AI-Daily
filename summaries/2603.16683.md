# Learning Whole-Body Control for a Salamander Robot

- arXiv: https://arxiv.org/abs/2603.16683v1
- arXiv ID: 2603.16683

---

🔍 **核心问题**：如何为高度关节化的蝾螈机器人实现统一的全身控制，使其在陆地和两栖环境中可靠地协调运动，并解决从仿真到真实硬件的迁移问题。  
🛠 **方法**：采用强化学习将本体感知观测和指令速度映射到关节级动作，并通过系统级的真实-仿真匹配与仿真-真实迁移策略，确保策略在硬件上稳定部署。  
✨ **主要贡献**：首次在真实蝾螈机器人上实现了基于学习的全身控制，支持平坦与不平坦地面的稳定行走，并在仿真中展示了从步行到游泳的无缝过渡，为理解跨物理模式运动提供了新视角。
