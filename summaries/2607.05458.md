# Learning to Control LLM Agent Harnesses with Offline Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.05458v1
- arXiv ID: 2607.05458

---

🔍 **核心问题**：现有LLM代理通常通过修改提示、模型或手工流程来改进，而执行框架（harness）被视为固定基础设施，本文旨在证明该框架本身是可学习的控制层。

🛠 **方法**：将框架操作形式化为有限时域的Harness MDP，使用轻量级控制器选择结构化执行动作，同时冻结LLM执行器；控制器通过离线强化学习中的优势加权回归训练，仅依赖终端任务评分奖励。

✨ **主要贡献**：提出Harness Maturity Score分离任务质量与执行模式可靠性，实验表明学习控制器能稳定提升验证行为，并在多个领域（如tau-bench零售、AgentBench DB-Bench）显著改善最终任务质量，消融实验证明改进并非源于模仿或简单增加检查。
