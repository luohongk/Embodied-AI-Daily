# Edit3DGS: Unified Framework for Dynamic Head Editing via 2D Instruction-Guided Diffusion and 3D Gaussian Splatting

- arXiv: https://arxiv.org/abs/2606.17432v1
- arXiv ID: 2606.17432

---

🔍 **核心问题**：现有方法难以在动态3D头部编辑中同时实现语义可控性、时间一致性和高保真度，尤其是处理视频中面部表情、属性等细粒度修改时容易出现伪影或不连贯。

🛠 **方法**：提出Edit3DGS框架，结合2D文本引导扩散模型进行面部区域编辑，并通过3D高斯泼溅（3D Gaussian Splatting）将编辑后的帧聚合为连贯的3D动态头像，同时采用多视角批量编辑和轻量级修复策略确保时间一致性。

✨ **主要贡献**：首次实现基于2D指令的扩散模型与3D高斯泼溅的统一动态头部编辑框架，支持表情变换、属性修改等细粒度操作，生成无伪影、时间平滑的高保真3D头像，在虚拟化身、沉浸式通信等领域具有实用价值。
