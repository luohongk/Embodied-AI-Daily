# CommonRoad-Game: A Human-in-the-Loop Simulation Framework for Autonomous Driving

- arXiv: https://arxiv.org/abs/2607.01382v1
- arXiv ID: 2607.01382

---

🔍 **核心问题**：现有自动驾驶仿真平台缺乏实时人机交互接口，且计算开销大，难以支持早期研究中快速原型开发与灵活实验。

🛠 **方法**：提出CommonRoad-Game框架，采用多线程架构与稳健同步机制，将仿真时间与真实时间对齐，并集成场景生成模块记录驾驶日志，支持可复现的交互测试。

✨ **主要贡献**：实现了稳定时间同步与可扩展多智能体仿真，无缝集成CommonRoad运动规划器，提供轻量级人机交互环境，代码已开源。
