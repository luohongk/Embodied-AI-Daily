# Generalized Kalman filter based temporal difference reinforcement learning

- arXiv: https://arxiv.org/abs/2607.20010v1
- arXiv ID: 2607.20010

---

🔍 **核心问题**：传统基于卡尔曼滤波的时序差分强化学习受限于线性高斯假设，无法处理非线性模型和非高斯概率分布下的价值函数估计与不确定性量化问题。

🛠 **方法**：基于条件期望理论，将价值函数和Q值函数视为不确定量，通过随机推断框架进行递归估计，并利用多项式混沌展开或集成近似对随机问题进行离散化，以实现高效计算。

✨ **主要贡献**：提出了一种广义时序差分学习框架，能同时估计价值函数的条件期望及其二阶概率矩，从而量化学习过程中的不确定性，并将经典方法扩展至非线性、非高斯系统，在线性质量-弹簧-阻尼系统和非线性热传导问题中验证了其准确性与适用性。
