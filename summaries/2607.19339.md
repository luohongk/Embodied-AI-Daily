# OmniReasoner: Thinking with Long Audio-Video via Native Tool Use

- arXiv: https://arxiv.org/abs/2607.19339v1
- arXiv ID: 2607.19339

---

🔍 **核心问题**：长音频-视频推理中关键证据稀疏、跨模态且难以用统一高保真输入保存，导致全模态大语言模型处理困难。

🛠 **方法**：提出OmniReasoner框架，通过监督微调和强化学习训练模型自主决定是否及何时调用“缩放工具”（zoom-in tool），先构建低成本的全局预览，再在必要时对特定时间区间进行高保真视听分析；引入TimeAnchor机制确保工具调用的时间参数在不同采样粒度下有效，并设计时间增强数据引擎自动合成训练轨迹。

✨ **主要贡献**：在无需人工标注时间区间的情况下，显著提升长音频-视频推理的答案准确性和时间定位能力，同时将高保真计算集中在信息丰富的区域。
