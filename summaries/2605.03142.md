# MARS-DA: A Hierarchical Reinforcement Learning Framework for Risk-Aware Multi-Agent Bidding in Power Grids

- arXiv: https://arxiv.org/abs/2605.03142v1
- arXiv ID: 2605.03142

---

🔍 **核心问题**：解决可再生能源并网后电力市场波动性增大，传统强化学习方法难以平衡利润最大化与风险管理，且易过拟合特定市场条件的问题。

🛠 **方法**：提出MARS-DA分层强化学习框架，包含一个顶层元控制器动态混合两个专用基智能体的动作——“安全智能体”优化日前分配，“投机智能体”瞄准实时套利机会。

✨ **主要贡献**：开源了基于PJM市场数据的高保真双结算电力市场竞价环境，并验证MARS-DA在极端市场波动下能实现更优的风险调整收益，且保持稳健的市场状态对齐。
