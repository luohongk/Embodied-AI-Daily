# AutoPass: Evidence-Guided LLM Agents for Compiler Performance Tuning

- arXiv: https://arxiv.org/abs/2606.20373v1
- arXiv ID: 2606.20373

---

🔍 **核心问题**：现有编译器性能调优方法难以处理复杂的微架构效应和运行时噪声，且依赖黑盒调优或离线训练，缺乏对编译器内部状态的利用。

🛠 **方法**：提出AutoPass多智能体框架，通过让LLM查询编译器内部优化状态和分析中间表示，结合运行时反馈迭代调整优化配置，实现无需训练或微调的推理式调优。

✨ **主要贡献**：在LLVM编译器上实现，无需离线训练即可应用于新平台；在x86-64和ARM64上分别比LLVM -O3获得1.043倍和1.117倍的几何平均加速，超越专家启发式和经典自动调优方法。
