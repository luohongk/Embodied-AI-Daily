# Dehaze-GaussianImage: Zero-Shot Dehazing via Efficient 2D Gaussian Splatting Representation

- arXiv: https://arxiv.org/abs/2606.16163v1
- arXiv ID: 2606.16163

---

🔍 **核心问题**：现有单图像去雾方法在像素级优化中存在计算冗余，且隐式神经网络缺乏物理可解释性，难以兼顾表示效率与重建保真度。

🛠 **方法**：提出Dehaze-GaussianImage，首个将2D高斯泼溅（2DGS）引入图像去雾的零样本框架，通过将雾霾图像建模为连续可演化的各向异性高斯场，并嵌入大气散射模型到高斯参数空间，实现传输介质与清晰纹理的几何级解耦。

✨ **主要贡献**：在完全无监督且参数极少的条件下达到最先进性能，通过显式结构保持约束抑制传统物理先验的伪影，展示了显式高斯表示在低级视觉任务中的潜力。
