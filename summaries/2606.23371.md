# TSD: A Physics-Inspired Trajectory Saliency Detector for Efficient Imitation Learning

- arXiv: https://arxiv.org/abs/2606.23371v1
- arXiv ID: 2606.23371

---

🔍 **核心问题**：机器人模仿学习中高质量数据采集成本高、数据稀缺，且大量过渡性动作对任务成功贡献有限，需高效识别关键轨迹片段。

🛠 **方法**：提出无需训练的即插即用框架TSD，基于物理启发指标（空间熵捕捉精细操作、向心加速度检测敏捷动作）识别轨迹显著性，并用于数据集压缩（减少训练成本）与扩展（提升采集效率）。

✨ **主要贡献**：TSD在仿真与真实实验中平均减少25%数据量仍保持或超越原模型性能，验证了其压缩与扩展策略的有效性，为高效机器人学习提供低成本、可扩展的数据合成路径。
