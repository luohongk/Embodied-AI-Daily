# Societal Alignment Frameworks Can Improve LLM Alignment

- arXiv: https://arxiv.org/abs/2503.00069v2
- arXiv ID: 2503.00069

---

🔍 **核心问题**：当前大语言模型（LLM）的对齐方法难以充分捕捉人类价值观的复杂性，导致目标设定不完整，类似“不完全契约”问题。  
🛠 **方法**：提出从社会、经济和契约对齐等社会对齐框架中汲取洞见，并分析不确定性在LLM对齐中的表现，同时倡导参与式对齐界面设计。  
✨ **主要贡献**：将LLM对齐目标的不明确性重新定义为机遇而非缺陷，强调通过社会对齐框架改进技术，并呼吁更广泛的社会参与。
