# ReMoBot: Retrieval-Based Few-Shot Imitation Learning for Mobile Manipulation with Vision Foundation Models

- arXiv: https://arxiv.org/abs/2408.15919v4
- arXiv ID: 2408.15919

---

🔍 **核心问题**：解决移动操作机器人在有限演示数据和部分可观测条件下（如第一人称视觉）难以生成准确动作的问题。

🛠 **方法**：提出ReMoBot框架，利用视觉基础模型从演示中检索相关信息，结合状态相似性、历史轨迹对齐和动作序列一致性来识别相关演示，并以无需训练的方式选择控制命令。

✨ **主要贡献**：在仅20个演示样本下，ReMoBot在真实世界任务中达到70%-80%的成功率，并展现出对机器人位置、物体尺寸和材料属性的鲁棒泛化能力。
