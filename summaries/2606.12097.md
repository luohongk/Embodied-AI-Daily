## 0. 概览（Concise Summary）

- **主题（Topic）**：长期（一个月）概率性风电功率预测。具体任务是基于随机微分方程（SDE）和状态空间模型，对未来一个月的风速和风电功率进行高时间分辨率（10分钟）的概率性预测。
- **问题（Problem）**：现有长期风电预测方法要么是确定性预测，在长时域上误差累积严重；要么是概率性预测，但通常假设风速的分布参数（如威布尔分布的参数）是固定不变的，无法捕捉其随时间的动态变化，从而忽略了参数不确定性对未来预测的影响。
- **方法（Method）**：提出一个四阶段的条件概率预测框架：
    1.  **参数估计与校正**：从历史SCADA数据中估计每月威布尔参数，并使用Godambe协方差矩阵校正因数据序列相关性导致的估计误差。
    2.  **参数预测**：使用异方差卡尔曼滤波器（Kalman Filter）对双变量VAR(1)状态空间模型进行滤波和预测，得到未来一个月威布尔参数的MMSE（最小均方误差）点预测。
    3.  **风速SDE模拟**：以预测的威布尔分布为不变分布，构建并比较了三种正风速SDE模型：Ornstein-Uhlenbeck-Weibull变换模型、Fokker-Planck漂移优先模型和Fokker-Planck扩散优先模型。
    4.  **功率映射**：将模拟的风速集合通过一个校准好的XGBoost功率曲线模型，转换为风电功率的概率预测。
- **创新（Innovation）**：
    1.  **动态参数预测**：首次将威布尔参数的动态变化纳入长期风电预测框架，通过状态空间模型和卡尔曼滤波预测未来一个月的分布参数，而非使用历史固定值。
    2.  **条件概率框架**：构建了一个清晰的条件概率预测流水线，将参数预测、SDE模拟和功率映射有机结合，其输出是给定预测参数下的条件概率预测。
    3.  **三种SDE模型比较**：系统性地比较了三种不同的威布尔平稳SDE模型，并从统计精度和计算效率两个维度进行了评估，为实际应用提供了选择依据。
- **意义（Significance）**：为长期（月度）风电功率预测提供了一种全新的、严谨的概率性方法。该方法能够量化从参数估计、参数预测到风速随机过程的全链路不确定性，为电网调度、储能规划、备用容量安排等下游决策提供了更具信息量的输入。
- **一句话总结**：本文提出一个结合状态空间参数预测与三种随机微分方程的条件概率框架，实现了对未来一个月高分辨率风电功率的概率预测，并证明了不同SDE模型在统计精度上等价，但扩散优先模型计算效率更高。

## 1. 研究动机（Motivation）

随着风电渗透率的不断提高，准确的风电功率预测对电网的稳定运行和调度至关重要。然而，风电功率预测面临一个根本性挑战：**随着预测时域（如从小时级扩展到月级）的延长，确定性预测的误差会急剧累积，变得不可靠**。因此，对于长期预测，研究重点应从预测单一的未来轨迹，转向**刻画未来可能结果的概率分布**，即概率性预测。

在概率性预测领域，随机微分方程（SDE）因其能够结合物理驱动的动力学特性和预设的平稳分布，成为模拟风速的有力工具。现有研究（如Zárate-Miñano等人）已经提出了基于OU过程与威布尔变换的SDE模型。然而，这些工作存在一个**根本性局限**：它们通常假设风速概率分布的参数（如威布尔的形状和尺度参数）是**固定不变的**，仅从历史数据中估计一次。这意味着这些模型反映的是**过去的**风况特征，而非预测**未来的**不确定性。这种静态处理忽略了参数随时间的动态变化（如季节性、月际波动）以及参数估计本身的不确定性，这对于需要前瞻性信息的运营决策（如备用容量安排、储能规划）来说是不够的。

因此，本文的核心动机是**填补这一空白**：构建一个能够**动态预测未来一个月威布尔分布参数**，并在此基础上进行概率性风速和功率预测的完整框架。该框架需要能够量化从参数预测到随机模拟的全链路不确定性。

## 2. 创新点（Innovation）

本文的关键创新点在于提出了一个**条件概率预测流水线**，其核心是**对预测的威布尔不变分布进行条件化**。具体创新如下：

1.  **动态参数预测与不确定性量化**：区别于以往将威布尔参数视为常数的做法，本文首次将**状态空间模型（VAR(1)）** 与**异方差卡尔曼滤波器**应用于月度威布尔参数的预测。这不仅预测了参数的未来值（MMSE点预测），还通过卡尔曼滤波的预测协方差矩阵量化了参数预测的不确定性。这是整个框架的基石，使得后续的SDE模拟能够基于“预测的未来风况”而非“历史平均风况”。

2.  **三种威布尔平稳SDE模型的系统比较**：本文并非提出一个全新的SDE模型，而是系统地构建并比较了三种能够生成具有相同威布尔平稳分布的风速路径的SDE模型：
    - **Ornstein-Uhlenbeck-Weibull (OU-Weibull) 模型**：通过一个单调变换将高斯OU过程映射为威布尔过程，其优点是可以通过精确的AR(1)离散化进行模拟，避免了欧拉-丸山法的离散误差。
    - **Fokker-Planck 漂移优先 (Drift-First) 模型**：先指定一个线性均值回归漂移项，然后通过Fokker-Planck方程的平稳解推导出扩散项。
    - **Fokker-Planck 扩散优先 (Diffusion-First) 模型**：先指定一个恒定的扩散项，然后推导出相应的漂移项。
    这种系统性的比较是新颖的，它揭示了在统计精度上三者等价，但在计算效率上扩散优先模型具有显著优势。

3.  **条件概率预测的清晰定义**：论文明确界定了其输出是“**给定MMSE预测的威布尔不变分布**下的条件概率预测”。这种清晰的概率组成（Kalman预测律 → MMSE点估计 → 条件SDE模拟 → 功率映射）使得模型的不确定性来源透明化，并为未来进行完全边缘化（即考虑整个Kalman预测分布）的研究指明了方向。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

本文提出的预测框架是一个四阶段的流水线：

1.  **阶段一：威布尔参数估计与校正**
    - **输入**：历史月度SCADA风速数据。
    - **处理**：对每个月的风速数据，使用**最大似然估计（MLE）** 估计威布尔分布的形状参数 $k_m$ 和尺度参数 $\lambda_m$。
    - **关键步骤**：由于风速数据存在序列相关性，标准MLE的协方差会低估真实不确定性。因此，使用**Godambe（三明治）协方差矩阵**进行校正，该矩阵通过**Newey-West HAC估计器**计算，考虑了序列相关性对估计方差的影响。
    - **输出**：每个月的参数估计值 $\hat{\theta}_m = (\hat{k}_m, \hat{\lambda}_m)$ 及其校正后的协方差矩阵。

2.  **阶段二：威布尔参数的状态空间预测**
    - **输入**：阶段一得到的月度参数估计序列 $\hat{\eta}_m = (\log\hat{k}_m, \log\hat{\lambda}_m)$（对数变换后）。
    - **模型**：构建一个**双变量VAR(1)状态空间模型**。状态方程描述潜在参数的对数 $\eta_m$ 的动态演化，观测方程将潜在状态与有噪声的MLE估计 $\hat{\eta}_m$ 联系起来。
    - **关键步骤**：使用**异方差卡尔曼滤波器**进行滤波和预测。观测噪声的协方差矩阵由阶段一得到的校正后协方差矩阵决定，这意味着不同月份的估计精度不同，精度高的月份在滤波中权重更大。
    - **输出**：对下一个月的参数预测，包括**MMSE点预测** $(\hat{k}_{49}, \hat{\lambda}_{49})$ 和完整的**Kalman预测分布**（其协方差 $P_{49|48}$ 量化了参数预测的不确定性）。本文的数值实验仅使用MMSE点预测作为后续SDE的目标不变分布。

3.  **阶段三：风速的随机微分方程模拟**
    - **输入**：阶段二预测的威布尔参数 $(\hat{k}_{49}, \hat{\lambda}_{49})$ 和从历史数据中校准的均值回复率 $\hat{\alpha}$。
    - **处理**：构建三种正风速SDE模型，其**平稳分布**均为 $Weibull(\hat{k}_{49}, \hat{\lambda}_{49})$。通过蒙特卡洛模拟生成大量风速路径集合。
    - **输出**：一个包含 $B=100$ 条10分钟分辨率风速路径的集合。

4.  **阶段四：功率映射**
    - **输入**：阶段三模拟的风速集合。
    - **处理**：使用一个在历史清洁数据上训练好的**XGBoost回归模型**作为确定性功率曲线，将每条风速路径映射为功率路径。
    - **输出**：一个包含 $B=100$ 条10分钟分辨率风电功率路径的集合，即最终的**条件概率功率预测**。

### 3.2 关键算法与数学推导

**1. 威布尔分布与MLE**
风速 $v$ 的威布尔分布概率密度函数为：
$$f(v;k, \lambda) = \frac{k}{\lambda} \left( \frac{v}{\lambda} \right)^{k-1} \exp\left[ -\left( \frac{v}{\lambda} \right)^k \right], \quad v \ge 0$$
参数通过最大化对数似然函数估计。

**2. Godambe协方差矩阵校正**
由于数据序列相关，MLE $\hat{\theta}_m$ 的渐近协方差为：
$$\sqrt{|T_m|} (\hat{\theta}_m - \theta_m^*) \Rightarrow \mathcal{N}\left(0, I(\theta_m^*)^{-1} J(\theta_m^*) I(\theta_m^*)^{-1}\right)$$
其中 $I(\theta)$ 是Fisher信息矩阵，$J(\theta)$ 是得分过程的长期协方差（由Newey-West估计器估计）。这个“三明治”矩阵比逆Fisher信息矩阵更大，反映了序列相关导致的“有效样本量减少”。

**3. 状态空间模型与卡尔曼滤波**
状态方程（VAR(1)的伴随形式）：
$$x_m = c + F x_{m-1} + u_m, \quad u_m \sim \mathcal{N}(0, Q)$$
观测方程：
$$\hat{\eta}_m = H x_m + e_m, \quad e_m \sim \mathcal{N}\left(0, |T_m|^{-1} \hat{V}_m\right)$$
其中 $H$ 是选择矩阵，观测噪声协方差 $|T_m|^{-1} \hat{V}_m$ 是异方差的，由阶段一的校正协方差决定。

**4. 三种SDE模型的核心公式**

- **Fokker-Planck平稳恒等式**（Proposition 1）：对于平稳密度为 $p(x)$ 的扩散过程 $dX(t) = a(X(t)) dt + b(X(t)) dW(t)$，在零概率流条件下，有：
  $$b^2(x) = \frac{2}{p(x)} \int_0^x a(u) p(u) du, \quad x > 0$$

- **Fokker-Planck漂移优先模型**：
  指定漂移 $a(x) = \alpha(\mu_W - x)$，其中 $\mu_W$ 是威布尔均值。代入恒等式得到扩散系数 $b^2(x)$。

- **Fokker-Planck扩散优先模型**：
  指定恒定扩散 $b^2(x) = 2\alpha\sigma_W^2$，其中 $\sigma_W^2$ 是威布尔方差。代入恒等式推导出漂移：
  $$a(x) = \alpha \sigma_W^2 \left( \frac{k-1}{x} - \frac{k x^{k-1}}{\lambda^k} \right)$$

- **Ornstein-Uhlenbeck-Weibull模型**：
  使用OU过程 $dX_t = -\alpha X_t dt + \sqrt{2\alpha} dW_t$，其平稳分布为 $\mathcal{N}(0,1)$。通过单调变换 $g(x) = F_W^{-1}(\Phi(x))$ 得到风速过程 $V_t = g(X_t)$，其中 $F_W^{-1}$ 是威布尔分位数函数，$\Phi$ 是标准正态CDF。

**5. 均值回复率 $\alpha$ 的校准**
利用OU-Weibull模型的可逆性，将历史风速 $V_i$ 转换为高斯序列 $X_i = \Phi^{-1}(F_W(V_i))$。然后对 $X_i$ 拟合AR(1)模型 $X_{i+1} = \phi X_i + \sqrt{1-\phi^2} \epsilon_{i+1}$，得到 $\hat{\phi}$，进而得到 $\hat{\alpha} = -\Delta^{-1} \log \hat{\phi}$。

### 3.3 模型、训练与数据集

- **数据集**：英国Kelmarsh风电场1号Senvion MM92涡轮机的SCADA数据（2017年1月至2021年6月），时间分辨率为10分钟。
- **数据预处理**：一个四步清洗流程，包括物理范围过滤、基于图像处理的异常模式检测、去除限电/维护时段，以及使用XGBoost对缺失功率值进行插补（仅用于可视化）。
- **模型训练**：
    - **威布尔参数估计**：对每个月（约4000个数据点）独立进行MLE。
    - **Godambe协方差**：使用Newey-West估计器，带宽由数据驱动选择。
    - **VAR(1)状态空间模型**：使用前48个月（2017-2020）的月度参数估计序列进行训练。卡尔曼滤波器在此训练数据上运行。
    - **均值回复率 $\alpha$**：使用2017-2020年所有一月份的数据，通过**池化MLE**进行校准。
    - **XGBoost功率曲线**：使用清洗后的历史风速和功率数据训练。

### 3.4 实验设置与结果

- **预测目标**：对2021年1月进行一个月预测，时间分辨率为10分钟。
- **SDE模拟**：三种模型各生成 $B=100$ 条风速路径。通过稳定性分析（$B=50, 100, 150$）确认 $B=100$ 是计算成本与采样稳定性的良好折衷。
- **评估指标**：
    - **风速域**：连续排名概率评分（CRPS）、Wasserstein距离（$W_1$）、预测区间（PI）覆盖率、分位数-分位数（Q-Q）图。
    - **功率域**：Wasserstein距离、月能量产量偏差、超越概率误差。
- **主要结果**：
    1.  **风速预测**：三种SDE模型在概率精度上**统计等价**，平均CRPS值在1.569至1.575 m/s之间，差异仅为0.006 m/s，远小于蒙特卡洛标准误差。因此，**扩散优先模型因计算效率高（运行时间约为OU-Weibull模型的1/7）而被推荐**。
    2.  **预测区间校准**：所有模型的80% PI覆盖率约为73.5-73.9%，低于名义值80%。论文指出，这种偏差主要源于1月18-22日的一次持续强风风暴事件，该事件处于威布尔分布的尾部，模型缺乏应对这种持续性极端事件的机制。
    3.  **功率预测**：模拟与观测功率分布之间的Wasserstein距离为26.1-27.6 kW，低于额定容量的1.4%。月能量产量偏差约为-7.3%。在0-1500 kW范围内，超越概率误差低于1.6个百分点，在额定功率附近约为2.2个百分点。

### 3.5 技术细节

- **对数空间变换**：对威布尔参数 $k$ 和 $\lambda$ 取对数，将正数约束转化为无约束实数，便于状态空间建模，并通过Delta方法传播协方差。
- **异方差观测噪声**：卡尔曼滤波器的观测噪声协方差矩阵随月份变化，反映了不同月份数据量和序列相关性导致的估计精度差异。这是使用卡尔曼滤波器而非普通最小二乘法的关键原因。
- **边界行为分析**：论文对扩散优先模型在 $x \to 0^+$ 处的边界行为进行了分析，指出当形状参数 $k \ge 2$ 时，零边界不可达，保证了模拟的正性。对于本研究的 $k=2.6272$，此条件满足。
- **精确模拟算法**：OU-Weibull模型利用其高斯AR(1)结构，实现了精确的离散时间模拟，避免了欧拉-丸山法的离散误差。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    - 解决了长期风电预测中，**忽视分布参数动态变化**这一关键空白。
    - 提出了一个**模块化、可解释**的条件概率预测框架，清晰地分离了参数预测、随机模拟和功率映射三个环节。
    - 通过严谨的比较，为实际应用提供了**选择SDE模型的实用指南**：在统计精度无显著差异时，应优先考虑计算效率更高的扩散优先模型。
    - 证明了在月度时间尺度上，基于SDE的概率预测是可行的，并能提供具有决策价值的概率信息（如超越概率）。
