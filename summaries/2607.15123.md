# NIFA: Nonlinear IMC enhanced FPGA for efficient ML inference

- arXiv: https://arxiv.org/abs/2607.15123v1
- arXiv ID: 2607.15123

---

🔍 **核心问题**：传统FPGA中集成的ReRAM模拟存内计算（IMC）仅支持静态权重向量矩阵乘法，无法高效处理Transformer模型依赖的非线性运算和动态矩阵乘法，且ADC模块占用超70%面积与功耗，限制了能效与可扩展性。

🛠 **方法**：提出一种无ADC的IMC块架构，用模拟内容可寻址存储器（ACAM）替代ADC，在块内原生执行非线性运算；同时进行FPGA感知的设计空间探索，优化交叉阵列尺寸，并开发利用ACAM实现动态矩阵乘法的高效映射方法。

✨ **主要贡献**：在CNN和Transformer基准测试中，分别实现最高40倍和1.9倍的能效提升，以及4.1倍和2.5倍的面积效率提升；显著提高了FPGA深度学习推理效率，并在长序列Transformer任务中保持稳健增益。
