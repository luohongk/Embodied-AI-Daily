# Scalable Maximum Entropy Reinforcement Learning for Diffusion Policies via Adjoint Matching

- arXiv: https://arxiv.org/abs/2606.22630v1
- arXiv ID: 2606.22630

---

🔍 **核心问题**：扩散策略在在线强化学习中难以高效训练，因为缺乏真实数据时无法直接应用标准优化方法（如分数匹配）。  
🛠 **方法**：提出基于伴随匹配的算法，利用随机最优控制的最新进展，实现无需模拟的训练，避免显式似然估计或通过扩散过程反向传播的高成本。  
✨ **主要贡献**：显著降低计算开销，同时保持竞争性能，并通过多种扩展提升实际应用中的鲁棒性和稳定性，使扩散策略更适用于在线RL场景。
