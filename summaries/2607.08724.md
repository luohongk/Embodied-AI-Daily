# Latent Memory Palace: Reasoning for Control as Autoregressive Variational Inference

- arXiv: https://arxiv.org/abs/2607.08724v1
- arXiv ID: 2607.08724

---

🔍 **核心问题**：如何让连续控制策略具备类似语言模型的灵活推理能力，同时保持对空间理解和精确动作的细粒度控制。

🛠 **方法**：提出Latent Memory Palace（LMP），将推理建模为自回归潜空间中的变分推断，并通过潜空间强化学习优化其变分下界，从而生成可自适应调整推理步数的控制策略。

✨ **主要贡献**：首次将变分推断视角引入控制策略的潜空间推理，LMP-π在仿真和真实场景中表现优异，且能自适应分配测试时计算资源；同时LMP-tok作为变长动作分词器，显著提升下游自回归策略的性能。
