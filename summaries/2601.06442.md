# WHU-PCPR: A cross-platform heterogeneous point cloud dataset for place recognition in complex urban scenes

- arXiv: https://arxiv.org/abs/2601.06442v2
- arXiv ID: 2601.06442

---

🔍 **核心问题**：现有点云位置识别数据集在场景、平台和传感器多样性上不足，限制了相关研究的发展。

🛠 **方法**：构建了跨平台异构点云数据集WHU-PCPR，包含车载移动激光扫描（MLS）和便携式头盔激光扫描（PLS）系统采集的数据，覆盖城市和校园道路场景，轨迹总长82.3公里。

✨ **主要贡献**：提供了首个兼具跨平台异构点云、复杂定位场景和大规模空间覆盖的数据集，并基于此评估了多种代表性方法，指出了关键挑战和未来方向。
