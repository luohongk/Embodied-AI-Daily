# Calibrating Biophysical Models for Grape Phenology Prediction via Multi-Task Learning

- arXiv: https://arxiv.org/abs/2508.03898v2
- arXiv ID: 2508.03898

---

🔍 **核心问题**：传统生物物理模型预测葡萄物候期精度不足，而深度学习方法受限于稀疏的品种级数据集，难以实现精细化管理。

🛠 **方法**：提出一种混合建模方法，结合多任务学习与循环神经网络，通过多任务学习预测可微分生物物理模型的参数，实现跨品种共享学习并保留生物结构。

✨ **主要贡献**：在真实与合成数据集上，该方法在预测物候阶段及其他作物状态变量（如抗寒性、小麦产量）上，显著优于传统生物物理模型和基线深度学习方法。
