# RRT-Rope: A deterministic shortening approach for fast near-optimal path planning in large-scale uncluttered 3D environments

- arXiv: https://arxiv.org/abs/2606.31948v1
- arXiv ID: 2606.31948

---

🔍 **核心问题**：在大规模无遮挡3D环境（如无人机勘探的矿井采场）中，现有路径规划算法（如RRT变体）收敛速度慢且路径成本高，难以快速找到近最优解。

🛠 **方法**：提出RRT-Rope算法，先用改进版RRT-Connect快速计算可行路径，再通过确定性缩短技术（利用树分支的中间节点）对路径进行后处理优化。

✨ **主要贡献**：在模拟环境中，RRT-Rope的路径成本和计算时间均优于其他流行RRT变体及缩短方法，在代表性采场中比次优算法快70%。
