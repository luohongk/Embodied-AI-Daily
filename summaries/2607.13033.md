# DenseReward: Dense Reward Learning via Failure Synthesis for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2607.13033v1
- arXiv ID: 2607.13033

---

🔍 **核心问题**：机器人强化学习缺乏可靠的视觉-语言奖励模型，难以提供密集且信息丰富的反馈，主要受限于大规模失败数据获取困难和奖励信号稀疏。

🛠 **方法**：提出DenseReward模型，通过自动化仿真管道生成物理真实的失败轨迹（如碰撞、抓取失败、物体掉落等），并基于视觉观察和语言指令预测密集的帧级奖励分数。

✨ **主要贡献**：无需人工标注即可合成多样化失败数据，实现细粒度任务进度估计；在模拟和真实操作中优于通用VLM和现有奖励模型，并成功用于模型预测控制和强化学习。
