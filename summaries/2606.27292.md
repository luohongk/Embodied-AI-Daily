# BOWConnect: Parallel Bayesian Optimization over Windows with Learned Local Cost Maps for Sample-Efficient Kinodynamic Motion Planning

- arXiv: https://arxiv.org/abs/2606.27292v1
- arXiv ID: 2606.27292

---

🔍 **核心问题**：现有基于采样的运动规划方法在高维状态空间中样本效率低、动态约束下成本启发式不可靠，且在狭窄通道环境中性能差。

🛠 **方法**：提出BOWConnect，一种双向并行运动规划器，通过将贝叶斯优化与窗口学习（BOW）集成到并行树探索框架中，让每个工作节点学习局部成本地图和约束，引导采样生成动态可行且无碰撞的控制；同时采用双向架构并行生长前向和后向树，结合空间哈希和边界值求解器生成动力学一致的桥接轨迹。

✨ **主要贡献**：在十个基准环境中实现100%成功率，在复杂场景（如狭窄通道和非凸空间）中达到最快或接近最快的规划时间；在地面车辆和四旋翼飞行器上的实际部署验证了实时无碰撞规划能力，并开源了代码。
