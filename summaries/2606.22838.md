# FPAS: Frontier-Based Path Planning with Adaptive Sampling for Large-Scale Unknown Environments

- arXiv: https://arxiv.org/abs/2606.22838v1
- arXiv ID: 2606.22838

---

🔍 **核心问题**：在大规模未知环境中，现有路径规划方法在长距离导航时存在计算瓶颈或路径效率低下的问题。

🛠 **方法**：提出FPAS框架，重新定义前沿概念以引导目标导向任务，并基于前沿导出的开放度指标实现自适应采样，在开阔区域稀疏化节点以降低计算量，在狭窄通道保持密集采样以确保连通性。

✨ **主要贡献**：显著提升计算效率，同时保持高竞争力的目标到达性能，有效平衡了探索效率与路径质量。
