# Deep Learning for Dynamic Programming with Recursive Utility Using First-order Conditions

- arXiv: https://arxiv.org/abs/2607.09461v1
- arXiv ID: 2607.09461

---

🔍 **核心问题**：解决离散时间动态规划中递归效用函数因非线性确定性等价难以评估的挑战。  
🛠 **方法**：提出CEFOL算法，引入独立神经网络表示确定性等价，并利用贝尔曼方程和模型特定一阶条件构建残差，通过最小化残差学习值函数、策略函数和拉格朗日乘子。  
✨ **主要贡献**：直接处理控制变量的一般等式与不等式约束（包括偶尔约束），无需惩罚函数或问题重构；在风险敏感消费、Epstein-Zin模型等应用中，贝尔曼诊断和最优性残差达1.0e-4至1.0e-3量级，且学习结果与VFI基准高度吻合。
