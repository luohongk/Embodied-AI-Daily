# Advancing Multimodal Fusion on Heterogeneous Medical Data with Hybrid Geometry Attention

- arXiv: https://arxiv.org/abs/2607.19086v1
- arXiv ID: 2607.19086

---

🔍 **核心问题**：现有多模态融合方法难以有效捕捉复杂跨模态交互、计算成本高，且缺乏对异构医疗数据（如影像、临床记录、组学数据）的通用性和可扩展性。  
🛠 **方法**：提出CURE框架，通过轻量级级联统一表示学习，利用混合几何感知融合层（HyFuse）逐步整合各模态，其中残差卷积模块提取多尺度特征，混合空间注意力机制学习粗到细的结构线索，并辅以可学习的后期融合与共享信息精炼模块。  
✨ **主要贡献**：在16个公共数据集上，CURE相比主流方法性能提升最高达3.97%，计算成本降低最高达87.8%，同时具备对多种模态配置的适应性和泛化能力。
