# Task-Error Residual Learning for Real-Robot Five-Ball Juggling

- arXiv: https://arxiv.org/abs/2606.16978v1
- arXiv ID: 2606.16978

---

🔍 **核心问题**：如何通过残差学习高效提升真实机器人多球杂耍（三至五球）的稳定性，解决传统强化学习样本效率低、信息利用不足的问题。

🛠 **方法**：采用带方向性任务误差监督的残差学习，结合任务误差模型驱动样本选择，并对比了牛顿式雅可比更新、复合贝叶斯优化和随机搜索等方法。

✨ **主要贡献**：首次在仿人Barrett WAM机械臂上实现稳定三、四、五球杂耍，从第二次尝试开始收敛且任务误差单调递减；证明方向性反馈与信息性先验结合的必要性，其中固定雅可比牛顿更新最可靠；指出残差学习的瓶颈在于监督信号的信息含量及利用方式，而非系统精度。
