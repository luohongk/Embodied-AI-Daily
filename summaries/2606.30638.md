# Open-Vocabulary and Referring Segmentation for 3D Gaussians Using 2D Detectors

- arXiv: https://arxiv.org/abs/2606.30638v1
- arXiv ID: 2606.30638

---

🔍 **核心问题**：现有基于3D高斯泼溅（3DGS）的语言驱动场景理解方法，依赖密集的CLIP特征和预定义实例数量，导致语义理解受限且无法处理复杂空间推理与指代表达式定位。

🛠 **方法**：提出GaussDet，利用离散的开放词汇2D目标检测器（具备指代表达能力），为每个高斯体学习实例特征以分解场景为3D实例组，并通过多视图2D检测的语义投票生成视图聚合语义标签分布（VASD），实现零样本从简单语言查询到复杂指代定位的扩展。

✨ **主要贡献**：在开放词汇分割（LeRF-OVS、ScanNet）和指代表达式定位（Ref-LeRF）任务上取得一致改进，尤其在严格零样本设置下，指代定位的mIoU提升16.7%。
