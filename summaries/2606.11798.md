# Deterministic Policy Gradient for Learning Equilibrium in Time-Inconsistent Control Problems

- arXiv: https://arxiv.org/abs/2606.11798v1
- arXiv ID: 2606.11798

## 0. 概览（Concise Summary）

- **主题（Topic）**：连续时间强化学习（Continuous-Time RL）在时间不一致控制问题中的应用。
- **问题（Problem）**：传统强化学习方法通常假设时间一致性偏好，无法处理因非线性期望、非指数贴现等原因导致的时间不一致性问题。现有基于随机策略的连续时间RL方法在时间不一致场景下面临耦合的扩展HJB方程、采样困难、熵正则化近似误差等技术挑战。
- **方法（Method）**：提出一种基于确定性策略梯度（DPG）的无模型强化学习算法（DPG-FPI）。该算法将原始时间不一致问题转化为一个等价的**两阶段固定点问题**：第一阶段，对于给定的辅助函数，使用DPG学习一个辅助时间一致控制问题的最优策略；第二阶段，基于更新后的策略，利用内部固定点迭代和鞅特征来学习辅助函数。
- **创新（Innovation）**：首次将确定性策略梯度方法应用于连续时间时间不一致控制问题的均衡策略学习；提出一个统一的两阶段actor-critic框架，通过交替更新策略和辅助函数来学习均衡；为内部固定点迭代的收敛性提供了理论保证；利用鞅特征实现了对不可观测辅助函数及其导数的无模型学习。
- **意义（Significance）**：解决了时间不一致控制问题中无模型RL学习的核心挑战，为均值-方差投资组合管理、非指数贴现下的最优跟踪投资组合等经典金融问题提供了有效的学习算法。相比基于随机策略的方法，DPG-FPI在收敛速度、精度和稳定性方面表现更优。
- **一句话总结**：本文提出了一种基于确定性策略梯度的无模型连续时间强化学习算法，通过两阶段固定点迭代框架，有效学习时间不一致控制问题中的均衡策略。

## 1. 研究动机（Motivation）

**核心问题**：在金融、经济等领域，许多控制问题存在**时间不一致性**——即当前被认为最优的决策，在未来可能不再被采纳。例如：
- **均值-方差投资组合管理**：目标函数包含非线性期望（方差项），导致动态规划失效。
- **非指数贴现**：决策者采用双曲线贴现等非指数贴现函数，导致偏好随时间变化。
- **状态依赖风险厌恶**：效用函数依赖于初始状态。

**为什么重要**：时间不一致性使得传统最优控制理论（基于动态规划原理）无法直接应用。一种被广泛接受的替代方案是将问题重新建模为**当前自我与未来自我之间的内部博弈**，并寻求**子博弈完美纳什均衡**。Björk等人（2017）提出了扩展的Hamilton-Jacobi-Bellman（HJB）系统来刻画这种均衡。然而，该系统的数值求解面临两大挑战：
1. **高维复杂性**：扩展HJB系统由一族无限维非线性偏微分方程组成，求解困难。
2. **模型依赖**：求解需要完全知道模型系数（如漂移、扩散项），而在实际应用中这些系数往往是未知的。

**现有方法的局限**：近年来，连续时间RL取得了显著进展，但大多假设时间一致性偏好。将RL扩展到时间不一致场景面临以下技术障碍：
- **耦合的扩展探索HJB方程**：时间不一致性导致RL的核心工具（如探索HJB方程）变得复杂且耦合，难以直接推广。
- **随机策略的采样困难**：在连续时间、状态和动作空间中，从随机均衡策略的分布中采样非常困难，且其显式形式通常未知。
- **熵正则化的近似误差**：熵正则化会改变原始控制问题，在时间不一致场景下，即使温度参数很小，也可能导致显著的近似误差。

**本文的解决思路**：受Cheng等人（2025）关于确定性策略梯度（DPG）在时间一致控制中成功应用的启发，本文选择**确定性策略**作为替代方案，以规避上述随机策略带来的技术难题。核心问题是：当目标不再是寻求全局最优值，DPG能否学习到均衡策略？

## 2. 创新点（Innovation）

1. **两阶段固定点重构**：将原始时间不一致控制问题等价转化为一个**两阶段固定点问题**。第一阶段是给定辅助函数 $(f, g)$ 的标准时间一致最优控制问题；第二阶段是给定策略后，通过固定点条件更新辅助函数。这种分解不仅澄清了均衡的结构，也为迭代学习算法提供了自然基础。

2. **确定性策略梯度（DPG）在时间不一致问题中的首次应用**：在辅助时间一致控制问题中，利用DPG学习最优确定性策略。通过使用**优势率函数**（advantage rate function）的鞅特征，实现了无模型策略改进，避免了随机策略的采样困难和熵正则化误差。

3. **内部固定点迭代的收敛性理论**：为学习辅助函数 $(f, g)$，提出了一个内部迭代方案，并在温和的模型假设下（Assumption 3.1 和 4.1），严格证明了该迭代收敛到固定点（Theorem 4.2）。

4. **基于鞅特征的无模型学习**：建立了价值函数、优势率函数以及辅助函数及其相关优势率函数的鞅特征。这些特征将扩展HJB方程替换为鞅正交条件，可通过随机逼近实现参数更新，从而无需知道模型动态。

5. **局部探索技巧**：为了实现探索，在优势率函数和辅助函数的鞅特征中采用当前动作附近的局部探索。针对时间不一致问题中辅助函数对初始时间-状态 $(s, y)$ 的依赖，进一步采用对 $(s, y)$ 的局部探索（均匀分布和正态分布采样），使得鞅正交条件对任意 $(s, y)$ 都可行。

6. **结构化神经网络与目标网络**：算法利用结构化神经网络来利用理论分析揭示的解析形式，并采用目标网络技术来平滑迭代固定点更新。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

本文提出的**DPG-FPI算法**是一个两阶段actor-critic框架：

- **Actor阶段（第一阶段）**：固定辅助函数 $(f, g)$，将原始时间不一致问题转化为一个辅助时间一致最优控制问题。使用确定性策略梯度（DPG）更新策略参数 $\phi$，以最大化修改后的奖励函数 $\tilde{r}^{f,g}$ 的期望。
- **Critic阶段（第二阶段）**：给定更新后的策略 $\mu^\phi$，通过内部固定点迭代更新辅助函数 $(f, g)$。该迭代利用鞅特征将固定点条件转化为可实现的优化目标。

**算法流程**：
1. 初始化策略参数 $\phi$，辅助函数网络参数 $\theta_f, \theta_g$。
2. 重复以下步骤直到收敛：
   - **Actor更新**：对于当前 $(f, g)$，使用DPG更新 $\phi$，以最大化 $\tilde{r}^{f,g}$ 下的期望回报。
   - **Critic更新**：对于当前策略 $\mu^\phi$，通过内部迭代更新 $(f, g)$，使其满足固定点条件（即概率表示）。
   - 交替进行，直到策略和辅助函数都收敛。

### 3.2 关键算法与数学推导

**1. 两阶段固定点重构**

原始时间不一致问题的奖励函数为：
$$J(t, x; a) := \mathbb{E}_{t,x}\left[\int_t^T r(t, x, s, X_s^a, a_s) ds + F(t, x, X_T^a)\right] + G(x, \mathbb{E}_{t,x}[X_T^a])$$

通过引入修改后的运行奖励函数 $\tilde{r}^{f,g}$：
$$\tilde{r}^{f,g}(t, x, a) := r(t, x, t, x, a) - \mathcal{L}^a f(t, x, t, x) + \mathcal{L}^a f^{t,x}(t, x) - \mathcal{L}^a (G \diamond g)(t, x) + \partial_y G(x, g(t, x)) \mathcal{L}^a g(t, x)$$

其中 $\mathcal{L}^a$ 是生成算子：
$$\mathcal{L}^a \varphi(t, x) := \partial_t \varphi(t, x) + b(t, x, a)^\top \partial_x \varphi(t, x) + \frac{1}{2} \text{Tr}\left( \Sigma(t, x, a) \partial_{xx}^2 \varphi(t, x) \right)$$

则原始问题等价于：
- **第一阶段**：对于给定 $(f, g)$，求解标准时间一致控制问题：
  $$V(t, x) := \sup_{a \in \mathcal{A}} \mathbb{E}_{t,x}\left[ \int_t^T \tilde{r}^{f,g}(s, X_s^a, a_s) ds + F(T, X_T^a, X_T^a) + G(X_T^a, X_T^a) \right]$$
- **第二阶段**：对于给定策略 $\hat{a}^{f,g}$，寻找固定点 $(f, g)$ 满足：
  $$\begin{cases}
  f(t, x, s, y) = \mathbb{E}_{t,x}\left[ \int_t^T r(s, y, \ell, X_\ell^{\hat{a}^{f,g}}, \hat{a}^{f,g}(\ell, X_\ell^{\hat{a}^{f,g}})) d\ell + F(s, y, X_T^{\hat{a}^{f,g}}) \right] \\
  g(t, x) = \mathbb{E}_{t,x}\left[ X_T^{\hat{a}^{f,g}} \right]
  \end{cases}$$

**2. 确定性策略梯度（DPG）公式**

对于参数化确定性策略 $\mu^\phi$，在辅助时间一致问题中，性能准则为：
$$V^\phi(t, x) := \mathbb{E}\left[ \int_0^T \tilde{r}^{f,g}(s, X_s^\phi, \mu^\phi(s, X_s^\phi)) ds + F(T, X_T^\phi, X_T^\phi) + G(X_T^\phi, X_T^\phi) \right]$$

DPG公式（Theorem 3.1）：
$$\partial_\phi V^\phi(t, x) = \mathbb{E}\left[ \int_t^T \partial_\phi \mu^\phi(s, X_s^\phi)^\top \partial_a A^\phi(s, X_s^\phi, \mu^\phi(s, X_s^\phi)) ds \mid X_t^\phi = x \right]$$

其中优势率函数 $A^\phi(t, x, a) = \mathcal{L}^a V^\phi(t, x) + \tilde{r}^{f,g}(t, x, a)$。

**3. 鞅特征（Theorem 3.2）**

优势率函数可以通过鞅条件来表征：存在 $\hat{V}$ 和 $\hat{q}$ 满足：
- $\hat{V}(T, x) = F(T, x, x) + G(x, x)$，$\hat{q}(t, x, \mu^\phi(t, x)) = 0$
- 对于所有 $a$ 在 $\mu^\phi(t, x)$ 的邻域内，过程 $\left( \hat{V}(s, X_s^{t,x,a}) + \int_t^s (\tilde{r} - \hat{q})(u, X_u^{t,x,a}, \alpha_u) du \right)_{s \in [t,T]}$ 是一个鞅。

则 $\hat{V} = V^\phi$ 且 $\hat{q} = A^\phi$。

**4. 内部固定点迭代的收敛性（Theorem 4.2）**

对于给定策略 $\mu^{f,g}$，定义迭代序列：
$$\begin{cases}
f_{k+1}(t, x, s, y) = \mathbb{E}_{t,x}\left[ \int_t^T r(s, y, \ell, X_\ell^{f_k,g_k}, \mu^{f_k,g_k}(\ell, X_\ell^{f_k,g_k})) d\ell + F(s, y, X_T^{f_k,g_k}) \right] \\
g_{k+1}(t, x) = \mathbb{E}_{t,x}\left[ X_T^{f_k,g_k} \right]
\end{cases}$$

在Assumption 3.1和4.1下，该序列收敛到固定点 $(f, g)$。证明思路：
- 首先证明在时间区间 $[T-\varepsilon, T]$ 上，迭代是一个压缩映射（contraction）。
- 然后通过向后归纳法，将收敛性扩展到整个时间域 $[0, T]$。
- 关键估计：$\|\tilde{f}_{k+1} - \tilde{f}_k\|_{(2)} + \|\tilde{g}_{k+1} - \tilde{g}_k\|_{(2)} \leq C \delta^k$，其中 $\delta \in (0,1)$。

### 3.3 模型、训练与数据集

**模型**：
- **状态动态**：$dX_s = b(s, X_s, a_s) ds + \sigma(s, X_s, a_s) dW_s$，其中 $b$ 和 $\sigma$ 未知。
- **奖励函数**：$r(t, x, s, y, a)$ 和终端奖励 $F, G$ 已知。
- **策略**：确定性反馈策略 $a: [0,T] \times \mathbb{R}^n \to \mathcal{A}$，参数化为 $\mu^\phi$。

**训练**：
- **算法**：DPG-FPI（Algorithm 1），包含外循环（actor-critic交替）和内循环（固定点迭代）。
- **神经网络**：使用结构化神经网络来近似价值函数 $V$、优势率函数 $A$、辅助函数 $f$ 和 $g$。网络结构利用理论分析揭示的解析形式（如均值-方差问题中的二次型形式）。
- **目标网络**：用于平滑固定点更新，提高稳定性。
- **局部探索**：在优势率函数和辅助函数的鞅特征中，对当前动作 $a$ 添加局部噪声（如高斯噪声）以实现探索。对于辅助函数 $f(t,x,s,y)$，还对 $(s,y)$ 进行探索（从均匀分布和正态分布中采样）。

**数据集**：
- 通过与环境交互生成轨迹数据。在每个episode中，从初始状态 $(t,x)$ 开始，使用当前策略 $\mu^\phi$ 生成状态轨迹 $\{X_s\}_{s \in [t,T]}$。
- 对于辅助函数的学习，需要从多个初始时间-状态对 $(s,y)$ 采样，以学习 $f$ 对 $(s,y)$ 的依赖。

### 3.4 实验设置与结果

**实验1：均值-方差投资组合管理**
- **问题**：经典的时间不一致问题，目标是最小化方差的同时最大化期望收益。
- **设置**：已知扩展HJB系统的显式解，因此可以精确参数化网络，隔离学习动态与近似误差。
- **结果**：
  - 经过10000个episode迭代，学习到的均衡策略和均衡价值函数与真实值高度吻合。
  - 与基于随机策略的q-learning算法相比，DPG-FPI在收敛速度、精度以及对探索和模型参数的稳定性方面表现更优（与Cheng et al. 2025在时间一致控制中的发现一致）。

**实验2：非指数贴现下的最优跟踪投资组合**
- **问题**：决策者采用非指数贴现函数（如双曲线贴现），导致时间不一致。
- **设置**：价值函数和辅助函数具有半解析表达式，因此采用结构化神经网络来利用这一结构。
- **结果**：
  - 学习到的均衡策略和均衡价值函数与理论结果紧密匹配。
  - 与时间一致情况（指数贴现，使用DPG算法）相比，时间不一致情况下的学习任务表现出更大的方差和不稳定性，这突出了学习非标准时间偏好下均衡策略的额外难度。

### 3.5 技术细节

- **Assumption 3.1**：确保DPG公式的良定义性，包括系数函数的Lipschitz连续性、增长条件、策略的光滑性等。
- **Assumption 4.1**：确保内部固定点迭代的收敛性，包括系数函数的高阶光滑性、策略对辅助函数的Lipschitz依赖性等。
- **Lemma 4.1**：证明迭代序列 $\{(\tilde{f}_k, \tilde{g}_k)\}$ 在 $(3)$-范数下一致有界，这是证明收敛性的关键。
- **Theorem 4.2**：通过时间区间划分和向后归纳法，证明迭代序列在 $(2)$-范数下是柯西序列，从而收敛到固定点。
- **鞅特征**（Propositions 4.4和4.5）：为辅助函数 $f$ 和 $g$ 及其优势率函数建立了鞅特征，使得可以通过随机逼近来学习这些函数，而无需知道模型动态。

## 4. 意义与影响（Significance and Impact）

**解决的问题**：
- 填补了连续时间RL在时间不一致控制问题中的空白，提出了一种无模型学习均衡策略的统一方法。
- 克服了基于随机策略的方法在时间不一致场景下面临的耦合HJB方程、采样困难和熵正则化误差等关键挑战。

**贡献**：
1. **理论贡献**：建立了时间不一致控制问题与确定性策略梯度之间的桥梁；证明了内部固定点迭代的收敛性。
2. **算法贡献**：提出了DPG-FPI算法，该算法在actor-critic框架下交替更新策略和辅助函数，利用鞅特征实现无模型学习。
3. **实证贡献**：在两个经典金融应用（均值-方差投资组合和非指数贴现跟踪投资组合）中验证了算法的有效性，展示了其优于随机策略方法的性能。

**对未来研究/应用的影响**：
- 为时间不一致控制问题的RL研究开辟了新方向，特别是确定性策略在连续时间RL中的应用。
- 为金融领域的实际应用（如投资组合管理、风险管理）提供了可行的无模型学习工具。
- 提出的两阶段固定点框架可能适用于其他涉及时间不一致性的领域（如行为经济学、动态契约理论）。

**局限与开放问题**：
- 算法依赖于一些理论假设（如Assumption 3.1和4.1），这些假设在实际应用中可能不完全满足。
- 内部固定点迭代的收敛性证明要求辅助函数
