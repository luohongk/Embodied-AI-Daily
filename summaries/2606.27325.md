# Not All Actions Are Equal: Rethinking Conditioning for Dexterous World Model

- arXiv: https://arxiv.org/abs/2606.27325v1
- arXiv ID: 2606.27325

---

🔍 **核心问题**：现有世界模型在处理高自由度（high-DoF）灵巧动作时，将整个动作序列压缩为单一表征，导致优化不平衡，难以建模细微动作效果，影响动作保真度。

🛠 **方法**：提出DexAC-WM，通过动作标记化保留维度级语义，结合局部细化和全局调制对齐动作信号与视觉动态；并引入语义分支提供丰富的物体-场景先验，增强高层语义基础。

✨ **主要贡献**：在EgoDex和EgoVerse上显著提升FID、FVD和PCK指标，验证了结构化动作建模与语义基础对高自由度控制世界模型的可扩展性和有效性。
