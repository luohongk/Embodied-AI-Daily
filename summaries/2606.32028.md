# DVG-WM: Disentangled Video Generation Enables Efficient Embodied World Model for Robotic Manipulation

- arXiv: https://arxiv.org/abs/2606.32028v1
- arXiv ID: 2606.32028

---

🔍 **核心问题**：现有基于视频的具身世界模型在预测机器人操作时，难以同时兼顾低层动态建模的精确性与高层语义驱动的视觉合成质量，导致推理速度慢或预测结果粗糙，无法保留接触细节。

🛠 **方法**：提出解耦视频生成世界模型（DVG-WM），将世界模型显式分解为动态学习与视觉合成两部分。先基于初始观测和语言指令生成中间视觉状态序列，再通过流匹配将动态直接映射到视频潜在空间，并引入潜在退化机制以再生接触细节。

✨ **主要贡献**：在LIBERO和真实平台实验中，DVG-WM在提升视频质量的同时实现了最高3.97倍的加速，验证了解耦视频生成可作为高效的具身世界模型用于机器人操作。
