# Native Active Perception as Reasoning for Omni-Modal Understanding

- arXiv: https://arxiv.org/abs/2606.19341v1
- arXiv ID: 2606.19341

---

🔍 **核心问题**：传统长视频理解模型采用“全量观看”范式，计算成本随视频时长线性增长，且交互式框架依赖全局预扫描，未能将推理复杂度与视频原始长度解耦。

🛠 **方法**：提出首个原生全模态智能体OmniAgent，将视频理解建模为基于POMDP的迭代“观察-思考-行动”循环，通过按需行动选择性提取音视频线索并存入持久化文本记忆；同时引入智能体监督微调（通过最佳N轨迹合成与双阶段质量控制）和基于TAURA的智能体强化学习（利用轮次级熵引导信用分配）。

✨ **主要贡献**：OmniAgent在10个基准测试（如VideoMME、LVBench）中达到开源模型最优性能，其中7B参数模型在LVBench上超越10倍大的Qwen2.5-VL-72B（50.5% vs 47.3%），并展现出正向测试时扩展特性——推理轮次增加时性能持续提升，验证了主动感知的有效性。
