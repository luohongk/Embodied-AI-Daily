# MOAR Planner: Multi-Objective and Adaptive Risk-Aware Path Planning for Infrastructure Inspection with a UAV

- arXiv: https://arxiv.org/abs/2606.30575v1
- arXiv ID: 2606.30575

---

🔍 **核心问题**：无人机在基础设施巡检中需近距离绕障飞行，同时应对天气、通信可靠性、电池续航等动态风险因素，现有方法难以实时平衡安全、时间和能耗的多目标优化。

🛠 **方法**：提出MOAR路径规划器，采用风险感知代价函数（集成预计算代价地图、损伤成本与插入成本概念）及自适应速度规划框架，通过离散状态-动作空间图搜索实时生成轨迹。

✨ **主要贡献**：在仿真与真实飞行测试中，该方法生成的轨迹在安全、时间、能耗等指标上覆盖主流算法约90%的性能范围，显著提升了无人机在关键任务中的自主导航可靠性与实时性。
