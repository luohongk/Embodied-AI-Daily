# AutoSpec: Safety Rule Evolution for LLM Agents via Inductive Logic Programming

- arXiv: https://arxiv.org/abs/2606.24245v2
- arXiv ID: 2606.24245

---

🔍 **核心问题**：大语言模型（LLM）智能体在自动化复杂任务时存在安全风险，现有方法中手工规则过于僵化（高误报或高漏报），而神经分类器缺乏可解释性，难以在安全关键场景中部署。

🛠 **方法**：提出AutoSpec框架，通过归纳逻辑编程（ILP）引导的CEGIS（反例引导归纳合成），从用户的安全/不安全标注中自动演化专家设计的规则，迭代挖掘反例、学习判别谓词并生成规则编辑，直至收敛。

✨ **主要贡献**：在两个领域（代码执行和具身智能）的291条轨迹上，将规则F1分数提升至0.98和0.93，误报率降低高达94%，且仅需4-5次迭代收敛；相比启发式CEGIS，F1提升达4.8倍；生成的规则可读、可审计，并能泛化到未见场景。
