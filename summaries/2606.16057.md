# A Smart-Scheduled Hybrid (SSH) EKF-FGO State Estimation

- arXiv: https://arxiv.org/abs/2606.16057v1
- arXiv ID: 2606.16057

---

🔍 **核心问题**：在机器人状态估计中，如何通过显式控制优化调度来平衡估计精度与计算成本，特别是解决滤波与优化混合系统中调度策略未被充分研究的问题。

🛠 **方法**：提出一种智能调度混合（SSH）EKF-FGO框架，将EKF的实时状态传播与周期性触发的因子图批量优化相结合，并固定求解器结构与计算负载，以隔离优化调度作为独立设计变量进行实验分析。

✨ **主要贡献**：通过平面SLAM仿真实验，首次系统表征了优化调度对预优化漂移、瞬态误差和运行时间的影响，并识别出能以较低计算成本保留全局优化大部分收益的工作区间，揭示了优化调度在混合状态估计中的关键作用。
