# Memory Retrieval in Visuomotor Policies for Long-Horizon Robot Control

- arXiv: https://arxiv.org/abs/2606.25136v1
- arXiv ID: 2606.25136

---

🔍 **核心问题**：在部分可观测环境中（如家庭），机器人需要从长期历史记忆中检索多样化信息以支持自主控制，但现有方法面临两个挑战：策略可能学习到过去信息与动作之间的虚假相关性，以及记忆误差累积导致模型漂移和级联失败。

🛠 **方法**：提出HALO框架，结合基于注意力的记忆检索机制。通过蒸馏视觉-语言模型（VLM）先验，利用演示轨迹生成记忆相关的问答对，并联合视频问答目标训练以抑制虚假相关性；同时采用稀疏注意力机制，仅检索历史中最相关的部分，减少闭环控制中记忆误差的影响。

✨ **主要贡献**：HALO实现了从长达8分钟的历史经验中可靠检索任务相关信息，显著提升了长期控制任务的鲁棒性，并通过实验验证了其在抑制虚假相关性和缓解误差累积方面的有效性。
