# EFlow: Learning Evidence Flow for Long-Video Reasoning with Adaptive Reflection

- arXiv: https://arxiv.org/abs/2607.00867v2
- arXiv ID: 2607.00867

---

🔍 **核心问题**：现有工具增强的视频推理框架常将时间定位与答案推理交织在同一轨迹中，导致早期语义假设偏差（即“过早语义承诺”），使得证据检索不完整并强化错误推理。

🛠 **方法**：提出EFlow框架，基于Qwen3-VL，通过分离时间定位与逻辑推理（分别使用CoT for Temporal Grounding和CoT for Reasoning），并引入置信度感知的反射机制，在证据不足时重新评估整个视频。同时构建专用轨迹数据集，结合监督微调、强化学习与强化微调进行训练。

✨ **主要贡献**：在五个视频理解基准上，EFlow显著提升了长视频推理性能，有效解决了过早语义承诺问题，实现了证据优先的可靠推理。
