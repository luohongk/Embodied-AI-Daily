# Lifecycle-Aware Dynamic Analysis for Secure ML Model Execution

- arXiv: https://arxiv.org/abs/2606.19023v1
- arXiv ID: 2606.19023

---

🔍 **核心问题**：现有ML模型安全扫描方案依赖静态规则或已知攻击签名，难以跨框架泛化且无法检测新型攻击路径。

🛠 **方法**：提出动态生命周期感知方法Moat，基于ML模型执行各阶段与宿主系统交互的结构化规律，通过实时监控异常行为实现安全检测，并实现参考系统Re-Moat。

✨ **主要贡献**：在Hugging Face Hub的77,974个真实模型、31个CVE概念验证及334个先进数据集模型上验证，该方法能检测所有攻击类别且误报率接近零，优于现有方案。
