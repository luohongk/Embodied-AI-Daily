# Quantized Evolution Strategies: High-precision Fine-tuning of Quantized LLMs at Low-precision Cost

- arXiv: https://arxiv.org/abs/2602.03120v2
- arXiv ID: 2602.03120

---

## 0. 概览（Concise Summary）

- **主题（Topic）**：大语言模型（LLM）的量化与微调，具体为在低精度（如INT4）量化空间中对LLM进行全参数微调。
- **问题（Problem）**：现有量化后微调方法面临两大挑战：1）梯度消失与离散化误差导致优化停滞（stagnation problem）；2）存储高精度优化器状态（如误差残差）会抵消量化带来的内存优势。
- **方法（Method）**：提出**量化进化策略（QES）**，一种无需反向传播的优化范式。核心包括：1）**累积误差反馈**机制，模拟Delta-Sigma调制，将微小梯度信号累积至跨过量化阈值，实现高精度更新轨迹；2）**无状态种子重放**机制，通过重放历史随机种子和奖励来在线重构误差状态，将优化器内存降至推理级水平。
- **创新（Innovation）**：首次将信号处理中的Delta-Sigma调制原理引入量化LLM优化，解决了离散空间中梯度估计的停滞与不准确问题；同时通过种子重放实现了内存与计算的有效权衡。
- **意义（Significance）**：使得在消费级硬件上直接对量化模型进行全参数微调成为可能，显著降低了LLM微调的门槛，并为在相同内存下扩展更大规模模型提供了新方向。
- **一句话总结**：QES通过累积误差反馈和无状态种子重放，在低精度成本下实现了对量化LLM的高精度全参数微调。

## 1. 研究动机（Motivation）

- **问题**：大语言模型（LLM）在部署时通常使用后训练量化（PTQ）来降低内存占用（如INT4），但量化后的模型参数空间是离散且不可微的，导致传统的基于反向传播的微调方法（如SGD、Adam）无法直接应用。
- **为什么重要**：PTQ使推理变得高效，但模型成为静态工件。任何微调都需在量化前进行，这需要大规模计算集群，普通用户难以负担。因此，能够直接在量化模型上进行微调的方法至关重要。
- **挑战**：
    1.  **梯度消失（Stagnation）**：进化策略（ES）或零阶（ZO）方法在连续空间有效，但在离散的量化参数空间中，微小的梯度更新信号（如 $\alpha \hat{g}$）往往小于量化步长（如1），导致舍入后更新量为0，优化停滞。
    2.  **离散化不准确**：即使更新量非零，舍入操作也会引入误差，导致参数更新方向不准确，降低优化效率。
    3.  **内存瓶颈**：为解决上述问题而引入的高精度误差残差向量（如FP16）会消耗大量显存，抵消量化带来的内存优势。

## 2. 创新点（Innovation）

- **关键创新1：累积误差反馈（Accumulated Error Feedback）**
    - **新方法**：受Delta-Sigma调制启发，QES维护一个高精度误差向量 $e_t$，将每次离散化更新后的残差（$u_t - \Delta W_t$）累积起来，并反馈到下一次更新中。
    - **与现有方案不同**：不同于直接舍入更新（导致停滞）或随机舍入（导致高方差），QES通过确定性累积，使得微小的梯度信号能在多次迭代中累积，直到跨过量化阈值（$|u| \ge 0.5$），从而在离散网格上模拟出高精度连续优化的轨迹。
    - **为何奏效**：该机制保证了虚拟连续参数 $\Theta_t = W_t + e_t$ 遵循理想的高精度梯度上升动态，而物理量化参数 $W_t$ 与理想轨迹的偏差被严格限制在半个量化步长内（$\|e_T\|_\infty \le \Delta/2$）。

- **关键创新2：无状态种子重放（Stateless Seed Replay）**
    - **新方法**：不直接存储高精度误差向量 $e_t$，而是通过维护一个轻量级的历史缓冲区 $H_t$（仅包含过去K步的随机种子 $S$ 和标量奖励 $F$），在每次更新时在线重放历史步骤，重新模拟误差累积过程。
    - **与现有方案不同**：将优化器状态的内存复杂度从 $O(d)$（d为参数数量）降低到 $O(K)$（K为窗口大小，通常为50），实现了推理级的内存占用。
    - **为何奏效**：由于衰减因子 $\gamma < 1$，历史误差贡献呈指数级衰减（$\gamma^K \approx 0$），因此截断历史的影响可忽略。同时，由于离散更新稀疏，使用当前权重 $W_t$ 近似历史权重 $W_\tau$ 进行边界检查的误差极小。

## 3. 主要内容（Main Content）【重点】

### 3.1 设计架构与方法

QES的整体架构如图1所示。其核心思想是在离散的量化参数空间中进行优化，同时通过累积误差反馈保持与高精度优化轨迹的“时间等价性”。

**流程概述**：
1.  **扰动生成**：对当前量化权重 $W_t$，使用随机舍入生成离散扰动 $\delta_i$。
2.  **适应度评估**：对每个扰动后的模型 $W_t + \delta_i$ 进行推理，获得奖励 $F_i$。
3.  **梯度估计**：使用ES公式估计梯度方向 $\hat{g}_t$。
4.  **累积误差更新**：将梯度与历史误差结合，得到高精度更新量 $u_t$。
5.  **离散化与误差累积**：将 $u_t$ 舍入为离散更新 $\Delta W_t$，并将残差 $e_{t+1} = u_t - \Delta W_t$ 累积到下一次迭代。
6.  **权重更新**：$W_{t+1} = W_t + \Delta W_t$。

### 3.2 关键算法与数学推导

**1. 标准ES梯度估计（连续空间）**：
$$\hat{g} = \frac{1}{N\sigma} \sum_{i=1}^{N} F_i \cdot \epsilon_i$$
其中 $\epsilon_i \sim \mathcal{N}(0, I)$ 是高斯扰动，$\sigma$ 是扰动尺度，$F_i$ 是归一化后的奖励。

**2. 离散扰动生成**：
为了在量化空间操作，使用随机舍入将连续高斯噪声 $\sigma\epsilon$ 转换为离散扰动 $\delta$：
$$\delta = \lfloor \sigma\epsilon \rfloor + b, \quad \text{where } b \sim \text{Bernoulli}(\sigma\epsilon - \lfloor \sigma\epsilon \rfloor)$$

**3. 边界门控（Boundary Gating）**：
确保更新后的权重仍在量化码本 $[0, 2^B - 1]$ 内：
$$W'_{ij} = \begin{cases} W_{ij} + \delta_{ij} & \text{if } 0 \le W_{ij} + \delta_{ij} < 2^B \\ W_{ij} & \text{otherwise} \end{cases}$$

**4. 离散空间梯度估计**：
$$\hat{g} = \frac{1}{N\sigma} \sum_{i=1}^{N} F_i \cdot \delta_i$$

**5. 累积误差反馈（核心）**：
这是QES解决停滞问题的关键，其更新动态如下：
$$u_t = \alpha \hat{g}_t + \gamma e_{t-1} \quad \text{(高精度更新量)}$$
$$\Delta W_t = \text{Round}(u_t) \quad \text{(离散化更新)}$$
$$e_t = u_t - \Delta W_t \quad \text{(更新累积误差)}$$
其中 $\alpha$ 是学习率，$\gamma \in (0,1]$ 是衰减因子。

**6. 时间等价性证明**：
定义虚拟连续参数 $\Theta_t \triangleq W_t + e_t$。通过上述更新规则可以推导出：
$$\Theta_{t+1} = \Theta_t + \alpha \hat{g}_t$$
这表明虚拟参数 $\Theta_t$ 遵循理想的高精度梯度上升动态。而物理参数 $W_T$ 与理想轨迹的偏差仅为最终残差 $e_T$，且 $\|e_T\|_\infty \le \Delta/2$。

### 3.3 模型、训练与数据集

- **模型**：
    - **SFT任务**：RoBERTa-large。
    - **推理任务**：Qwen2.5-1.5B, Qwen2.5-3B, Llama-3.1-8B。
- **量化**：使用GPTQ将模型量化为INT4、INT8，以及使用LLM-Compressor量化为W8A8（权重和激活均为INT8）。
- **数据集**：
    - **SFT**：SNLI, MNLI, RTE, SST-5（分类任务）。
    - **推理**：Countdown（算术表达式生成），GSM8K（小学数学应用题）。
- **训练设置**：
    - **优化器**：QES（Algorithm 1）。
    - **超参数**：种群大小 $N=8$（SFT）或 $N=50$（推理），衰减因子 $\gamma=0.9$，重放窗口 $K=50$（推理）或 $K=16$（SFT）。具体学习率 $\alpha$ 和扰动尺度 $\sigma$ 见附录A。

### 3.4 实验设置与结果

- **基线方法**：Base Model（未微调）、QuZO（量化零阶方法）、MeZO（全精度零阶方法）、First-Order（全精度/量化一阶方法）、Full Residual（QES的Oracle变体，存储FP16残差）。
- **主要结果**：
    - **SFT任务（表1）**：在W8（8-bit权重）设置下，QES平均准确率（44.4%）显著优于QuZO（34.2%）和W8一阶方法（41.0%），甚至超过了全精度MeZO（36.5%）。
    - **推理任务（表2）**：在所有模型大小和量化格式上，QES均取得一致且显著的提升。例如，在Qwen2.5-3B W8A8的GSM8K任务上，QuZO仅4.40%，而QES达到80.82%。
    - **扩展性案例（附录C）**：在Llama-3.1-8B INT4上，QES将GSM8K准确率从64.14%提升至82.64%，证明了其跨模型家族和更大规模的有效性。
- **消融实验**：
    - **无状态种子重放 vs. Full Residual（表6）**：两者性能非常接近，证明种子重放引入的近似误差极小。
    - **窗口大小K与衰减因子γ（附录D）**：在固定 $\gamma=0.9$ 下，即使 $K=20$ 也能保持大部分性能，提供了计算-内存权衡的灵活性。
    - **更新比率与边界命中率（附录D）**：更新比率约 $10^{-2}$，边界命中率 $\rho < 10^{-5}$，解释了种子重放高保真度的原因。

### 3.5 技术细节

- **累积误差反馈的数学原理**：通过将误差残差 $e_t$ 视为“虚拟连续参数” $\Theta_t$ 的小数部分，QES确保了离散更新 $\Delta W_t$ 的长期累积效果与连续梯度上升一致。这解决了式(10)中统计无状态方法（如QuZO）的“累积量化损失”问题。
- **无状态种子重放的具体实现（Algorithm 2）**：
    1.  从历史缓冲区 $H$ 中取出过去K步的种子和奖励对 $(S_\tau, F_\tau)$。
    2.  对于每一步 $\tau$，使用历史种子重新生成噪声 $\epsilon$，并重新计算梯度 $\hat{g}$。
    3.  模拟误差累积过程，使用当前权重 $W_t$ 进行边界检查，得到一个“代理残差” $\tilde{e}$。
    4.  最后，使用当前步的种子和奖励，结合重放得到的代理残差 $\tilde{e}$，执行正式的更新。
- **内存与计算权衡**：种子重放将优化器状态从 $O(d)$（如FP16残差，1.5B模型需2.44GB）降低到 $O(K)$（约29.7KB）。代价是每次更新需额外进行K次重放计算，导致总训练时间增加约12.5%-16.7%。

## 4. 意义与影响（Significance and Impact）

- **解决的问题与贡献**：
    1.  **解决了量化模型微调的“不可能三角”**：即同时实现全参数更新、低内存占用和有效优化。QES通过累积误差反馈解决了优化停滞，通过种子重放解决了内存瓶颈。
    2.  **提供了理论保证**：证明了QES的虚拟参数动态与连续梯度上升等价，且物理参数与理想轨迹的偏差有界，为方法的有效性提供了理论支撑。
    3.  **显著提升了性能**：在多个任务上大幅超越现有量化微调方法，甚至在某些情况下超越全精度零阶方法。

- **对未来研究/应用的影响**：
    - **民主化LLM微调**：使得在消费级GPU上对量化大模型进行全参数微调成为可能，降低了技术门槛。
    - **扩展模型规模的新范式**：作者指出，在相同硬件下，可以量化一个参数多4倍（如INT4）的模型，并使用QES直接在量化空间训练。结合QES的低内存需求，未来可能训练比现有方法大1-2个数量级的模型。
    - **跨领域应用**：QES的框架（累积误差+状态重放）可能启发其他离散优化问题，如神经架构搜索、硬件设计等。

- **局限与开放问题**：
    - **量化格式限制**：目前仅测试了标准线性整数量化（INT4, INT8, W8A8），扩展到更激进的格式（如二值网络、FP4）是未来方向。
    - **超参数自适应**：窗口大小K和衰减因子γ目前是手动设定的，未来可研究自适应算法。
    - **计算开销**：种子重放引入了额外的计算开销，虽然可控，但在超大规模模型上仍需优化。

## 5. 澄清与简化（Clarifications and Simplifications）

- **类比：Delta-Sigma调制与“存钱罐”**
    - **问题**：想象你有一个只能存整数元硬币的存钱罐（量化参数空间）。你每天想存0.3元（微小梯度信号），但直接存会失败（更新量为0）。
    - **QES的解决方案**：你准备了一个“零钱袋”（误差残差 $e_t$）。第一天，你把0.3元放进零钱袋。第二天，你又想存0.3元，加上零钱袋里的0.3元，总共0.6元，还是不够1元。第三天，再存0.3元，零钱袋里变成0.9元。第四天，再存0.3元，零钱袋里变成1.2元。这时，你拿出1元硬币投入存钱罐（触发离散更新 $\Delta W_t = 1$），零钱袋里还剩0.2元（更新残差 $e_t$）。这样，通过累积，你成功地将微小的“存款”转化为了有效的“投资”，这就是**累积误差反馈**。
    - **无状态种子重放**：你不想随身携带一个记录每天存了多少钱的账本（高精度误差向量），因为太占地方。于是你只记录每天“存钱”这个动作的“种子”（比如当天的日期和心情）。当你需要知道零钱袋里有多少钱时，你根据这些种子，重新模拟一遍过去几天的存钱过程，就能算出当前零钱袋的余额。这就是**种子重放**。

- **“时间等价性”的通俗解释**：
    - 想象你（物理量化参数 $W_t$）和一个理想中的你（虚拟连续参数 $\Theta_t$）在赛跑。理想中的你每一步都迈得精准且连续。而你只能在整数米的标记点上落脚（离散网格）。
    - 普通方法（如QuZO）是：理想中的你迈出0.3米，你也想迈0.3米，但规则不允许，你只能原地踏步（停滞）或随机迈出1米（不准确），导致你很快被甩开。
    - QES的方法是：你手里有一个“误差背包”（$e_t$）。理想中的你迈出0.3米，你虽然不能迈步，但你把0.3米记在背包里。下一次，他又迈0.3米，你背包里就有0.6米。直到某次，他迈了0.4米，你背包里总共1.0米，这时你正好可以向前迈出1米（离散更新），并把背包清空。这样，虽然你每一步的落脚点都是整数，但你的**平均位置**和理想中的你几乎完全一致，这就是“时间等价性”。

## 6. 补充说明（Additional Notes）

- **相关工作关联**：
    - **与梯度压缩的关系**：QES的累积误差反馈机制与通信高效训练中的1-bit SGD [Seide et al., 2014] 和误差反馈（Error Feedback）[Karimireddy et al., 2019] 有相似之处，但QES是首次将其应用于无反向传播的量化优化场景。
    - **与量化感知训练（QAT）的关系**：QAT通常需要全精度潜权重和STE，内存开销大。QES则完全在量化空间操作，无需潜权重。
    - **与LoRA的关系**：QLoRA冻结量化权重，只更新低秩适配器。QES则进行全参数更新，理论上表达能力更强。
