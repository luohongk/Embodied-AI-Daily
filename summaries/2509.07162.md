# First Plan Then Evaluate: Multi-Target Planning with Post-Planning Success Evaluation Improves Learning-Based Grasping Pipelines

- arXiv: https://arxiv.org/abs/2509.07162v2
- arXiv ID: 2509.07162

---

🔍 **核心问题**：传统生成器-评估器-规划器框架中，执行低排名抓取目标会降低成功率，而多次轨迹优化耗时且精度与效率难以兼顾。  
🛠 **方法**：提出“先规划后评估”框架，先对一组生成的抓取目标规划轨迹，再评估各轨迹终点处的抓取成功概率，最终执行最可能成功的轨迹。  
✨ **主要贡献**：在多种物体、生成器和运动规划器上显著优于传统方法，并在真实世界不同环境（如货架和桌面高度）中成功泛化。
