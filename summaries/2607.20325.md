# MR-Compare: A Mixed-Reality Framework for Spatially Grounded Visual Comparison of 3D Gaussian Splatting and Mesh Reconstructions with the Physical Environment

- arXiv: https://arxiv.org/abs/2607.20325v1
- arXiv ID: 2607.20325

---

🔍 **核心问题**：在混合现实环境中，如何实现3D高斯泼溅与网格重建结果相对于真实物理环境的空间对齐与视觉比较。

🛠 **方法**：基于PC连接的Meta Quest 3头显，开发了MR-Compare框架，包含两阶段配准流程和用于跨媒介比较的3D滑块，并提出了利用高斯各向异性的零样本滤波器以提升配准鲁棒性。

✨ **主要贡献**：在两种静态室内场景中，通过30人用户研究验证了框架在五种重建工作流下均达到厘米级平移误差，其中桌面端3DGS-MCMC工作流表现最优，且用户感知可用性高、工作负荷低。
