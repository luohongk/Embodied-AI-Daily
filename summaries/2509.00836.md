# One-Step Model Predictive Path Integral for Manipulator Motion Planning Using Configuration Space Distance Fields

- arXiv: https://arxiv.org/abs/2509.00836v3
- arXiv ID: 2509.00836

---

🔍 **核心问题**：解决机械臂运动规划中传统优化方法易陷入局部最小值、梯度消失，以及无梯度MPPI方法计算成本高的问题。

🛠 **方法**：将配置空间距离场（CDF）与模型预测路径积分（MPPI）结合，利用CDF的可靠梯度信息统一MPPI代价函数，并将预测步长缩短为单步。

✨ **主要贡献**：在2D环境实现近100%成功率，在7自由度Franka机械臂仿真中保持高成功率，控制频率超过750Hz，显著优于优化方法和标准MPPI基线。
