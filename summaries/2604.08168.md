# ViVa: A Video-Generative Value Model for Robot Reinforcement Learning

- arXiv: https://arxiv.org/abs/2604.08168v2
- arXiv ID: 2604.08168

---

🔍 **核心问题**：现有基于视觉语言模型的价值函数难以捕捉机器人操作中的时间动态和物理交互，导致长周期任务中价值估计不可靠。

🛠 **方法**：提出ViVa模型，将预训练视频生成器改造为联合预测未来本体感知和标量价值的模型，通过预期身体动态来锚定价值估计。

✨ **主要贡献**：在三个任务上取得最先进的指标评估结果，产生可靠的价值信号以准确追踪任务进度和检测执行错误；集成到RECAP框架后平均成功率达80%，展示了视频生成模型用于价值估计的潜力。
