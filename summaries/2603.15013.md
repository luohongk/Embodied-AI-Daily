# CycleRL: Sim-to-Real Deep Reinforcement Learning for Robust Autonomous Bicycle Control

- arXiv: https://arxiv.org/abs/2603.15013v2
- arXiv ID: 2603.15013

---

🔍 **核心问题**：解决传统控制方法在欠驱动、非线性动力学特性的自主自行车中，因模型失配和现实不确定性导致的适应性差、鲁棒性不足的问题。

🛠 **方法**：提出CycleRL框架，在高保真NVIDIA Isaac Sim环境中，利用近端策略优化（PPO）算法建立从感知到动作的直接映射，并设计复合奖励函数以同时实现平衡维持、速度跟踪和转向控制，同时通过系统域随机化缩小仿真到现实的差距。

✨ **主要贡献**：在仿真中实现99.90%的平衡成功率、1.15°的航向跟踪误差和0.18 m/s的速度跟踪误差，并通过硬件部署验证了深度强化学习在自主自行车控制中的有效性，展现出优于传统方法的适应性。
