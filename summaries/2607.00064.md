# Solution space path planning for supporting en-route air traffic control

- arXiv: https://arxiv.org/abs/2607.00064v1
- arXiv ID: 2607.00064

---

🔍 **核心问题**：现有路径规划算法与空中交通管制员实际需求脱节，缺乏兼具可解释性、计算高效且适配人类决策逻辑的冲突解脱方案。

🛠 **方法**：提出基于解空间框架的路径规划算法，集成距离、时间间隔和区域三种意图冲突检测方法，并设计顶点搜索（SSPPV）与边搜索（SSPPE）两种变体，在满足间隔标准、机动性限制等运营约束下生成无冲突路径。

✨ **主要贡献**：SSPPV结合区域冲突检测在MUAC Delta扇区场景中平均计算时间仅3.69毫秒，验证了算法在可解释性、实时性和解质量上的综合优势，为战术级空中交通管制提供了实用决策支持工具。
