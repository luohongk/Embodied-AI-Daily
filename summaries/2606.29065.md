# A Unified Framework for Multi-Contact Path Planning in the Rolling Robot Systems

- arXiv: https://arxiv.org/abs/2606.29065v1
- arXiv ID: 2606.29065

---

🔍 **核心问题**：解决多接触滚动机器人系统中，在无滑移非完整约束和曲面流形约束下的路径规划难题。  
🛠 **方法**：基于Montana接触坐标公式建立多球滚动运动学模型，在球面接触流形上构建Voronoi路线图，结合球冠障碍物和互斥区域进行流形碰撞检测，并通过流形一致的对数-指数平滑优化离散路径，最终利用正向仿真验证运动可行性。  
✨ **主要贡献**：首次提出统一框架实现多接触无滑移滚动路径规划，支持流形上的碰撞检测与路径平滑，为扩展到非球面几何、动态障碍物及物理机器人实验奠定基础。
