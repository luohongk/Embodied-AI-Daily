# FLYNN: Robust Neural Network for Robot Navigation using Fly Brain Topology

- arXiv: https://arxiv.org/abs/2607.00025v2
- arXiv ID: 2607.00025

---

🔍 **核心问题**：解决深度神经网络在面对新环境或感官缺失时脆弱性高、鲁棒性不足的问题。  
🛠 **方法**：基于果蝇大脑突触级连接组结构，构建循环神经网络（FLYNN），并在MuJoCo仿真中训练其执行视觉导航任务。  
✨ **主要贡献**：FLYNN在参数规模相近时性能媲美手工设计网络，且无需额外训练即可显著抵抗分布外数据和感官缺失（如完全失明），其内部状态的高表征模块化可能是鲁棒性的关键。
