# Do You Need a Frontier Model as a Citation Verifier? Benchmarking Rubric LLMs for Deep-Research Source Attribution

- arXiv: https://arxiv.org/abs/2607.08700v1
- arXiv ID: 2607.08700

---

🔍 **核心问题**：研究在深度研究系统的引用质量评估中，是否需要使用最昂贵的前沿模型作为评判者，以及如何校准评判者的偏差。

🛠 **方法**：构建对抗性长文本基准，使用8个来自3个模型系列的现成LLM评判者，对1248个评分决策（含378个争议案例）进行人工审核，从源相关性和事实支持两个维度评估引用质量。

✨ **主要贡献**：发现廉价模型（如GPT-5-mini）在源相关性上F1达0.908，而所有模型在事实支持上统计无显著差异；但模型在通过率漂移、假阳性率等方向性偏差上差异显著，表明标量F1掩盖了关键偏差，校准评判者无需最昂贵模型。
