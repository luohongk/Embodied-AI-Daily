# Robust bipedal locomotion on flowable slopes via foot-driven terrain manipulation

- arXiv: https://arxiv.org/abs/2607.11855v1
- arXiv ID: 2607.11855

---

🔍 **核心问题**：双足机器人在流动性地形（如颗粒斜坡）上行走时，足地接触会导致地表大变形和固-液转变，耦合地形与机器人动力学，导致性能下降或失败，而现有方法缺乏可靠手段表征流动性地形的动态影响。

🛠 **方法**：通过研究带“爪”结构（足底薄板）的小型双足机器人（1.4 kg）的足地动力学，系统分析爪间距对地形屈服和阻力的影响，并设计可主动调节爪深度的足部，以适应刚性和颗粒地形；进一步在大型自主双足机器人（15 kg）上验证原理。

✨ **主要贡献**：发现中等爪间距能优化力分布，使基底应力接近屈服阈值，实现高达30度颗粒斜坡的稳定行走；提出通过足部设计主动调控地形交互的“肢体中心”方法，替代传统依赖身体运动调节地形效应的“身体中心”控制策略。
