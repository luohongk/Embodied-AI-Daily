# Scalable Multi-Task Data Generation via Reinforcement Learning for Language-Conditioned Bimanual Dexterous Manipulation

- arXiv: https://arxiv.org/abs/2606.22471v1
- arXiv ID: 2606.22471

---

🔍 **核心问题**：训练双臂灵巧操作的通用策略时，缺乏大规模、高质量的数据集，现有方法（如人类遥操作）任务多样性有限，且基于强化学习的方法常依赖手工设计的任务特定奖励。

🛠 **方法**：提出一种基于强化学习的系统化数据生成流程，整合了可泛化的奖励设计、有效的域随机化以及语言条件任务标注，用于合成多样化的双臂灵巧操作数据集，并训练语言条件多任务策略。

✨ **主要贡献**：通过该流程生成的数据显著提升了模型在三个代表性操作任务上的泛化能力，为双臂灵巧操作提供了可扩展的高质量数据生成方案。
