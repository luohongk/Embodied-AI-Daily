# Improving Autonomous Nano-drones Performance via Automated End-to-End Optimization and Deployment of DNNs

- arXiv: https://arxiv.org/abs/2607.12593v1
- arXiv ID: 2607.12593

---

🔍 **核心问题**：解决纳米无人机（nano-UAV）在有限计算和内存资源下，高效部署视觉卷积神经网络（CNN）进行自主导航的难题，避免传统人工迭代开发流程的低效和易错性。

🛠 **方法**：采用自动化端到端优化与部署流程，将PULP-Dronet CNN从初始训练到闭环评估全流程集成在超低功耗多核系统芯片（ULP multicore SoC）上，通过工具链自动压缩和加速模型。

✨ **主要贡献**：相比原始手工设计的CNN，实现了2倍内存占用减少和1.6倍推理速度提升，同时保持相同预测精度；在实际飞行中，障碍物避让峰值刹车速度达1.65 m/s，熟悉环境自由飞行速度达1.96 m/s（基线为0.5 m/s），并能完成90度弯道循线，且计算功耗仅占无人机总功率预算的1.6%；开源了与Crazyflie 2.1兼容的完整软件设计。
