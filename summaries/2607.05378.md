# CompactionRL: Reinforcement Learning with Context Compaction for Long-Horizon Agents

- arXiv: https://arxiv.org/abs/2607.05378v1
- arXiv ID: 2607.05378

---

🔍 **核心问题**：长程交互的智能体LLM受限于有限上下文窗口，任务未完成时轨迹可能超出最大长度，而现有强化学习方法未有效整合上下文压缩技术。

🛠 **方法**：提出CompactionRL，通过联合优化任务执行与摘要生成，采用token级损失归一化和跨轨迹广义优势估计，使LLM智能体能从压缩后的长程轨迹中学习。

✨ **主要贡献**：在开源模型上实现一致性能提升，如GLM-4.5-Air在SWE-bench Verified上Pass@1达66.8%（+7.0），GLM-4.7-Flash提升5.5-6.8点，并已部署于GLM-5.2模型的RL训练流程。
