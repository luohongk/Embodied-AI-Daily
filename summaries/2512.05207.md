# Hierarchical Reinforcement Learning for the Dynamic VNE with Alternatives Problem

- arXiv: https://arxiv.org/abs/2512.05207v2
- arXiv ID: 2512.05207

---

🔍 **核心问题**：解决动态虚拟网络嵌入中，虚拟网络请求具有多种可选拓扑结构（VNEAP）时，因额外决策层带来的嵌入挑战，提升请求接受率和收益。

🛠 **方法**：提出分层强化学习框架HRL-VNEAP，高层策略负责选择最优替代拓扑或拒绝请求，低层策略将选定拓扑嵌入底层网络。

✨ **主要贡献**：在真实拓扑和多负载场景下，HRL-VNEAP相比最强基线，接受率提升20.7%，总收益提升36.2%，收益成本比提升22.1%，并量化了与最优解的差距。
