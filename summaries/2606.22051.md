# GeoFlow-SLAM++: A Robust Multi-Camera Visual-Inertial SLAM System with Relocalization

- arXiv: https://arxiv.org/abs/2606.22051v1
- arXiv ID: 2606.22051

---

🔍 **核心问题**：单目和RGB-D视觉惯性SLAM系统受限于视野狭窄、传感器特定失效模式及跨会话重定位不可靠等问题。

🛠 **方法**：提出GeoFlow-SLAM++，一种紧耦合多相机视觉惯性SLAM系统，支持ORB和基于SuperPoint+LightGlue的神经网络特征前端，融合多相机重投影约束、IMU预积分、跨视角地点识别及双流光流/特征跟踪，并可选集成跨视角伪深度预测。

✨ **主要贡献**：在EuRoC、OpenLORIS、TUM、Hilti及自采数据集上验证，神经网络特征前端提升外观挑战场景鲁棒性，多相机设计在Hilti上实现高精度定位，统一跨视角重定位在手持数据集上达到与LiDAR相当的性能。
