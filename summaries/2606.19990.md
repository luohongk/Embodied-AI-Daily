# Reward as An Agent for Embodied World Models

- arXiv: https://arxiv.org/abs/2606.19990v1
- arXiv ID: 2606.19990

---

🔍 **核心问题**：现有强化学习方法在优化世界模型时，因依赖保守的轨迹采样而限制了探索的多样性和动态发现，且缺乏可靠的验证机制，导致扩展探索易引发奖励欺骗（reward hacking），无法实现真正的性能提升。

🛠 **方法**：提出“奖励即智能体”（Reward as an Agent）框架，主动评估生成行为以提供鲁棒奖励信号，缓解分布偏移下的奖励欺骗；同时引入动态感知轨迹多样化方法（DynDiff-GRPO），通过扩展动作空间探索来丰富轨迹和状态-动作覆盖，两者结合形成可靠奖励基础与多样化采样的统一方案。

✨ **主要贡献**：在具身世界模型中验证了该方法，有效缓解了奖励欺骗问题，并在多个开源世界模型上显著提升了准确性，证明了基于稳健验证的广泛探索可以成功扩展。
