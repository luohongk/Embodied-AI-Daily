# Learning Fair Pareto-Optimal Policies in Multi-Objective Reinforcement Learning

- arXiv: https://arxiv.org/abs/2606.18111v1
- arXiv ID: 2606.18111

---

🔍 **核心问题**：在多目标强化学习中，现有方法无法在动态或未知用户偏好下提供一组既公平又帕累托最优的策略集合。  
🛠 **方法**：提出三种算法，包括将广义基尼福利函数与多策略多目标Q学习结合、状态增强的多策略多目标Q学习（用于非平稳策略）及其随机策略扩展。  
✨ **主要贡献**：证明了公平策略存在于凸覆盖集中，并验证了非平稳和随机策略能通过适应历史不平等性提升公平性，实验表明该方法可学习适应不同偏好的公平策略集。
