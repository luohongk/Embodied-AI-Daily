# MetaSkill-Evolve: Recursive Self-Improvement of LLM Agents via Two-Timescale Meta-Skill Evolution

- arXiv: https://arxiv.org/abs/2607.05297v1
- arXiv ID: 2607.05297

---

🔍 **核心问题**：现有LLM智能体通过技能自改进提升性能，但改进过程本身是固定的、非递归的，无法适应任务多样性和持续优化需求。

🛠 **方法**：提出MetaSkill-Evolve框架，采用双时间尺度机制——任务技能在快速循环中进化，而元技能（包含分析器、检索器、分配器、提议器和进化器五个组件）在慢速循环中自我改进，且所有组件共享同一冻结基座模型。

✨ **主要贡献**：首次实现智能体技能改进过程的递归自进化，在OfficeQA、SealQA和ALFWorld三个基准上分别提升+23.54、+16.09和+1.92个百分点的测试准确率，显著优于无技能、静态技能和单层进化基线。
