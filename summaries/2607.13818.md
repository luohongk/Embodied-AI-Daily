# Learning Robust Execution in Robotic Manipulation with Agentic Reinforcement Learning

- arXiv: https://arxiv.org/abs/2607.13818v1
- arXiv ID: 2607.13818

---

🔍 **核心问题**：机器人操作任务中，由于不确定性、长时执行和误差累积，执行稳定性差且易失败，现有视觉-语言-动作模型缺乏执行质量评估与恢复机制。

🛠 **方法**：提出两种运行时执行质量评估指标，并构建一个智能体强化学习框架，通过高层决策（而非直接学习低层动作）选择少量执行模式来调节过程，在执行退化时触发恢复机制，使机器人回到先前正常状态。

✨ **主要贡献**：在LIBERO基准测试中，标准设置下成功率提升最高达13.7%，干扰设置下提升最高达39.2%，显著增强了执行鲁棒性。
