# FLORA: A deep learning approach to predict forest attributes from heterogeneous LiDAR data

- arXiv: https://arxiv.org/abs/2606.32023v1
- arXiv ID: 2606.32023

---

🔍 **核心问题**：解决异构LiDAR数据（传感器、飞行参数、季节等差异）下，全国尺度森林属性（如优势高、总蓄积量等）预测的鲁棒性问题。

🛠 **方法**：提出FLORA深度学习框架，采用八叉树（octree）骨干网络提取点云特征，并通过后期融合门控机制整合生态与时空辅助变量，在法国32,052个国家森林清查样地上训练单一模型。

✨ **主要贡献**：单一模型在叶期与非叶期数据上均优于季节专用模型，实现跨季节鲁棒性；辅助变量对物种特异性蓄积量预测贡献显著；优势高预测rRMSE约12.3%（R²=0.88），总蓄积量rRMSE约39%（R²=0.74），为异构国家LiDAR计划提供稳健基线。
