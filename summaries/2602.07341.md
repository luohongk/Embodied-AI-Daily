# Scalable Dexterous Robot Learning with AR-based Remote Human-Robot Interactions

- arXiv: https://arxiv.org/abs/2602.07341v2
- arXiv ID: 2602.07341

---

🔍 **核心问题**：解决灵巧机器人手臂-手系统在操作任务中，因缺乏高效数据采集和策略训练方法而导致的效率低下问题。  
🛠 **方法**：提出两阶段方法，先通过基于增强现实（AR）的远程人机交互系统收集专家演示数据，进行行为克隆（BC）预训练；再结合对比学习增强的强化学习（RL）和事件驱动增强奖励，优化策略并提升安全性。  
✨ **主要贡献**：在PyBullet仿真和真实实验中，该方法相比基线显著加速训练过程，并提高操作任务的成功率；消融实验证实对比学习有效防止策略崩溃。
