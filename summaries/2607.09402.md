# Data-Efficient Deep Learning: Empirical Guidelines for Training Set Size Estimation in Inertial Sensor Classification

- arXiv: https://arxiv.org/abs/2607.09402v1
- arXiv ID: 2607.09402

---

🔍 **核心问题**：惯性传感器分类任务（如人体活动识别）中，深度学习模型依赖大规模数据集，但缺乏确定达到目标精度所需最小样本量的数据驱动指南。

🛠 **方法**：通过系统实证评估学习曲线收敛速率，提出统一框架分析二分类与多分类场景下的性能，并推导出基于数据集大小的经验公式；定义稳定性点指标，即学习曲线稳定在渐近最大值附近所需样本量。

✨ **主要贡献**：发现精度遵循一致的对数增长模式，证明模型在远少于传统启发式方法建议的样本量下即可达到实际稳定性；提供从小规模试点研究外推总数据需求的通用框架，优化记录成本与模型可靠性之间的权衡。
