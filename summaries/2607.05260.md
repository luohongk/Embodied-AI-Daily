# Shifting from Discrete to Continuous Reference Data: QSM-Derived Horizontal Tree Biomass Distribution for Deep Learning Biomass Estimation

- arXiv: https://arxiv.org/abs/2607.05260v1
- arXiv ID: 2607.05260

---

🔍 **核心问题**：传统基于LiDAR的地上生物量（AGB）估算依赖离散样地聚合数据，存在边界效应，导致小样地模型性能下降。

🛠 **方法**：利用定量结构模型（QSM）生成连续水平生物量分布（HBD）参考数据，训练稀疏3D U-Net模型，并与标准森林清查（FI）及无边界效应的QSM样地聚合参考进行对比。

✨ **主要贡献**：在100 m²小样地中，HBD参考将相对均方根误差（RRMSE）降低16.84±4.37%，R²提升0.22±0.05，证明连续HBD参考能有效校正边界效应，显著提升小样地AGB估算性能。
