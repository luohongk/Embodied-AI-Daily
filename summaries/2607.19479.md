# ModPack: An Extensible Teleoperation Interface for Bimanual Mobile Manipulation

- arXiv: https://arxiv.org/abs/2607.19479v1
- arXiv ID: 2607.19479

---

🔍 **核心问题**：现有遥操作系统通常针对特定机器人硬件和任务领域定制，缺乏可扩展性和适应性，难以支持多样化的机器人形态和任务需求。

🛠 **方法**：提出ModPack模块化遥操作系统，核心是一个集成了计算、电源、通信和数据存储的可穿戴“背包”，并支持即插即用的功能模块，包括关节级遥操作（带触觉反馈）、移动操作和主动感知。

✨ **主要贡献**：在两种不同机器人平台和真实移动操作任务中验证了系统的灵活性和可复用性，并开源了完整的硬件设计和软件栈，为数据收集和策略学习提供了统一框架。
