# Safe Navigation using Neural Radiance Fields via Reachable Sets

- arXiv: https://arxiv.org/abs/2604.26899v2
- arXiv ID: 2604.26899

---

🔍 **核心问题**：解决机器人在复杂障碍环境中实现安全导航的问题，需处理障碍物、目标及自身几何形状的多样性。  
🛠 **方法**：利用可达集表示机器人在状态空间中的实时能力，结合神经辐射场（NeRF）计算和存储障碍物或本体的体积表示，并通过含线性矩阵不等式约束的最优控制进行路径规划。  
✨ **主要贡献**：在两种不同场景下验证了该方法，通过可达集与约束最优控制成功实现了多障碍环境中的安全导航。
