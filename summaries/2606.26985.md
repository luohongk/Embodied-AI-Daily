# Vis4GS: A Visual Analytic Tool for 3D Gaussian Splatting Reconstruction

- arXiv: https://arxiv.org/abs/2606.26985v1
- arXiv ID: 2606.26985

---

🔍 **核心问题**：3D高斯泼溅（3DGS）的优化过程难以解释，现有工具仅展示最终重建场景，无法分析高斯属性如何导致可见伪影或在训练中演变。

🛠 **方法**：提出Vis4GS，一个多视图可视化分析工具，通过交互式高斯分析视图、属性时间线视图、高斯密集化树视图及日志控制面板，将渲染伪影与高斯属性、视角覆盖、训练进度和高斯谱系关联。

✨ **主要贡献**：支持高斯选择、模糊/针状伪影评分、视角覆盖分析及多尺度谱系探索（克隆/分裂/剪枝事件），实现从场景级伪影到原始级证据的结构化诊断，用户研究证实其可用性与伪影理解能力显著优于原始3DGS查看器。
