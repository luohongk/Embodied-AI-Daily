# SparseLGS: Sparse View Language Embedded Gaussian Splatting

- arXiv: https://arxiv.org/abs/2412.02245v3
- arXiv ID: 2412.02245

---

🔍 **核心问题**：现有基于高斯泼溅的3D语义场景理解方法依赖密集多视角输入，在稀疏或无位姿输入的真实场景中难以应用。

🛠 **方法**：提出SparseLGS，利用学习型密集立体模型处理无位姿稀疏输入，通过三步区域匹配解决多视角语义不一致问题，并提取低维CLIP特征构建双射关系以降低存储成本，同时引入重建损失优化高斯位置与形状。

✨ **主要贡献**：首次解决稀疏无位姿输入下的3D语义场重建问题，仅需3-4视图即可达到与密集输入方法相当的语义场质量，且在相同稀疏输入下质量显著领先，计算速度提升5倍。
