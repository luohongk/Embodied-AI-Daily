# Extending a Large View Synthesis Model for Multi-view Panoptic Segmentation

- arXiv: https://arxiv.org/abs/2607.19765v1
- arXiv ID: 2607.19765

---

🔍 **核心问题**：如何利用大型视图合成模型实现多视图全景分割，而无需显式3D重建或针对分割任务的模型训练。

🛠 **方法**：提出一种流水线，将输入视图的全景标签编码为二进制通道表示，直接输入冻结的视图合成模型，通过跨视图注意力机制传播到新视图，从而生成目标视图的分割结果。

✨ **主要贡献**：首次将大型视图合成模型从外观渲染扩展到3D场景理解，在ScanNet上分割质量与基于高斯的方法相当，新视图合成性能提升超过7 dB，且无需微调即可跨数据集迁移至Replica。
