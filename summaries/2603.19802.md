# Evaluating Vision Foundation Models for Pixel and Object Classification in Microscopy

- arXiv: https://arxiv.org/abs/2603.19802v2
- arXiv ID: 2603.19802

---

🔍 **核心问题**：在显微镜图像分析中，传统基于手工特征的浅层学习方法在像素分类（语义分割）和对象分类任务上仍被广泛使用，但缺乏大规模预训练数据，本研究旨在探索视觉基础模型（VFMs）能否提升这些任务的性能。

🛠 **方法**：评估多种通用型（SAM、SAM2、SAM3、DINOv3）和领域专用型（μSAM、PathoSAM、KRONOS）视觉基础模型，结合浅层学习与注意力探针技术，在五个多样且具有挑战性的显微镜数据集上进行实验。

✨ **主要贡献**：实验证明视觉基础模型相比手工特征在像素和对象分类上取得一致改进，为实际应用提供了明确路径，并建立了显微镜领域视觉基础模型的基准，指导未来发展方向。
